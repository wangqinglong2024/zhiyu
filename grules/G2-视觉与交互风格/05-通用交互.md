# 05 · 通用交互规范

> 引用：[00-index.md](./00-index.md) · [04-状态与组件.md](./04-状态与组件.md)
> 落地：通过 `ui-kit` 与项目级 hooks 强制约定。

---

## 一、防重复提交

- **方案**：所有提交按钮在请求未返回前 **立即** 进入 `loading + disabled` 双重态。
- **实现**：项目级 `useAsyncAction(fn)` hook：
  - 返回 `{ run, isLoading }`；
  - 重复调用在第一次未完成时被忽略（不排队）；
  - 异常自动 Toast（错误码 → i18n 文案）。
- **按钮约束**：所有提交按钮必须接收 `loading` prop；不允许自实现"点一次后变灰"逻辑。
- **表单整体**：表单未通过校验时主按钮禁用（`disabled`）。

## 二、搜索

| 项 | 默认值 |
|----|--------|
| 触发 | **自动**（输入即查询）|
| 防抖 | **300ms** |
| 最小字符 | 1（中文 1，拉丁 2）|
| 清空 | 输入框右侧 X 图标，点击立即清空并刷新列表 |
| 历史 | 不保存（隐私） |
| 占位文案 | i18n `common.search.placeholder`，含示例如"搜索课程 / 拼音 / 标签" |

- 搜索失败 / 无结果 → 表格内"未找到匹配项"空状态，提供"清空筛选"按钮。
- 高级筛选位于搜索框右侧抽屉触发图标 `sliders` 内。

## 三、列表刷新策略

| 操作 | 默认策略 |
|------|---------|
| 创建成功 | **重新请求当前页**（保证排序、计数一致）|
| 编辑成功 | **重新请求当前页**（避免乐观更新与服务端校验冲突）|
| 删除成功 | **乐观移除该行 + 后台请求**；失败时插回并 Toast |
| 状态切换（启用/禁用、上下架）| 乐观更新 + 失败回滚 |
| 列表筛选 / 排序 / 翻页 | 重新请求 |
| 跨页选中 | 不支持跨页选中（避免误操作）；当前页全选用 checkbox |

> 实现：统一通过 TanStack Query 的 `invalidateQueries(['x', 'list', filters])`；项目内不允许直接 `setState` 篡改列表数据。

## 四、表单离开保护

- **启用场景**：编辑型表单（含 Modal 内）。
- **触发条件**：表单 dirty（任意字段被修改）且未保存。
- **行为**：
  - 同站路由跳转：弹"未保存的更改将丢失，确认离开吗？"确认弹窗（[04 §八](./04-状态与组件.md)）；
  - 关闭浏览器 / 刷新：触发原生 `beforeunload` 提示；
  - Modal / Drawer 关闭按钮 / 点遮罩（如允许）：弹同样的确认弹窗。
- **实现**：`useFormGuard(form)` hook，订阅 React Hook Form `formState.isDirty`。

## 五、分页

| 项 | 值 |
|----|---|
| 默认每页 | **20** |
| 可选每页 | 10 / 20 / 50 / 100 |
| 分页器位置 | 表格底部右侧 |
| 显示内容 | 上一页 / 页码（最多 5 个数字 + 省略号）/ 下一页 / 总条数 / 每页条数选择 |
| 跳页 | 桌面提供"跳至第 N 页"输入框；移动端隐藏 |
| 移动端 | 仅显示"上一页 / 当前 N / 下一页 / 共 X 条"，每页条数选择隐藏 |
| URL 同步 | `?page=2&pageSize=20`，便于分享与刷新保留状态 |

## 六、键盘与无障碍

- **全局快捷键**：
  - `?`：弹出"快捷键帮助"Modal；
  - `g h`：返回首页；
  - `g a`（管理端）：跳 `/admin`；
  - `/`：聚焦顶部搜索框（如当前页存在）；
  - `ESC`：关闭最上层弹窗 / Drawer / Toast。
- 任意可交互元素必须可 Tab 聚焦；focus ring 使用 `--ring`（红色半透明，不被毛玻璃吞没）。
- 模态焦点陷阱由 Radix 提供；关闭后焦点回到触发按钮。

---

## 七、复制 / 文件下载 / 上传

- **复制**：使用 `navigator.clipboard.writeText`，成功后右上 Toast"已复制"。
- **下载**：通过隐藏 `<a>` + `URL.createObjectURL`；大文件后台生成 → 完成后 Toast 内提供下载按钮。
- **上传**：
  - Dropzone + 点击两种方式；拖入时容器 `--brand` 红色虚线 + 浅红底；
  - 文件类型 / 大小限制以 Toast 失败提示；
  - 上传中显示进度条（`--brand`），可取消。

---

## 八、错误反馈

- **可恢复错误（4xx）**：内联在表单 / 字段，**不弹 Toast**；
- **不可恢复错误（5xx / 网络）**：右上 Toast，文案走 i18n（错误码 → `errors.50001` 等）；
- **未授权 401**：自动跳 `/auth/login?redirect=...`；
- **无权限 403**：跳 `/403` 毛玻璃页（含返回首页按钮）；
- **资源不存在 404**：跳 `/404` 毛玻璃页（含搜索建议）。

> 错误展示样式：见 [04 §一 状态色](./04-状态与组件.md)；与品牌红 `--brand` 自然融合。

---

## 九、动效 / 过渡

- **页面切换**：内容区淡入 120ms（`opacity 0→1`），无水平位移；
- **菜单项 hover**：颜色变化 `--motion-fast`；
- **按钮 hover/active**：背景与阴影渐变 `--motion-base`；
- **弹窗 / Drawer 进出**：`--motion-slow`，详见 [04 §五 §六](./04-状态与组件.md)；
- 所有动效尊重 `prefers-reduced-motion`。

# 04 · 状态与组件规范

> 引用：[00-index.md](./00-index.md) · [01-设计Token.md](./01-设计Token.md)
> 实现库：**shadcn/ui** + Radix Primitives，主题层在 `system/packages/ui-kit/`。
> 规则：所有组件**毛玻璃化**（按钮、Tab、Toast、Modal、Drawer 等），见 [01 §四](./01-设计Token.md)。

---

## 一、状态标签色彩映射

> 通用规则。状态标签全部使用 `<StatusTag status="..." />` 组件，禁止手写颜色。
> 颜色严格遵守"黑白红 + 中性语义"。错误 / 拒绝 / 取消统一使用品牌红。

| 状态语义 | 背景色 | 文字色 | 适用枚举 |
|---------|--------|-------|---------|
| 初始 / 草稿 | `--bg-active` | `--text-secondary` | `draft / pending` |
| 进行中 / 处理中 | `--bg-active` + 1px `--border-strong` 虚线 | `--text-primary` | `active / processing / in_progress` |
| 成功 / 完成 | `--success-soft` | `--success` | `done / completed / success / paid` |
| 警告 / 待注意 | `--warning-soft` | `--warning` | `warning / pending_review / overdue` |
| 失败 / 拒绝 / 取消 | `--danger-soft` | `--danger`（= `--brand`）| `failed / rejected / cancelled / refunded` |
| 已归档 / 已关闭 | `--bg-active` | `--text-tertiary` | `archived / closed` |

- 形状：`radius-pill`，高度 22px，内边距左右 8px，字号 `text-tag`。
- 图标可选（在文字左 4px），尺寸 12px。

## 二、按钮

| 类型 | 场景 | 背景 | 文字 | 边框 | 高度 | 圆角 | 备注 |
|------|------|------|------|------|------|------|------|
| 主按钮 `primary` | 创建 / 提交 | `--brand` | `--text-on-brand` | 无 | 36 / 32 / 28 | `radius-sm` | hover→`--brand-hover` ；active→`--brand-active` |
| 次要按钮 `secondary` | 筛选 / 导出 | `glass-button`（毛玻璃）| `--text-primary` | `--border-subtle` | 36 / 32 / 28 | `radius-sm` | hover 增加 `--bg-hover` |
| 危险按钮 `danger` | 删除 / 重置 | `--danger-soft` | `--danger` | `--danger` 1px | 36 | `radius-sm` | hover→实色 `--brand` 底 + 白字 |
| 文字按钮 `ghost` | 表格内操作 | 透明 | `--text-primary` | 无 | 28 | `radius-sm` | hover 加 `--bg-hover` |
| 链接按钮 `link` | 表单内跳转 | 透明 | `--brand` | 无 | 行内 | — | hover 下划线 |
| 禁用态 | 条件不满足 | `--bg-active` | `--text-disabled` | `--border-subtle` | 同上 | 同上 | `cursor: not-allowed`，禁止 hover 变化 |
| Loading 态 | 提交中 | 同原态，左侧加 14px Spinner | 文字保持 | — | 同上 | — | 自动禁用点击；详见 [05 §一](./05-通用交互.md) |

- 尺寸三档：`sm=28 / md=32 / lg=36`，默认 `md`；移动端默认 `lg`（更易触控）。
- 主按钮全局只允许同时存在 1 个（每个区段 / 弹窗内）。
- 图标按钮（icon-only）使用 36×36 / 32×32 正方形，提供 `aria-label`。

## 三、表格

| 属性 | 值 |
|------|---|
| 容器 | `<GlassCard>` 包裹，圆角 `radius-md` |
| 表头背景 | `transparent`（毛玻璃透出底色）；文字 `text-caption`，字重 600，色 `--text-secondary` |
| 行高 | 桌面 48px / 移动 56px |
| 斑马纹 | **不开**（与毛玻璃叠加易脏）|
| hover 行 | `--bg-hover` |
| 选中行 | `--brand-soft` 底 + 文字 `--text-primary` |
| 边框 | 行间 1px `--border-subtle`；外层 `<GlassCard>` 自带边框，表格自身无外边框 |
| 操作列 | 固定右侧；按钮间距 8px；文字按钮（ghost）|
| 排序图标 | `chevrons-up-down` 12px，激活后 `chevron-up/down` + `--brand` |

### 3.1 空状态

- 表格内容空：行内插画（48×48）+ 文字"暂无数据"（`text-secondary`）+ 可选 `New` 主按钮，居中，纵向 `space-6` 留白。

### 3.2 加载

- 骨架屏行数 = 当前 `pageSize`，每行高度 = 表格行高，4 个矩形条占位（按列宽分布）。

### 3.3 大数据量

- 行数 > 100 必须使用 TanStack Virtual 虚拟滚动；表头 sticky。

## 四、表单

| 属性 | 值 |
|------|---|
| 标签位置 | 桌面**上方**；移动端上方 |
| 标签字号 / 字重 | `text-caption` / 500 |
| 字段间距 | `space-4`（16px）|
| 输入框高度 | 36px（默认）/ 32px（紧凑）/ 44px（移动）|
| 输入框背景 | `glass-button` 风格（半透明、`--glass-blur-sm`）|
| 输入框边框 | `--border-strong` 1px；focus → 2px ring `--ring`（详见 01 §1.1）|
| 必填标记 | 标签后 4px 处，红色 `*`，`--brand` |
| 错误提示颜色 | `--danger` |
| 错误提示字号 | `text-caption` |
| 错误提示位置 | 字段下方 4px，与标签同列对齐 |
| 字数限制提示 | 字段右下角，`text-caption text-tertiary`，超限变 `--danger` |
| 帮助文案 | 字段下方 4px，`text-caption text-tertiary`，与错误提示**互斥**（错误时隐藏帮助）|

- Select / Combobox / DatePicker 弹层一律毛玻璃面板（`glass-panel`）；
- 表单按钮区：右下角，主按钮在右、取消按钮在左（与弹窗一致）。

## 五、弹窗 Modal

| 属性 | 值 |
|------|---|
| 默认宽度 | 480px |
| 大弹窗宽度 | 720px |
| 全屏断点 | < 640px 下宽度 = `100vw - 16px`，高度 = `90vh`，从底部弹起 |
| 遮罩层 | `--bg-overlay` |
| 容器 | `glass-modal`（高强度毛玻璃 + 大圆角 `radius-lg`）|
| 标题字号 / 字重 | `text-h3` / 600 |
| 内容区内边距 | 24px |
| 底部按钮对齐 | **右对齐**：取消（次要按钮）在左、确认（主按钮）在右 |
| 关闭方式 | 右上角 X 图标按钮 + ESC + 点遮罩（**危险弹窗禁用点遮罩关闭**）|
| 进入动效 | 透明度 0→1 + 缩放 0.96→1，`--motion-slow` |

## 六、侧边抽屉 Drawer

| 属性 | 值 |
|------|---|
| 默认宽度 | 桌面 480px / 移动 100% |
| 滑出方向 | **右侧** |
| 容器 | `glass-modal` 风格 |
| 遮罩层 | `--bg-overlay` |
| 关闭方式 | 右上角 X + ESC + 点遮罩 |
| 顶部 | 标题（`text-h3`）+ X 按钮，下方 1px `--border-subtle` |
| 底部 | 操作区固定底部：取消左、主按钮右；含 `safe-area-inset-bottom` |

## 七、消息提示 Toast

| 属性 | 值 |
|------|---|
| 位置 | **顶部居中**，距离顶栏底部 16px |
| 容器 | `glass-toast`（毛玻璃 + `radius-md` + `shadow-md`）|
| 进入 | 自顶向下 16px → 0，`--motion-base` |
| 自动消失 | 成功 3s / 错误 5s / 警告 4s |
| 同时上限 | 3 条，超出时最旧的提前消失 |
| 成功 | 左侧 16px `check-circle` 图标 `--success`，文字 `--text-primary` |
| 错误 | 左侧 `x-circle` 图标 `--danger`，文字 `--text-primary` |
| 警告 | 左侧 `alert-triangle` 图标 `--warning`，文字 `--text-primary` |
| 信息 | 左侧 `info` 图标 `--text-secondary`，文字 `--text-primary` |
| 关闭 | 右侧 X 图标按钮（鼠标悬停时显示）|

## 八、确认弹窗（危险操作）

| 属性 | 值 |
|------|---|
| 触发 | 删除 / 重置 / 不可撤销操作 |
| 容器 | Modal（默认宽度 480px，`glass-modal`）|
| 标题 | "确认删除 / 确认重置 / 确认 XX"（i18n key `confirm.delete.title` 等）|
| 内容 | "确定要 XX「{name}」吗？此操作不可撤销。"（i18n + 插值）|
| 图标（标题左侧）| `alert-triangle` 20px `--danger` |
| 取消按钮 | 次要按钮 `secondary` |
| 确认按钮 | **危险按钮 `danger`** |
| 关闭 | X + ESC（**禁用点遮罩关闭**）|
| 二次确认（高危）| 需输入对象名称匹配后才能点击确认按钮；适用于"删除用户 / 删除课程"等 |

## 九、组件目录约定

- 全部毛玻璃组件来自 `system/packages/ui-kit/src/components/`：
  - `GlassNav`、`GlassCard`、`GlassButton`、`GlassModal`、`GlassDrawer`、`GlassTabs`、`GlassToast`、`StatusTag`、`Skeleton`、`PageHeader`、`PageContent`、`EmptyState`、`Confirm`。
- 业务页面**禁止**直接使用 shadcn/ui 原始组件（如 `<Card>`、`<Dialog>`），必须经 `ui-kit` 包二次封装以保证毛玻璃主题统一。

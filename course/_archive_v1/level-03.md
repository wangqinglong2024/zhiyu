# Level 3 · 段落表达

> 对标：中国小学三年级 · HSK 3 · CEFR B1 初

---

## 一、等级定位

三年级是小学语文最关键的转折点：从"字词"过渡到"段落"。本级完成：
1. **识字突破 900**：累计 **900 字**，会写 700 字
2. **段落意识觉醒**：理解总分、并列、因果三大段落逻辑
3. **写作从句到篇**：能写 200-300 字完整记叙文
4. **查字典能力**：部首查字法、音序查字法独立使用

学习者完成本级后：
- 能独立阅读无拼音的少儿读物（如《小猪唏哩呼噜》）
- 能写小日记、简单说明、读后感
- 能参与小组讨论、主题演讲 1 分钟

---

## 二、学习目标

| 能力 | 目标 |
|---|---|
| 听 | 听懂 1-2 分钟故事并复述大意；辨识故事起因、经过、结果 |
| 说 | 讲完整故事（5-8 句）；当众演讲 1 分钟；小组讨论发表观点 |
| 读 | 独立读 500-800 字短篇；理解段意、主旨；学会圈划关键词 |
| 写 | 写 200-300 字记叙文；分段表达；用上 3 个以上成语与修辞 |
| 文化 | 了解中国地理概况、历史朝代歌、名人故事（孔子、李白、鲁班） |

## 四、九大知识模块详解

### 模块 1 · 汉字与书写
**新增 300 字重点**：形声字大量涌入
- **形声字系统认知**：
  - 形旁表义，声旁表音
  - 举例：清、请、情、晴、睛、精（青作声旁）
  - 举例：跑、抱、炮、饱、袍、泡（包作声旁）
- **同音字辨析**：在 / 再；做 / 作；那 / 哪
- **多音字入门 20 个**：为、得、长、乐、行、少、种、教、空、数、好、觉、答、干、重、还、着、发、调、和
- **书写质量要求**：结构匀称、笔画饱满、田字格四边留白均匀
- **硬笔书法启蒙**：中性笔 / 钢笔基础

### 模块 2 · 拼音与语音
- 查字典音序法复习
- 同音字区分：shi 的同音字群（是、十、事、时、市、师、实、食、史、世）
- 多音字辨析训练
- 朗读能力：停顿、重音、语调升降
- 绕口令：《四是四》《吃葡萄不吐葡萄皮》《板凳与扁担》

### 模块 3 · 词汇（累计 2200 词 / 本级新增 1000）
> HSK 3 大纲词汇全覆盖

**新增分类**：
- **近反义词系统**（200 对）：
  - 近义词：漂亮-美丽、高兴-快乐、喜欢-喜爱、奔跑-飞奔
  - 反义词：大-小、快-慢、明-暗、前-后、胖-瘦、勇敢-胆小
- **形容词扩展**（150 词）：
  - 外貌：胖乎乎、瘦瘦的、白白的、黑黝黝、圆圆的
  - 性格：勇敢、胆小、活泼、文静、聪明、糊涂、老实、调皮
  - 心情：激动、失望、羡慕、骄傲、谦虚
- **动词搭配**（200 词）：
  - "打"系列：打扫、打字、打球、打架、打扮、打算
  - "开"系列：开门、开车、开会、开心、开学、开始
- **连词**（30 词）：因为、所以、虽然、但是、如果、就、只要、才、还、而、并且、而且、不但、还、既、又、或者
- **拟声词**（40 词）：叮咚、哗啦、呼呼、嘀嗒、咕噜、汪汪、喵喵、哞哞、呱呱、咯咯
- **成语（本级新增 30 个）**：
  - 历史成语：完璧归赵、负荆请罪、纸上谈兵、凿壁偷光、悬梁刺股、囊萤映雪
  - 寓言成语：揠苗助长、掩耳盗铃、刻舟求剑、井底之蛙、杯弓蛇影、狐假虎威
  - 人物品质：大公无私、坚贞不屈、任劳任怨、废寝忘食、鞠躬尽瘁
  - 景物描写：春意盎然、绿树成荫、硕果累累、白雪皑皑、波光粼粼、星罗棋布

### 模块 4 · 语法与句法
**新句型（21-35 号）**：
21. `S + 把 + O + V + 结果`（他把碗打破了。）
22. `S + 被 + O + V + 了`（花瓶被猫打翻了。）
23. `一边 … 一边 …`（他一边走一边唱。）
24. `不但 … 而且 …`（他不但聪明，而且勤奋。）
25. `只要 … 就 …`（只要努力就能成功。）
26. `只有 … 才 …`（只有刻苦才能进步。）
27. `除了 … 以外`（除了小明，都来了。）
28. `越 … 越 …`（雨越下越大。）
29. `是 … 的`（他是昨天来的。）——强调时间/方式
30. `连 … 都 …`（连小孩都知道。）
31. `跟 … 一样 / 不一样`
32. `像 … 一样`（他跑得像风一样快。）
33. `没 … 那么 / 这么 …`
34. `从 … 到 …`（从北京到上海。）
35. `往 / 向 + 方位`（往前走。）

**结构扩展**：
- 主谓宾补：我吃完了饭。
- 定语前置：我的书 / 漂亮的花 / 三个学生
- 状语：认真地听 / 高兴地笑 / 在家里睡

**查字典能力（重点）**：
- 部首查字法四步：数部首笔画 → 查部首页码 → 数剩余笔画 → 定字
- 音序查字法三步：定声母 → 按韵母排序找页 → 对字定音义
- 一字多义辨识：根据上下文选最合适的义项

> 以上语法点对标 HSK 3 语法考点

### 模块 5 · 修辞（扩展到 6 种）
1. **比喻**（明喻、暗喻、借喻）
   - 明喻：月亮像玉盘。
   - 暗喻：妈妈是我的港湾。
   - 借喻：小白鸽（喻和平）飞过天空。
2. **拟人**：太阳公公起床了。
3. **排比**：他爱画画，爱唱歌，爱跳舞。
4. **夸张**：声音大得能把天震破。
5. **反问**：难道我不对吗？
6. **设问**：什么是幸福？幸福是…

### 模块 6 · 阅读理解
**文本类型（本级重点）**：
- **记叙文 · 写人**：《我的妈妈》《卖火柴的小女孩》《好朋友》
- **记叙文 · 写事**：《一件难忘的事》《第一次…》《去海边》
- **记叙文 · 写景**：《秋天的公园》《雨中的校园》《美丽的乡村》
- **记叙文 · 写物**：《我的小乌龟》《台灯》《桂花》
- **寓言故事**：伊索寓言选（《狼与小羊》）、中国寓言（《纪昌学射》）
- **童话故事**：安徒生（《丑小鸭》《皇帝的新装》改写版）、格林（《白雪公主》）
- **民间故事**：《孟姜女哭长城》《牛郎织女》《梁山伯与祝英台》（简版）

**阅读策略 5 式**：
1. 预测：看题目猜内容
2. 圈划：关键词、时间、地点、人物
3. 提问：5W1H（Who What When Where Why How）
4. 概括：段意 → 主旨
5. 评价：对主人公的看法

**HSK 3 阅读真题训练**：
- 句子理解与短文阅读
- 选词填空
- 段落排序练习

### 模块 7 · 写作与口语
**记叙文三要素训练**：
- 时间：什么时候
- 地点：在哪里
- 人物：谁
- 事件：发生了什么（起因 → 经过 → 结果）

**常见文体**：
- 写人 · 外貌 + 性格 + 一件事（200 字）
- 写事 · 六要素完整（250 字）
- 写景 · 按时间或空间顺序（200 字）
- 写物 · 总分总结构（200 字）
- 想象 · 续写故事（150 字）
- 应用 · 通知、留言条、感谢信

**段落结构训练**：
- 总分结构：先总说，后分说（举例说明）
- 并列结构：几个方面平行描述
- 因果结构：先说结果，后说原因；或反之

**口语进阶**：
- 讲故事比赛：3-5 分钟完整故事
- 小组辩论启蒙：正反方各 1 分钟
- 新闻播报（模拟）：播报一则校园新闻
- 采访练习：扮演小记者采访同学

## 六、三级考核体系（Level 3）

### 第一级 · 课时小测验
- 每课时学完后 3-5 题即时测试
- 题型：多音字选音 / 近反义词 / 成语填空
- 无通过门槛，答错加入 SRS 复习队列

### 第二级 · 单元阶段测评
- 每单元学完后 10-15 题
- 题型：听短文答题 / 改病句 / 阅读段落大意 / 看图写作
- 通过标准：70 分

### 第三级 · 级别综合考核（100 分制）

| 维度 | 分值 | 题型 |
|---|---|---|
| 听力 | 15 | 听短文答 5 题 / 听指令做事 |
| 口语 | 15 | 讲故事 3 分钟 / 即兴演讲 1 分钟 |
| 识字 | 5 | 认读 + 多音字选音 + 形近字辨析 |
| HSK 考点 | 10 | HSK 3 阅读理解 / 完成句子 / 书写汉字 |
| 写字 | 10 | 默写 + 形声字推断 + 部首查字 |
| 词汇 | 10 | 成语填空 / 近反义词 / 词语搭配 |
| 语法 | 10 | 句型转换 / 改病句 / 连词填空 |
| 阅读 | 15 | 400 字短文 + 5 题（含主旨与推断） |
| 写作 | 10 | 200-300 字记叙文 |

**通过线**：总分 85 分，单项不低于 60 分
**重考间隔**：24 小时
**通过奖励**：Level 3 数字证书（对标 HSK 3 / CEFR B1 / 小学三年级）

## 七、AI 软件产品化建议

### 1. AI 写作批改官
- 用户交作文 → AI 分析：错别字、病句、段落结构、好词好句
- 给出"提升版"对照范文
- 生成个性化薄弱点清单

### 2. 段落搭建游戏
- 给出散乱句子，用户按逻辑（时间/空间/总分）排序
- 或选"最佳首句""最佳末句"

### 3. 朝代时间轴
- 可交互的 3D 朝代时间轴
- 点击每个朝代看代表人物、文学、事件
- 连连看：朝代与代表诗人匹配（唐-李白、宋-苏轼）

### 4. 虚拟采访练习
- AI 扮演各种人物（古代诗人、科学家、运动员）
- 用户以记者身份采访，问答训练

### 5. 词语接龙 · 成语大会
- 单人闯关：以末字为下一词首字
- 多人 PK：在线实时 PK，限时 10 秒
- 成语分类挑战（历史成语、寓言成语等）

### 6. 查字典挑战赛
- 电子字典模拟：用户在限时内用部首法查 10 个字
- 纸质字典扫码验证（给家长 OCR 判定学习进度）

## HSK 3 考点清单

| 板块 | 考点内容 |

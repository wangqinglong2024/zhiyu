# Level 7 · 文言文启蒙

> 对标：中国初中七年级 · HSK 6 · CEFR B2+

---

## 一、等级定位

七年级正式进入初中，核心任务是文言文系统化启蒙与现代文学经典精读。本级完成：
1. **识字达 2,200**：累计 **2,200 字**，会写 1,800 字
2. **文言文体系化**：学习 12 篇经典文言（先秦到唐）
3. **名著精读**：《朝花夕拾》《西游记》再读
4. **现代文学**：鲁迅、老舍、朱自清、林海音等作家入门

学习者完成本级后：
- 达到 HSK 6（5456 词升级方向）
- 能独立翻译短篇浅显文言
- 能写 800 字记叙文、议论文
- 掌握鲁迅早期作品基本解读

## 四、九大知识模块详解

### 模块 1 · 汉字与书写
**新增 400 字**：
- 文言高频：曾、竟、遂、既、乃、辄、须臾、少顷、俄而
- 人体内脏：肝、胆、脾、胃、肾、肺、肠、胸、腹、腰
- 天文地理：陆、洋、泽、岭、峡、岸、滩、礁、屿、潟
- 制度职官：侯、卿、宰、辅、宦、戚、爵、仕、贬、谪
- 诗文抒情：慨、喟、叹、噫、呜、嗟、咦、哉、矣、乎

**字源入门**：
- 象形字：日、月、山、水、人、口
- 指事字：上、下、本、末、刃
- 会意字：明、休、林、从、好
- 形声字（80% 汉字）：例 海 = 氵 + 每
- 假借字：来（本义"麦"借作"来往")

### 模块 2 · 拼音与语音
- 朗诵提升：古诗文的音韵美（平仄、押韵、节奏）
- 语流音变的系统整理
- 四级普通话水平模拟测试（目标：二甲 87 分）

### 模块 3 · 词汇（累计 7000 词 / 本级新增 1500）
**现代文学相关**（200 词）：彷徨、呐喊、徬徨、寂寥、惆怅、苦闷、忧愤、觉醒、启蒙、革新、斗争、压迫
**文言实词 120 个**（每个配多个义项）：
- 名词类：兵（兵器 / 士兵 / 军队 / 战争）、师（军队 / 老师 / 效法）、徒（徒弟 / 徒然 / 空）
- 动词类：使（派遣 / 出使 / 使唤 / 如果）、走（逃跑 / 跑）、去（离开 / 距离 / 除去）、过（经过 / 错误 / 拜访）
- 形容词类：鲜（少 / 新鲜 / 鲜艳）、善（好 / 擅长 / 赞同）、穷（走投无路 / 贫穷 / 穷尽）

**文言虚词 18 个系统**：
- 之：代词 / 结构助词"的" / 取消句子独立性 / 动词"去"
- 其：代词 / 表推测 / 表反问 / 表祈使
- 以：因为 / 用 / 凭借 / 而 / 认为
- 而：表并列 / 转折 / 承接 / 修饰
- 于：在 / 对 / 被 / 比 / 向
- 乎：疑问 / 反问 / 感叹 / 句中语气
- 者：…的人 / 的事 / 结构助词
- 也：判断语气 / 陈述语气 / 句中停顿
- 则、焉、乃、若、所、因、为、何、且、与

**成语累计 450 个 · 本级新增 120 个**：
- 出自《论语》：学而不倦、温故知新、见贤思齐、三人行必有我师、己所不欲勿施于人、任重道远、有教无类
- 出自《庄子》：庖丁解牛、望洋兴叹、贻笑大方、鹏程万里、邯郸学步、庄周梦蝶、朝三暮四
- 出自《史记》：四面楚歌、破釜沉舟、鸿鹄之志、指鹿为马、毛遂自荐、完璧归赵、负荆请罪、背水一战、一鸣惊人、孺子可教
- 出自唐诗宋词：柳暗花明、春风得意、青梅竹马、老骥伏枥、风吹草动
- 描写学习：学富五车、才高八斗、博学多才、博览群书、学贯中西、融会贯通、登峰造极

### 模块 4 · 语法与句法
**初中系统化语法（现代汉语）**：
**词性 12 类**：
1. 名词
2. 动词（及物/不及物）
3. 形容词
4. 数词
5. 量词
6. 代词
7. 副词
8. 介词
9. 连词
10. 助词（结构、动态、语气）
11. 叹词
12. 拟声词

**短语 5 类**：
- 并列短语：山水 / 伟大光荣
- 偏正短语：好朋友 / 认真地学
- 主谓短语：天亮 / 月明
- 动宾短语：学语文 / 踢足球
- 补充短语：跑得快 / 写完

**句子六大成分**：主、谓、宾、定、状、补

**文言特殊句式**：
1. **判断句**："……者，……也"（廉颇者，赵之良将也）
2. **省略句**：省主语、省宾语、省介词
3. **倒装句**：宾语前置（何陋之有）、状语后置（战于长勺）、定语后置
4. **被动句**：为所式、见……于式、于字句
5. **疑问句**：疑问代词前置

### 模块 7 · 写作与口语

**文体训练**：
- **记叙文 800 字**：描写精细化；倒叙/插叙技巧；以小见大立意
- **议论文 600 字**：三要素清晰；论证方法丰富；过渡段流畅
- **说明文 600 字**：说明顺序明确；语言准确；信息密度
- **散文 600 字**：形散神聚；意境营造；情景交融
- **书信 & 演讲稿**：正式场合写作

**写作技法专题**：
- **立意**：高（有思想）、深（有内涵）、新（有角度）、准（切题）
- **选材**：典型、新颖、真实
- **结构**：总分总、起承转合、对比、并列
- **语言**：准确、生动、形象、含蓄
- **细节**：动作、神态、心理、环境

**口语交际**：
- 话题讨论：对某社会现象发表看法
- 即兴评述：看新闻短片 30 秒后 2 分钟评述
- 讲故事大赛：自创或改编故事

## 七、AI 软件产品化建议

### 1. 文言文智能翻译工坊
- 四栏对照：原文 | 字字译 | 流畅译 | 古人意图
- 点击任一字弹出"本义 / 引申义 / 语境义"
- 自动识别活用、通假、倒装
- AI 生成同难度新练习

### 2. 鲁迅研究社
- 时间线：鲁迅人生与作品对应
- 人物关系网：鲁迅与同时代文人
- 作品精读带：按难度排序，逐步解锁
- AI 讨论："如果鲁迅活在今天会说什么？"

### 3. 诸子百家辩论场
- 选一话题（如"人性善恶"）
- 孔、孟、荀、韩非子 AI 化身分别立论
- 用户选择立场与 AI 辩论
- 最终 AI 评判逻辑与引经据典能力

### 4. 古诗词吟诵大师
- 提供多种吟诵风格（唐调、湖湘调、常州调）
- AI 教学：字、腔、气、韵四要素
- 录音评分 → 与名家范本对比

### 5. 800 字作文 AI 升格
- 第一稿 → AI 分段打分
- 第二稿 → 用户按提示修改
- 第三稿 → AI 生成"特级教师范文"对照
- 立意、结构、语言、细节四维提升

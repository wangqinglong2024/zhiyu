## 5. 拼读规则

| 规则 | 说明 | 例 |
|---|---|---|
| 两拼 | 声母 + 韵母 | b-a → bā |
| 三拼 | 声母 + 介音 + 韵母 | x-i-ā → xiā |
| 整体认读 | 不拆分 | yī（不拆 y-i）|
| j q x + ü | ü 去两点 | jū（写 ju）|
| y + i | 省略 | 一 yī |
| w + u | 省略 | 五 wǔ |

## 6. 变调

| 规则 | 说明 | 例 |
|---|---|---|
| 三声连读 | 前一个变二声 | 你好 → ní hǎo |
| 半三声 | 三声后接非三声，前变半三声 | 你说 → ní（半三）shuō |
| "一"变调 | 一 + 一/二/三声 → 四声；一 + 四声 → 二声 | 一天 yī → yì；一次 yí cì |
| "不"变调 | 不 + 四声 → 二声 | 不要 bù → bú yào |
| 轻声 | 双音节后字弱化 | 妈妈 māma |

## 7. 儿化

- 北方话特征，南方学习者仅作认知
- 拼音写为原字 + r：花儿 huār，玩儿 wánr

## 8. 隔音符号

a / o / e 开头的音节连接前面音节时加 ' 防混淆：
- 西安 = Xī'ān（不是 Xiān）
- 海鸥 = hǎi'ōu

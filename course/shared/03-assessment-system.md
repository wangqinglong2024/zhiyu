# shared · 三级考核体系

> 4 轨道共用考核标准。所有节 / 章 / 阶段考核必须遵循本规范。

---

## 1. 三级总览

```
┌─ 第一级：节小测（12 题 / 节）  ── 即时反馈，错题入 SRS
├─ 第二级：章单元测（30-50 题 / 章） ── 80% 通过，否则重学
└─ 第三级：阶段综合考核（80-150 题 / 阶段） ── 80-85% 通过 + 颁发证书
```

---

## 2. 第一级 · 节小测

### 出题规则
- 每节 12 个知识点 = 12 道题，1:1 对应
- 题型必须 **混合** 5 类（按知识点类型自动选）：
  - 选择（4 选 1）
  - 填空（输入 / 选项填）
  - 听力（音频 → 选 / 写）
  - 跟读（ASR 评分）
  - 翻译（中→母 或 母→中）

### 通过
- 9/12 ≥ 75%
- 答错题自动加入 SRS 复习队列
- 不限重考次数，立即可重考

### App 反馈
- 单题作答即出对错 + 1 行解释
- 错题展示正确答案 + 关键字源 / 文化点

---

## 3. 第二级 · 章单元测

### 出题规则
- 30-50 题，从本章 144 知识点中随机抽 + 题型平衡
- 题型分布：
  - 选择 30%
  - 填空 25%
  - 听力 20%
  - 跟读 15%
  - 翻译 10%
- 增设 5 道 **章主题应用题**（情景模拟）

### 通过
- 总分 ≥ 80%
- 单项（听 / 读 / 写 / 说）≥ 60%
- 不通过 → 推送本章关键节点重学清单 → 24h 间隔后再考

---

## 4. 第三级 · 阶段综合考核

### 出题规则
- 80-150 题（按阶段递增，阶段 1 = 80，阶段 12 = 150）
- 覆盖本阶段全 12 章 × 12 节 = 1728 知识点中按 SRS 优先抽
- 必含：
  - 听力（含连续对话 ≥ 3 段）
  - 阅读（含中长文 ≥ 1 篇）
  - 写作（看图写话 / 仿写句 / 主题段落）
  - 口语（自由话题 + ASR 评分）

### 通过
- 阶段 1-10：80%；单项 60%
- 阶段 11-12：85%；单项 60%
- 通过 → 数字证书（含 HSK 等级 / CEFR 等级 / 阶段编号 + 区块链可验证）
- 不通过 → 24h 后无限重考

---

## 5. HSK 模拟考集成（HSK 轨道独有）

HSK 轨在每个阶段综合考核之外，额外提供：
- 完整 HSK 真题模拟（按 HSK 1-9 各级官方题型）
- 计时模拟（按真实考试时长）
- 模拟成绩 → HSK 真考报名建议

---

## 6. 跨轨认证机制

- 4 轨道使用同一阶段编号
- 通过任意一轨阶段 N 综合考核 → 自动解锁其他轨阶段 N（其他轨可直接进 N，但需补学该轨独有词汇 / 场景）
- HSK 轨阶段 9 综合考通过 ≈ HSK 6 通过水平（可参加官方 HSK 6）

---

## 7. 反作弊与公平

- 同 IP / 同设备短时多账号刷题 → 风控锁定
- 跟读 ASR 拒识高频帐号 → 人工抽查
- 证书生成时记录设备指纹 + 答题轨迹哈希 → 可追溯

# 应用端：类目与文章浏览

> 所有接口均**公开可访问**（与 F1 RLS 一致：`china_categories` 全量公开；`china_articles` / `china_sentences` 仅 `published`）。
> 不要求登录；登录态仅影响是否带回阅读进度（见 [03-应用端-学习进度.md](./03-应用端-学习进度.md)）。

---

## C1 · 列出 12 类目

**对应需求中的操作**：F2-用户 §一 · 应用端「Tab 进入后是 12 个类目卡片」

**方法**：GET
**路径**：`/api/v1/china/categories`
**权限**：公开

**业务校验**：无。

**请求参数**：无（按 `sort_order asc` 固定顺序）。

**成功响应 200**：

```json
{
  "code": 0,
  "message": "ok",
  "data": {
    "items": [
      {
        "code": "01",
        "name_i18n":        { "zh":"中国历史","en":"Chinese History","vi":"Lịch sử Trung Quốc","th":"ประวัติศาสตร์จีน","id":"Sejarah Tiongkok" },
        "description_i18n": { "zh":"朝代更替、历史事件、传奇人物","en":"Dynasties, historical events, legendary figures","vi":"...","th":"...","id":"..." },
        "sort_order": 1
      }
      /* ... 共 12 项 */
    ]
  }
}
```

**错误场景**：仅基础设施错误（5xx），无业务错误。

> 该接口可强缓存（`Cache-Control: public, max-age=3600`），变更频率极低。

## C2 · 列出某类目下已发布文章

**对应需求中的操作**：F2-用户 §一 · 应用端「进入后是对应类目下的文章列表」

**方法**：GET
**路径**：`/api/v1/china/categories/:code/articles`
**权限**：**分级公开**（PM 答 F2-Q11）
- 类目 `01..03`：完全公开。
- 类目 `04..12`：必须登录，否则返回 401 + `redirect_to=/login`。

**业务校验**：
- `:code` 必须存在于 `china_categories`，否则 404。
- 若 `code >= '04'` 且未登录，返回 401。

**请求参数**：

| 参数 | 位置 | 类型 | 必填 | 说明 | 校验 |
|------|------|------|------|------|------|
| `code` | path | string | ✅ | 类目编码 | 正则 `^(0[1-9]\|1[0-2])$` |
| `page` | query | int ≥ 1 | ❌ | 默认 1 | — |
| `page_size` | query | int ≥ 1 | ❌ | 默认 20，上限 100 | — |
| `sort` | query | string | ❌ | 默认 `-published_at`；白名单：`published_at`、`title_pinyin` | — |

**成功响应 200**（仅返回 `status='published'` 文章）：

```json
{
  "code": 0,
  "message": "ok",
  "data": {
    "items": [
      {
        "code": "A7K2P9X3M4QR",
        "title_pinyin": "qín shǐ huáng tǒng yī liù guó",
        "title_i18n":   { "zh":"秦始皇统一六国","en":"...","vi":"...","th":"...","id":"..." },
        "category_code": "01",
        "sentence_count": 3,
        "published_at": "2026-04-20T10:00:00+08:00"
      }
    ],
    "pagination": { "page":1, "page_size":20, "total":42, "total_pages":3, "has_next":true }
  }
}
```

**错误场景**：

| HTTP | code | 触发条件 | 错误信息 |
|------|------|---------|---------|
| 400 | 40001 | `code` 格式不合法 | 类目编码格式不正确 |
| 401 | 40101 | 类目 `04..12` 且未登录 | 请先登录后再查看该类目 |
| 404 | 40400 | 类目不存在 | 类目不存在 |

## C3 · 文章详情（含全部已发布句子）

**对应需求中的操作**：F2-用户 §一 · 应用端「逐句查看文章，每个句子一个卡片」

**方法**：GET
**路径**：`/api/v1/china/articles/:code`
**权限**：**分级公开**（PM 答 F2-Q11）
- 文章所属类目 `01..03`：完全公开。
- 类目 `04..12`：必须登录，否则 401 + `redirect_to=/login`。

**业务校验**：
- 文章必须 `status='published' AND deleted_at IS NULL`，否则 404（不暴露下架/草稿存在）。
- 若所属类目 `code >= '04'` 且未登录，返回 401。

**请求参数**：

| 参数 | 位置 | 类型 | 必填 | 说明 |
|------|------|------|------|------|
| `code` | path | string | ✅ | 文章 12 位编码，正则 `^[A-Z0-9]{12}$` |

**成功响应 200**：

```json
{
  "code": 0,
  "message": "ok",
  "data": {
    "code": "A7K2P9X3M4QR",
    "title_pinyin": "qín shǐ huáng tǒng yī liù guó",
    "title_i18n":   { "zh":"秦始皇统一六国","en":"...","vi":"...","th":"...","id":"..." },
    "category": {
      "code": "01",
      "name_i18n":        { "zh":"中国历史","en":"...","vi":"...","th":"...","id":"..." },
      "description_i18n": { "zh":"...","en":"...","vi":"...","th":"...","id":"..." }
    },
    "sentences": [
      {
        "id": "5b7c...",
        "seq_no": 1,
        "seq_label": "0001",
        "pinyin": "gōng yuán qián 221 nián , qín wáng yíng zhèng wán chéng tǒng yī dà yè .",
        "content": {
          "zh": "公元前221年，秦王嬴政完成统一大业。",
          "en": "In 221 BCE, ...",
          "vi": "Năm 221 TCN, ...",
          "th": "ปี 221 ก่อน ค.ศ. ...",
          "id": "Pada 221 SM, ..."
        },
        "audio": {
          "status": "ready",
          "url":    "https://storage.zhiyu.app/china-tts/A7K2P9X3M4QR/0001.mp3",
          "duration_ms": 4820
        }
      }
      /* ... */
    ],
    "published_at": "2026-04-20T10:00:00+08:00",
    "prev": { "code": "B6J1N8W2L3PQ", "title_i18n": { "zh":"商鞅变法","en":"...","vi":"...","th":"...","id":"..." } },
    "next": { "code": "C5H0M7V1K2NP", "title_i18n": { "zh":"汉武帝盛世","en":"...","vi":"...","th":"...","id":"..." } }
  }
}
```

> 句子顺序按 `seq_no asc`；返回内嵌 `audio.status`，前端据此决定播放按钮初始态（`pending` / `ready` / `failed`）。
> **不下发** 4 种非中文语言的 `audio_url`（PM 答 F1-Q4：仅中文 TTS）。
> `prev` / `next`：同类目下按 `published_at desc` 排序的相邻文章，用于应用端阅读页「上一篇/下一篇」按钮（PM 答 F2-Q8）；首/尾文章对应字段为 `null`。

**错误场景**：

| HTTP | code | 触发条件 | 错误信息 |
|------|------|---------|---------|
| 400 | 40001 | `code` 格式不合法 | 文章编码格式不正确 |
| 404 | 40400 | 文章不存在/未发布/已删除 | 文章不存在 |

## C5 · 全文朗读清单（先放在浏览组，便于复用文章详情上下文）

> C4 单句 TTS 见 [02-应用端-TTS与朗读.md](./02-应用端-TTS与朗读.md)。

**对应需求中的操作**：F2-用户 §一 · 应用端「全文朗读，逐个句子读出来」

**方法**：GET
**路径**：`/api/v1/china/articles/:code/audio-playlist`
**权限**：公开

**业务校验**：与 C3 相同（文章必须已发布）。

**请求参数**：

| 参数 | 位置 | 类型 | 必填 | 说明 |
|------|------|------|------|------|
| `code` | path | string | ✅ | 文章编码 |

**成功响应 200**：

```json
{
  "code": 0,
  "message": "ok",
  "data": {
    "article_code": "A7K2P9X3M4QR",
    "items": [
      { "sentence_id":"5b7c...", "seq_no":1, "audio_status":"ready",   "audio_url":"https://.../0001.mp3" },
      { "sentence_id":"6c8d...", "seq_no":2, "audio_status":"pending", "audio_url":null },
      { "sentence_id":"7d9e...", "seq_no":3, "audio_status":"ready",   "audio_url":"https://.../0003.mp3" }
    ]
  }
}
```

**前端编排约定**：
- 顺序播放，遇 `audio_status != 'ready'` 时调用 C4 触发生成；生成成功后继续播放。
- 任一句失败则停止全文朗读并提示错误（PM 答 Q10：不限次重试，由用户决定再次点击）。

**错误场景**：

| HTTP | code | 触发条件 | 错误信息 |
|------|------|---------|---------|
| 400 | 40001 | `code` 格式不合法 | 文章编码格式不正确 |
| 404 | 40400 | 文章不存在/未发布 | 文章不存在 |

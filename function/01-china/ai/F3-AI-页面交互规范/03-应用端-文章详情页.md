## 元素规格

| 元素 | 组件类型 | 规格 |
|------|---------|------|
| 标题区 | `<header>` | 拼音字号 16，中文标题 24，本地语言 18 |
| 全文朗读按钮 | `Button.Primary` | 状态：闲置 / 播放中（变图标 ⏸）/ 暂停 |

> **2026-04 变更**：取消阅读进度功能，移除「重新开始」「阅读进度」相关元素；C6/C7 接口下线。
| 句子卡片 | `Card` | 顺序：seq_label → 拼音 → 中文 → 本地语言；右上角播放按钮 |
| 句子播放按钮 | `IconButton(Speaker)` | 4 态：`pending`（默认可点）/ `processing`（loading 旋转）/ `ready`（播放/暂停切换）/ `failed`（红点 + 可重点） |
| 上/下篇导航 | `LinkButton` | 来自 C3 响应的 `prev` / `next`；为 null 时灰显 |

## 交互行为

| # | 用户操作 | 技术实现 |
|---|---------|---------|
| C3-01 | 进入页面 | 调 C3 → 渲染卡片；不再回填阅读进度（C7 已下线） |
| C3-02 | 点击单句播放按钮（status=ready） | 直接 `<audio>` 播放该 url |
| C3-03 | 点击单句播放按钮（status=pending/failed） | 调 C4：成功 → 替换 status=ready 并立即播放；失败 → toast 错误 + 该按钮变 `failed` |
| C3-04 | C4 返回 202 | 立即按钮显 loading；500ms 后调 AUX 状态轮询，每 2s 一次，最多 30s；命中 ready 即播 |
| C3-05 | C4 返回 502/504 | toast「语音生成失败/超时，请稍后重试」，按钮可再点（不限次） |
| C3-06 | 点击「全文朗读」 | 顺序播放 seq=1..N：每句先看 status，pending → 调 C4，ready → 播放结束后等 300ms 间隔进入下一句；遇失败停止并 toast |
| C3-07 | 全文朗读中点击「⏸」 | 暂停当前 audio，再点恢复 |
| ~~C3-08~~ | ~~重新开始~~ | **已下线（2026-04）** |
| ~~C3-09~~ | ~~进度上报~~ | **已下线（2026-04）**，不再调 C6 |
| C3-10 | 切换 UI 语言 | 重渲染本地语言行；中文模式下隐藏本地语言行 |
| C3-11 | 点击「上一篇/下一篇」 | `router.push('/china/articles/' + prev.code)`；若按钮灰显则忽略 |
| C3-12 | 接口 401 | 跳 `/login?redirect=` 当前 URL |
| C3-13 | 接口 404 | 居中「文章不存在或已下架」+ 返回 |
| C3-14 | 进度回填检测到被服务端重置 | C7 返回 `last_seq_no=0` 且本地缓存有 >0 旧值 → toast「文章已更新，阅读进度已重置」并清本地缓存（PM 答 F3-Q4/Q12：仅"重排/中间或开头插入/删除"清进度，"末尾追加/编辑单句"不清；判定全部由后端在 A11/A13/A14 副作用中完成，前端只读取） |
| C3-15 | 读取个人中心拼音偏好 | 启动时读取 user-account 域 `user_settings.china_pinyin_visible`（默认 `true`）；为 `false` 时所有句子卡片隐藏拼音行（PM 答 F3-Q9：跨 UI 语言通用，配置入口在个人中心）；未登录用户取本地 localStorage `china_pinyin_visible` 默认 `true` |

## 条件显示规则

| 条件 | 显示 | 隐藏 |
|------|------|------|
| UI 语言 = 中文 | 拼音 + 中文 | 本地语言行 |
| 未登录 | 「全文朗读」按钮仍可用 | — |
| `prev` = null | 上一篇按钮灰显 | — |
| `next` = null | 下一篇按钮灰显 | — |
| `audio.status` = `failed` | 红色感叹号 + tooltip 错误信息 | — |
| `china_pinyin_visible = false` | — | 所有句子卡片的拼音行（不论 UI 语言）|

## 空态 / 异常

| 场景 | 处理 |
|------|------|
| 文章 0 句（理论上不应发布） | 显示「该文章暂无内容」 |
| 全文朗读中网络断开 | 暂停播放 + toast「网络已断开」 |
| 同一句被并发点击 2 次 | 第二次点击与第一次去重（前端按 sentence_id 维护 in-flight Set） |

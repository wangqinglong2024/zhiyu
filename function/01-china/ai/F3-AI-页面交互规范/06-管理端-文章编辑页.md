## 区域 1：基本信息（D-2 内联区域）

### 表单字段

| 字段标签 | 组件 | 数据字段（F1） | 必填 | 校验（F1） | 说明 |
|---------|------|--------------|------|-----------|------|

| 标题 en | 同上 | `title_i18n.en` | ✅ | 1–40 | |
| 标题 vi | 同上 | `title_i18n.vi` | ✅ | 1–40 | |
| 标题 th | 同上 | `title_i18n.th` | ✅ | 1–40 | |
| 标题 id | 同上 | `title_i18n.id` | ✅ | 1–40 | |

### 接口

- 挂载：[A3 GET /admin/v1/china/articles/:id](../F2-AI-接口规范/05-管理端-文章CRUD.md#a3--文章详情不含句子)
- 保存：[A5 PATCH /admin/v1/china/articles/:id](../F2-AI-接口规范/05-管理端-文章CRUD.md#a5--编辑文章基本信息)

### 交互

| # | 用户操作 | 技术实现 |
|---|---------|---------|
| A3-01 | 进入页面 | 并行调 A3（基本信息）+ A9 page=1（句子列表） |

## 区域 2：句子列表（含分页 + 搜索）

### 表格/卡片列定义

每个句子用一个 `Card` 渲染（非表格行），字段：

| 元素 | 数据字段 | 渲染 |
|------|--------|------|
| seq_label | `seq_label` | 等宽 4 位编号 |
| 拼音 | `pinyin` | 单行省略 |
| 中文 | `content_zh` | 主行突出 |
| 音频状态 | `audio_status` | `ready`(✅+时长) / `pending`(⚪) / `processing`(⏳) / `failed`(❌+原因) |
| 操作 | — | [▶预听]（仅 ready 可点）/ [✎编辑] / [🗑 删除] |

### 顶部摘要

来自 A9 响应 `summary.total_sentences`：「共 {N} 句」（PM 答 F3-Q3：是句子条数/卡片数，不是字符数）。

### 编辑模式开关（PM 答 F3-Q8）

- 默认进入「只读模式」（🔒）：句子卡片不展示「✎ 编辑」按钮；用户**双击**任一句子卡片的中文区域 → 打开 D-3 单句编辑 Drawer。
- 切到「编辑模式」（🔓）：所有句子卡片直接显示「✎ 编辑」「🗑 删除」按钮；双击行为同样有效。
- 切换 UI：顶部一个 Toggle 按钮（图标随状态变化）。

### 接口

- 列表：[A9 GET /admin/v1/china/articles/:id/sentences](../F2-AI-接口规范/06-管理端-句子CRUD.md#a9--句子列表分页)
- 新建：[A11 POST .../sentences](../F2-AI-接口规范/06-管理端-句子CRUD.md#a11--新建句子开头结尾中间)（开头/结尾/中间）
- 编辑：[A12 PATCH /admin/v1/china/sentences/:id](../F2-AI-接口规范/06-管理端-句子CRUD.md#a12--编辑句子)
- 删除：[A13 DELETE /admin/v1/china/sentences/:id](../F2-AI-接口规范/06-管理端-句子CRUD.md#a13--删除句子软删--自动重排)
- 重排：[A14 POST .../sentences:reorder](../F2-AI-接口规范/06-管理端-句子CRUD.md#a14--批量重排句子)
- 预听：复用 [C4](../F2-AI-接口规范/02-应用端-TTS与朗读.md#c4--触发获取句子-tts)

### 交互

| # | 用户操作 | 技术实现 |
|---|---------|---------|
| A3-10 | 输入句子搜索 | 300ms 防抖；调 A9 带 `q`（F2 增补，详见 Q1） |
| A3-11 | 点击「+ 添加句子」 | 弹下拉菜单：「开头 / 结尾 / 在某句后…」；选「中间」时再让用户先点击列表中目标句卡片，然后弹 D-4 |
| A3-12 | D-4 提交 | 调 A11；成功后用响应 `affected_sentence_ids` 局部刷新或简单整页重拉 A9 |
| A3-13 | 点击 [▶预听] | 直接 `<audio>` 播放 `audio.url` |
| A3-14 | 点击 [✎编辑]（编辑模式下）/ 双击句子中文区域（任一模式） | 打开 D-3 句子编辑 Drawer，预填该句 `pinyin` + 5 语言（PM 答 F3-Q8） |
| A3-15 | D-3 提交 | 调 A12；若 `content_zh` 改变 → 后端清缓存，前端列表中将该句 `audio_status` 改为 `pending`。**编辑单句不清空用户阅读进度**（PM 答 F3-Q12） |
| A3-16 | 点击 [🗑 删除句子] | 弹 D-5「确定删除该句？删除后剩余句子会自动重排、所有句子音频缓存失效，且所有用户该文章的阅读进度会被重置。」→ 调 A13 → 成功 toast + 整页重拉 A9 |
| A3-17 | 点击「重排」 | 弹 D-7 句子重排 Drawer（拖拽列表）；提交时调 A14 全量 `ordered_ids`；成功 toast「已重排，所有句子的音频已失效，且所有用户的阅读进度已重置」（PM 答 F3-Q4/Q12） |
| A3-18 | 翻页 | 仅更新 `page` 调 A9 |
| A3-19 | 切换「编辑模式」开关 | 仅前端切换；显示/隐藏所有卡片的 [✎编辑][🗑] 按钮，不影响双击行为（PM 答 F3-Q8） |

## 区域 3：底部状态切换 + 删除

| # | 用户操作 | 技术实现 |
|---|---------|---------|
| A3-20 | `draft` 时点击「发布」 | 弹 D-5 → 调 A6；422 错误时具体提示「请补全所有句子的 5 语言翻译和拼音」或「至少添加 1 句」 |
| A3-21 | `published` 时点击「下架」 | 弹 D-5「确定下架？所有用户该文章的阅读进度将被清空（不可恢复）。」→ 调 A7 |
| A3-22 | 点击「删除文章」 | 弹 D-5「确定删除整篇文章？30 天内不可恢复。」→ 调 A8 → 跳回 P-A-2 |

## 条件显示规则

| 条件 | 显示 | 隐藏 |
|------|------|------|
| `status='draft'` | 「发布」按钮 | 「下架」按钮 |
| `status='published'` | 「下架」按钮 | 「发布」按钮 |
| `sentence_count = 0` | 「发布」按钮 disabled + tooltip「请先添加句子」 | — |
| 任一句子 5 语言/拼音不全 | 「发布」按钮 disabled + tooltip 列出缺失项 | — |
| 当前页所有 Drawer/区域无脏数据 | 「保存基本信息」按钮 disabled | — |

## 未保存离开提示

- 基本信息区任一字段被改但未点保存 → 路由切换前弹 D-6。
- D-3 / D-4 Drawer 打开期间被改但未提交 → 关闭 Drawer 前弹 D-6。

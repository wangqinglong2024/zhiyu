## 1. 表：`course_ai_prompts`（Prompt 模板库）

**对应需求中的对象**：[temp/02 §2.7 AI 出稿 Prompt 模板](../../../temp/02-知识点与题型内容模板.md) + [temp/04 §4.6.1](../../../temp/04-管理端模块设计.md)
**性质**：管理端 CRUD。最少 21 条（7 类 KP + 12 类题型 + TTS + 配图）。

### 一、字段定义

| 字段名 | 类型 | 必填 | 默认值 | 索引 | 对应需求中的信息 | 校验规则 |
|--------|------|------|--------|------|----------------|---------|
| `id` | `uuid` | ✅ | `gen_random_uuid()` | PK | — | — |
| `prompt_code` | `text` | ✅ | — | `uq_course_prompts_code` (UNIQUE) | prompt 编码 | 正则 `^prompt_[a-z0-9_]+_v\d+$` |
| `scope` | `text` | ✅ | — | `idx_course_prompts_scope` | 用途范围 | 枚举 `kp_<type>\|q_<type>\|tts\|image\|translate\|review_check` |
| `template` | `text` | ✅ | — | — | prompt 正文（含占位符） | 长度 1–8000 |
| `variables` | `jsonb` | ✅ | `'{}'` | — | 占位符说明 | 见 §variables schema |
| `version` | `int` | ✅ | `1` | — | 版本 | ≥ 1 |
| `is_active` | `bool` | ✅ | `false` | `idx_course_prompts_active` | 当前是否启用（每 scope 仅 1 条 active） | — |
| `created_by` | `uuid` | ❌ | `null` | — | — | FK → `auth.users.id`，`ON DELETE SET NULL` |
| `created_at` | `timestamptz` | ✅ | `now()` | — | — | — |
| `updated_at` | `timestamptz` | ✅ | `now()` | — | — | — |

### `variables` JSON Schema

```jsonc
{
  "word_zh":          { "type": "string", "required": true,  "desc": "目标词" },
  "track":            { "type": "enum",   "required": true,  "values": ["ec","fc","hk","dl","share"] },
  "stage":            { "type": "int",    "required": true,  "range": [0,6] },
  "target_languages": { "type": "array",  "required": true,  "default": ["vi","th","id","en"] },
  "known_vocab_list": { "type": "array",  "required": true,  "desc": "本主题 stage ≤ 当前 stage 已收录词表（白名单）" }
}
```

### 二、AI 补充字段

| 字段名 | 类型 | 补充理由 |
|--------|------|---------|
| `is_active` | `bool` | [temp/04 §4.6.1](../../../temp/04-管理端模块设计.md) "灰度切换：v3 启用，v2 保留只读，v1 归档"；同 scope 仅 1 条 active |
| `variables` | `jsonb` | 编辑器渲染表单 + 运行时校验 prompt 输入参数 |
| `created_by` | `uuid` | prompt 编辑审计 |

### 三、DDL 草案

```sql
create table zhiyu.course_ai_prompts (
  id          uuid primary key default gen_random_uuid(),
  prompt_code text not null,
  scope       text not null,
  template    text not null,
  variables   jsonb not null default '{}'::jsonb,
  version     int  not null default 1,
  is_active   bool not null default false,
  created_by  uuid,
  created_at  timestamptz not null default now(),
  updated_at  timestamptz not null default now(),

  constraint fk_course_prompts_creator
    foreign key (created_by) references auth.users(id) on delete set null,
  constraint uq_course_prompts_code unique (prompt_code),
  constraint ck_course_prompts_code_format
    check (prompt_code ~ '^prompt_[a-z0-9_]+_v\d+$'),
  constraint ck_course_prompts_template_len
    check (char_length(template) between 1 and 8000),
  constraint ck_course_prompts_version check (version >= 1)
);

-- 同 scope 仅 1 条 active
create unique index uq_course_prompts_one_active_per_scope
  on zhiyu.course_ai_prompts (scope) where is_active = true;
create index idx_course_prompts_scope  on zhiyu.course_ai_prompts (scope);
create index idx_course_prompts_active on zhiyu.course_ai_prompts (is_active);

create trigger tg_course_prompts_before_update_set_updated_at
  before update on zhiyu.course_ai_prompts
  for each row execute function zhiyu.set_updated_at();
```

### 四、RLS 策略

| 操作 | 角色 | 策略 |
|------|------|------|
| 全部 | `service_role` | 全量 |
| 其它 | — | 拒绝（仅管理端可见） |

## 2. 表：`course_ai_generation_jobs`（AI 生成任务）

**对应需求中的对象**：[temp/03 §3.4.2](../../../temp/03-数据库schema.md) + [temp/06 §6.3 流程 B](../../../temp/06-关键业务流程.md)
**性质**：每次发起一次 AI 生成（KP 批 / Question 批 / TTS / 配图）写一行；带状态机。

### 一、字段定义

| 字段名 | 类型 | 必填 | 默认值 | 索引 | 对应需求中的信息 | 校验规则 |
|--------|------|------|--------|------|----------------|---------|
| `id` | `uuid` | ✅ | `gen_random_uuid()` | PK | — | — |
| `job_type` | `text` | ✅ | — | `idx_course_aij_type` | 任务类型 | 枚举 `gen_kp_batch\|gen_q_batch\|gen_audio\|gen_image\|regen_single\|translate_batch` |
| `prompt_id` | `uuid` | ✅ | — | — | 使用的 prompt | FK → `course_ai_prompts.id`，`ON DELETE RESTRICT` |
| `model` | `text` | ✅ | — | `idx_course_aij_model` | 模型标识 | 1–100，例 `gpt-5-pro` / `claude-sonnet-4` / `azure-tts-xiaoxiao` |
| `params` | `jsonb` | ✅ | `'{}'` | — | 模型参数 | `{temperature, top_p, max_tokens, ...}` |
| `input` | `jsonb` | ✅ | — | — | 输入上下文 | 含 track/stage/词表等；用于审计与重试 |
| `output` | `jsonb` | ❌ | `null` | — | LLM 原始输出 | 用于审计 |
| `generated_count` | `int` | ✅ | `0` | — | 实际生成数量 | ≥ 0 |
| `failed_items` | `jsonb` | ❌ | `null` | — | 失败子集（便于重跑） | `[{ index, reason, raw }]` |
| `status` | `text` | ✅ | `'queued'` | `idx_course_aij_status` | Job 状态 | 枚举 `queued\|running\|done\|failed\|partial` |
| `cost_usd` | `numeric(10,4)` | ❌ | `null` | — | 实际 token 成本 | ≥ 0 |
| `budget_usd` | `numeric(10,4)` | ❌ | `null` | — | 预算上限 | ≥ 0 |
| `idempotency_key` | `text` | ❌ | `null` | `uq_course_aij_idemp` (UNIQUE) | 幂等键（[temp/06 §6.3](../../../temp/06-关键业务流程.md)） | 防止重复创建 |
| `created_by` | `uuid` | ❌ | `null` | — | — | FK → `auth.users.id`，`ON DELETE SET NULL` |
| `started_at` | `timestamptz` | ❌ | `null` | — | 开始执行时间 | — |
| `finished_at` | `timestamptz` | ❌ | `null` | — | 结束时间 | `status in (done,failed,partial)` 时必须非空 |
| `created_at` | `timestamptz` | ✅ | `now()` | `idx_course_aij_created` | — | — |

### 二、AI 补充字段

| 字段名 | 类型 | 补充理由 |
|--------|------|---------|
| `idempotency_key` | `text` | [temp/06 §6.3](../../../temp/06-关键业务流程.md) "Job 必须幂等：重跑不会创建重复 KP"；按 `(track, kp_type, title_zh hash)` 派生 |
| `failed_items` | `jsonb` | "失败可只重失败子集"——按 index 重投 |
| `budget_usd` | `numeric` | [temp/04 §4.6.2](../../../temp/04-管理端模块设计.md) "预算上限：超出自动停"——worker 累计 cost ≥ budget 时主动 fail |

### 三、DDL 草案

```sql
create table zhiyu.course_ai_generation_jobs (
  id              uuid primary key default gen_random_uuid(),
  job_type        text not null,
  prompt_id       uuid not null,
  model           text not null,
  params          jsonb not null default '{}'::jsonb,
  input           jsonb not null,
  output          jsonb,
  generated_count int  not null default 0,
  failed_items    jsonb,
  status          text not null default 'queued',
  cost_usd        numeric(10,4),
  budget_usd      numeric(10,4),
  idempotency_key text,
  created_by      uuid,
  started_at      timestamptz,
  finished_at     timestamptz,
  created_at      timestamptz not null default now(),

  constraint fk_course_aij_prompt
    foreign key (prompt_id) references zhiyu.course_ai_prompts(id) on delete restrict,
  constraint fk_course_aij_creator
    foreign key (created_by) references auth.users(id) on delete set null,

  constraint ck_course_aij_type
    check (job_type in ('gen_kp_batch','gen_q_batch','gen_audio','gen_image','regen_single','translate_batch')),
  constraint ck_course_aij_status
    check (status in ('queued','running','done','failed','partial')),
  constraint ck_course_aij_count check (generated_count >= 0),
  constraint ck_course_aij_cost check (cost_usd is null or cost_usd >= 0),
  constraint ck_course_aij_budget check (budget_usd is null or budget_usd >= 0),
  constraint ck_course_aij_finished_consistency
    check (
      (status in ('queued','running') and finished_at is null) or
      (status in ('done','failed','partial') and finished_at is not null)
    )
);

create unique index uq_course_aij_idemp
  on zhiyu.course_ai_generation_jobs (idempotency_key) where idempotency_key is not null;
create index idx_course_aij_status  on zhiyu.course_ai_generation_jobs (status);
create index idx_course_aij_type    on zhiyu.course_ai_generation_jobs (job_type);
create index idx_course_aij_model   on zhiyu.course_ai_generation_jobs (model);
create index idx_course_aij_created on zhiyu.course_ai_generation_jobs (created_at desc);

-- 补 KP / Q 表的反向 FK（建表时为避免循环依赖，此处用 alter 补加）
alter table zhiyu.course_knowledge_points
  add constraint fk_course_kp_ai_job
  foreign key (ai_job_id) references zhiyu.course_ai_generation_jobs(id) on delete set null;

alter table zhiyu.course_questions
  add constraint fk_course_q_ai_job
  foreign key (ai_job_id) references zhiyu.course_ai_generation_jobs(id) on delete set null;
```

### 四、RLS 策略

| 操作 | 角色 | 策略 |
|------|------|------|
| 全部 | `service_role` | 全量 |
| 其它 | — | 拒绝（管理端专用） |

## 3. 表：`course_content_review_log`（内容审核 / 学员举报留痕）

**对应需求中的对象**：[temp/03 §3.4.4](../../../temp/03-数据库schema.md) + [temp/06 §6.5 / §6.10](../../../temp/06-关键业务流程.md)
**性质**：高写入审计表。承载审核员 approve/reject/edit/regen + 学员 report。

### 一、字段定义

| 字段名 | 类型 | 必填 | 默认值 | 索引 | 对应需求中的信息 | 校验规则 |
|--------|------|------|--------|------|----------------|---------|
| `id` | `uuid` | ✅ | `gen_random_uuid()` | PK | — | — |
| `target_type` | `text` | ✅ | — | `idx_course_crl_target` | 目标对象类型 | 枚举 `kp\|question\|lesson\|exam` |
| `target_id` | `uuid` | ✅ | — | `idx_course_crl_target` | 目标对象 id | — |
| `action` | `text` | ✅ | — | `idx_course_crl_action` | 操作类型 | 枚举 `approve\|reject\|edit\|regen\|report\|dismiss` |
| `actor_id` | `uuid` | ❌ | `null` | `idx_course_crl_actor` | 操作人（学员或审核员） | FK → `auth.users.id`，`ON DELETE SET NULL` |
| `actor_role` | `text` | ✅ | — | — | 角色（learner/reviewer/admin/super） | 枚举 |
| `reason` | `text` | ❌ | `null` | — | 备注 | ≤ 500 |
| `report_type` | `text` | ❌ | `null` | — | 学员举报类型（仅 action='report' 时） | 枚举 `wrong_answer\|wrong_translation\|wrong_audio\|wrong_image\|other` |
| `diff` | `jsonb` | ❌ | `null` | — | 修改前后差异（仅 action='edit'） | — |
| `created_at` | `timestamptz` | ✅ | `now()` | `idx_course_crl_created` | — | — |

### 二、AI 补充字段

| 字段名 | 类型 | 补充理由 |
|--------|------|---------|
| `actor_role` | `text` | 区分"学员举报"和"审核员审核"两种行为，便于运营聚合 |
| `report_type` | `text` | [temp/05 §5.9](../../../temp/05-用户端模块设计.md) 学员举报固定类型 |

### 三、DDL 草案

```sql
create table zhiyu.course_content_review_log (
  id          uuid primary key default gen_random_uuid(),
  target_type text not null,
  target_id   uuid not null,
  action      text not null,
  actor_id    uuid,
  actor_role  text not null,
  reason      text,
  report_type text,
  diff        jsonb,
  created_at  timestamptz not null default now(),

  constraint fk_course_crl_actor
    foreign key (actor_id) references auth.users(id) on delete set null,
  constraint ck_course_crl_target_type
    check (target_type in ('kp','question','lesson','exam')),
  constraint ck_course_crl_action
    check (action in ('approve','reject','edit','regen','report','dismiss')),
  constraint ck_course_crl_actor_role
    check (actor_role in ('learner','reviewer','content_admin','ai_operator','super')),
  constraint ck_course_crl_report_type
    check (report_type is null or report_type in
      ('wrong_answer','wrong_translation','wrong_audio','wrong_image','other')),
  constraint ck_course_crl_reason_len
    check (reason is null or char_length(reason) <= 500),
  constraint ck_course_crl_report_type_only_with_report
    check ((action = 'report' and report_type is not null) or
           (action <> 'report' and report_type is null))
);

create index idx_course_crl_target  on zhiyu.course_content_review_log (target_type, target_id);
create index idx_course_crl_action  on zhiyu.course_content_review_log (action);
create index idx_course_crl_actor   on zhiyu.course_content_review_log (actor_id);
create index idx_course_crl_created on zhiyu.course_content_review_log (created_at desc);
```

### 四、RLS 策略

| 操作 | 角色 | 策略 |
|------|------|------|
| `insert` | `authenticated` | 仅允许 `action='report' and actor_id=auth.uid()`（通过 RPC `fn_report_question` 写入） |
| `select / update / delete` | `authenticated` | 拒绝 |
| 全部 | `service_role` | 全量 |

## 4. 表：`course_media_assets`（媒资库）

**对应需求中的对象**：[temp/03 §3.4.3](../../../temp/03-数据库schema.md) + [temp/04 §4.8](../../../temp/04-管理端模块设计.md)
**性质**：所有 TTS 音频 / AI 生图 / 上传图片 / Lottie 笔顺统一存储；按 `hash` 去重。

### 一、字段定义

| 字段名 | 类型 | 必填 | 默认值 | 索引 | 对应需求中的信息 | 校验规则 |
|--------|------|------|--------|------|----------------|---------|
| `id` | `uuid` | ✅ | `gen_random_uuid()` | PK | — | — |
| `kind` | `text` | ✅ | — | `idx_course_ma_kind` | 媒资类型 | 枚举 `audio\|image\|video\|svg\|lottie` |
| `url` | `text` | ✅ | — | — | CDN URL | URL，1–500 |
| `hash` | `text` | ✅ | — | `uq_course_ma_hash` (UNIQUE) | sha256 内容 hash（去重用） | 64 字符 hex |
| `meta` | `jsonb` | ✅ | `'{}'` | — | 元信息 | 见 §meta schema |
| `source` | `text` | ✅ | — | `idx_course_ma_source` | 来源 | 枚举 `tts\|generated_image\|human_recorded\|uploaded\|stock_library` |
| `voice_profile` | `text` | ❌ | `null` | — | TTS 声线（仅 kind='audio'） | 枚举 `xiaoxiao_zh\|yunyang_zh\|...` |
| `ref_kp_id` | `uuid` | ❌ | `null` | `idx_course_ma_kp` | 关联 KP（弱引用） | FK → `course_knowledge_points.id`，`ON DELETE SET NULL` |
| `ref_q_id` | `uuid` | ❌ | `null` | `idx_course_ma_q` | 关联 Question | FK → `course_questions.id`，`ON DELETE SET NULL` |
| `created_by` | `uuid` | ❌ | `null` | — | — | FK → `auth.users.id`，`ON DELETE SET NULL` |
| `created_at` | `timestamptz` | ✅ | `now()` | — | — | — |

### `meta` JSON Schema（按 `kind` 不同）

```jsonc
// kind='audio'
{ "duration_ms": 1820, "bitrate_kbps": 128, "format": "mp3" }

// kind='image'
{ "width": 1024, "height": 768, "format": "webp", "size_bytes": 45821 }

// kind='lottie'
{ "duration_ms": 3500, "frames": 105, "size_bytes": 23456 }
```

### 二、AI 补充字段

| 字段名 | 类型 | 补充理由 |
|--------|------|---------|
| `hash` | `text` | [temp/04 §4.8](../../../temp/04-管理端模块设计.md) "上传时按 hash 命中已有就直接复用" |
| `voice_profile` | `text` | 多声线管理（女声标准/男声标准等） |
| `ref_kp_id` / `ref_q_id` | `uuid` | "每个媒资能反查被哪些 KP/Q 引用，删除前必拦截"——弱引用，删除时置空 |

### 三、DDL 草案

```sql
create table zhiyu.course_media_assets (
  id            uuid primary key default gen_random_uuid(),
  kind          text not null,
  url           text not null,
  hash          text not null,
  meta          jsonb not null default '{}'::jsonb,
  source        text not null,
  voice_profile text,
  ref_kp_id     uuid,
  ref_q_id      uuid,
  created_by    uuid,
  created_at    timestamptz not null default now(),

  constraint fk_course_ma_kp
    foreign key (ref_kp_id) references zhiyu.course_knowledge_points(id) on delete set null,
  constraint fk_course_ma_q
    foreign key (ref_q_id) references zhiyu.course_questions(id) on delete set null,
  constraint fk_course_ma_creator
    foreign key (created_by) references auth.users(id) on delete set null,

  constraint uq_course_ma_hash unique (hash),
  constraint ck_course_ma_kind
    check (kind in ('audio','image','video','svg','lottie')),
  constraint ck_course_ma_source
    check (source in ('tts','generated_image','human_recorded','uploaded','stock_library')),
  constraint ck_course_ma_hash_format
    check (hash ~ '^[a-f0-9]{64}$'),
  constraint ck_course_ma_url_len
    check (char_length(url) between 1 and 500)
);

create index idx_course_ma_kind   on zhiyu.course_media_assets (kind);
create index idx_course_ma_source on zhiyu.course_media_assets (source);
create index idx_course_ma_kp     on zhiyu.course_media_assets (ref_kp_id);
create index idx_course_ma_q      on zhiyu.course_media_assets (ref_q_id);
```

### 四、RLS 策略

| 操作 | 角色 | 策略 |
|------|------|------|
| `select` | `anon`、`authenticated` | 全部放行（CDN URL 本身公开） |
| 写 | `service_role` | 全量 |

策略命名：`course_ma_select_public`、`course_ma_write_service`。

# 二 · 枚举定义

> 本规范不使用 PostgreSQL 原生 `enum` 类型（不可在线增删值），统一用 `text + CHECK` 实现。
> 前端共用枚举字面量见 [shared-schemas/src/course/enums.ts](../../../system/packages/shared-schemas/src/course/enums.ts)（待生成）。

---

## 1. KP 类型 `kp_type`（7 类）

来源：[temp/00 §0.5](../../../temp/00-总览与设计原则.md)

| 值 | 中文名 | 说明 | 内容字段差异 |
|----|--------|------|------------|
| `pinyin` | 拼音点 | 单字拼音音节 | `content.pinyin / tone` |
| `hanzi` | 汉字点 | 单字写法/笔顺 | `content.hanzi / strokes / radicals / svg_url` |
| `word` | 词汇点 | 1–4 字常用词 | `content.word_zh / pinyin / pos / examples` |
| `phrase` | 短语点 | 固定搭配 | `content.phrase_zh / pinyin / scenario` |
| `grammar` | 语法点 | 句式/语法 | `content.pattern / explanation / examples` |
| `sentence` | 句子点 | 完整句 | `content.sentence_zh / pinyin / context` |
| `dialog` | 对话点 | 多轮对话 | `content.turns: [{role, zh, pinyin}]` |

约束（[02-表定义-知识点与题目.md §1](./02-表定义-知识点与题目.md)）：
```sql
constraint ck_course_kp_type
  check (kp_type in ('pinyin','hanzi','word','phrase','grammar','sentence','dialog'))
```

## 2. 题型 `q_type`（12 类）

来源：[temp/02 §2.4](../../../temp/02-知识点与题型内容模板.md)

| 值 | 题型名 | 适配 KP 类型 | payload 关键字段 |
|----|--------|-------------|-----------------|
| `mcq_meaning` | 中→外文意思 | word/phrase/sentence | `stem_zh, options_i18n[4], answer_index` |
| `mcq_zh` | 外→中 | word/phrase | `stem_i18n, options_zh[4], answer_index` |
| `listen_pick` | 听音选词 | pinyin/word/sentence | `audio_url, options_zh[4], answer_index` |
| `listen_pinyin` | 听音填拼音 | pinyin/word | `audio_url, expected_pinyin` |
| `tone_pick` | 选声调 | pinyin | `audio_url, syllable_no_tone, answer_tone(1-4,0)` |
| `match_pairs` | 连线 | word | `pairs: [{left_zh, right_i18n}]`，6–8 对 |
| `sort_words` | 词序排列 | sentence/grammar | `tokens_zh[], correct_order[]` |
| `fill_blank_choice` | 选词填空 | grammar/sentence | `template_zh(含___), options_zh[4], answer_index` |
| `type_pinyin` | 输入拼音 | word/sentence | `audio_url 或 hanzi, expected_pinyin` |
| `type_zh_ime` | IME 打字 | word | `prompt_i18n, expected_zh` |
| `image_pick` | 看图选词 | word/phrase | `image_url, options_zh[4], answer_index` |
| `dialog_cloze` | 对话填空 | dialog | `turns_with_blank, options_zh[4], answer_index` |

约束：
```sql
constraint ck_course_q_type
  check (q_type in (
    'mcq_meaning','mcq_zh','listen_pick','listen_pinyin','tone_pick',
    'match_pairs','sort_words','fill_blank_choice','type_pinyin',
    'type_zh_ime','image_pick','dialog_cloze'
  ))
```

> ⚠️ 12 题型是否首发全部上线，见 [12-待确认问题清单.md Q7](./12-待确认问题清单.md)。

## 3. 内容审核状态 `content_status`

来源：[temp/04 §4.4 / §4.5 / §4.7](../../../temp/04-管理端模块设计.md) + [temp/06 §6.4 流程 C](../../../temp/06-关键业务流程.md)

适用：`course_knowledge_points.status` / `course_questions.status` / `course_lessons.status` / `course_exams.status`

| 值 | 含义 | 进入条件 |
|----|------|---------|
| `draft` | 草稿（AI 刚生成或人工新建） | 初始 |
| `reviewing` | 审核中 | 提交审核 |
| `approved` | 已通过 | 审核员 approve |
| `rejected` | 驳回 | 审核员 reject（带 reason） |
| `archived` | 归档（停用旧版本） | 灰度切换或人工下架 |

约束：
```sql
constraint ck_course_xxx_status
  check (status in ('draft','reviewing','approved','rejected','archived'))
```

附加字段：`is_published bool default false`，仅当 `status='approved' AND is_published=true` 时学员可见。

## 4. 节级学习状态 `lesson_progress_status`

来源：[temp/05 §5.4](../../../temp/05-用户端模块设计.md)

适用：`course_user_progress.status`

| 值 | 含义 |
|----|------|
| `locked` | 锁定（前置节未通过） |
| `unlocked` | 已解锁未开始 |
| `in_progress` | 学习中（已进入但未通过节末小测） |
| `passed` | 已通过节末小测 |

```sql
constraint ck_course_up_status
  check (status in ('locked','unlocked','in_progress','passed'))
```

## 5. 答题上下文 `answer_context_type`

适用：`course_user_answers.context_type` / `course_user_srs` 调用方

| 值 | 触发场景 |
|----|---------|
| `practice` | 节内练习（不计 SRS） |
| `lesson_quiz` | 节末小测 |
| `chapter_test` | 章测试 |
| `stage_exam` | 阶段考试 |
| `hsk_mock` | HSK 模考（dl 主题） |
| `srs_review` | SRS 复习 |

> SRS 计入策略：`practice` **不更新** SRS；其余 5 种 **均更新** SRS（详见 [04 §3 course_user_srs](./04-表定义-学员进度.md)）。

## 12. 主题码 `track_code`

来源：[temp/01 §1](../../../temp/01-课程目录骨架.md)

| 值 | 中文名 | Stage 数 | 备注 |
|----|--------|---------|------|
| `share` | 共享拼音入门 | 1（Stage 0） | 所有主题前置必修 |
| `ec` | 电商客服 | 6 | Stage 1–6 |
| `fc` | 工厂沟通 | 6 | Stage 1–6 |
| `dl` | 日常生活+HSK | 6 | Stage 1–6 |
| `hk` | 旅游酒店 | 6 | Stage 1–6 |

```sql
constraint ck_course_xxx_track
  check (track in ('share','ec','fc','dl','hk'))
```

> 写死，不允许动态新增。新增新主题需 DDL 变更 + 前端枚举同步发版。

## 13. 学员母语/UI 语言 `ui_lang`

适用：`auth_user_profiles.ui_lang` / `auth_user_profiles.native_lang`（账号域，非课程域，仅枚举值定义在此）

| 值 | 名称 |
|----|------|
| `vi` | 越南语 |
| `th` | 泰语 |
| `id` | 印尼语 |
| `en` | 英语 |

> 注意：**不含 `zh`**——中文是学习目标，UI 不会以中文呈现给目标学员。
> i18n jsonb 列同样仅 4 key：`{vi, th, id, en}`。

## 14. 管理员角色 `admin_role`

来源：[temp/03 §3.5.1](../../../temp/03-数据库schema.md)

| 值 | 权限 |
|----|------|
| `super` | 全部 |
| `content_admin` | KP/题目/课程目录 CRUD |

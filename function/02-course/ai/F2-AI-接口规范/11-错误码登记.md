# 横切：错误码登记

> 课程域错误码统一前缀 `COURSE_*`，遵循 [grules/G1-04 §四](../../../grules/G1-架构与技术规范/04-API接口规范.md)。
> HTTP 状态码与业务 code 双层；客户端按 code 决定 i18n 文案与重试策略。

---

## 1. 通用错误（与 G1 共用，本节列出常用以便对照）

| code | HTTP | 描述 |
|------|------|------|
| `40001` | 400 | 请求参数格式错 |
| `40002` | 400 | 排序字段不在白名单 |
| `40101` | 401 | 未登录 / token 失效 |
| `40301` | 403 | 角色权限不足 |
| `40400` | 404 | 实体不存在 |
| `42900` | 429 | 限流 |
| `50000` | 500 | 服务内部错误 |

## 3. Schema / 一致性（4xx · 400）

| code | 描述 |
|------|------|
| `COURSE_KP_CONTENT_SCHEMA_MISMATCH` | content 不匹配 kp_type |
| `COURSE_PAYLOAD_SCHEMA_MISMATCH` | payload 不匹配 q_type / user_answer 不匹配 q_type |
| `COURSE_LESSON_CODE_MISMATCH` | lesson.code 三段与上级不一致 |
| `COURSE_EXAM_BLUEPRINT_OUT_OF_SCOPE` | blueprint.kp 超出 scope_ref 范围 |
| `COURSE_KP_NOT_IN_LESSON` | last_kp_id 不属于该节（C7） |
| `COURSE_AI_PROMPT_SCOPE_MISMATCH` | prompt.scope 与 job_type 不兼容 |
| `COURSE_QUIZ_INCOMPLETE_ANSWERS` | 提交节末小测/考试时缺答 |
| `COURSE_REPORT_TYPE_REQUIRED` | action='report' 但缺 report_type |

## 9. 错误响应统一格式

```json
{
  "code": "COURSE_PREREQUISITE_NOT_MET",
  "http_status": 403,
  "message": "请先完成前置阶段的学习",
  "details": {
    "required_stage_id": "<uuid>",
    "current_user_progress": "..."
  },
  "request_id": "uuid-v4",
  "server_time": "2026-04-15T10:00:00+08:00"
}
```

由 Hono 全局中间件统一封装，业务代码只需 `throw new BizError('COURSE_xxx', { details })`。

## 10. 维护规则

- 新增错误码 → 必须在本文件登记 + 在 `apps/api-app/src/errors/course.ts`（待生成）枚举
- 删除错误码 → 必须先 deprecation 一个版本，前端无引用后再删
- i18n 文案 → 后端只下发 code + 默认 zh message；前端按 code 查 5 语言文案表（`zh / vi / th / id / en`，与 [F1-12-Q10](../F1-AI-数据模型规范/12-待确认问题清单.md) 一致）

## 11. 业务码 ↔ 5 位数字映射（[grules/G1-04 §四](../../../grules/G1-架构与技术规范/04-API接口规范.md)）

> 本文件中 `COURSE_*` 字符串是**业务命名**（便于代码读写）；对外响应的 `code` 字段返回 **5 位数字**。
> 映射表同时登记到 `packages/shared-config/src/error-codes.js`，前后端共享。

### 号段划分（课程域）

| 范围 | 类别 | HTTP |
|------|------|------|
| 40300–40399 | 结构/字段校验错误 | 400 |
| 40400–40499 | 业务上下文冲突 | 400/409 |
| 40500–40599 | 鉴权/鉴别 | 401/403 |
| 40600–40699 | 资源未找到 | 404 |
| 40700–40799 | 状态不允许 | 409 |
| 40800–40899 | 限流 | 429 |
| 40900–40999 | 依赖服务 | 502/504 |
| 50300–50399 | 服务端未知错误 | 500 |

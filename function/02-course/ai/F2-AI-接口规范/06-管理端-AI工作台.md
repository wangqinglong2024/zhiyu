## A15 · Prompt 库 CRUD

| 操作 | 方法 | 路径 |
|------|------|------|
| 列表 | GET | `/admin/v1/course/ai-prompts?scope=&active_only=&page=` |
| 详情 | GET | `/admin/v1/course/ai-prompts/:id` |
| 新建 | POST | `/admin/v1/course/ai-prompts` |
| 编辑（同版本） | PATCH | `/admin/v1/course/ai-prompts/:id` |
| 软删 | DELETE | `/admin/v1/course/ai-prompts/:id` |

权限：`ai_operator+`

**新建请求体**：

```json
{
  "prompt_code": "prompt_kp_word_v3",
  "scope": "kp_word",
  "template": "你是中文教学内容生成助手...{{word_zh}}...",
  "variables": {
    "word_zh": { "type":"string", "required": true },
    "track":   { "type":"enum", "required": true, "values":["ec","fc","hk","dl","share"] }
  },
  "version": 3
}
```

**业务校验**：
- `prompt_code` 唯一且符合正则 `^prompt_[a-z0-9_]+_v\d+$`
- 同 `scope` 已有 active 的不影响（新建默认 `is_active=false`）
- `template` 长度 1–8000

**响应 201**：返回新建对象。

## A16 · Prompt 启用 / 归档

| 操作 | 方法 | 路径 |
|------|------|------|
| 启用 | POST | `/admin/v1/course/ai-prompts/:id:activate` |
| 归档 | POST | `/admin/v1/course/ai-prompts/:id:archive` |

权限：`ai_operator+`

**业务校验**：
- 启用：事务内，将同 `scope` 旧的 active 自动改为 `is_active=false`，本条 `is_active=true`
- 归档：仅当 `is_active=false`；置 `is_active=false`（已是）+ 软删（`deleted_at=now()`），新 Job 不可再选

**响应 200**：

```json
{ "code": 0, "data": { "scope": "kp_word", "active_prompt_code": "prompt_kp_word_v3" } }
```

## A17 · 创建 AI Job

**方法**：POST
**路径**：`/admin/v1/course/ai-jobs`
**权限**：`ai_operator+`

**请求体**：

```json
{
  "job_type": "gen_kp_batch",         // 6 种 job_type
  "prompt_id": "<uuid>",               // 可省略 → 取 scope 当前 active
  "model": "gpt-5-pro",
  "params": { "temperature": 0.3, "max_tokens": 1500 },
  "input": {
    "track": "ec",
    "stage_no": 2,
    "kp_type": "word",
    "target_languages": ["vi","th","id","en"],
    "target_count": 50,
    "known_vocab_list_source": "auto"  // auto = 后端从 DB 拉取该赛道 stage<=2 已发布词
  },
  "budget_usd": 5.00,
  "idempotency_key": "ec_s2_word_batch_2026_04_15_a"
}
```

**业务校验**：
- `idempotency_key` 唯一；命中已有 → 返回旧 job_id（不创建）
- `prompt_id` 必须 `scope` 与 `job_type` 兼容（如 `gen_kp_batch` → `scope='kp_<type>'`）
- `budget_usd` ≥ 0；为空表示用平台默认上限

**副作用**：
- 写 `course_ai_generation_jobs` 行（`status='queued'`）
- worker（`apps/api-app` 或独立 worker）异步执行

**响应 202**：

```json
{
  "code": 0,
  "data": {
    "job_id": "<uuid>",
    "status": "queued",
    "estimated_finish_at": "2026-04-15T10:35:00+08:00"
  }
}
```

**错误**：

| HTTP | code | 触发 |
|------|------|------|
| 400 | `COURSE_AI_PROMPT_SCOPE_MISMATCH` | prompt 与 job_type 不兼容 |
| 401 | 40101 | 未登录 |
| 403 | `COURSE_ADMIN_ONLY` | 角色不足 |
| 409 | `COURSE_AI_JOB_DUPLICATE` | idempotency_key 已存在 → data 仍返回旧 job_id |

## A18 · AI Job 列表 / 详情 / 重试

### A18.1 列表

**方法**：GET
**路径**：`/admin/v1/course/ai-jobs?status=&job_type=&model=&page=&sort=-created_at`

**响应 200**：

```json
  "data": {

        "model": "gpt-5-pro",
        "status": "partial",
        "generated_count": 47,
        "failed_count": 3,
        "cost_usd": 4.82,
        "budget_usd": 5.00,
        "created_at": "...",
        "started_at": "...",
        "finished_at": "..."
      }
    ],
    "pagination": {...}
  }
}
```

### A18.2 详情

**方法**：GET
**路径**：`/admin/v1/course/ai-jobs/:id`

**响应 200**：完整字段含 `input / output / failed_items / params`。

### A18.3 重试

**方法**：POST
**路径**：`/admin/v1/course/ai-jobs/:id:retry`
**权限**：`ai_operator+`

**请求体**：

```json
{
  "scope": "failed_only",   // failed_only / all
  "override_model": null,   // 可选，覆盖原模型
  "override_budget_usd": null
}
```

**业务校验**：
- 原 job 必须 `status ∈ {done, partial, failed}`
- `scope='failed_only'` 时仅当 `failed_items` 非空

**副作用**：
- 创建新 job（`idempotency_key=<原>:retry-<N>`）
- `input.parent_job_id = <原 id>`，便于追溯

**响应 202**：与 A17 一致。

**错误**：

| HTTP | code | 触发 |
|------|------|------|
| 400 | `COURSE_AI_JOB_NOT_RETRIABLE` | 原 job 状态不允许重试 |
| 400 | `COURSE_AI_JOB_NO_FAILED_ITEMS` | failed_only 但无失败项 |

## I3 · Worker 心跳（内部）

**方法**：POST
**路径**：`/internal/v1/course/ai-jobs/:id:heartbeat`
**权限**：`service-role`

**请求体**：

```json
{
  "phase": "running",
  "progress": 0.6,
  "current_cost_usd": 2.45
}
```

**副作用**：
- 更新 `started_at`（首次）；写监控指标
- 若 `current_cost_usd >= budget_usd`：worker 必须自杀；本接口返 `409 COURSE_AI_BUDGET_EXCEEDED` 并将 job 标 `partial`

**响应 200**：`{ "code":0, "data": { "should_continue": true } }`

> 详细回调见 [09-AI补充接口.md](./09-AI补充接口.md)。

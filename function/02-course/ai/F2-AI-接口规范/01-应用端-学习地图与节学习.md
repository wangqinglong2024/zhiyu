# 应用端：学习地图与节学习

> 学员端的"探索 → 进入节 → 学习卡片 → 听音"全链路。
> 所有接口需登录（除 C1 列出主题）；RLS 强制 `status='approved' AND is_published=true AND deleted_at IS NULL`。

---

## C1 · 列出 5 主题

**对应需求中的操作**：[temp/05 §5.2 选择主题](../../../temp/05-用户端模块设计.md)

**方法**：GET
**路径**：`/api/v1/course/tracks`
**权限**：公开（未登录可看以诱导注册）

**业务校验**：无

**请求参数**：无

**成功响应 200**：

```json
{
  "code": 0,
  "message": "ok",
  "data": {
    "items": [
      {
        "code": "share",
        "name_i18n": { "vi":"...","th":"...","id":"...","en":"Pinyin Foundations" },
        "description_i18n": { "vi":"...","th":"...","id":"...","en":"Required basics for all tracks" },
        "stage_count": 1,
        "lesson_count": 24,
        "is_required": true,
        "sort_order": 0
      }
      /* ec / fc / dl / hk */
    ]
  }
}
```

`Cache-Control: public, max-age=3600`。

## C2 · 切换/选择当前主题

**对应需求中的操作**：[temp/05 §5.2](../../../temp/05-用户端模块设计.md) 选主题按钮 / 切换主题入口

**方法**：POST
**路径**：`/api/v1/course/me/select-track`
**权限**：登录

**业务校验**：
- 学员若未通过 share Stage 0，强制写为 `share`（无论提交什么）
- 提交的 `track` 必须存在于 `course_tracks`

**请求体**：

```json
{ "track": "ec" }
```

**成功响应 200**：

```json
{
  "code": 0,
  "data": {
    "current_track": "ec",
    "shared_stage_done": false,
    "redirect_to_track": "share"  // 若强制改写，给出提示
  }
}
```

**错误**：

| HTTP | code | 触发 |
|------|------|------|
| 400 | `COURSE_INVALID_TRACK_CODE` | track 不在枚举 |
| 401 | 40101 | 未登录 |

## C3 · 学习地图

**对应需求中的操作**：[temp/05 §5.4 学习地图](../../../temp/05-用户端模块设计.md)

**方法**：GET
**路径**：`/api/v1/course/tracks/:track/map`
**权限**：登录

**业务校验**：
- `:track` 必须存在
- 学员当前主题 ≠ `:track` 时，仍可查看（用于切换前预览），但 `progress` 字段全为 `locked`

**请求参数**：

| 参数 | 位置 | 类型 | 必填 | 说明 |
|------|------|------|------|------|
| `track` | path | enum | ✅ | `share/ec/fc/dl/hk` |
| `include` | query | csv | ❌ | 默认 `stages,chapters,lessons,progress`；可裁剪 |

## C4 · 进入节

**对应需求中的操作**：[temp/05 §5.5 节内学习](../../../temp/05-用户端模块设计.md) + [temp/06 §6.7 流程 F](../../../temp/06-关键业务流程.md)

**方法**：GET
**路径**：`/api/v1/course/lessons/:lesson_id`
**权限**：登录

**业务校验**：
- 节必须 `status='approved' AND is_published=true`
- 节所属 stage 必须满足前置：学员已通过上一 stage（除非 `is_quiz_required=false` 节，详见 F1-Q4）
- 若不满足前置：`403 COURSE_PREREQUISITE_NOT_MET`

**请求参数**：

| 参数 | 位置 | 说明 |
|------|------|------|
| `lesson_id` | path | UUID |

**副作用**：
- 若 `course_user_progress` 不存在，自动创建 `status='in_progress', started_at=now()`
- 若已 `unlocked` → `in_progress`

**成功响应 200**：

```json
{
  "code": 0,
  "data": {
    "lesson": {
      "id": "<uuid>",
      "code": "ec-1-1-1",
      "title_zh": "...",
      "objective_i18n": { "vi":"...","th":"...","id":"...","en":"..." },
      "is_quiz_required": true,
      "estimated_minutes": 8
    },
    "kp_cards": [
      {
        "kp_id": "<uuid>",
        "kp_code": "kp_ec_word_000123",
        "kp_type": "word",
        "title_zh": "你好",
        "card_order": 1,
        "preview": { "pinyin": "nǐ hǎo" }
      }
      /* ... 按 card_order asc */
    ],
    "checkpoint": {
      "last_kp_id": "<uuid>",
      "last_position": 5
    },
    "progress": { "status":"in_progress", "best_score": null, "attempts": 0 }
  }
}
```

**错误**：

| HTTP | code | 触发 |
|------|------|------|
| 401 | 40101 | 未登录 |
| 403 | `COURSE_PREREQUISITE_NOT_MET` | 前置 stage 未通过 |
| 404 | 40400 | 节不存在或未发布 |

## C6 · 触发/获取 KP 朗读音频

**对应需求中的操作**：[temp/05 §5.5.2 听音](../../../temp/05-用户端模块设计.md) + [temp/06 §6.10 流程 I TTS](../../../temp/06-关键业务流程.md)

**方法**：POST
**路径**：`/api/v1/course/kps/:kp_id/audio`
**权限**：登录

**业务校验**：
- KP 必须可见
- 若 `course_media_assets` 已有命中（按 `text+voice_profile` hash），直接返回 URL
- 否则同步触发 TTS（短，<2s 直接返回；长则返 `202 + audio_status='pending'`，前端轮询 C5 取最新）

**请求体**：

```json
{
  "voice_profile": "xiaoxiao_zh",   // 默认
  "speed": 1.0                       // 0.5–2.0
}
```

**成功响应 200**（同步返回）：

```json
{
  "code": 0,
  "data": {
    "audio_url": "https://cdn.example.com/audio/abc.mp3",
    "duration_ms": 1820,
    "cache_hit": true
  }
}
```

**响应 202**（异步生成中）：

```json
{
  "code": 0,
  "data": { "audio_status": "pending", "job_id": "<uuid>" }
}
```

**错误**：

| HTTP | code | 触发 |
|------|------|------|
| 401 | 40101 | 未登录 |
| 404 | 40400 | KP 不存在 |
| 429 | `COURSE_RATE_LIMIT` | 单 IP 20/分 |

限流：单 IP 20/分（缓存命中不计入）。

## C7 · 上报节内位置（断点续学）

**对应需求中的操作**：[temp/05 §5.5.3 退出确认](../../../temp/05-用户端模块设计.md)

**方法**：PUT
**路径**：`/api/v1/course/lessons/:lesson_id/checkpoint`
**权限**：登录

**业务校验**：
- 节必须可见且学员已 `in_progress`
- `last_kp_id` 必须属于本节

**请求体**：

```json
{
  "last_kp_id": "<uuid>",
  "last_position": 5
}
```

**成功响应 200**：

```json
{ "code": 0, "data": { "saved": true } }
```

**错误**：

| HTTP | code | 触发 |
|------|------|------|
| 400 | `COURSE_KP_NOT_IN_LESSON` | last_kp_id 不属于本节 |
| 401 | 40101 | 未登录 |
| 404 | 40400 | 节不存在 |

> 该接口高频调用（卡片切换即 PUT），前端需 debounce 1s。

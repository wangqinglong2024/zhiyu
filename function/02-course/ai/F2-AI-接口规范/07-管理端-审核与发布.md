## A19 · 审核队列

**方法**：GET
**路径**：`/admin/v1/course/review/queue`
**权限**：`reviewer+`

**请求参数**：

| 参数 | 类型 | 说明 |
|------|------|------|
| `target_type` | enum | `kp/question/lesson/exam`（可选） |
| `track` | enum | 5 主题筛选 |
| `kp_type / q_type` | enum | 进一步筛选 |
| `created_by_ai` | bool | true=AI 生产；false=人工新建 |
| `assigned_to_me` | bool | 只看分给我的 |
| `priority` | enum | `high/normal/low`（按 KP 关联节数派生） |
| `page / page_size / sort` | — | 默认 `-created_at` |

**响应 200**：

```json
{
  "code": 0,
  "data": {
    "items": [
      {
        "target_type": "kp",
        "target_id": "<uuid>",
        "kp_code": "kp_ec_word_000123",
        "title_zh": "你好",
        "track": "ec",
        "status": "reviewing",
        "ai_job_id": "<uuid>",
        "created_at": "...",
        "submitted_at": "..."
      }
    ],
    "pagination": {...}
  }
}
```

## A20 · 审核动作（approve / reject / edit）

通用路径：`POST /admin/v1/course/review/:target_type/:id:<action>`

| Target | Action | 路径 | 权限 |
|--------|--------|------|------|
| kp | approve / reject / edit | `/admin/v1/course/review/kp/:id:approve` 等 | `reviewer+` |
| question | 同上 | `/admin/v1/course/review/question/:id:approve` 等 | `reviewer+` |
| lesson | 同上 | `/admin/v1/course/review/lesson/:id:approve` 等 | `reviewer+` |
| exam | 同上 | `/admin/v1/course/review/exam/:id:approve` 等 | `reviewer+` |

### approve

**请求体**：

```json
{
  "auto_publish": true,            // 通过即发布
  "comment": null
}
```

**副作用**：
- 状态 `reviewing → approved`
- 若 `auto_publish=true`：同时 `is_published=true`
- 写 `course_content_review_log` (action=`approve`)

**响应 200**：

```json
{
  "code": 0,
  "data": { "id":"<uuid>", "status":"approved", "is_published": true }
}
```

### reject

**请求体**：

```json
{
  "reason": "翻译不准；例句过于书面化",
  "request_revision": true        // true=回 draft，false=终态 rejected
}
```

**副作用**：状态 `reviewing → rejected` 或 `→ draft`；写 review_log（带 reason）。

### edit

**请求体**：

```json
{
  "patch": {
    "title_zh": "你好",
    "content.translation_i18n.vi": "Xin chào!"
  },
  "comment": "修订越南语",
  "approve_after_edit": true
}
```

**副作用**：
- 应用 patch（仅审核员可改"内容字段"，非结构性字段）
- 派生新版本（同 [05 §A8 KP 编辑](./05-管理端-KP与题目CRUD.md)）
- `approve_after_edit=true` → 立即 approve + 可选 publish
- 写 review_log (action=`edit`，带 diff)

**响应 200**：

```json
{
  "code": 0,
  "data": { "id":"<new_id>", "version": 4, "status": "approved", "is_published": false }
}
```

**错误**：

| HTTP | code | 触发 |
|------|------|------|
| 400 | `COURSE_INVALID_REVIEW_ACTION` | 状态机不允许该动作 |
| 401 | 40101 | 未登录 |
| 403 | `COURSE_ADMIN_ONLY` | 角色不足 |
| 404 | 40400 | 目标不存在 |
| 409 | `COURSE_STALE_VERSION` | If-Match 不一致 |

## A21 · 发布 / 下架

通用：`POST /admin/v1/course/:target_type/:id:publish` / `:unpublish`

| Target | 权限 | 备注 |
|--------|------|------|
| kp / question / lesson / exam | `content_admin+` | 必须 `status='approved'` |

**业务校验**：
- publish：要求 `status='approved' AND is_published=false`
- unpublish：仅 super 可下架已发布超过 7 天的内容（`COURSE_PUBLISH_DURATION_LIMIT`）

**副作用**：
- publish：`is_published=true`，同时学员侧立即可见
- 节发布需先确保该节绑定的所有 KP 都已 published（否则 `409 COURSE_LESSON_HAS_UNPUBLISHED_KPS`）
- 题发布需先确保关联 KP 已 published

**响应 200**：

```json
{ "code": 0, "data": { "id":"<uuid>", "is_published": true } }
```

## A27 · 处理举报

| 操作 | 方法 | 路径 |
|------|------|------|
| 驳回 | POST | `/admin/v1/course/reports/:log_id:dismiss` |
| 转为审核任务 | POST | `/admin/v1/course/reports/:log_id:adopt` |

**adopt 副作用**：
- 把目标对象从 `approved` 强制回到 `reviewing`，进入审核队列（高优先级）
- 写一条 `action='edit'` 的关联 review_log，标注"由学员举报触发"

**响应 200**：

```json
{ "code": 0, "data": { "report_status": "adopted", "target_status": "reviewing" } }
```

**错误**：

| HTTP | code | 触发 |
|------|------|------|
| 400 | `COURSE_REPORT_ALREADY_HANDLED` | 已处理 |
| 404 | 40400 | report 不存在 |

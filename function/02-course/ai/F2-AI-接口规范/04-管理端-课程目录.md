# 管理端：课程目录

> 管理 `tracks / stages / chapters / lessons` 四级目录与节-KP 绑定。
> 全部接口要求 `Authorization: Bearer <admin_token>`，最低角色 `content_admin`（除主题列表为只读）。

---

## A1 · 列出主题

**方法**：GET
**路径**：`/admin/v1/course/tracks`
**权限**：`readonly+`

**响应 200**：与 [C1](./01-应用端-学习地图与节学习.md) 类似，附 `is_published / sort_order`。

> 主题写死 5 个，无 CUD 接口。

---

## A2 · 阶段 CRUD

| 操作 | 方法 | 路径 |
|------|------|------|
| 列表 | GET | `/admin/v1/course/stages?track=<code>` |
| 详情 | GET | `/admin/v1/course/stages/:id` |
| 新建 | POST | `/admin/v1/course/stages` |
| 编辑 | PATCH | `/admin/v1/course/stages/:id` |
| 软删 | DELETE | `/admin/v1/course/stages/:id` |

权限：`content_admin+`

**新建请求体**：

```json
{
  "track": "ec",
  "stage_no": 1,
  "name_i18n": { "vi":"...","th":"...","id":"...","en":"Greetings" },
  "hsk_level": "A1",
  "sort_order": 1
}
```

**业务校验**：
- `(track, stage_no)` 唯一
- `track` ∈ 5 个枚举
- `stage_no` 0–6（share=0，其余 1–6）
- `name_i18n` 必须含 4 key

**软删校验**：阶段下若有任何 chapter 未软删，返 `409 COURSE_STAGE_NOT_EMPTY`。

## A3 · 章节 CRUD + 调序

| 操作 | 方法 | 路径 |
|------|------|------|
| 列表 | GET | `/admin/v1/course/chapters?stage_id=<id>` |
| 详情 | GET | `/admin/v1/course/chapters/:id` |
| 新建 | POST | `/admin/v1/course/chapters` |
| 编辑 | PATCH | `/admin/v1/course/chapters/:id` |
| 软删 | DELETE | `/admin/v1/course/chapters/:id` |
| 批量调序 | POST | `/admin/v1/course/chapters:reorder` |

权限：`content_admin+`

**新建请求体**：

```json
{
  "stage_id": "<uuid>",
  "chapter_no": 1,
  "name_i18n": { "vi":"...","th":"...","id":"...","en":"..." },
  "objective_i18n": { ... },
  "sort_order": 1
}
```

**调序请求体**：

```json
{
  "stage_id": "<uuid>",
  "ordered_chapter_ids": ["<uuid>", "<uuid>", ...]
}
```

后端在事务内按数组顺序更新 `chapter_no` 与 `sort_order`。

## A4 · 节 CRUD + 调序 + 绑定 KP

### A4.1 列表 / 详情 / 新建 / 编辑 / 软删

| 操作 | 方法 | 路径 |
|------|------|------|
| 列表 | GET | `/admin/v1/course/lessons?chapter_id=<id>&status=<>&q=<>&page=&page_size=&sort=` |
| 详情 | GET | `/admin/v1/course/lessons/:id` |
| 新建 | POST | `/admin/v1/course/lessons` |
| 编辑 | PATCH | `/admin/v1/course/lessons/:id` |
| 软删 | DELETE | `/admin/v1/course/lessons/:id` |
| 调序 | POST | `/admin/v1/course/lessons:reorder` |
| 绑定 KP | POST | `/admin/v1/course/lessons/:id:bind-kps` |
| 解绑 KP | POST | `/admin/v1/course/lessons/:id:unbind-kps` |

**新建请求体**：

```json
{
  "chapter_id": "<uuid>",
  "lesson_no": 1,
  "code": "ec-1-1-1",
  "title_zh": "你好，欢迎光临",
  "title_i18n": { "vi":"...","th":"...","id":"...","en":"..." },
  "objective_i18n": { ... },
  "is_quiz_required": true,
  "estimated_minutes": 8,
  "sort_order": 1
}
```

**业务校验**：
- `code` 三段必须与 `chapter -> stage -> track` 一致
- `(chapter_id, lesson_no)` 唯一

### A4.2 绑定 KP

**请求体**：

```json
{
  "kp_orders": [
    { "kp_id": "<uuid>", "card_order": 1 },
    { "kp_id": "<uuid>", "card_order": 2 }
  ]
}
```

**业务校验**：
- 每个 `kp_id` 必须存在
- `card_order` 在节内唯一
- 操作覆盖（先 DELETE 旧 lesson_kp，再 INSERT），事务内完成

**响应 200**：`{ "code":0, "data": { "bound_count": 12 } }`

## A5 · 节列表查询条件

`A4.1 列表` 支持以下 query：

| 参数 | 类型 | 说明 |
|------|------|------|
| `chapter_id` | uuid | 章 id（可选） |
| `track` | enum | 5 主题 |
| `stage_no` | int | — |
| `status` | enum | `draft/reviewing/approved/rejected/archived` |
| `is_published` | bool | — |
| `q` | string | 模糊匹配 `title_zh` / `code` |
| `page` / `page_size` | int | — |
| `sort` | enum | 白名单 `lesson_no/created_at/updated_at` |

## 共通错误

| HTTP | code | 触发 |
|------|------|------|
| 400 | `COURSE_INVALID_LESSON_CODE` | code 格式错或与上级不一致 |
| 401 | 40101 | 未登录 |
| 403 | `COURSE_ADMIN_ONLY` | 角色不足 |
| 404 | 40400 | 实体不存在 |
| 409 | `COURSE_STAGE_NOT_EMPTY` / `COURSE_CHAPTER_NOT_EMPTY` | 软删但下级非空 |
| 409 | `COURSE_LESSON_NO_DUPLICATE` | (chapter_id, lesson_no) 冲突 |

> 编辑冲突策略详见 [10-并发与冲突处理.md](./10-并发与冲突处理.md)。

## C14 · 开始考试

**对应需求中的操作**：[temp/05 §5.7](../../../temp/05-用户端模块设计.md) + [temp/06 §6.9 流程 H](../../../temp/06-关键业务流程.md)

**方法**：POST
**路径**：`/api/v1/course/exams/:exam_id:start`
**权限**：登录

**业务校验**：
- 学员必须解锁
- 阶段考已通过 (`max_attempts=1`) → `403 COURSE_STAGE_EXAM_ALREADY_PASSED`
- 学员是否有该 exam 的 `in_progress` attempt：返回旧 attempt（不创建新的）
- 抽题：基于 `course_exams.blueprint`，事务内创建 `course_user_exam_attempts` 行 + snapshot
- 计时：`expires_at = started_at + time_limit_sec`

**请求体**：无

**成功响应 200**：

```json
{
  "code": 0,
  "data": {
    "attempt_id": "<uuid>",
    "attempt_no": 1,
    "started_at": "2026-04-15T10:00:00+08:00",
    "expires_at": "2026-04-15T10:30:00+08:00",
    "question_ids": ["<uuid>", "<uuid>", ...]
  }
}
```

**错误**：

| HTTP | code | 触发 |
|------|------|------|
| 401 | 40101 | 未登录 |
| 403 | `COURSE_EXAM_LOCKED` | 未解锁 |
| 403 | `COURSE_STAGE_EXAM_ALREADY_PASSED` | 阶段考已过 |
| 404 | 40400 | exam 不存在 |

## C16 · 提交考试

**方法**：POST
**路径**：`/api/v1/course/exam-attempts/:attempt_id:submit`
**权限**：登录（仅本人）

**业务校验**：
- 状态必须 `in_progress`
- `now() <= expires_at`（否则由 cron 标 `expired`，本接口返 `410`）
- 每题必须经 C9 提交（按 attempt_id 关联）

**请求体**：无（题目答案已经通过 C9 写入流水）

**副作用**：
- 计算总分 → 更新 attempt：`state='submitted', score, passed, finished_at=now()`
- 若 `scope_type='stage_exam' AND passed`：解锁下一 stage（trigger）
- 若 `scope_type='lesson_quiz'`：等同 C10（节末小测路径，二选一）

**成功响应 200**：

```json
{
  "code": 0,
  "data": {
    "score": 88,
    "passed": true,
    "wrong_questions": [
      { "question_id": "<uuid>", "user_answer": {...}, "correct_answer": {...}, "explanation_i18n": {...} }
    ]
  }
}
```

**错误**：

| HTTP | code | 触发 |
|------|------|------|
| 400 | `COURSE_QUIZ_INCOMPLETE_ANSWERS` | 缺少题目答案 |
| 410 | `COURSE_EXAM_EXPIRED` | 已超时 |

## C17 · 主动放弃考试

## C18 · 举报题目

**对应需求中的操作**：[temp/05 §5.9 题目反馈](../../../temp/05-用户端模块设计.md)

**方法**：POST
**路径**：`/api/v1/course/questions/:q_id:report`
**权限**：登录

**业务校验**：
- 同一学员对同一题 24h 内仅允许 1 条 report
- `report_type` 必须合法

**请求体**：

```json
{
  "report_type": "wrong_translation",
  "reason": "印尼语翻译有误，应为 ..."
}
```

**副作用**：写入 `course_content_review_log` (action=`report`, actor_role=`learner`)

**成功响应 200**：

```json
{ "code": 0, "data": { "reported": true } }
```

**错误**：

| HTTP | code | 触发 |
|------|------|------|
| 400 | `COURSE_INVALID_REPORT_TYPE` | report_type 不合法 |
| 401 | 40101 | 未登录 |
| 404 | 40400 | 题目不存在 |
| 429 | `COURSE_REPORT_DUPLICATE` | 24h 内重复举报 |

## I1 · TTS 服务回调

**方法**：POST
**路径**：`/internal/v1/course/tts/callback`
**权限**：`service-role`，header `X-Signature`（HMAC-SHA256，密钥从 `env.TTS_CALLBACK_SECRET`）

**请求体**：

```json
{
  "job_id": "<uuid>",
  "task_id": "<provider_task_id>",
  "status": "success",                 // success / failed
  "audio_url": "https://provider.cdn.com/xxx.mp3",
  "duration_ms": 1820,
  "voice_profile": "xiaoxiao_zh",
  "kp_id": "<uuid|null>",
  "text_hash": "<sha256>",
  "error": null
}
```

**业务校验**：
- HMAC 签名校验失败 → `401`
- `job_id` 不存在 → `404`
- 已收到过 `task_id` → 幂等返回 `200 OK`（防重投）

**副作用**：
- 下载 / mirror 音频到本平台 CDN（按平台策略）
- 计算音频 hash → 命中 `course_media_assets` 复用，否则插入新行
- 更新关联 `course_ai_generation_jobs.generated_count += 1` / `failed_items 追加`
- 若该 job 全部子任务完成：`status='done'/'partial'/'failed'`，写 `finished_at`

**响应 200**：`{ "code":0, "data": { "received": true } }`

**错误**：401 / 404 / 422（payload 无法解析）

## I2 · 图像生成回调

**方法**：POST
**路径**：`/internal/v1/course/image/callback`
**权限**：`service-role`

**请求体**：与 I1 类似，字段差异：

```json
{
  "job_id": "<uuid>",
  "task_id": "...",
  "status": "success",
  "image_url": "...",
  "width": 1024,
  "height": 768,
  "format": "webp",
  "size_bytes": 45821,
  "kp_id": "<uuid|null>",
  "q_id": "<uuid|null>",
  "error": null
}
```

副作用同 I1：mirror + dedup + 更新 job。

## I5 · 考试限时到期 cron 触发

**方法**：POST
**路径**：`/internal/v1/course/exam-attempts:expire`
**权限**：`service-role`（仅 cron 调用）

**业务**：
- `UPDATE course_user_exam_attempts SET state='expired', score=<计算>, finished_at=now() WHERE state='in_progress' AND now() > expires_at`
- 返回处理数量

**响应 200**：

```json
{ "code":0, "data": { "expired_count": 12 } }
```

调度：每分钟（`scripts/cron/expire-exam-attempts.ts`）。

## I6 · LLM Mock 适配（开发环境）

**说明**：[zhiyu-docker-policy](../../../../README.md) 要求"missing API keys 必须用 mock，不阻塞 docker 测试"。
- `apps/api-app` 内置 `MockLLMAdapter`：
  - `gen_kp_batch` → 返回固定 fixtures（5 条 word KP）
  - `gen_q_batch` → 返回 5 条 mcq_meaning 题
  - `gen_audio` → 返回 placeholder.mp3 URL
- 通过 `env.LLM_PROVIDER=mock` 启用
- mock 适配器仍然走 worker → I3 心跳 → 完成
- 生产环境必须 `LLM_PROVIDER=openai|claude|qwen` 等，校验 API key 存在

## 错误码（横切）

| code | 描述 |
|------|------|
| `COURSE_INTERNAL_BAD_SIGNATURE` | HMAC 签名校验失败 |
| `COURSE_INTERNAL_DUPLICATE_TASK` | 同 task_id 已处理（幂等成功，info 级日志） |
| `COURSE_INTERNAL_JOB_NOT_FOUND` | 回调中的 job_id 不存在 |
| `COURSE_INTERNAL_PAYLOAD_INVALID` | 422 payload 解析失败 |

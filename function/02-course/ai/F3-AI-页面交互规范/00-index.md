## 二、全局约定

- **样式与组件**：所有 UI 严格遵循 [G2-视觉与交互风格](../../../../grules/G2-视觉与交互风格/)；本文件不重复描述按钮颜色、字号等基础视觉。
- **路由模式**：纯 CSR / SPA（[G1-05 §二](../../../../grules/G1-架构与技术规范/05-编码规范.md)），客户端只渲染单一 HTML 入口。
- **多语言**：所有 `_i18n` jsonb 5 key（`zh / vi / th / id / en`，[F1-12-Q10](../F1-AI-数据模型规范/12-待确认问题清单.md) 封板）。学员端按 `course_user_settings.ui_lang` 取对应 key，缺失回退 `en` → `zh`；学习目标本体的中文原文额外固定显示（`title_zh / sentence_zh`）。
- **学员当前赛道**：从 `course_user_settings.current_track` 读取；切换 Tab 同步写回（[F2-Q13](../F2-AI-接口规范/12-待确认问题清单.md)）。share Stage 0 完成前所有主题赛道入口锁定（[temp/01 §1.6](../../../temp/01-课程目录骨架.md)）。
- **数量基线**（[temp/01 §1.7](../../../temp/01-课程目录骨架.md)）：5 赛道、25 stage、148 chapter、888 lesson、~12 000 KP；每 stage **固定 6 章**（share Stage 0 例外为 4 章）、每章 **固定 6 节**、每节 **固定 12 KP**、节末小测 **固定 5 题**。所有 UI 文案 / 进度计算 / 占位骨架均按此基线渲染。
- **节末小测可跳过**（[F1-12-Q4](../F1-AI-数据模型规范/12-待确认问题清单.md)）：默认 `is_quiz_required=false`；学员可跳过，跳过后该节 `progress.status` 保持 `in_progress`，仍可解锁下一节。
- **媒资缓存**（[F1-12-Q13 / F2-12-Q11](../F1-AI-数据模型规范/12-待确认问题清单.md)）：仅学员端点击 KP 听音时触发；管理端预览不触发缓存。
- **C 端读 / 写实现**（[F2-12-Q1](../F2-AI-接口规范/12-待确认问题清单.md)）：本规范统一以 RESTful 路径表达；学员侧读走 PostgREST + RLS，写走 Hono RPC，前端不感知。
- **表格 / 卡片骨架屏**：列表加载 < 200ms 不显示 skeleton，超过后显示 G2 标准 skeleton。
- **Toast**：成功 2s、失败 4s，右上角；`request_id` 在失败 toast 折叠面板中可复制。
- **必填星标**：所有"保存时校验"的字段在 label 前加红色 `*`。
- **未保存提示**：所有有表单的页面 / Drawer 通过 `route.beforeEach` + `beforeunload` 双拦截 → D-2。
- **错误码**：5 位数字 + `COURSE_*` 业务命名（[F2 11-错误码登记.md](../F2-AI-接口规范/11-错误码登记.md)）。
- **限流提示**（[F2-12-Q11](../F2-AI-接口规范/12-待确认问题清单.md)）：学员端 429 → 顶部提示条；管理端业务无限流。
- **暗黑模式**：[G2 §六](../../../../grules/G2-视觉与交互风格/06-响应式与暗黑模式.md) 自动跟随系统；学员端考试页强制 light 模式（避免阅读疲劳与公平性）。
- **响应式**：学员端 mobile-first（≥ 360px）；管理端桌面优先（≥ 1280px），≤ 1024px 退化为可滚动表格。
- **分页**：管理端列表统一 `page` / `page_size`（默认 20），保留筛选条件到 URL query。
- **乐观更新**：勾选已学 KP / 切换收藏走乐观更新；失败回滚 + Toast。
- **冲突**：管理端同条记录被并发编辑时，最后保存为准 + Toast「该内容在你编辑期间被 {display_name} 修改过」（[F2 §并发](../F2-AI-接口规范/10-并发与冲突处理.md)）。

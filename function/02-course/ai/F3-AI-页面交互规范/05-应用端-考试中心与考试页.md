## P-C-7 · 考试进行页

**路由**：`/learn/exams/:attempt_id`
**权限**：登录学员（仅 attempt 归属本人）
**模式**：跟随系统主题（[F3-Q9 封板](./13-待确认问题清单.md)）；阻止后退 + 关闭提示。

### 布局（桌面 / 移动统一，不降级 — [F3-Q10 封板](./13-待确认问题清单.md)）

```
┌──────────────────────────────────────────────────────────┐
│  HSK 4 模考 A 卷               ⏱ 01:23:45  [交卷]         │
├──────────────────────────────────────────────────────────┤
│  题号 12 / 100   ━━━━░░░░░░░░░░░░  12%                    │
├──────────────────────────────────────────────────────────┤
│   听力第 12 题                                            │
│   [ 🔊 仅可播 1 次（HSK 真实规则） ]                        │
│   ○ A …  ○ B …  ○ C …  ○ D …                              │
│                                                           │
│   [ ‹ 上一题 ]                       [ 下一题 › ]          │
├──────────────────────────────────────────────────────────┤
│  题号导航（桌面底部横排 / 移动点「12/100」弹 Drawer）：    │
│  1✓ 2✓ 3 4✓ 5 ... 12🔵 ... 100                          │
└─────────────────────────────────────────────────────────┘
```

> **移动端**（§§Q10 封板）：顶部倍计时常驻；题号区折叠为「12/100 ▲」状态条，点击从底部升起全屏题号 Drawer（类 D-5，10×10 题号网格，颜色表示已答 / 未答 / 当前）；主体选项块全宽 100% / 44px 高。

### 关键交互

| # | 操作 | 实现 |
|---|------|------|
| C7-01 | 进入 | 调 [C15 GET /api/v1/course/exam-attempts/:id](../F2-AI-接口规范/03-应用端-考试与举报.md) 取卷面 + 已答 + 剩余时间；恢复定时器 |
| C7-02 | 选答案 | 本地暂存；defer 1s 调 [C16 PATCH /api/v1/course/exam-attempts/:id/answers](../F2-AI-接口规范/03-应用端-考试与举报.md) 自动保存（按 question_id 增量）；状态指示「☁ 已保存」/「⏳ 保存中」 |
| C7-03 | 点击「下一题」 | 立即触发保存 + 跳下一题 |
| C7-04 | 点击「上一题」 | 可改答（HSK 模考类） |
| C7-05 | 题号导航点击 | 跳到任意题；保留答案 |
| C7-06 | 倒计时 = 0 | 自动调 [C17 POST /api/v1/course/exam-attempts/:id/submit](../F2-AI-接口规范/03-应用端-考试与举报.md)；显示结果页 |
| C7-07 | 点击「交卷」 | 弹 D-12 二次确认；显示已答 / 未答统计；确认后调 C17 |
| C7-08 | 听力题 🔊 | 仅 HSK 模考限播 1 次（按题目 `payload.audio_play_limit`）；其它考试不限 |
| C7-09 | 中途离开 | 弹 D-1：「考试将继续计时，未答题视为弃权。」选「继续考试」回当前题；选「确认离开」仅本地退出 |
| C7-10 | 网络断开 | 顶部条；本地累积修改；恢复网络自动 flush；持续断 60s → 顶部红条「连接异常，无法保存」 |
| C7-11 | 浏览器刷新 / 关闭 | `beforeunload` 拦截 → 提示后允许退出；恢复进入 P-C-7 自动续 |
| C7-12 | 题目内点举报 | D-14 → C18 |

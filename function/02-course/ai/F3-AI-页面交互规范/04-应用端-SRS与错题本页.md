# P-C-4 / P-C-5 应用端 · SRS 与错题本页

> 两个独立路由共用本文件：SRS 复习页（P-C-4）+ 错题本只读页（P-C-5）。

---

## P-C-4 · SRS 复习页

**路由**：`/learn/srs`
**权限**：登录学员
**入口**：P-C-1 浮动工具栏「📚 SRS 今日复习」、节末小测结果页「复习错题」按钮

### 关键约定

- 队列长度：今日到期题 ≤ 50（[F2-Q3 封板](../F2-AI-接口规范/12-待确认问题清单.md)），优先 box=1（错题）→ box=2 → … → box=5。
- 单日上限即 50；超过部分顺延到次日。
- 答题逻辑与节末小测完全相同（C9 即时判分），但**没有"提交汇总"步骤**——逐题完成。
- 答完最后一题显示总结卡。

### 布局

```
┌──────────────────────────────────────────────────────────┐
│  [← 返回]   今日复习                  3 / 32              │
├──────────────────────────────────────────────────────────┤
│  ━━━━━━━━░░░░░░░░░░░░░░░░░░░░░░░░░░░  9%                  │
├──────────────────────────────────────────────────────────┤
│   听音选词                                                │
│   [ 🔊 再听一次 ]                                          │
│   ○ A …  ○ B …  ○ C …  ○ D …                              │
│                                                           │
│   ┌──────────────────────────────────────┐               │
│   │ ❌ 错了。正确：早上                    │               │
│   │ 来自：1.3.4 早上好                     │               │
│   │ 已升盒：box 1（明天再次复习）           │               │
│   └──────────────────────────────────────┘               │
│                                                           │
│         [  ←  ]   [  下一题  →  ]                          │
└──────────────────────────────────────────────────────────┘
```

完成全部题目后：

```
┌──────────────────────────────────────────────────────────┐
│         🎯 今日复习完成 32 题                              │
│         正确率 78%                                        │
│         ⬆ 升盒 22 题 / ⬇ 降盒 10 题                      │
│         明日预计：18 题待复习                              │
│         [ 进入错题本 ]   [ 返回学习地图 ]                  │
└──────────────────────────────────────────────────────────┘
```

### 元素规格

| 元素 | 组件 | 说明 |
|------|------|------|
| 顶部进度 | `<span>` + `Progress` | `当前 / 总数` |
| 题目区 | `QuestionRenderer` | 复用 P-C-3 的 12 题型组件 |
| 反馈卡 | `FeedbackCard` | 含「来自：{lesson_name}」+ 「升/降盒」语义 |
| 总结卡 | `SummaryCard` | 完成后整页替代 |

### 数据来源

- 进入：调 [C11 GET /api/v1/course/me/srs/queue](../F2-AI-接口规范/02-应用端-答题与SRS.md) 取队列（最多 50 题）。
- 单题：调 [C9 POST /api/v1/course/answers](../F2-AI-接口规范/02-应用端-答题与SRS.md)（带 `context_type='srs'`），后端写答题流水 + 调整 SRS box。
- 总结：本地汇总 C9 各题响应；不再发额外接口。

### 交互行为

| # | 用户操作 | 技术实现 |
|---|---------|---------|
| C4-01 | 进入页面 | 调 C11；空队列 → 显示「今日没有复习任务，明天再来 🌱」+ [返回学习地图] |
| C4-02 | 答题 | 与 C3-02/03 一致；上一题不可回看（防作弊） |
| C4-03 | 关闭页面 | 已答题已写库，无草稿；可放心离开 |
| C4-04 | 中途退出 | 顶部「← 返回」直接跳；本地总结仅当前会话有效 |
| C4-05 | 完成全部 | 显示总结卡；「进入错题本」跳 P-C-5 |
| C4-06 | 网络断开 | 顶部条；C9 失败标 pending；下次答其它题时一起重试 |

## P-C-5 · 错题本页

**路由**：`/learn/wrong`
**权限**：登录学员
**入口**：P-C-1 浮动工具栏「❌ 错题本」、SRS 总结卡「进入错题本」

### 关键约定

- **只读列表**（[F2-Q8 封板](../F2-AI-接口规范/12-待确认问题清单.md)），本期不提供「一键重练 N 题」接口；学员需通过 SRS 队列自然消化（box=1 优先级最高）。
- 列表展示最近 90 天的错题；超期数据按月归档（不显示）。

### 布局

```
┌──────────────────────────────────────────────────────────┐
│  [← 返回]   错题本                          [筛选 ▼]     │
├──────────────────────────────────────────────────────────┤
│  最近答错 5 题（共 23 题）                                 │
│                                                           │
│  ┌──────────────────────────────────────────────────┐   │
│  │ ❌ 听音选词 · 早上                                 │   │
│  │   你的答案：晚上                                   │   │
│  │   来自：1.3.4 早上好                               │   │
│  │   答错时间：3 小时前                                │   │
│  │   下次复习：明天 09:00（box 1）                     │   │
│  │   [ 查看 KP ] [ 立即去 SRS 复习 ]                  │   │
│  └──────────────────────────────────────────────────┘   │
│  ...                                                      │
│  [ 加载更多 ]                                             │
└──────────────────────────────────────────────────────────┘
```

### 元素规格

| 元素 | 组件 | 说明 |
|------|------|------|
| 筛选 | `Popover` | 按 q_type / 来源 lesson / box 多选 |
| 错题卡 | `WrongCard` | 显示题型 + 中文 + 你的答案 + 正确答案 + 来源 lesson + 下次复习时间 |
| 查看 KP | `Link` | 跳 P-C-2 对应 KP |
| 立即去 SRS 复习 | `Link` | 跳 P-C-4（自动取 box=1 优先） |
| 加载更多 | `Button` | 滚动到底自动触发；或手动点击 |

### 数据来源

- 调 [C12 GET /api/v1/course/me/wrong-questions](../F2-AI-接口规范/02-应用端-答题与SRS.md)：分页 + 按 `last_wrong_at desc`。

### 交互行为

| # | 用户操作 | 技术实现 |
|---|---------|---------|
| C5-01 | 进入页面 | 调 C12 page=1；骨架屏 |
| C5-02 | 应用筛选 | 重新调 C12（带 query）；URL 同步 |
| C5-03 | 点击「查看 KP」 | 路由跳 P-C-2 + `?focus_kp_id=...`，P-C-2 自动定位到该 KP |
| C5-04 | 点击「立即去 SRS 复习」 | 路由跳 P-C-4；C11 默认按 box 优先 |
| C5-05 | 滚动到底 | 触发下一页；空响应隐藏「加载更多」 |
| C5-06 | 错题已 box ≥ 5（已掌握） | 卡片右上角显示「✓ 已掌握」徽标，不再出现在 SRS 队列 |

### 条件显示

| 条件 | 显示 |
|------|------|
| 列表为空 | 「🎉 没有错题，继续保持！」 |
| 网络断开 | 顶部离线条；缓存数据可继续浏览 |
| 加载失败 | 整页错误 + [重试] |

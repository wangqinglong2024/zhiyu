### 交互

| # | 操作 | 实现 |
|---|------|------|
| A4-01 | 进入 | 调 A21；URL query 反映筛选 / 搜索 / 分页 |
| A4-02 | 搜索 🔍 | 5 语言全字段 ILIKE（[F2-Q7 封板](../F2-AI-接口规范/12-待确认问题清单.md)）；不支持拼音 |
| A4-03 | 「+ 新建」 | 弹 D-3 KP 编辑 Drawer（空表单） |
| A4-04 | 「编辑」 | 弹 D-3（带数据） |
| A4-05 | 「删除」 | D-1；[A22 DELETE /kps/:id](../F2-AI-接口规范/05-管理端-KP与题目CRUD.md)；如已被某节绑定 → 422 + Toast「请先在节中解绑」 |
| A4-06 | 「重生题目」（行操作） | 弹 D-7 创建 AI Job（`job_type='gen_questions_for_kp'`），自动填 kp_id |
| A4-07 | 批量发布 | 勾选行 → [A23 POST /kps/batch-publish](../F2-AI-接口规范/05-管理端-KP与题目CRUD.md) |
| A4-08 | 「导入」 | D-15 CSV/JSON 上传；预览 + 校验后批量插入 |

| `examples[].sentence_zh` | 1–80 字 |
| `examples[].translation_i18n` | 5 key；zh / en 必填 |

| 操作 | 实现 |

### D-4 · 题目编辑 Drawer（720px）

```
┌─ 题目编辑 ─────────────────[× 关闭][保存草稿][提审][发布]┐
│ Q code  q_ec_00012345           关联 KP  [kp_ec_000123 早上]│
│ 题型 (q_type) [listen_pick ▼]    难度 [1 ▼]                  │
│ ─────────────────────────────────────────────                │
│ 题干 (stem_i18n)                                             │
│   zh: 听音选词 · vi: ... · th: ... · id: ... · en: Pick the word│
│                                                               │
│ Payload（按题型差异）                                         │
│  audio_text:    早上                                          │
│  audio_voice:   xiaoxiao_zh                                   │
│  options[]:     早上 / 晚上 / 早安 / 晚安                      │
│  correct_index: 0                                             │
│ ─────────────────────────────────────────────                │
│ 解释 (explanation_i18n)                                      │
│   zh: ...· en: ...                                            │
│ ─────────────────────────────────────────────                │
│ 媒资    🔊 audio_url 自动生成（hash: a3f7...）                │
│ 历史版本（v1, v2 ...）  [查看 diff]                            │
│ 最近 7 天准确率  78% (1 234 次作答)                           │
└────────────────────────────────────────────────────────────────┘
```

| 字段 | 校验 |
|------|------|
| `stem_i18n` | 5 key；zh / en 必填 |
| `payload` | 按 `q_type` 走 [F1-09 §payload schema](../F1-AI-数据模型规范/09-校验规则汇总.md) Zod |
| `payload_hash` | **后端自动生成**（[F1-Q9](../F1-AI-数据模型规范/12-待确认问题清单.md)）；前端不展示，但保存时若命中 dedup（同 KP + 同 q_type + 同 hash）→ 报 `40402 COURSE_DUPLICATE_PAYLOAD_HASH` Toast「已存在相同题目（{q_code}）」 |
| `correct_answer` | 必填，与 `payload` 结构一致 |

| 按钮 | 实现 |
|------|------|
| 保存草稿 | 新建 [A30 POST /questions](../F2-AI-接口规范/05-管理端-KP与题目CRUD.md) / 编辑 [A31 PATCH /questions/:id](../F2-AI-接口规范/05-管理端-KP与题目CRUD.md)；status=draft |
| 提审 | [A32 POST /questions/:id/submit-for-review](../F2-AI-接口规范/05-管理端-KP与题目CRUD.md)；status=reviewing |
| 发布 | reviewer+ 角色才显示；[A33 POST /questions/:id/publish](../F2-AI-接口规范/05-管理端-KP与题目CRUD.md)；status=approved |

### 条件显示

| 条件 | 行为 |
|------|------|
| 题目已发布 | 编辑保存 → 自动提交审核（`status: approved → reviewing`），需要 reviewer+ 重新批准 |
| 题目报错次数 ≥ 3 | 列表行红色高亮 + 顶部条「该题被举报较多，建议审核」 |
| `audio_url = null`（听力题） | Drawer 中按钮「生成音频」→ 调 [A34 POST /questions/:id/regen-media](../F2-AI-接口规范/05-管理端-KP与题目CRUD.md) |

## 性能 / 可访问性

- 列表虚拟滚动（rc-table + react-window）；万行不卡。
- Drawer 表单内字段顺序锁定（避免 Tab 跳跃）；i18n 5 字段水平排列。
- KP/Q code 列展示可一键复制。

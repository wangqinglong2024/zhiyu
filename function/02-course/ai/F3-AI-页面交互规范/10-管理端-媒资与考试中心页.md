## P-A-10 · 媒资库页

**路由**：`/admin/course/media`
**权限**：editor+

### 关键约定

- 媒资缓存只在**学员端**触发（[F2-Q11 封板](../F2-AI-接口规范/12-待确认问题清单.md)）；管理端只能查看 / 软删 / 替换，不能"主动生成"。
- 删除走"DB 软删 + 7 天 cron 真删 CDN 文件"（[F2-Q9 封板](../F2-AI-接口规范/12-待确认问题清单.md)）。
- voice_profile 是抽象标识，由后端 `MediaProvider` adapter 映射供应商（[F1-Q13 封板](../F1-AI-数据模型规范/12-待确认问题清单.md)）。

### 布局

```
┌──────────────────────────────────────────────────────────┐
│  媒资库      [类型 ▼][来源 ▼][命中 ▼][搜索 🔍]   [清理]   │
├──────────────────────────────────────────────────────────┤
│ ☐ │ kind │ hash       │ 来源文本 / 关联     │ voice / 尺寸 │ 命中次数 │ 大小 │ 操作 │
│───┼──────┼────────────┼────────────────────┼─────────────┼─────────┼─────┼─────│
│ ☐ │ tts  │ a3f7bc...  │ "早上" / kp_ec_000123│ xiaoxiao_zh │ 1 234   │ 18 KB│ ⋯  │
│ ☐ │ tts  │ b91c2d...  │ "你好" / kp_share_001│ yunyang_zh │ 5 678   │ 22 KB│ ⋯  │
│ ☐ │ image│ c33ee1...  │ "天安门" / kp_share_…│ 1024x768   │ 89      │ 320KB│ ⋯  │
│ ...                                                     │
└──────────────────────────────────────────────────────────┘
```

### 数据来源

- [A56 GET /admin/v1/course/media](../F2-AI-接口规范/08-管理端-媒资与考试中心.md)：分页 + 筛选 + 命中数（来自 `course_media_assets.hit_count`）。

### 交互

| # | 操作 | 实现 |
|---|------|------|
| A10-01 | 进入 | 调 A56；默认按命中倒序 |
| A10-02 | 点击行 | 弹 D-10 预览 Modal（音频可播 / 图片放大） |
| A10-03 | 「替换」 | 弹 Modal 上传新文件 → [A57 POST /media/:id/replace](../F2-AI-接口规范/08-管理端-媒资与考试中心.md)；保留 hash（指向新 url） |
| A10-04 | 「删除」 | D-1 二次确认；[A58 DELETE /media/:id](../F2-AI-接口规范/08-管理端-媒资与考试中心.md)；DB 软删 + 入 `course_media_assets_pending_purge` 队列；7 天后 cron 真删 CDN |
| A10-05 | 「清理」按钮 | 顶部入口；查看待清理队列（7 天窗口）；可手动撤销 |
| A10-06 | 批量删除 | [A59 POST /media/batch-delete](../F2-AI-接口规范/08-管理端-媒资与考试中心.md) |

### 条件显示

| 条件 | 行为 |
|------|------|
| 媒资 `hit_count = 0` 超 30 天 | 标灰 + Tooltip「无人使用，建议清理」 |
| 已入待清理队列 | 状态标「⏳ 7 天后清理」+ 可撤销 |
| 关联 KP / Q 已删除 | 标「🔗 孤立」+ 提示可清理 |

## P-A-11 · 考试中心管理页

**路由**：`/admin/course/exams`
**权限**：editor+

### 关键约定

- 考试模板只承载 `chapter_test / stage_exam / hsk_mock` 三类（[F2-Q2](../F2-AI-接口规范/12-待确认问题清单.md)）；节末小测**不在此**。
- 模板核心是 `blueprint`（jsonb）：定义按 KP 范围 + q_type 配比抽题。

### 布局

```
┌──────────────────────────────────────────────────────────┐
│  考试中心   [类型 ▼][主题 ▼][状态 ▼]      [+ 新建模板]   │
├──────────────────────────────────────────────────────────┤
│ ☐ │ 模板名             │ 类型       │ 题数 │ 时长 │ 通过线 │ 状态  │ 操作 │
│───┼───────────────────┼──────────┼─────┼─────┼───────┼──────┼─────│
│ ☐ │ EC Stage 1 阶段考  │ stage_exam│ 50   │ 45m │ 70    │ 启用 │ ⋯  │
│ ☐ │ HSK 4 模考 A 卷    │ hsk_mock  │ 100  │ 120m│ 60    │ 启用 │ ⋯  │
│ ☐ │ EC Ch1.1 章测试   │ chapter_test│ 30  │ 25m │ 60    │ 草稿 │ ⋯  │
│ ...                                                     │
└──────────────────────────────────────────────────────────┘
```

### 数据来源

- [A60 GET /admin/v1/course/exams](../F2-AI-接口规范/08-管理端-媒资与考试中心.md)：分页 + 各模板使用次数（attempt count）。

### 交互

| # | 操作 | 实现 |
|---|------|------|
| A11-01 | 进入 | 调 A60 |
| A11-02 | 「+ 新建模板」 | 弹 D-11 |
| A11-03 | 「编辑」 | D-11（带数据） |
| A11-04 | 「启用 / 归档」 | [A61 POST /exams/:id/activate](../F2-AI-接口规范/08-管理端-媒资与考试中心.md) / `archive` |
| A11-05 | 「删除」 | D-1；若已被 attempt 引用 → 拒绝 |

### D-11 · 考试 blueprint 编辑 Drawer（720px）

```
┌─ 考试模板编辑 ────────────────────[× 关闭][保存][试抽]┐
│ 名称 (title_i18n)                                       │
│   zh: EC Stage 1 阶段考  *                               │
│   en: EC Stage 1 Final Exam  *                            │
│   vi/th/id: ...                                          │
│ ──────────────────────────                              │
│ 类型     [stage_exam ▼]    主题  [ec ▼]                  │
│ 总题数   50    时长(分)  45    通过线  70                │
│ 每题播听 [允许多次 ▼]                                     │
│ ──────────────────────────                              │
│ Blueprint（按 KP 范围 + 题型配比抽题）                   │
│ ┌────────────────────────────────────────────────┐     │
│ │ KP 范围（多选）                                  │     │
│ │   ☑ Stage 1 全部 KP                              │     │
│ │   ☐ 仅 Chapter 1                                 │     │
│ │ 题型配比                                         │     │
│ │   listen_pick     20% (10 题)                    │     │
│ │   mcq_meaning     30% (15 题)                    │     │
│ │   mcq_zh          20% (10 题)                    │     │
│ │   sort_words      20% (10 题)                    │     │
│ │   tone_pick       10% (5 题)                     │     │
│ │ 难度分布                                         │     │
│ │   1: 30% / 2: 40% / 3: 20% / 4: 10%              │     │
│ └────────────────────────────────────────────────┘     │
│ ──────────────────────────                              │
│ 试抽预览（最近一次）   [立即试抽]                         │
│   ✅ 成功抽出 50 题（耗时 1.2s）                          │
│   ⚠ 警告：listen_pick 只抽到 8 题（KP 候选不足）          │
└──────────────────────────────────────────────────────────┘
```

| 字段 | 校验 |
|------|------|
| `title_i18n` | 5 key；zh / en 必填 |
| `total_questions` | 与配比之和必须一致（前端实时校验，否则保存按钮 disabled） |
| `duration_min` | ≥ 1，≤ 240 |
| `pass_score` | 0–100 |
| `blueprint.kp_scope` | 至少 1 项 |
| `blueprint.q_type_ratio` | 之和 = 100% |
| `blueprint.difficulty_dist` | 之和 = 100%（可空 → 不限制） |

| 操作 | 实现 |
|------|------|
| 保存 | 新建 [A62 POST /exams](../F2-AI-接口规范/08-管理端-媒资与考试中心.md) / 编辑 [A63 PATCH /exams/:id](../F2-AI-接口规范/08-管理端-媒资与考试中心.md) |
| 试抽 | [A64 POST /exams/:id/dry-pick](../F2-AI-接口规范/08-管理端-媒资与考试中心.md)；服务端按 blueprint 抽 1 次返回题目 ID 列表 + 警告（不入库 / 不影响学员） |
| 启用前置校验 | 试抽必须成功且无警告才能启用 |

### 条件显示

| 条件 | 行为 |
|------|------|
| 模板已被学员产生 attempt | 「删除」disabled；编辑保存后顶部条「修改不影响已有 attempt」 |
| 类型 `chapter_test` | 必须额外指定 `chapter_id` 字段 |
| 类型 `hsk_mock` | 听力题强制 `audio_play_limit=1`；其它题型自动添加 |

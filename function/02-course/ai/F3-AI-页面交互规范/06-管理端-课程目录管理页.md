```
└──────────────────────────────────────────────────────────┘
```

### 元素规格

| 元素 | 组件 | 说明 |
|------|------|------|
| Stage 折叠 | `Collapse` | 默认全部折叠；点击展开后显示该 Stage 的章列表 |
| Stage 操作 | 「编辑」「调序」「删」 | 删除前需 D-1 确认 + 仅当所有章为空时允许 |
| 章列表表格 | `Table` | 列：序号 / 章名称（含 i18n.zh）/ 节数（已发布/总）/ KP 数 / 状态 / 操作 |
| 行操作 | `Dropdown` | 编辑章基本信息 / 调序节 / 进入节列表 / 删除 |
| 状态徽标 | `Badge` | 草稿 / 审核中 / 已发布 / 已下架 |
| 批量操作 | `Dropdown` | 批量发布 / 批量下架 / 批量删除（仅勾选行）|

### 数据来源

- [A2 GET /admin/v1/course/tracks/:track_code/stages](../F2-AI-接口规范/04-管理端-课程目录.md)：包含每个 stage 的 chapter 列表 + 节数 + KP 数。

### 交互

| # | 操作 | 实现 |
|---|------|------|
| A2-01 | 进入 | 调 A2；骨架表格 |
| A2-02 | 展开 Stage | 本地 toggle（数据已包含） |
| A2-03 | 「+ 新建阶段」 | 弹 Drawer 编辑 stage 基本信息（name_i18n 5 key、stage_no、hsk_level、sort_order）；保存调 [A3 POST /stages](../F2-AI-接口规范/04-管理端-课程目录.md)；i18n 字段 `zh / en` 必填，其它语言可空字符串但 key 必须存在 |

| A2-06 | 调序章 | 弹 D-6 调序 Drawer（拖拽 6 章）；保存调 [A6 POST /chapters/batch-reorder](../F2-AI-接口规范/04-管理端-课程目录.md) |
| A2-07 | 新建章 | Stage 折叠面板内「+ 新建章」按钮；Drawer + [A7 POST /chapters](../F2-AI-接口规范/04-管理端-课程目录.md) |

| A2-10 | 点击章 → 进入节列表 | 跳 `/admin/course/chapters/{chapter_id}/lessons`（同样 P-A-2 视图，深一层显示 6 节列表，复用本页结构） |
| A2-11 | 批量发布 | 选中行后 [A10 POST /chapters/batch-publish](../F2-AI-接口规范/04-管理端-课程目录.md)；进度 Toast |

### 条件显示

| 条件 | 说明 |
|------|------|
| `stage.is_published = false` | 不影响学员（草稿仅管理端可见） |
| `chapter.lesson_count < 6` | 章名称右侧黄点 + Tooltip「未达 6 节基线」 |
| share Stage 0 | 唯一允许 4 章；不显示「未达 6 节」警告 |
| 学员有进度的章 | 删除按钮 disabled + Tooltip「有学员进度，不可删除」 |

## 全局：i18n 编辑器组件

| 字段 | 校验 |
|------|------|
| zh | 必填（学习目标显示用 + UI 主语言） |
| en | 必填（回退） |
| vi / th / id | 可空字符串，但 key 必须存在 |

> i18n 编辑器统一封装在 `packages/ui-kit/i18n-editor`，所有管理端表单引用。

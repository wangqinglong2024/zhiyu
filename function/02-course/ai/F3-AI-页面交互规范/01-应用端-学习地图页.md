## 元素规格

| 元素 | 组件类型 | 规格 | 说明 |
|------|---------|------|------|
| 主题 Tab | `Tabs` | 5 个：ec / fc / hk / dl / share；下划线高亮 | 切换写回 `course_user_settings.current_track` |
| Share Stage 0 标识 | `Badge` | 学员未完成 share 时，主题主题 Tab 显示锁 | 完成 share Stage 0 解锁全部主题主题 |
| 阶段标题 | `<h2>` | `stage.name_i18n[ui_lang]` + 进度条 | 进度 = passed_lesson_count / total_lesson_count |
| 章节点 | `ChapterCard` | 毛玻璃卡片（`zy-china-card` 同样式）；包含：emoji + 编号 + 名称 + 节进度 N/6 + 状态徽标（✅ 已通过 / 🟡 N/6 / 🔒 待解锁） | 与 [system ChinaCategoryCardsPage](../../../../system/apps/web-app/src/pages/china/ChinaCategoryCardsPage.tsx) 一致 |
| 布局 | `Waterfall` | 4 列 → 3 列 → 2 列 → 1 列（1100 / 768 / 640 / 默认 断点）；`zy-grid-12` 同源 | 锁态卡片右上角 🔒 |
| 章展开面板 | `Collapse` | 默认展开「当前进入的章」；其余折叠 | 同时只展开 1 个 |
| 节列表项 | `LessonItem` | 圆点状态 + 编号 + `name_i18n[ui_lang]` + 状态文案（已通过 / 学习中 N/12 KP / 待解锁） | 点击跳 P-C-2 |
| 桌面浮动工具栏 | `FloatingToolbar` | 4 个入口：SRS / 错题本 / 考试 / 统计；徽标显示数量 | **仅桌面**；F3-Q6 封板 |
| 移动端底部 Tab | `BottomTabs` | 2 项：📚 学习 / 👤 我的 | 考试 / SRS / 错题本 均在「我的」内 |
| 顶部 Header | `AppHeader` | G2 标准；含语言切换 + 头像菜单 | 全局组件 |

## 数据来源

- 进入页面：调 [C1 GET /api/v1/course/me](../F2-AI-接口规范/01-应用端-课程目录浏览.md)（取 `current_track / shared_stage_done / unlocked_stages`）。
- 主题切换：调 [C2 GET /api/v1/course/tracks/:track_code/map](../F2-AI-接口规范/01-应用端-课程目录浏览.md)（一次性返回该主题全部 stage / chapter / lesson + 学员进度），SWR 缓存 60s。
- 浮动工具栏徽标：调 [C5 GET /api/v1/course/me/dashboard](../F2-AI-接口规范/01-应用端-课程目录浏览.md)（SRS 数 / 错题数 / 待考次数）。

## 交互行为

| # | 用户操作 | 技术实现 |
|---|---------|---------|
| C1-01 | 进入页面 | 并行 C1 + C5；C1 完成后用 `current_track` 调 C2；骨架屏直至 C2 返回 |
| C1-02 | 切换主题 Tab | 乐观更新本地 state；并行：① 调 [C7 PATCH /me/settings](../F2-AI-接口规范/02-应用端-答题与SRS.md) `{current_track}` ② 调 C2 取新地图；切换前若 share 未完成且目标是主题主题 → D-1 二次确认「需先完成 share Stage 0」 |
| C1-03 | 点击章节点 | 展开该章（折叠其它）；如该章 `state='locked'` → Toast「请先完成上一章 / 阶段考」 |
| C1-04 | 点击节列表项 | 路径跳 `/learn/lessons/{lesson_code}`；锁状态 → Toast 提示前置依赖 |
| C1-05 | 点击「SRS 今日复习」 | 路径跳 `/learn/srs` |
| C1-06 | 点击「错题本」 | 路径跳 `/learn/wrong` |
| C1-07 | 点击「考试中心」 | 路径跳 `/learn/exams` |
| C1-08 | 点击「我的统计」 | 路径跳 `/learn/me` |
| C1-09 | 切换 UI 语言 | 全局事件：① PATCH `/me/settings {ui_lang}` ② 重渲染所有 `_i18n[ui_lang]` 键，无需重新请求 |
| C1-10 | 退出登录 | 头像菜单 → 走 G3 标准登出；路由跳 `/login` |

## 条件显示规则

| 条件 | 显示 | 隐藏 |
|------|------|------|
| `shared_stage_done = false` 且目标主题 ≠ share | 主题主题 Tab 显示 🔒 + 点击 D-1 | — |
| `stage.unlocked_at != null` | 阶段卡片正常进度条 | — |
| `stage.unlocked_at = null` | 阶段卡片显示「🔒 完成 S{n-1} 阶段考解锁」 | — |
| `lesson.progress.status = 'passed'` | ✅ 已通过 | — |
| `lesson.progress.status = 'in_progress'` | 🟡 学习中（N/12 KP） | — |
| `lesson.progress.status = 'unlocked'` 且未学 | ⚪ 未开始 | — |
| `lesson.locked_reason != null` | 🔒 + 提示文案 | — |

## 空态 / 异常

| 场景 | 处理 |
|------|------|
| C2 失败 | 整页错误态 + [重试]；本地缓存若有则降级显示 + 顶部条「最新数据加载失败」 |
| 学员从未学过 | 默认进入 `share` 主题；显示引导浮层「先完成基础 Stage 0 再选主题方向」 |
| 学员已通过全部 stage | 顶部条「🎉 你已完成本主题，建议进入考试中心 HSK 模考」 |
| 网络断开 | 全局离线提示条；缓存数据可继续浏览，操作均禁用 |

## 性能 / 可访问性

- 学习地图首屏关键 LCP < 1.5s（C1 + C2 并行 + skeleton）。
- 章节点支持键盘左右切换 + Enter 展开（`role="button"` + `aria-expanded`）。
- 节列表项 `aria-label` 包含状态语义（"已通过 / 学习中 8/12 KP"）。
- 长地图（HSK 6 stage）虚拟滚动（`react-window`），按章为单位渲染。

## 元素规格

| 元素 | 组件类型 | 规格 | 说明 |
|------|---------|------|------|
| 顶部进度条 | `Progress` | 比例 = `learned_kp_count / total_kp_count`（固定 12） | KP 看完即 +1，不可回退 |
| KP 卡片 | `KPCard` | 居中；卡片内布局按 `kp_type` 7 类自适应 | 见下表 |
| 中文原文 | `<h1>` | `kp.title_zh`（不可变学习目标） | 高对比，显眼 |
| 拼音 | `<p class="pinyin">` | `kp.pinyin`（带声调符） | 上方或紧贴中文 |
| 翻译 | `<p class="translation">` | `kp.translation_i18n[ui_lang]`，回退 `en` → `zh` | — |
| 播放发音 | `Button(Icon)` | 触发媒资缓存 + 播放 | 见交互 C2-03 |
| 收藏 | `Button(Star)` | 切换 `course_user_kp_marks.starred` | 乐观更新 |
| 例句 | `ExampleBlock` | `kp.examples[]`（最多 2 条） | 同样可点击 🔊 |
| 上/下导航 | `Button` | 第 1 个隐藏「上一个」；最后 1 个变「完成本节」按钮 | 键盘 ← / → |
| 完成本节 | `Button(Primary)` | 12 KP 全部看完后弹 D-13（节末小测可跳过提示） | — |
| 设置 | `Popover` | 字号 / 自动播放发音 / 隐藏拼音 | 写入 `course_user_settings.lesson_prefs` |

### KP 卡片按类型差异（[F1-02 §kp_type 七类](../F1-AI-数据模型规范/02-表定义-知识点与题目.md)）

| kp_type | 卡片要点 |
|---------|---------|
| `char` | 单字 + 拼音 + 笔顺动画（可选）+ 翻译 + 1 例 |
| `word` | 词 + 拼音 + 翻译 + 2 例句 |
| `phrase` | 短语 + 拼音 + 翻译 + 2 例 |
| `sentence` | 句 + 拼音（可隐藏）+ 翻译 + 朗读全句 |
| `dialogue` | 对话双方 + 角色头像 + 整段朗读 |
| `pattern` | 句型 + 模板「A + 是 + B」+ 替换示例 ×3 |
| `culture` | 文化点：图片 + 文字说明 + 1 选用例 |

## 数据来源

- 进入页：调 [C3 GET /api/v1/course/lessons/:lesson_code](../F2-AI-接口规范/01-应用端-课程目录浏览.md)（节基础信息 + 12 KP 数组 + 学员断点 `last_kp_index`）。
- 进度自动定位到 `last_kp_index`；首次进入从 0 开始。
- 媒资点击播放：调 [C6 POST /api/v1/course/media/tts](../F2-AI-接口规范/02-应用端-答题与SRS.md)（请求 hash 命中即返 URL，未命中触发后台缓存 + 202，前端按 1s/2s/4s/8s 轮询 [C5 dashboard 已含 media 也可用 C6 复请](../F2-AI-接口规范/02-应用端-答题与SRS.md)）。

## 交互行为

| # | 用户操作 | 技术实现 |
|---|---------|---------|
| C2-01 | 进入节 | 调 C3；如 `lesson.locked` → 路由跳 P-C-1 + Toast；定位到 `last_kp_index` |
| C2-02 | 切换 KP（前进） | 本地 index++；同时 PATCH 进度（debounce 1s）写 `course_user_progress.last_kp_index` 与 `learned_kp_count`（取 max） |
| C2-03 | 点击 🔊 播放发音 | 调 C6 → 200 即 `<audio>` 播；202 → 启动轮询 1/2/4/8s；超 60s → Toast「发音生成失败，可稍后再试」；按钮在等待期 disabled + spinner |
| C2-04 | 点击收藏 | 乐观切换；POST `/api/v1/course/me/kp-marks {kp_id, starred}`；失败回滚 |
| C2-05 | 点击「完成本节，进入小测」 | 弹 D-13：「节末小测有 5 题（可选）。要现在测验吗？」 → 「测一下」跳 P-C-3 / 「先跳过」直接 PATCH 进度 `status='in_progress'`（不变 passed），跳回 P-C-1 |
| C2-06 | 自动播放发音（设置启用） | 切换到新 KP 时自动调 C6（仅命中缓存才播；未命中不阻塞） |
| C2-07 | 修改字号 / 隐藏拼音 | 写本地 + PATCH `/me/settings.lesson_prefs`；立即生效 |
| C2-08 | 应用 / 标签页失焦超过 5min | 暂停所有播放 + 暂存本地进度（不发请求） |
| C2-09 | 重新进入网络断开 | 顶部离线条；本地缓存 KP 仍可浏览；播放按钮 disabled |
| C2-10 | 举报 KP 内容（在 ⚙ 设置内） | 弹 D-14；提交调 [C18 POST /api/v1/course/reports](../F2-AI-接口规范/03-应用端-考试与举报.md) |

## 条件显示规则

| 条件 | 显示 | 隐藏 |
|------|------|------|
| `lesson.is_quiz_required = true` | 「完成本节，进入小测」按钮文案变「完成本节并测验（必做）」，D-13 不出现 | D-13 |
| 当前 KP `audio_url = null` | 🔊 按钮显示等待 spinner | — |
| 当前 KP `kp_type = 'culture'` 且无 example | 不展示例句区 | 例句区 |
| 学员设置「隐藏拼音」 | 拼音隐藏，长按中文短暂显示 | 拼音 |

## 空态 / 异常

| 场景 | 处理 |
|------|------|
| `lesson.kps.length < 12` | 仍可学，但顶部黄条提示「该节内容尚未补全（{n}/12 KP）」 |
| C3 失败 | 整页错误 + [重试] |
| C6 失败（5xx） | 单 KP Toast「发音暂不可用」+ 按钮不再 disabled |
| 进度 PATCH 失败 | 静默重试 3 次；最终失败仅本地保留 |
| 学员被强制下线（401） | 跳 `/login?redirect=...` |

## 性能 / 可访问性

- 进入页 LCP < 1.2s。
- KP 切换走纯 CSS 过渡（200ms）；不重新请求。
- `<audio>` 复用单实例，避免内存堆积。
- 键盘：← / → 切换 KP；Space 播放发音；Enter 收藏。
- 视障模式：屏幕阅读读「中文 + 拼音 + 翻译 + 第 N / 12」，跳过装饰图。

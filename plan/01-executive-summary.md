## 一、核心思考：为什么这件事值得做，又为什么难做

### 1.1 市场判断

面向东南亚的中文学习是真实刚需——华人基数大、商务往来频繁、旅游和务工需求强，赛道前景没有问题。

但冷静下来需要正视一个事实：**这个想法一点都不冷门。** 有做中文教学想法和相关资源的人比比皆是。按理说，如果门槛很低，市面上应该涌现大量同类精品 App。但现实是，海外优质、精细化、贴合东南亚用户习惯的中文学习 App 屈指可数。

从这一点判断：**这件事绝对没有表面看起来那么简单。** 想法人人都能想到，不代表人人都能做成、做好、长期运营活下去。

### 1.2 隐性门槛分析

| 门槛 | 说明 |
|------|------|
| **想法零门槛，落地全是高门槛** | 识字、拼音、日常对话只是表层功能，支撑产品活下去的是底层整套体系 |
| **资源 ≠ 产品化能力** | 线下教学、碎片化知识与系统化课程、标准化题库、轻量化软件体验完全是两回事 |
| **多区域本地化极难** | 东南亚各国语言、文化、习惯、审美、痛点完全不同，一套内容无法通用 |
| **海外合规壁垒** | 隐私法规、付费资质、内容审核、应用商店规则——国内逻辑完全不适用 |
| **内容需长期迭代** | 固定课程很快过时，持续产出是长期成本压力，多数团队做完 demo 就无力更新 |
| **运营和变现是最大卡点** | 没有本地化投放渠道、不懂消费习惯、跑不通盈利模式就没法长久存活 |
| **竞品少是因为淘汰率高** | 大量入局者在本地化、合规、内容、运营、变现环节陆续出局 |

### 1.3 客观结论

1. 中文出海学习是真实刚需，东南亚市场空间足够，大方向没问题
2. 项目门槛不在想法，而在落地、本地化、合规、内容、长期运营和商业化
3. 正是因为大部分人只会空想、无法落地打磨，才给了认真做精细化、差异化产品的团队突围机会
4. 理性看待项目，不盲目跟风、不轻视隐性壁垒，提前理清所有难点再稳步推进

## 二、品牌与产品定位

### 2.1 品牌名称

**知语 Zhiyu**（知 + 语 = 懂语言，知世界）

| 维度 | 说明 |
|------|------|
| **命名逻辑** | 知 = 知道/智慧；语 = 语言。"知语"即"通过语言认知世界"，也谐音"知遇"（知遇之恩） |
| **中国文化底蕴** | 知、语均为常见汉字，自带中文基因，学习者第一眼就能感受到中文氛围 |
| **国际友好** | Zhiyu 拼音发音简单，两个音节，英文用户也容易记忆和传播 |
| **语言中立** | 品牌不绑定"中文"二字，未来可扩展 Zhiyu English 等产品线 |
| **域名/商标** | zhiyu.app / zhiyu.com 可注册验证 |

### 2.2 品牌宣传语体系

| 场景 | 文案 |
|------|------|
| **品牌主 Slogan** | **知语，知世界。** / Learn a language, know the world. |
| **App 副标题** | 有趣的中文学习平台 / Fun Chinese Learning |
| **App Store 描述** | 知语 — 学中文，玩游戏，探索中国文化 |
| **社交传播短句** | "今天你知语了吗？" |

> **品牌哲学**：学一门语言，不只是学文字和语法，而是打开一扇理解世界的窗。"知语"的"知"既是认知，也是智慧——学语言的过程就是变得更聪明的过程。品牌不限于中文，未来可扩展任何语言。

### 2.3 产品定位

**知语 Zhiyu** 是面向东南亚市场的语言学习平台。首发产品为**中文学习**，以三大核心模块构建完整学习生态：

| 模块 | 定位 | 核心价值 |
|------|------|---------|
| 🏛️ 发现中国 | 登录即可浏览全部内容的文化窗口 | 激发兴趣，吸引注册，为课程引流 |
| 📚 系统课程 | 12 级 K-12+HSK 双轨课程体系 | 从零到精通系统掌握中文，核心变现 |
| 🎮 游戏模式 | 12 款可无限玩的 2D 学习游戏 | 巩固知识，上瘾留存，社交 PK |

### 2.4 一句话定位

> **知语 Zhiyu = 了解中国的窗口 + 学中文的课堂 + 练中文的游乐场**

### 2.5 三模块飞轮

```
文化内容激发兴趣 → 用户想学中文
       ↓
课程提供系统知识 → 用户掌握基础
       ↓
游戏巩固应用能力 → 用户获得成就感
       ↓
成就感驱动更深入的文化探索 → 飞轮加速
```

### 2.6 未来扩展蓝图

```
知语 Zhiyu（平台）
  ├── 中文课程（首发，当前全部精力）
  │     ├── 发现中国
  │     ├── 12级系统课程
  │     └── 12款中文游戏
  ├── 英语课程（远期，验证模式后复制）
  │     ├── Discover English-Speaking World
  │     ├── English Courses
  │     └── English Games
  └── 更多语言……
```

> 当前阶段：**100% 聚焦中文课程**，不分散精力。品牌和技术架构留好扩展空间即可。

## 三、多语言体系设计

这是产品的核心差异化能力之一，贯穿所有模块。

### 3.1 UI 语言

- 前期支持：**汉语、英语、越南语** 三种切换
- 后期扩展：印尼语、泰语、马来语
- 同一时间操作按钮的 UI **仅显示一种语言**

### 3.2 内容语言（学习语言 + 解释语言）

用户可配置两个维度：

| 配置项 | 选项 | 默认值 |
|--------|------|--------|
| 学习语言模式 | 拼音+中文 / 纯中文 | 拼音+中文 |
| 解释语言开关 | 开启/关闭 | 开启 |
| 解释语言 | 跟随系统 UI 语言 | 越南语 |

### 3.3 内容展示规则

最多情况下一段文字展示三行：

```
Zuótiān qīngchén, wǒ zuò zài yángtái shàng kàn rìchū.    ← 拼音（选择"拼音+中文"时显示）
昨天清晨，我坐在阳台上看日出。                                ← 中文（始终显示）
Sáng qua, tôi ngồi trên ban công ngắm bình minh.           ← 解释语言（开启时显示，跟随UI语言）
```

**特殊规则**：
- 若用户系统语言为中文 → 禁止开启解释语言（中文解释中文无意义）
- 若关闭解释语言 → 仅显示拼音+中文（或纯中文）
- 默认配置：拼音+中文 + 越南语解释（产品首发越南市场）

### 3.4 内容存储方案

所有教学内容最终存储于 **Supabase 数据库**（Dify 仅作为内容生产工具，生成后写入 Supabase）。格式示例：

```
Zuótiān qīngchén, wǒ zuò zài yángtái shàng kàn rìchū. Tiānbiān zhújiàn bèi rǎn chéng júhóngsè, fēicháng měilì.
昨天清晨，我坐在阳台上看日出。天边逐渐被染成橘红色，非常美丽。
---
Sáng qua, tôi ngồi trên ban công ngắm bình minh. Chân trời dần được nhuộm màu cam, rất đẹp.
```

根据用户选择的语言模式，前端按需加载和渲染对应的内容层。

## 四、三大模块概览

### 模块一：发现中国（Discover China）

- **分层开放**：未登录访客只能浏览前 3 个类目（中国历史、中国美食、名胜风光）；登录用户可浏览全部 12 类目全部内容
- 12 大类目：**中国历史、中国美食、名胜风光、传统节日、艺术非遗、音乐戏曲、文学经典、成语典故、哲学思想、当代中国、趣味汉字、中国神话传说**
- 图文 + 短视频(1-3分钟) + 音频，简洁易懂，搭配多语言对照
- 每周 3-5 篇更新，贴合节日热点
- 板块联动引导至系统课程
- 详见 [02-discover-china.md](02-discover-china.md)

### 模块二：系统课程（Courses）

- **分层开放**：未登录访客不可访问课程模块；登录用户（免费）可学习 L1-L3（含完整考核）；L4-L12 按级购买 $6/级（3 年有效）
- K-12 + HSK 3.0 双轨对标，4 大阶段：启蒙段(L1-3) → 基础段(L4-6) → 核心段(L7-9) → 高阶段(L10-12)
- 9 大知识模块贯穿 12 级，难度螺旋上升
- AI 辅助内容生产，持续更新迭代
- 考核体系：每课时小测验 + 每单元阶段测评 + 每级综合考核（学点考点，即时反馈）；考核融合课程知识与 HSK 考点，综合考核 85 分通过方可升级
- 详见 [03-course-system.md](03-course-system.md)

### 模块三：游戏模式（Games）

- **12 款独立 2D 游戏**，每款对应一个课程 Level，强化该级别知识点
- 每款游戏可**无限重复玩**（非闯关模式），每局随机出题，越玩越上瘾
- 全部支持**单人模式 + 1v1 PK + 多人对战**
- 游戏类型多样：切水果式、棋牌式、消消乐式、打字竞速等
- **访问规则**：必须登录才能进入游戏模块；登录后 12 款游戏全部解锁，无需课程进度门槛
- 详见 [04-game-design.md](04-game-design.md)

### 每日金句（Daily Quote）

- 首页每日更新一句中文金句（古诗词名句/成语/哲学名言）+ 拼音 + 英文翻译 + 解读
- 节日自动切换节日主题金句（春节/中秋/端午等）
- 一键生成精美分享图片（自带品牌水印 + 二维码）→ 免费获客
- Phase 1 全局统一 → Phase 2 按 HSK 等级分发 → Phase 3 个性化推荐
- 预期：DAU +15-20%，分享率 +30%
- 详见 [08-operations-marketing.md](08-operations-marketing.md) 第八节

## 五、目标市场与用户画像

### 5.1 市场策略

越南首发 → 印尼扩展 → 泰国/马来西亚 → 更广泛东南亚市场

**为什么首发越南**：华人华侨基数大、中越商贸往来密切、务工需求旺盛、互联网渗透率高、获客成本相对低。

### 5.2 目标用户

| 用户群体 | 年龄 | 核心需求 | 付费意愿 |
|---------|------|---------|---------|
| 工厂/商务青年 | 18-35岁 | 会中文加薪、商务沟通 | 高 |
| 大学生 | 18-24岁 | 中文考级（HSK）、求职竞争力 | 中 |
| 华人华侨后代 | 15-35岁 | 文化认同、与长辈沟通 | 中高 |
| 文化兴趣者 | 15-40岁 | 喜欢中国影视/美食/旅游 | 低→中 |
| 零基础外国人 | 18-40岁 | 从零学中文，系统性需求 | 中 |

### 5.3 入学分流

| 用户类型 | 入口 |
|---------|------|
| 零基础 | 从 Level 1 开始 |
| 有基础 | 通过**入学测试**（识字+词汇+阅读+HSK 模拟）直接分配到对应级别 |
| 母语儿童 | 按年龄自动映射（6 岁→L1，7 岁→L2，……18 岁→L12） |

### 核心差异化

1. **12 级双轨课程体系（K-12 + HSK 3.0）** — 既有母语级深度，又有国际标准通用性，竞品做不到
2. **文化+课程+游戏三合一闭环** — 完整学习生态
3. **多语言内容对照** — 拼音+中文+母语三行对照，降低学习门槛
4. **价格碾压** — 低于所有竞品，性价比拉满
5. **平台可扩展** — 品牌不绑定中文，未来可复制到其他语言

## 七、商业模式摘要

### 收入结构

课程按级购买（核心）+ 游戏皮肤/特效 + **知语币（虚拟币）推荐体系**驱动增长

### 课程定价

| 课程等级 | 定价 | 有效期 |
|---------|------|--------|
| **L1-L3** | **免费**（含完整考核） | 永久 |
| **L4-L12**（每级） | **$6 / 级** | 3 年 |
| **L4-L12 全部** | **$54**（9 级 × $6） | 3 年 |

> 定价远低于 Duolingo Super（$84-168/年）、HelloChinese Premium（$108/年），一次购买 3 年有效。详见 [06-business-model.md](06-business-model.md)

### 推广激励（知语币体系）

- 推荐用户付费 → 推荐人与被推荐人各获得付费金额 20% 的**知语币**（Zhiyu Coins）
- 知语币仅可在平台内使用（兑换课程、皮肤、道具），**不可提现、不可转让**
- 采用虚拟币而非现金返佣，规避越南跨境支付合规（FCT、SBV AML、FATF）与 App Store 政策风险
- 详见 [06-business-model.md](06-business-model.md) 知语币章节

## 八、风险控制

| 风险类型 | 具体风险 | 应对策略 |
|---------|---------|---------|
| 内容风险 | 文化/历史描述错误 | 邀请中文老师审核，参考义务教育课程标准 |
| 版权风险 | 统编教材版权 | 仅做知识点引用，不复制原文；古诗文公版可直接使用；其他内容提供同题材改写版 |
| 技术风险 | 卡顿、闪退、AI 评测不准确 | 定期维护优化，每周功能测试 |
| 变现风险 | 付费转化率低 | 优化转化路径，定期调整定价和优惠策略 |
| 用户流失 | 内容枯燥、游戏趣味性不足 | 持续更新内容，定期收集用户反馈 |
| 竞争风险 | 同类 App 竞争 | 打造差异化：12级双轨+文化+游戏闭环 |
| 合规风险 | 隐私法规、应用商店规则 | 提前研究目标市场法规，合规先行 |

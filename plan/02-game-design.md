# PlayLingo — 游戏化设计方案

> 把教学变成游戏，把故事变成过场动画，把练习变成战役
> 版本：v2.0 | 日期：2026-04-15

---

## 一、设计哲学

### 1.1 核心原则

1. **学习是游戏推进的燃料**：用户必须掌握中文知识才能推进游戏，但用户的主观体验是"我在玩一个策略游戏"
2. **过场动画代替课文**：传统语言 App 的"故事""对话""语法讲解"全部以游戏过场动画/剧情CG的形式呈现
3. **荣誉感驱动**：段位、军衔、领地版图、勋章体系，让用户有"我在不断变强"的进阶感
4. **即时策略节奏**：不是回合制答题，而是有时间压力、有策略选择、有即时反馈的游戏节奏
5. **高品质 2D 插画美术 + 精致 UI**：非像素风格，而是精致的 2D 插画/手绘风，UI/UX 必须精致，对标高品质独立游戏

### 1.2 美术风格定义

**主风格：精致 2D 插画风（High-Quality 2D Illustration）**

- **不是**像素风、不是卡通简笔画、不是教育 App 风格
- **是**：类似《原神》2D 宣传图、《Genshin》卡牌插画、《Hades》质感的精致手绘风
- 所有美术素材由 AI（Midjourney/Stable Diffusion/Flux）生成高品质原图，再经后期调整统一风格
- 角色立绘精致、场景背景层次丰富、UI 元素有质感（玻璃拟态/渐变/微动效）

```

美术生产流程：
  1. AI 生成高品质插画（Midjourney/Flux + 统一风格 LoRA）
  2. 后期处理：色调统一、裁切适配、添加游戏内 UI 元素
  3. 动画：Spine/Lottie 制作角色动画和 UI 微动效
  4. 场景：AI 生成背景 + 分层处理（远景/中景/近景）实现视差滚动
  5. UI：Figma 精致设计 → React 组件实现（含交互动效）
```

### 1.3 参考游戏（美术与玩法）

| 参考作品 | 借鉴元素 |
|---------|---------|
| **Kingdom Rush** (塔防) | 策略部署感、关卡地图推进、英雄系统 |
| **Clash Royale** (即时策略) | 实时对战节奏、卡牌收集、段位系统 |
| **文明 (Civilization)** | 领地扩张、科技树（=语言技能树）、文明发展感 |
| **Hades / Dead Cells** | 高品质 2D 美术参考、流畅动画 |
| **原神** (2D 宣传/卡面) | 人物插画品质标杆、UI 质感参考 |
| **刺客信条：探索之旅** | 游戏与教育内容的融合方式 |

## 二、汉语学习顺序体系（教学设计核心）

> **铁律**：游戏关卡的推进顺序 = 汉语学习的科学顺序。不是随机堆砌关卡，而是严格按照汉语教学（对外汉语 / HSK 大纲）的认知规律设计。

### 2.1 汉语学习的科学顺序（六大阶段）

```
阶段一：语音基础（拼音与声调）—— 一切的起点
  为什么先学这个：汉语是声调语言，声调错误直接导致意义完全不同
  （妈 mā / 麻 má / 马 mǎ / 骂 mà）
  学习内容：
    1. 声母（23个）：b p m f d t n l g k h j q x zh ch sh r z c s y w
    2. 韵母（24个）：单韵母 a o e i u ü → 复韵母 → 鼻韵母
    3. 声调（4声+轻声）：阴平 阳平 上声 去声
    4. 拼读规则：声母+韵母+声调的组合规律
    5. 声调辨听：听辨不同声调
  汉语特色设计：声调是汉语最独特的特征，必须贯穿整个学习过程

阶段二：汉字基础（笔画 → 部首 → 独体字）
  为什么紧接着学：汉字是表意文字，不像拼音文字可以"拼读"，
  必须建立"形音义"三维关联
  学习内容：
    1. 基本笔画（横竖撇捺点等 8 种基本笔画）
    2. 笔顺规则（从上到下、从左到右、先横后竖等）
    3. 常见偏旁部首（人 口 手 水 木 火 土 日 月 等 50 个高频部首）
    4. 独体字（一二三四五六七八九十 大小多少 上下左右 等）
    5. 汉字构造规律（左右结构、上下结构、包围结构）
  汉语特色设计：偏旁部首是汉字的"密码"，理解部首能举一反三

阶段三：核心词汇（HSK 1 级 ~150 词 + 量词系统）
  为什么这个顺序：有了拼音和汉字基础，开始积累高频词
  学习内容：
    1. 数字与量词（一个人、两本书、三杯水 —— 量词是汉语独有特征）
    2. 代词（我你他她它、这那哪）
    3. 日常名词（人、家、学校、公司、饭、水、钱、车 等）
    4. 基础动词（是、有、去、来、吃、喝、看、听、说、写 等）
    5. 基础形容词（大小、多少、好坏、快慢、高低 等）
    6. 时间词（今天、明天、昨天、现在 等）
  汉语特色设计：量词系统是汉语的标志性特征，每个名词对应特定量词

阶段四：句型与语法（基础句式 → HSK 2 级语法）
  为什么这个顺序：词汇量足够后，开始组装句子
  学习内容：
    1. 基础句式：SVO（主语+动词+宾语："我吃饭"）
    2. 疑问句（...吗？ / 什么？谁？哪里？怎么？为什么？）
    3. 否定句（不 / 没有）
    4. "的"字结构（我的书、红色的花）
    5. 介词短语（在...、从...到...、对...、给...）
    6. 时态表达（了/过/着 —— 汉语无动词变位，用助词表时态）
    7. 能愿动词（能、会、可以、想、要）
    8. 比较句（A比B...、A跟B一样...）
    9. 把字句、被字句（汉语特色语法）
  汉语特色设计：汉语没有动词变位/名词性数变化，但有独特的"了过着"时态系统和"把被"句式

阶段五：实用对话与场景（HSK 2-3 综合应用）
  为什么这个顺序：语法框架建立后，在真实场景中综合运用
  学习内容：
    1. 问候与自我介绍
    2. 餐厅点餐
    3. 购物砍价
    4. 问路导航
    5. 看病就医
    6. 工作面试（核心：越南用户在中国企业求职场景）
    7. 工厂/职场日常用语
    8. 电话/视频通话
  汉语特色设计：融入真实的越南人在中国/中国企业工作场景

阶段六：进阶表达与文化（HSK 3-4+）
  学习内容：
    1. 成语故事（汉语独特的四字结构）
    2. 复杂语法（虽然...但是...、不但...而且...、因为...所以...）
    3. 书面语 vs 口语区别
    4. 中国文化常识
    5. HSK 3-4 级目标词汇和语法点
```

## 四、核心游戏机制（MVP 版本详细设计）

### 4.1 关卡类型与过关标准

#### 类型一：知识教学关（先教后练）

**形式**：交互式教学 + 即时练习

```
场景示例 —— "声母海湾·第1关：b p m f"

教学阶段（60秒）：
  → 动画演示声母 b 的发音口型（AI 生成动画）
  → 播放标准发音（Azure TTS）
  → 显示发音要领提示（母语解释）
  → 依次教学 b p m f 四个声母

练习阶段（不限时，但有评分）：
  → 听音辨认：播放发音，选择对应声母（4选1）
  → 跟读练习：用户朗读，AI 评估发音准确度
  → 配对练习：将声母与对应拼音音节连线

过关标准：
  · 听音辨认准确率 ≥ 80%（5题对4题）
  · 跟读评分 ≥ 60 分（Azure Speech 评估）
  · 满分条件（3星）：准确率 100% + 跟读 ≥ 85 分

星级评定：
  · ★☆☆：过关（准确率 ≥ 80%）
  · ★★☆：良好（准确率 ≥ 90% 且跟读 ≥ 75）
  · ★★★：完美（准确率 100% 且跟读 ≥ 85）
```

#### 类型二：词汇战役关（游戏化练习）

**形式**：即时策略 + 语言知识

```
场景示例 —— "词汇平原·第3关：美食词汇战"

过场动画（20秒，中文配音+越南语字幕）：
  → NPC："欢迎来到美食街区！想买东西，
     你得先学会这些词..."

游戏阶段：
  1. [词汇收集] 屏幕上出现饮食图片，
     玩家需要在限时内将正确的汉字名称拖到对应图片上
     → 正确匹配：获得"字能"点数，充入能量条
     → 错误匹配：能量条不增加，图片抖动提示

  2. [句型组装] 出现打散的汉字卡片，
     玩家需要在限时内拖拽排列成正确句子
     → "我 / 想 / 要 / 一杯 / 水"
     → 正确排列后自动播放这句话的标准发音

  3. [Boss 对话] NPC 开始提问：
     → "你要什么？" → 选择正确回答
     → "多少钱？" → 在虚拟钱包中选出正确金额
     → 全部答对 = 击败关卡 Boss

过关标准：
  · 三个阶段均完成
  · 总正确率 ≥ 70%
  · Boss 对话阶段 ≥ 2/3 答对

星级评定：
  · ★☆☆：过关（总正确率 ≥ 70%）
  · ★★☆：良好（总正确率 ≥ 85% 且用时 < 平均值）
  · ★★★：完美（总正确率 ≥ 95% 且 Boss 全对）

结算画面：
  → 显示本关学会的词汇（自动加入词库）
  → 获得 XP + 金币
  → 星级评分 + 分享按钮
```

#### 类型三：汉字书写关（触屏手写）

**形式**：手写识别 + 笔画规范

```
场景示例 —— "汉字谷地·第5关：写出 人 大 天"

教学阶段：
  → 动画演示笔画顺序（一笔一画动态展示）
  → 显示笔画名称和规则

练习阶段：
  → 描红练习：按照虚线描写汉字（引导模式）
  → 自由书写：看拼音/图片写汉字（无引导）
  → AI 评估笔画顺序和字形规范

过关标准：
  · 描红练习：每字至少完成 2 次
  · 自由书写：3 个汉字中至少 2 个被 AI 正确识别
  · 笔画顺序正确率 ≥ 70%

星级评定：
  · ★☆☆：过关（2/3 识别正确）
  · ★★☆：良好（3/3 识别正确）
  · ★★★：完美（3/3 识别正确 + 笔画顺序全部正确 + 字形评分 ≥ 80）
```

#### 类型四：听力挑战关

**形式**：听中文音频 → 完成任务

```
场景示例 —— "语法要塞·第2关：听指令闯迷宫"

游戏阶段：
  → 播放中文指令："向左走"、"向右转"、"往前走三步"
  → 玩家操作游戏角色按指令移动
  → 指令逐渐变复杂："先向右走，然后向左转"

过关标准：
  · 正确执行指令 ≥ 7/10
  · 到达迷宫终点

星级评定：
  · ★☆☆：到达终点（允许 3 次错误）
  · ★★☆：到达终点（允许 1 次错误）
  · ★★★：零错误到达终点
```

### 4.2 小游戏设计（MVP 3款 — 杀手级创意）

> 设计原则：这 3 款小游戏必须让竞品玩家看到后说"卧槽这也行？"
> 不是换皮教育练习题，而是真正让人上瘾的游戏，只不过玩的过程中你把中文学了。

#### 小游戏一：声调狙击手（Tone Sniper）— Beat Saber × 中文声调

```
核心创意：
  把中文四声变成可视化的「声波轨迹」，用手指像切水果一样
  沿着正确的声调走势方向去「斩」飞来的词语。
  节奏感 + 视觉冲击 + 声调肌肉记忆，三位一体。

玩法：
  · 屏幕中央是准星区域，中文词语以「声波弹」形式从四面八方飞来
  · 每个声波弹上方显示汉字/拼音，同时播放该字的发音
  · 玩家必须在声波弹进入准星时，沿正确声调方向滑动斩击：
    一声 → 水平横划 ──    （平调）
    二声 → 右上斜划 ╱     （升调）
    三声 → V 形划  ╲╱    （降升调）
    四声 → 右下斜划 ╲     （降调）
  · 正确斩击：声波弹爆裂为粒子特效 + 连击 combo 数 +1
  · 连击效果：
    5 combo  → 屏幕边缘开始发光
    10 combo → 背景音乐 bass drop + 慢动作特效
    20 combo → 全屏霓虹爆炸 + 「完美节奏！」弹幕
  · 斩错方向：声波弹撞碎准星 + combo 归零 + 屏幕震动
  · 速度随时间递增：从 1 秒/个 → 0.3 秒/个（地狱难度）

模式：
  · 生存模式：3 条命，错 3 次结束，挑战最高分
  · 限时模式：60 秒尽可能多斩，排行榜竞赛
  · 每日挑战：固定词库 + 固定节奏，全服统一排名

过关标准：
  · ★☆☆：正确率 ≥ 60% 且存活 30 秒
  · ★★☆：正确率 ≥ 80% 且 combo ≥ 10
  · ★★★：正确率 ≥ 95% 且 combo ≥ 20

学习价值：
  · 声调辨识 → 听到发音后 0.5 秒内判断声调（训练直觉）
  · 肌肉记忆 → 手指划出声调形状，建立「声调 = 动作」的身体记忆
  · 高频重复 → 60 秒内可练习 40-80 个声调判断

为什么震惊对手：
  Duolingo 的声调练习 = 听音选 ABCD。我们的 = Beat Saber 式节奏斩击。
  没有任何语言学习 App 做过这种事。
```

#### 小游戏二：部首大爆炸（Radical Blitz）— 水果忍者 × 汉字拆解

```
核心创意：
  目标汉字的「幽灵轮廓」悬浮在屏幕上方。偏旁部首像陨石一样
  从天而降。你要像水果忍者一样斩中正确的偏旁，让它飞入轮廓
  的正确位置完成汉字。斩错偏旁 = 炸弹爆炸扣血。

玩法：
  · 屏幕上方显示目标汉字的「半透明轮廓」和拼音提示
  · 偏旁部首从屏幕下方/两侧不断飞入（夹杂干扰偏旁和炸弹）
  · 玩家滑动斩击正确偏旁 → 偏旁以粒子轨迹飞入轮廓对应位置
  · 全部偏旁就位 → 汉字「点亮」+ 爆炸组合特效 + 标准发音播放
  · 连续组装触发连锁：
    2 连组 → 「不错！」+ 时间 +3 秒
    3 连组 → 「太强了！」+ 全屏金光 + 双倍积分
    5 连组 → 「汉字大师！」+ 屏幕震动 + 三倍积分
  · 斩中炸弹/错误偏旁 → 爆炸扣血 + 连锁中断 + 画面闪红
  · 难度递进：
    初级：2 部件汉字（如 好=女+子），干扰少
    中级：3 部件汉字（如 想=木+目+心），干扰多
    高级：同时组装 2 个汉字，偏旁交叉飞入

模式：
  · 闯关模式：10/20/30 字过关，难度递增
  · 无尽模式：无限汉字，速度越来越快，挑战极限
  · Boss 战：限时 10 秒内组装一个复杂汉字（如「赢」= 亡+口+月+贝+凡）

过关标准：
  · ★☆☆：完成 ≥ 60% 汉字且 HP > 0
  · ★★☆：完成 ≥ 80% 且连锁 ≥ 3
  · ★★★：100% 完成 + 零失误 + 连锁 ≥ 5

学习价值：
  · 偏旁识别 → 高速场景下快速辨认 50+ 常用偏旁
  · 汉字结构 → 知道每个字由哪些部件组成、放在什么位置
  · 形义关联 → 完成组装时播放发音 + 显示含义，三维绑定

为什么震惊对手：
  竞品的汉字练习 = 描红/抄写。我们的 = 水果忍者式爽快斩击 + 拼装。
  你在玩的时候根本不觉得在学习，但每局下来你记住了 10-30 个汉字结构。
```

#### 小游戏三：拼音漂移（Pinyin Drift）— 赛车 × 听力反应

```
核心创意：
  俯视角赛车游戏。你的赛车在多车道高速公路上飞驰。
  耳机里播放中文词语/句子，每个路口前方的车道上标着不同选项
  （汉字/拼音/翻译），你必须漂移到正确车道通过检查点。
  答对加速，答错撞墙减速。和好友的「幽灵车」异步竞速。

玩法：
  · 俯视角赛车视角，赛车自动前进，速度逐渐递增
  · 前方出现 3-4 条分叉车道，每条标有不同答案
  · 同时播放音频（中文发音/句子）
  · 玩家左右滑动切换车道，在到达检查点前选中正确答案
  · 正确通过：
    → 漂移加速特效（轮胎冒烟 + 速度线）
    → 连续正确触发「极速模式」（画面模糊 + 氮气加速）
    → 每 10 连正确 → 速度永久提升一档
  · 错误通过：
    → 撞上路障 + 画面震动 + 减速 2 秒
    → 连续错误 → 赛车着火（3 连错 = 爆炸 Game Over）
  · 特殊道具（路上随机出现）：
    → 🛡️ 护盾：抵消一次撞墙
    → ⏱️ 慢动作：3 秒减速看清选项
    → 💎 双倍分：下 5 题得分翻倍

模式：
  · 单人竞速：跑最远距离，全服排行
  · 幽灵对决：和好友的历史最佳成绩异步竞速（看到好友的幽灵车）
  · 每日赛道：固定题目/速度曲线，全服统一排名
  · 组队接力（v1.2）：3 人队伍轮流驾驶

题型轮换（根据学习进度自动调整）：
  · 听音选字：播放发音 → 选正确汉字
  · 听音选拼音：播放发音 → 选正确拼音
  · 看字选义：显示汉字 → 选正确翻译
  · 听句补空：播放句子（缺一个词）→ 选正确补充

过关标准：
  · ★☆☆：存活 ≥ 30 秒（约 15 题）
  · ★★☆：存活 ≥ 60 秒 且正确率 ≥ 80%
  · ★★★：存活 ≥ 90 秒 且连续正确 ≥ 20

学习价值：
  · 听力反应速度 → 在时间压力下训练即时听力理解
  · 多维度复习 → 听/读/拼音/翻译全覆盖
  · 高频强化 → 60 秒可练 30+ 道题，效率远超传统卡片

为什么震惊对手：
  没有任何语言 App 把听力练习做成赛车游戏。
  异步幽灵竞速让你不知不觉和朋友较劲，每天都想刷新记录。
  你的朋友会问"你在玩什么赛车游戏？"然后发现是学中文的。
```

## 八、词库与复习系统（SRS - 间隔重复）

### 8.1 系统设计

```
核心机制：间隔重复算法（Spaced Repetition System）

每学会一个新词，自动进入 SRS 复习队列：

  初次学习 → 10 分钟后复习
  第 1 次正确 → 1 天后复习
  第 2 次正确 → 3 天后复习
  第 3 次正确 → 7 天后复习
  第 4 次正确 → 14 天后复习
  第 5 次正确 → 30 天后复习
  第 5 次以上 → 标记为"已掌握"

  任何一次复习答错 → 间隔减半，重新开始

复习形式（随机选择）：
  · 看汉字选含义
  · 看含义选汉字
  · 听发音选汉字
  · 看拼音写汉字
  · 看图片选汉字
```

### 8.2 每日复习提醒

```
触发规则：
  · 有词汇到期需要复习时，Push 通知提醒
  · 每日首次打开 App 时，如有待复习词汇，优先引导复习
  · 复习完成后再进入新关卡

复习激励：
  · 完成每日复习 = 5 金币
  · 连续 7 天完成复习 = "记忆大师"临时称号
```

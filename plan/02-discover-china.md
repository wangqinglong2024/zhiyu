# 发现中国（Discover China）— 模块设计文档

> 知语 Zhiyu 免费文化探索模块
> 版本：V1.0 | 日期：2026-04-17

---

## 一、模块定位

**发现中国** 是 知语 Zhiyu 的内容流量入口，以 **12 大文化类目** 覆盖中国文化的方方面面。未登录访客可浏览前 3 个类目（作为引流诱饵），登录用户解锁全部 12 类目全部内容。

核心目的：
1. **获客引流** — 公开前 3 个类目吸引访客 → 引导注册登录
2. **注册激活** — 登录后解锁全部 12 类目全部内容，形成“已登录即有收获”的正反馈
3. **课程导流** — 文化内容引导用户进入系统课程学习中文

> 产品逻辑：先让用户觉得中国有意思 → 再引导他注册 → 再引导他学中文 → 而不是一上来就推课程。

## 三、内容分级与访问模型

### 3.1 访问规则

| 用户类型 | 可访问内容 |
|---------|----------|
| **未登录访客** | 仅可浏览 **前 3 个类目**（01 中国历史、02 中国美食、03 名胜风光）的全部内容，其他 9 个类目显示「登录查看」遮罩 |
| **登录用户（免费）** | 全部 12 个类目均可浏览，所有内容免费 |
| **付费用户** | 与登录用户相同（发现中国全免费） |

### 3.2 内容分层

| 层级 | 说明 | 访问权限 |
|------|------|---------|
| **公开内容** | 前 3 个类目的全部内容 | 未登录即可查看 |
| **登录内容** | 第 4-12 类目的全部内容 | 登录用户 + 付费用户 |

### 3.3 登录引导触点

- 访客进入第 4-12 类目 → 登录遮罩："登录即可解锁全部 12 类目"
- 不设付费墙，课程转化通过 L4+ 按级购买驱动。

## 四、内容格式标准

### 4.1 单篇内容结构

```
标题（多语言：中文 + 拼音 + 用户母语）
封面图 / 视频缩略图
正文（多语言对照，300-800 字）
核心词汇表（中文 + 拼音 + 翻译，3-8 个）
趣味小测（1-3 题）
相关推荐（同类目其他文章 / 关联课程 / 关联游戏）
```

### 4.2 内容形式

| 形式 | 适用场景 |
|------|---------|
| 图文卡片 | 快速浏览、社交分享（主力形式） |
| 短视频（1-3 分钟） | 美食制作、景点展示、神话动画 |
| 音频解说 | 配合阅读、碎片时间听 |
| 互动测验 | 每篇文末，强化记忆和参与感 |

## 六、内容生产流程

### 6.1 生产工具链

```
选题策划 → AI 初稿（Dify / GPT）→ 人工审核润色 → 多语言翻译 → 配图 / 配视频 → 发布
```

### 6.2 团队分工

| 角色 | 职责 |
|------|------|
| 内容策划 | 选题、热点追踪、排期 |
| AI + 编辑 | AI 生成初稿 → 编辑审核、润色、本地化 |
| 翻译 | 越南语 / 英语翻译校对 |
| 设计 | 封面图、配图、短视频制作 |

### 6.3 更新节奏

| 阶段 | 频率 | 说明 |
|------|------|------|
| MVP 上线前 | 每类目 5+ 篇预备 | 保证上线时内容充足（合计 60+ 篇） |
| 上线后常规 | 每周 3-5 篇（分散到各类目） | 保持新鲜感 |
| 节日 / 热点 | 临时加更 | 蹭热点，提升传播 |

## 八、变现逻辑

```
3 个公开类目（未登录） → 访客进入   →   9 个类目提示登录
                                            ↓
                                   注册登录（漏斗第 1 层）
                                            ↓
                           12 类目全部内容免费 → 内容引导至课程/游戏
                                            ↓
                              课程 L4+ 需付费 → 课程购买转化（漏斗第 2 层，核心变现）
```

关键：**发现中国是获客漏斗的顶部**，用 3 个免登录公开类目引流，再用 9 个登录墙驱动注册，最后用课程付费墙（L4+）驱动课程付费。

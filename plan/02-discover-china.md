# 发现中国（Discover China）— 模块设计文档

> 知语 Zhiyu 免费文化探索模块
> 版本：v6.0（终版） | 日期：2026-04-16

---

## 一、模块定位

**发现中国** 是 知语 Zhiyu 的内容流量入口（非会员可免费访问前 3 个模块，会员解锁全部 4 个模块），面向所有对中国文化感兴趣的用户。

核心目的：
1. **获客引流** — 零门槛内容吸引下载，降低获客成本
2. **兴趣激发** — 让用户在了解中国文化的过程中产生学中文的动力
3. **付费导流** — 自然引导至系统课程和游戏模块

> 产品逻辑：先让用户觉得中国有意思 → 再引导他去学中文 → 而不是一上来就推课程。

## 三、内容策略

### 3.1 内容分级

| 级别 | 可见范围 | 作用 |
|------|---------|------|
| 非会员内容 | 前 3 个模块完整可访问（注册即可） | 获客引流，体验产品价值 |
| 会员内容 | 第 4 个模块需会员解锁 | 驱动会员转化 |
| 关联内容 | 文末推荐相关课程/游戏 | 导流至课程和游戏模块 |

### 3.2 内容格式标准

每篇内容包含：

```
标题（多语言）
封面图 / 视频缩略图
正文（多语言对照，300-800字）
核心词汇表（中文+拼音+翻译，3-8个）
趣味小测（1-3题）
相关推荐（其他文化文章 / 课程 / 游戏关卡）
```

### 3.3 内容与课程联动

| 文化内容 | 可关联的课程 Level | 联动方式 |
|---------|-----------------|---------|
| 春节的由来 | L1-L3（启蒙段） | "想学会用中文说'新年快乐'吗？→ 课程 L1" |
| 八大菜系 | L4-L6（基础段） | "学会用中文点菜 → 课程 L5" |
| 古诗赏析 | L7-L9（核心段） | "读懂这首古诗 → 课程 L8" |
| 成语故事深度 | L10-L12（高阶段） | "用这个成语写一段话 → 课程 L11" |

# PlayLingo — 发现中国模块设计方案

> 让全世界爱上中国——全方位的中华文化探索平台
> 版本：v3.0 | 日期：2026-04-15

---

## 一、模块定位

**发现中国** 是 PlayLingo 三大模块之一，定位为**内容驱动的中华文化探索平台**。它不是简单的百科全书，而是精心策划的文化之旅，用精美图文、音频讲解和互动体验，让用户全方位了解中国、爱上中国，从而自然激发学中文的强烈动机。

### 核心设计哲学

1. **兴趣先于教学**：不直接教中文，而是讲好故事、展示美景、传递智慧，让学习动机自然萌发
2. **双语沉浸**：所有内容中越双语呈现，关键词标注拼音，阅读即学习
3. **深度而非广度**：每篇文章深入讲透一个主题，而非蜻蜓点水
4. **互动而非被动**：阅读后有测验、词汇卡片、讨论区，不是单向灌输

└─────────────────────────────────────┘
```

### 内容规范

| 维度     | 标准                                                         |
| -------- | ------------------------------------------------------------ |
| 字数     | 越南语主体 800-1500 字 + 中文原文 200-500 字                 |
| 难度分级 | ⭐ 入门（HSK1-2词汇）/ ⭐⭐ 进阶（HSK3词汇）/ ⭐⭐⭐ 高级（HSK4+词汇） |
| 中文标注 | 所有汉字标注拼音，关键词可点击查看释义                       |
| 音频     | 每篇至少1段中文音频朗读（AI TTS 生成）                       |
| 配图     | 每篇 3-5 张配图（AI 生成 / 公域素材）                        |
| 词汇提取 | 每篇 8-15 个核心词汇，自动关联到词汇本                       |
| 测验     | 每篇 5 道互动题（2理解 + 2词汇 + 1文化常识）                 |
| SEO      | 每篇优化越南语 SEO 标题/描述/关键词                          |

### 难度适配策略

| 用户水平 | 文章呈现方式                                       |
| -------- | -------------------------------------------------- |
| 零基础   | 越南语主体，中文仅出现核心词汇（带拼音），音频辅助 |

用户可在设置中选择偏好的难度级别，系统据此调整推荐内容和展示方式。

### AI 辅助创作流水线

```

## 五、跨模块联动设计

### 文化 → 课程联动

- 文章中的中文词汇标记为「课程可学」，点击可查看对应课程
- 文章底部推荐「相关课程」："想学会文章里的中文？去 Level 1 第3课"
- 课程中的「文化角」环节链接回文化文章

### 文化 → 游戏联动

- 成语典故系列可生成成语闯关小游戏（v1.1）
- 文章底部推荐相关游戏关卡："学了拼音知识？去拼音群岛试试身手"
- 游戏过场动画中引用文化内容

### 文化 → 词汇联动

- 文章中所有标注词汇一键加入用户词汇本
- 词汇本中标记词汇来源（"来自：《西游记：孙悟空大闹天宫》"）
- SRS 复习时展示词汇的文化背景

### 免费与付费边界

| 内容     | 非会员          | 付费会员  |
| -------- | ----------------- | --------- |

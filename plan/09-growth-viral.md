> 版本：v5.0（终版） | 日期：2026-04-16

## 一、增长飞轮模型

### 1.1 核心飞轮

```
免费文化内容 → 吸引下载/注册
       ↓
试学课程 + 试玩游戏 → 体验价值
       ↓
付费转化 → 会员/课程/道具
       ↓
深度学习 → 获得成果（证书/成就）
       ↓
社交分享 → 朋友圈/社群传播
       ↓
推荐返佣 → 老用户推荐新用户
       ↓
新用户进入 → 飞轮加速
```

### 1.2 飞轮加速器

| 加速器 | 作用 |
|--------|------|
| **发现中国（免费内容）** | 降低获客门槛，提供传播素材 |
| **推荐返佣** | 经济激励驱动老带新 |
| **社交分享** | 成就感+炫耀心理驱动自发传播 |
| **排行榜** | 竞争心理驱动活跃 + 拉人 |
| **家人共享** | 年度/终身会员共享权益，天然裂变 |

## 二、推荐裂变系统

### 2.1 双向激励机制

```
推荐人 A                          被推荐人 B
   │                                  │
   ├── 分享邀请链接/二维码              │
   │         ──────────────────►      │
   │                                  ├── 注册
   │                                  ├── 首单 9 折优惠
   │                                  └── 首次付费完成
   │         ◄──────────────────      │
   ├── 获得 B 首付金额 20% 返佣        │
   ├── 积累推荐人数解锁额外奖励        │
   └── 获得推荐成就徽章                │
```

### 2.2 推荐人分层奖励

| 累计推荐人数 | 额外奖励 | 称号 |
|------------|---------|------|
| 1 人 | 道具礼包 | 推荐新手 |
| 5 人 | 1 个月会员 | 推荐达人 |
| 10 人 | 基础道具包 × 3 | 推荐高手 |
| 20 人 | 3 个月会员 | 推荐大师 |
| 50 人 | 年度会员 | 推荐导师 |
| 100 人 | 终身会员 | 推荐传奇 |

### 2.3 推荐码机制

| 维度 | 说明 |
|------|------|
| 邀请码格式 | 6 位字母数字（如 LB3X7K） |
| 邀请链接 | `lingobase.com/join/LB3X7K` |
| 二维码 | 自动生成，可保存分享 |
| 绑定规则 | 被推荐人注册后绑定，永久有效 |
| 返佣确认 | 被推荐人首次付费 + 7 天无退款确认期 |
| 提现 | 累计 ¥50 起提现（PayPal / 银行 / 平台余额） |

## 五、用户生命周期管理

### 5.1 生命周期阶段

```
新用户 → 活跃用户 → 付费用户 → 忠诚用户 → 传播用户
  ↓          ↓          ↓          ↓
沉默用户 ← 流失用户（各阶段都可能流失）
```

### 5.2 各阶段策略

#### 新用户（注册后 0-7 天）

| 策略 | 说明 |
|------|------|
| 新手引导 | 3 步快速上手（选语言 → 入学测试 → 开始学习） |
| 首日推送 | 欢迎 + "你的第一课已准备好" |
| 首周签到 | 连续 7 天签到奖励丰厚 |
| 试学引导 | 引导完成 L1 第一单元 |
| 游戏入门 | 引导试玩第 1-3 关 |

#### 活跃用户（D7+，未付费）

| 策略 | 说明 |
|------|------|
| 学习提醒 | 每日固定时间推送（用户可设置） |
| 进度展示 | 强调"你已经学了 X 个汉字" |
| 付费墙 | 到达免费内容边界时展示会员优惠 |
| 限时优惠 | "前 7 天注册用户专享 7 折" |

#### 付费用户

| 策略 | 说明 |
|------|------|
| 专属内容 | 会员专属学习报告 |
| 进度追踪 | 周报 / 月报推送 |
| 续费提醒 | 到期前多次温馨提醒 |
| VIP 社群 | 付费用户专属学习群 |

#### 忠诚/传播用户

| 策略 | 说明 |
|------|------|
| 推荐激励 | 突出返佣和推荐奖励 |
| KOC 培养 | 邀请创作体验分享内容 |
| 内测资格 | 新功能优先体验 |
| 专属称号 | 社区内的特殊标识 |

### 5.3 流失挽回

| 流失阶段 | 策略 |
|---------|------|
| 3 天未登录 | Push：你的学习进度在等你 |
| 7 天未登录 | Push + 邮件：送你一张双倍积分卡 |
| 14 天未登录 | Push + 邮件：限时优惠回归 |
| 30 天未登录 | 邮件：专属回归礼包（免费 7 天会员） |
| 60 天未登录 | 最后一封：我们很想你（不再频繁打扰） |

## 六、病毒传播系数估算

### 6.1 K-Factor 模型

```
K = 邀请数 × 转化率

目标：K > 0.5（每 2 个用户带来 1 个新用户）
理想：K > 1（自然增长，无需付费获客）
```

### 6.2 优化路径

| 阶段 | K 值估计 | 优化重点 |
|------|---------|---------|
| MVP | 0.2-0.3 | 完善分享功能，优化邀请流程 |
| 增长期 | 0.4-0.6 | 返佣体系上线，KOL 推荐 |
| 成熟期 | 0.6-0.8 | 品牌口碑积累，自然传播 |
| 目标 | > 0.8 | 接近自增长临界点 |

## 七、增长实验计划

### 7.1 A/B 测试优先级

| 实验 | 假设 | 指标 |
|------|------|------|
| 付费墙位置 | L1 完成 vs L2 第一课 | 付费转化率 |
| 定价方案 | ¥39 vs ¥49 月度 | ARPU |
| 新手引导 | 3步 vs 5步 | D1 留存 |
| 分享奖励 | 积分 vs 道具 vs 试学 | 分享率 |
| 推送时间 | 早8点 vs 晚8点 vs 自定义 | 打开率 |
| 邀请文案 | 功能描述 vs 成果展示 vs 优惠引导 | 邀请转化 |

### 7.2 数据驱动

- 所有核心页面埋点（注册、首课、付费、分享、推荐）
- 周度增长会议：复盘数据 → 制定实验 → 执行 → 评估
- 月度增长报告：渠道 ROI、LTV、CAC、K-Factor

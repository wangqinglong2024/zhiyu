# PlayLingo — 运营与营销方案

> 越南首战 → 东南亚扩张 → 全球化
> 版本：v2.0 | 日期：2026-04-15

## 一、运营总纲

### 1.1 核心理念

```
"让每个越南人都知道，学中文可以很便宜、很好玩"

关键词：便宜 + 好玩 + 赚钱（佣金）
价格锚点：$2.99/月 vs Duolingo $12.99/月 vs 线下课 $50-100/月
```

### 1.2 运营节奏

| 阶段 | 时间 | 重点 |
|------|------|------|
| MVP 内测 | 第 1 月 | 200 种子用户，验证核心学习链路 |
| 冷启动 | 第 2-3 月 | TikTok 矩阵 + KOL 合作 + 佣金推广员招募 |
| 增长期 | 第 4-6 月 | 付费投放 + PvP 社交裂变 + B2B 企培 |
| 爆发期 | 第 7-12 月 | 品牌化 + 印尼扩张 + 大型活动 |

## 四、KOL / KOC 合作

### 4.1 合作策略

```
阶段 1（冷启动）：KOC 为主（粉丝 1K-50K）
  · 成本低（免费账号 + 佣金分成）
  · 内容真实，转化率高
  · 目标：合作 20 个 KOC

阶段 2（增长期）：KOL 进入（粉丝 50K-500K）
  · 付费合作 + 佣金分成
  · 单条视频 $50-200
  · 目标：合作 5-10 个 KOL

选择标准：
  · 内容与中国/学习/游戏相关
  · 粉丝画像覆盖 18-35 岁越南年轻人
  · 互动率 > 3%（真实流量）
```

### 4.2 合作形式

| 形式 | 说明 | 成本 |
|------|------|------|
| 免费体验 | 送高级会员 + 佣金权益 | 0 |
| 专属推广链接 | KOL 专属推广链接，追踪转化 | 佣金 |
| 定制内容 | 按需制作 PlayLingo 主题视频 | $50-200/条 |
| 长期代言 | 月度合作，定期发布内容 | 可谈 |

## 五、付费投放

### 5.1 启动时机

```
触发条件（满足任一）：
  · 自然增长达到 500 DAU
  · LTV/CAC > 3
  · 月收入 > $500
  · MVP 核心数据验证通过（次日留存 >40%、7 日留存 >20%）
```

### 5.2 投放策略

```
Facebook/Instagram Ads：
  · 按 CPI 优化
  · 目标 CPI < $0.30（越南）
  · A/B 测试：价格导向 vs 游戏导向 vs 赚钱导向
  · 预算：起步 $10/天，逐步放大

TikTok Ads：
  · Spark Ads（加热自然内容）
  · 目标 CPI < $0.20
  · 年轻用户效果更好
  · 预算：起步 $10/天

Google Ads：
  · 搜索广告："学中文 app" 越南语关键词
  · 目标 CPI < $0.50
  · 低优先级，SEO 优先
```

### 5.3 投放漏斗

```
广告曝光 → 落地页/商店页 → 注册 → 完成新手引导 → 7 日留存 → 付费
  100%  →   30%      →  50%  →    70%       →   25%    →  5%

目标 CPI: $0.20-0.30
目标 CPA (付费): $4-6
计划 LTV: $15+ (5 个月 ARPU)
```

## 六、佣金推广运营

### 6.1 推广员管理

```
招募渠道：
  · 现有付费用户中招募活跃用户
  · TikTok/Facebook 发布"赚钱学中文"招募帖
  · 大学校园代理（学生推广员）
  · KOC 自动升级为推广员

推广员类型：
  · 普通用户推广员（自然分享）
  · 学生推广员（校园渠道）
  · 教师推广员（教学场景推荐）
  · KOL/KOC 推广员（内容渠道）
```

### 6.2 推广物料

```
App 内一键生成：
  · 个人专属推广链接（点击即绑定，无需输入邀请码）
  · 带推广链接二维码的分享海报（多模板可选）
  · 短视频用推广片头/片尾
  · 一句话推荐文案（越南语，可复制）
  · 佣金收益报告（实时更新）

推广话术模板（越南语）：
  · "Tôi đang học tiếng Trung qua game, rẻ lắm chỉ 69K/tháng!"
    （我在用游戏学中文，很便宜只要 69K/月！）
  · "Bấm link này để giảm 30% tháng đầu!"
    （点这个链接首月减 30%！）
```

### 6.3 防作弊运营

```
监控指标：
  · 单个推广员每日新增上限：50 人
  · 推荐用户 7 日留存必须 > 10%（否则不计佣金）
  · 同 IP/设备注册的推荐不计佣金
  · 推荐人和被推荐人不能同设备

处罚措施：
  · 疑似刷量：暂停佣金，人工审核
  · 确认违规：永久取消推广资格
  · 严重违规：冻结账户余额
```

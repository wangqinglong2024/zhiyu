# 游戏模式（Games）— 100 关趣味闯关设计

> LingoBase 游戏化练习模块
> 版本：v5.0（终版） | 日期：2026-04-16

---

## 一、模块定位

游戏模式是 LingoBase 的**趣味练习场**，通过闯关机制巩固系统课程所学知识，提升用户活跃度和留存率。

| 维度 | 说明 |
|------|------|
| 定位 | 课程知识的游戏化巩固，不替代系统课程 |
| 关卡总数 | **100 关** |
| 难度分级 | **4 级**，对应课程 4 大阶段 |
| 变现 | 免费基础关卡 + 付费高级关卡 + 道具内购 |

---

## 二、四级关卡体系

### 2.1 关卡分布

| 级别 | 对应课程阶段 | 关卡范围 | 关卡数 | 开放条件 |
|------|------------|---------|--------|---------|
| **青铜** | 启蒙段（L1-3） | 第 1-25 关 | 25 关 | 免费开放前 10 关 |
| **白银** | 基础段（L4-6） | 第 26-50 关 | 25 关 | 课程达到 L4 或付费解锁 |
| **黄金** | 核心段（L7-9） | 第 51-75 关 | 25 关 | 课程达到 L7 或付费解锁 |
| **钻石** | 高阶段（L10-12） | 第 76-100 关 | 25 关 | 课程达到 L10 或付费解锁 |

### 2.2 解锁逻辑

```
方式一：课程进度解锁（推荐）
  学完课程 L1 → 解锁青铜关卡对应部分
  学完课程 L4 → 解锁白银关卡
  ...以此类推

方式二：付费解锁
  购买对应关卡包直接解锁（无需课程进度）

方式三：连续通关
  连续 3 星通过当前级别所有关卡 → 可挑战下一级别（无需课程/付费）
```

## 三、关卡设计

### 3.1 单关结构

每一关包含：

```
关卡 X
  ├── 关卡预览（目标+知识点提示）
  ├── 挑战内容（5-10 道互动题）
  ├── 计时 + 积分
  ├── 结算页（星级评定+积分+奖励）
  └── 错题回顾
```

### 3.2 游戏题型

| 题型 | 说明 | 对应知识模块 |
|------|------|------------|
| **拼音配对** | 汉字与拼音连线 / 选择正确拼音 | M2 拼音与语音 |
| **汉字拼图** | 拖拽偏旁部首组合汉字 | M1 汉字与书写 |
| **词语消消乐** | 在字母矩阵中找出正确词汇 | M3 词汇与语义 |
| **句子排序** | 打乱词序，重新排列成正确句子 | M4 语法与句法 |
| **听音选图** | 听中文音频，选择对应图片 | 听力 |
| **语音闯关** | 跟读句子，AI 评分 | 口语 |
| **填字游戏** | 根据提示填入正确汉字 | M1 + M3 |
| **成语接龙** | 用前一个成语最后一个字接下一个 | M3 词汇 |
| **古诗补全** | 给出上句，选择/填写下句 | M9 古诗文 |
| **文化问答** | 关于中国文化的趣味选择题 | M8 文化 |
| **翻译挑战** | 中文↔母语翻译选择 | 综合 |
| **限时打字** | 限时内输入正确中文 | M1 + M7 |

### 3.3 各级别题型分布

| 级别 | 主要题型 | 次要题型 | 特殊题型 |
|------|---------|---------|---------|
| 青铜 | 拼音配对、汉字拼图、听音选图 | 词语消消乐、翻译挑战 | — |
| 白银 | 句子排序、填字游戏、语音闯关 | 成语接龙、文化问答 | 限时打字 |
| 黄金 | 阅读理解、古诗补全、成语接龙 | 语音闯关进阶、翻译挑战 | 文言文趣味题 |
| 钻石 | 综合挑战、文言文解读、学术表达 | 高级成语、文化深度题 | Boss 关卡 |

## 七、特殊关卡

### 7.1 Boss 关卡

每 25 关的最后一关为 **Boss 关卡**（第 25、50、75、100 关）：

| Boss 关 | 名称 | 特殊规则 |
|---------|------|---------|
| 第 25 关 | 启蒙守护者 | 混合题型，限时 5 分钟，三次机会 |
| 第 50 关 | 基础挑战者 | 进阶题型，限时 8 分钟，两次机会 |
| 第 75 关 | 核心大师 | 综合题型（含文言文），限时 10 分钟，两次机会 |
| 第 100 关 | 终极学者 | 全知识点覆盖，限时 15 分钟，一次机会 |

通过 Boss 关获得特殊成就徽章和丰厚奖励。

### 7.2 每日挑战

- 每天刷新 1 个限时挑战关卡
- 不限级别，题目随机
- 完成获得每日积分奖励
- 连续完成 7 天获得额外大奖

### 7.3 节日活动关卡

- 春节、中秋等节日推出限时特别关卡
- 关卡主题贴合节日文化
- 限时奖励（限定徽章、限定道具）

## 九、游戏与课程联动

### 9.1 知识同步

| 联动方式 | 说明 |
|---------|------|
| 内容对齐 | 游戏关卡的词汇/语法/文化知识与对应 Level 课程一致 |
| 进度互通 | 课程中学到的生字自动出现在游戏复习中 |
| 错题收集 | 游戏中答错的题目自动加入课程复习计划 |
| 成就共享 | 游戏成就和课程成就在同一系统中展示 |

### 9.2 引导策略

- 游戏中遇到不会的知识点 → 弹出"去课程学习"引导
- 课程章节学完后 → 弹出"去游戏巩固"引导
- 游戏连续 3 关低于 2 星 → 推荐回到课程复习

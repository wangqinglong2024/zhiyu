# 游戏模式（Games）— 12 款可无限玩的 2D 学习游戏

> 知语 Zhiyu 游戏化练习模块
> 版本：v8.0（终版） | 日期：2026-04-16

---

## 一、设计哲学

### 1.1 核心原则

**不做闯关，做上瘾。**

传统语言学习 App 的闯关模式有一个致命问题：通关了就没动力再玩了。知语的游戏要做到：

1. **无限可玩** — 每局随机出题，永远有新鲜感，像切水果一样可以一直玩
2. **一局一局上瘾** — 单局 2-3 分钟，结束后立刻想"再来一局"
3. **围绕课程成长** — 每款游戏的题目素材 100% 来自对应 Level 的课程内容
4. **社交 PK** — 所有游戏都支持单人练习 + 1v1 实时 PK + 多人对战房间
5. **成就驱动** — 累计积分、段位升降、个人最高分记录，形成长期追求
6. **登录即可全开** — 12 款游戏对**所有已登录用户**完全开放，无需课程进度门槛，通过皮肤/特效/通行证变现
7. **横屏优先** — 所有游戏强制横屏（landscape），同时支持 Web 桌面和移动端 H5

### 1.2 访问模型

| 用户类型 | 游戏模块权限 |
|---------|------------|
| **未登录访客** | 不可进入游戏模块；点击游戏入口 → 跳转登录/注册 |
| **登录用户（免费）** | **12 款游戏全部解锁**，可玩单人 / 1v1 / 多人对战全部模式 |
| **付费用户** | 与登录用户相同 + 付费用户专属皮肤/称号折扣 + 部分赛季通行证首赠 |

> 关键决策：不再用"完成对应 Level 课程第 1 单元"作为解锁门槛。原因有三：
> 1. 课程是付费主阵地，游戏是留存/裂变主阵地，二者漏斗应分离
> 2. 高 Level 游戏（G10-G12）若卡住付费才能玩，会导致免费用户流失
> 3. 对低水平玩家来说，玩高 Level 游戏时能被难题"劝回课程"，这本身就是一种引流

### 1.3 与课程的关系

```
12 级课程（付费为主）         12 款游戏（登录即玩）
         ↕                          ↕
  题库 100% 来自课程          入口无课程依赖
         ↕                          ↕
  学完 Lx 课程 → 被推荐去玩对应 Gx 巩固
  Gx 中答错 → SRS 加入，并推荐回课程 Lx 复习
```

每款游戏的题库**完全来自对应 Level 的课程知识点**。玩游戏就是在复习课程，但体验是纯游戏的快感。玩家可以直接挑战高 Level 游戏——被题难住时，系统会推荐对应课程。

## 三、12 款游戏详细设计

### 游戏 1：汉字切切切（Level 1 · 启蒙）

| 维度 | 说明 |
|------|------|
| **灵感** | Fruit Ninja（切水果） |
| **画面** | 汉字卡片从屏幕底部飞出，玩家滑动手指切割 |
| **核心玩法** | 屏幕飞出汉字卡片，每张卡片标注拼音或图片提示；玩家需要切**正确的汉字**（如屏幕提示"māo"，切"猫"得分），切错扣一条命 |
| **无限模式** | 无尽模式，速度逐渐加快，直到三条命耗尽 |
| **题库来源** | L1 课程的 300 个基础汉字 |
| **单局时长** | 1-3 分钟 |

**游戏模式**：

| 模式 | 说明 |
|------|------|
| **单人·经典** | 无尽模式，挑战个人最高分 |
| **单人·限时** | 60 秒内切对最多汉字 |
| **1v1 PK** | 同屏比赛，同时出题，谁先切对谁得分 |
| **多人混战** | 2-4 人实时对战，积分最高者赢 |

**上瘾机制**：
- 连续切对 Combo 加分（3连+5连+10连有特效和音效）
- 个人最高分记录，每次都想刷新
- 每日挑战：特殊模式（只出形近字/只出同声调）
- 段位系统：青铜→白银→黄金→铂金→钻石→星耀→王者

### 游戏 2：拼音泡泡龙（Level 2 · 基础读写）

| 维度 | 说明 |
|------|------|
| **灵感** | Bubble Shooter（泡泡龙） |
| **画面** | 顶部排列汉字泡泡，底部发射拼音泡泡 |
| **核心玩法** | 顶部有各种汉字泡泡，玩家从底部射出**拼音泡泡**，命中对应汉字即可消除。3 个以上连消额外加分。顶部泡泡逐渐下降，触底则 Game Over |
| **无限模式** | 消完一层自动生成新一层，越来越快 |
| **题库来源** | L2 课程 800 字 + 偏旁部首 |
| **单局时长** | 2-5 分钟 |

**游戏模式**：

| 模式 | 说明 |
|------|------|
| **单人·生存** | 泡泡不断下降，坚持越久分越高 |
| **单人·清屏** | 每关一屏泡泡，全消后自动进入下一屏（无限循环） |
| **1v1 PK** | 你消的泡泡会变成垃圾泡泡送到对手那边 |
| **多人对战** | 4 人竞速，比谁先清完 3 屏 |

**上瘾机制**：
- 连消特效（彩虹泡泡、炸弹泡泡）
- 声调颜色区分（一声蓝、二声绿、三声橙、四声红）
- 每日挑战：只用声母消/只用韵母消

### 游戏 3：词语消消乐（Level 3 · 段落理解）

| 维度 | 说明 |
|------|------|
| **灵感** | Candy Crush（三消游戏） |
| **画面** | 棋盘上摆满词语方块，交换位置触发消除 |
| **核心玩法** | 棋盘上每个格子是一个词语。交换相邻两个格子，如果形成**近义词配对/反义词配对/正确搭配词组**则消除得分。新词语从顶部落下，保持棋盘满。无法消除时 Game Over |
| **无限模式** | 棋盘自动补充新词，可以一直玩 |
| **题库来源** | L3 课程 2000 词汇 + 近反义词 + 搭配 |
| **单局时长** | 3-5 分钟 |

**游戏模式**：

| 模式 | 说明 |
|------|------|
| **单人·无尽** | 无限消除，挑战最高分 |
| **单人·限步** | 限制 30 步内得到最高分 |
| **1v1 PK** | 同棋盘，比同时间内谁分高 |
| **多人对战** | 4 人同棋盘限时竞速 |

**上瘾机制**：
- 4 消/5 消/L 消产生特殊方块（炸弹、彩虹、横扫）
- 连锁消除（cascade）爽感
- 每局结束展示"你学到了 X 组近义词"

### 游戏 4：语法大厨（Level 4 · 篇章理解）

| 维度 | 说明 |
|------|------|
| **灵感** | Overcooked（胡闹厨房） |
| **画面** | 中国风餐厅厨房，顾客排队点菜 |
| **核心玩法** | 顾客点出一个"菜名"（目标句子的意思），玩家需要从传送带上抓取正确的**词语食材**，按正确**语序**放入锅中"烹饪"成句子。完整正确的句子端给顾客得分，超时或语序错误则顾客离开扣分 |
| **无限模式** | 顾客源源不断，速度递增 |
| **题库来源** | L4 课程的句型结构、关联词、复句 |
| **单局时长** | 2-4 分钟 |

**游戏模式**：

| 模式 | 说明 |
|------|------|
| **单人·经营** | 无尽顾客，挑战服务数量 |
| **单人·限时** | 90 秒内完成最多订单 |
| **1v1 PK** | 各自厨房，同时接单，比谁完成多 |
| **多人协作** | 2 人合作厨房，一人选词一人排序 |

**上瘾机制**：
- 连续正确获得 Combo 加速
- 特殊顾客（VIP）给双倍分
- 餐厅升级系统（累计积分装修餐厅）
- "因为……所以……"等关联词是特殊食材，用对加分

### 游戏 5：成语接龙大战（Level 5 · 深度阅读）

| 维度 | 说明 |
|------|------|
| **灵感** | 文字类棋牌对战 |
| **画面** | 中国象棋风格棋盘，成语卡牌 |
| **核心玩法** | 回合制。系统给出一个成语，玩家需要用**最后一个字**（或同音字）接出新成语。接不出则倒计时 10 秒内从提示列表中选择。10 秒内无法接出则输掉本回合。三回合制 |
| **无限模式** | 单人模式下与 AI 无限接龙，记录最长连接数 |
| **题库来源** | L5 课程 80+ 成语 + 扩展成语库 |
| **单局时长** | 2-5 分钟 |

**游戏模式**：

| 模式 | 说明 |
|------|------|
| **单人·无尽** | 与 AI 无限接龙，挑战最长链 |
| **单人·限时** | 60 秒内接出最多成语 |
| **1v1 PK** | 真人对战接龙，接不出就输 |
| **多人车轮战** | 3-6 人轮流接龙，淘汰制 |

**上瘾机制**：
- 每个成语显示释义和故事（学完即用）
- 段位星数排名系统
- 用生僻成语接龙额外加分（奖励学习新成语）
- 每日"成语之星"排名

### 游戏 6：汉字华容道（Level 6 · 综合提升）

| 维度 | 说明 |
|------|------|
| **灵感** | 华容道 / 数字滑块谜题 |
| **画面** | 4×4 或 5×5 方格棋盘，汉字方块 |
| **核心玩法** | 棋盘上的汉字方块打乱了顺序，玩家通过滑动方块（利用空位）将汉字排成**正确的成语/句子/古诗句**。步数越少、速度越快分越高。每局随机生成新题目 |
| **无限模式** | 解完一题自动出下一题，难度递增 |
| **题库来源** | L6 课程的成语、句子、缩句扩句 |
| **单局时长** | 1-3 分钟/题，可连续解题 |

**游戏模式**：

| 模式 | 说明 |
|------|------|
| **单人·无尽** | 连续解题，3 次超时结束 |
| **单人·极速** | 3 分钟内解最多题 |
| **1v1 PK** | 同题竞速，谁先解出谁赢 |
| **多人竞速** | 4 人同题，排名计分 |

**上瘾机制**：
- 最少步数挑战（显示"理论最优解"对比）
- 解题速度排行榜
- 每日一题（特殊超难题，全服排名）

### 游戏 10：辩论擂台（Level 10 · 高中语文）

| 维度 | 说明 |
|------|------|
| **灵感** | 卡牌对战（炉石传说简化版） |
| **画面** | 辩论赛场景，正反方对阵 |
| **核心玩法** | 系统给出一个辩题 + 自动分配正/反方。双方各有 5 张手牌，每张卡牌**预设固定的攻击力（说服力）和防御力（逻辑严密度）数值**。3 回合内轮流出牌 → 服务端按预设数值结算，对方可出反驳卡抵消。总说服力高者胜。**不使用 AI 动态评分**，完全类似传统 CCG 的确定性数值对战，保证公平与可实现性。|
| **无限模式** | 随机辩题 + AI 对手（按卡牌数值+简单策略），永远有新辩论 |
| **题库来源** | L10 课程议论文素材、论证方法、名言警句（卡牌文案来源，数值由策划预设） |
| **单局时长** | 3-5 分钟 |

**游戏模式**：

| 模式 | 说明 |
|------|------|
| **单人·练习** | vs AI 辩手，难度可调 |
| **1v1 辩论** | 真人对战，限时出牌 |
| **多人辩论赛** | 4 人淘汰赛 |
| **每日辩题** | 全服同一辩题，投票最佳辩手 |

**上瘾机制**：

### 4.1 段位层级

| 段位 | 图标 | 小段位数 | 每段星数 | 该段总星数 | 累计星数 |

### 4.2 核心规则

| 规则 | 说明 |
|------|------|
| **赢了** | +1 🌟 |
| **输了** | -1 🌟 |
| **最低** | 1 🌟（青铜 III 最低 1 星，不会再掉） |

| **赛季重置** | 每赛季（3 个月）段位软重置，保留基础段位 |

### 4.3 段位设计逻辑

```
新手：从青铜 III 1🌟 开始
       ↓
保持 50% 胜率：约 180 局可达王者（每日 5 局 ≈ 36 天）
保持 60% 胜率：约 120 局可达王者（每日 5 局 ≈ 24 天）
       ↓
王者段位：按排名竞争，持续活跃才能保持排名
```

## 五、解锁与变现

### 5.1 解锁逻辑

**所有 12 款游戏完全免费，且登录即开**，无需购买、无需课程进度门槛：

| 游戏 | 解锁条件 |
|------|---------|
| 游戏 1-12 | **登录即可解锁全部 12 款** |

> 理由：课程是付费主阵地，游戏是留存+裂变主阵地；不把游戏作为课程付费门槛。玩家可直接挑战高 Level 游戏，遇到难题时由系统推荐回对应课程。

### 5.2 变现方式

| 来源 | 说明 |
|------|------|
| **皮肤商城** | 游戏皮肤/特效/音效（纯装饰，不影响游戏平衡）|
| **赛季通行证** | 30 天限定皮肤链 + 专属段位边框 + 额外积分/知语币 |
| **知语币消费** | 通过推荐获得的知语币可直接在皮肤商城等价兑换 |

> 游戏本身不收费，不卖游戏解锁。皮肤和特效是主要的游戏侧收入来源。

### 5.3 参考定价（USD）

| 商品 | 参考价 |
|------|-------|
| 基础皮肤 | $0.99 |
| 稀有皮肤 | $1.99 – $2.99 |
| 特效包 | $0.99 – $1.99 |


> 所有皮肤均可用积分或**知语币**兑换，也可直接付费购买。知语币与现金保持 1:1 等价关系（1 币 = $0.01），不可提现。

## 七、技术实现要点

### 7.1 2D 游戏引擎

| 方案 | 说明 |
|------|------|
| **PixiJS** | 轻量级 2D 渲染引擎，Web 原生，性能好 |
| **Phaser 3** | 成熟的 HTML5 游戏框架，适合 2D 游戏 |
| **选择** | 推荐 **Phaser 3**（生态好、文档全、适合多类型 2D 游戏） |

### 7.1.1 横屏适配

所有 12 款游戏强制**横屏（landscape）**模式，同时支持 Web 桌面和移动端 H5。

| 维度 | 方案 |
|------|------|
| 屏幕方向 | Phaser `scale.orientation = LANDSCAPE`，移动端自动提示旋转 |
| 分辨率 | 设计基准 1920×1080，Phaser ScaleManager 自适应缩放 |
| 移动端 H5 | 全屏 API + viewport meta 强制横屏，支持触控操作 |
| Web 桌面 | 自适应浏览器窗口，保持 16:9 比例 |
| 未来 App | Capacitor 内嵌 WebView，锁定横屏 |

### 7.2 实时对战

```
客户端 ←→ WebSocket ←→ 对战服务器（Node.js）
                              ├── 匹配系统
                              ├── 房间管理
                              ├── 状态同步
                              └── 结算服务
```

- 使用 WebSocket 实现实时通信
- 匹配服务用 Redis 做队列和段位匹配计算
- 对战回放存储在 Cloudflare R2

### 7.3 防作弊

| 措施 | 说明 |
|------|------|
| 服务端出题 | 题目由服务端下发，客户端不预加载答案 |
| 答题时间校验 | 过快的答题（<0.3 秒）标记为异常 |
| 分数服务端计算 | 客户端只提交答案，分数由服务端算 |
| 对战同步校验 | 双方答题时间戳比对 |

# 运营与推广策略

> 知语 Zhiyu 用户获取、品牌建设与运营体系
> 版本：V1.0 | 日期：2026-04-17

---

## 一、运营策略总览

### 1.1 核心逻辑

```
内容获客（低成本）→ 产品留存（体验好）→ 口碑裂变（自增长）→ 付费转化（变现）
```

### 1.2 阶段策略

| 阶段 | 时间 | 重心 | 目标 |
|------|------|------|------|
| **冷启动** | 第 1-2 月 | 种子用户积累 + 口碑验证 | 500 核心用户 |
| **增长期** | 第 3-6 月 | 内容营销 + KOL 合作 | 10,000 MAU |
| **加速期** | 第 7-12 月 | 付费投放 + 渠道扩展 | 50,000 MAU |
| **成熟期** | 12 月后 | 品牌建设 + 新市场拓展 | 100,000+ MAU |

## 四、KOL / KOC 合作

### 4.1 合作策略

| 类型 | 定义 | 合作方式 | 预算 |
|------|------|---------|------|
| **头部 KOL** | 粉丝 50 万+ | 品牌合作视频 | ¥3,000-10,000/条 |
| **中腰部 KOL** | 粉丝 5-50 万 | 产品测评/教程 | ¥500-3,000/条 |
| **KOC（素人）** | 粉丝 < 5 万 | 真实体验分享 | 免费课程解锁 + 少量报酬 |
| **中文老师** | 教育类创作者 | 课程推荐/联名 | 分成模式 |

### 4.2 合作对象画像

| 类型 | 平台 | 内容方向 |
|------|------|---------|
| 越南中文 KOL | Facebook / TikTok / YouTube | 在越南教中文的老师、学中文的学生 |
| 文化类博主 | YouTube / Instagram | 介绍中国文化、旅游、美食 |
| 教育类创作者 | TikTok / YouTube | 语言学习方法论 |
| 在越华人 | Facebook / Zalo | 中越文化交流 |

### 4.3 效果跟踪

每位 KOL 分配唯一推荐链接，跟踪：
- 点击量 → 注册量 → 付费转化 → ROI
- 按效果评估续约合作

## 八、每日金句模块（Daily Quote）

> 目标：提升日活跃度（DAU）和分享率，增强用户每日打开 App 的理由。

### 8.1 产品形态

| 维度 | 说明 |
|------|------|
| **位置** | App 首页顶部卡片，打开 App 即可看到 |
| **内容** | 一句中文金句 + 拼音 + 英文翻译 + 下方一段 2-3 句的解读/故事 |
| **更新频率** | 每天 0:00 UTC 刷新，全球统一 |
| **数据来源** | 后台数据库预写入，运营人员提前排期 |
| **节日特别** | 节日当天自动切换为**节日主题金句**（春节/元宵/清明/端午/中秋/重阳/冬至等） |
| **分享** | 金句卡片支持一键生成精美图片 → 分享到社交媒体（自带品牌水印 + 二维码） |

### 8.2 金句内容分类

| 类别 | 举例 | 占比 |
|------|------|------|
| **古诗词名句** | "海内存知己，天涯若比邻" | 25% |
| **经典哲学** | "千里之行，始于足下" — 老子 | 20% |
| **成语故事** | "画龙点睛" — 关键的一笔让整件事活起来 | 20% |
| **现代名言** | "世界上怕就怕认真二字" | 15% |
| **节日专题** | "但愿人长久，千里共婵娟"（中秋） | 10% |
| **学习激励** | "学而时习之，不亦说乎" | 10% |

### 8.3 分发策略

| 阶段 | 策略 | 说明 |
|------|------|------|
| **Phase 1（上线）** | 全局统一 | 所有用户每天看到相同金句，运营成本低 |
| **Phase 2（6 个月后）** | 用户画像分发 | 根据用户 HSK 等级推送不同难度金句 |
| **Phase 3（12 个月后）** | 个性化推荐 | 结合用户学习偏好（诗词/成语/哲学）智能分发 |

### 8.4 运营价值

| 指标 | 预期提升 |
|------|---------|
| DAU | +15-20%（"今日金句"成为每日打开理由） |
| 分享率 | +30%（精美金句卡片天然适合社交分享） |
| D1 留存 | +5%（次日打开看新金句） |
| 品牌传播 | 分享图片自带品牌水印 + 下载二维码 → 免费获客 |

### 8.5 技术要点

- 后台 CMS 管理金句库，支持提前 30 天排期
- 节日日历自动匹配节日金句（支持中国传统节日 + 国际节日）
- 分享图片生成：服务端渲染（模板 + 金句 + 用户昵称 + 品牌水印 + 二维码）
- Phase 2 起接入用户画像标签系统

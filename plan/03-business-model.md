# HanYu — 商业模式与收费标准

> 会员卖游戏体验，AI 按量透传，不靠套壳赚钱
> 版本：v1.0 | 日期：2026-04-15

---

## 一、商业模式核心逻辑

### 1.1 收入结构（成熟期目标）

```
会员订阅（游戏内容/皮肤/装饰/特权）   60%   ← 核心收入，边际成本趋零
B2B 企业培训                          20%   ← 第二增长曲线
广告收入（免费用户）                    10%   ← 补充收入
AI 按量付费（透传成本，不赚利润）         5%   ← 赚口碑和用户信任
增值服务（HSK 考试包/证书/赛季通行证）   5%   ← 补充收入
```

### 1.2 为什么这个结构比"AI 订阅"更健康

| 维度 | AI 订阅模式 | 我们的模式 |
|------|------------|-----------|
| **边际成本** | 用户越多 → API 成本线性增长 | 游戏内容一次制作 → 边际成本趋零 |
| **毛利率** | 40-60%（AI API 成本高） | 80-90%（内容分发无额外成本） |
| **用户增长的成本压力** | 每多一个活跃用户都增加成本 | 用户增长几乎不增加内容成本 |
| **被替代风险** | 大模型降价/免费 → 失去价值 | 游戏内容+社区不可替代 |
| **定价灵活性** | 必须覆盖 API 成本，价格有下限 | 可以极低定价抢市场 |

## 二、个人订阅（B2C）

### 2.1 免费版 vs 会员版对比

| 功能 | 免费版 | 会员版 |
|------|--------|--------|
| **基础关卡** | 拼音群岛 + 日常平原前 50% | 全部领地全部关卡 |
| **PvP 对战** | 每日 5 场 | 无限制 |
| **小游戏** | 基础 3 款 | 全部小游戏 |
| **过场动画** | 基础剧情线 | 全部剧情线 |
| **皮肤/装饰** | 5 套基础皮肤 | 全部常规皮肤 + 每月限定 |
| **称号** | 军衔称号 | 军衔 + 成就 + 自定义称号 |
| **边框** | 段位边框 | 段位 + 成就 + 限定边框 |
| **宠物/坐骑** | 无 | 会员专属宠物 |
| **排行榜** | 可查看 | 显示会员标识 + 专属特效 |
| **赛季通行证** | 免费线 | 免费线 + 付费线 |
| **每月钻石赠送** | 无 | 300 钻石/月 |
| **AI 每日免费额度** | 10 HanYu Coins/天 | 30 HanYu Coins/天 |
| **广告** | 课间插页广告 | 无广告 |
| **离线** | 不支持 | 已下载关卡可离线 |

### 2.2 定价方案

| 套餐 | 越南市场 | 印尼市场 | 月均成本 | 占月收入比 |
|------|---------|---------|---------|-----------|
| **免费版** | $0 | $0 | — | — |
| **月付** | $2.99/月 | $1.99/月 | $2.99/$1.99 | <1% |
| **季付** | $6.99/季 | $4.99/季 | $2.33/$1.66 | <0.7% |
| **年付** | $19.99/年 | $14.99/年 | $1.67/$1.25 | <0.5% |
| **终身** | $49.99 | $39.99 | — | 一次性 |

**定价策略**：
- **心理锚点**：$2.99/月 = 越南一杯奶茶的钱。"每天不到 1 毛钱"
- **年付推荐**：UI 默认展示年付，月付显示"最贵"标签
- **首月 $0.99**：首次付费用户享受首月特价，降低决策门槛
- **终身制造紧迫感**："限时优惠 $49.99，原价 $79.99"

### 2.3 付费转化设计

**免费用户的"甜蜜痛点"**：
```
痛点 1 - 关卡锁定
  · 玩到日常平原 50% 时，剩余关卡需要会员解锁
  · 用户已经投入时间建立了进度 → 沉没成本驱动付费

痛点 2 - PvP 次数限制
  · 免费用户每天 5 场 PvP
  · 打到黄金段位时，5 场不够用 → "想继续打，得开会员"

痛点 3 - 皮肤/装饰吸引
  · 看到会员用户的华丽皮肤和特效 → 社交攀比驱动

痛点 4 - 广告打断
  · 每 3 个关卡之间插入 15 秒广告 → 影响游戏节奏

痛点 5 - 赛季限定
  · 赛季限定皮肤只有会员付费线才有 → FOMO（错过恐惧）
```

**不靠限制 AI 来逼付费**（核心差异）：
- 免费用户每天也有 10 Coins 的 AI 免费额度
- AI 功能是增值体验，不是核心游戏流程的一部分
- 即使完全不用 AI 功能，游戏本身的关卡和 PvP 也足够有趣

## 三、AI 按量付费（HanYu Coins）

### 3.1 定价原则

**铁律：平台不在 AI 功能上赚差价。**

HanYu Coins 的定价严格基于实际 API 成本 + 5% 运营费（服务器、通信开销）。

### 3.2 HanYu Coins 价格体系

```
充值价格：
  · $0.99 = 100 Coins（最小充值）
  · $4.99 = 550 Coins（赠 50）
  · $9.99 = 1200 Coins（赠 200）

每日免费额度：
  · 免费用户：10 Coins/天（约 2-3 次 AI 对话）
  · 会员用户：30 Coins/天（约 8-10 次 AI 对话）
```

### 3.3 各 AI 功能消耗

| 功能 | 消耗 Coins | 对应 API 成本 | 说明 |
|------|-----------|-------------|------|
| AI 中文对话（1轮） | 3 Coins | ~$0.002 | GPT-4o-mini，约 1500 tokens |
| AI 发音评估（1次） | 2 Coins | ~$0.001 | Azure Speech |
| AI 作文批改（1篇） | 8 Coins | ~$0.005 | GPT-4o-mini，约 3000 tokens |
| AI 语法解释（1次） | 3 Coins | ~$0.002 | GPT-4o-mini |
| AI 场景模拟（1场） | 10 Coins | ~$0.008 | 多轮对话，约 5000 tokens |

### 3.4 透明计费的用户价值

```
为什么透明计费是竞争优势：

1. 信任建立
   · "我们不赚 AI 的差价" → 用户信任 → 愿意为其他功能付费

2. 消除焦虑
   · 竞品：$29.99/月的会员，用户担心"我用不了这么多"
   · 我们：用多少付多少，$0.99 可以用很久

3. 口碑传播
   · "这个 App 的 AI 功能不赚钱" → 社交媒体自发传播

4. 差异化
   · 市场上所有竞品都在"AI 功能 = 高价会员"
   · 我们打破这个逻辑 → 鲜明的品牌记忆点
```

## 四、B2B 企业培训

### 4.1 目标客户

| 类型 | 企业规模 | 需求场景 | 预算 |
|------|---------|---------|------|
| 在越南/印尼的中国制造企业 | 100-5000 人 | 产线工人基础中文 | $200-1000/月 |
| 中国建筑企业海外项目部 | 50-500 人 | 工程沟通中文 | $200-500/月 |
| 中国零售/餐饮海外门店 | 20-200 人 | 服务场景中文 | $100-300/月 |
| 东南亚本地企业（与中国有贸易） | 不限 | 商务中文 | $200-500/月 |

### 4.2 B2B 产品方案

```
企业版功能（在个人版基础上新增）：

1. 管理后台
   · 批量创建员工账号
   · 查看全员学习进度仪表盘
   · 设置学习任务和截止日期
   · 导出学习报告（PDF/Excel）

2. 行业定制内容包
   · 制造业词汇包：设备名称、操作指令、安全规范
   · 建筑业词汇包：施工术语、材料名称
   · 零售业词汇包：客服用语、商品名称
   · 定制包由 AI 生成 + 行业专家审校

3. 培训证书
   · 完成指定课程后颁发数字证书
   · 证书可验证（含二维码）
   · 对接 HSK 考试大纲

4. 企业排行榜
   · 部门间排名
   · 月度学习之星
   · 激励员工学习积极性
```

### 4.3 B2B 定价

| 套餐 | 价格 | 人数上限 | 包含内容 |
|------|------|---------|---------|
| **入门包** | $99/月 | 30 人 | 全功能 + 管理后台 + 基础报告 |
| **标准包** | $299/月 | 100 人 | + 行业定制内容 + 证书 |
| **专业包** | $599/月 | 300 人 | + 专属客服 + 详细报告 |
| **企业包** | $999/月 | 1000 人 | + 定制课程开发 + API 集成 |
| **定制** | 面议 | 不限 | 完全定制 |

### 4.4 B2B 销售策略

```
渠道优先级：

1. 中国商会（越南中国商会、印尼中国商会）
   → 一次接触多家企业

2. LinkedIn 找 HR/行政负责人
   → 精准触达决策者

3. 中国出海企业微信群
   → 口碑传播

4. 地推：工业园区张贴海报/派发传单
   → 成本低，覆盖面广

销售话术核心：
"你的越南/印尼员工会说中文 = 减少沟通错误 = 减少产线事故 = 省钱"
"员工学中文不是福利，是降低管理成本的投资"
```

## 五、广告收入

### 5.1 广告形式

| 形式 | 场景 | 频率 | 预估 eCPM |
|------|------|------|----------|
| **插页广告** | 每 3 个关卡之间 | ~每 10 分钟 1 次 | $0.5-2.0 |
| **激励视频** | 看广告获得金币/体力/Streak Freeze | 用户主动触发 | $2.0-5.0 |
| **Banner 广告** | 非游戏页面底部 | 持续展示 | $0.3-0.8 |

### 5.2 广告收入预估

```
免费用户日均使用时长：15 分钟
每次会话广告展示：
  · 插页广告：1-2 次（eCPM $1.0 平均）
  · 激励视频：1 次（eCPM $3.0）
  · Banner：持续展示（eCPM $0.5）

单个免费用户日均广告收入：
  · 插页：$0.001-0.002
  · 激励视频：$0.003
  · Banner：$0.0005
  · 合计：约 $0.004-0.006/天 = $0.12-0.18/月

广告平台：Google AdMob（覆盖东南亚最好）
```

## 八、单位经济（Unit Economics）

| 指标 | 数值 | 说明 |
|------|------|------|
| **CAC** | $0.3-1.0 | 裂变+自然流量可压到 $0.3 |
| **ARPU（付费用户）** | $2.0-3.0/月 | 加权平均（月付+年付） |
| **内容边际成本** | ~$0/用户 | 游戏内容一次制作，无限分发 |
| **AI 功能边际成本** | ~$0.05/月/活跃用户 | 免费额度的 API 成本 |
| **服务器边际成本** | ~$0.01/月/用户 | 静态资源 CDN 化后极低 |
| **毛利率** | 85-90% | 核心优势：内容的零边际成本 |
| **LTV** | $15-30 | 假设平均付费 8-12 个月 |
| **LTV/CAC** | 15-60x | 远超健康线（>3x） |
| **付费回收期** | <1 个月 | 首月即回收获客成本 |

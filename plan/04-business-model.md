# PlayLingo — 商业模式与收费标准

> 会员卖游戏体验，AI 按量透传，分享赚佣金
> 版本：v2.0 | 日期：2026-04-15

---

## 一、商业模式核心逻辑

### 1.1 收入结构（成熟期目标）

```
会员订阅（游戏内容/皮肤/装饰/特权）   60%   ← 核心收入，边际成本趋零
B2B 企业培训                          20%   ← 第二增长曲线
广告收入（免费用户）                    10%   ← 补充收入
AI 按量付费（透传成本，不赚利润）         5%   ← 赚口碑和用户信任
增值服务（HSK 考试包/证书/赛季通行证）   5%   ← 补充收入
```

### 1.2 成本结构优势

| 维度 | AI 订阅模式（竞品） | 我们的模式 |
|------|------------|-----------|
| **边际成本** | 用户越多 → API 成本线性增长 | 游戏内容一次制作 → 边际成本趋零 |
| **毛利率** | 40-60% | 80-90% |
| **被替代风险** | 大模型降价/免费 → 失去价值 | 游戏内容+社区不可替代 |
| **定价灵活性** | 必须覆盖 API 成本，价格有下限 | 可以极低定价打价格战 |

## 二、个人订阅（B2C）

### 2.1 免费版 vs 会员版

| 功能 | 免费版 | 会员版 |
|------|--------|--------|
| **基础关卡** | 拼音群岛全部 + 其他地图前 50% | 全部地图全部关卡 |
| **小游戏** | 3 款基础小游戏 | 全部小游戏 |
| **过场动画** | 基础剧情线 | 全部剧情线 |
| **PvP 对战**（v1.1） | 每日 5 场 | 无限制 |
| **皮肤/装饰**（v1.1） | 基础皮肤 | 全部常规皮肤 + 每月限定 |
| **每月钻石赠送** | 无 | 300 钻石/月 |
| **AI 每日免费额度**（v1.1） | 10 Coins/天 | 30 Coins/天 |
| **广告** | 关卡间插页广告 | 无广告 |
| **间隔复习** | 完整功能 | 完整功能 |
| **离线** | 不支持 | 已下载关卡可离线 |
| **词库导出** | 不支持 | 支持导出 |

### 2.2 定价方案

| 套餐 | 越南市场 | 印尼市场 | 月均成本 |
|------|---------|---------|---------|
| **免费版** | $0 | $0 | — |
| **月付** | $2.99/月 | $1.99/月 | $2.99/$1.99 |
| **季付** | $6.99/季 | $4.99/季 | $2.33/$1.66 |
| **年付** | $19.99/年 | $14.99/年 | $1.67/$1.25 |
| **终身** | $49.99 | $39.99 | 一次性 |

**定价策略**：
- **心理锚点**：$2.99/月 = 越南一杯奶茶。"每天不到 1 毛钱"
- **年付优先推荐**：UI 默认展示年付，月付标注"最贵"
- **首月 $0.99**：首次付费用户享首月特价，降低决策门槛
- **终身制造紧迫感**："限时优惠 $49.99，原价 $79.99"

### 2.3 付费转化设计

```
免费用户的"甜蜜痛点"设计：

痛点 1 - 关卡锁定
  · 拼音群岛免费开放（让用户体验完整的入门阶段）
  · 汉字谷地及以后的地图，后 50% 关卡需要会员
  · 用户已投入时间建立进度 → 沉没成本驱动付费

痛点 2 - 广告打断
  · 每 3 个关卡之间插入 15 秒广告
  · 影响游戏节奏 → 会员去广告

痛点 3 - PvP 次数限制（v1.1）
  · 免费用户每天 5 场 PvP
  · 打到黄金段位时，5 场不够 → "想继续打，得开会员"

痛点 4 - 皮肤/装饰吸引（v1.1）
  · 看到会员用户的华丽皮肤 → 社交攀比驱动

不靠限制 AI 来逼付费：
  · 免费用户也有 AI 免费额度
  · AI 是增值体验，不是核心流程
```

## 三、分享赚佣金机制（一级分销）

### 3.1 机制概述

**核心理念**：不只是"分享得会员"，而是"分享赚钱"——让用户成为你的分销渠道。

```
运作方式：
  A 分享专属推广链接（含唯一用户标识 ?ref=uid）
  → B 通过链接注册，系统自动绑定 A-B 推荐关系（无需手动输入邀请码）
  → B 付费 → A 获得 B 每笔付费的 X% 现金佣金
  → B 再分享给 C，C 的付费只给 B 佣金（与 A 无关）
  → 仅一级，不构成多层分销

关键：
  · 链接注册 = 自动绑定，零摩擦
  · A 推荐 B → A 赚钱
  · B 推荐 C → B 赚钱（A 不赚 C 的钱）
```

### 3.2 佣金阶梯（推荐越多，提点越高）

| 推广等级 | 累计本账户及推荐用户付费金额 | 佣金比例 
|---------|----------------|---------|
| **初级推广员** | $0 - 9.99 | 5% | 
| **高级推广员** | $10 - 99.99 | 10% |
| **金牌推广员** | $100 - 999.99 | 15% | 
| **超级推广员** | $1000 - 9999.99 | 20% | 

**佣金计算示例**：
- B 购买会员 $3.00 → A 在"初级"阶段（5%）→ A 获得 $0.15
- B 购买皮肤 $20.00 → A 在"高级"阶段（10%）→ A 获得 $2.00

### 3.3 用户端展示方案

```
推广中心页面（App 内"我的推广"入口）：

┌─────────────────────────────┐
│  我的推广                      │
│                               │
│  当前等级：高级推广员 ★★        │
│  佣金比例：10%                 │
│  距下一级：还需推荐消费 $180.26  │
│                               │
│  ┌─────────────────────────┐ │
│  │ 累计推荐消费：$ ...       │ │
│  │ 累计收益：$ ...         │ │
│  │ 可提现：$ ...            │ │
│  │ [立即提现]               │ │
│  └─────────────────────────┘ │
│                               │
│  [生成推广海报]  [复制推广链接]      │
│                               │
│  最近推荐明细                   │
│  · Nguyen V. 4/12 月付 → +$0.45│
│  · Tran H.  4/10 年付 → +$3.00│
│  · Tran H.  4/10 皮肤 → +$12.00                     │
│                               │
│  佣金规则说明 >                 │
└─────────────────────────────┘

提现规则：
  · 最低提现：$10
  · 提现方式：MoMo / 银行转账
  · 结算周期：被推荐人付费 30 天后佣金可提现
  · 退款处理：被推荐人在 30 天内退款，则佣金取消
```

### 3.4 合规设计

```
为什么合规：

1. 仅一级分销，不存在多级关系
   → 不符合传销/金字塔定义（越南竞争法明确禁止非法多层分销）
   → 类似 Uber/Grab 的推荐奖励机制

2. 佣金基于真实商品交易
   → 只有被推荐人实际付费后才计佣
   → 不奖励"纯拉人头"

3. 设有防作弊机制
   → 设备指纹检测、IP 检测、支付工具关联检测
   → 30 天冷却期防刷

4. 透明披露
   → 用户协议中明确佣金规则
   → App 内展示完整的佣金计算方式
   → 用户可随时查看明细和申请提现

5. 税务合规
   → 超过一定金额提醒用户报税
   → 平台保留交易记录供审计

6. 退出机制
   → 如遇当地法律政策变化，可随时将佣金机制降级为"推荐送会员"模式
   → 系统设计支持热切换（佣金 ↔ 会员奖励）
```

## 四、AI 按量付费（PlayLingo Coins）

### 4.1 定价原则

**铁律：平台不在 AI 功能上赚差价。** Coins 定价 = API 实际成本 + 5% 运营费。

### 4.2 价格体系

```
充值价格：
  · $0.99 = 100 Coins（最小充值）
  · $4.99 = 550 Coins（赠 50）
  · $9.99 = 1200 Coins（赠 200）

每日免费额度：
  · 免费用户：10 Coins/天（约 2-3 次 AI 对话）
  · 会员用户：30 Coins/天（约 8-10 次 AI 对话）
```

### 4.3 各功能消耗

| 功能 | 消耗 Coins | 对应 API 成本 |
|------|-----------|-------------|
| AI 中文对话（1轮） | 3 Coins | ~$0.002 |
| AI 发音评估（1次） | 2 Coins | ~$0.001 |
| AI 作文批改（1篇） | 8 Coins | ~$0.005 |
| AI 语法解释（1次） | 3 Coins | ~$0.002 |

## 五、广告收入

### 5.1 广告形式

| 形式 | 场景 | 频率 | 预估 eCPM |
|------|------|------|----------|
| **插页广告** | 每 3 个关卡之间 | ~每 10 分钟 | $0.5-2.0 |
| **激励视频** | 看广告获得金币/Streak Freeze | 用户主动触发 | $2.0-5.0 |
| **Banner** | 非游戏页面底部 | 持续展示 | $0.3-0.8 |

**广告平台**：Google AdMob（东南亚覆盖最好）

### 5.2 单用户广告收入

```
免费用户日均使用 15 分钟：
  · 插页：$0.001-0.002
  · 激励视频：$0.003
  · Banner：$0.0005
  · 合计：~$0.005/天 = ~$0.15/月
```

## 九、单位经济

| 指标 | 数值 | 说明 |
|------|------|------|
| **CAC** | $0.3-1.0 | 裂变+佣金机制可压到 $0.3 |

| **服务器边际成本** | ~$0.01/月/用户 | CDN 化后极低 |
| **毛利率** | 85-90% | 核心优势 |
| **LTV/CAC** | 20-40x | 远超健康线 3x |

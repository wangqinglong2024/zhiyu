### 2.1 佣金阶梯与动力

```
推广等级体系：

| 推广等级 | 累计本账户及推荐用户付费金额 | 佣金比例

| **初级推广员** | $0 - 9.99 | 5% | 
| **高级推广员** | $10 - 99.99 | 10% |
| **金牌推广员** | $100 - 999.99 | 15% |

用户心理：
  "推荐别人学中文，自己每月都有被动收入"
  "达到金牌推广，佣金率比上班时薪还高"

```

```
场景 4：课程分享
  触发：用户学完一个主题
  呈现："把这个课程推荐给朋友，他们 7 折，你赚佣金"
  → 生成特定课程的推荐链接
```

## 三、社交裂变机制

### 3.1 PvP 对战裂变

```
设计逻辑：
  对战 → 赢了想炫耀 → 输了想复仇 → 都想拉好友

具体流程：
  1. 用户 A 发起 PvP 匹配
  2. 匹配队列中对战（或邀请好友专场）
  3. 对战结束，双方看到结果 + 分享按钮
  4. "邀请好友对战！好友通过链接注册自动绑定推荐关系"
  5. 好友加入 → 对战圈扩大 → 更多人加入

裂变倍数估计：
  · 每个 PvP 玩家平均邀请 2-3 个好友
  · 好友转化率 ~30%
  · 有效 K-factor ≈ 0.6-0.9
```

### 3.2 荣誉分享裂变

```
可分享成就：
  · 学习连续 30 天打卡
  · 段位升级（青铜→白银→黄金...）
  · 完成主题世界（如"拼音群岛通关"）
  · 词汇量里程碑（100/500/1000 词）
  · PvP 排名（周榜/月榜进入前 100）
  · 佣金里程碑（$10/$50/$100）

分享模板（自动生成）：
┌────────────────────────────┐
│  🎮 PlayLingo              │
│                            │
│  [精致游戏画面截图]          │
│                            │
│  Tôi đã học được 500       │
│  từ tiếng Trung!           │
│  （我已经学了500个中文词！）  │
│                            │
│  Day 30 🔥 连续打卡          │
│                            │
│  👇 点击链接首月 7 折！      │
│  playlingo.app/r/XXXX      │
└────────────────────────────┘
```

### 3.3 组队学习裂变

```
学习小组（v2.0 功能）：
  · 3-5 人组队，每周学习任务
  · 全队完成奖励宝箱
  · 队长邀请链接（自动绑定推荐关系）
  · 组队 PvP（团队对团队）

裂变路径：
  用户创建小组 → 邀请 2-4 好友 → 好友注册 → 组队学习 → 好友还会邀请
```

## 四、游戏化增长策略

### 4.1 军衔段位系统

```
段位体系：
  列兵 → 下士 → 中士 → 上士 →
  少尉 → 中尉 → 上尉 →
  少校 → 中校 → 上校 →
  准将 → 少将 → 中将 → 上将 → 元帅

升段条件：
  · PvP 积分累计
  · 不活跃 30 天自动降段
  → 防流失：掉段前 3 天推送提醒

分享动力：
  · 升段动画 + 一键分享
  · 段位显示在个人主页和排行榜
  · 高段位有专属头像框和称号
```

### 4.2 连续打卡机制

```
streak 系统：
  · 每日完成 1 关 = 打卡成功
  · 连续 7 天 → 奖励宝箱（随机皮肤/金币）
  · 连续 30 天 → 高级奖励 + 分享卡片
  · 连续 100 天 → 传说称号 + 永久装饰
  · 断签可用金币修复（1 次/月免费）

推送策略：
  · 每日固定时间打卡提醒（用户自定义时间）
  · 即将断签提醒："还差 2 小时，别丢了 15 天连续打卡！"
  · streak 里程碑庆祝推送
```

### 4.3 限时活动

```
每周活动：
  · 周末 PvP 锦标赛（奖励双倍金币）
  · 主题挑战（如"一周掌握餐厅用语"）

每月活动：
  · 月度排行争夺赛（Top 10 奖励高级皮肤）
  · 新内容开放活动（新世界探索首杀奖励）

节日活动（越南历法）：
  · 春节（Tết）：特殊春节课程 + 红包奖励
  · 中秋节：灯谜答题挑战
  · 教师节：教师推广员专属奖励翻倍
```

## 五、用户生命周期管理

### 5.1 新手引导优化

```
Day 0（注册日）：
  · 选择学习目标（工作/旅游/兴趣/考试）
  · 快速能力测试（3 分钟）
  · 进入第 1 关（拼音声母 b/p/m/f）
  · 完成第 1 关后立即正反馈（"你已经会说 4 个中文音了！"）
  · 引导设置每日打卡提醒

Day 1-3（激活期）：
  · 推送："继续昨天的学习，今天学声调"
  · 完成 3 天打卡后引导 PvP 体验
  · 展示佣金推广入口

Day 4-7（转化窗口期）：
  · 免费内容将尽提示
  · 限时 7 折首月优惠
  · 推荐好友可延长免费期
  · 展示推广员收益案例
```

### 5.2 流失预警与召回

```
流失信号：
  · 2 天未登录 → 低风险
  · 5 天未登录 → 中风险
  · 10 天未登录 → 高风险
  · 30 天未登录 → 流失

召回策略：
  2 天：推送"你的连续打卡快断了！"
  5 天：推送"上次学到拼音 zh/ch/sh，继续吧"
  10 天：推送"回来续学，送你 1 天高级体验"
  30 天：邮件/短信"我们想你了，限时 5 折回归优惠"
  60 天：推荐人收到通知"你推荐的 XX 快流失了，提醒 TA 回来"
```

### 5.3 付费用户续费策略

```
续费提醒节点：
  · 到期前 7 天：展示学习报告 + 续费优惠
  · 到期前 3 天：推送"即将失去高级功能"
  · 到期前 1 天：最后提醒 + 对比免费版限制
  · 到期当天：降级提示 + 限时恢复按钮

年付引导：
  · 月付 $2.99 vs 年付 $29.99（节省 17%）
  · 连续 3 个月付费后推荐年付
  · 年付用户专属称号和装饰
```

## 六、病毒系数优化

### 6.1 K-factor 目标

```
K = 邀请数 × 转化率

目标分解：
  · 每个活跃用户月均发出 3 次邀请
  · 邀请点击率 40%
  · 注册转化率 50%
  · K-factor = 3 × 0.4 × 0.5 = 0.6

加上佣金激励后：
  · 推广员月均邀请 10+ 次
  · 整体 K-factor 目标 0.8-1.0

K > 1.0 时（病毒增长）：
  · 不需要额外投放也能自然增长
  · 触发条件：推广员占 DAU >5% 且人均邀请 >10/月
```

### 6.2 优化手段

```
提高邀请数：
  ✓ 每次成就都弹出分享入口
  ✓ 佣金实时推送刺激再分享
  ✓ PvP 自带社交邀请属性
  ✓ 降低分享门槛（一键生成分享素材）

提高转化率：
  ✓ 被推荐人首月 7 折
  ✓ 落地页展示推荐人的学习成就
  ✓ 新用户注册后立即进入游戏（不是教程墙）
  ✓ 推广链接自动绑定推荐关系（URL 参数 ?ref=uid，注册即绑定，零摩擦）
```

## 七、数据驱动增长

### 7.1 核心追踪指标

| 指标 | 定义 | 目标 |
|------|------|------|
| K-factor | 每用户带来的新用户数 | >0.8 |
| 邀请点击率 | 邀请链接点击/发送 | >40% |
| 注册转化率 | 注册/落地页访问 | >50% |
| 佣金用户占比 | 有推广收入的用户/总用户 | >5% |
| 推广员月均推荐 | 活跃推广员月均有效推荐数 | >5 人 |
| 分享率 | 每日分享次数/DAU | >15% |
| PvP 邀请转化 | PvP 邀请后注册率 | >30% |

### 7.2 A/B 测试计划

```
优先级 1（MVP 后立即测试）：
  · 分享弹窗时机（关卡完成 vs 获得成就 vs 升级）
  · 佣金展示方式（金额 vs 百分比 vs 对比）
  · 落地页设计（游戏画面 vs 学习效果 vs 价格对比）

优先级 2：
  · 首月折扣力度（7折 vs 5折 vs 免费）
  · 推广物料模板（简约 vs 游戏截图 vs 收益截图）
  · 通知推送时间和话术
```

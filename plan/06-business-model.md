> 知语 Zhiyu 变现体系与定价策略
> 版本：V1.0 | 日期：2026-04-17

## 一、收入结构总览

知语 Zhiyu 采用 **“免费体验 + 单级购买 + 皮肤变现”** 三层漏斗。L1-L3 免费（含完整考核）→ L4-L12 按级购买 $6/级 → 游戏皮肤/特效增值。推广激励不使用现金返佣——采用**平台虚拟币“知语币”**，合规风险极低，长期绑定用户。

```
收入结构
  ├── 课程单级购买（核心收入，占比 75-85%）
  │     └── L4-L12 共 9 级 × $6 = $54（3 年有效期）
  ├── 游戏皮肤 / 特效（占比 10-20%）
  └── 知语币虚拟币体系（驱动增长，不形成现金支出）
```

### 核心策略

| 策略 | 说明 |
|------|------|
| **游戏登录即玩** | 12 款游戏对所有已登录用户完全开放，无课程进度门槛 |
| **免费 3 级体验** | L1-L3 课程完全免费，包含完整学习内容 + 升级考核，让用户充分体验产品价值 |
| **按级付费** | L4-L12 每级 $6，按需购买，不强制一次性购买全部，降低付费门槛 |
| **3 年有效期** | 每个购买的课程 3 年内可无限学习，到期需续购，平衡用户权益与长期收入 |
| **皮肤变现** | 游戏皮肤和特效是游戏侧唯一付费项，纯装饰不影响平衡 |
| **虚拟币代替现金返佣** | 推广奖励以知语币发放，闭环消费，规避越南跨境支付合规风险 |

## 二、三级访问权益对比

### 2.1 权益详情

| 功能模块 | 未登录访客 | 登录用户（免费） | 付费用户（按级购买） |
|---------|----------|----------------|------|
| **发现中国** | 前 3 个类目 | ✅ 全部 12 个类目 + 全部内容 | ✅ 全部 12 个类目 + 全部内容 |
| **课程 L1-L3** | ❌ 跳转登录 | ✅ 完整课程 + 升级考核 | ✅ 完整课程 + 升级考核 |
| **课程 L4-L12** | ❌ 跳转登录 | ❌ 付费墙（$6/级） | ✅ 已购买的等级（3 年有效） |
| **游戏 12 款** | ❌ 跳转登录 | ✅ **全部 12 款** | ✅ **全部 12 款** |

### 2.2 设计逻辑

- **未登录 → 登录**：靠"9 个隐藏类目 + 全部游戏 + L1-L3 课程"的解锁吸引力驱动注册。
- **免费用户 → 付费**：L3 学完后，L4 付费墙是核心转化触点。L1-L3 自带完整考核，用户已体验到产品价值。
- **持续付费**：逐级解锁 L4→L5→…→L12，每次 $6，用户按需购买，无压力。
- **3 年有效期**：给用户充足学习时间，同时保证长期收入可持续。

## 三、课程定价体系（USD 全球统一）

### 3.1 定价方案

| 课程等级 | 定价（USD）| 有效期 | 说明 |
|---------|----------|--------|------|
| **L1-L3** | **免费** | 永久 | 注册即可学习，包含完整内容 + 升级考核 |
| **L4-L12**（每级） | **$6 / 级** | **3 年** | 单级解锁，包含该级全部课程内容 + 升级考核 |
| **L4-L12 全部**（9 级） | **$54** | **3 年** | 等同于逐级购买，无额外折扣 |

> **定价策略**：$6/级 击穿东南亚付费门槛（约 ₫150,000）。9 级合计 $54 远低于 Duolingo Super 年费（$84-168/年）、HelloChinese Premium（$108/年）。一次性购买 3 年有效，无续订焦虑。

### 3.2 免费课程（L1-L3）详情

L1-L3 是完整的课程体验，**不是阉割版**：

| 包含内容 | 说明 |
|---------|------|
| 全部 9 大知识模块 | 与付费级别结构一致 |
| AI 辅助学习 | 发音纠正、智能出题 |
| 升级考核 | 三级考核体系（课时小测验 + 单元测评 + 级别综合考核），融合课程内容与 HSK 考点，综合考核 85 分通过 |
| 学习证书 | 通过考核获得对应级别证书 |
| HSK 考点训练 | L1=HSK 1, L2=HSK 2, L3=HSK 3 |

> **目的**：让用户在免费阶段充分感受产品质量，建立信任后自然转化。3 个免费级 = 3 次"Aha Moment"。

### 3.3 3 年有效期规则

## 五、推广激励体系：知语币（Zhiyu Coins）

### 5.1 为什么不做现金返佣

| 维度 | 现金返佣 | 知语币（本方案）|
|------|---------|---------------|
| 越南用户提现到银行卡 | ❌ 触发 FCT、SBV AML、FATF 合规风险 | ✅ 无需提现，闭环消费 |
| App Store / Google Play 政策 | ❌ 苹果明确禁止"cash referral rewards" | ✅ 属于 in-app loyalty points，合规 |
| 退款/欺诈风险 | ❌ 推荐人已提现 + 被推荐人退款 → 损失 | ✅ 未消费可撤销，已消费的虚拟品作废成本为 0 |
| 账务处理 | ❌ 跨境汇款 + 个税代扣 | ✅ 平台内账本即可 |
| 用户粘性 | ❌ 提现即流失 | ✅ 必须使用平台才能消耗 |
| 对 $6 定价的课程产品 | ❌ 经济上不可行（$1.2 佣金成本过高） | ✅ 零现金成本 |

结论：**对 $6/级课程定价 + 越南首发市场，虚拟币是唯一合理的推广激励形式。**

### 5.2 知语币规则

| 规则 | 定义 |
|------|------|
| **命名** | 知语币（Zhiyu Coins） |
| **比价** | 1 币 = $0.01（固定，平台内统一） |
| **用途** | 兑换课程 / 皮肤 / 特效 / AI 加速包 |
| **不可做** | 不可提现、不可转让、不可交易、不可折现 |
| **上限** | 单用户账户余额上限 100,000 币（$1000 等值），防大额操作 |
| **法律性质** | 平台内 loyalty points，非证券、非电子货币、非支付工具 |
| **退款** | 自己或自己下级产生支付，都会赠送 20% 的对应价格的知语币。如果产生退款，则对应扣除，可以扣除为负数，下次再获得时优先抒充负数余额。出现负数时，向用户提示：“因您推荐的用户发起了课程退款，对应的推荐奖励已扣回，您的知语币余额为 XX 币。后续获得新奖励时将自动抒充。” |
| **禁止充值** | 禁止付费充值知语币，纯平台赠品，平台倒闭无需赔偿

### 5.3 知语币来源（Credit）

| 来源 | 数量 | 触发条件 |
|------|------|---------|
| **推荐付费** | 被推荐人付费金额的 **20%**（折算为知语币） | 被推荐人完成付费且 30 天内未退款 | 注意，被推荐人自己也会得到20%知语币
| **每日签到** | 1-10 币（概率分布：1=40%, 2=20%, 3-4各8%, 5-6各6%, 7-8各4%, 9-10各2%）| 每日登录签到（付费用户双倍）|
| **游戏连胜** | 1 币 | PK 5 连胜 |
| **新手引导** | 100 币 | 完成首次入学测试 |
| **节日活动** | 不定 | 春节/中秋/双11 等 |

### 5.4 知语币消耗（Debit）

| 用途 | 扣减 |
|------|------|
| 单级课程解锁 | 600 币（= $6） |
| 基础皮肤 | 99 币 |
| 稀有皮肤 | 199-299 币 |
| 全部课程（9级）| 5,400 币（= $54）|

### 5.5 推荐流程（Referral Flow）

```
用户 A 分享唯一推荐链接 → 用户 B 点击并注册（系统自动绑定，无需邀请码）
     ↓
用户 B 购买 L4 课程 $6 → 系统记录 referrals(A, B, 该订单)
     ↓
30 天冷却期通过（B 未退款）→ 向 A 和 B 账户同时发放 120 币（= $1.20）
     ↓
用户 B 后续购买其他级别 / 皮肤 → A 继续获得 20% 知语币（同样 30 天冷却）
     ↓
A 的知语币仅可在平台消费
```

### 5.6 推荐里程碑奖励（纯平台内激励）

| 累计成功推荐人数 | 额外奖励 |
|-----------------|---------|
| 3 人 | 100币
| 10 人 | 300 币 + 任意基础皮肤兑换券 |
| 30 人 | 1000币 + 任意稀有皮肤兑换券 |
| 100 人 | 3000币 + 3级课程解锁券 |
注意，是推荐新用户注册才奖励，老用户不奖励
技术上防脚本攻击

> 里程碑奖励以虚拟品发放，不涉及现金。

### 5.7 防作弊

- 同 IP / 同设备指纹注册 → 不计推荐
- 被推荐人付费后 30 天内退款 → 反向扣除推荐人相应知语币（若余额不足则扣至 0 并记负债账本）
- 推荐关系一人一链，不可修改
- 知语币流水表完整审计

## 七、支付实现

### 7.1 Paddle MoR（Merchant of Record）

| 维度 | 说明 |
|------|------|
| 角色 | **Merchant of Record**（Paddle 是法律意义上的卖方，知语是供应商） |
| 优势 | 支持中国大陆公司签约、全球 VAT/GST 自动处理、欺诈保护、退款处理全包 |
| 费率 | ~5% + $0.50/笔（含支付处理 + 税务合规 + 欺诈保护） |
| 支持方式 | 信用卡（Visa/Mastercard/JCB）、PayPal、Apple/Google Pay、部分区域本地支付 |
| 购买管理 | Paddle Billing 内置购买管理（退款/失败催收） |
| 税务 | **全自动** — Paddle 代缴全球 VAT/GST/Sales Tax，公司无需在各国注册税号 |

> **Paddle 经济测算**：单课 $6，Paddle 扣费约 $0.80（5% + $0.50），净收入约 $5.20/级。全部 9 级 $54，Paddle 扣 9 笔共约 $7.20，净收入约 $46.80。若用户一次性购买 9 级（$54 单笔），Paddle 扣费约 $3.20（5% + $0.50），净收入约 $50.80。

### 7.2 支付流程

```
用户选择套餐 → 前端调用 Paddle Checkout → Paddle 托管支付页面
                                              ↓
                                    Paddle Webhook 通知后端
                                              ↓
                                    后端更新用户课程权限 + 写 coin_transactions（推荐人返佣）
                                              ↓
                                    前端刷新权限
```

### 7.3 iOS 特殊处理

- iOS App 内数字内容购买**必须走 Apple IAP**（苹果 30% 抽成）
- **策略**：引导用户通过 **Web 端** 完成购买（Paddle），绕开苹果税
- App 内不展示购买按钮，只展示"已购买？登录恢复"
- Apple 明确**禁止 App 内展示 Web 购买链接**——采用"在设置中管理购买"等中性措辞引导
- 若必须上架 iOS IAP：Web 价 $6/级，iOS IAP 价 $7.99/级（覆盖苹果税）

### 7.4 中国大陆公司签约说明

- **Paddle 支持中国大陆公司直接签约**（需营业执照 + 法人身份认证）
- Paddle 作为 MoR，向全球买家收款后扣除费用将净收入转给公司
- 公司**无需自行处理全球税务**
- 结算方式：银行电汇（支持 USD 或 CNY）
- 后续如注册新加坡公司，可迁移 Paddle 账户至新主体

### 7.5 越南合规一览（仅课程购买，不涉及用户提现）

| 风险项 | 现状 | 缓解 |
|-------|------|------|
| 跨境收款 | 越南用户通过 Visa/Mastercard 付给 Paddle | ✅ 无合规问题，Paddle MoR 承接 |
| VAT（越南 8-10%） | Paddle 自动代缴 | ✅ 无需行动 |
| FCT（外国承包商税） | Paddle 承担供应商义务 | ✅ 无需行动 |
| 用户获取现金 | **本产品不向用户发放现金** | ✅ 用知语币替代 |
| App Store 规则 | 数字商品需 IAP | ⚠️ 引导 Web 购买 |

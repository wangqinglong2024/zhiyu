# HanYu — 裂变引流与增长策略

> 游戏天然适合裂变：对战邀请、段位炫耀、限定皮肤
> 版本：v1.0 | 日期：2026-04-15

---

## 一、裂变设计哲学

### 1.1 游戏化裂变 vs 传统教育裂变

| 传统教育 App 裂变 | 我们的游戏化裂变 |
|-----------------|----------------|
| "邀请好友获得 7 天会员" | "邀请好友来 PK，赢了送限定皮肤" |
| 分享动机：利益驱动 | 分享动机：**荣誉感 + 竞技 + 社交** |
| 分享内容：推荐链接 | 分享内容：**战绩海报、段位截图、对战邀请** |
| 被邀请人感受：被打广告 | 被邀请人感受：**被约打游戏** |

### 1.2 核心裂变引擎

```
三大裂变引擎：

引擎 1：PvP 对战邀请（最强裂变力）
  → 用户自发邀请好友来 PK
  → 好友必须下载注册才能参战
  → 对战本身就是产品体验

引擎 2：荣誉炫耀分享（自然传播）
  → 段位晋级、勋章获得、赛季排名
  → 用户自发截图分享到社交媒体
  → 精美的分享海报自动生成

引擎 3：利益激励（补充推动力）
  → 传统的邀请奖励机制
  → 会员天数 + 金币 + 钻石
```

## 二、PvP 对战裂变（核心引擎）

### 2.1 好友对战邀请流程

```
1. 用户 A 在 PvP 界面点击"挑战好友"
2. 系统生成对战邀请链接/二维码
3. 用户 A 分享到 Facebook/Zalo/WhatsApp/Line
4. 好友 B 点击链接：
   a. 已安装 → Deep Link 直接进入对战房间
   b. 未安装 → 跳转下载页（保存邀请参数）
      → 下载注册后自动进入对战房间
5. 对战完成 → 双方获得奖励
6. 好友 B 大概率会继续玩（因为刚体验了最有趣的功能）
```

### 2.2 对战邀请的病毒系数

```
预估病毒系数计算：

假设：
  · 每个活跃用户每周发起 1 次好友对战邀请
  · 邀请接受率：30%（游戏邀请 > 普通 App 推荐）
  · 接受后注册率：50%（需要注册才能对战）
  · 新用户次周留存率：40%

每周病毒系数 = 1 × 0.3 × 0.5 × 0.4 = 0.06/周

月病毒系数 ≈ 0.06 × 4 = 0.24/月

即每个用户每月带来 0.24 个新留存用户
目标优化到 0.3+ → 有机增长可持续
```

### 2.3 组队学习裂变

```
军团系统的裂变效应：

创建军团需要 3 人以上 → 邀请好友加入
军团任务需要团队协作 → 活跃成员邀请更多人
军团排名 → 为了排名招募强力成员
军团 vs 军团（未来） → 大规模拉人

每个军团最多 30 人
活跃军团平均 15 人
军团长平均邀请 5 人 → 高质量裂变（有社交关系绑定）
```

## 三、荣誉炫耀裂变

### 3.1 自动生成分享海报

```
触发时机（自动弹出分享引导）：

高情绪点触发：
  · PvP 对战胜利（尤其是连胜、逆转胜）
  · 段位晋级（铜→银→金→钻石...）
  · 解锁新勋章
  · 完成领地征服
  · 获得限定皮肤
  · 赛季排名公布
  · 连续打卡里程碑（7天/30天/100天）

低情绪点不触发：
  · 对战失败
  · 答错题
  · 关卡未通关
```

### 3.2 海报设计规范

```
海报类型：

[对战胜利海报]
  · 展示：对战双方段位 + 比分 + "胜利"大字
  · 底部：邀请二维码 + "来挑战我"
  · 风格：电竞风 + 金色光效

[段位晋级海报]
  · 展示：新段位图标 + 玩家角色（当前皮肤）
  · 底部：邀请二维码 + "我升到XX段了"
  · 风格：史诗感 + 段位主题色

[打卡海报]
  · 展示："我已在 HanYu 学中文 XX 天"
  · 数据：学会 XX 个汉字 / 完成 XX 个关卡
  · 底部：邀请二维码
  · 风格：简洁精致

[勋章获得海报]
  · 展示：勋章图标 + 获得条件描述
  · 底部：邀请二维码
  · 风格：成就解锁感

每张海报自动包含：
  · 玩家昵称和角色头像
  · 成就/战绩数据
  · HanYu 品牌 Logo
  · 邀请二维码（含玩家专属邀请码）
  · 下载链接文字
```

## 四、利益激励裂变

### 4.1 邀请奖励规则

```
推荐人（邀请者）奖励：

邀请 1 人注册 + 完成新手教程
  → 获得 100 金币 + 50 HanYu Coins

邀请 1 人成为付费会员
  → 获得 30 天免费 Premium

邀请 3 人成为付费会员
  → 获得 1 年免费 Premium

邀请 10 人成为付费会员
  → 获得终身免费 Premium
  → 解锁"推广大使"身份

邀请 50 人成为付费会员
  → 升级为"城市合伙人"
  → 所有被推荐人的付费金额 15% 现金返佣
```

```
被推荐人（新用户）奖励：

通过邀请链接注册
  → 获得"邀请新兵"限定皮肤（独占，无法其他方式获得）
  → 获得 7 天免费 Premium 体验
  → 获得 50 HanYu Coins

首次付费
  → 享受首月 $0.99 特价
```

### 4.2 推广大使体系

| 等级 | 条件 | 奖励 |
|------|------|------|
| **学习伙伴** | 邀请 1-2 人付费 | 30-60 天免费 Premium |
| **学习达人** | 邀请 3-9 人付费 | 1 年免费 Premium + 专属称号 |
| **推广大使** | 邀请 10-49 人付费 | 终身免费 + 10% 现金返佣 + 限定皮肤套装 |
| **城市合伙人** | 邀请 50+ 人付费 | 终身免费 + 15% 返佣 + 专属运营支持 + 限定坐骑 |

### 4.3 推广大使仪表盘

```
后台功能：
  · 实时数据：邀请注册数 / 付费转化数 / 累计收益
  · 生成工具：自定义推荐链接 + 海报 + 二维码
  · 返佣明细：逐笔记录 + 可提现余额
  · 排行榜：本月推荐最多的大使 Top 10
  · 推广素材：官方图片/文案/视频素材下载

提现规则：
  · 最低提现：$10
  · 提现方式：MoMo / GoPay / 银行转账
  · 结算周期：每月 1 日结算上月
  · 冷却期：被推荐人付费 30 天后才计入可提现
```

## 六、防作弊机制

### 6.1 推荐奖励防刷

```
规则：
  1. 被推荐人必须完成新手教程（约 10 分钟） → 防止纯注册刷量
  2. 被推荐人需累计学习 3 天 → 防止注册即弃
  3. 付费推荐奖励需被推荐人连续付费 30 天 → 防止付费后退款
  4. 同一设备/IP 不允许重复注册 → 防止多开
  5. 同一支付工具关联的账号不超过 3 个 → 防止自买自推
  6. 推广大使返佣有 30 天冷却期 → 防止刷单

技术实现：
  · 设备指纹（Device Fingerprint）检测
  · IP 地理位置 + 登录时段分析
  · 支付工具关联分析
  · 异常行为检测（如短时间大量注册）
  · 人工审核阈值：单用户单日推荐 >10 人触发人工审核
```

### 6.2 PvP 防作弊

```
规则：
  · 答案校验在服务端（客户端不发送正确答案）
  · 答题速度异常检测（<0.5秒答对视为可疑）
  · 同 IP 对战检测（防止自打自）
  · 段位异常检测（短时间内段位飙升）
  · 举报系统 → 人工审核 → 封号/降段
```

## 七、增长飞轮模型

```
HanYu 增长飞轮：

        ┌───────────────────────┐
        │  更多用户注册           │
        │  (PvP 邀请 + 荣誉分享)  │
        └──────────┬────────────┘
                   │
                   ▼
        ┌───────────────────────┐
        │  PvP 匹配更快、更有趣   │
        │  （更多对手 = 更好体验）  │
        └──────────┬────────────┘
                   │
                   ▼
        ┌───────────────────────┐
        │  用户留存提升            │
        │  （游戏好玩 + 社交绑定）  │
        └──────────┬────────────┘
                   │
                   ▼
        ┌───────────────────────┐
        │  更多用户邀请好友         │
        │  （想 PK + 想炫耀）      │
        └──────────┬────────────┘
                   │
                   ▼
              回到第一步 ↑

飞轮加速器：
  · 赛季制度 → 每 3 个月重置段位 → 重新冲刺 → 重新活跃
  · 限定内容 → FOMO 驱动回流
  · 军团系统 → 社交绑定 → 难以流失
  · 内容更新 → 新关卡 → 回流
```

## 八、关键裂变指标

| 指标 | 目标值 | 说明 |
|------|--------|------|
| **裂变系数 K** | >0.3 | 每个用户带来 0.3+ 个新用户 |
| **邀请发送率** | >15% | 15%+ 的活跃用户发送过邀请 |
| **邀请接受率** | >20% | 收到邀请后 20%+ 的人注册 |
| **分享率** | >25% | 25%+ 的用户分享过海报/战绩 |
| **推荐用户 7 日留存** | >35% | 被推荐用户比自然用户留存更高 |
| **推广大使转化率** | >0.5% | 0.5%+ 的用户成为推广大使 |

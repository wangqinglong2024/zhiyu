# HanYu — 财务合规与公司架构方案

> 版本：v1.0 | 日期：2026-04-15

## 一、公司架构

### 1.1 现有实体

你已有一家**中国有限公司**（以下简称"中国主体"）。这是项目的核心运营实体。

### 1.2 推荐架构

```
中国有限公司（主体）
├── 产品开发与技术中心
├── 内容生产与审核
├── 财务与行政
├── 知识产权持有
│
├── [未来可选] 新加坡子公司 / 独立公司
│   ├── 海外收入结算
│   ├── 东南亚客户合同签署
│   └── 应用商店开发者账号持有
│
└── [未来可选] 越南/印尼当地代理/合作方
    └── 本地化运营支持
```

### 1.3 架构选择分析

**方案 A：仅中国主体（初创期推荐）**

| 优势 | 劣势 |
|------|------|
| 零额外注册成本 | 海外收入需结汇，有外汇管制 |
| 运营简单，一套账 | 部分国家对中国公司有审核偏见 |
| 中国企业所得税 25%（小微可减免） | 东南亚用户对中国公司信任度不一 |

**方案 B：中国主体 + 新加坡公司（规模化后推荐）**

| 优势 | 劣势 |
|------|------|
| 新加坡企业税 17%（前 $200K 减免至 ~8.5%） | 注册成本 $2,000-3,000 |
| 海外收入直接在新加坡结算，无需结汇 | 每年审计/合规费 $1,500-3,000 |
| 东南亚用户/企业信任度高 | 需要新加坡注册地址和本地董事 |
| 应用商店账号更便于全球发布 | 两套账务增加复杂度 |
| 腾讯云新加坡节点合同直签 | |

**建议时间线**：
- **Phase 0-2**（前 9 个月）：仅用中国主体，不增加复杂度
- **Phase 3+**（年收入 >$50K 后）：注册新加坡公司，海外收入通过新加坡结算
- **触发条件**：月收入稳定超过 $5,000 或准备接 B2B 企业合同时

## 二、收入结算与外汇合规

### 2.1 初创期（仅中国主体）

```
收入流向：

用户付费（美元/越南盾/印尼盾）
    │
    ▼
支付平台（Stripe / MoMo / Midtrans）
    │
    ▼
Stripe 账户（美元）
    │
    ▼
中国银行外汇账户
    │  需提交：
    │  · 服务合同（与用户/平台的协议）
    │  · 发票（或 Invoice）
    │  · 银行要求的其他材料
    ▼
人民币结汇入对公账户
```

**关键注意事项**：

1. **外汇登记**：中国企业接收外汇收入需在银行做外汇登记，首次办理需提交营业执照、服务合同等
2. **Stripe Atlas / 香港 Stripe**：Stripe 目前不直接支持中国大陆企业。可选方案：
   - 使用 Stripe 香港（需香港银行账户）
   - 使用 Paddle/LemonSqueezy 作为代收代付（Merchant of Record）
   - 直接对接本地支付（MoMo API / Midtrans），资金回流到中国账户
3. **增值税**：出口软件服务在中国可申请增值税免税或零税率（需办理出口退税备案）

### 2.2 规模化后（中国 + 新加坡）

```
收入流向（方案 B）：

东南亚用户付费
    │
    ▼
支付平台 → 新加坡公司银行账户
    │
    ├── 留存运营资金（服务器、推广、海外人员）
    │
    └── 技术服务费 → 中国母公司
        （需签订关联交易协议，转让定价合规）
```

## 三、税务规划

### 3.1 中国税务

| 税种 | 税率 | 说明 |
|------|------|------|
| **企业所得税** | 25%（小微 5%） | 年应纳税所得 ≤300 万享小微优惠 |
| **增值税** | 6%（技术服务） | 出口软件服务可申请免征 |
| **个人所得税** | 3-45%（超额累进） | 创始人工资/分红 |
| **附加税** | ~12%（增值税附加） | 城建税+教育费附加 |

**税务优化建议**：
1. 初创期尽量保持小微企业资质（年应纳税所得 ≤300 万，5% 税率）
2. 申请高新技术企业认定（符合条件可享 15% 企业所得税）
3. 出口软件服务申请增值税免征
4. 研发费用加计扣除（AI 和游戏开发相关支出可加计 100% 扣除）

### 3.2 新加坡税务（规模化后适用）

| 税种 | 税率 | 说明 |
|------|------|------|
| **企业所得税** | 17% | 前 $100K 仅 4.25%，$100K-$200K 仅 8.5% |
| **GST** | 9% | 年收入 >$1M SGD 需注册 |
| **预扣税** | 0-15% | 支付技术服务费给中国母公司时 |

**新加坡优势**：
- 与中国有避免双重征税协定（DTA）
- 与越南、印尼也有 DTA
- 新创企业前 3 年享额外税务减免

### 3.3 越南/印尼税务考量

```
作为无本地实体的外国企业：

越南：
  · 不需在越南注册实体（纯线上 SaaS 服务）
  · 但越南近年加强对数字服务的税收监管
  · 2022 年起要求外国数字服务提供商注册纳税
  · 税率：增值税 5% + 企业所得税 5%（简化税率）
  · 建议：通过越南税务网站注册外国数字服务提供商

印尼：
  · 类似越南，对外国数字服务征税
  · 增值税 11%（2025 年 12% ）
  · 需要注册 Appointed VAT Collector
  · 建议：通过 Google Play / App Store 代扣代缴（它们已是指定代扣方）
```

## 四、数据合规

### 4.1 各国数据隐私法规

| 国家 | 法规 | 核心要求 | 我们的应对 |
|------|------|---------|-----------|
| **越南** | 个人数据保护法令（PDPD 2023） | 数据处理需同意；跨境传输需评估 | 新加坡服务器（非中国），隐私政策越南语版 |
| **印尼** | 个人数据保护法（PDP 2022） | 类似 GDPR；数据处理需合法基础 | 同上，隐私政策印尼语版 |
| **新加坡** | PDPA | 数据收集/使用/披露需同意 | 服务器在新加坡，天然合规 |
| **中国** | 个人信息保护法（PIPL） | 敏感信息需单独同意；出境需评估 | 用户数据存新加坡，不回传中国 |

### 4.2 数据架构合规设计

```
关键设计原则：

1. 用户数据存储在新加坡（腾讯云新加坡节点）
   · 不将东南亚用户的个人数据传回中国
   · 中国开发团队通过后台管理系统访问（脱敏数据）

2. 隐私政策多语言
   · 越南语版 + 印尼语版 + 英语版 + 中文版
   · 明确告知数据收集范围、使用目的、存储位置

3. 用户同意机制
   · 首次注册时展示隐私政策，明确获取同意
   · AI 功能使用前单独提示"您的对话数据将用于改善服务"
   · 用户可随时删除账号和所有数据（GDPR-style）

4. 数据最小化
   · 只收集业务必需的数据
   · 学习数据匿名化后用于产品优化
   · 支付信息由第三方支付平台处理，不自行存储卡号

5. AI 数据使用
   · 用户的 AI 对话数据不用于训练模型
   · 仅用于提供即时服务
   · 明确告知用户并获取同意
```

## 五、知识产权保护

### 5.1 需保护的 IP

| IP 类型 | 内容 | 保护方式 |
|---------|------|---------|
| **商标** | HanYu / 汉语 品牌名 + Logo | 中国商标注册 + 马德里国际注册（覆盖越南/印尼） |
| **软件著作权** | 游戏引擎代码、App 代码 | 中国软件著作权登记（免费，有助于高企认定） |
| **美术作品** | 像素美术、角色设计、UI 设计 | 美术著作权（自动产生，保留原始创作文件） |
| **课程内容** | 关卡内容、题目、剧本 | 著作权（自动产生） |
| **域名** | hanyu.app 或类似域名 | 域名注册 |

### 5.2 IP 保护优先级

```
Phase 1（立刻）：
  · 注册域名
  · 中国商标申请（品牌名 + Logo）
  · 软件著作权登记（有助于高企认定）

Phase 2（上线前）：
  · 越南/印尼商标注册（通过马德里体系或当地代理）
  · 完善用户协议和服务条款

Phase 3（规模化后）：
  · 美术作品批量版权备案
  · 竞品侵权监测
```

## 六、应用商店合规

### 6.1 Google Play

```
开发者账号：
  · 注册费：$25（一次性）
  · 主体：中国公司或未来新加坡公司
  · 审核周期：3-7 天

合规要求：
  · 隐私政策链接（必填）
  · 数据安全表单（声明收集哪些数据）
  · 内容分级（IARC）
  · 订阅条款透明展示
  · 未成年人保护（如面向 13 岁以下需额外合规）

抽成：
  · 订阅类：年收入 <$1M 部分 15%，超出 30%
  · 非订阅类：15%（<$1M）
```

### 6.2 Apple App Store

```
开发者账号：
  · 注册费：$99/年
  · 主体：推荐用中国公司注册（需 D-U-N-S 编号）
  · 审核：通常 1-3 天

合规要求：
  · 隐私政策
  · App Tracking Transparency（ATT）弹窗
  · App Privacy 标签
  · 订阅条款

抽成：
  · 小型开发者（年收入 <$1M）：15%
  · 超出：30%

特殊注意：
  · Apple 不允许在 App 内引导用户到外部订阅页面
  · 但可以通过邮件/短信引导（不违规）
  · PWA 版不受 Apple 抽成限制
```

## 八、财务管理

### 8.1 账务建议

```
初创期（自己+代账）：
  · 使用中国公司对公账户收支
  · 聘请代账公司处理月度记账和报税（约 ¥300-500/月）
  · 自己用 Excel/Notion 记录运营数据

规模化后（兼职/全职财务）：
  · 正式会计处理中国+新加坡两套账
  · 使用财务软件（金蝶/用友）
  · 季度出具财务报表
  · 年度审计（新加坡公司法定要求）
```

### 8.2 资金规划

```
启动资金建议：¥10-15 万人民币

分配：
  · 服务器+工具订阅（6个月）：¥10,000
  · 美术外包：¥15,000
  · 域名+商标+法律文件：¥5,000
  · 推广预算（6个月）：¥20,000
  · 支付平台保证金/接入费：¥5,000
  · 应急储备：¥20,000-50,000
  · AI API 费用（开发+测试+初期运营）：¥5,000

如果创始人自己开发（+AI辅助），开发人力成本 = 0
总启动资金可控制在 ¥10 万以内
```

## 九、合规时间线

| 阶段 | 时间 | 合规动作 |
|------|------|---------|
| **立刻** | 本月 | 注册域名、申请中国商标、软件著作权登记 |
| **MVP 前** | 1-3 个月 | 起草服务条款+隐私政策（AI 生成初稿+法律顾问审核） |

| **越南上线** | 4 个月 | 越南数字服务税注册（如适用）、越南语法律文件 |

## 一、课程体系总览

### 1.1 设计哲学

> **本固根深，应试无忧；不逐应试，自成底气。**

知语 Zhiyu 课程体系的核心逻辑：**不围绕考试设计课程，而是围绕真正掌握语言来设计课程。** 当学生真正具备语言能力时，通过任何考试都是自然结果。

### 1.2 双轨对标体系

课程同时对标两套标准：

| 轨道 | 标准 | 适用场景 |
|------|------|---------|
| **母语轨** | 中国义务教育课程标准（K-12） | 系统性语文能力培养，适合华裔/深度学习者 |
| **二语轨** | HSK 3.0 / CEFR 国际标准 | 国际认证、留学申请、职场需求 |

两轨并行、内容融合——学生学同一套课程，同时获得两套标准的能力提升。

## 二、九大知识模块

9 大知识模块贯穿 12 个级别，每个级别的难度螺旋上升。

### 2.1 模块总览

| 编号 | 知识模块 | 说明 |
|------|---------|------|
| M1 | **汉字与书写** | 识字量递增、笔画笔顺、偏旁部首、书写规范 |
| M2 | **拼音与语音** | 声母韵母、声调、音变、朗读节奏 |
| M3 | **词汇与语义** | 词汇量递增、近反义词、多义词、成语俗语 |
| M4 | **语法与句法** | 从简单句到复合句，语序、关联词、句式变换 |
| M5 | **修辞与表达** | 比喻、拟人、排比等修辞手法，从识别到运用 |
| M6 | **阅读理解** | 从句子理解到篇章分析，从记叙文到议论文 |
| M7 | **写作与口语** | 从看图写话到议论文，从日常口语到正式演讲 |
| M8 | **文学与文化** | 文学常识、作家作品、文化背景、跨文化对比 |
| M9 | **古诗文** | 从简单古诗背诵到文言文精读赏析 |

## 四、AI 辅助教学功能

| **AI 写作批改** | 自动评分 + 语法错误标注 + 改进建议 | L4+ |
| **AI 古文工坊** | 文言文翻译辅助 + 断句练习 + 古今对照 | L7+ |

基于艾宾浩斯遗忘曲线，自动安排复习时间：

```
首次学习 → 1天后 → 2天后 → 4天后 → 7天后 → 15天后 → 30天后
```

- 答对则延长间隔，答错则缩短间隔
- 适用于：生字、词汇、成语、古诗文、语法点
- 每日复习量控制在 15-30 分钟内

## 五、课程内容结构

### 5.1 每个 Level 的内容组织

```
Level X
  ├── Unit 1（单元）
  │     ├── Lesson 1（课时）

  ├── Unit 2

  ├── Unit N
```

### 5.2 每个 Level 的规模

### 5.3 内容题型

## 六、升级考试体系

### 6.1 考试结构

每个 Level 结束时需通过升级考试才能进入下一级：

| 考核维度 | 权重 | 评估方式 |
|---------|------|---------|
| **听力** | 20% | AI 出题 + 自动评分 |
| **口语** | 20% | AI 语音评测 + 评分 |
| **阅读** | 30% | 多题型综合 + 自动评分 |
| **写作** | 20% | AI 批改 + 评分 |
| **文化** | 10% | 知识问答 |

### 6.2 通过标准

| 项目 | 标准 |
|------|------|
| 总分 | **85 分**（满分 100 分） |
| 单项最低分 | 不低于 60 分（防止严重偏科） |
| 补考机制 | 不通过可无限次重考，间隔 24 小时 |
| 补考前 | 系统自动推荐薄弱模块复习 |

### 6.3 证书与成就

- 通过每个 Level 获得电子证书（可分享到社交媒体）
- 通过每个阶段（如完成启蒙段 L1-3）获得阶段证书
- 证书标注对应 HSK 等级和 CEFR 等级
- 成就系统联动（详见游戏设计文档）

## 七、入学分级测试

### 7.1 测试结构

| 模块 | 题量 | 时长 | 说明 |
|------|------|------|------|
| 识字量 | 20题 | 5分钟 | 认字选义 |
| 词汇量 | 20题 | 5分钟 | 词义匹配 |
| 阅读理解 | 10题 | 10分钟 | 短文理解 |
| HSK 模拟 | 10题 | 5分钟 | 标准 HSK 题型 |

### 7.2 分级规则

- 根据正确率自动分配到对应 Level
- 测试采用自适应出题：答对则出更难的题，答错则降级
- 零基础检测：若前 10 题正确率 < 30%，直接分配到 L1
- 高水平检测：若全部答对，分配到 L10 并推荐挑战 L11/12

## 八、内容版权策略

### 8.1 原则

| 内容类型 | 策略 |
|---------|------|
| 古诗文（公版） | 直接使用，标注出处 |
| 义务教育课程标准（知识点） | 引用知识点体系，不复制教材原文 |
| 统编教材课文 | **不使用原文**，提供同题材改写版本 |
| 现当代文学作品 | 引用片段（合理使用），或取得授权 |
| 自主原创 | AI 生成 + 人工审核润色 |

### 8.2 改写规则

对于有版权的课文/文章：
- 保留相同知识点和教学目标
- 改写故事情节和人物设定
- 同题材、不同表达
- 标注"本文为教学改编版本"

## 九、内容生产流程

### 9.1 生产流程

```
教学大纲设计 → AI 生成初稿（Dify 工作流）→ 教学专家审核 → 多语言翻译 → 练习题生成 → 质量测试 → 上线
```

### 9.2 内容存储

所有教学内容存储于 **Dify 知识库**，结构化管理：

```
Level > Unit > Lesson > Content Block
                          ├── text_zh（中文正文）
                          ├── text_pinyin（拼音）
                          ├── text_vi（越南语翻译）
                          ├── text_en（英语翻译）
                          ├── audio_url（音频链接）
                          ├── exercises（练习题 JSON）
                          └── metadata（级别、模块、知识点标签）
```

### 9.3 内容更新机制

- 核心课程内容一旦上线，保持稳定（不频繁改动）
- 练习题库持续扩充（AI 生成 + 人工审核）
- 根据用户反馈和数据优化难度曲线
- 每季度进行一次内容质量审查

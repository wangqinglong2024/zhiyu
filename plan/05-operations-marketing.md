# HanYu — 运营推广方案

> 游戏化产品的内容营销 + 本地化推广
> 版本：v1.0 | 日期：2026-04-15

## 一、推广总纲

### 1.1 核心思路转变

旧计划以"AI 中文学习"为卖点推广。新计划的推广核心是：**"一个好玩的中文游戏"**。

| 旧推广思路 | 新推广思路 |
|-----------|-----------|
| "AI 帮你学中文" | "一边打游戏一边学中文" |
| 强调功能（发音纠正、AI 对话） | 强调体验（攻城略地、段位竞技、华丽皮肤） |
| 教育 App 的推广方式 | **游戏的推广方式** |
| 目标情绪：求知欲 | 目标情绪：**爽、酷、想赢** |

### 1.2 三阶段路径

```
Phase 0 - 验证期（第1个月）
  目标：验证"游戏化学中文"的概念吸引力
  预算：$0（纯内容引流）
  核心：TikTok 游戏概念视频 + Facebook 社群测试

Phase 1 - 冷启动（第2-4个月）
  目标：5,000 注册用户
  预算：$500-1,000/月
  核心：TikTok 矩阵 + 游戏社区渗透 + KOL 试玩

Phase 2 - 增长期（第5-12个月）
  目标：50,000 注册用户
  预算：$2,000-5,000/月
  核心：付费广告 + 裂变系统 + PvP 竞技赛事 + B2B
```

## 五、KOL 合作方案

### 5.1 KOL 类型拓展

传统语言学习 App 只找教育类 KOL。我们还要找**游戏类 KOL**：

| KOL 类型 | 粉丝量 | 合作方式 | 预算/位 |
|---------|--------|---------|---------|
| 游戏类 KOL（核心） | 10K-200K | 试玩视频 + 专属推荐码 | $50-500 |
| 中文教学 KOL | 5K-100K | 深度体验 + 教学联动 | $50-300 |
| 生活方式 KOL | 10K-100K | "我的学习方式"类内容 | $50-200 |
| 微型 KOL / 素人 | 1K-10K | 免费终身会员 + 推荐码 | $0 |

### 5.2 合作脚本（游戏化版）

```
合作内容要求：
1. 真实试玩至少 3 天
2. 展示至少 2 个游戏玩法（PvP 对战 + 关卡闯关 或 小游戏）
3. 展示游戏画面品质（强调美术和体验）
4. 展示段位/成就（荣誉感）
5. 提供专属推荐码和下载链接
6. 时长：60-120 秒

我们提供：
- 终身 Premium 账号
- 专属推荐码（推荐人付费后 KOL 获 10% 佣金）
- 游戏内"KOL 限定皮肤"（仅通过 KOL 推荐码注册的用户获得）
- 素材支持
```

### 5.3 游戏内 KOL 限定皮肤策略

```
核心创意：每个合作 KOL 都有一套联名限定皮肤

机制：
  · 用户通过 KOL 的推荐码注册 → 自动获得该 KOL 的联名皮肤
  · 皮肤设计融入 KOL 的个人元素
  · 限时供应（合作期结束后不再发放）
  · 已获得的皮肤永久保留

价值：
  · 对 KOL：独特的合作体验，不只是打广告
  · 对用户：稀缺感，"只有通过这个 KOL 才能拿到"
  · 对我们：强化推荐码使用率，可追踪 KOL 贡献
```

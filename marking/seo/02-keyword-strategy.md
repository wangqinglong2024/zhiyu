## 3. 5 层意图模型

| 层 | 意图 | 用户在哪 | 关键词特征 | 知语对应页面 |
|---|------|---------|----------|------------|
| L1 | **Awareness** | 漏斗顶 | "is chinese hard to learn", "why learn mandarin" | 博客长文 |
| L2 | **Information** | 漏斗中上 | "HSK 4 vocabulary", "stroke order rules" | 词条页 + 长文 |
| L3 | **Comparison** | 漏斗中 | "duolingo vs hellochinese", "best app for HSK" | vs 文章 + 评测 |
| L4 | **Commercial** | 漏斗中下 | "free chinese learning app", "HSK 4 course price" | 课程介绍页 |
| L5 | **Transactional** | 漏斗底 | "download zhiyu app", "zhiyu sign up" | 转化页 |

**优先级**：L2 > L3 > L4 > L1 > L5（L2 是流量主战场，L3 转化最猛）

## 4. 4 个市场的本地化关键词差异

### 4.1 北美（en-US）
- 优先词：`learn chinese`, `mandarin`, `simplified chinese`, `HSK`
- 文化锚词：`chinese new year`, `chinese zodiac`, `chinese tattoo`, `feng shui`
- 工具词：`pinyin keyboard`, `chinese to english translator`
- **避免**：`mandarin chinese`（重复词）— 倾向用 `mandarin`

### 4.2 英国 / 澳洲（en-GB / en-AU）
- 与 US 大体一致，但偏好：`mandarin lessons`、`learn mandarin`
- 文化兴趣偏：`tea ceremony`, `traditional medicine`, `tai chi`

### 4.3 西欧（de/fr/es）
- 关键词必须本地化：`Chinesisch lernen`, `apprendre le chinois`, `aprender chino`
- 教育认证关注度高：`HSK Prüfung`, `examen HSK`
- 必须做 hreflang，不能只用英语 + 翻译插件

### 4.4 东南亚（vi/th/id/ms）
- 越南语：`học tiếng trung`, `HSK [N]`, `tiếng phổ thông`
- 泰语：`เรียนภาษาจีน`, `HSK`
- 印尼语：`belajar bahasa mandarin`, `kursus mandarin`
- 商务场景搜索量极高：`tiếng trung thương mại`（中文 商务）`belajar mandarin bisnis`
- **东南亚是知语 P0 市场**，关键词竞争远低于英语

## 5. 关键词采集 SOP（数据团队执行）

### 步骤 1：种子词库
从下列 5 个来源拉种子，去重后 ≈ 5000 词：
1. Google Suggest API（输入 50 个根词，扩展到 a-z、how/what/why、年份等）
2. Ahrefs `Matching Terms` 报告（根词 5000 → 扩展 5 万）
3. SEMrush `Keyword Magic`（按主题）
4. Reddit r/ChineseLanguage 顶帖标题词频统计
5. Quora `learn chinese` 话题相关问题

### 步骤 2：扩展长尾
对每个种子词，用 Ahrefs/SEMrush 拉：
- Search Suggestions
- Questions
- Also Rank For
导出 CSV，目标 50 万 - 100 万长尾

### 步骤 3：分类打标
每个关键词必须打 6 个 tag（写脚本批量）：
- `track`：HSK / 字典 / 拼音 / 成语 / 文化 / 课程 / 通用
- `intent`：L1-L5
- `market`：US/UK/AU/DE/FR/ES/VN/TH/ID/MY
- `volume`：搜索量级
- `kd`：Keyword Difficulty
- `serp_features`：是否有 AI Overview / Featured Snippet / PAA

### 步骤 4：优先级评分
公式：
```
priority = (volume × intent_weight × (100 - kd)) / 100
```
其中：
- `intent_weight`：L1=0.5, L2=1.0, L3=1.5, L4=2.0, L5=3.0

按 priority 降序，前 5000 关键词为"P0 关键词"，必须 90 天内有页面承接。

### 步骤 5：竞品 Gap 分析
导出 Duolingo / HelloChinese / Pleco / Du Chinese / ChinesePod 排名前 100 的所有 URL → 关键词 → 我们没有的词 = **冷启动收割池**

## 6. 关键词与页面的映射关系

**铁律**：1 个核心关键词 = 1 个 canonical 页面，绝不竞食（Cannibalization）。

```
核心关键词 ─┐
            ├→ 1 个 canonical URL
变体词     ─┘
长尾词 (10-100) ─→ 同一 URL（写进文中 H2/H3）
```

实现方式：
- 内容编辑用关键词分组工具（如 Keyword Insights、SurferSEO Cluster）
- 输出 CSV：`primary_kw, variants[], target_url, page_type`
- 工程把 CSV 导入 CMS，自动生成页面

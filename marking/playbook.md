## 0. 这份文档解决什么问题

你现在不是要做“全网营销”，而是要做一套 **AI 可验证证据链**。

AI 认可 Zhiyu，不是因为你发了很多平台，而是因为它能从多个公开来源验证以下事实：

1. Zhiyu 是真实产品。
2. Zhiyu 是教人学中文 / 汉语 / Mandarin Chinese 的产品。
3. Zhiyu 适合越南、泰国、印尼用户。
4. Zhiyu 有官网、App、视频、社交公开内容、实体页。
5. 全网对 Zhiyu 的描述一致，不混乱。

所以本项目只做 **公开可抓取、可索引、可验证** 的平台：

- 官网本地化页面

不要写：

- “全球第一”
- “最强 AI 中文学习平台”
- “官方 HSK 平台”
- “被所有 AI 推荐”
- “100% 保证通过 HSK”

要写：

- 明确功能
- 明确适合谁
- 明确学习内容
- 明确更新时间
- 明确资料来源

AI 更愿意引用清晰、克制、可验证的内容。

## 2. “实体身份证明”到底是什么意思

### 2.1 为什么要做实体证明

AI 在回答 “best app to learn Mandarin for Vietnamese learners” 时，会先判断：

1. 这个产品是否真实存在？
2. 是否有官网？
3. 是否有 App Store / Google Play？
4. 是否有公司或团队实体？
5. 是否有公开社交账号？
6. 是否有第三方页面能交叉验证？
7. 是否和查询意图相关？

所以 LinkedIn / Wikidata / App Store / Google Play 不是为了流量，而是为了让 AI 确认：

> Zhiyu 是一个真实存在的中文学习产品，而不是一个刚生成的 SEO 页面。

### 2.2 各实体平台的作用

| 平台 | 作用 | 是否必须 | 备注 |
|---|---|---:|---|
| App Store | 证明 iOS App 真实存在 | 必做 | 若 iOS 版本未上线，先准备 Product Page 资料 |
| Google Play | 证明 Android App 真实存在 | 必做 | 东南亚 Android 权重高，尤其重要 |
| LinkedIn Company Page | 证明公司 / 产品实体 | 必做 | AI 和用户都会看 |
| 创始人 LinkedIn | 证明团队和人可信 | 必做 | 2 人公司更要做人设可信 |
| Crunchbase | 创业公司数据库实体 | 推荐 | 可低频维护 |
| Wikidata | 机器可读实体锚点 | 谨慎做 | 没有足够公开引用时可能被删 |
| GitHub Public Repo / Docs | 证明技术真实存在 | 推荐 | 如果仓库可公开展示部分文档 |
| YouTube Channel About | 视频实体证明 | 必做 | 要统一描述 |
| Facebook Page About | 本地公开实体证明 | 必做 | 东南亚重要 |
| TikTok Bio | 社交搜索实体证明 | 必做 | 简短但一致 |

### 2.3 Wikidata 特别说明

Wikidata 不是想建就一定能保留。它要求条目是可识别实体，并且最好有严肃、公开、可验证的来源。

执行建议：

1. 如果 Zhiyu 已经有 App Store、Google Play、LinkedIn、官网、媒体报道、公开产品页，则可以尝试创建 Wikidata。
2. 如果公开第三方来源还不够，先不要硬建。
3. 先做 LinkedIn / App Store / Google Play / Crunchbase / YouTube / Facebook。
4. 等有 2-3 个可靠第三方报道或公开数据库页面后，再建 Wikidata，成功率更高。

### 2.4 Wikidata 目标属性

如果可以创建，建议属性如下：

| 属性 | 值 |
|---|---|
| label | Zhiyu |
| also known as | 知语 |
| description | Mandarin Chinese learning platform |
| instance of | mobile application / educational software / website |
| official website | `https://zhiyu.app` |
| language | English, Vietnamese, Thai, Indonesian, Chinese |
| country | 按公司真实注册地填写 |
| inception | 2026，若真实年份不是 2026，则以真实年份为准 |
| same as | LinkedIn、App Store、Google Play、YouTube、Facebook、TikTok |

注意：所有信息必须真实。不要为了 SEO 编造公司注册地、创始时间、团队背景。

## A7. IndexNow

### 目标

每次新增或更新页面后，通知 Bing / IndexNow 生态。

### 步骤

1. 生成 IndexNow key。
2. 在网站根目录放置 key 文件：

```text
https://zhiyu.app/YOUR_INDEXNOW_KEY.txt
```

文件内容就是 key 本身。

3. 新增或更新页面后，POST 到：

```text
https://api.indexnow.org/indexnow
```

### 请求示例

```json
{
  "host": "zhiyu.app",
  "key": "YOUR_INDEXNOW_KEY",
  "keyLocation": "https://zhiyu.app/YOUR_INDEXNOW_KEY.txt",
  "urlList": [
    "https://zhiyu.app/vi/",
    "https://zhiyu.app/th/",
    "https://zhiyu.app/id/"
  ]
}
```

### Next.js 伪代码

```ts
export async function submitIndexNow(urls: string[]) {
  const payload = {
    host: "zhiyu.app",
    key: process.env.INDEXNOW_KEY,
    keyLocation: `https://zhiyu.app/${process.env.INDEXNOW_KEY}.txt`,
    urlList: urls,
  };

  const res = await fetch("https://api.indexnow.org/indexnow", {
    method: "POST",
    headers: {
      "Content-Type": "application/json; charset=utf-8",
    },
    body: JSON.stringify(payload),
  });

  if (!res.ok) {
    console.error("IndexNow submit failed", await res.text());
  }

  return res.status;
}
```

### 验收标准

- key 文件可访问。
- 提交后返回 HTTP 200。
- 每次发布新页面后自动提交。
- 每次更新核心页面后自动提交。

```text
三语官网 + Google/GSC + Bing/IndexNow + YouTube + Facebook + TikTok + Reddit/Quora 轻量验证 + App Store/Google Play/LinkedIn 实体证明
```

```text
Zhiyu 是一个真实存在的中文学习平台，适合越南、泰国、印尼学习者，用来学习 HSK、拼音、词汇、发音和中国文化。
```

## 二、核心玩法

### 2.1 基础规则

1. 4×4 或 5×5 方格棋盘，摆放汉字方块（有 1 个空位）
2. 方块上的汉字被**打乱顺序**
3. 玩家通过**滑动方块**（利用空位移动），将汉字排列成：
   - 正确的**成语**（如"守株待兔"）
   - 正确的**句子**（如"今天天气很好"）
   - 正确的**古诗句**（如"床前明月光"）
4. 步数越少、速度越快 → 分数越高
5. 每局随机生成新题目

### 2.2 题目类型

| 类型 | 棋盘 | 难度 | 举例 |
|------|------|------|------|
| 四字成语 | 4×1 行 | ★ | 守株待兔 |
| 五字古诗 | 5×1 行 | ★★ | 床前明月光 |
| 七字古诗 | 7×1 行或 4×2 | ★★★ | 两个黄鹂鸣翠柳 |
| 短句 | 4×4 | ★★★★ | "小明今天去学校上课了" |
| 复合 | 5×5 | ★★★★★ | 成语+句子混合 |

### 2.3 辅助系统

| 功能 | 说明 |
|------|------|
| 提示 | 高亮 1 个方块的正确位置（每局 3 次） |
| 目标预览 | 点击眼睛图标查看目标结果 2 秒 |
| 最优解 | 解题后显示理论最少步数对比 |
| 撤销 | 可撤销最近 3 步 |

## 三、游戏模式

### 3.1 单人·无尽

| 维度 | 说明 |
|------|------|
| 规则 | 连续解题，难度递增 |
| 结束 | 累计 3 次超时（每题限时 60-120 秒） |
| 目标 | 连续解题数最多 |

### 3.2 单人·极速

| 维度 | 说明 |
|------|------|
| 规则 | 3 分钟内解最多题 |
| 题型 | 以简单题为主（4 字成语） |
| 目标 | 最多解题数 |

### 3.3 1v1 PK

| 维度 | 说明 |
|------|------|
| 规则 | 同题竞速，谁先解出谁赢 |
| 赛制 | Bo3（3 题 2 胜） |
| 实时 | 能看到对手的步数（但看不到布局） |

### 3.4 多人竞速

| 维度 | 说明 |
|------|------|
| 人数 | 4 人同题 |
| 规则 | 按完成顺序排名 |
| 加分 | 步数最少者额外加分 |

## 四、视觉与交互

### 4.1 画面风格

| 元素 | 设计 |
|------|------|
| 棋盘 | 木质/竹制底板 |
| 方块 | 木牌/石板风格，篆刻汉字 |
| 移动动画 | 平滑滑动 + 轻微弹性 |
| 解题成功 | 金色闪光 + 成语/句子从方块中"浮出"完整展示 |

### 4.2 横屏布局

```

│  [题号]  [步数]          [⏱时间]  [最优解] │
│                                            │
│          ┌──┬──┬──┬──┐                     │
```

### 4.3 交互

| 操控 | 方式 |
|------|------|
| 移动端 | 点击空位相邻方块 → 自动滑入空位 / 滑动方块方向 |
| 桌面 | 点击方块 / 键盘方向键 |

## 五、上瘾机制

- **最少步数挑战**：解题后显示你的步数 vs 理论最优解，追求完美
- **解题速度排行**：每日一题全服排名
- **收集系统**：解出的成语/诗句收入"已解锁"图鉴
- **段位追求**：华容道专属段位

### 成就

| 成就 | 条件 |
|------|------|
| 初解华容道 | 首次解出 |
| 步数最优 | 10 次达到理论最少步数 |
| 速解大师 | 4 字成语 10 秒内解出 |
| 百题斩 | 累计解出 100 题 |

## 七、技术要点

| 技术点 | 方案 |
|--------|------|
| 棋盘状态 | 二维数组表示方块位置，空位标记 |
| 合法性 | 只允许空位相邻方块移动 |
| 最优解 | A* / IDA* 算法计算最少步数（服务端预算） |
| 题目生成 | 从正确排列随机打乱 N 步（保证可解） |
| 同步 | PK 模式 WebSocket 同步完成状态 |

## 二、核心玩法

### 2.1 基础规则

1. 玩家操控一个**冒险者角色**在像素风地图上探索
2. 遇到怪物触发**回合制战斗**
3. 战斗方式：怪物出示一道 HSK 题目 → 玩家通过答题"攻击"
4. **答对** → 对怪物造成伤害
5. **答错** → 怪物反击，扣玩家 HP
6. 击败怪物 → 获得**经验值 + 金币 + 装备掉落**
7. 角色**永久成长**（等级/装备/技能不重置）

### 2.2 战斗题型

| 题型 | 举例 | 伤害 |
|------|------|------|
| **听力理解** | "听录音，选出正确的理解" | 高 |
| **阅读理解** | "阅读短文，回答问题" | 高 |
| **完形填空** | "选择最合适的词填入空格" | 中 |
| **词汇辨析** | "'反映'和'反应'哪个正确？" | 中 |
| **语法改错** | "找出句中的语法错误并改正" | 低 |

### 2.3 地图与怪物

| 地图层 | 主题 | 怪物 | 难度 | HSK 等级 |
|--------|------|------|------|----------|
| 第 1 层 | 校园 | 错字怪、拼音虫 | ★ | HSK 4 |
| 第 2 层 | 街市 | 语法妖、词汇精 | ★★ | HSK 5 |
| 第 3 层 | 都市 | 阅读兵、听力怪 | ★★★ | HSK 6 |
| 第 4 层 | 学府 | 完形将、改错师 | ★★★★ | HSK 7 |
| 第 5 层 | 巅峰 | 综合王、辨析龙 | ★★★★★ | HSK 8 |
| Boss 层 | 各层末尾 | 大 Boss（多道题） | 特殊 | 综合 |

> 无尽地牢模式：第 5 层后循环加难度，每层怪物更强。

### 2.4 角色系统

| 维度 | 说明 |
|------|------|
| 等级 | 1-99，答题获经验升级 |
| HP | 生命值，升级增加 |
| 攻击力 | 答对造成的伤害基础值 |
| 防御 | 减少怪物反击伤害 |
| 技能 | 答题连对触发特殊技能（如"双倍伤害""回复HP""跳过听力"） |

### 2.5 装备系统

| 装备槽 | 举例 | 获取 |
|--------|------|------|
| 武器（笔） | 铅笔/钢笔/金笔 | 怪物掉落 / 商店 |
| 防具（袍） | 校服/西装/战袍 | 怪物掉落 |
| 饰品（徽章） | 词汇徽/语法徽/听力徽 | 成就奖励 |

> 装备影响属性，稀有装备有特殊效果（如"听力题伤害+30%"）

## 三、游戏模式

### 3.1 单人·冒险（核心模式）

| 维度 | 说明 |
|------|------|
| 规则 | 无尽地牢，角色永久养成 |
| 目标 | 探索更深层，角色更强 |
| 暂停 | 随时暂停退出，进度自动保存 |
| 每日 | 每天体力 100 点（每战消耗 5-10），体力自动恢复 |

### 3.2 1v1 PK

| 维度 | 说明 |
|------|------|
| 规则 | 双方各自面对同一道 HSK 题，同时答题 |
| 胜负 | 先答对者先手攻击对方角色 |
| 赛制 | 5 回合，HP 先到 0 者输 |
| 角色 | 使用各自的角色等级和装备 |

### 3.3 多人 Boss 战（合作）

| 维度 | 说明 |
|------|------|
| 人数 | 2-4 人合作 |
| 规则 | 共同挑战超强 Boss，每人回合答题攻击 |
| 协作 | 不同人答不同题型，团队配合 |
| 奖励 | 击败 Boss 稀有装备掉落（全队共享） |

### 3.4 竞技场

| 维度 | 说明 |
|------|------|
| 规则 | PK 排位赛，段位升降 |
| 赛季 | 30 天一赛季 |
| 奖励 | 赛季结算发放排名奖励 |

## 五、上瘾机制

### 5.1 角色永久成长

- 等级永远不重置，形成长期追求
- 装备收集欲：稀有装备掉率低但效果强
- 技能树：升级解锁不同技能（听力专精/阅读专精/词汇语法专精）

### 5.2 每日 Boss

- 每天全服一个超强 Boss
- 全服玩家合力打伤害（累计制）
- 击败后全服奖励

### 5.3 成就

| 成就 | 条件 |
|------|------|
| 初入江湖 | 击败第 1 层 Boss |
| HSK 达人 | 累计答对 100 道 HSK 题 |
| 装备收藏家 | 收集 20 件不同装备 |
| 满级大佬 | 角色达到 99 级 |
| Boss 终结者 | 击败全部 5 层 Boss |

## 七、技术要点

| 技术点 | 方案 |
|--------|------|
| 地图系统 | Phaser Tilemap，像素地图分层渲染 |
| 战斗系统 | 回合状态机：出题→等待答题→计算伤害→动画→下回合 |
| 角色存档 | 服务端存储角色状态（等级/装备/进度） |
| 题目下发 | 按当前层对应 HSK 等级从服务端请求题目 |
| Boss 战协作 | WebSocket 多人同步，轮流答题 |
| 装备系统 | 随机属性生成（服务端），防篡改 |

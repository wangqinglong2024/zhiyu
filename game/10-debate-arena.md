## 二、核心玩法

### 2.1 基础规则

1. 每局给出一个**辩题**（如"科技是把双刃剑？正方 vs 反方"）
2. 系统随机分配正方/反方
3. 每人有一副**论据卡牌**（5 张手牌）
4. 双方轮流出牌，每张卡牌代表一个论据/论证方法
5. 卡牌有**攻击力（说服力）**和**防御力（逻辑严密度）**
6. 对方可以出"反驳卡"抵消攻击
7. 3 回合后，比较双方累计说服力 → 高者胜

### 2.2 卡牌类型

| 卡牌类型 | 说明 | 攻击力 |
|---------|------|--------|
| **事实论据卡** | 引用数据/事实 | 高（8-10） |
| **道理论据卡** | 引用名言/经典 | 中高（7-9） |
| **类比论证卡** | 类比推理 | 中（5-7） |
| **反证法卡** | 用反例反驳 | 高（8-10），只能在反驳阶段用 |
| **情感牌** | 情感诉求/共鸣 | 低（3-5），但有特殊效果 |
| **逻辑陷阱卡** | 揭示对方逻辑谬误 | 特殊：降低对方上一张牌效果 |

### 2.3 回合流程

```
回合 1：双方各出 1 张论据卡 → 计算说服力
回合 2：双方各出 1 张 → 可出反驳卡抵消对方上回合的牌
回合 3：总结陈词 → 各出最后 1-2 张牌
结算：总说服力对比 → 判定胜负
```

### 2.4 卡牌获取

| 途径 | 说明 |
|------|------|
| 课程学习 | 完成 L10 课程单元解锁新卡牌 |
| 战斗奖励 | 胜利后随机获得新卡 |
| 每日签到 | 每天 1 张随机卡 |
| 成就解锁 | 特殊成就解锁稀有卡 |

## 三、游戏模式

### 3.1 单人·练习

| 维度 | 说明 |
|------|------|
| 对手 | AI 辩手（4 个难度） |
| 目的 | 练习卡牌搭配和辩论策略 |
| 奖励 | 经验值（无段位变化） |

### 3.2 1v1 排位

| 维度 | 说明 |
|------|------|
| 匹配 | ELO 段位匹配 |
| 赛制 | 单局，随机辩题 |

| 维度 | 说明 |
|------|------|
| 人数 | 4 人单淘汰 |
| 规则 | 半决赛 → 决赛 |
| 时间 | 约 10-15 分钟 |
| 奖励 | 冠军丰厚奖励 |

### 3.4 每日辩题

| 维度 | 说明 |
|------|------|
| 规则 | 每天一个热点辩题 |
| 模式 | 正方/反方各阵营，全服对战 |
| 统计 | 全服正方/反方胜率展示 |

## 五、上瘾机制

### 5.1 卡牌收集

- 总共 100+ 张不同卡牌
- 稀有度：普通（白）→ 优秀（绿）→ 稀有（蓝）→ 传说（金）
- 传说卡有特殊论据和华丽特效
- 收集进度可视化

### 5.2 卡组构筑

- 从收集的卡牌中选 15 张组成卡组
- 不同卡组搭配不同策略（事实流/情感流/逻辑流）
- Meta 策略变化（每赛季调整卡牌平衡）

### 5.3 成就

| 成就 | 条件 |
|------|------|
| 初辩 | 完成第一场辩论 |
| 逻辑大师 | 使用反证法卡获胜 20 次 |
| 收藏家 | 收集 50 张不同卡牌 |
| 连胜之王 | 排位连胜 5 场 |
| 辩神 | 达到辩神段位 |

## 七、技术要点

| 技术点 | 方案 |
|--------|------|
| 卡牌引擎 | Phaser 3 Sprite + Tween 动画 |

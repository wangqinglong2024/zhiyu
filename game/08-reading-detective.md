## 二、核心玩法

### 2.1 基础规则

1. 每局给出一篇 **300-800 字短文**（现代文/文言文/新闻/议论文）
2. 文中隐藏 **5 个谜题**（修辞手法、作者意图、逻辑关系、情感变化、文体特征）
3. 玩家阅读文章后，逐个"破案"回答 5 个谜题
4. 全部答对 → "完美破案"（SSS 评级）
5. 速度越快、准确率越高 → 分数越高
6. 每局随机从题库抽取文章

### 2.2 谜题类型

| 谜题类型 | 举例 | 分值 |
|---------|------|------|
| **修辞识别** | "第 3 段用了什么修辞手法？" 比喻/拟人/排比/夸张 | 20 分 |
| **作者意图** | "作者写这段的目的是？" 讽刺/赞美/警示/抒情 | 25 分 |
| **逻辑关系** | "第 2 段和第 4 段是什么关系？" 因果/对比/递进/转折 | 20 分 |
| **情感变化** | "主人公的情感从___变为___" | 20 分 |
| **文体判断** | "本文属于什么文体？" 记叙文/说明文/议论文/散文 | 15 分 |

### 2.3 评级系统

| 评级 | 条件 |
|------|------|
| SSS | 5/5 正确 + 总时间 < 2 分钟 |
| SS | 5/5 正确 |
| S | 4/5 正确 |
| A | 3/5 正确 |
| B | 2/5 正确 |
| C | 1/5 或 0/5 |

## 三、游戏模式

### 3.1 单人·破案

| 维度 | 说明 |
|------|------|
| 规则 | 独立阅读和解谜，追求最高评级 |
| 目标 | SSS 收集（每篇文章追求完美评级） |
| 无限性 | 每次随机文章，永远有新案件 |

### 3.2 1v1 PK

| 维度 | 说明 |
|------|------|
| 规则 | 同一篇文章，同 5 个谜题，比谁先全部答对 |
| 策略 | 速度 vs 准确率的平衡 |
| 胜负 | 先完成 5 题且正确率 ≥ 3/5 者胜 |

### 3.3 多人竞速

| 维度 | 说明 |
|------|------|
| 人数 | 4 人同题 |
| 规则 | 积分制：正确 + 速度 |
| 排名 | 总分排名 |

### 3.4 侦探排位

| 维度 | 说明 |
|------|------|
| 规则 | 段位赛，赢升输降 |
| 段位 | 实习→初级→高级→专家→名侦探 |
| 赛季 | 30 天一赛季 |

### 4.1 画面风格

## 五、上瘾机制

### 5.1 破案手记收集

- 每篇文章解出后，解析笔记收入"破案手记"
- 按文体分类收集：记叙文卷 / 散文卷 / 议论文卷 / 文言文卷
- 集齐某卷 → 解锁专属头衔和皮肤

### 5.2 侦探等级

累计破案数决定等级：实习（10+）→ 初级（50+）→ 高级（200+）→ 专家（500+）→ 名侦探（1000+）

### 5.3 成就

| 成就 | 条件 |
|------|------|
| 首次破案 | 完成第一个案件 |
| 完美侦探 | 累计 10 次 SSS 评级 |
| 修辞专家 | 修辞题累计答对 100 道 |
| 速度之王 | 5 题全对且 < 90 秒 |
| 文体全通 | 每种文体至少破案 10 次 |

| 技术点 | 方案 |

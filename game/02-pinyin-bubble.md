## 二、核心玩法

### 2.1 基础规则

1. 屏幕顶部排列多行**汉字泡泡**，每个泡泡内是一个汉字
2. 屏幕底部中央有一个**拼音发射器**，显示当前拼音弹珠
3. 玩家**瞄准并发射**拼音泡泡，命中对应汉字即可消除
4. **3 个以上**相同声调/偏旁的泡泡相邻可连消，额外加分
5. 消除后悬空的泡泡自动掉落消除（cascade）
6. 顶部泡泡每 30 秒下降一行，**触底则 Game Over**

### 2.2 声调颜色系统

| 声调 | 颜色 | 示例 |
|------|------|------|
| 一声（阴平） | 🔵 蓝色 | mā 妈 |
| 二声（阳平） | 🟢 绿色 | má 麻 |
| 三声（上声） | 🟠 橙色 | mǎ 马 |
| 四声（去声） | 🔴 红色 | mà 骂 |
| 轻声 | ⚪ 白色 | ma 吗 |

> 视觉上通过颜色区分声调，玩家在游戏中自然建立声调直觉。

### 2.3 特殊泡泡

| 泡泡类型 | 效果 | 出现概率 |
|---------|------|---------|
| 🌈 彩虹泡泡 | 可匹配任意汉字 | 5% |
| 💣 炸弹泡泡 | 爆炸消除周围 1 圈 | 3% |
| ⭐ 星星泡泡 | 消除同一行所有泡泡 | 2% |
| 🔥 火焰泡泡 | 消除同一声调所有泡泡 | 1% |

### 2.4 难度递进

| 阶段 | 下降速度 | 泡泡种类 | 特殊规则 |
|------|---------|---------|---------|
| 初始 | 60秒/行 | 10 个汉字 | 纯拼音匹配 |
| 中等 | 45秒/行 | 20 个汉字 | 加入偏旁分类消除 |
| 困难 | 30秒/行 | 30 个汉字 | 同音字干扰 |
| 极难 | 20秒/行 | 40+ 汉字 | 偏旁+声调混合规则 |

## 三、游戏模式

### 3.1 单人·生存（Solo Survival）

| 维度 | 说明 |
|------|------|
| 规则 | 泡泡不断下降，坚持越久分越高 |
| 目标 | 存活时间最长 + 最高分 |
| 结束条件 | 泡泡触底 |
| 记录 | 最高分、最长存活时间、最多连消 |

### 3.2 单人·清屏（Solo Clear）

| 维度 | 说明 |
|------|------|
| 规则 | 每屏固定排列泡泡，全部消除后进入下一屏 |
| 目标 | 用最少射击次数清完 |
| 无限性 | 清完一屏自动下一屏，无限循环 |
| 评价 | 每屏按用弹数评星（3星/2星/1星） |

### 3.3 1v1 PK

| 维度 | 说明 |
|------|------|
| 匹配 | 段位匹配 |
| 规则 | 你消的泡泡会变成**垃圾泡泡**送到对手那边 |
| 策略 | 连消越多，送对手的垃圾越多 |
| 胜负 | 对手先触底者输 |
| 时长 | 不限时，平均 3-5 分钟 |

### 3.4 多人对战（4 人竞速）

| 维度 | 说明 |
|------|------|
| 人数 | 4 人 |
| 规则 | 同一题库，比谁先清完 3 屏 |
| 排名 | 按完成速度排 1-4 名 |

## 四、视觉与交互设计

### 4.1 画面风格

| 元素 | 设计 |
|------|------|
| 背景 | 星空/海洋主题（横屏全景） |
| 泡泡 | 圆形半透明，内嵌汉字，声调颜色外框 |
| 发射器 | 底部中央，箭头指示瞄准方向 |
| 瞄准线 | 半透明弹道线，实时显示弹射路径 |
| 连消特效 | 泡泡炸裂 + 星光飞散 + 分数飘字 |

### 4.2 横屏适配

| 场景 | 处理 |
|------|------|
| **移动端** | 横屏锁定，左侧泡泡区域，右侧信息面板 |
| **桌面** | 16:9 居中，鼠标瞄准+点击发射 |
| **操控** | 移动端：触摸拖拽瞄准→松手发射；桌面：鼠标移动瞄准→点击发射 |

### 4.3 音效

| 事件 | 音效 |
|------|------|
| 发射 | "嘭"发射声 |
| 命中消除 | 泡泡破裂声 + 汉字读音 |
| 连消 | 连锁水泡破裂 + 叮叮叮 |
| 垃圾泡泡（PK） | 沉闷落下声 |
| 特殊泡泡 | 对应特效音 |

## 五、上瘾机制

### 5.1 连消爽感

- 3 连消 → "Good！" + 基础特效
- 5 连消 → "Amazing！" + 华丽特效
- 10 连消 → "Incredible！" + 全屏特效
- cascade 连锁越多，视觉奖励越华丽

### 5.2 成就

| 成就 | 条件 | 奖励 |
|------|------|------|
| 泡泡新手 | 首次清完 1 屏 | 头像框 |
| 精准射手 | 单屏命中率 100% | 称号 |
| 连消大师 | 单局触发 5+ 连消 | 皮肤碎片 |
| 声调达人 | 累计消除 500 个四声泡泡 | 积分 ×300 |
| 泡泡王者 | 段位达到王者 | 限定泡泡样式 |

### 5.3 每日挑战

- **单声调挑战**：只出某一声调的汉字
- **偏旁专场**：按偏旁部首分组消除
- **极速清屏**：限时 60 秒清最多屏
- **无特殊泡泡**：纯技术挑战

## 七、技术要点

### 7.1 Phaser 3 场景结构

```
PinyinBubbleGame
  ├── BootScene           # 资源预加载
  ├── MenuScene           # 模式选择
  ├── GameScene           # 核心游戏场景
  │     ├── BubbleGrid       # 泡泡网格管理（六角形排列）
  │     ├── Launcher         # 发射器+瞄准线
  │     ├── BubblePhysics    # 泡泡碰撞+吸附逻辑
  │     ├── MatchChecker     # 拼音-汉字匹配检测
  │     ├── CascadeHandler   # 连消+悬空检测
  │     └── EffectManager    # 特效+音效
  ├── ResultScene         # 结算
  └── PKScene             # PK 分屏
```

### 7.2 关键技术

| 技术点 | 方案 |
|--------|------|
| 泡泡网格 | 六角形网格（hex grid），泡泡自动吸附到最近网格点 |
| 弹射物理 | Phaser Arcade Physics + 墙壁反弹 |
| 匹配算法 | BFS 搜索相邻同类泡泡 → ≥3 消除 |
| 悬空检测 | 消除后 DFS 检查连通性，不与顶部连通的泡泡掉落 |
| PK 垃圾行 | WebSocket 推送对手连消数 → 本地生成随机垃圾行 |

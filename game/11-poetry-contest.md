## 二、核心玩法

### 2.1 基础规则

1. 混合题型抢答竞技
2. 每局 **8-12 道题**，来自多种题型
3. 抢答机制：题目出现后，先答对的得分
4. 连对加分（Streak 连击）
5. 单局积分排名

### 2.2 题型池

| 题型 | 说明 | 分值 |
|------|------|------|
| **字填诗** | "春眠不觉晓，处处闻___" 填空 | 10 |
| **诗填字** | 给诗句，选择正确的字 | 10 |
| **上下句** | 给上句选下句 / 给下句选上句 | 15 |
| **九宫格** | 9 个字中选出一句五言诗 | 15 |
| **看图猜诗** | 展示意境图，猜对应的诗句 | 20 |
| **听音辨诗** | 播放朗诵音频，辨别是哪首诗 | 20 |
| **诗人连线** | 诗句与诗人配对 | 15 |
| **朝代排序** | 按年代排列 4 首诗的顺序 | 20 |
| **赏析题** | "此句表达了什么情感？" | 25 |
| **出处题** | "此句出自哪首诗/哪位诗人？" | 15 |

### 2.3 Streak 连击系统

| 连对数 | 加成 |
|--------|------|
| 2 连 | ×1.2 |
| 3 连 | ×1.5 |
| 5 连 | ×2.0 |
| 8 连 | ×3.0 |
| 10+ 连 | ×5.0 |

> 答错或超时 → 连击中断，倍率归 1。

## 三、游戏模式

### 3.1 单人·闯关

| 维度 | 说明 |
|------|------|
| 规则 | 连续答题，3 次答错结束 |
| 目标 | 最高分 / 最多连续正确 |
| 难度 | 随答对数递增 |

### 3.2 1v1 抢答

| 维度 | 说明 |
|------|------|
| 规则 | 10 道题，先答对的得分 |
| 特殊 | 抢答时间 8 秒（越快分数越高） |
| 段位 | 段位排位赛 |

### 3.3 多人大会（4-8 人）

| 维度 | 说明 |
|------|------|
| 规则 | 12 道混合题型抢答 |
| 计分 | 积分制，按题型分值 × Streak 倍率 |
| 排名 | 总分排名 |
| 体验 | 模拟电视综艺竞赛感 |

### 3.4 每周百人大会

| 维度 | 说明 |
|------|------|
| 时间 | 每周六晚 8 点（固定时间） |
| 人数 | 100 人海选 → 10 人决赛 |
| 赛制 | 海选 20 题积分 → 前 10 进决赛 → 淘汰赛 |
| 奖励 | 冠军赛季专属皮肤 + 大量积分 |
| 社交 | 实时弹幕/观众投票 |

## 五、上瘾机制

### 5.1 Streak 追求

- 连击倍率形成"不想断"的心理
- 历史最高 Streak 记录
- 全服最长 Streak 排行榜

### 5.2 百人大会社交

- 每周固定时间形成"赴约"习惯
- 比赛中的紧张感和荣耀感
- 观众弹幕互动

### 5.3 诗词段位

专属段位：诗童 → 才子/才女 → 诗人 → 词人 → 诗圣

### 5.4 成就

| 成就 | 条件 |
|------|------|
| 首秀 | 参加第一次大会 |
| 连击达人 | Streak ≥ 10 |
| 百人冠军 | 赢得每周百人大会 |
| 全题型大师 | 每种题型答对 ≥ 50 道 |
| 诗圣 | 段位达到诗圣 |
| 九宫格高手 | 九宫格题 5 秒内答对 20 次 |

## 七、技术要点

| 技术点 | 方案 |
|--------|------|
| 题目池 | 服务端 5000+ 题，按题型/难度分库 |
| 抢答同步 | WebSocket 毫秒级抢答判定（服务端时间戳） |
| 百人场 | WebSocket Room，最多 100 连接 + 观众连接 |
| 音频 | 听音辨诗题的朗诵音频预加载 |
| 图片 | 看图猜诗的 AI 生成意境图预缓存 |
| 防作弊 | 服务端出题 + 验证，客户端不预发答案 |

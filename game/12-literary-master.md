## 二、核心玩法

### 2.1 基础规则

1. **8-16 人**同时参赛
2. 每轮出一道题（混合 L1-L12 全部知识点）
3. 答错或超时 → **淘汰**
4. 答对者继续，直到只剩 **1 人**
5. 越到后面，题目越难，限时越短
6. 最后存活者成为本局"文豪"

### 2.2 综合题型池（涵盖全 12 级）

| 级别来源 | 题型举例 | 难度 |
|---------|---------|------|
| L1-L2 | 汉字笔画、拼音声调、偏旁部首 | ★ |
| L3-L4 | 词汇搭配、语法填空、量词选择 | ★★ |
| L5-L6 | 成语填空、缩句扩句、病句修改 | ★★★ |
| L7-L8 | 古诗默写、修辞手法、阅读理解 | ★★★★ |
| L9-L10 | HSK 综合题、论证辨析、虚词辨析 | ★★★★ |
| L11-L12 | 诗词赏析、写作技法、综合分析 | ★★★★★ |

### 2.3 特殊机制

| 机制 | 说明 |
|------|------|
| **复活币** | 每局 1 次复活机会（答错后可使用） |
| **提示** | 每局 2 次提示（排除 1 个错误选项） |
| **加时** | 每局 1 次延长答题时间 5 秒 |
| **难度跳升** | 后半段题目难度骤升，每题限时递减 |

### 2.4 淘汰进程

```
第 1-3 题：★ 简单题（20秒/题）→ 淘汰不认真的
第 4-6 题：★★★ 中等题（15秒/题）→ 淘汰基础不牢的
第 7-9 题：★★★★ 难题（12秒/题）→ 淘汰大部分人
第 10+ 题：★★★★★ 极难（10秒/题）→ 决出最强者
```

## 三、游戏模式

### 3.1 快速赛（8 人）

| 维度 | 说明 |
|------|------|
| 人数 | 8 人 |
| 时间 | 约 5 分钟 |
| 目标 | 快速体验淘汰刺激 |
| 匹配 | 即时匹配，不限段位 |

### 3.2 排位赛（16 人）

| 维度 | 说明 |
|------|------|
| 人数 | 16 人 |
| 赛制 | 完整淘汰 → 最终 1 人 |
| 段位 | 书生 → 学士 → 翰林 → 大学士 → 文豪 → 圣贤 |
| 星数 | 根据名次 +/- 星数 |

### 3.3 主题赛

| 维度 | 说明 |
|------|------|
| 规则 | 限定题目范围（如"只出古诗词"或"只出 L9-L12"） |
| 目的 | 专项能力测试 |
| 频率 | 每天一个主题 |

### 3.4 赛季总决赛

| 维度 | 说明 |
|------|------|
| 时间 | 赛季末（每 30 天一次） |
| 资格 | 段位"翰林"以上可参加 |
| 人数 | 64 人大赛 |
| 赛制 | 分组淘汰 → 半决赛 → 决赛 |
| 奖励 | 赛季皮肤 + 专属头衔 + 大量积分 |

## 五、上瘾机制

### 5.1 淘汰紧张感

- 每道题都可能是最后一题
- 存活人数实时可见，压迫感递增
- "大逃杀"式紧张刺激

### 5.2 段位追求

专属段位：书生 → 学士 → 翰林 → 大学士 → 文豪 → 圣贤
- 圣贤段位全服限 Top 100

### 5.3 复仇机制

- 被淘汰后可"观战"到结束
- 记录淘汰你的题目 → 推送到 SRS 复习
- 下局再遇同题必答对 → 解锁"复仇"成就

### 5.4 成就

| 成就 | 条件 |
|------|------|
| 初次登殿 | 参加第一场文豪争霸 |
| 首冠 | 第一次获得"文豪"称号 |
| 全对通关 | 一局中 0 错误存活到最后 |
| 连冠 | 连续 3 局获得冠军 |
| 圣贤 | 段位达到圣贤 |
| 知识全面 | 12 种级别的题都答对过 |
| 赛季王者 | 赢得赛季总决赛 |

## 七、技术要点

| 技术点 | 方案 |
|--------|------|
| 房间管理 | WebSocket Room（8/16 人），服务端控制出题节奏 |
| 同步 | 服务端广播题目 → 客户端倒计时 → 收集答案 → 广播淘汰结果 |
| 题库 | L1-L12 全级别题库分库存储，按难度/级别标签 |
| 防作弊 | 服务端出题+验证+计时，答案不预发 |
| 观战 | 被淘汰者自动切换为观战模式（只读 WebSocket） |
| 赛季系统 | 服务端定时任务管理赛季重置/结算 |
| 大赛 | 64 人赛事需预约+分组+多轮 WebSocket 管理 |

## 二、核心玩法

### 2.1 基础规则

1. 系统给出一个**关键字**（如"月""花""风""春"）
2. 双方轮流说出**含有该字的古诗句**
3. 每回合限时 **15 秒**
4. 说出的诗句**不能重复**
5. 15 秒内答不出 → 输掉本局
6. 输入方式：打字 / 从提示候选中选择 / 语音输入（后期）

### 2.2 飞花令进阶规则

| 规则变体 | 说明 | 难度 |
|---------|------|------|
| 基础飞花令 | 诗句中任意位置含关键字 | ★ |
| 定位飞花令 | 关键字必须在第 N 个字位置 | ★★★ |
| 双字飞花令 | 诗句必须同时含 2 个关键字 | ★★★★ |
| 主题飞花令 | 限定朝代/诗人/题材 | ★★★★★ |

### 2.3 诗句展示

每次正确说出诗句后：
- 🖌️ 全屏展示诗句（自动排版）
- 🎵 AI 朗诵音频自动播放
- 🖼️ 配套意境背景画
- 📖 显示作者、出处、朝代

## 三、游戏模式

### 3.1 单人·练习（vs AI）

| 维度 | 说明 |
|------|------|
| AI 难度 | 初级（只用常见诗）/ 中级 / 高级 / 诗仙（冷门诗也出） |
| 目标 | 练习积累，一直对到自己词穷 |
| 记录 | 最长对诗数 |

### 3.2 1v1 对决

| 维度 | 说明 |
|------|------|
| 匹配 | 段位匹配 |
| 规则 | 严格飞花令规则，15 秒/回合 |
| 赛制 | Bo3（每局不同关键字） |
| 星数 | 胜 +1🌟，败 -1🌟 |

### 3.3 多人淘汰（4-8 人）

| 维度 | 说明 |
|------|------|
| 规则 | 轮流答题，答不出淘汰 |
| 时间 | 每人 10 秒（随人数减少递减） |
| 胜负 | 最后 1 人存活获胜 |

### 3.4 每日主题挑战

| 维度 | 说明 |
|------|------|
| 规则 | 每天一个特殊关键字，全服参与 |
| 排名 | 按最长对诗数排名 |
| 奖励 | 前 100 名丰厚积分 |

## 五、上瘾机制

### 5.1 诗词储备量可视化

- "你已掌握 X 首古诗词" 进度条
- 按朝代/诗人分类的诗词图谱
- 集齐某诗人全部作品 → 解锁该诗人"传记卡"

### 5.2 段位系统

专属段位：诗童 → 秀才 → 举人 → 进士 → 状元 → 诗仙

### 5.3 成就

| 成就 | 条件 |
|------|------|
| 诗词入门 | 飞花令中说出 10 首不同诗 |
| 李白门徒 | 使用 10 首李白诗句 |
| 冷门诗人 | 使用冷门诗人作品 5 次 |
| 飞花百回 | 单局对诗 ≥ 20 回合 |
| 诗仙降临 | 段位达到诗仙 |

## 七、技术要点

| 技术点 | 方案 |
|--------|------|
| 诗词库 | 服务端古诗词数据库（5000+ 首，含全文/作者/朝代） |
| 关键字索引 | 按每个汉字建立倒排索引 → 快速查"含X字的诗句" |
| 验证 | 服务端验证诗句存在 + 含关键字 + 未重复 |
| 候选生成 | 从合法诗句中随机选 3 个作为提示 |
| AI 对手 | 按难度从候选池中选（简单选 Top 热门，困难选冷门） |
| 音频 | Azure TTS 实时生成朗诵音频 / 常用诗句预缓存 |

## 一、关卡系统概述

### 1.1 总量与分布

| 区域 | 关卡数 | 关卡类型分布 | Boss 关 |
|------|--------|------------|---------|
| 拼音群岛 | 20 | 知识教学 14 + 听力挑战 5 + Boss 1 | 第 20 关 |
| 汉字谷地 | 15 | 知识教学 7 + 汉字书写 6 + Boss 1 + 听力 1 | 第 15 关 |
| 词汇平原 | 15 | 词汇战役 10 + 听力挑战 3 + Boss 1 + 知识教学 1 | 第 15 关 |
| 语法要塞 | 10 | 知识教学 4 + 词汇战役 3 + 听力挑战 2 + Boss 1 | 第 10 关 |
| **合计** | **60** | | **4 个 Boss** |

### 1.2 关卡通用框架

每个关卡由以下阶段组成：

```
关卡加载 → [可选：过场动画] → 教学阶段 → 练习/挑战阶段 → 结算画面
```

### 1.3 关卡通用 UI 框架（游戏画面内 — 横屏 16:9）

> 所有关卡均在游戏模式下运行，H5 和 PC 统一横屏 16:9 全屏。详见 [响应式布局](10-responsive-layout.md)。

```
┌────────────────────────────────────────────────┐
│ ← 暂停   Bài 1-1   ⭐⭐☆         ❤️❤️❤️  🪙+20 │ ← 顶栏：返回/暂停 + 关卡编号 + 星级 + 生命 + 金币
│ ━━━━━━━━━━━━━━━━━━━━━━━━━━░░░░  68%            │ ← 进度条
├─────────────────────────┬──────────────────────┤
│                         │                      │
│    游戏/教学主内容区域     │   操作/选项区域        │ ← 横屏优势：左右分栏
│    （内容展示/题目）       │   （选项按钮/交互）     │
│                         │                      │
│                         │                      │
└─────────────────────────┴──────────────────────┘
```

**暂停菜单**（点击左上角暂停按钮）：
- 弹出半透明遮罩 + 居中弹窗
- 选项：「Tiếp tục」（继续） / 「Chơi lại」（重玩） / 「Thoát」（退出回地图）
- 退出时二次确认：进度不保存

**生命值系统**：
- 每关开始 3 颗心（❤️❤️❤️）
- 答错 1 题扣 1 颗心
- 心耗尽 → Game Over → 选择「重玩」或「退出」
- 免费用户心耗尽后需等 30 分钟恢复，或看激励视频广告恢复 1 颗心
- 会员用户心无限（不显示心，关卡内答错仍有反馈但不会 Game Over）

## 二、关卡类型一：知识教学关

### 2.1 用户故事

> 作为零基础学习者，我需要先被教会新知识，然后通过即时练习巩固，以便真正学会。

### 2.2 关卡结构

```
教学阶段（Learning Phase）
  → 讲解新知识点（有动画演示 + 音频播放 + 文字说明）
  → 用户主动点击「Tiếp」（下一步）推进
  → 每次教学 3-5 个新知识点

练习阶段（Practice Phase）
  → 将刚学的知识混合练习
  → 8-12 道练习题
  → 答对有正面反馈（绿色 ✓ + 音效 + 金币 +5）
  → 答错有纠正反馈（红色 ✗ + 震动 + 显示正确答案 + 扣心）
```

### 2.3 教学阶段交互细节

**拼音教学示例**（第 1 关：声母 b/p/m/f）— 横屏 16:9：

```
┌────────────────────────────────────────────────┐
│ ← 暂停  Bài 1-1  ⭐☆☆  ━━━░░░░░░░  25%  ❤❤❤  │
├────────────────────────┬───────────────────────┤
│                        │                       │
│          b             │  "b" phát âm như       │
│                        │  tiếng Việt chữ "b"   │
│     🔊 [发音按钮]       │  trong "bàn"          │ ← 左侧大字 + 发音
│                        │                       │    右侧越南语讲解
│                        │  [  Tiếp（下一步）  ]  │
│                        │                       │
└────────────────────────┴───────────────────────┘
```

- 每个知识点有「再听一次」按钮，可重复播放发音
- 越南语讲解尽量用越南人熟悉的对比（如拼音发音与越南语的对应关系）
- 教学阶段无时间限制，用户自主推进

### 2.4 练习阶段题型（知识教学关通用）

| 题型 | 描述 | 使用场景 |
|------|------|---------|
| **听音选择** | 播放音频，从 4 个选项中选正确的 | 拼音/声调/词汇辨识 |
| **看图选字** | 显示图片，选择对应的汉字/拼音 | 词汇学习 |
| **配对连线** | 左列汉字/拼音，右列释义，拖拽连线 | 词义匹配 |
| **填空选择** | 句子中空出一个词，从 3 个选项中选 | 语法/词汇 |
| **排序组句** | 打散的词语卡片，拖拽排列成正确句子 | 语法/句型 |
| **判断正误** | 显示词/句 + 音频，判断是否匹配 | 综合复习 |

**题型交互规范**：

**听音选择题**（横屏 16:9）：
```
┌────────────────────────────────────────────────┐
│ ← 暂停  Bài 1-1  ⭐☆☆  ━━━━━━━░░░  60%  ❤❤❤  │
├────────────────────────┬───────────────────────┤
│                        │                       │
│   🔊 [自动播放发音]     │  ┌──────┐ ┌──────┐   │
│      点击可重复播放      │  │  bā  │ │  pā  │   │ ← 4 个选项
│                        │  └──────┘ └──────┘   │
│                        │  ┌──────┐ ┌──────┐   │
│                        │  │  mā  │ │  fā  │   │
│                        │  └──────┘ └──────┘   │
└────────────────────────┴───────────────────────┘
```
- 选中选项后立即判定（无需确认按钮）
- 正确：选项变绿 + ✓ 图标 + "Đúng rồi!"（答对了！）+ 音效
- 错误：选项变红 + ✗ 图标 + 正确选项高亮绿色 + "Đáp án đúng là..."（正确答案是...）
- 停留 1.5 秒后自动进入下一题

**配对连线题**：
```
┌─────────────────────────────┐
│  左列          右列          │
│  ┌────┐       ┌────────┐    │
│  │ 大  │ ───→  │ to, lớn │   │ ← 拖拽画线
│  └────┘       └────────┘    │
│  ┌────┐       ┌────────┐    │
│  │ 小  │       │ nhỏ, bé │   │
│  └────┘       └────────┘    │
│  ┌────┐       ┌────────┐    │
│  │ 好  │       │ tốt,hay │   │
│  └────┘       └────────┘    │
└─────────────────────────────┘
```
- H5 端：手指拖拽画线连接
- PC 端：鼠标点击左列项 → 再点击右列项完成连线
- 全部连线完成后统一判定

**排序组句题**（横屏 16:9）：
```
┌────────────────────────────────────────────────┐
│  Sắp xếp thành câu đúng（排列成正确的句子）       │
├────────────────────────┬───────────────────────┤
│                        │                       │
│  答题区域（空白横线）：   │  词卡区域（乱序）：      │
│  ┌──┐ ┌──┐ ┌──┐ ┌──┐ │  ┌────┐ ┌──┐         │
│  │__│ │__│ │__│ │__│ │  │ 吃饭 │ │我│         │ ← 可拖拽的词卡
│  └──┘ └──┘ └──┘ └──┘ │  └────┘ └──┘         │
│                        │  ┌──┐ ┌────┐         │
│                        │  │想│ │ 要  │         │
│                        │  └──┘ └────┘         │
│                        │                       │
│  [   Kiểm tra（检查）]  │                       │
└────────────────────────┴───────────────────────┘
```
- 点击词卡 = 放入下一个空位
- 再次点击已放置的词卡 = 取回
- 也可拖拽调整顺序
- 点击「检查」按钮统一判定

### 2.5 过关标准（知识教学关）

| 星级 | 条件 |
|------|------|
| ★☆☆ | 练习准确率 ≥ 60%（如 12 题对 8 题） |
| ★★☆ | 练习准确率 ≥ 80% |
| ★★★ | 练习准确率 ≥ 95%（≤ 1 题错误） |
| 未过关 | 准确率 < 60% 或生命值耗尽 |

## 三、关卡类型二：词汇战役关

### 3.1 用户故事

> 作为已有一定基础的学习者，我想在有游戏感的情境中运用词汇，以便在实战中巩固记忆。

### 3.2 关卡结构

```
过场动画（可选，20 秒剧情引入）
  → 词汇收集阶段（限时配对/拖拽）
  → 句型组装阶段（排序组句）
  → Boss 对话阶段（选择正确回答）
  → 结算
```

### 3.3 三阶段详细设计

**阶段一：词汇收集**（限时 60 秒）

```
┌─────────────────────────────┐
│  ⏱️ 00:45                    │ ← 倒计时
│                             │
│  ┌─────┐  ┌─────┐  ┌─────┐ │
│  │ 🍚  │  │ 🥤  │  │ 🍜  │ │ ← 物品图片（3-4 个）
│  │     │  │     │  │     │ │
│  └──┬──┘  └──┬──┘  └──┬──┘ │
│     │       │       │      │
│  ┌──┴──────┴──────┴────┐   │
│  │ 🀄 米饭  🀄 水  🀄 面条 │  │ ← 汉字标签（拖到对应图片上）
│  │ 🀄 茶   🀄 饺子        │  │    含干扰项
│  └─────────────────────┘   │
└─────────────────────────────┘
```

- 将汉字标签拖拽到对应图片上
- 正确匹配：图片+标签绿色高亮 + 粒子特效 + 得分 +10
- 错误匹配：图片抖动 + 标签弹回 + 扣心
- 图片可以从 HTML/CSS 用 emoji 表示，或使用 AI 生成的小图标

**阶段二：句型组装**（3-5 题）

- 与知识教学关的「排序组句题」相同交互方式
- 句子内容与本关词汇主题相关
- 每题拼对后播放完整句子的标准发音

**阶段三：Boss 对话**（3 回合）

```
┌─────────────────────────────┐
│                             │
│  ┌─────────────────────┐    │
│  │    NPC 立绘（上半身） │    │
│  │                     │    │
│  └─────────────────────┘    │
│                             │
│  ┌─────────────────────┐    │
│  │ NPC: "你要什么？"    │    │ ← 对话框（中文 + 拼音 + 越南语）
│  │ Nǐ yào shénme?      │    │
│  │ Bạn muốn gì?        │    │
│  └─────────────────────┘    │
│                             │
│  选择回答:                    │
│  A) 我要一碗米饭 ✓           │ ← 正确选项
│  B) 你好吗                   │ ← 干扰选项
│  C) 这是什么                 │ ← 干扰选项
└─────────────────────────────┘
```

- NPC 提问带有中文语音播放
- 3 个选项，选对推进，选错扣分但不影响流程
- 全部对话结束 = Boss 击败

### 3.4 过关标准（词汇战役关）

| 星级 | 条件 |
|------|------|
| ★☆☆ | 整体正确率 ≥ 60% 且 Boss 对话至少答对 2/3 |
| ★★☆ | 整体正确率 ≥ 80% 且用时 < 平均值 |
| ★★★ | 整体正确率 ≥ 95% 且 Boss 全对 |

## 四、关卡类型三：汉字书写关

### 4.1 用户故事

> 作为汉字初学者，我想通过手写练习掌握正确的笔画和笔顺，以便记住汉字怎么写。

### 4.2 关卡结构

```
笔画/笔顺教学 → 描红练习（引导模式）→ 自由书写/笔画组装 → 结算
```

### 4.3 教学阶段

```
┌─────────────────────────────┐
│                             │
│        ┌─────────┐          │
│        │         │          │
│        │    大    │          │ ← 大字展示目标汉字
│        │         │          │
│        └─────────┘          │
│                             │
│  🔊 dà (lớn)               │ ← 发音 + 越南语释义
│                             │
│  笔顺动画：                  │
│  1. 横（一）→ 2. 撇（ノ）    │ ← 动画逐笔演示
│  → 3. 捺（＼）               │
│                             │
│  [  再看一遍  ]  [ 开始练习 ] │
└─────────────────────────────┘
```

- 笔顺动画：在田字格中一笔一笔动态绘制汉字
- 每笔停顿 0.5 秒，展示笔画名称
- 可重复播放

### 4.4 练习阶段——触屏设备（H5 端）

**描红练习**：
```
┌─────────────────────────────┐
│      ┌───────────┐          │
│      │ · · · · · │          │
│      │ · 大 · · │          │ ← 田字格 + 灰色虚线汉字
│      │ · · · · · │          │
│      └───────────┘          │
│                             │
│  沿虚线描写...                │
│  用手指在田字格内描写          │
│                             │
│  笔画 1/3  ✓               │ ← 当前进度
└─────────────────────────────┘
```

- 灰色虚线显示目标汉字
- 用户用手指在屏幕上描写
- 系统检测笔画是否沿正确路径（容差范围内）
- 每个汉字描红 2 次
- 描红不扣分，是引导练习

**自由书写**：
- 显示目标汉字的拼音 + 释义，但不显示汉字本身
- 用户凭记忆在空白田字格中书写
- 简化识别：只判断关键笔画路径是否大致正确（不要求书法级精美）

### 4.5 练习阶段——PC 端（鼠标设备）

**笔画拖拽组装**：

由于 PC 端鼠标手写体验差，改为拖拽笔画组件的方式：

```
┌─────────────────────────────┐
│      ┌───────────┐          │
│      │           │          │
│      │  [田字格]  │          │ ← 将笔画拖入田字格的正确位置
│      │           │          │
│      └───────────┘          │
│                             │
│  可用笔画：                   │
│  ┌──┐ ┌──┐ ┌──┐ ┌──┐      │
│  │一│ │丿│ │㇏│ │丨│      │ ← 可拖拽的笔画组件
│  └──┘ └──┘ └──┘ └──┘      │    含干扰笔画
│                             │
│  提示："大" = 横 + 撇 + 捺    │ ← 笔顺提示
└─────────────────────────────┘
```

- 从底部笔画区拖拽正确笔画到田字格中
- 必须按正确笔顺依次拖入（先横后撇后捺）
- 拖入错误笔画 → 弹回 + 提示正确笔画
- 拖入正确笔画 → 吸附到正确位置 + 绿色特效
- 含 1-2 个干扰笔画

### 4.6 过关标准（汉字书写关）

| 星级 | 条件 |
|------|------|
| ★☆☆ | 描红完成 + 自由书写/组装正确 ≥ 60% |
| ★★☆ | 全部完成 + 正确率 ≥ 80% |
| ★★★ | 全部完成 + 正确率 100% + 笔顺全部正确 |

## 五、关卡类型四：听力挑战关

### 5.1 用户故事

> 作为学习者，我想通过纯听力练习提升中文听力理解能力，以便在真实场景中听懂中文。

### 5.2 关卡结构

```
指令说明 → 听力挑战（多种题型交替）→ 结算
```

### 5.3 题型

**题型一：听指令操作**

```
┌─────────────────────────────┐
│  🔊 "向左走"                 │ ← 播放中文指令音频
│  (再听一次)                  │
│                             │
│  ┌───────────────────────┐  │
│  │        🧍              │  │ ← 简易迷宫/格子地图
│  │   ← · ↑ · →           │  │    角色在中间
│  │        ↓               │  │
│  └───────────────────────┘  │
│                             │
│   ←     ↑     →     ↓      │ ← 方向按钮
└─────────────────────────────┘
```

- 播放中文方向指令（向左/向右/向前/向后/停）
- 用户按方向按钮操作角色移动
- 共 10 条指令，指令逐渐变复杂：
  - 简单："向左走"
  - 中等："先向右，再向前走两步"
  - 困难："走到红色的门那里"

**题型二：听音选图**

```
┌─────────────────────────────┐
│  🔊 播放词语/短句             │
│                             │
│  ┌─────┐  ┌─────┐          │
│  │ 🍎  │  │ 🍊  │          │ ← 4 个图片选项
│  └─────┘  └─────┘          │
│  ┌─────┐  ┌─────┐          │
│  │ 🍇  │  │ 🍌  │          │
│  └─────┘  └─────┘          │
└─────────────────────────────┘
```

- 播放中文词语或短句
- 选择对应的图片
- 即时判定

**题型三：听句补空**

```
┌─────────────────────────────┐
│  🔊 播放句子（含空缺）        │
│                             │
│  "我想要一____水"             │
│                             │
│  A) 杯   B) 个   C) 瓶     │
└─────────────────────────────┘
```

- 播放包含目标词的完整句子
- 文字显示中空出一个词
- 选择正确的词填空

### 5.4 过关标准（听力挑战关）

| 星级 | 条件 |
|------|------|
| ★☆☆ | 正确率 ≥ 60%（10 题对 6 题） |
| ★★☆ | 正确率 ≥ 80% |
| ★★★ | 正确率 ≥ 95% |

## 六、Boss 关卡设计

### 6.1 Boss 关特点

- 每个地图区域最后一关是 Boss 关
- Boss 关综合该区域所有知识点
- 题目数量更多（15-20 题）
- 无教学阶段，纯挑战
- Boss 有 NPC 形象（带有立绘的角色）
- Boss 被击败后播放胜利动画 + 解锁下一区域

### 6.2 四个 Boss 关设计

- 「下一关」→ 如果下一关已解锁直接进入，否则返回世界地图

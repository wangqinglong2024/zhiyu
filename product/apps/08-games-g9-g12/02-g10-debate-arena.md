## 一、游戏概述

| 属性 | 说明 |
|------|------|
| **一句话描述** | 围绕辩题打出论据卡牌，用预设数值碾压对手——论据就是武器，逻辑就是铠甲 |
| **灵感来源** | 卡牌对战（炉石传说简化版 CCG） |
| **题库范围** | L10 课程议论文素材、论证方法（举例论证/对比论证/引用论证/比喻论证）、名言警句、成语 400 个。卡牌文案来源于题库，**数值由策划预设，非 AI 动态评分** |
| **单局时长** | 3-5 分钟 |
| **情绪目标** | 策略博弈、智力对抗、收集满足、荣耀感 |

### 关键设计决策

> **G10 辩论擂台明确不使用 AI 动态评分。** 所有卡牌的攻击力（说服力）和防御力（逻辑严密度）均为**策划预设的固定数值**，完全类似传统 CCG（集换式卡牌游戏）的确定性数值对战。
>
> **原因**：
> 1. AI 评分不可控且不可复现，无法保证竞技公平
> 2. 固定数值让玩家可以学习卡牌策略，形成 Metagame
> 3. 降低服务端复杂度，避免依赖大模型实时推理
> 4. 玩家可以通过收集和组合卡牌获得策略深度

### 用户故事

| 角色 | 故事 | 验收 |
|------|------|------|
| 作为策略型玩家 | 我想通过收集强力论据卡牌和合理出牌策略赢得辩论，以便体验策略对战的乐趣 | 卡牌数值固定可见，策略可学习 |
| 作为收集型玩家 | 我想收集各种稀有论据卡牌并升级，以便构建最强卡组 | 卡牌收集系统完整，有稀有度分级 |
| 作为学习者 | 我想在辩论中学到议论文写作技巧和论证方法，以便提升写作能力 | 卡牌文案包含真实论证素材 |

## 二、游戏子模式

### 2.1 子模式总览

| 子模式 | 人数 | 影响段位 | 结束条件 | 特殊规则 |
|--------|------|---------|---------|---------|
| **单人·练习** | 1 人 | 否 | 3 回合结束 | vs AI 辩手（按卡牌数值+简单策略），AI 难度可调（初级/中级/高级） |
| **1v1 辩论** | 2 人 | 是 | 3 回合结束 | 真人对战，限时出牌 |
| **多人辩论赛** | 4 人 | 是 | 两轮淘汰（半决赛→决赛） | 4 人抽签分组，半决赛 2 场→决赛 1 场 |
| **每日辩题** | 不限 | 否 | 3 回合结束 | 全服同一辩题，投票最佳辩手（按胜率+出牌精彩度排名） |

### 2.2 单人·练习模式

- AI 对手按难度等级使用不同策略：
  - **初级 AI**：随机出牌
  - **中级 AI**：优先出高攻击力卡牌
  - **高级 AI**：根据己方出牌选择最优反驳卡
- AI 手牌从对应难度的卡池中随机抽取
- 胜利获得少量经验值和金币，用于卡牌升级

### 2.3 1v1 辩论模式

- 匹配同段位对手
- 系统随机从辩题库抽取辩题 + 随机分配正/反方
- 每回合出牌限时 30 秒
- 3 回合后总说服力高者胜
- 胜 +1 星，负 -1 星

### 2.4 多人辩论赛模式

- 4 人匹配成功后抽签分组
- 半决赛：A vs B，C vs D（各 3 回合）
- 决赛：两场胜者对决（3 回合）
- 冠军 +2 星，亚军 +1 星，半决赛负者不扣星

### 2.5 每日辩题模式

- 每日 00:00 UTC 发布一个全服统一辩题
- 玩家与 AI 对战（高级难度），但出牌记录被保存
- 按胜率 + 出牌策略丰富度综合评分排名
- 前 3 名获得「今日最佳辩手」称号 + 知语币奖励（+50/+30/+20）

## 四、核心交互

### 4.1 辩论开始

**触发**：匹配成功 / 选择 AI 对手后

**视觉流程**：
1. **辩论赛场入场**（0ms）：画面渐入辩论赛场背景（传统书院风格，木质讲台、毛笔字横幅）
2. **辩题展示**（300ms）：屏幕中央大字展示辩题（H1 32px，白色），持续 2 秒
3. **正反方分配**（2300ms）：辩题下方出现分配结果：
   - `你是【正方】：支持"应该限制"` / `你是【反方】：反对"应该限制"`
   - 正方 Sky 蓝色标签，反方 Rose 红色标签
4. **抽牌动画**（3300ms）：5 张卡牌从牌堆飞入手牌区域（每张间隔 200ms），正面依次翻转展示
5. **开始提示**（4500ms）：`第 1 回合 · 开始！`（H2 24px，Amber 色），淡入淡出 1 秒
6. **倒计时启动**（5500ms）：30 秒出牌倒计时开始

### 4.2 选择和出牌

#### 4.2.1 选择卡牌

**触发**：点击/长按手牌中的某张卡牌

**视觉反馈**：
1. **卡牌提升**（0ms）：被选中卡牌向上位移 30px，其余手牌微缩至 95%，选中卡牌缩放至 110%
2. **卡牌光效**（0ms）：选中卡牌边框变为稀有度对应颜色光晕（2px→4px 发光）
3. **详情浮层**（长按 500ms 触发）：卡牌旁弹出详情浮层（200×120px 毛玻璃面板），展示完整论据文案 + 出处 + 数值 + 论证方法解释
4. **取消选择**：点击其他区域 → 卡牌回落，光效消失

#### 4.2.2 拖拽出牌

**触发**：选中卡牌后拖拽至出牌区域

**视觉反馈**：
1. **拖拽跟手**（0ms）：卡牌跟随手指/鼠标移动，卡牌缩放至 80%，半透明化（80% 不透明度）
2. **进入出牌区**：出牌区域虚线变实线 + Sky 蓝色光晕 + 区域微放大
3. **释放出牌**：
   - 卡牌吸附至出牌区域中央，恢复 100% 大小和不透明度
   - 卡牌正面完整展示在出牌区
   - 出牌区域背景变为对应稀有度颜色半透明
   - 手牌中对应位置消失，其余手牌重新排列（间距微调）
4. **出牌确认**：出牌后底部出现 `[确认出牌]`（Rose 红色主按钮）和 `[收回]`（灰色次按钮）
   - 确认前可收回（卡牌飞回手牌）
   - 确认后不可撤回
5. **等待对方**：确认出牌后，己方出牌区显示已出卡牌，等待对方出牌

**音效**：卡牌拖拽音（纸牌滑动）+ 出牌确认音（印章盖戳感）

#### 4.2.3 点击出牌（替代拖拽）

- 选中卡牌后，点击出牌区域 → 卡牌飞入出牌区域（300ms 抛物线轨迹）
- 后续流程与拖拽出牌相同

### 4.3 回合结算

**触发**：双方均确认出牌（或一方超时自动出牌）

**视觉流程**：
1. **双方卡牌揭示**（0ms）：对方出牌区域的卡牌翻转至正面（3D 翻牌动画 500ms）
2. **卡牌对冲飞行**（500ms）：双方卡牌从出牌区域向中央 VS 分隔线飞出，在中央碰撞
3. **碰撞特效**（800ms）：碰撞点产生粒子爆发（正方蓝色粒子 + 反方红色粒子）
4. **数值结算展示**（1200ms）：碰撞点上方展示结算过程：

```
┌────────────────────────────────────────────┐
│   正方出牌：举例论证                         │
│   ⚔️ 说服力 85                              │
│                VS                           │
│   反方出牌：对比论证                         │
│   🛡️ 防御力 40 → 抵消 40 说服力             │
│   ⚔️ 说服力 70                              │
│                                             │
│   本回合结果：                               │
│   正方净说服力：85 - 40 = +45               │
│   反方净说服力：70 - 0 = +70 ← 本回合优势！  │
└────────────────────────────────────────────┘
```

5. **说服力条更新**（2000ms）：顶部双方说服力数值平滑增加，优势方数值 Amber 金色闪烁
6. **回合结束提示**（2500ms）：`第 2 回合结束 · 反方领先！`（Body 16px，领先方颜色）
7. **下回合准备**（3500ms）：提示淡出 → 进入下一回合出牌阶段

### 4.4 说服力结算规则

> **核心规则：完全基于预设固定数值，无 AI 动态评分。**

| 步骤 | 计算方式 | 说明 |
|------|---------|------|
| **1. 双方出牌** | 正方出 A 卡（ATK_a, DEF_a），反方出 B 卡（ATK_b, DEF_b） | — |
| **2. 防御抵消** | 正方净说服力 = ATK_a - DEF_b（最低为 0） | 对方防御力抵消我方攻击力 |
| | 反方净说服力 = ATK_b - DEF_a（最低为 0） | 同上 |
| **3. 累加** | 双方净说服力分别累加至总说服力 | 3 回合累计 |
| **4. 论证克制** | 特定论证方法有 20% 克制加成（详见 §4.5） | 策略深度 |
| **5. 最终判定** | 3 回合后总说服力高者胜 | 平局时比较最后一回合净说服力 |

### 4.5 论证方法克制关系

| 论证方法 | 克制 | 被克制 | 克制效果 |
|---------|------|-------|---------|
| **举例论证** 🔖 | 比喻论证 🎭 | 对比论证 ⚖️ | ATK +20% |
| **对比论证** ⚖️ | 举例论证 🔖 | 引用论证 📜 | ATK +20% |
| **引用论证** 📜 | 对比论证 ⚖️ | 比喻论证 🎭 | ATK +20% |
| **比喻论证** 🎭 | 引用论证 📜 | 举例论证 🔖 | ATK +20% |

- 克制关系在出牌后结算时自动生效
- 克制触发时，克制方卡牌发出对应颜色光芒 + `克制！+20%` 浮字

### 4.6 反驳卡机制

每张卡牌同时具备攻击（说服力）和防御（逻辑严密度）两个维度：
- **出牌时**：卡牌的 ATK 作为进攻值，DEF 作为被动防御值
- **无额外反驳卡**：防御力始终生效，是卡牌的固有属性
- 高防御力卡牌可有效抵消对方的高攻击力
- 策略选择：出高 ATK 低 DEF（全攻型）还是均衡型，取决于对局判断

### 4.7 辩论结束

**触发**：3 回合全部结束

**视觉流程**：
1. **画面暗淡**（0ms）：全屏覆盖半透明暗层
2. **结果揭晓**（500ms）：屏幕中央大字显示：
   - 胜利方：`正方获胜！` 或 `反方获胜！`（H1 32px，胜方颜色 + 光芒特效）
   - 平局：`旗鼓相当！`（H1 32px，Amber 色）
3. **总说服力对比**（1000ms）：
   - 横向双条形图：正方蓝色条 | 反方红色条
   - 数值标注（如 `正方 245 : 反方 210`）
4. **每回合回顾**（1500ms）：3 行回顾每回合双方出牌和净说服力
5. **停留 3 秒**后过渡到通用结算页

**音效**：胜利 → 华丽胜利音乐；失败 → 低沉惜败音；平局 → 悬疑/僵持音

## 七、PK 模式特殊规则

### 7.1 手牌公平性

- PK 模式下，双方手牌从**各自卡组**中随机抽取 5 张
- 为保证公平性，双方手牌的**总 ATK 差异不超过 15%**
  - 若差异过大，系统会为弱势方替换 1 张低稀有度卡牌为更高稀有度（从公共池补充）
- 此规则对玩家透明，不展示调整过程

### 7.2 出牌时间压力

| 回合 | 出牌时限 | 说明 |
|------|---------|------|
| 第 1 回合 | 30 秒 | 充足时间思考策略 |
| 第 2 回合 | 25 秒 | 略微紧迫 |
| 第 3 回合 | 20 秒 | 关键决策回合，时间最短 |

### 7.3 精彩回放

- PK 结束后，系统自动生成「精彩回放」（3 回合出牌过程快速重播 15 秒）
- 回放数据保存在服务端，玩家可在个人中心回顾历史对战
- 支持分享回放卡片（含辩题 + 结果 + 关键出牌截图）

## 八、上瘾机制

### 8.1 卡牌收集与养成

- 每周最佳卡组排行 → 策略分享社区话题

### 8.3 辩神称号追求

- 5 级称号体系 → 长期目标
- 称号在对战中可见 → 社交炫耀

### 8.4 每日辩题的日常活跃

- 每日一个新辩题 → 日常仪式感
- 全服排名竞争 → 每日挑战动力
- 前 3 名「今日最佳辩手」→ 短期荣耀

| **拖拽中** | 卡牌被拖拽移动 | 跟手移动 + 出牌区高亮 |

| **回合切换** | 进入下一回合 | 回合提示 + 新一轮倒计时 |
| **辩论结束** | 3 回合结束 | 结果揭晓 + 说服力对比 + 过渡到结算页 |

### 9.2 PK 模式状态

| 状态 | 描述 | 设计要求 |
|------|------|---------|

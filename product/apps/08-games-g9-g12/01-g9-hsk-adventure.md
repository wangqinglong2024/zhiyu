## 一、游戏概述

| 属性 | 说明 |
|------|------|
| **一句话描述** | 操控角色在像素风地牢中冒险，遇怪答题攻击，角色永久养成——答题就是战斗力 |
| **灵感来源** | 回合制 RPG（勇者斗恶龙/Pokémon 战斗系统简化版） |
| **题库范围** | L9 课程 HSK 7-8 考点（词汇 10,000+、成语 330 个、古诗词 100 首），覆盖听力/阅读/词汇/语法四大模块，题目由服务端从题库随机抽取 |
| **单局时长** | 3-8 分钟（可随时暂停，单人模式进度自动保存） |
| **情绪目标** | 冒险感、成长满足、收集癖、挑战欲 |

### 用户故事

| 角色 | 故事 | 验收 |
|------|------|------|
| 作为 HSK 考生 | 我想通过 RPG 式答题刷 HSK 真题，以便在游戏中不知不觉地提升综合能力 | 答题内容覆盖 HSK 7-8 四大模块 |
| 作为养成型玩家 | 我想看到角色等级提升、装备越来越强，以便获得长期成就感 | 角色数据跨局保存，等级/装备永久保留 |
| 作为 PK 玩家 | 我想在竞技场用自己培养的角色和别人对战，以便展示实力 | 竞技场模式对等化处理，答题决定胜负 |

## 二、游戏子模式

### 2.1 子模式总览

| 子模式 | 人数 | 影响段位 | 结束条件 | 特殊规则 |
|--------|------|---------|---------|---------|
| **单人·冒险** | 1 人 | 否 | 角色 HP 归零（Game Over）或 玩家主动退出 | 无尽地牢探索，角色永久养成，进度自动保存 |
| **1v1 PK** | 2 人 | 是 | 一方角色 HP 归零 或 答完 10 题 | 双方角色属性对等化，纯答题对决 |
| **多人 Boss 战** | 2-4 人 | 否 | Boss HP 归零（胜利）或 全部玩家 HP 归零（失败） | 合作答题攻击超强 Boss |
| **竞技场** | 2 人 | 是 | 同 1v1 PK | 排位赛，按段位匹配，连续胜利可触发段位晋级赛 |

### 2.2 单人·冒险模式

- 玩家操控角色进入「无尽地牢」
- 地牢分层（第 1 层 → 第 2 层 → ……），每层 3-5 个怪物 + 1 个宝箱
- 击败当前层所有怪物 → 出现层传送门 → 进入下一层
- 每 10 层出现 Boss（详见 §六）
- 角色 HP 归零 → Game Over → 保留已获经验和装备 → 下次从最新解锁的检查点（每 5 层一个检查点）继续
- 可随时暂停退出，进度自动保存

### 2.3 1v1 PK 模式

- 双方角色属性**强制对等化**（HP/攻击力/防御力统一为基准值），不受各自养成进度影响
- 双方同时收到同一道题目
- 先答对者发动攻击（造成固定伤害），后答对者造成 70% 伤害
- 答错者被对方攻击（受到固定伤害）
- 共计 10 题，10 题后 HP 高者胜；若 HP 相同则平局
- 胜 +1 星，负 -1 星

### 2.4 多人 Boss 战模式

- 2-4 名玩家合作对战一个超强 Boss
- Boss HP 为基准值 × 玩家人数（如 4 人 Boss HP 为 2000）
- 每轮所有玩家同时答题，答对者对 Boss 造成伤害，答错者被 Boss 反击
- Boss 每 3 轮进行「全体攻击」（所有玩家各受一次伤害）
- Boss HP 归零 → 全队胜利 → 根据各自贡献（正确数/速度）分配奖励
- 全部玩家 HP 归零 → 失败 → 获得少量经验值安慰奖
- 不影响段位

### 2.5 竞技场模式

- 规则与 1v1 PK 相同
- 按段位匹配对手
- 连续 3 胜触发「段位晋级赛」（加星数翻倍）
- 竞技场独有的赛季结算奖励（限定装备皮肤）

## 四、核心交互

### 4.1 地图探索

#### 4.1.1 角色移动

**触发**：点击方向键 / 键盘 WASD / 点击地图某个可通行 Tile

**视觉流程**：
1. **方向键按下**（0ms）：按下的方向键高亮（白色 30% → 白色 60%），角色转向目标方向
2. **角色移动**（0ms）：角色精灵播放 4 帧行走动画，以每 Tile 200ms 的速度移动
3. **地图跟随**（0ms）：摄像机跟随角色居中，边缘平滑过渡
4. **脚步音效**（每步）：轻微的石砖脚步声

#### 4.1.2 自动行走

**触发**：点击「自动行走」按钮

**行为**：
- 按钮变为激活态（Sky 蓝色背景 → Amber 背景 + `自动行走中`）
- 角色自动寻路至最近的未击败怪物
- 到达怪物位置 → 触发战斗
- 击败怪物后 → 自动寻路至下一个怪物
- 所有怪物击败 → 自动走向层传送门
- 玩家点击任意方向键或再次点击按钮 → 取消自动行走

#### 4.1.3 遇怪触发战斗

**触发**：角色与怪物精灵重叠（接触）

**过渡动画**（地图→战斗画面切换）：
1. **画面闪烁**（0ms）：屏幕闪白 2 次（每次 100ms），模拟经典 RPG 遇怪闪屏
2. **黑幕过渡**（200ms）：从角色位置向外展开圆形黑幕，300ms 铺满全屏
3. **战斗画面入场**（500ms）：黑幕从中间撕裂打开，露出战斗画面，角色从左侧滑入，怪物从右侧滑入
4. **VS 动画**（800ms）：屏幕中央闪出 `VS` 字样（Display 48px，Rose 红色），持续 500ms 后淡出
5. **题目入场**（1300ms）：题目面板从底部滑入，倒计时开始

### 4.2 战斗流程

#### 4.2.1 答题攻击（答对）

**触发**：玩家选择正确选项

**视觉反馈**（按时间顺序）：
1. **选项变绿**（0ms）：正确选项背景变为绿色半透明（`rgba(34,197,94,0.2)`），前方显示 ✅
2. **「答对了！」提示**（100ms）：题目区上方飘出 `答对了！` 文字（H3 20px，绿色），300ms 上浮淡出
3. **角色攻击动画**（400ms）：角色精灵播放 3 帧攻击动画（挥剑/释放技能），持续 400ms
4. **攻击特效**（500ms）：从角色方向飞出一道光斩/魔法弹（Rose 红色粒子轨迹），命中怪物
5. **怪物受击**（700ms）：怪物精灵闪烁 3 次（透明度 100%→30% 交替）+ 向后微偏移 8px → 回弹
6. **伤害数字**（700ms）：怪物头顶弹出伤害数字（如 `-45`，Rose 红色 H2 24px），上浮 + 放大 → 缩小淡出，持续 800ms
7. **怪物 HP 条减少**（800ms）：HP 条平滑减少动画（300ms 过渡）
8. **音效**：攻击挥击音 + 命中打击音

**伤害计算**（服务端）：
- 伤害 = 角色 ATK × (1 - 怪物 DEF / (怪物 DEF + 100))
- 暴击（答题速度 <3 秒）：伤害 × 1.5，伤害数字变为 Amber 金色 + `暴击！` 标签

#### 4.2.2 被怪物攻击（答错）

**触发**：玩家选择错误选项 或 倒计时归零

**视觉反馈**：
1. **选项变红**（0ms）：错误选项背景变为 Rose 红半透明，前方显示 ❌
2. **正确答案标记**（200ms）：正确选项背景变绿 + ✅（让玩家看到正确答案）
3. **简短解析**（200ms）：题目区下方展示 1 行解析（Body S 14px，白色 60%），持续 2 秒
4. **怪物攻击动画**（500ms）：怪物精灵播放攻击动画，发出攻击特效（暗红色冲击波）
5. **角色受击**（700ms）：角色精灵闪烁 3 次 + 后退 8px → 回弹
6. **伤害数字**（700ms）：角色头顶弹出受到的伤害数字（Rose 红色），上浮淡出
7. **角色 HP 条减少**（800ms）：HP 条平滑减少
8. **知识点记录**（后台）：答错的题目自动加入 SRS 复习队列

**音效**：错误蜂鸣音 + 怪物攻击音 + 角色受伤呻吟音

#### 4.2.3 怪物被击败

**触发**：怪物 HP 归零

**视觉反馈**：
1. **怪物 HP 条清零**（0ms）：HP 条变空
2. **怪物死亡动画**（200ms）：怪物精灵闪烁 + 逐渐透明 + 向下坠落 → 消散为像素碎片（粒子效果）
3. **胜利音效**（200ms）：短旋律胜利音
4. **掉落物展示**（600ms）：怪物消散位置弹出 2-3 个奖励图标（浮动旋转），停留 1 秒：
   - 经验值：蓝色菱形 + `+30 EXP`（Sky 蓝色）
   - 金币：金色圆形 + `+15 金币`（Amber 色）
   - 装备（概率掉落 20%）：装备图标 + 稀有度颜色边框 + 装备名称
5. **奖励飞入**（1600ms）：经验值飞入顶部 EXP 条 → EXP 条增长；金币飞入金币计数 → 数值增加；装备飞入背包图标 → 背包闪烁提示
6. **经验值满** → 触发升级动画（详见 §4.3）
7. **过渡回地图**（2500ms）：战斗画面以圆形黑幕收缩 → 回到地图画面，怪物消失

#### 4.2.4 角色阵亡（Game Over）

**触发**：角色 HP 归零

**视觉反馈**：
1. **角色 HP 条清零**（0ms）
2. **角色倒下动画**（200ms）：角色精灵向后倒下 + 变灰
3. **画面变暗**（500ms）：全屏覆盖暗色半透明层（`rgba(0,0,0,0.7)`）
4. **Game Over 文字**（800ms）：屏幕中央 `GAME OVER`（Display 48px，Rose 红色），伴随低沉音效
5. **战绩总结**（1500ms）：Game Over 下方淡入本次冒险数据：
   - `到达层数：第 7 层`
   - `击败怪物：12 只`
   - `获得经验：+480 EXP`
   - `获得装备：2 件`
6. **按钮入场**（2500ms）：
   - `[继续冒险]`（从检查点恢复，Rose 红色主按钮）
   - `[返回大厅]`（灰色次按钮）

### 4.3 角色升级

**触发**：角色经验值达到升级阈值

**视觉流程**（可在战斗中或地图中触发）：
1. **EXP 条满溢**（0ms）：EXP 条填满 + 金色光芒闪烁
2. **升级光柱**（200ms）：角色位置向上射出金色光柱（全屏高度），持续 1.5 秒
3. **等级跳变**（400ms）：角色等级数字从旧等级翻牌至新等级（如 `Lv.12` → `Lv.13`），Amber 金色放大动画
4. **属性提升展示**（800ms）：角色旁弹出属性提升面板（300×200px 毛玻璃面板）：
   - `HP 上限 +20`（绿色）
   - `ATK +3`（Rose 红色）
   - `DEF +2`（Sky 蓝色）
5. **HP 全满恢复**（800ms）：升级后 HP 自动恢复至满值
6. **面板自动关闭**（3000ms）：属性面板淡出

**音效**：华丽的升级音效（上升感的和弦 + 金币散落音）

**升级公式**（服务端）：
- 升级所需经验 = 100 × 当前等级 × 1.2（向上取整）
- 属性成长：每级 HP+20, ATK+3, DEF+2

## 六、Boss 战特殊规则

### 6.1 Boss 出现条件

| 条件 | 说明 |
|------|------|
| **触发层数** | 每 10 层（第 10 层、第 20 层、第 30 层……） |
| **前置条件** | 击败当前层所有普通怪物后，Boss 出现在层传送门前 |
| **Boss 外观** | 32×32px 像素精灵（比普通怪物大一倍），头顶金色王冠图标，周围有火焰/冰霜/雷电光效 |

### 6.2 Boss 属性

| 层数 | Boss 名称示例 | HP | ATK | DEF | 特殊行为 |
|------|-------------|-----|-----|-----|---------|
| 10 | 词汇魔王 | 800 | 40 | 20 | 每 3 题释放「混淆术」（选项顺序随机打乱） |
| 20 | 语法巨龙 | 1500 | 60 | 35 | HP<50% 后出题难度提升至 HSK 8 级 |
| 30 | 成语天尊 | 2500 | 80 | 50 | 每 2 题释放「时间压缩」（倒计时减半至 10s） |
| 40+ | 文学圣贤 | 3500+ | 100+ | 65+ | 综合全部特殊行为 |

### 6.3 Boss 多阶段行为

| 阶段 | HP 范围 | 行为变化 |
|------|---------|---------|
| **阶段一** | 100%–51% | 正常出题（与当前层数对应的难度） |
| **阶段二** | 50%–1% | 题目难度提升一档（如 HSK 7 → HSK 8）+ Boss 攻击力 ×1.5 |
| **阶段切换动画** | HP 降至 50% 时 | Boss 精灵变红闪烁 + 释放冲击波（全屏震动 + Rose 红色光芒）+ 弹出 `Boss 狂暴化！` 提示 |

### 6.4 Boss 掉落

| 掉落 | 概率 | 说明 |
|------|------|------|
| **大量经验值** | 100% | Boss 等级 × 50 EXP |
| **大量金币** | 100% | Boss 等级 × 20 金币 |
| **优秀装备** | 30% | 绿色品质 |
| **稀有装备** | 15% | 蓝色品质 |
| **史诗装备** | 5% | Amber 金色品质 |
| **传说装备** | 1% | 仅 10 层整数 Boss | Rose 红色 + 金色光效 |

### 6.5 多人 Boss 战特殊规则

- 2-4 人合作，Boss HP = 基准 HP × 玩家人数
- 每轮所有玩家同时收到同一道题目
- 各自答题独立：答对者对 Boss 造成伤害，答错者被 Boss 反击
- Boss 每 3 轮释放「全体攻击」，所有玩家各受 Boss ATK × 0.5 伤害
- Boss 被击败后，根据各玩家贡献（正确数 × 40% + 速度 × 30% + 存活 × 30%）分配奖励
- 贡献最高者有额外奖励（掉落概率 +20%）

## 七、PK 模式（竞技场）

### 7.1 属性对等化

为保证竞技场公平性，PK 模式下双方角色属性**强制对等化**：

| 属性 | 对等化值 | 说明 |
|------|---------|------|
| **HP** | 800 | 固定值，不受等级/装备影响 |
| **ATK** | 50 | 固定值 |
| **DEF** | 25 | 固定值 |
| **暴击率** | 10% | 固定值 |

> 设计理由：PK 模式的核心是**答题能力的比拼**，而非角色养成进度的比拼。如果高等级玩家碾压新手，会导致新手流失。装备和等级的价值体现在单人冒险和 Boss 战中。

### 7.2 PK 规则

| 规则 | 说明 |
|------|------|
| **题目来源** | HSK 7-8 综合题，双方收到完全相同的题目 |
| **题目数量** | 固定 10 题 |
| **每题倒计时** | 15 秒 |
| **先手奖励** | 先答对者造成满额伤害（50），后答对者造成 70% 伤害（35） |
| **双方答对** | 均造成 70% 伤害（35） |
| **一方答错** | 答错者被攻击（受到 50 伤害），答对者不受伤 |
| **双方答错** | 无伤害 |
| **超时** | 视为答错 |
| **10 题后** | HP 高者胜；HP 相同则平局 |
| **段位影响** | 胜 +1 星，负 -1 星，平局不变 |

### 7.3 PK 外观差异化

虽然属性对等化，但玩家的角色外观（皮肤）和称号在 PK 中可见：
- 角色精灵显示玩家购买/获得的皮肤
- 角色名称旁显示称号（如 `Lv.45 勇者` 或 `Lv.3 冒险家`）
- 装备的视觉效果（武器外观）在 PK 中可见但不影响数值

## 八、上瘾机制

### 8.1 角色永久养成

- 核心驱动力：角色**永远在变强**，等级、装备、经验值跨局保存
- 每次冒险结束都有收获（经验值 + 金币 + 可能的装备）
- 升级时 HP 全满恢复 + 属性提升 → 正向反馈
- 下次冒险可以挑战更深的层数 → 目标驱动

### 8.2 装备收集

- 5 种稀有度 + 多种装备类型 → 收集癖
- Boss 掉落稀有/史诗/传说装备 → 挑战 Boss 的动机
- 装备属性对比 → 换装的满足感
- 传说装备极低掉落率 → 稀缺性驱动

### 8.3 每日 Boss

- 每日 00:00 UTC 刷新一个「每日 Boss」（特殊 Boss，全服玩家可挑战）
- 全服伤害排行榜（前 3 名获得知语币奖励：+50/+30/+20）
- 每日 Boss 限挑战 3 次
- 独占掉落表（每日 Boss 专属装备，仅此途径获得）

### 8.4 HSK 答题准确率

- 实时统计各模块（听力/阅读/词汇/语法）答题准确率
- 以雷达图形式展示在个人中心
- 准确率达标（如词汇 >90%）→ 解锁「词汇大师」成就
- 与课程 L9 的考核成绩关联展示 → 看到游戏提升了自己的考试能力

### 9.1 地图探索状态

| **HP 回复** | 角色经过回复泉 | 回复泉发光 + HP 条增长 + `+20% HP` 飘字 |
| **触发战斗** | 角色触碰怪物 | 遇怪闪屏过渡 → 进入战斗画面 |
| **层通关** | 所有怪物击败 | 传送门激活（灰色→蓝紫色旋涡），进入后过渡动画 → 下一层加载 |
| **检查点** | 角色经过检查点旗帜 | 旗帜从灰变绿 + `进度已保存` 提示（Toast 1.5 秒） |

### 9.2 战斗状态

| 状态 | 描述 | 设计要求 |
|------|------|---------|
| **题目加载中** | 等待服务端下发题目 | 题目面板 Skeleton + `出题中...` |
| **答题中** | 倒计时进行，等待玩家选择 | 选项可点击，倒计时条运行 |
| **答对·攻击** | 正确选项被选中 | 选项变绿 → 角色攻击动画 → 怪物受伤 → 伤害数字 |
| **答错·受击** | 错误选项被选中 / 超时 | 选项变红 + 正确答案标记 → 怪物攻击 → 角色受伤 |
| **怪物被击败** | 怪物 HP 归零 | 死亡动画 → 掉落物展示 → 奖励飞入 → 回到地图 |
| **Boss 狂暴化** | Boss HP 降至 50% | 全屏震动 + Boss 变红 + 提示弹出 + 后续题目更难 |
| **角色升级** | EXP 满溢 | 升级光柱 + 等级跳变 + 属性提升展示 + HP 全满 |
| **角色阵亡** | 角色 HP 归零 | 倒下动画 → 画面变暗 → Game Over + 战绩总结 |

### 9.3 PK 模式状态

| 状态 | 描述 | 设计要求 |
|------|------|---------|
| **同时答题** | 双方收到同一题目 | 各自独立选择，15 秒倒计时 |
| **先手攻击** | 一方先答对 | 先答对者角色攻击动画，对方受击 |
| **双方答对** | 双方都答对 | 双方各受 70% 伤害攻击 |
| **一方答错** | 一方错另一方对 | 答错者被攻击，答对者不受伤 |
| **双方答错** | 双方都答错 | 无攻击，进入下一题 |
| **10 题结束** | 10 题全部答完 | 对比 HP → 胜/负/平局 → 结算 |
| **对手断线** | 对手网络断开 | 等待 15 秒重连，超时判己方获胜 |

### 9.4 多人 Boss 战状态

| 状态 | 描述 | 设计要求 |
|------|------|---------|
| **全员答题** | 所有玩家同时答题 | 各自独立作答，右侧显示队友答题状态（已答/未答） |
| **Boss 受击** | 有玩家答对 | 答对者角色攻击 Boss，伤害数字在 Boss 头顶显示 |
| **全体攻击** | Boss 每 3 轮释放 | 全屏 Rose 红冲击波 + 所有玩家受伤 + 警告提示 |
| **队友阵亡** | 某玩家 HP 归零 | 该玩家角色灰化 + `已阵亡` 标签，其余玩家继续 |
| **Boss 被击败** | Boss HP 归零 | 全体庆祝动画 + 贡献排名 + 奖励分配 |
| **全灭** | 全部玩家 HP 归零 | 失败画面 + 安慰奖经验值 |

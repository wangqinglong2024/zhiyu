## 一、应用端概述

### 1.1 产品形态

- 首发：PWA Web（移动端优先）
- 后续：Android / iOS 原生或混合
- 游戏场景：强制横屏（landscape），Phaser 3 引擎

| Tab | 图标 | 名称 | 说明 |
|-----|------|------|------|

| 2 | 📚 | 系统课程 | K-12+HSK 双轨学习 |

## 三、系统课程

### 3.1 课程首页

#### 3.1.1 入学测试入口

- 首次进入课程 Tab 时，顶部展示入学测试引导卡片
- "测一测你的中文水平" → 进入测试 → 推荐起始 Level
- 用户可忽略，默认从 L1 开始
- 完成入学测试奖励 100 知语币

#### 3.1.2 Level 地图

12 个 Level 以纵向地图路径展示（类似游戏关卡地图）：

```
  L1  🟢 启蒙之旅（当前进度 85%）
   │
  L2  🟢 基础筑基（已解锁）
   │
  L3  🟢 进阶之门（已解锁）
   │
  L4  🔒 扩展视野（$6 解锁）
   │
  ...
   │
  L12 🔒 博古通今（$6 解锁）
```

每个 Level 节点展示：
- Level 名称（多语言）
- 进度百分比（已开始的 Level）
- 状态图标：进行中 🟢 / 已解锁 ⚪ / 已购买待学 🔵 / 锁定 🔒 / 已完成 ✅
- HSK/CEFR 等级标签
- 点击进入 Level 详情

#### 3.1.3 付费墙

- 点击 L4+ 锁定 Level → 弹出购买弹窗
- 内容：Level 名称 + 包含内容预览 + 价格 $6 + 有效期 3 年
- 知语币足够时显示 "或使用 600 知语币兑换"
- 购买 → Paddle 支付流程 → 成功后 Level 解锁

### 3.2 Level 详情页

#### 3.2.1 单元列表

```
Level 3：进阶之门
HSK 2 / CEFR A2
━━━━━━━━━━━━━━━━━
📖 单元 1：日常交际（4 课时）      ✅ 已完成
   └── 单元测评 ✅ 82分
📖 单元 2：校园生活（5 课时）      🟢 进行中 3/5
   └── 单元测评 🔒
📖 单元 3：购物出行（4 课时）      🔒
   └── 单元测评 🔒
...
━━━━━━━━━━━━━━━━━
🏆 级别综合考核                    🔒
```

#### 3.2.2 进入条件

- 前一单元全部课时学完 → 解锁本单元测评
- 本单元测评通过（≥70 分）→ 解锁下一单元
- 全部单元测评通过 → 解锁级别综合考核

### 3.3 课时学习页

#### 3.3.1 学习内容

```
课时标题
─────────────────
[教学内容区]
  · 文本内容（拼音+中文+解释语言，遵循用户设置）
  · 重点词汇高亮
  · 音频逐句播放
  · 插图/动画辅助
─────────────────
[重点词汇卡片]
  · 横滑卡片：汉字 → 拼音 → 释义 → 例句
  · 点击播放发音
─────────────────
[语法点]（如有）
  · 语法规则说明
  · 例句对照
─────────────────
[底部按钮]
  "开始测验" → 进入课时小测验
```

#### 3.3.2 课时小测验（第一级考核）

- 3-5 道题，即时反馈
- 答对：绿色 ✓ + 简短鼓励
- 答错：红色 ✗ + 显示正确答案 + 解析 + 自动加入 SRS 复习队列
- 无通过门槛，全部答完即可继续
- 题型：选择题 / 拼音标注 / 排序组句

### 3.4 单元阶段测评（第二级考核）

- 10-15 道题
- 通过标准：70 分
- 考试界面：
  - 顶部进度条（第 3/15 题）
  - 题目区域
  - 选项/输入区
  - 下一题按钮
- 结果页：
  - 分数 + 通过/未通过
  - 各题正误详情
  - 薄弱知识点提示
  - 未通过：显示"继续复习"按钮，可重考（无间隔）
- 题型：选择题 / 听力选择 / 拼音标注 / 排序组句 / 填空

### 3.5 级别综合考核（第三级考核）

- 全面考核本 Level 所有知识点，融合 HSK 考点
- 通过标准：总分 85 分，单项不低于 60 分
- 考核模块：听力 / 阅读 / 词汇语法 / 书写（按 Level 逐步增加）
- 不通过：需等待 24 小时后重考
- 通过后：
  - 庆祝动画 + 电子证书展示（标注 HSK 等级 + CEFR 等级）
  - 证书保存至个人中心
  - 解锁下一 Level（L4+ 需购买）

## 四、游戏模式

### 4.1 游戏大厅

#### 4.1.1 布局

```
[顶部栏]
  我的段位：黄金 II ⭐⭐⭐     总星数：47
─────────────────────────────
[游戏列表 - 网格/列表切换]
  🎮 G1 汉字切切切     L1    ⭐ 12胜
  🎮 G2 拼音泡泡龙     L2    ⭐ 8胜
  🎮 G3 词语消消乐     L3    ⭐ 6胜
  ...
  🎮 G12 文学宗师      L12   🔒
─────────────────────────────
[底部]
  🛍️ 皮肤商城          🏆 段位排行榜
```

#### 4.1.2 游戏卡片

每款游戏卡片：
- 游戏封面插图
- 游戏名称（多语言）
- 对应课程 Level 标签
- 胜场/总场数据
- 当前使用皮肤缩略图

#### 4.1.3 游戏解锁规则

- 全部 12 款游戏对登录用户免费开放
- 游戏难度题库按 Level 对应（G1 用 L1 词库，G2 用 L2 词库…）
- 未学对应课程也可玩，但会提示 "去学对应课程效果更佳"

### 4.2 游戏通用流程

#### 4.2.1 进入游戏

```
选择游戏 → 选择模式 → 匹配/开始 → 游戏画面 → 结算
```

#### 4.2.2 游戏模式

| 模式 | 说明 |
|------|------|
| **单人练习** | 单人闯关，不影响段位 |
| **1v1 PK** | 陌生人实时匹配，影响段位 |
| **多人对战** | 3-6 人匹配（部分游戏支持） |

MVP 阶段：仅陌生人匹配，无好友系统。

#### 4.2.3 匹配流程

```
点击"开始匹配" → 匹配动画（搜索对手…）→ 匹配成功（展示对手段位+昵称）→ 3秒倒计时 → 开始
```

- 匹配超时（30秒）：提示 "未找到对手，是否进入 AI 对战？"
- AI 对战不影响段位

#### 4.2.4 游戏结算页

```
🏆 胜利！/ 😢 惜败！
─────────────────
得分：850 分
段位变化：黄金 II ⭐⭐⭐ → 黄金 II ⭐⭐⭐⭐
知识点回顾：
  · "奋不顾身" → 答对 ✅
  · "破釜沉舟" → 答错 ❌（点击查看详解）
─────────────────
[再来一局]  [返回大厅]
```

## 六、登录与注册

### 6.1 触发场景

| 场景 | 触发 |
|------|------|
| 发现中国第 4+ 类目 | 点击被遮罩的类目卡片 |
| 系统课程 Tab | 点击底部导航 |

| 个人中心 Tab | 点击底部导航 |

### 6.2 登录方式

| 方式 | 优先级 |
|------|--------|
| Google 账号 | 主推（东南亚覆盖率高） |
| Apple 账号 | iOS 必须 |
| 邮箱+密码 | 备选 |

### 6.3 登录流程

```
登录弹窗（非全屏，保持上下文）
  ① Google / Apple 一键登录
  ② 或 邮箱登录
     · 邮箱 + 密码
     · "忘记密码" → 邮箱验证码重置
  ③ 或 注册
     · 邮箱 + 密码 + 昵称
     · 勾选隐私协议+服务条款
  ④ 推荐码（可选）
     · "有推荐码？填写后双方获得奖励"
```

## 九、全局设计规范

### 9.1 视觉风格

- 遵循 Cosmic Refraction 设计系统
- 毛玻璃材质 + 光折射效果
- 主色调：Rose / Sky / Amber
- 支持明暗模式切换
- 移动端优先设计（375px 起）

### 9.2 动效规范

| 场景 | 动效 |
|------|------|
| 页面切换 | 横向滑动过渡 |
| 弹窗出现 | 底部上滑 + 背景模糊 |
| 段位晋级 | 专属星光粒子动画 |
| 考核通过 | 庆祝撒花动画 |
| 收藏/取消 | 心形弹跳动画 |
| 游戏结算 | 积分滚动 + 段位变化动画 |

### 9.3 加载与异常

| 状态 | 展示 |
|------|------|
| 加载中 | 骨架屏（Skeleton） |
| 空数据 | 插图 + 引导文案 |
| 网络异常 | "网络不给力，点击重试" |
| 服务器错误 | "服务开小差了，请稍后再试" |

### 9.4 无障碍

- 所有图片含 alt 文本
- 颜色对比度符合 WCAG 2.1 AA
- 支持屏幕阅读器

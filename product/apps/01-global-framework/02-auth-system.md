## 一、登录墙触发场景清单

> 登录墙以非全屏弹窗形式呈现，保持用户当前浏览上下文不丢失。

| # | 触发场景 | 触发条件 | 触发位置 |
|---|---------|---------|---------|
| 1 | 访问受限类目 | 未登录用户点击发现中国第 4-12 类目的卡片（卡片带半透明遮罩 + "登录解锁" 标签） | 发现中国类目首页 |
| 2 | 切换到系统课程 Tab | 未登录用户点击底部 Tab 2（系统课程） | 底部 Tab Bar |
| 3 | 切换到游戏模式 Tab | 未登录用户点击底部 Tab 3（游戏模式） | 底部 Tab Bar |
| 4 | 切换到个人中心 Tab | 未登录用户点击底部 Tab 4（个人中心） | 底部 Tab Bar |
| 5 | 收藏文章 | 未登录用户在文章详情页点击收藏按钮（心形图标） | 文章详情页 |

### 登录墙触发后行为

- 弹出登录弹窗（详见第二节）
- Tab Bar 选中态**不切换**（场景 2/3/4）
- 用户可关闭弹窗回到当前页面继续浏览
- 登录成功后：
  - 场景 1：当前页面刷新，解锁全部 12 类目
  - 场景 2/3/4：自动切换到用户原本想去的 Tab
  - 场景 5：自动完成收藏操作并显示成功 Toast

## 二、登录弹窗详细设计

### 2.1 弹窗布局概述

登录弹窗为**非全屏弹窗**，从屏幕底部上滑弹出，覆盖屏幕下方约 70% 区域，上方仍可看到底部页面内容（背景添加半透明黑色遮罩，透明度 50%）。

弹窗圆角：顶部左右使用 `--radius-3xl`（32px），底部无圆角（贴合屏幕底部）。

弹窗弹出动效：从底部上滑（Slide Up），时长 300ms，Standard 缓动曲线。

### 2.2 弹窗内容结构（从上到下）

```
[拖拽条]（居中横线，宽 40px，高 4px，圆角 --radius-full，中性灰色）
                        间距 --space-6（24px）
[品牌 Logo]（知语 Zhiyu Logo，居中，宽度 120px）
                        间距 --space-2（8px）
[欢迎语]（"欢迎来到知语" / "Welcome to Zhiyu" / "Chào mừng đến với Zhiyu"）
                        间距 --space-8（32px）
[Google 登录按钮]
                        间距 --space-3（12px）
[Apple 登录按钮]
                        间距 --space-6（24px）
```

### 2.3 Google 一键登录按钮

| 属性 | 说明 |
|------|------|
| **按钮标签文字** | "使用 Google 账号登录" / "Sign in with Google" / "Đăng nhập bằng Google" |
| **图标** | Google 官方彩色 G Logo，位于按钮左侧 |

### 2.8 关闭弹窗

- **拖拽关闭**：向下拖拽弹窗顶部拖拽条，下拉超过 120px 后释放即关闭

- **关闭动效**：弹窗向下滑出（Slide Down），时长 250ms，Accelerate 缓动
- 关闭弹窗不影响当前页面内容

## 四、忘记密码流程

### 4.1 流程概述

```
步骤 1：输入邮箱 → 步骤 2：发送验证码 → 步骤 3：输入验证码 → 步骤 4：设置新密码 → 步骤 5：重置成功
```

全部步骤在登录弹窗内完成，使用左右推入（Push/Pop）动效切换步骤。

### 4.2 步骤 1：输入邮箱

| 元素 | 说明 |
|------|------|
| **标题** | "重置密码" / "Reset Password" / "Đặt lại mật khẩu" |
| **副标题** | "请输入注册时使用的邮箱" / "Enter your registered email" / "Nhập email đã đăng ký" |
| **邮箱输入框** | 与登录区邮箱输入框规范一致 |
| **"发送验证码" 按钮** | 全宽药丸形，Rose 色背景，白色文字，高度 48px |
| **可点击条件** | 邮箱格式校验通过时可点击，否则置灰 |
| **点击后流程** | 按钮加载态 → 发送验证码 → 成功进入步骤 2 / 失败提示 |
| **邮箱未注册** | 错误 Toast："该邮箱尚未注册" / "This email is not registered" / "Email này chưa được đăng ký" |
| **返回按钮** | 左上角返回箭头（Lucide `arrow-left`），点击返回登录表单 |

### 4.3 步骤 2：输入验证码

| 元素 | 说明 |
|------|------|
| **标题** | "输入验证码" / "Enter Verification Code" / "Nhập mã xác nhận" |
| **副标题** | "验证码已发送至 user@example.com" / "Code sent to user@example.com" / "Mã đã gửi đến user@example.com"（显示用户邮箱） |
| **验证码输入** | 6 位数字输入框，每位独立方框（宽 48px × 高 56px，`--radius-md` 圆角），自动跳转下一位 |
| **校验规则** | 6 位纯数字 |
| **输入完成后** | 6 位全部输入 → 自动提交验证 → 验证成功进入步骤 3 / 验证失败 |
| **验证失败文案** | "验证码错误，请重新输入" / "Invalid code, please try again" / "Mã không hợp lệ, vui lòng thử lại"，输入框边框变红，内容清空 |
| **重新发送** | 底部文字："没有收到？重新发送" / "Didn't receive? Resend" / "Chưa nhận được? Gửi lại"，发送后进入 60 秒倒计时，倒计时中文字变灰不可点击："60 秒后可重新发送" |
| **返回按钮** | 返回步骤 1 |

## 五、登录态管理

### 5.1 Token 管理

| 场景 | 行为 |
|------|------|
| **登录成功** | 保存登录凭证到本地，后续请求自动携带 |
| **Token 过期** | 自动尝试刷新凭证；刷新失败 → 弹出登录弹窗，提示"登录已过期，请重新登录" / "Session expired, please log in again" / "Phiên đã hết hạn, vui lòng đăng nhập lại" |
| **自动登录** | App 启动时自动检查凭证有效性；有效 → 静默登录；无效 → 保持未登录状态（不弹窗） |
| **异地登录** | 同一账号在新设备登录 → 旧设备收到提示 Toast："你的账号在其他设备登录，如非本人操作请及时修改密码" / "Your account was logged in on another device" / "Tài khoản của bạn đã đăng nhập trên thiết bị khác"，旧设备自动退出登录 |

### 5.2 登录状态持久化

- 登录状态在关闭浏览器/App 后保持
- 凭证有效期：30 天自动续期（每次活跃使用时重置）
- 30 天无活跃 → 凭证过期 → 下次打开需重新登录

## 六、登出流程

### 6.1 登出入口

- 位置：个人中心 → 功能列表 → "退出登录"（列表最底部）
- 文字："退出登录" / "Log out" / "Đăng xuất"
- 图标：Lucide `log-out`，错误色（红色）
- 文字颜色：错误色（红色）

### 6.2 二次确认弹窗

| 元素 | 说明 |
|------|------|
| **弹窗标题** | "确认退出登录？" / "Log out?" / "Đăng xuất?" |
| **弹窗内容** | "退出后需要重新登录才能使用全部功能" / "You'll need to log in again to access all features" / "Bạn cần đăng nhập lại để sử dụng tất cả tính năng" |

### 6.3 登出后行为

- 清除本地登录凭证
- 清除个性化缓存数据（收藏列表、学习进度缓存等）
- 保留用户偏好设置（语言、主题模式）
- 自动跳转到 Tab 1（发现中国），以未登录访客身份浏览
- Tab Bar 角标全部清除

## 七、状态矩阵

### 7.1 登录弹窗状态矩阵

| 状态 | 表现 |
|------|------|
| **空状态** | 不适用（弹窗始终有内容） |
| **加载中** | 登录/注册按钮显示旋转加载指示器 + 文字"登录中…"/"注册中…"；所有输入框和按钮不可操作 |
| **首次加载** | 弹窗弹出后即显示完整表单，无需预加载 |
| **成功** | 弹窗关闭 + 成功 Toast（注册成功时显示）+ 跳转目标页 |
| **错误** | 表单字段保留已填内容；按钮恢复可点击；错误 Toast 从顶部弹出 |
| **部分加载** | 不适用 |
| **离线** | 弹窗可弹出但所有提交按钮不可点击，底部显示提示："当前无网络连接，请联网后重试" / "No internet connection" / "Không có kết nối mạng"；Google/Apple 按钮同样不可点击 |

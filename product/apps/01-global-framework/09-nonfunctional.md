# 非功能需求与验收标准

> 关联文档：[PRD 规范 §一.1](../../../grules/product-design.md) | [UI 设计规范 §五](../../../grules/ui-design.md)
> 优先级：P0

---

## 一、非功能需求

### 1.1 性能指标

| 指标 | 目标值 | 测量条件 |
|------|--------|---------|
| **首屏加载时间（冷启动）** | ≤ 2 秒 | 4G 网络环境、中端移动设备 |
| **首屏加载时间（PWA 已安装）** | ≤ 1 秒 | 有缓存状态下 |
| **Tab 切换响应时间** | ≤ 100ms | 视觉反馈延迟（选中态切换） |
| **Tab 内容渲染时间** | ≤ 300ms | 已缓存数据的页面渲染 |
| **页面过渡动效** | ≤ 500ms | Push/Pop/Fade 过渡完成时间 |
| **Toast 弹出延迟** | ≤ 100ms | 操作触发到 Toast 可见 |
| **登录弹窗弹出** | ≤ 300ms | 触发到弹窗完全可见 |
| **语言切换生效** | ≤ 200ms | 选择语言到 UI 完全切换 |
| **主题切换生效** | ≤ 300ms | 选择模式到所有元素过渡完成 |
| **骨架屏最大显示时间** | ≤ 10 秒 | 超时后转为错误状态 |

### 1.2 网络与超时策略

| 场景 | 超时时间 | 超时处理 |
|------|---------|---------|
| **页面数据请求** | 10 秒 | 显示错误状态 + 重试按钮 |
| **登录/注册请求** | 15 秒 | 按钮恢复可点击 + 错误 Toast |
| **图片加载** | 10 秒 | 显示占位图 |
| **验证码发送** | 15 秒 | 按钮恢复可点击 + 错误 Toast |
| **下拉刷新** | 8 秒 | 刷新动画停止 + 错误 Toast |
| **加载更多** | 8 秒 | 显示"加载失败，点击重试" |

### 1.3 并发与可用性

| 指标 | 目标值 |
|------|--------|
| **预期峰值并发用户** | 1,000（MVP 阶段） |
| **可用性 SLA** | 99.9%（全年宕机 < 8.7 小时） |
| **数据同步延迟** | ≤ 3 秒（用户操作到服务端确认） |

### 1.4 兼容性

| 维度 | 要求 |
|------|------|
| **浏览器** | Chrome ≥ 90、Safari ≥ 14、Firefox ≥ 90、Edge ≥ 90 |
| **移动端** | iOS ≥ 14、Android ≥ 10 |
| **屏幕宽度** | 最小 320px（iPhone SE），最大 2560px |
| **PWA** | 支持 Chrome、Edge、Safari（iOS 16.4+） |

### 1.5 无障碍

| 要求 | 标准 |
|------|------|
| **合规等级** | WCAG 2.1 AA |
| **色彩对比度** | 正文 ≥ 4.5:1，大标题 ≥ 3:1 |
| **触控区域** | 可点击元素最小 44 × 44pt |
| **键盘导航** | 所有核心操作可通过 Tab + Enter 完成 |
| **屏幕阅读器** | 装饰元素 `aria-hidden`，交互元素有 `aria-label` |
| **动效敏感** | 支持 `prefers-reduced-motion`，可关闭动画 |

### 1.6 安全

| 要求 | 说明 |
|------|------|
| **传输加密** | 全站 HTTPS |
| **凭证存储** | 凭证不明文存储，使用安全的本地存储方案 |
| **输入防护** | 所有输入框防 XSS，服务端二次校验 |
| **密码安全** | 密码不可逆加密存储；登录连续失败 5 次锁定 15 分钟 |
| **验证码安全** | 验证码 5 分钟有效；同一邮箱 60 秒内不可重复发送 |

## 二、验收标准汇总

### 2.1 底部 Tab 导航栏

| # | 验收条件 | 预期结果 |
|---|---------|---------|
| 1 | 打开 App，检查底部 Tab Bar | 显示 4 个 Tab，默认选中"发现中国" |
| 2 | 点击各 Tab 切换 | 选中态图标变 Rose 色填充 + 文字变 Rose 色；切换响应 < 100ms |
| 3 | 未登录点击 Tab 2/3/4 | Tab 不切换，弹出登录弹窗 |
| 4 | 登录成功后（从 Tab 2 触发） | 自动切换到 Tab 2（系统课程） |
| 5 | 页面滚动时检查 Tab Bar | Tab Bar 始终可见，不隐藏 |
| 6 | 有新文章时检查 Tab 1 | Tab 1 图标右上角显示红点角标 |
| 7 | 有未读通知时检查 Tab 4 | Tab 4 图标右上角显示数字角标 |
| 8 | 进入对应 Tab 后检查角标 | 角标消除 |
| 9 | 切换 UI 语言后检查 Tab 文字 | Tab 名称跟随语言切换 |
| 10 | 横屏游戏模式下 | Tab Bar 隐藏 |

### 2.2 登录与注册系统

| # | 验收条件 | 预期结果 |
|---|---------|---------|
| 1 | 未登录点击发现中国第 4 类目 | 弹出登录弹窗（非全屏） |
| 2 | 未登录点击收藏按钮 | 弹出登录弹窗 |
| 3 | 检查登录弹窗布局 | 包含 Logo、Google/Apple 按钮、邮箱密码区、忘记密码、注册入口 |
| 4 | 非 iOS 设备检查 | Apple 登录按钮不显示 |
| 5 | 输入无效邮箱格式并离开输入框 | 输入框边框变红，显示错误提示文案 |
| 6 | 输入少于 8 位密码并离开 | 显示密码长度错误提示 |
| 7 | 邮箱密码均有效时 | 登录按钮从置灰变为可点击 |
| 8 | 点击登录按钮 | 按钮变为加载态（旋转指示器 + "登录中…"） |
| 9 | 登录成功 | 弹窗关闭 + 跳转目标页 |
| 10 | 密码错误 | 错误 Toast + 已填内容保留 |
| 11 | 向下拖拽弹窗 > 120px | 弹窗关闭 |
| 12 | 点击遮罩区域 | 弹窗关闭 |
| 13 | 完成注册流程 | 自动登录 + 成功 Toast + 跳转 |
| 14 | 注册时不勾选隐私协议 | 注册按钮置灰不可点击 |
| 15 | 输入无效推荐码 | 显示"推荐码无效"错误提示 |
| 16 | 忘记密码完整流程 | 邮箱 → 验证码 → 新密码 → 成功 → 回到登录 |
| 17 | 验证码输入错误 | 红色边框 + 内容清空 + 错误文案 |
| 18 | 退出登录 | 二次确认弹窗 → 确认 → 清除凭证 → 跳转 Tab 1 |
| 19 | 离线状态打开登录弹窗 | 提交按钮不可点击 + 离线提示文字 |

### 2.3 多语言切换系统

| # | 验收条件 | 预期结果 |
|---|---------|---------|
| 1 | 首次打开 App（浏览器语言为 vi） | UI 默认为越南语 |
| 2 | 首次打开 App（浏览器语言为 en-US） | UI 默认为英语 |
| 3 | 个人中心切换系统语言为英语 | 所有 UI 文字即时切换为英语，无需刷新 |
| 4 | 切换语言后关闭重新打开 | 保持上次选择的语言 |
| 5 | 登录弹窗右上角地球图标 | 点击弹出语言选择面板，未登录也可切换 |
| 6 | 设置学习语言模式为"纯中文" | 所有内容的拼音行消失 |
| 7 | 开启解释语言 | 中文下方出现翻译行 |
| 8 | UI 语言切换为汉语 | 解释语言开关自动关闭且不可操作 |
| 9 | UI 语言从汉语切换回越南语 | 解释语言开关恢复可操作，自动开启 |
| 10 | 文章详情页快捷设置面板 | 可在阅读中切换拼音模式和解释语言 |

### 2.4 深色/浅色模式

| # | 验收条件 | 预期结果 |
|---|---------|---------|
| 1 | 首次打开默认"跟随系统" | App 主题与系统一致 |
| 2 | 切换为深色模式 | 整个界面即时切换为深色，过渡动效 300ms |
| 3 | 切换为浅色模式 | 整个界面即时切换为浅色 |
| 4 | 选择"跟随系统"后手动更改系统设置 | App 自动跟随切换，无需重新打开 |
| 5 | 关闭 App 再打开 | 保持上次主题选择 |
| 6 | 检查 Tab Bar、Toast、弹窗在深色模式下 | 所有组件正确适配深色模式 |

### 2.5 推送通知系统

| # | 验收条件 | 预期结果 |
|---|---------|---------|
| 1 | 首次打开 App | 不请求通知权限 |
| 2 | 完成首节课 | 弹出通知权限引导弹窗 |
| 3 | 点击"暂时不用" | 弹窗关闭，7 天后再次提示 |
| 4 | 允许通知后 | Toast 确认 + 后续正常推送 |
| 5 | 当天已学习 → 学习提醒时间 | 不发送学习提醒 |
| 6 | 当天未学习 → 学习提醒时间 | 发送学习提醒推送 |
| 7 | App 在前台收到通知 | 顶部 Toast 显示，不弹系统通知 |
| 8 | 点击应用内通知 Toast | 跳转到对应页面 |
| 9 | 设置中关闭推送通知 | 不再收到任何推送 |

### 2.6 全局状态规范

| # | 验收条件 | 预期结果 |
|---|---------|---------|
| 1 | 页面数据加载中 | 显示骨架屏（形状匹配最终内容），带 Shimmer 动画 |
| 2 | 首次冷启动 | 品牌 Logo 呼吸动画 + 全屏骨架屏 |
| 3 | 收藏列表为空 | 显示心形图标 + "还没有收藏" + CTA 按钮 |
| 4 | 网络断开 | 顶部 Amber 色离线 Banner |
| 5 | 网络恢复 | Banner 消失 + Toast "网络已恢复" + 自动刷新 |
| 6 | 服务器错误 | "服务开小差了" + 重试按钮 |
| 7 | 点击重试 | 按钮加载态 → 成功恢复内容 / 失败保持错误页 |
| 8 | 加载超过 10 秒 | 自动转为错误状态 |

### 2.7 通用组件

| # | 验收条件 | 预期结果 |
|---|---------|---------|
| 1 | 操作成功 | 顶部成功 Toast，2 秒自动消失 |
| 2 | 操作失败 | 顶部错误 Toast，需手动关闭 |
| 3 | 同时触发 4 个 Toast | 最多显示 3 个，最早的被替换 |
| 4 | 危险操作点击 | 模态弹窗 + 二次确认，确认按钮为错误色 |
| 5 | 模态弹窗打开后按 Tab | 焦点锁定在弹窗内 |
| 6 | 弹窗关闭后 | 焦点回到触发元素 |
| 7 | Bottom Sheet 向下拖拽 > 30% 高度 | 面板关闭 |
| 8 | Bottom Sheet 向下拖拽 < 30% 高度 | 面板回弹 |
| 9 | 列表下拉超过 60px | 触发刷新动画 |
| 10 | 列表滚动到底部 | 自动加载更多，显示加载指示器 |
| 11 | 无更多数据 | 显示"已经到底了" |

### 2.8 PWA

| # | 验收条件 | 预期结果 |
|---|---------|---------|
| 1 | 首次访问 | 不显示安装提示 |
| 2 | 第 2 次访问且浏览 ≥ 3 分钟 | 顶部显示安装 Banner |
| 3 | 关闭 Banner 后 7 天内再访问 | 不再显示 Banner |
| 4 | 已安装 PWA 后访问 | 不显示安装 Banner |
| 5 | 断网后浏览已访问过的文章 | 可正常阅读（含已缓存图片） |
| 6 | 断网后尝试登录 | 操作不可执行 + 离线提示 |
| 7 | 检测到新版本 | 顶部 Banner 提示刷新 |
| 8 | 强制更新场景 | 全屏弹窗，不可关闭，必须更新 |

## 三、商业指标关联

| 功能模块 | 关联商业指标 | 度量方式 |
|---------|------------|---------|
| **登录墙** | 注册转化率 | 触发登录墙的用户中完成注册的比例，目标 ≥ 30% |
| **登录弹窗** | 登录成功率 | 弹窗展示后成功登录的比例，目标 ≥ 60% |
| **推送通知** | 通知权限获取率 | 展示引导后允许通知的比例，目标 ≥ 50% |
| **推送通知** | DAU 回访贡献率 | 通过推送通知回到 App 的 DAU 占比，目标 ≥ 15% |
| **PWA 安装** | 安装转化率 | 展示 Banner 后安装的比例，目标 ≥ 20% |
| **多语言** | 语言切换使用率 | 非默认语言用户占比，用于验证本地化覆盖 |

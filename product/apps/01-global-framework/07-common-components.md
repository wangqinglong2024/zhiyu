## 一、Toast 组件

### 1.1 定位与作用

Toast 用于轻量级操作反馈，不阻断用户当前操作。

### 1.2 位置

| 属性 | 值 |
|------|-----|
| **位置** | 屏幕顶部水平居中，距顶部 `--space-4`（16px）+ 状态栏安全区 |
| **禁止** | 禁止底部弹出（会被 Tab Bar 遮挡） |

### 1.3 动画

| 属性 | 值 |
|------|-----|
| **进入动画** | 从上方滑入（`translateY(-100%) → translateY(0)`），300ms，Decelerate 缓动 |
| **退出动画** | 向上滑出（`translateY(0) → translateY(-100%)`），200ms，Accelerate 缓动 |
| **透明度** | 进入时从 0 → 1 渐显；退出时从 1 → 0 渐隐 |

### 1.4 持续时间

| Toast 类型 | 持续时间 | 关闭方式 |
|-----------|---------|---------|
| **成功** | 2 秒后自动消失 | 自动消失 / 向右滑动关闭 |
| **信息** | 3 秒后自动消失 | 自动消失 / 向右滑动关闭 |
| **警告** | 4 秒后自动消失 | 自动消失 / 向右滑动关闭 |
| **错误** | 不自动消失，需手动关闭 | 点击右侧 × 关闭 / 向右滑动关闭 |

### 1.5 四种类型定义

| 类型 | 左侧图标 | 左侧色条 | 背景色 | 使用场景 |
|------|---------|---------|--------|---------|
| **成功** | Lucide `check-circle`（成功色） | 成功色（绿色） | 毛玻璃面板 | 操作成功（收藏、保存、注册等） |
| **信息** | Lucide `info`（Sky 色） | Sky 色 | 毛玻璃面板 | 系统提示、应用内通知 |
| **警告** | Lucide `alert-triangle`（Amber 色） | Amber 色 | 毛玻璃面板 | 需注意的提示（到期提醒等） |
| **错误** | Lucide `x-circle`（错误色） | 错误色（红色） | 毛玻璃面板 | 操作失败（登录失败、网络错误等） |

### 1.6 Toast 内容结构

```
┌──┬────────────────────────────────────┬──┐
│色│  [图标] Toast 内容文字               │×│
│条│                                     │  │
└──┴────────────────────────────────────┴──┘
```

- 左侧色条：宽度 4px，圆角跟随 Toast 左侧圆角
- 图标：20px，位于文字左侧，间距 `--space-2`（8px）
- 文字：Body S 级别（14px），单行显示，超长截断省略
- 关闭按钮（仅错误类型）：Lucide `x`（16px），位于右侧，间距 `--space-2`

### 1.7 Toast 尺寸

| 属性 | 值 |
|------|-----|
| **宽度** | 自适应内容宽度，最小 200px，最大 屏幕宽度 - 32px（左右各 `--space-4`） |
| **高度** | 44px（单行内容） |
| **内边距** | 上下 `--space-3`（12px），左右 `--space-4`（16px） |
| **圆角** | `--radius-lg`（16px） |
| **阴影** | Level 3（弹窗级阴影） |

### 1.8 堆叠规则

- 多个 Toast 垂直堆叠，新 Toast 在最上方
- 堆叠间距：`--space-2`（8px）
- **最多同时显示 3 个** Toast
- 超过 3 个时，最早的 Toast 被新 Toast 替换（先淡出再入场）
- 堆叠时整体向下位移，使用 200ms Standard 缓动

## 二、模态弹窗

### 2.1 定位与作用

模态弹窗用于需要用户关注并做出决策的场景（确认操作、危险操作二次确认等）。弹窗出现时阻断底层操作。

### 2.2 尺寸

| 属性 | 移动端 | 平板/桌面端 |
|------|--------|-----------|
| **宽度** | 屏幕宽度 - 48px（左右各 `--space-6`） | 固定 400px 居中 |
| **最大高度** | 屏幕高度 × 80% | 屏幕高度 × 70% |
| **内边距** | `--space-6`（24px） | `--space-6`（24px） |
| **圆角** | `--radius-2xl`（24px） | `--radius-2xl`（24px） |
| **位置** | 屏幕垂直居中 | 屏幕垂直居中 |

### 2.3 背景遮罩

| 属性 | 值 |
|------|-----|
| **颜色** | 黑色，透明度 50%（浅色模式）/ 70%（深色模式） |
| **模糊** | 背景叠加 `backdrop-blur(4px)` |
| **出现动效** | 从透明度 0 → 目标透明度，200ms Standard 缓动 |
| **点击遮罩** | 非危险操作弹窗：点击遮罩可关闭弹窗；危险操作弹窗：点击遮罩不可关闭（必须选择按钮） |

### 2.4 弹窗出现/消失动效

| 属性 | 值 |
|------|-----|
| **出现** | `scale(0.95) → scale(1)` + `opacity(0) → opacity(1)`，300ms Spring 缓动 |
| **消失** | `scale(1) → scale(0.95)` + `opacity(1) → opacity(0)`，200ms Accelerate 缓动 |

### 2.5 关闭方式

| 关闭方式 | 适用场景 |
|---------|---------|
| 点击遮罩 | 信息确认类弹窗 |
| 点击右上角 × 按钮 | 所有弹窗（Lucide `x`，24px） |
| 点击取消按钮 | 所有带取消按钮的弹窗 |
| 键盘 Esc 键 | 所有弹窗（桌面端） |

### 2.6 按钮布局

| 场景 | 按钮布局 |
|------|---------|
| **单按钮（确认）** | 全宽居中按钮 |
| **双按钮（取消+确认）** | 两个按钮并排，左侧取消、右侧确认 |
| **危险操作** | 两个按钮并排，左侧"取消"（中性灰色文字按钮），右侧"确认"（错误色背景按钮） |

#### 按钮样式详细

| 按钮类型 | 样式 |
|---------|------|
| **主按钮（确认）** | 药丸形，Rose 色背景，白色文字，高度 44px |
| **次按钮（取消）** | 药丸形，无背景（透明），中性灰色文字，高度 44px |
| **危险按钮** | 药丸形，错误色背景，白色文字，高度 44px |

### 2.7 危险操作二次确认规范

以下操作必须使用模态弹窗进行二次确认：

| 操作 | 弹窗标题 | 弹窗内容 | 确认按钮文字 |
|------|---------|---------|------------|
| **退出登录** | "确认退出登录？" | "退出后需要重新登录才能使用全部功能" | "退出登录"（错误色） |
| **清除缓存** | "确认清除缓存？" | "清除后需要重新加载所有数据" / "Cached data will be cleared and reloaded" / "Dữ liệu cache sẽ bị xóa và tải lại" | "清除"（错误色） |
| **取消收藏** | 不弹窗，直接执行 + 撤销 Toast | — | — |

### 2.8 焦点管理

- 弹窗打开 → 焦点自动移到弹窗内第一个可交互元素
- 焦点锁定在弹窗内（Tab 键循环不会跳出弹窗）
- 弹窗关闭 → 焦点回到触发弹窗的元素

## 三、底部弹出面板（Bottom Sheet）

### 3.1 定位与作用

Bottom Sheet 用于展示辅助信息或操作选项，从屏幕底部上滑弹出。适合不需要全屏关注的内容。

### 3.2 高度规格

| 高度类型 | 占屏幕高度 | 使用场景 |
|---------|-----------|---------|
| **小型** | 30% | 简单操作选项（如语言选择、排序方式） |
| **中型** | 50% | 设置面板、筛选面板 |
| **大型** | 70% | 登录弹窗、详细表单 |

### 3.3 通用结构

```
┌─────────────────────────────────────┐
│              [拖拽条]                │  ← 宽 40px，高 4px，圆角 full，居中
│                                     │
│  [标题]（可选）                       │
│  ─────────────────────              │
│                                     │
│  [内容区域]                          │
│  · 列表选项 / 表单 / 信息           │
│                                     │
│                                     │
└─────────────────────────────────────┘
```

### 3.4 视觉规范

| 属性 | 值 |
|------|-----|
| **顶部圆角** | `--radius-3xl`（32px），底部无圆角 |
| **背景** | 毛玻璃面板（与页面背景一致） |
| **拖拽条** | 宽 40px × 高 4px，圆角 `--radius-full`，中性灰色 30% 透明度，居中，距顶部 `--space-3`（12px） |
| **阴影** | Level 4（模态级阴影） |
| **遮罩** | 与模态弹窗一致（黑色半透明） |

### 3.5 拖拽关闭

| 属性 | 值 |
|------|-----|
| **拖拽方向** | 向下拖拽 |
| **关闭阈值** | 下拉超过面板高度的 30% 后释放 → 关闭 |
| **未达阈值** | 释放后面板回弹到原位，200ms Spring 缓动 |
| **关闭动效** | 面板向下滑出 + 遮罩淡出，250ms Accelerate 缓动 |
| **其他关闭方式** | 点击遮罩关闭 |

### 3.6 内容规范

- 面板内部可滚动（当内容超过面板高度时）
- 内部滚动到顶部后继续下拉 → 触发面板关闭（而非继续滚动）
- 面板最大高度：屏幕高度 × 90%（留出状态栏可见）

## 四、下拉刷新

### 4.1 触发条件

| 属性 | 值 |
|------|-----|
| **触发手势** | 在页面/列表顶部向下拉动 |
| **触发距离** | 下拉超过 60px |
| **触发区域** | 仅在列表/页面已滚动到最顶部时可触发 |

### 4.2 动画表现

| 阶段 | 视觉表现 |
|------|---------|
| **下拉中（< 60px）** | 顶部出现知语 Logo 缩小版（24px），随下拉距离从小到大缩放，透明度从 30% 到 100% |
| **达到触发距离（≥ 60px）** | Logo 完全显示 + 轻微 Haptic 反馈（如设备支持） |
| **释放后加载中** | Logo 开始旋转（360° 循环），1 秒一周 |
| **加载完成** | Logo 停止旋转，向上缩小消失（200ms），页面内容回弹 |
| **加载失败** | Logo 停止旋转并消失，错误 Toast 弹出 |

### 4.3 刷新完成反馈

- 数据有更新 → Toast 提示："已刷新" / "Refreshed" / "Đã làm mới"
- 数据无更新 → Toast 提示："已是最新" / "Up to date" / "Đã cập nhật"
- 加载失败 → 错误 Toast："刷新失败" / "Refresh failed" / "Làm mới thất bại"

### 4.4 防重复

- 刷新过程中再次下拉无效（忽略手势）
- 刷新完成后立即再次下拉 → 最短间隔 2 秒才可再次触发

## 五、上拉加载更多

### 5.1 触发条件

| 属性 | 值 |
|------|-----|
| **触发方式** | 列表滚动到底部时自动触发（无需用户操作） |
| **触发距离** | 距离列表底部 200px 时提前触发预加载（提升流畅感） |

### 5.2 加载指示器

| 属性 | 值 |
|------|-----|
| **位置** | 列表最底部，居中显示 |
| **视觉** | 圆形旋转加载指示器（24px，Rose 色）+ 文字 "加载中…" / "Loading…" / "Đang tải…" |
| **高度** | 指示器区域总高度 56px |

### 5.3 加载结果

| 结果 | 表现 |
|------|------|
| **加载成功** | 指示器消失，新数据以 Stagger 动画依次渐入（每项延迟 50ms） |
| **加载失败** | 指示器替换为可点击文字："加载失败，点击重试" / "Failed to load, tap to retry" / "Tải thất bại, nhấn để thử lại"，文字 Sky 色 |
| **无更多数据** | 显示终止标记："— 已经到底了 —" / "— That's all —" / "— Đã hết —"，Body S 级别（14px），中性灰色 40% 透明度，居中 |

### 5.4 防重复

- 加载过程中不重复触发
- 已显示"无更多数据"后不再触发加载

# 多语言切换系统

> 关联文档：[产品总纲 §四](../../00-product-overview.md) | [执行摘要 §三](../../../plan/01-executive-summary.md)
> 优先级：P0（UI 语言切换、学习语言模式）/ P1（解释语言开关）
> 情绪目标：亲切、无障碍、自主

## 一、UI 语言切换

### 1.1 支持语言列表

| 阶段 | 支持语言 | 语言代码 |
|------|---------|---------|
| MVP v1.0 | 汉语、英语、越南语 | zh / en / vi |
| v1.1 | + 印尼语 | id |
| v2.0+ | + 泰语、马来语 | th / ms |

### 1.2 切换入口位置

| 入口 | 位置 | 说明 |
|------|------|------|
| **主入口** | 个人中心 → 基础设置 → 系统语言 | 所有已登录用户可用 |
| **辅助入口** | 登录弹窗右上角地球图标（Lucide `globe`） | 未登录用户也可切换 UI 语言 |

### 1.3 切换交互

#### 个人中心入口

| 元素 | 说明 |
|------|------|
| **设置项标签** | "系统语言" / "Language" / "Ngôn ngữ" |
| **当前值** | 显示当前语言名称（如 "English"） |
| **点击行为** | 底部弹出面板（Bottom Sheet），列出所有支持的语言选项 |

#### 语言选择面板

```
[拖拽条]
                间距 --space-4（16px）
[标题]（"选择语言" / "Select Language" / "Chọn ngôn ngữ"）
                间距 --space-4（16px）
[语言列表]
  🇨🇳 中文（简体）          ✓（若当前选中）
  ─────────────
  🇺🇸 English              ✓（若当前选中）
  ─────────────
  🇻🇳 Tiếng Việt           ✓（若当前选中）
```

- 每个语言项高度 56px，左侧国旗 emoji + 语言名称（用该语言的原文显示）
- 当前选中项右侧显示 Rose 色对勾（Lucide `check`）
- 点击其他语言项 → 立即切换

#### 登录弹窗入口

- 登录弹窗右上角显示地球图标（Lucide `globe`，24px）
- 点击弹出与上述相同的语言选择面板
- 此入口让未登录用户也能在首次使用时切换到熟悉的语言

### 1.4 切换后页面表现

| 行为 | 说明 |
|------|------|
| **即时生效** | 所有 UI 文字（按钮标签、Tab 名称、标题、提示文案等）立即切换为目标语言 |
| **无需刷新** | 不触发页面刷新或重新加载，纯前端切换 |
| **内容区域** | 发现中国文章标题/课程名称等内容文字跟随 UI 语言显示对应翻译版本 |
| **Toast 提示** | 切换成功后显示 Toast："语言已切换" / "Language changed" / "Đã đổi ngôn ngữ"（使用新语言显示） |
| **持久化** | 选择结果保存到本地，下次打开 App 保持选择 |

### 1.5 默认语言规则

- 首次打开 App → 检测浏览器/系统语言设置
- 若浏览器语言为 zh/zh-CN/zh-TW → 默认汉语
- 若浏览器语言为 vi → 默认越南语
- 其他所有情况 → 默认英语

## 二、学习语言模式

### 2.1 模式定义

| 模式 | 内容展示 | 适用人群 |
|------|---------|---------|
| **拼音+中文** | 拼音显示在中文上方，中文主体内容始终显示 | 初学者、需要拼音辅助的用户 |
| **纯中文** | 不显示拼音，仅显示中文主体内容 | 中高级学习者、母语用户 |

默认值：**拼音+中文**

### 2.2 切换入口

| 入口 | 位置 |
|------|------|
| **主入口** | 个人中心 → 基础设置 → 学习语言模式 |
| **快捷入口** | 文章详情页 / 课时学习页右上角设置图标（Lucide `settings-2`） |

### 2.3 切换交互

#### 设置页入口

| 元素 | 说明 |
|------|------|
| **设置项标签** | "学习语言模式" / "Learning Mode" / "Chế độ học" |
| **切换控件** | 分段选择器（Segmented Control），两个选项并排 |
| **选项 A** | "拼音+中文" / "Pinyin + Chinese" / "Pinyin + Tiếng Trung" |
| **选项 B** | "纯中文" / "Chinese Only" / "Chỉ Tiếng Trung" |
| **选中态** | 选中项背景为 Rose 色 10% 透明度，文字为 Rose 色；未选中项文字为中性灰色 |

#### 文章/课时内快捷切换

- 右上角设置图标 → 点击弹出迷你设置面板（Bottom Sheet，高度约 30% 屏幕）
- 面板内包含：学习语言模式切换 + 解释语言开关 + 字体大小调节
- 切换后内容区域即时渲染更新

### 2.4 切换后内容渲染变化

#### 拼音+中文模式

```
Zuótiān qīngchén, wǒ zuò zài yángtái shàng kàn rìchū.    ← 拼音行（Body S 级别，中性灰色）
昨天清晨，我坐在阳台上看日出。                                ← 中文行（Body 级别，主文字色）
Sáng qua, tôi ngồi trên ban công ngắm bình minh.           ← 解释语言行（Body S 级别，Sky 色）
```

#### 纯中文模式

```
昨天清晨，我坐在阳台上看日出。                                ← 中文行（Body 级别，主文字色）
Sáng qua, tôi ngồi trên ban công ngắm bình minh.           ← 解释语言行（Body S 级别，Sky 色）
```

- 拼音行消失时使用 200ms 淡出动效，内容区高度自适应收缩
- 持久化：选择保存到本地，下次打开保持

## 三、解释语言开关

### 3.1 功能定义

| 属性 | 说明 |
|------|------|
| **功能** | 控制是否在中文内容下方显示解释语言翻译行 |
| **开启** | 中文内容下方显示对应的解释语言翻译（跟随 UI 语言） |
| **关闭** | 不显示解释语言翻译行 |
| **默认值** | 开启 |
| **解释语言** | 自动跟随 UI 语言设置。例如：UI 语言为越南语 → 解释语言为越南语；UI 语言为英语 → 解释语言为英语 |

### 3.2 切换入口

| 入口 | 位置 |
|------|------|
| **主入口** | 个人中心 → 基础设置 → 解释语言 |
| **快捷入口** | 文章/课时内快捷设置面板（同学习语言模式） |

### 3.3 切换交互

| 元素 | 说明 |
|------|------|
| **设置项标签** | "解释语言" / "Explanation Language" / "Ngôn ngữ giải thích" |
| **切换控件** | 开关（Toggle Switch），Rose 色选中态 |
| **开关下方说明** | 开启时显示："当前解释语言：越南语" / "Current: Vietnamese" / "Hiện tại: Tiếng Việt"（跟随 UI 语言） |

### 3.4 特殊规则：中文 UI 时的禁用逻辑

> **核心规则**：当 UI 语言为汉语时，解释语言开关**自动关闭且不可开启**（中文解释中文无意义）。

| 场景 | 行为 |
|------|------|
| UI 语言切换为汉语 | 解释语言开关自动变为关闭状态；开关控件变为禁用态（`opacity: 0.45` + 不可操作）；开关下方显示说明文字："中文界面下无需解释语言" |
| UI 语言从汉语切换为其他语言 | 解释语言开关恢复为启用态；自动恢复为开启状态 |
| 用户手动关闭解释语言（非中文 UI） | 正常关闭，下方说明文字消失 |

### 3.5 切换后内容渲染

- 关闭解释语言 → 翻译行使用 200ms 淡出消失，内容区高度自适应
- 开启解释语言 → 翻译行使用 200ms 淡入出现
- 持久化：选择保存到本地

### 配置组合与最终渲染

| UI 语言 | 学习语言模式 | 解释语言开关 | 最终内容渲染 |

| 汉语 | 拼音+中文 | 强制关闭 | 拼音 + 中文（两行） |
| 汉语 | 纯中文 | 强制关闭 | 仅中文（一行） |

## 五、首次打开 App 的语言初始化逻辑

### 5.1 初始化流程

```
App 首次打开
    │
    ├── 检测浏览器/系统语言
    │     ├── zh/zh-CN/zh-TW → UI 语言设为汉语
    │     ├── vi → UI 语言设为越南语
    │     └── 其他 → UI 语言设为英语
    │
    ├── 学习语言模式 → 默认"拼音+中文"
    │
    ├── 解释语言开关
    │     ├── UI 为汉语 → 强制关闭且禁用
    │     └── UI 为非汉语 → 默认开启
    │
    └── 保存初始配置到本地
```

### 5.2 未登录用户

- 语言配置保存在浏览器本地存储
- 登录后自动同步到服务端用户配置

### 5.3 已登录用户切换设备

- 新设备登录 → 从服务端拉取用户语言配置 → 覆盖本地默认值
- 若服务端无配置记录 → 使用本地检测的默认值

## 六、状态矩阵

### 6.1 语言切换面板

| 状态 | 表现 |
|------|------|
| **空状态** | 不适用（语言列表固定） |
| **加载中** | 不适用（语言列表本地数据，无需加载） |
| **首次加载** | 面板弹出即显示完整语言列表 |
| **成功** | 切换后 Toast 提示 + 面板自动关闭 + UI 即时更新 |
| **错误** | 不适用（本地操作，不依赖网络） |
| **部分加载** | 不适用 |
| **离线** | UI 语言切换正常工作（纯本地操作）；学习语言模式切换正常；解释语言开关切换正常。内容区域显示缓存的已加载内容 |

### 6.2 用户故事验收

- 「作为越南用户，我想要将界面切换为越南语，以便用母语理解所有操作提示」
- 「作为初学者，我想要同时看到拼音和中文，以便学习正确发音」
- 「作为中文母语用户，我不需要解释语言翻译，系统应该自动帮我关闭这个功能」
- 「作为进阶学习者，我想要关闭拼音显示，以便训练自己直接阅读中文的能力」

## 二、界面语言切换

### 2.1 交互流程

```
点击"界面语言"行
    ↓
弹出底部面板 (Bottom Sheet)
┌──────────────────────────────────┐
│  界面语言                         │
│                                  │
│  ○ 中文                     ✓    │
│  ○ English                       │
│  ○ Tiếng Việt                    │
│                                  │
│  [ 取消 ]                        │
└──────────────────────────────────┘
    ↓
选择新语言 → 全局 UI 即时切换 → Bottom Sheet 关闭
```

### 2.2 规则

| 规则 | 说明 |
|------|------|
| 支持语言 | 中文、English（英语）、Tiếng Việt（越南语） |
| 默认语言 | 首次使用时跟随系统语言；系统语言不在支持列表内时默认英语 |
| 切换生效 | 即时生效，无需重新加载应用 |
| 持久化 | 选择保存到用户偏好，下次登录自动恢复 |
| 联动 | 切换界面语言后，解释语言（若已开启）自动跟随更新 |

### 2.3 特殊规则

- 选择「中文」作为界面语言时 → 「解释语言」开关自动关闭且不可开启（中文解释中文无意义）
- 该规则在底部面板选中"中文"的瞬间生效，不需要额外提示

## 四、学习语言模式 + 解释语言

### 4.1 学习语言模式

```
点击"学习语言模式"行
    ↓
弹出底部面板 (Bottom Sheet)
┌──────────────────────────────────┐
│  学习语言模式                     │
│                                  │
│  ○ 拼音 + 中文                ✓  │
│     同时展示拼音和中文             │
│                                  │
│  ○ 纯中文                        │
│     仅展示中文，不显示拼音         │
│                                  │
│  [ 取消 ]                        │
└──────────────────────────────────┘
```

| 规则 | 说明 |
|------|------|
| 默认值 | 拼音+中文 |
| 切换生效 | 即时生效，所有内容页面重新渲染 |
| 持久化 | 保存到用户偏好 |

### 4.2 解释语言

| 元素 | 规格 |
|------|------|
| 显示 | 列表行右侧为开关 Toggle（非箭头） |
| 默认值 | 开启（UI 语言非中文时）；关闭（UI 语言为中文时） |
| 开启时 | 内容页面第三行展示用户 UI 语言的解释/翻译 |
| 关闭时 | 内容页面仅展示拼音+中文（或纯中文） |
| 联动规则 | UI 语言选择中文 → 自动关闭且 Toggle 禁用（灰色，不可操作） |

## 五、推送通知开关

### 5.1 三项独立通知控制

| 通知类型 | 说明 | 默认 |
|---------|------|------|
| 学习提醒 | 每日学习提醒、课程到期提醒、复习提醒 | 开启 |
| 活动通知 | 节日活动、促销优惠、赛季开始/结束 | 开启 |
| 系统消息 | 知语币到账、推荐返利、课程购买确认 | 开启 |

每项为独立的 Toggle 开关，可分别控制。

### 5.2 系统通知权限

- 首次开启任何通知开关时，若用户未授权浏览器推送权限 → 触发系统权限请求弹窗
- 用户拒绝系统权限后 → 三个开关均变为禁用态 → 开关下方显示提示文字"请在浏览器设置中允许通知权限" + \[去设置\] 链接

### 5.3 交互

- 切换 Toggle 立即生效
- 关闭时 Toast"已关闭{通知类型}"
- 开启时无 Toast（避免打扰）

|------|------|
| 用户已绑定推荐人 | 不显示此菜单项 |

### 9.3 注销确认流程

```
输入密码 → 点击"确认注销账号"
    ↓

    ↓

```

### 9.4 Google/Apple 账号注销

使用 Google 或 Apple 登录的用户无密码：
- 密码输入框替换为"请通过 Google/Apple 验证身份"+ \[验证身份\] 按钮
- 点击后调起对应的 OAuth 重新认证流程
- 认证成功后继续注销流程

### 9.5 冷静期状态

注销申请提交后，账号进入 30 天冷静期：

| 表现 | 说明 |
|------|------|
| 用户会收到邮件 | "您的知语账号注销申请已提交，30 天后将永久生效。如需撤销，请在此期间登录账号。" |
| 自动退出登录 | 提交注销后自动退出到访客模式 |
| 冷静期内登录 | 登录时弹出提示："您的账号正在注销中（剩余 {N} 天），登录将自动撤销注销。" + \[继续登录（撤销注销）\] + \[取消\] |
| 撤销注销 | 点击"继续登录"→ 注销申请撤销 → Toast"注销已撤销，欢迎回来！" → 正常使用 |
| 30 天到期 | 系统自动执行账号删除：删除个人数据，匿名化订单记录 |

### 9.6 GDPR 合规数据处理

| 数据类型 | 注销后处理 |
|---------|----------|
| 个人身份信息（姓名、邮箱、头像、昵称） | 永久删除 |
| 学习进度、收藏、证书、游戏数据 | 永久删除 |
| 知语币余额和流水 | 永久删除 |
| 推荐关系 | 解除绑定，不影响已发放的推荐人返利 |
| 支付/订单记录 | 脱敏保留 7 年（合规要求） |
| Paddle 支付信息 | 由 Paddle 按其数据保留政策处理 |
| 统计数据 | 匿名化保留（不可关联到个人） |

## 十一、状态矩阵

| 状态 | 表现 |
|------|------|
| **空状态 (Empty)** | 不适用（设置页始终有内容） |
| **加载中 (Loading)** | 设置项列表使用 Skeleton 行（左侧图标灰块 + 右侧文字灰条） |
| **首次加载 (First Load)** | 同加载中 |
| **成功 (Success)** | 各项切换即时生效，无额外 Toast（除关闭通知外） |
| **错误 (Error)** | 设置保存失败：Toast"保存失败，请重试"；密码验证失败：Toast"密码错误" |
| **部分加载 (Partial)** | 不适用（设置数据体量小，一次加载） |
| **离线 (Offline)** | 顶部离线 Banner；语言/主题/通知开关仍可操作（本地即时生效）；注销/退出登录按钮禁用 |

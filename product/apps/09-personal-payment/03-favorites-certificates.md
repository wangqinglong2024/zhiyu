# 我的收藏与我的证书

> 关联：[00-index.md](00-index.md) · `plan/03-course-system.md`（证书规则） · `product/apps/00-apps-overview.md`（收藏功能）
> 本文档定义「我的收藏」页面（收藏的发现中国文章）和「我的证书」页面（级别通过证书）的完整需求。

## 一、我的收藏页

### 1.1 页面结构

```
┌─────────────────────────────────────┐
│  ← 返回          我的收藏            │
├─────────────────────────────────────┤
│  共 {N} 篇收藏                       │
├─────────────────────────────────────┤
│  ┌─────────────────────────────┐    │
│  │ ┌──────┐                    │    │
│  │ │缩略图 │ 文章标题（最多2行）  │    │
│  │ │72×72 │ [中国美食] 分类标签  │    │
│  │ └──────┘ 收藏于 2026-03-15   │    │
│  └─────────────────────────────┘    │
│  ┌─────────────────────────────┐    │
│  │ ┌──────┐                    │    │
│  │ │缩略图 │ 文章标题（最多2行）  │    │
│  │ │72×72 │ [中国历史] 分类标签  │    │
│  │ └──────┘ 收藏于 2026-03-10   │    │
│  └─────────────────────────────┘    │
│  ...                                │
│                                     │
│  [触底加载更多]                       │

```

### 1.2 收藏列表

- 排序规则：按收藏时间倒序（最新收藏在最上方）
- 分页加载：每次加载 20 条，触底自动加载更多
- 下拉刷新：下拉触发刷新整个列表

### 1.3 收藏卡片设计

每张卡片为横向布局的 `.glass-card`：

| 元素 | 位置 | 规格 |
|------|------|------|
| 缩略图 | 左侧 | 72×72px，圆角 `radius-md`（12px） |
| 文章标题 | 右侧上方 | Body（16px，`font-weight: 600`），最多 2 行，超出截断 `-webkit-line-clamp: 2` |
| 分类标签 | 右侧中间 | 标签样式，Caption（12px），Amber 色背景 10% + Amber 色文字，内容为所属类目名称（如"中国美食"） |
| 收藏日期 | 右侧下方 | Caption（12px，中性灰），格式：`收藏于 YYYY-MM-DD` |

### 1.4 左滑/长按删除收藏操作

**方式一：左滑删除（移动端首选）**

```
← 左滑卡片
┌────────────────────────┬──────┐
│  文章卡片内容     ←←←   │ 取消 │
│                        │ 收藏 │
└────────────────────────┴──────┘
```

- 左滑距离 > 80px 时露出红色「取消收藏」按钮
- 点击「取消收藏」→ 卡片向左滑出并消失（300ms `ease-out` 动画）→ 列表自动上移填补空位
- 取消收藏成功后 Toast 提示："已取消收藏"
- Toast 附带「撤销」文字按钮，点击可恢复收藏（5 秒内有效）

**方式二：长按删除（辅助）**

- 长按卡片 400ms → 弹出底部面板 (Bottom Sheet)：
  - 「取消收藏」— 红色文字
  - 「取消」— 中性灰文字
- 点击「取消收藏」→ 同左滑删除效果

### 1.5 点击跳转文章详情

点击卡片任意区域（非左滑操作区域）→ 跳转对应的「发现中国」文章详情页。
若文章已被下架或删除 → Toast 提示"该文章已下架"，并从收藏列表中移除该条目。

### 1.6 空状态设计

```
┌─────────────────────────────────────┐
│                                     │
│        [收藏相关品牌插画]             │
│                                     │
│     还没有收藏，去发现中国看看吧       │
│                                     │
│      [ 🔵 去发现中国 ]               │
│                                     │
└─────────────────────────────────────┘
```

| 元素 | 规格 |
|------|------|
| 插画 | 居中，200×160px，知语品牌风格插画（书签/爱心/中国元素） |
| 引导文案 | Body（16px），中性灰，居中 |
| CTA 按钮 | 主按钮（Rose 色背景，药丸形），文字"去发现中国" |
| 按钮行为 | 点击后切换到底部导航「发现中国」Tab |

### 1.7 状态矩阵

| 状态 | 表现 |
|------|------|
| **空状态 (Empty)** | 展示空状态插画 + 引导文案 + CTA（见 1.6） |
| **加载中 (Loading)** | 显示 3-4 个卡片形状的 Skeleton 骨架（左侧正方形 + 右侧三行矩形条） |
| **首次加载 (First Load)** | 同加载中 |
| **成功 (Success)** | 取消收藏成功后 Toast"已取消收藏" + 撤销按钮 |
| **错误 (Error)** | 列表加载失败：页面中部错误插画 + "加载失败，请重试" + \[重试\] 按钮 |
| **部分加载 (Partial)** | 已加载的卡片正常展示；触底加载更多时底部显示加载指示器（三点动画） |
| **离线 (Offline)** | 顶部离线 Banner；展示上次缓存的收藏列表；左滑删除操作禁用 |

### 2.2 证书列表

- 排序规则：按获得时间倒序（最新获得在最上方）
- 证书对应关系：每通过一个 Level 的综合考核，获得一张证书

### 2.3 证书卡片设计

每张卡片为横向布局的 `.glass-card`：

| 元素 | 位置 | 规格 |
|------|------|------|
| 证书缩略图 | 左侧 | 96×68px（4:3 横向），圆角 `radius-md`，展示证书缩略预览 |
| Level 名称 | 右侧上方 | Body（16px，`font-weight: 600`），格式：`L{N} {级别名称}` |
| 评测等级 | 右侧中间 | Body S（14px），格式：`评测等级：{等级}`，颜色见下表 |
| 获得日期 | 右侧下方 | Caption（12px，中性灰），格式：`获得于 YYYY-MM` |

#### 评测等级映射

| 综合考核得分 | 评测等级 | 颜色 |
|------------|---------|------|
| 95-100 分 | 卓越 | Amber 色（金） |
| 85-94 分 | 优秀 | Rose 色 |

> 综合考核通过标准为 85 分，因此所有获得证书的用户评测等级至少为「优秀」。

#### HSK/CEFR 标签

每张证书卡片底部显示该 Level 对应的 HSK 和 CEFR 等级标签：

| 标签 | 样式 |
|------|------|
| `HSK {N}` | 小标签，Sky 色背景 10% + Sky 色文字 |
| `CEFR {等级}` | 小标签，Amber 色背景 10% + Amber 色文字 |

#### "分享证书"按钮

点击「分享证书」按钮后的流程：

```
点击 [📤 分享证书]
    ↓
系统生成证书分享图片（含证书完整内容 + 知语品牌水印 + 二维码）
    ↓
生成中：按钮变为 Loading 态（"生成中…"）
    ↓
生成完成 → 调起系统分享面板（Web Share API）
    ↓
分享面板选项：
  · 保存到相册
  · 分享到 WhatsApp
  · 分享到 Facebook
  · 复制图片
  · 其他系统支持的分享渠道
    ↓
分享成功 → Toast"分享成功"
用户取消 → 无提示
```

**分享图片规格**：
- 尺寸：1080×1440px（3:4 竖版，适合社交媒体）
- 内容：完整证书内容 + 底部知语 Logo + "扫码下载知语 Zhiyu" + 品牌二维码
- 格式：PNG

#### "下载证书"按钮

点击「下载证书」按钮：
- 生成高清证书 PNG 图片（1080×1440px）
- 触发浏览器下载 → 保存到本地
- Toast 提示"证书已保存"

**Web Share API 不可用时**（桌面浏览器）：
- "分享证书"按钮改为"复制证书图片"，点击复制图片到剪贴板
- Toast 提示"证书图片已复制到剪贴板"

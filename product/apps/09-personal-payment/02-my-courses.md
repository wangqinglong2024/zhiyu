### 1.2 总进度条

页面顶部显示总学习进度条：

| 元素 | 规格 |
|------|------|
| 进度条 | 高度 8px，圆角 `radius-full`，背景中性灰 20%，填充色 Rose 渐变 |
| 进度文字 | Body（16px，`font-weight: 600`），格式：`{已完成Level数} / 12 级完成` |
| 计算规则 | 仅统计已获得证书的 Level 为"完成"；进行中的 Level 不计入已完成数 |

### 1.3 课程卡片设计

每个 Level 使用 `.glass-card` 毛玻璃卡片，卡片内部结构：

| 元素 | 位置 | 规格 |
|------|------|------|
| Level 编号 + 级别名称 | 左上 | Body（16px，`font-weight: 600`），格式：`L{N} {级别名称}` |
| 价格/状态标签 | 右上 | Caption（12px），标签样式 |
| 学习进度条 | 中部 | 高度 4px，圆角 `radius-full`；未开始时灰色空条 |
| 进度百分比 | 进度条右侧 | Caption（12px），格式：`{N}%` |
| 状态图标+文字 | 进度条右侧 | 见下表 |
| 有效期信息 | 底部 | Caption（12px），见下方有效期规则 |

#### 状态标签

| 状态 | 图标 | 文字 | 颜色 |
|------|------|------|------|
| 已完成 | ✅ | 已完成 | 语义成功色 `#22c55e` |
| 进行中 | 🟢 | 进行中 | Rose 色 |
| 未开始（已解锁/已购买） | ⚪ | 未开始 | 中性灰 |
| 锁定（未购买） | 🔒 | — | 中性灰 |
| 已过期 | ⏰ | 已过期 | 语义错误色 `#ef4444` |

#### 价格/状态右上角标签

| 课程状态 | 标签文字 | 标签样式 |
|---------|---------|---------|
| L1-L3 | `免费` | 语义成功色背景 10% + 成功色文字 |
| L4-L12 已购买 | `已购买` | Sky 色背景 10% + Sky 色文字 |
| L4-L12 未购买 | `$6` | Amber 色背景 10% + Amber 色文字 |
| L4-L12 已过期 | `已过期` | 语义错误色背景 10% + 错误色文字 |

### 1.4 有效期展示规则

| 课程类型 | 有效期显示 |
|---------|----------|
| L1-L3（免费） | `永久有效`，中性灰文字 |
| L4-L12（有效期内，剩余 > 30 天） | `有效期至 YYYY-MM-DD`，中性灰文字 + 下一行 `剩余 {N} 天` |
| L4-L12（有效期内，剩余 ≤ 30 天） | `有效期至 YYYY-MM-DD`，语义错误色红色文字 + ⚠️ 图标 + 下一行 `剩余 {N} 天，即将到期！` 红色文字 |
| L4-L12（有效期内，剩余 ≤ 7 天） | 同上，额外在卡片顶部右侧显示红色闪烁圆点角标 |
| L4-L12（已过期） | 整张卡片降低透明度至 60%；`已过期` 红色标签；显示 \[续费\] 按钮 |

### 1.5 卡片点击行为

| 课程状态 | 点击行为 |
|---------|---------|
| 已完成 | 跳转该 Level 详情页（可复习） |
| 进行中 | 跳转该 Level 详情页（继续学习） |
| 未开始（已解锁/已购买） | 跳转该 Level 详情页（开始学习） |
| 锁定（未购买） | 跳转课程购买确认页（见 `05-purchase-payment.md`） |
| 已过期 | 弹出续费引导弹窗（见第四节） |

## 四、续费/重新购买流程

### 4.1 续费入口

| 入口 | 场景 |
|------|------|
| 我的课程页已过期卡片上的 \[续费\] 按钮 | 有效期已到期 |
| 我的课程页即将到期卡片上的 \[续费\] 按钮 | 有效期 ≤ 30 天时显示 |
| 到期提醒推送中的跳转链接 | 收到 30/7/1 天到期推送后点击 |

### 4.2 续费流程

```
点击 [续费] 按钮
    ↓
弹出续费确认底部面板 (Bottom Sheet)
┌──────────────────────────────────┐
│  续费确认                         │
│                                  │
│  课程：L4 扩展视野                │
│  原价：$6                        │
│  续费优惠价：$4.8（8 折）          │
│  有效期：续费之日起 3 年           │
│                                  │
│  ℹ️ 您的学习进度已保留，续费后     │
│     可继续之前的学习               │
│                                  │
│  [ 🔵 确认续费 $4.8 ]             │
│  [ 取消 ]                        │
└──────────────────────────────────┘
    ↓
点击"确认续费" → 进入 Paddle 支付流程（见 05-purchase-payment.md）
    ↓
支付成功 → 课程状态恢复为"已购买"，有效期重新计算 3 年 → 返回我的课程页
支付失败 → 提示重试（见 05-purchase-payment.md 支付失败页）
```

### 4.3 续费定价规则

| 场景 | 价格 | 说明 |
|------|------|------|
| 有效期内续费（≤ 30 天时） | $4.8（原价 $6 × 80%） | 20% 续购折扣 |
| 有效期到期后重新购买 | $4.8（原价 $6 × 80%） | 同样享受 20% 续购折扣 |
| 从未购买过的 Level | $6 | 原价，无折扣 |

**续费有效期计算**：
- 若在有效期内续费：新有效期 = 当前到期日 + 3 年
- 若已过期后续费：新有效期 = 续费当日 + 3 年

### 4.4 知语币抵扣续费

续费时同样支持知语币抵扣（见 `05-purchase-payment.md`）：
- 续费价 $4.8 = 480 知语币可全额抵扣
- 知语币余额不足时，差额通过 Paddle 支付

## 五、状态矩阵

| 状态 | 表现 |
|------|------|
| **空状态 (Empty)** | 用户未学习过任何课程（L1-L3 也未开始）：在课程列表上方显示引导卡片，插画 + "开始你的中文学习之旅吧！" + \[去课程首页\] 按钮跳转课程 Tab |
| **加载中 (Loading)** | 顶部总进度条 Skeleton；课程卡片区域显示 3-4 个卡片形状的 Skeleton 骨架 |
| **首次加载 (First Load)** | 同加载中 |
| **成功 (Success)** | 所有课程卡片正常渲染，已购/未购/过期状态区分清晰 |
| **错误 (Error)** | 页面中部显示错误插画 + "课程信息加载失败" + \[重试\] 按钮 |
| **部分加载 (Partial)** | 课程列表优先渲染卡片框架，进度数据异步填充；进度未加载时进度条显示 Skeleton |
| **离线 (Offline)** | 顶部离线 Banner；展示上次缓存的课程数据；"去购买"和"续费"按钮置灰禁用 |

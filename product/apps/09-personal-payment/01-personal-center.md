# 个人中心 Tab 主页与资料编辑

> 关联：[00-index.md](00-index.md) · 全局框架 `01-global-framework/`
> 本文档定义个人中心 Tab 主页的布局、功能入口、数据展示，以及个人资料编辑页的完整交互。

## 一、个人中心 Tab 主页

### 1.1 页面布局

个人中心 Tab 主页采用纵向滚动布局，自上而下分为五个区域：

```
┌─────────────────────────────────────┐
│  ① 用户信息区                        │
│  ┌──────┐                            │
│  │ 头像  │ 昵称                       │
│  │ 64×64 │ 注册时间：2025-03-15       │
│  └──────┘ [编辑资料 >]               │
├─────────────────────────────────────┤
│  ② 段位展示栏                        │
│  [段位图标] 黄金 II  ⭐⭐⭐            │
│  本赛季排名：第 128 名                │
├─────────────────────────────────────┤
│  ③ 学习数据统计卡片区（2×2 网格）     │
│  ┌───────────┐  ┌───────────┐       │
│  │ 已学课程    │  │ 学习天数    │       │
│  │   3 / 12   │  │    47 天   │       │
│  └───────────┘  └───────────┘       │
│  ┌───────────┐  ┌───────────┐       │
│  │ 游戏战绩    │  │ 知语币余额  │       │
│  │ 186胜/320场 │  │   320 币 > │       │
│  └───────────┘  └───────────┘       │
├─────────────────────────────────────┤
│  ④ 功能菜单列表                      │
│  📖 我的课程                    >    │
│  ⭐ 我的收藏                    >    │
│  📜 我的证书                    >    │
│  💰 知语币                      >    │
│  🎁 推荐好友                    >    │
│  ⚙️ 设置                       >    │
├─────────────────────────────────────┤
│  ⑤ 底部安全区（≥80px）               │
└─────────────────────────────────────┘
```

#### ① 用户信息区

| 元素 | 规格 | 说明 |
|------|------|------|
| 头像 | 64×64px，圆形，`radius-full` | 右下角叠加白色编辑小图标（12×12 铅笔 icon） |
| 昵称 | H3（20px，`font-weight: 600`） | 最多显示 12 个字符，超出截断加 `…` |
| 注册时间 | Caption（12px，中性灰） | 格式：`注册时间：YYYY-MM-DD` |
| 编辑资料入口 | Body S（14px）+ 右箭头图标 | 点击头像或"编辑资料"文字均可跳转资料编辑页 |

#### ② 段位展示栏

| 元素 | 规格 | 说明 |
|------|------|------|
| 段位图标 | 32×32px | 与段位对应的专属图标（青铜/白银/黄金/铂金/钻石/星耀/王者） |
| 段位名称 | Body（16px，`font-weight: 600`） | 格式：`黄金 II`，使用 Amber 色 |
| 星数 | 16px 星形图标 | 实心星=已获得，空心星=未获得 |
| 赛季排名 | Body S（14px，中性灰） | 格式：`本赛季排名：第 XXX 名` |

整个段位展示栏使用 `.glass-card` 毛玻璃卡片样式，点击可跳转游戏模式 Tab 的段位排行榜页。

#### ③ 学习数据统计卡片区

四张统计卡片以 2×2 网格排列，每张卡片使用 `.glass-card` 样式：

| 卡片 | 数据 | 数字样式 | 点击行为 |
|------|------|---------|---------|
| 已学课程 | `{已完成Level数} / 12` | H2（24px，`font-weight: 700`），Rose 色 | 跳转「我的课程」页 |
| 学习天数 | `{天数} 天` | H2（24px，`font-weight: 700`），Sky 色 | 无跳转 |
| 游戏战绩 | `{胜场}胜 / {总场}场` | H2（24px，`font-weight: 700`），Amber 色 | 跳转游戏模式 Tab |
| 知语币余额 | `{余额} 币` + 右箭头 | H2（24px，`font-weight: 700`），Amber 色 | 跳转「知语币」页面 |

**特殊规则**：
- 知语币余额为负数时，数字使用语义错误色 `#ef4444`（红色）
- 知语币余额为 0 时，数字使用中性灰色
- 所有数字使用 `font-variant-numeric: tabular-nums` 确保对齐

#### ④ 功能菜单列表

每一项为标准列表行：

| 元素 | 规格 |
|------|------|
| 左侧图标 | 20×20px，Lucide Icons，继承文字颜色 |
| 功能名称 | Body（16px） |
| 右侧箭头 | `ChevronRight` 图标，中性灰 |
| 行高 | 56px |
| 分隔线 | 底部 1px 分隔线，左侧对齐图标右边缘 |

菜单项列表：

| 图标 | 名称 | 点击跳转 | 角标 |
|------|------|---------|------|
| `BookOpen` | 我的课程 | 我的课程页 | 有课程到期 < 30 天时显示红点 |
| `Star` | 我的收藏 | 我的收藏页 | 无 |
| `Award` | 我的证书 | 我的证书页 | 无 |
| `Coins` | 知语币 | 知语币页面 | 无 |
| `Gift` | 推荐好友 | 推荐返利页面 | 无 |
| `Settings` | 设置 | 设置页 | 无 |

### 1.2 访客模式（未登录）的个人中心表现

未登录用户点击底部导航栏「个人中心」Tab 时，触发登录墙弹窗（详见 `01-global-framework/02-auth-system.md`）。

**若用户关闭登录墙弹窗**，展示访客引导页：

```
┌─────────────────────────────────────┐
│                                     │
│        [默认头像 96×96]              │
│                                     │
│     登录后解锁完整个人中心            │
│                                     │
│  · 查看学习进度和段位排名             │
│  · 管理已购课程和收藏                │
│  · 赚取和使用知语币                  │
│  · 分享证书和推荐好友                │
│                                     │
│    [ 🔵 立即登录 / 注册 ]            │
│                                     │
└─────────────────────────────────────┘
```

| 元素 | 说明 |
|------|------|
| 默认头像 | 96×96px，使用知语品牌灰色占位头像 |
| 引导标题 | H2（24px，`font-weight: 700`），居中 |
| 权益列表 | Body（16px），每行前加 `·` 符号，中性灰 |
| 登录按钮 | 主按钮（Rose 色背景，药丸形），宽度占容器 80%，文字 "立即登录 / 注册" |

点击登录按钮 → 弹出登录弹窗 → 登录成功后自动刷新为完整个人中心主页。

### 1.3 状态矩阵

| 状态 | 表现 |
|------|------|
| **空状态 (Empty)** | 未登录时显示访客引导页（见 1.2） |
| **加载中 (Loading)** | 用户信息区 + 统计卡片区 + 菜单列表各自显示 Skeleton 骨架屏：头像圆形灰块、昵称矩形灰条、统计数字矩形灰块、菜单列表行灰条 |
| **首次加载 (First Load)** | 同加载中状态，首次加载完成后无特殊差异 |
| **成功 (Success)** | 所有数据正常渲染，无额外反馈 |
| **错误 (Error)** | 页面顶部展示错误 Banner："加载失败，请检查网络后重试" + \[重试\] 按钮；已缓存的菜单列表仍可展示 |
| **部分加载 (Partial)** | 用户基础信息（头像/昵称/菜单）先渲染；统计数据和段位信息异步加载，未完成时各显示 Skeleton |
| **离线 (Offline)** | 顶部展示离线 Banner（全局组件）；展示上次缓存的用户信息和统计数据；统计数据旁标注"离线数据，可能不是最新" |

### 2.2 可编辑字段

| 字段 | 输入类型 | 必填 | 默认值 | 说明 |
|------|---------|------|--------|------|
| 头像 | 图片上传 | 否 | 系统默认头像 | 点击触发上传流程（见 2.3） |
| 昵称 | 文本输入 | 是 | 注册时随机生成（如"知语学员8392"） | 2-12 个字符，详见 2.4 |
| 性别 | 单选 | 否 | 保密 | 男 / 女 / 保密 |
| 生日 | 日期选择器 | 否 | 空 | 底部弹出滚轮选择器；年份范围 1940-当年 |
| 所在国家/地区 | 下拉选择 | 否 | 根据 IP 自动推断 | 国家/地区列表按首字母排序，优先展示越南、中国、美国 |
| 学习目标 | 多选标签 | 否 | 空 | 最多选择 3 个 |

### 2.3 头像上传流程

```
点击头像区域
    ↓
弹出底部面板 (Bottom Sheet)
  ┌──────────────────────┐
  │  📷 拍照              │
  │  🖼️ 从相册选择        │
  │  ─────────────────── │
  │  取消                 │
  └──────────────────────┘
    ↓
选择图片来源（拍照/相册）
    ↓
调起系统相机/相册
    ↓
选择图片后进入裁剪页面
  ┌──────────────────────┐
  │  取消        完成      │
  │                       │
  │   ┌────────────┐     │
  │   │  裁剪框      │     │
  │   │  1:1 正方形  │     │
  │   │  可缩放平移  │     │
  │   └────────────┘     │
  │                       │
  └──────────────────────┘
    ↓
点击"完成"→ 上传裁剪后图片
    ↓
上传中：头像位置显示加载指示器（圆形进度环叠加在头像上，半透明遮罩）
    ↓
上传成功：头像即时替换为新图，Toast 提示"头像更新成功"
上传失败：Toast 提示"头像上传失败，请重试"，头像恢复为原图
```

**头像技术约束**：
- 支持格式：JPG、PNG、WebP
- 最大文件大小：5MB
- 裁剪后输出尺寸：512×512px
- 超出大小限制时：Toast 提示"图片大小不能超过 5MB，请重新选择"

### 2.4 昵称规则

| 规则 | 说明 |
|------|------|
| 字符长度 | 最少 2 个字符，最多 12 个字符（中英文各算 1 个字符） |
| 允许字符 | 中文、英文字母、数字、下划线 `_`、短横线 `-` |
| 禁止字符 | 空格、特殊符号（@#$%^&* 等）、emoji |
| 敏感词过滤 | 提交时进行服务端敏感词校验，命中则返回错误提示"昵称包含不当内容，请修改" |
| 唯一性 | 不要求唯一（不同用户可使用相同昵称） |
| 实时校验 | 输入时实时检查字符长度和格式，不符合时输入框下方显示红色提示文字 |

**昵称输入框交互**：
- 输入框右侧显示字符计数器 `{当前字数}/12`
- 未修改时计数器为中性灰色
- 接近上限（≥10 字符）时计数器变为 Amber 色
- 超出上限时计数器变为红色，禁止继续输入

### 2.5 保存/取消操作

| 操作 | 触发 | 行为 |
|------|------|------|
| 保存 | 点击右上角"保存"按钮 | ① 按钮变为 Loading 态（旋转图标+disabled）→ ② 提交所有已修改字段 → ③ 成功：Toast"保存成功" + 返回个人中心主页 → ④ 失败：Toast"保存失败，请重试"，停留在编辑页 |
| 取消/返回 | 点击左上角"←"返回按钮 | 若有未保存的修改 → 弹出确认弹窗："你有未保存的修改，确定要离开吗？" + \[继续编辑\] + \[放弃修改\]；若无修改 → 直接返回 |
| 保存按钮状态 | — | 未做任何修改时，"保存"按钮为禁用态（灰色，不可点击）；任何字段发生变化后，按钮激活为 Rose 色可点击态 |

### 2.6 状态矩阵

| 状态 | 表现 |
|------|------|
| **空状态 (Empty)** | 不适用（必有用户数据） |
| **加载中 (Loading)** | 头像区域显示圆形 Skeleton；各表单字段显示矩形 Skeleton 条 |
| **首次加载 (First Load)** | 同加载中 |
| **成功 (Success)** | 保存成功后 Toast 提示"保存成功"，自动返回个人中心主页 |
| **错误 (Error)** | 保存失败：Toast 提示"保存失败，请重试"；头像上传失败：Toast 提示"头像上传失败，请重试"；昵称敏感词：输入框下方红色文字"昵称包含不当内容，请修改" |
| **部分加载 (Partial)** | 不适用 |
| **离线 (Offline)** | 顶部离线 Banner；保存按钮变为禁用态；Toast 提示"网络已断开，请在联网后保存" |

## 三、教学内容区

### 3.1 多语言文本渲染规则

教学内容的文本渲染遵循 3 个用户配置项的组合：

| 配置项 | 选项 | 设置位置 |
|--------|------|---------|
| 学习语言模式 | 拼音+中文 / 纯中文 | 个人中心 → 基础设置 |
| 解释语言开关 | 开启 / 关闭 | 个人中心 → 基础设置 |
| 解释语言 | 跟随 UI 语言（越南语/英语/汉语） | 自动跟随系统语言设置 |

#### 6 种渲染组合

| # | 学习语言模式 | 解释语言开关 | 渲染结果 |
|---|------------|------------|---------|
| 1 | 拼音+中文 | 开启（越南语） | 三行：拼音 → 中文 → 越南语 |
| 2 | 拼音+中文 | 开启（英语） | 三行：拼音 → 中文 → 英语 |
| 3 | 拼音+中文 | 关闭 | 两行：拼音 → 中文 |
| 4 | 纯中文 | 开启（越南语） | 两行：中文 → 越南语 |
| 5 | 纯中文 | 开启（英语） | 两行：中文 → 英语 |
| 6 | 纯中文 | 关闭 | 一行：纯中文 |

#### 渲染示例（组合 1：拼音+中文+越南语）

```
Zuótiān qīngchén, wǒ zuò zài yángtái shàng kàn rìchū.
昨天清晨，我坐在阳台上看日出。
Sáng qua, tôi ngồi trên ban công ngắm bình minh.
```

#### 文本排版规则

| 层级 | 字号 | 颜色 | 行高 | 字体 |
|------|------|------|------|------|
| 拼音行 | Body S（14px） | 灰色次要文字色 | 1.5 | Inter / 系统字体 |
| 中文行 | Body L（18px） | 主文字色（黑/白跟随主题） | 1.7 | "PingFang SC", "Noto Sans CJK SC", sans-serif |
| 解释语言行 | Body S（14px） | Sky 色（`#0284c7` Dark / `#7dd3fc` Light） | 1.5 | Inter / 系统字体 |

- 三行之间行间距 4px（`--space-1`）
- 段落之间间距 24px（`--space-6`）
- 每段三行文本视觉上作为一个**文本块**，用微弱的背景色区分（`rgba(255,255,255,0.03)` Dark / `rgba(0,0,0,0.02)` Light）

#### 特殊规则

- 用户系统语言为中文时 → 解释语言开关被强制关闭且不可开启（中文解释中文无意义）→ 仅渲染组合 3 或组合 6
- 解释语言跟随 UI 语言设置，用户切换 UI 语言后解释语言自动切换

### 3.2 重点词汇在正文中高亮

#### 高亮规则

- 正文中的**重点词汇**（由后台课程内容标注）使用特殊样式高亮
- 高亮样式：Amber 色下划线（`border-bottom: 2px solid var(--amber)`）+ 文字颜色不变
- 高亮区域：仅中文行中的词汇高亮（拼音行和解释语言行不高亮）

#### 点击弹出词义浮层

用户点击高亮词汇后，弹出浮层（Popover）：

```
        ┌────────────────────────┐
        │  阳台  yángtái          │
        │  🔊                    │
        │                        │
        │  balcony               │
        │  ban công               │
        │                        │
        │  例句：                  │
        │  我家的阳台能看到大海。   │
        └────────────────────────┘
```

| 元素 | 规格 | 说明 |
|------|------|------|
| 浮层形态 | Popover，毛玻璃材质，指向高亮词的箭头 | 圆角 16px（`--radius-lg`） |
| 汉字 | H3 字号（20px），加粗 | — |
| 拼音 | Body S 字号（14px），灰色 | 紧跟汉字右侧 |
| 发音按钮 | 🔊 图标（Lucide `Volume2`），24px | 点击播放该词发音 |
| 释义 | Body S 字号（14px） | 按用户解释语言设置展示；若关闭解释语言则不展示释义（仅展示汉字+拼音+发音） |
| 例句 | Caption 字号（12px），灰色 | 展示一句包含该词的例句 |
| 关闭方式 | 点击浮层外任意区域 | 浮层淡出关闭（200ms） |

### 3.3 音频逐句播放

#### 音频播放按钮

- 每段教学文本（文本块）的**中文行左侧**有一个小喇叭图标按钮
- 图标：Lucide `Volume2`，16px，灰色
- 图标位置：中文行行首左侧，距文字 8px

#### 播放交互

| 操作 | 行为 |
|------|------|
| 点击小喇叭 | 播放**该句**的中文发音音频 |
| 播放中视觉 | 小喇叭图标变为 Rose 色 + 声波动画（3 条竖线交替伸缩）；对应文本块背景变为浅 Rose 色高亮 |
| 播放完毕 | 小喇叭恢复灰色；背景色恢复 |
| 再次点击（播放中） | 停止当前播放 |
| 连续播放 | 不支持自动连播。用户需逐句手动点击。（避免初学者跟不上） |
| 音频加载失败 | 小喇叭图标变为灰色带斜线（Lucide `VolumeX`），点击后 Toast "音频加载失败，请检查网络"（多语言） |

#### 音频数据

- 每段教学文本对应一条音频 URL（存储在 Supabase `audio_url` 字段）
- 音频格式：MP3 / OGG（前端按浏览器兼容性自动选择）
- 音频播放使用 Web Audio API，支持后台播放

### 3.4 插图/动画辅助展示

- 部分课时内容包含插图或简单动画（由后台配置）
- 插图位置：在相关文本块之间，居中展示
- 插图尺寸：宽度 100%（页面内边距内），高度自适应，最大高度 300px
- 插图圆角：16px（`--radius-lg`）
- 加载占位：图片 Skeleton + 底部"图片加载中…"（多语言）
- 加载失败：灰色占位框 + "图片加载失败"文字

## 四、重点词汇卡片区

- "重点词汇" / "Key Vocabulary" / "Từ vựng trọng điểm"

## 五、语法点区（如本课有语法点）

### 5.1 区域标题

- "语法点" / "Grammar Points" / "Điểm ngữ pháp"
- H3 字号（20px），加粗，多语言
- 仅当本课时包含语法点时展示此区域（由后台配置）

### 5.2 语法规则说明

```
┌────────────────────────────────────────────────┐
│  📐 "把" 字句（bǎ zì jù）                       │
│                                                  │
│  用法：主语 + 把 + 宾语 + 动词 + 其他成分         │
│                                                  │
│  说明：                                          │
│  "把"字句用于强调对某个事物做了什么动作，            │
│  把宾语提到动词前面。                              │
│                                                  │
│  ─────────────────────────────────────────────  │
│                                                  │
│  ✅ 正确例句：                                    │
│  我把书放在桌子上了。                              │
│  Wǒ bǎ shū fàng zài zhuōzi shàng le.            │
│  I put the book on the table.                    │
│                                                  │
│  ❌ 常见错误：                                    │
│  × 我把去学校。                                   │
│  ✓ 我去学校。                                    │
│                                                  │
│  ─────────────────────────────────────────────  │
│                                                  │
│  句型结构图：                                     │
│  ┌──────┬───┬──────┬──────┬──────┐             │
│  │ 主语  │ 把 │ 宾语  │ 动词  │ 其他  │             │
│  │  我   │ 把 │  书   │  放   │在桌上 │             │
│  └──────┴───┴──────┴──────┴──────┘             │
└────────────────────────────────────────────────┘
```

| 元素 | 规格 | 说明 |
|------|------|------|
| 语法点标题 | H3 字号（20px），加粗 | 语法点名称 + 拼音（根据设置）。带 📐 图标前缀 |
| 用法公式 | Body 字号（16px），`code` 样式背景 | 语法结构公式，用灰色背景高亮，等宽字体 |
| 说明文本 | Body 字号（16px） | 多语言渲染（遵循 §三 渲染规则） |
| 正确例句 | Body 字号，绿色 ✅ 前缀 | 展示 1-3 个正确用法例句，多语言渲染 |
| 常见错误 | Body 字号，红色 ❌ 前缀 | 展示 1-2 个常见错误 + 正确写法对照 |
| 句型结构图 | 表格样式 | 将句子结构可视化为表格，每格对应一个语法成分 |

### 5.3 例句对照展示

所有语法例句遵循教学内容区的多语言渲染规则（§三），展示拼音/中文/解释语言。

## 六、底部操作区

### 6.1 固定底部栏

底部操作区固定在屏幕底部，不随内容滚动。高度 80px + 底部安全区。

### 6.2 首次学习模式

用户首次学习该课时（课时状态为"待学习"或"进行中"，且课时小测验未做过）：

| 元素 | 规格 | 说明 |
|------|------|------|
| 按钮文案 | "开始测验" | 主按钮（Rose 色，药丸形，宽度 70%，居中），多语言（EN: "Start Quiz" / VI: "Bắt đầu kiểm tra"） |
| 按钮状态 | 用户滚动浏览完内容后才可点击 | 用户未滚动到底部时按钮灰色禁用 + 文字"请先阅读完课时内容"（Caption 字号，灰色，按钮上方） |
| 点击行为 | 进入课时小测验页面（在课程考核 PRD 中定义） | — |

> **首次学习必须做测验才能标记完成**：用户不能跳过测验直接标记课时为"已完成"。测验完成后（无论对错），课时自动标记为已完成。

### 6.3 复习模式

用户重新查看已完成的课时（课时状态为"已完成"）：

| 元素 | 规格 | 说明 |
|------|------|------|
| 按钮文案 | "复习测验" | 次按钮（毛玻璃轮廓形，宽度 70%，居中），多语言（EN: "Review Quiz" / VI: "Ôn tập kiểm tra"） |
| 按钮状态 | 始终可点击 | — |
| 点击行为 | 进入课时小测验页面（复习模式，题目可能不同于首次） | — |

### 6.4 底部进度提示

- 底部按钮上方有一行进度文字
- 首次学习时："课时 X/Y · 请先阅读完课时内容" → 滚动到底后变为"课时 X/Y · 准备好了就开始测验吧！"
- 复习模式时："课时 X/Y · 已完成 ✅"

## 七、学习进度自动保存

### 7.1 保存规则

| 保存触发 | 保存内容 |
|---------|---------|
| 用户滚动页面 | 当前滚动位置（百分比或像素偏移） |
| 用户点击词汇卡片 | 已查看的词汇卡片索引 |
| 用户播放音频 | 已播放的音频句子索引 |
| 用户退出页面 | 上述所有数据打包保存 |
| 定时保存 | 每 30 秒自动保存一次当前状态 |

### 7.2 恢复规则

- 用户退出后再次进入同一课时 → 页面自动恢复到上次的滚动位置
- 顶部 Toast 提示"已恢复到上次学习位置"（多语言，2s 自动消失）
- 若课时内容有更新（后台编辑后重新发布）→ 不恢复位置，从头开始

### 7.3 离线缓存

- 已加载过的课时内容（文本 + 图片）缓存到本地
- 离线状态下可浏览已缓存的课时内容（只读）
- 音频不缓存（音频播放按钮展示为不可用态）
- 离线状态下不可进入测验

## 八、状态矩阵

| 页面状态 | 描述 | 设计要求 |
|---------|------|---------|
| **空状态 (Empty)** | 不适用 — 每个课时至少有教学内容 | — |
| **加载中 (Loading)** | 课时内容加载中 | 教学内容区 Skeleton：3-4 个文本行占位 + 1 个图片占位（200px 高灰色矩形）+ 词汇卡片区占位（1 个矩形卡片）+ 底部按钮占位 |
| **首次加载 (First Load)** | 首次进入课时 | 内容区淡入动画（300ms），词汇卡片区从右侧滑入（400ms） |
| **成功 (Success)** | 内容加载完成 | 正常展示教学内容 + 词汇 + 语法 + 底部按钮 |
| **错误 (Error)** | 内容加载失败 | 全屏错误提示"课时内容加载失败，请检查网络后重试"（多语言）+ 重试按钮 |
| **部分加载 (Partial)** | 文本已加载但图片/音频仍在加载 | 文本先展示；图片位置展示 Skeleton；音频按钮展示 Loading 态（灰色 + 转圈） |
| **离线 (Offline)** | 无网络 | 若有缓存：展示缓存内容（只读）+ 顶部 Banner "网络已断开，部分功能不可用"；音频和测验按钮不可用。若无缓存：全屏提示"请连接网络加载课时内容" + 返回按钮 |

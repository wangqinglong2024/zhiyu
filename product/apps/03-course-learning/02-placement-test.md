## 三、测试题目设计

### 3.1 题目数量和组成

| 模块 | 基础题量 | 自适应调整范围 | 内容 |
|------|---------|-------------|------|
| 识字能力 | 20 题 | 10-20 题 | 认字选义：展示汉字，选择正确释义 |
| 词汇能力 | 20 题 | 10-20 题 | 词义匹配：展示词语，选择对应解释或图片 |
| 阅读理解 | 10 题 | 5-10 题 | 短文理解：阅读一段文字，回答问题 |
| HSK 模拟 | 10 题 | 5-10 题 | 标准 HSK 题型混合（听力选择、语法填空等） |

> 总题量：实际作答 30-60 题（根据自适应算法动态调整，答题快的用户题量更少）

### 3.2 自适应出题逻辑

```
起始难度 = L3（中间水平）

每答一题：
  IF 答对 → 下一题难度 +1 级
  IF 答错 → 下一题难度 -1 级
  IF 连续答对 3 题 → 下一题难度 +2 级（跳级加速）
  IF 连续答错 3 题 → 下一题难度 -2 级（快速降级）

零基础快速检测：
  IF 前 10 题正确率 < 30% → 立即结束测试，推荐 L1

高水平快速检测：
  IF 前 15 题正确率 > 90% 且已达 L9+ 难度 → 可提前结束测试

终止条件（满足任一）：
  ① 已作答所有模块的最少题量（30 题）且置信度 ≥ 80%
  ② 已作答所有模块的最大题量（60 题）
  ③ 触发零基础快速检测
```

│  ━━━━━━━━━━━━━━━━━━━━━━░░░░░░░░░░░░              │

| 元素 | 规格 | 说明 |
|------|------|------|
| 退出按钮 | 顶部导航栏左侧 | 文字"退出测试"，点击 → 二次确认弹窗"确定退出吗？测试进度将不会保存"→ 确认 → 返回来源页 |
| 题号指示 | 顶部导航栏右侧 | "第 X/Y 题"（Y 为当前预估总题量，随自适应动态调整），多语言 |
| 进度条 | 导航栏下方 | 线性进度条，Rose 色填充，灰色背景 |
| 题干区域 | 页面中部 | Body L 字号（18px），黑色/白色（跟随主题）；题干中的中文文字使用粗体 |
| 选项区域 | 题干下方 | 每个选项为独立卡片（毛玻璃材质），点击选中后边框变为 Rose 色 + 轻微放大 `scale(1.02)` |

### 3.4 题目交互规则

- **不显示答题对错**：入学测试不即时反馈对错（区别于课时小测验），所有结果在最后统一展示
- **不可回退**：已答题目不可返回修改（自适应逻辑依赖顺序作答）
- **可跳过**：每道题下方有"跳过此题"链接，跳过的题计为答错（不影响体验，但影响推荐准确度）

### 3.5 时间限制

| 限制类型 | 规则 |
|---------|------|
| 单题限时 | **不限时** — 入学测试不对单题限时，让用户从容作答 |
| 总时长限制 | **不限时** — 无总时长上限。预计用时 5-15 分钟，但不强制 |
| 超时提醒 | 若作答超过 20 分钟，顶部 Toast 温和提示"测试不限时，你可以慢慢来 😊"（一次性提示，不重复） |

## 五、重新测试规则

### 5.1 重新测试入口

- 入口位置：「个人中心 → 入学测试」
- 已完成测试时，页面展示上次测试结果 + "重新测试"按钮

### 5.2 重新测试规则

| 规则 | 说明 |
|------|------|
| 可否重测 | **可以**，不限次数 |
| 结果处理 | 新结果**覆盖**前次结果，更新推荐 Level |
| 知语币奖励 | **不重复发放** — 仅首次完成发放 100 知语币 |
| 已有学习进度 | 重测不影响已有学习进度。仅更新推荐 Level，不会重置任何已完成的课程数据 |
| 冷却时间 | 无冷却时间限制，可随时重测 |

### 5.3 重新测试确认

- 点击"重新测试"按钮 → 弹出确认弹窗：
  - 标题："确认重新测试？"
  - 内容："重新测试将覆盖你之前的测试结果和推荐级别，但不会影响已有的学习进度。"
  - 按钮："确认重测"（Rose 色）/ "取消"（灰色）
  - 多语言

## 六、状态矩阵

### 6.1 测试说明页

| 页面状态 | 描述 | 设计要求 |
|---------|------|---------|
| **空状态** | 不适用 | — |
| **加载中** | 页面资源加载 | Skeleton 骨架屏：插图占位 + 文本行占位 + 按钮占位 |
| **首次加载** | 首次进入 | 正常展示说明页内容 |
| **成功** | 页面加载完成 | 正常展示 |
| **错误** | 加载失败 | 错误提示 + 重试按钮 |
| **部分加载** | 不适用 | — |
| **离线** | 无网络 | 提示"入学测试需要网络连接"+ 返回按钮 |

### 6.2 测试答题页

| 页面状态 | 描述 | 设计要求 |
|---------|------|---------|
| **空状态** | 不适用 | — |
| **加载中** | 题目加载中 | 题干区域 Skeleton + 选项区域 Skeleton（4 个卡片占位） |
| **首次加载** | 第一道题加载 | 正常展示题目 |
| **成功** | 用户作答后加载下一题 | 选项卡片切换动画（当前题向左滑出，新题从右滑入，300ms） |
| **错误** | 题目加载失败 | 题干区域展示"加载失败" + 重试按钮 |
| **部分加载** | 不适用 | — |
| **离线** | 答题过程中断网 | 顶部 Banner 提示 "网络已断开，请恢复网络后继续"；已答题目数据缓存本地；恢复网络后自动提交并加载下一题 |

### 6.3 测试结果页

| 页面状态 | 描述 | 设计要求 |
|---------|------|---------|
| **空状态** | 不适用 | — |
| **加载中** | 结果计算中 | 居中 Loading 动画 + "正在分析你的测试结果…"文案（多语言） |
| **首次加载** | 结果首次展示 | 庆祝动画 + 结果卡片入场动画（从下方滑入，500ms） |
| **成功** | 结果展示完成 | 正常展示推荐 Level + 雷达图（P1）+ 知语币奖励 |
| **错误** | 结果计算失败 | "结果生成失败，请重试" + 重试按钮；不扣除测试机会 |
| **部分加载** | 不适用 | — |
| **离线** | 提交答案后断网 | 答案缓存本地，提示"网络恢复后将自动提交并展示结果" |

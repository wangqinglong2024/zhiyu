# 多语言管理

> 关联文档：[管理后台概述 §七](../00-admin-overview.md)
> 路由路径：`/admin/i18n`
> 角色权限：超级管理员、内容运营
> 情绪目标：**精准、高效、可控** — 确保三语翻译准确无误，发布后立即同步到应用端

---

### 1.1 页面布局

### 1.6 文案列表表格

#### 表格布局

| 属性 | 说明 |
|------|------|
| **样式** | 毛玻璃面板（`.glass-card`），圆角 `rounded-2xl`（24px） |
| **表头** | 背景色 `rgba(255,255,255,0.03)`，Caption 级别 |
| **行高** | 自适应（最小 56px，文案内容多时自动撑高） |
| **分页** | 每页 50 条 |

#### 列定义

| # | 列名 | 宽度 | 内容说明 | 排序 |
|---|------|------|---------|------|
| 1 | 键名 | 220px | 等宽字体（如 `home.welcome_title`），Body S 级别，灰色；Hover 可复制（点击后 Toast"已复制"） | ✅ 字母序 |

| 5 | 操作 | 80px | "编辑"文字按钮，Sky 色 | 否 |

### 1.7 行内快速编辑

| 属性 | 说明 |
|------|------|
| **触发** | 双击"当前语言翻译"列的单元格 |
| **效果** | 单元格变为可编辑文本框，白色背景，自动 focus |
| **保存** | 按 Enter 或点击单元格外部 → 保存 → 状态更新为"已修改未发布" → 单元格恢复为只读 |
| **取消** | 按 Esc → 恢复原值 |
| **校验** | 不可为空（如需清空翻译，使用编辑弹窗） |

## 三、批量导入/导出

### 3.1 导出

#### 触发

点击页面标题右侧"导出"按钮。

#### 导出弹窗

| 属性 | 说明 |
|------|------|
| **弹窗标题** | "导出语言包" |
| **导出范围** | 单选组：全部文案 / 仅待翻译文案 / 仅当前筛选结果 |
| **导出格式** | 单选组：Excel (.xlsx) / CSV (.csv) |
| **导出列** | 键名 + 中文原文 + 英文翻译 + 越南语翻译 + 模块 + 翻译状态 |
| **按钮** | "取消" + "导出"（Rose 色实心） |
| **导出后** | 浏览器自动下载文件，文件名格式 `zhiyu_i18n_{日期}_{范围}.xlsx` → Toast"导出成功" |

### 3.2 导入

#### 触发

点击页面标题右侧"导入"按钮。

#### 导入弹窗

| 属性 | 说明 |
|------|------|
| **弹窗标题** | "导入语言包" |
| **上传区** | 虚线边框上传区域，支持 Excel (.xlsx) / CSV (.csv)，最大 10MB |
| **上传区文案** | Lucide `Upload` 32px + "点击或拖拽上传翻译文件" + "支持 .xlsx / .csv 格式" |

#### 导入预览

| 属性 | 说明 |
|------|------|
| **触发** | 文件上传成功后自动解析并展示预览 |
| **预览内容** | 匹配结果统计："共 X 条翻译，其中 Y 条已有翻译将被覆盖，Z 条为新增翻译" |
| **冲突处理** | 单选组："覆盖已有翻译"（默认）/ "跳过已有翻译（仅导入空缺项）" |
| **预览表格** | 展示前 10 条数据预览（键名 + 中文原文 + 英文翻译 + 越南语翻译 + 状态标签"新增"/"覆盖"/"跳过"） |

#### 导入按钮

| 按钮 | 样式 | 点击行为 |
|------|------|---------|
| **取消** | 灰色描边 | 关闭弹窗 |
| **确认导入** | Rose 色实心 | 执行导入 → 进度条展示 → 导入完成 → Toast"导入完成，共导入 X 条翻译"→ 关闭弹窗 → 列表刷新 → 导入的条目状态标记为"已修改未发布" |

#### 导入错误处理

| 场景 | 处理 |
|------|------|
| 文件格式错误 | Toast"文件格式不正确，请使用 .xlsx 或 .csv"（红色） |
| 键名不匹配 | 跳过不存在的键名，导入完成后提示"X 条键名不存在已跳过" |
| 内容为空 | 跳过空行 |

## 四、发布机制

### 4.1 发布按钮

| 属性 | 说明 |
|------|------|
| **位置** | 页面标题右侧 |
| **样式** | Rose 色实心按钮 + 数字角标（显示"已修改未发布"条目数量） |
| **状态** | 无待发布条目时：按钮 Disabled + 灰色 + 无角标 |

### 4.2 发布确认弹窗

| 属性 | 说明 |
|------|------|
| **弹窗标题** | "确认发布翻译" |
| **弹窗内容** | "确认将最新翻译同步到应用端？" + "本次发布包含 {N} 条文案变更。" + "发布后立即生效，用户将看到最新翻译。" |
| **变更预览** | 可展开的变更列表，显示键名 + 修改前后对比（最多展示前 20 条，超出显示"…及其他 X 条"） |
| **按钮** | "取消"（灰色描边）+ "确认发布"（Rose 色实心） |

### 4.3 发布执行

| 步骤 | 说明 |
|------|------|
| ① | 后端生成新版本号（自增整数，如 v42） |
| ② | 将所有"已修改未发布"条目的翻译同步到应用端语言包 CDN |
| ③ | 所有"已修改未发布"条目状态更新为"已翻译" |
| ④ | 写入发布记录（版本号+发布时间+操作人+变更数量+变更详情） |
| ⑤ | 写入系统操作日志 |

**发布后**：Toast"翻译已发布（版本 v{N}），共 {X} 条变更"（绿色）→ 列表刷新 → 角标消失。

**说明**：MVP 阶段发布后立即生效，无灰度发布机制。应用端下次打开时拉取最新语言包。

### 4.4 发布历史记录

> 路由路径：`/admin/i18n/history`
> 优先级：P2

| 属性 | 说明 |
|------|------|
| **入口** | 多语言管理页面标题下方"发布历史"文字链接（Sky 色） |
| **列表列** | 版本号（如 v42）+ 发布时间（YYYY-MM-DD HH:mm）+ 操作人 + 变更数量 |
| **操作** | "查看详情"→ 展开该版本的变更列表（键名 + 修改前 + 修改后） |
| **分页** | 每页 20 条 |

## 五、状态矩阵

### 5.1 语言包列表页

| 状态 | 表现 |
|------|------|
| **空状态 (Empty)** | 表格区域居中显示 Lucide `Languages` 48px 灰色图标 + "暂无文案数据"文案 + "请联系开发人员初始化语言包" |
| **加载中 (Loading)** | 表格区域 Skeleton 骨架屏（15 行） |
| **首次加载 (First Load)** | 全表格 Skeleton + 品牌 Logo 呼吸动画 |
| **成功 (Success)** | 正常展示文案列表；操作成功后顶部 Toast |
| **错误 (Error)** | "加载失败，请重试"+ Lucide `AlertCircle` + "重新加载"按钮 |
| **部分加载 (Partial)** | 翻页时底部加载指示器 |
| **离线 (Offline)** | 页面顶部 Amber 色 Banner："网络连接中断" |

### 5.2 翻译编辑弹窗

| 状态 | 表现 |
|------|------|
| **加载中** | 弹窗内容区 Skeleton 骨架屏 |
| **成功** | 保存成功 → Toast → 关闭弹窗 |
| **错误** | 保存失败 → Toast"保存失败"（红色）→ 弹窗不关闭，内容保留 |

### 5.3 导入/导出

| 状态 | 表现 |
|------|------|
| **导出中** | 导出按钮 Loading + Disabled，Toast"正在生成导出文件..." |
| **导出成功** | 浏览器下载启动 → Toast"导出成功" |
| **导入中** | 弹窗内显示进度条 |
| **导入成功** | Toast"导入完成"→ 关闭弹窗 → 列表刷新 |
| **导入失败** | Toast"导入失败：{原因}"→ 弹窗不关闭 |

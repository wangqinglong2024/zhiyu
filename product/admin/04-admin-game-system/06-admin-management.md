# 管理员管理

> 关联文档：[管理后台概述 §七](../00-admin-overview.md) | [登录与权限](../01-admin-dashboard/02-permissions.md)
> 路由路径：`/admin/administrators`
> 角色权限：**仅超级管理员**
> 情绪目标：**安全、掌控、严谨** — 管理员账号是系统安全的第一道防线

---

## 一、管理员列表

### 1.1 页面布局

| 属性 | 说明 |
|------|------|
| **页面标题** | "管理员管理"，H1 级别（32px，字重 700） |
| **权限提示** | 非超级管理员访问此页面时，跳转至 403 页面："您没有权限访问此页面" |
| **标题右侧** | "新建管理员"按钮，Rose 色实心按钮（`rounded-full`，高度 40px），Lucide `UserPlus` 图标 16px + 文字"新建管理员" |
| **列表区域** | 标题下方，间距 16px |

### 1.2 管理员列表表格

#### 表格布局

| 属性 | 说明 |
|------|------|
| **样式** | 毛玻璃面板（`.glass-card`），圆角 `rounded-2xl`（24px） |
| **表头** | 背景色 `rgba(255,255,255,0.03)`，Caption 级别 |
| **行高** | 56px |
| **行 Hover** | `rgba(255,255,255,0.04)` |
| **分页** | 每页 20 条 |
| **当前用户高亮** | 当前登录管理员所在行，左侧边框 3px Rose 色，背景加亮 `rgba(225,29,72,0.03)` |

#### 列定义

| # | 列名 | 宽度 | 内容说明 | 排序 |
|---|------|------|---------|------|
| 1 | 头像 | 56px | 40px 圆形头像；缺失时显示 Lucide `User` 灰色圆形占位 | 否 |
| 2 | 姓名 | 140px | Body 级别，白色；当前登录管理员后追加 Rose 色"(我)"标签 | 否 |
| 3 | 邮箱 | 220px | Body S 级别，灰色 `#a3a3a3` | 否 |
| 4 | 角色 | 120px | 标签样式：超级管理员 = Rose 色标签；内容运营 = Sky 色标签；用户运营 = Amber 色标签；游戏运营 = 绿色标签 `#22c55e` | 否 |
| 5 | 创建日期 | 120px | 格式 YYYY-MM-DD，Body S 级别 | ✅ 默认降序 |
| 6 | 最后登录 | 140px | 格式 YYYY-MM-DD HH:mm；超过 30 天未登录文字变红色 `#ef4444`；从未登录显示"从未登录"灰色斜体 | ✅ |
| 7 | 状态 | 80px | "正常" = 绿色圆点 + 文字；"已停用" = 红色圆点 + 文字 | 否 |
| 8 | 操作 | 240px | 操作按钮组，详见 §1.3 | 否 |

### 1.3 操作列

| 按钮 | 样式 | 条件 | 点击行为 |
|------|------|------|---------|
| **编辑** | 文字按钮，Sky 色 `#0284c7` | 始终显示 | 打开编辑管理员弹窗（详见 §3） |
| **重置密码** | 文字按钮，Amber 色 `#d97706` | 非自己账号时显示 | 打开重置密码确认弹窗（详见 §4） |
| **停用** | 文字按钮，错误色 `#ef4444` | 非自己账号 + 状态为"正常"时显示 | 打开停用确认弹窗（详见 §5.1） |
| **启用** | 文字按钮，成功色 `#22c55e` | 状态为"已停用"时显示 | 打开启用确认弹窗（详见 §5.2） |
| **删除** | 文字按钮，错误色 `#ef4444` | 非自己账号 + 非最后一个超级管理员时显示 | 打开删除确认弹窗（详见 §5.3） |

按钮间距 12px，水平排列。

**操作限制规则**：

| 规则 | 说明 |
|------|------|
| 不可操作自己 | 当前管理员行的"重置密码""停用""删除"按钮不显示 |
| 不可删除最后一个超级管理员 | 如系统中仅剩 1 个超级管理员，该管理员行的"停用"和"删除"按钮不显示 |
| 仅超级管理员可访问 | 其他角色访问 `/admin/administrators` 路由返回 403 |

## 二、新建管理员

### 2.1 触发

点击"新建管理员"按钮 → 打开新建管理员弹窗。

### 2.2 弹窗布局

| 属性 | 说明 |
|------|------|
| **弹窗宽度** | 520px |
| **弹窗标题** | "新建管理员" |
| **关闭** | 右上角 Lucide `X` 关闭按钮 |

### 2.3 表单字段

#### 字段 1：姓名

| 属性 | 说明 |
|------|------|
| **组件** | 文本输入框（`.glass-input`，`rounded-full`） |
| **标签** | "姓名"，标红星 |
| **Placeholder** | "请输入管理员姓名" |
| **校验** | 必填，2-20 个字符 |

#### 字段 2：邮箱

| 属性 | 说明 |
|------|------|
| **组件** | 文本输入框 |
| **标签** | "邮箱（登录账号）"，标红星 |
| **Placeholder** | "请输入邮箱地址" |
| **校验** | 必填，合法邮箱格式，不可与已有管理员邮箱重复 |
| **重复检测** | blur 后异步请求后端检查邮箱唯一性；重复时行内提示"该邮箱已被其他管理员使用" |

#### 字段 3：角色

| 属性 | 说明 |
|------|------|
| **组件** | 下拉单选（`.glass-input`，`rounded-md`） |
| **标签** | "角色"，标红星 |
| **选项** | 超级管理员 / 内容运营 / 用户运营 / 游戏运营 |
| **默认值** | 无（Placeholder "请选择角色"） |

**角色权限说明（弹窗内展示）**：

| 属性 | 说明 |
|------|------|
| **位置** | 角色下拉下方，灰色折叠区域 |
| **标题** | "角色权限说明"（可展开/收起） |
| **内容** | 表格展示 4 个角色的权限范围（参照管理后台概述 §1.3 角色权限表） |

### 2.4 弹窗按钮

| 按钮 | 样式 | 点击行为 |
|------|------|---------|
| **取消** | 灰色描边按钮 | 关闭弹窗 |
| **创建** | Rose 色实心按钮 | 校验全部字段 → 创建管理员 → 系统自动生成随机初始密码（12 位，含大小写字母+数字+特殊字符）→ 打开创建成功弹窗 |

### 2.5 创建成功弹窗

| 属性 | 说明 |
|------|------|
| **弹窗标题** | "管理员创建成功 ✓" |
| **弹窗内容** | "管理员「{姓名}」已创建成功。" |
| **初始密码展示** | 密码区域白色背景框，等宽字体显示密码明文，右侧"复制密码"按钮（Lucide `Copy`） |
| **安全提示** | Amber 色提示："⚠️ 初始密码仅此一次展示。密码已同时发送到 {邮箱}。管理员首次登录后将被强制修改密码。" |
| **按钮** | "我已复制密码，关闭"（Rose 色实心按钮） |

**邮件发送**：系统自动向新管理员邮箱发送创建通知邮件，包含：登录地址、邮箱账号、初始密码、"请在首次登录后立即修改密码"提示。

### 3.2 编辑弹窗

| 属性 | 说明 |
|------|------|
| **弹窗宽度** | 520px |
| **弹窗标题** | "编辑管理员" |

### 3.3 表单字段

#### 字段 1：姓名（可编辑）

| 属性 | 说明 |
|------|------|
| **组件** | 文本输入框，预填当前值 |
| **校验** | 必填，2-20 个字符 |

#### 字段 2：邮箱（不可编辑）

| 属性 | 说明 |
|------|------|
| **组件** | 文本输入框，Disabled + 灰色背景 |

## 四、重置密码

### 4.1 触发

点击管理员列表操作列的"重置密码"按钮。

### 4.2 确认弹窗

| 属性 | 说明 |
|------|------|
| **弹窗标题** | "确认重置密码" |
| **弹窗内容** | "确认重置管理员「{姓名}」({邮箱}) 的密码？" + "系统将生成新的临时密码并发送到其邮箱。该管理员下次登录时需修改密码。" |
| **按钮** | "取消"（灰色描边）+ "确认重置"（Amber 色实心） |

### 4.3 确认后

| 步骤 | 说明 |
|------|------|
| ① | 系统生成临时密码（12 位随机，同新建管理员规则） |
| ② | 发送密码重置邮件到管理员邮箱（包含新临时密码 + "请立即登录修改密码"提示） |
| ③ | 标记该管理员账号为"需修改密码"，下次登录强制修改 |
| ④ | 写入系统操作日志（操作人 + 目标管理员 + 时间，不记录密码内容） |
| ⑤ | Toast"密码已重置，新密码已发送至 {邮箱}"（绿色） |

## 五、停用/启用/删除管理员

### 5.1 停用管理员

#### 确认弹窗

| 属性 | 说明 |
|------|------|
| **弹窗标题** | "确认停用管理员" |
| **弹窗内容** | "确认停用管理员「{姓名}」({邮箱})？" + "停用后该管理员将无法登录管理后台，账号信息和操作记录保留。" |
| **按钮** | "取消"（灰色描边）+ "确认停用"（错误色 `#ef4444` 实心） |

#### 确认后

- 管理员状态 → "已停用"
- 如该管理员当前在线，其会话立即失效（下一次 API 请求返回 401）
- 写入系统操作日志
- Toast"管理员已停用"（绿色）→ 列表刷新

### 5.2 启用管理员

#### 确认弹窗

| 属性 | 说明 |
|------|------|
| **弹窗标题** | "确认启用管理员" |
| **弹窗内容** | "确认启用管理员「{姓名}」({邮箱})？启用后该管理员可正常登录。" |
| **按钮** | "取消"（灰色描边）+ "确认启用"（成功色 `#22c55e` 实心） |

#### 确认后

- 管理员状态 → "正常"
- 写入系统操作日志
- Toast"管理员已启用"（绿色）→ 列表刷新

### 5.3 删除管理员

#### 前置检查

| 检查项 | 处理 |
|--------|------|
| 是否为自己 | 不显示删除按钮 |
| 是否为最后一个超级管理员 | 不显示删除按钮；如通过 API 强制删除，返回错误"系统必须保留至少一个超级管理员" |

#### 确认弹窗（二次确认）

**第一次确认**：

| 属性 | 说明 |
|------|------|
| **弹窗标题** | "⚠️ 确认删除管理员" |
| **弹窗样式** | 顶部红色警告条 |
| **弹窗内容** | "确认删除管理员「{姓名}」({邮箱})？" + "**此操作不可恢复**。该管理员将无法登录，账号将被永久删除。其历史操作日志将保留以供审计。" |
| **按钮** | "取消"（灰色描边）+ "确认删除"（错误色 `#ef4444` 实心） |

#### 确认后

- 管理员记录硬删除（但 admin_logs 中该管理员的历史操作记录保留，操作人字段标记为"已删除管理员-{原姓名}"）
- 如该管理员当前在线，其会话立即失效
- 写入系统操作日志（操作人 + 删除目标 + 时间）
- Toast"管理员已删除"（绿色）→ 列表刷新

## 六、状态矩阵

### 6.1 管理员列表页

| 状态 | 表现 |
|------|------|
| **空状态 (Empty)** | 不适用（系统至少有 1 个超级管理员） |
| **加载中 (Loading)** | 表格区域 Skeleton 骨架屏（5 行） |
| **首次加载 (First Load)** | 全表格 Skeleton + 品牌 Logo 呼吸动画 |
| **成功 (Success)** | 正常展示管理员列表；当前用户行高亮；操作成功后顶部 Toast |
| **错误 (Error)** | "加载失败，请重试"+ Lucide `AlertCircle` + "重新加载"按钮 |
| **部分加载 (Partial)** | 不适用（管理员数据量小，无需翻页） |
| **离线 (Offline)** | 页面顶部 Amber 色 Banner："网络连接中断" |

### 6.2 新建/编辑管理员弹窗

| 状态 | 表现 |
|------|------|
| **加载中** | 编辑弹窗加载数据时，字段区域 Skeleton |
| **成功** | 创建成功 → 打开创建成功弹窗展示密码；编辑成功 → Toast → 关闭弹窗 |
| **错误** | 创建/编辑失败 → Toast（红色）→ 弹窗不关闭，内容保留 |
| **邮箱重复** | 邮箱字段红色边框 + 行内提示"该邮箱已被其他管理员使用" |

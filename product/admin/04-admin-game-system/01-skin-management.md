## 四、状态矩阵

本模块每个核心页面必须覆盖以下 7 种状态：

### 4.1 皮肤列表页

| 状态 | 表现 |
|------|------|
| **空状态 (Empty)** | 表格区域居中显示 Lucide `Palette` 48px 灰色图标 + "暂无皮肤"文案 + "新建皮肤"Rose 色 CTA 按钮 |
| **加载中 (Loading)** | 表格区域显示 Skeleton 骨架屏（10 行），模拟表格行形状 |
| **首次加载 (First Load)** | 全表格 Skeleton 骨架屏 + 品牌 Logo 呼吸动画 |
| **成功 (Success)** | 正常展示表格数据；操作成功后顶部 Toast 通知（2s 自动消失） |
| **错误 (Error)** | 表格区域显示"加载失败，请重试"+ Lucide `AlertCircle` 图标 + "重新加载"按钮 |
| **部分加载 (Partial)** | 翻页时底部显示加载指示器 |
| **离线 (Offline)** | 页面顶部显示 Amber 色 Banner："网络连接中断，显示的可能不是最新数据" |

### 4.2 新建/编辑皮肤页

| 状态 | 表现 |
|------|------|
| **空状态 (Empty)** | 新建时表单为空，各字段显示 Placeholder |
| **加载中 (Loading)** | 编辑时，表单区域显示 Skeleton 骨架屏，模拟字段形状 |
| **首次加载 (First Load)** | 编辑时同上 |
| **成功 (Success)** | 保存/上架成功 → Toast"已保存草稿"或"皮肤已上架"→ 跳转列表 |
| **错误 (Error)** | 保存失败 → Toast"保存失败，请重试"（红色，需手动关闭）→ 表单内容保留不清空 |
| **部分加载 (Partial)** | 图片上传中显示进度条 |
| **离线 (Offline)** | 顶部 Banner 提示网络中断 + 保存按钮 Disabled |

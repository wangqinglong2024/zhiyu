## 一、顶部指标卡片区

### 1.1 布局

| 属性 | 说明 |
|------|------|
| **位置** | 页面内容区最顶部 |
| **布局** | 一行 6 张卡片，等宽网格布局（`grid-cols-6`），卡片间距 16px |
| **卡片样式** | 毛玻璃面板（`.glass-card`），圆角 `rounded-2xl`（24px），内边距 20px |
| **响应式** | ≥1280px 一行 6 张；1024-1279px 一行 3 张（两行）；768-1023px 一行 2 张（三行） |

### 1.2 卡片定义

#### 卡片 1：总用户数

| 属性 | 说明 |
|------|------|
| **标题** | "总用户数" |
| **图标** | Lucide `Users`，24px，Sky 色 `#0284c7` |
| **主数值** | 注册用户总数，格式：≥10,000 显示为"X.X 万"（如 12,345 → "1.2 万"）；<10,000 显示完整数字，使用千分位分隔（如 "9,876"） |
| **副指标** | "今日新增 +XX"，正数绿色 `#22c55e`，零值灰色 `#a3a3a3` |
| **较上周期变化** | 与昨日同一时刻对比，格式："+12.3%" 绿色上箭头 / "-5.2%" 红色下箭头 / "0.0%" 灰色横线 |
| **角色可见** | 超级管理员、用户运营 |

#### 卡片 2：今日活跃用户数（DAU）

| 属性 | 说明 |
|------|------|
| **标题** | "今日活跃" |
| **图标** | Lucide `Activity`，24px，Sky 色 `#0284c7` |
| **主数值** | 今日（UTC 0:00 起）活跃用户数，格式同总用户数 |
| **副指标** | "DAU/MAU XX%"，显示日活占月活比例 |
| **较上周期变化** | 与昨日同一时刻对比，格式同上 |
| **角色可见** | 超级管理员、用户运营 |

#### 卡片 3：付费用户数

| 属性 | 说明 |
|------|------|
| **标题** | "付费用户" |
| **图标** | Lucide `CreditCard`，24px，Amber 色 `#d97706` |
| **主数值** | 累计付费用户总数，格式同总用户数 |
| **副指标** | "付费率 X.X%"（付费用户数 / 总用户数 × 100%），保留一位小数 |
| **较上周期变化** | 付费率与昨日对比，"+0.3%" / "-0.1%" |
| **角色可见** | 超级管理员、用户运营 |

#### 卡片 4：今日营收

| 属性 | 说明 |
|------|------|
| **标题** | "今日营收" |
| **图标** | Lucide `DollarSign`，24px，Amber 色 `#d97706` |
| **主数值** | 今日（UTC 0:00 起）收入金额，格式："$X,XXX.XX"（USD），保留两位小数 |
| **副指标** | "较昨日 +XX.X%" 或 "较昨日 -XX.X%"，正数绿色，负数红色 |
| **较上周期变化** | 与昨日全天对比（如今日未结束则按比例推算） |
| **角色可见** | 超级管理员、用户运营 |

#### 卡片 5：在线游戏人数

| 属性 | 说明 |
|------|------|
| **标题** | "在线游戏" |
| **图标** | Lucide `Gamepad2`，24px，Rose 色 `#e11d48` |
| **主数值** | 当前正在进行游戏对局的用户数，实时更新（每 30 秒刷新） |
| **副指标** | "今日峰值 XXX"，显示今日最高同时在线数 |
| **实时标识** | 主数值左侧绿色圆点闪烁（脉冲动画），表示实时数据 |
| **较上周期变化** | 与昨日同一时刻对比 |
| **角色可见** | 超级管理员、游戏运营 |

#### 卡片 6：待处理退款

| 属性 | 说明 |
|------|------|
| **标题** | "待处理退款" |
| **图标** | Lucide `AlertTriangle`，24px，错误色 `#ef4444` |
| **主数值** | 当前待审核退款申请数量，整数 |
| **副指标** | "最早 X 天前"，显示最早一笔待处理退款距今天数（超过 3 天文字变红色） |
| **特殊样式** | 当数量 > 0 时，卡片边框变为红色 `#ef4444`（1px），数值变为红色加粗 |
| **点击行为** | 整张卡片可点击，点击跳转至退款管理页面 `/admin/orders/refunds` |
| **角色可见** | 超级管理员、用户运营 |

### 1.3 卡片通用交互

| 行为 | 说明 |
|------|------|
| **Hover 态** | 卡片阴影从 Level 1 提升至 Level 2（`0 4px 12px` 弥散阴影），过渡 200ms |
| **加载中** | 骨架屏：标题行 + 数值行 + 副指标行三行矩形骨架闪烁 |
| **数据刷新** | 页面加载时请求一次；在线游戏人数每 30 秒自动刷新；其他卡片每 5 分钟自动刷新 |
| **数据异常** | 接口返回错误时，卡片主数值显示"--"，副指标显示"数据加载失败"，灰色 |

## 二、趋势图表区

### 2.1 布局

| 属性 | 说明 |
|------|------|
| **位置** | 指标卡片区下方，间距 24px |
| **布局** | 一行 3 张图表卡片，等宽网格布局（`grid-cols-3`），卡片间距 16px |
| **卡片样式** | 毛玻璃面板（`.glass-card`），圆角 `rounded-2xl`（24px），内边距 24px |
| **图表高度** | 240px |
| **响应式** | ≥1280px 一行 3 张；1024-1279px 一行 2 张 + 1 张换行；768-1023px 一行 1 张（纵向堆叠） |

### 2.2 图表 1：用户增长趋势

| 属性 | 说明 |
|------|------|
| **标题** | "用户增长趋势" |
| **图表类型** | 折线图 |
| **数据线** | 单条线：每日新增注册用户数 |
| **线条颜色** | Sky 色 `#0284c7` |
| **线条下方** | 渐变填充区域（Sky 色 10% 透明度渐变至透明） |
| **时间范围切换** | 图表右上角 Segmented Control（分段选择器）："7天" / "30天" / "90天"，默认"7天" |
| **X 轴** | 日期，格式 "MM/DD"（7天）/ "MM/DD"（30天）/ "MM月"（90天） |
| **Y 轴** | 用户数，自动缩放，标签格式同卡片数值规则 |
| **Hover 交互** | 鼠标悬停数据点时显示 Tooltip："2026-04-17\n新增用户：XX" |
| **角色可见** | 超级管理员、用户运营 |

### 2.3 图表 2：营收趋势

| 属性 | 说明 |
|------|------|
| **标题** | "营收趋势" |
| **图表类型** | 折线图 |
| **数据线** | 单条线：每日营收金额（USD） |
| **线条颜色** | Amber 色 `#d97706` |
| **线条下方** | 渐变填充区域（Amber 色 10% 透明度渐变至透明） |
| **时间范围切换** | 同用户增长趋势 |
| **X 轴** | 同用户增长趋势 |
| **Y 轴** | 金额，格式 "$X" 或 "$X.Xk" |
| **Hover 交互** | Tooltip："2026-04-17\n营收：$XXX.XX" |
| **角色可见** | 超级管理员、用户运营 |

### 2.4 图表 3：日活趋势

| 属性 | 说明 |
|------|------|
| **标题** | "日活趋势" |
| **图表类型** | 折线图 |
| **数据线** | 单条线：每日活跃用户数 |
| **线条颜色** | Rose 色 `#e11d48` |
| **线条下方** | 渐变填充区域（Rose 色 10% 透明度渐变至透明） |
| **时间范围切换** | 同用户增长趋势 |
| **X 轴** | 同用户增长趋势 |
| **Y 轴** | 用户数，自动缩放 |
| **Hover 交互** | Tooltip："2026-04-17\nDAU：XX" |
| **角色可见** | 超级管理员、用户运营 |

### 2.5 图表通用交互

| 行为 | 说明 |
|------|------|
| **加载中** | 图表区域显示矩形骨架屏（模拟图表形状） |
| **数据为空** | 图表区域居中显示"暂无数据"文字 + Lucide `BarChart3` 图标（32px，中性灰） |
| **时间范围切换** | 切换时图表数据平滑过渡动画（300ms），非重新渲染 |
| **数据刷新** | 页面加载时请求一次，每 5 分钟自动刷新 |

## 四、快捷操作区

### 4.1 布局

| 属性 | 说明 |
|------|------|
| **位置** | 内容概览卡片区下方，间距 24px |
| **标题** | "快捷操作"，字号 H3（20px），字重 600 |
| **布局** | 一行水平排列按钮，间距 12px |

### 4.2 按钮定义

| 按钮文案 | 图标（Lucide） | 跳转目标 | 按钮样式 | 角色可见 |
|---------|---------------|---------|---------|---------|
| "新建文章" | `Plus` | `/admin/content/articles/new` | 次要按钮（毛玻璃背景 + Sky 色文字） | 超级管理员、内容运营 |
| "金句排期" | `Calendar` | `/admin/content/quotes` | 次要按钮 | 超级管理员、内容运营 |
| "查看待审退款" | `RotateCcw` | `/admin/orders/refunds?status=pending` | 次要按钮（有待处理时文字变 Rose 色） | 超级管理员、用户运营 |
| "发送推送" | `Send` | `/admin/system/push/new` | 次要按钮 | 仅超级管理员 |
| "皮肤上架" | `Palette` | `/admin/games/skins/new` | 次要按钮 | 超级管理员、游戏运营 |
| "赛季配置" | `Trophy` | `/admin/games/seasons` | 次要按钮 | 超级管理员、游戏运营 |

### 4.3 按钮交互

| 行为 | 说明 |
|------|------|
| **默认态** | 毛玻璃背景 `rgba(255, 255, 255, 0.06)`，圆角 `rounded-lg`（16px），内边距 12px 20px |
| **Hover 态** | 背景变为 `rgba(255, 255, 255, 0.1)`，图标和文字色变亮 |
| **Active 态** | 缩小 `scale(0.97)`，100ms 回弹 |
| **角色过滤** | 按角色权限仅显示有权限的按钮，无权限按钮不渲染 |

## 五、最近操作日志

### 5.1 布局

| 属性 | 说明 |
|------|------|
| **位置** | 快捷操作区下方，间距 24px，仪表盘页面最底部区域 |
| **标题** | "最近操作"，字号 H3（20px），字重 600，右侧附"查看全部"链接（跳转系统日志页面） |
| **卡片样式** | 毛玻璃面板（`.glass-card`），圆角 `rounded-2xl`（24px），内边距 24px |

### 5.2 日志列表

| 属性 | 说明 |
|------|------|
| **条数** | 最近 10 条管理员操作记录 |
| **排序** | 按时间倒序（最新在上） |
| **每行结构** | `[操作人头像(24px)] 操作人姓名 · 操作描述 · 时间` |

### 5.3 每行展示字段

| 字段 | 格式 | 示例 |
|------|------|------|
| **操作人头像** | 圆形小头像 24px，无头像显示姓名首字 | — |
| **操作人姓名** | 字号 Body S（14px），白色，字重 500 | "张三" |
| **操作描述** | 字号 Body S（14px），中性灰 `#a3a3a3` | "发布了文章《春节的由来》" |
| **操作时间** | 字号 Caption（12px），中性灰 `#a3a3a3` | 24 小时内："X 分钟前" / "X 小时前"；超过 24 小时："04/17 14:30" |

### 5.4 操作描述文案模板

| 操作类型 | 描述文案 |
|---------|---------|
| 文章发布 | "发布了文章《{文章标题}》" |
| 文章下架 | "下架了文章《{文章标题}》" |
| 课程上线 | "上线了课程 {Level 名称}" |
| 金句排期 | "排期了 {日期} 的每日金句" |
| 用户封禁 | "封禁了用户 {用户昵称}" |
| 用户解封 | "解封了用户 {用户昵称}" |
| 退款审批 | "审批了退款订单 #{订单号}" |
| 知语币调整 | "调整了用户 {用户昵称} 的知语币 {+/-}{数量}" |
| 皮肤上架 | "上架了皮肤《{皮肤名称}》" |
| 赛季创建 | "创建了赛季《{赛季名称}》" |
| 推送发送 | "发送了推送通知《{推送标题}》" |
| 管理员创建 | "创建了管理员账号 {管理员姓名}" |

### 5.5 日志交互

| 行为 | 说明 |
|------|------|
| **Hover 态** | 当前行背景变为 `rgba(255, 255, 255, 0.03)` |
| **空状态** | 列表区域居中显示"暂无操作记录"文字 + Lucide `ScrollText` 图标（32px，中性灰） |
| **角色过滤** | 超级管理员：看到全部操作日志；其他角色：仅看到自己的操作记录 |
| **敏感操作标识** | 退款审批、用户封禁、知语币调整、管理员创建等操作，操作描述前显示 Lucide `AlertCircle` 图标（14px，Amber 色 `#d97706`） |
| **"查看全部"链接** | 仅超级管理员可见（跳转 `/admin/system/logs`），其他角色不显示此链接 |

## 七、状态矩阵

| 状态 | 场景描述 | 页面表现 |
|------|---------|---------|
| **空状态 (Empty)** | 系统刚上线，无用户/无内容/无订单数据 | 指标卡片主数值显示"0"，趋势图表显示"暂无数据"空状态，内容概览显示空状态插画 + "暂无内容，开始创建吧" + CTA 按钮 |
| **加载中 (Loading)** | 页面首次加载或数据刷新中 | 所有卡片和图表区域显示骨架屏（Skeleton），骨架形状与最终内容匹配 |
| **首次加载 (First Load)** | 登录后第一次进入仪表盘 | 各区域依次入场动画（从下方 `translateY(20px)` 淡入）：指标卡片 → 趋势图表 → 内容概览 → 快捷操作 → 操作日志，每组间隔 100ms |
| **成功 (Success)** | 数据加载完成 | 所有组件正常展示数据，骨架屏消失（200ms 淡出） |
| **错误 (Error)** | 数据加载失败（网络错误/服务端异常） | 失败区域显示错误提示："数据加载失败" + "重试"按钮（Lucide `RefreshCw` 图标），其他正常加载的区域不受影响 |
| **部分加载 (Partial)** | 部分接口成功、部分失败 | 成功的区域正常展示，失败的区域独立显示错误状态和重试按钮，互不影响 |
| **离线 (Offline)** | 网络断开 | 页面顶部显示黄色 Banner："网络连接已断开，数据可能不是最新"，已加载的数据继续展示但不再自动刷新 |

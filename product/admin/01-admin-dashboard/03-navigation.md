## 二、侧边栏交互

### 2.1 基本布局

| 属性 | 说明 |
|------|------|
| **位置** | 页面左侧固定定位 |
| **展开宽度** | 240px |
| **折叠宽度** | 64px（仅显示图标） |
| **高度** | 100vh（全屏高度） |
| **背景** | 毛玻璃面板（`.glass-card`），暗色模式下 `rgba(14, 14, 14, 0.85)` + `backdrop-filter: blur(20px)` |
| **分隔线** | 右边缘 1px 分隔线，`rgba(255, 255, 255, 0.06)` |
| **顶部** | 品牌 Logo 区域——展开态显示 Logo 图标（32px）+ "知语 Zhiyu"文字；折叠态仅显示 Logo 图标 |
| **底部** | 折叠/展开切换按钮 |

### 2.2 折叠/展开行为

| 行为 | 说明 |
|------|------|
| **切换按钮** | 侧边栏底部居中，Lucide `PanelLeftClose`（展开态）/ `PanelLeftOpen`（折叠态），24px |
| **切换动画** | 宽度过渡 240px ↔ 64px，300ms，缓动曲线 Standard `cubic-bezier(0.2, 0, 0, 1)` |
| **折叠态菜单** | 一级菜单仅显示图标（24px），居中对齐；Hover 时右侧弹出 Tooltip 显示菜单名称 |
| **折叠态二级菜单** | Hover 一级菜单图标时，右侧弹出浮层面板显示该一级菜单的全部二级菜单项 |
| **默认状态** | 展开态 |
| **状态记忆** | 折叠/展开状态存储在本地 `localStorage`，刷新后保持 |

### 2.3 菜单项交互

| 状态 | 表现 |
|------|------|
| **默认态** | 文字色 `#a3a3a3`（中性灰），图标色继承文字色 |
| **Hover 态** | 背景色 `rgba(255, 255, 255, 0.06)`，文字色变白 `#fafafa`，圆角 `rounded-md`（8px），过渡 150ms |
| **当前页高亮** | 背景色 `rgba(225, 29, 72, 0.1)`（Rose 10%），文字色 Rose `#e11d48`，图标色 Rose，左边缘 3px Rose 色竖条指示器 |
| **一级菜单展开/收起** | 点击含二级菜单的一级菜单项，右侧箭头图标 Lucide `ChevronDown` 旋转 180°（收起→展开），二级菜单列表动画展开/收起（高度过渡 200ms） |
| **二级菜单项** | 左侧缩进 40px（相对一级菜单），字号 Body S（14px） |
| **当前一级菜单** | 如果当前页面属于某一级菜单的二级菜单，则该一级菜单自动展开，一级菜单文字变为白色 |

### 2.4 徽标/计数

| 菜单项 | 徽标条件 | 徽标展示 |
|--------|---------|---------|
| **退款管理** | 有待处理退款时 | 菜单项右侧红色圆形徽标，显示待处理数量（超过 99 显示"99+"） |
| **系统日志** | 有未读告警时 | 菜单项右侧红色圆点（无数字） |

## 三、顶部栏

### 3.1 整体布局

```
┌──────────────────────────────────────────────────────────────────┐
│  面包屑导航                              管理员信息 + 退出登录    │
```

| 属性 | 说明 |
|------|------|
| **位置** | 内容区顶部固定（侧边栏右侧，水平铺满剩余宽度） |
| **高度** | 56px |
| **背景** | 毛玻璃效果，`rgba(14, 14, 14, 0.6)` + `backdrop-filter: blur(12px)` |
| **底部分隔线** | 1px，`rgba(255, 255, 255, 0.06)` |
| **左侧** | 面包屑导航 |
| **右侧** | 管理员信息区 |

### 3.2 左侧 — 面包屑导航

| 属性 | 说明 |
|------|------|
| **分隔符** | Lucide `ChevronRight` 图标，12px，中性灰 `#a3a3a3` |
| **文字样式** | 字号 Body S（14px），非末级为 Sky 色 `#0284c7` 可点击链接，末级为白色 `#fafafa` 不可点击 |

### 3.3 右侧 — 管理员信息区

| 元素 | 说明 |
|------|------|
| **头像** | 管理员头像（圆形，32px），无头像时显示姓名首字（Rose 色背景 + 白色文字） |
| **姓名** | 管理员姓名，字号 Body S（14px），白色 |
| **角色标签** | 角色名称标签，字号 Caption（12px），药丸形 `rounded-full`，颜色：超管=Amber `#d97706`，其他=Sky `#0284c7` |
| **退出登录按钮** | 文案"退出登录"，Lucide `LogOut` 图标 + 文字，字号 Body S（14px），中性灰，Hover 态变 Rose 色 |

## 四、面包屑导航规则

### 4.1 层级生成规则

| 页面层级 | 面包屑内容 | 示例 |
|---------|-----------|------|
| **一级页面（无二级菜单）** | 一级菜单名称 | `仪表盘` |
| **二级页面** | 一级菜单名称 / 二级菜单名称 | `内容管理 / 文章管理` |
| **三级页面（详情/编辑）** | 一级菜单名称 / 二级菜单名称 / 操作名称 | `内容管理 / 文章管理 / 编辑文章` |
| **三级页面（新建）** | 一级菜单名称 / 二级菜单名称 / 操作名称 | `内容管理 / 文章管理 / 新建文章` |

### 4.2 点击行为

| 面包屑层级 | 点击行为 |
|-----------|---------|
| **第一级（一级菜单名称）** | 无二级菜单时：跳转该页面；有二级菜单时：跳转该一级菜单下的第一个二级菜单页面 |
| **第二级（二级菜单名称）** | 跳转该二级菜单页面（列表页） |
| **最后一级（当前页）** | 不可点击，仅展示当前位置 |

### 4.3 特殊页面面包屑

| 页面 | 面包屑 |

## 五、响应式行为

### 5.1 断点定义

| 断点 | 宽度范围 | 侧边栏行为 | 内容区行为 |
|------|---------|-----------|-----------|
| **桌面端（默认）** | ≥ 1280px | 默认展开（240px），可手动折叠 | 自适应宽度，最大 `calc(100vw - 240px)` |
| **小桌面端** | 1024px - 1279px | 默认折叠（64px），可手动展开 | 自适应宽度 |
| **平板端** | 768px - 1023px | 默认隐藏，通过汉堡菜单按钮唤出（overlay 模式，覆盖在内容区上方） | 全宽 |

### 5.2 平板端 Overlay 模式

| 属性 | 说明 |
|------|------|
| **触发** | 顶部栏左侧新增汉堡菜单按钮（Lucide `Menu`，24px） |
| **展开** | 侧边栏从左侧滑入（宽度 280px），内容区右侧出现半透明遮罩 `rgba(0, 0, 0, 0.5)` |
| **关闭** | 点击遮罩区域、点击侧边栏内菜单项后、点击汉堡按钮 |
| **动画** | 侧边栏 `translateX(-100%) → translateX(0)`，300ms ease-out；遮罩 `opacity: 0 → 0.5`，200ms |

### 5.3 不支持移动端

| 规则 | 说明 |
|------|------|
| **最小支持宽度** | 768px |
| **小于 768px** | 显示全屏提示页："请使用电脑或平板设备访问管理后台"，居中显示 Lucide `Monitor` 图标 + 提示文字 |

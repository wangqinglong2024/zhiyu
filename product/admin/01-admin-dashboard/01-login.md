## 二、登录表单

### 2.1 邮箱输入框

| 属性 | 说明 |
|------|------|
| **标签** | 浮动标签（Float Label），文案："邮箱地址" |
| **Placeholder** | "请输入邮箱地址" |
| **输入类型** | `email` |
| **自动填充** | 启用浏览器 autocomplete="email" |
| **校验规则** | ① 非空校验："请输入邮箱地址"；② 格式校验（标准邮箱格式）："请输入有效的邮箱地址" |
| **校验时机** | 失焦（blur）时触发即时校验 |
| **错误展示** | 输入框下方红色文字（字号 Caption 12px），输入框边框变为错误色 `#ef4444` |
| **清除按钮** | 有内容时右侧显示 × 清除按钮（Lucide `X` 图标，16px） |
| **图标** | 左侧 Lucide `Mail` 图标，20px，中性灰色 |

### 2.2 密码输入框

| 属性 | 说明 |
|------|------|
| **标签** | 浮动标签（Float Label），文案："密码" |
| **Placeholder** | "请输入密码" |
| **输入类型** | `password`（默认隐藏） |
| **自动填充** | 启用浏览器 autocomplete="current-password" |
| **显示/隐藏切换** | 右侧眼睛图标按钮：隐藏态 Lucide `EyeOff`（默认）→ 点击切换为 `Eye`（明文显示）→ 再点击切回隐藏 |
| **校验规则** | 非空校验："请输入密码" |
| **校验时机** | 失焦（blur）时触发即时校验 |
| **错误展示** | 同邮箱输入框规则 |
| **图标** | 左侧 Lucide `Lock` 图标，20px，中性灰色 |

### 2.3 "记住我" 复选框

| 属性 | 说明 |
|------|------|
| **位置** | 密码输入框下方，左对齐 |
| **文案** | "记住我（7天）" |
| **默认状态** | 未勾选 |
| **行为** | 勾选后登录成功时会话有效期延长至 7 天；未勾选则会话有效期为 2 小时 |
| **样式** | 标准复选框，选中态为 Rose 色 `#e11d48` |

### 2.4 "登录" 按钮

| 属性 | 说明 |
|------|------|
| **文案** | "登 录" |
| **样式** | 全宽（与卡片内容区等宽），药丸形圆角（`rounded-full`），背景色 Rose `#e11d48`，白色文字，字号 Body（16px），字重 600 |
| **可用条件** | 邮箱和密码输入框均有值时可用（非空） |
| **禁用态** | 邮箱或密码为空时，按钮置灰（opacity 50%），不可点击 |
| **Hover 态** | 背景色加深 10%，鼠标变为 pointer |
| **Active 态** | 按钮缩小 `scale(0.97)`，100ms 弹性回弹 |
| **Loading 态** | 点击后按钮文案替换为旋转 Spinner + "登录中..."，按钮 Disabled 防止重复提交 |
| **键盘支持** | 在密码输入框按 Enter 键等同于点击登录按钮 |

## 三、安全机制

### 3.1 登录失败次数限制

| 规则 | 说明 |
|------|------|
| **最大连续失败次数** | 5 次 |
| **计数范围** | 以邮箱地址为维度，同一邮箱连续失败累计 |
| **失败提示（第 1-4 次）** | 登录表单上方显示红色警告 Toast："邮箱或密码错误，还剩 X 次尝试机会" |
| **失败提示（第 5 次）** | 登录表单上方显示红色警告 Toast："账号已锁定，请 15 分钟后重试" |
| **锁定时间** | 15 分钟 |
| **锁定期间表现** | 登录按钮置灰不可点击，按钮上方显示倒计时："账号已锁定，XX:XX 后可重试"（分:秒倒计时） |
| **锁定解除** | 15 分钟后自动解除，失败计数归零，按钮恢复可用 |
| **成功后重置** | 登录成功后失败计数归零 |

### 3.2 倒计时展示规则

| 状态 | 展示内容 |
|------|---------|
| **未锁定，首次失败** | "邮箱或密码错误，还剩 4 次尝试机会" |
| **未锁定，第 2 次失败** | "邮箱或密码错误，还剩 3 次尝试机会" |
| **未锁定，第 3 次失败** | "邮箱或密码错误，还剩 2 次尝试机会" |
| **未锁定，第 4 次失败** | "邮箱或密码错误，还剩 1 次尝试机会" |
| **锁定中** | "账号已锁定，14:59 后可重试"（实时倒计时） |
| **锁定解除** | 倒计时消失，表单恢复正常可输入状态 |

### 3.3 安全规则说明

| 规则 | 说明 |
|------|------|
| **密码错误不暴露原因** | 始终提示"邮箱或密码错误"，不区分"邮箱不存在"或"密码错误" |
| **不提供注册入口** | 登录页无"注册"链接或按钮 |
| **不提供密码找回** | 登录页无"忘记密码"链接——密码重置由超级管理员在管理员管理页面操作 |
| **HTTPS 强制** | 管理后台仅通过 HTTPS 访问 |

## 四、登录成功后跳转逻辑

| 场景 | 跳转目标 |
|------|---------|
| **正常登录** | 跳转至仪表盘首页（`/admin/dashboard`） |
| **会话过期重新登录** | 跳转至会话过期前最后访问的页面（URL 参数 `redirect` 携带原路径） |
| **登录后目标页面无权限** | 跳转至仪表盘首页，并 Toast 提示"您无权访问该页面" |

### 登录成功流程

```
用户输入邮箱+密码 → 点击"登录" → Loading 态
  → 验证成功 → 获取用户角色信息 → 初始化侧边栏菜单（按角色过滤）
  → 跳转至目标页面 → 显示绿色 Toast："登录成功，欢迎回来，[管理员姓名]"
```

## 五、会话管理

### 5.1 会话有效期

| 场景 | 有效期 | 说明 |
|------|--------|------|
| **未勾选"记住我"** | 2 小时 | 2 小时无操作后自动退出 |
| **勾选"记住我"** | 7 天 | 7 天内关闭浏览器重新打开无需再次登录 |

### 5.2 会话超时行为

| 步骤 | 说明 |
|------|------|
| **超时前 5 分钟** | 页面右上角弹出倒计时提醒浮层："您的会话即将过期，剩余 5:00"，提供"延续会话"按钮 |
| **点击"延续会话"** | 刷新会话有效期（重新计时），浮层消失 |
| **超时触发** | ① 清除本地凭证（Token）；② 页面显示全屏遮罩："您的会话已过期，请重新登录"；③ 遮罩上有"重新登录"按钮 |
| **点击"重新登录"** | 跳转至登录页，URL 携带 `redirect` 参数保存当前页面路径 |

### 5.3 主动退出

| 步骤 | 说明 |
|------|------|
| **触发** | 点击顶部栏右侧"退出登录"按钮 |
| **确认弹窗** | 模态对话框："确定要退出登录吗？"，按钮："取消"（次要按钮）+ "确认退出"（Rose 色主按钮） |
| **确认退出后** | ① 清除本地凭证；② 跳转至登录页；③ 显示绿色 Toast："已安全退出" |

## 六、无"注册"和"忘记密码自助找回"

| 规则 | 说明 |
|------|------|
| **账号创建** | 由超级管理员在"系统设置 → 管理员管理"页面创建，需填写姓名、邮箱、分配角色，系统生成临时密码并通过邮件发送 |
| **密码重置** | 由超级管理员在"管理员管理"页面点击"重置密码"操作，系统生成新的临时密码并通过邮件发送给对应管理员 |
| **首次登录** | 使用临时密码登录后，系统强制要求修改密码，修改成功后才可进入后台 |
| **修改密码弹窗** | 字段：当前密码 + 新密码 + 确认新密码。新密码规则：8-32 位，必须包含大小写字母和数字 |
| **登录页不展示任何引导** | 页面上无注册链接、无忘记密码链接、无任何第三方登录入口 |

## 七、状态矩阵

| 状态 | 场景描述 | 页面表现 |
|------|---------|---------|
| **空状态 (Empty)** | 不适用（登录页始终有表单） | — |
| **加载中 (Loading)** | 点击登录按钮后等待服务端响应 | 登录按钮显示 Spinner + "登录中..."，按钮 Disabled，表单输入框不可编辑 |
| **首次加载 (First Load)** | 打开登录页 | 登录卡片带入场动画（从下方 `translateY(20px)` 淡入，300ms ease-out），输入框自动 focus 到邮箱输入框 |
| **成功 (Success)** | 登录验证通过 | 绿色 Toast："登录成功，欢迎回来，[管理员姓名]"，2s 后自动消失，同时跳转目标页面 |
| **错误 (Error)** | 邮箱或密码错误 | 红色警告文字出现在表单上方："邮箱或密码错误，还剩 X 次尝试机会"；密码输入框轻微抖动动效（水平位移 ±4px，持续 300ms） |
| **锁定 (Locked)** | 连续 5 次失败触发锁定 | 表单上方红色警告区显示倒计时；登录按钮置灰 Disabled；邮箱和密码输入框仍可编辑（允许用户更正信息） |
| **会话过期 (Session Expired)** | 2 小时无操作或 7 天到期 | 全屏半透明遮罩 + 居中提示卡片："您的会话已过期，请重新登录" + "重新登录"按钮 |

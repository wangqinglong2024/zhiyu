## 一、数据流向说明

### 1.1 登录与鉴权数据流

```
管理员输入邮箱+密码
      ↓
前端表单校验（非空、邮箱格式）
      ↓ 通过
发送登录请求 → Supabase Auth 验证凭证
      ↓                    ↓
  验证成功               验证失败
      ↓                    ↓
返回 Token + 用户信息    返回错误码
（含角色、姓名、头像）     ↓
      ↓               前端累计失败次数
前端存储 Token         ↓ ≥5 次
（记住我=localStorage;    锁定该邮箱 15 分钟
 否则=sessionStorage）
      ↓
请求权限配置
（角色→可见菜单列表→仪表盘数据范围）
      ↓
初始化侧边栏菜单 + 跳转仪表盘
```

### 1.2 仪表盘数据流

```
仪表盘页面加载
      ↓
根据当前管理员角色，确定数据请求范围
      ↓
并行请求多个数据接口：
  ├── 指标卡片数据（总用户数、DAU、付费用户、营收、在线人数、待退款数）
  ├── 趋势图表数据（用户增长、营收、日活，按时间范围）
  ├── 内容概览数据（文章按类目统计、课程课时统计、游戏对局统计）
  └── 最近操作日志（最近 10 条）
      ↓
各区域独立渲染（某个接口失败不影响其他区域）
      ↓
定时刷新：
  ├── 在线游戏人数：每 30 秒
  └── 其他数据：每 5 分钟
```

### 1.3 权限数据流

```
管理员登录成功
      ↓
获取角色信息（super_admin / content_ops / user_ops / game_ops）
      ↓
前端根据角色匹配权限配置表
      ↓
生成可见菜单列表 → 渲染侧边栏
生成数据请求范围 → 请求仪表盘数据
      ↓
API 请求携带 Token → 服务端二次校验角色权限
      ↓ 校验通过
返回对应数据范围的结果
      ↓ 校验失败
返回 403 → 前端 Toast 提示"无权执行此操作"
```

### 1.4 会话数据流

```
登录成功 → 写入 Token
      ↓
每次页面导航 / API 请求 → 检查 Token 有效性
      ↓
Token 即将过期（剩余 <5 分钟）
      ↓
显示续期提醒 → 用户点击"延续会话" → 刷新 Token
      ↓
Token 过期（超时未操作）
      ↓
清除本地 Token → 全屏遮罩提示 → 跳转登录页（携带 redirect 参数）
```

## 二、非功能需求

### 2.1 页面加载性能

| 指标 | 目标 | 说明 |
|------|------|------|
| **登录页首屏加载** | ≤ 1.5s | 登录页静态资源少，应极快加载 |
| **仪表盘首屏加载** | ≤ 2s | 从登录成功到仪表盘数据可见 |
| **仪表盘骨架屏出现** | ≤ 300ms | 页面结构先渲染，数据后填充 |
| **指标卡片数据返回** | ≤ 1s | 单个接口响应时间 |
| **趋势图表数据返回** | ≤ 2s | 含数据库聚合计算 |
| **图表渲染完成** | ≤ 500ms | 从数据到达到图表渲染完成 |
| **页面切换（导航）** | ≤ 500ms | 从点击菜单到新页面内容可见 |

### 2.2 图表渲染性能

| 指标 | 目标 |
|------|------|
| **折线图数据点** | 单图表最大支持 365 个数据点（90 天 × 4 数据系列） |
| **饼图扇区** | 最大 12 个扇区（对应 12 个文化类目） |
| **图表动画帧率** | ≥ 30fps |
| **时间范围切换** | 数据重新请求 + 渲染 ≤ 1s |

### 2.3 会话与安全

| 指标 | 目标 | 说明 |
|------|------|------|
| **会话超时** | 2 小时无操作 / 7 天（记住我） | 根据用户选择 |
| **Token 传输** | HTTPS only | 禁止 HTTP 明文传输 |
| **Token 存储** | HttpOnly Cookie 或 localStorage（记住我） | 防 XSS 窃取 |
| **登录锁定** | 5 次失败锁定 15 分钟 | 防暴力破解 |
| **并发会话** | 允许同一账号最多 2 个并发会话 | 超出时最早的会话被踢出，显示提示"您的账号在其他设备登录" |
| **敏感操作日志** | 100% 记录 | 退款、封禁、知语币调整、角色变更等操作完整审计 |
| **密码存储** | 单向哈希 + 盐值 | 禁止明文存储 |

### 2.4 可用性

| 指标 | 目标 |
|------|------|
| **管理后台可用性 SLA** | 99.9%（全年宕机 < 8.7 小时） |
| **数据看板实时性** | 指标数据延迟 ≤ 5 分钟（在线人数 ≤ 30 秒） |
| **浏览器支持** | Chrome 90+、Firefox 90+、Edge 90+、Safari 15+（最新两个大版本） |
| **最小分辨率** | 1280 × 720（推荐 1920 × 1080） |

### 2.5 无障碍基线

| 规则 | 要求 |
|------|------|
| **色彩对比度** | 正文 ≥ 4.5:1，大标题 ≥ 3:1 |
| **键盘导航** | 所有核心操作可通过 Tab / Enter / Escape 完成 |
| **焦点指示** | 弥散光晕（glow），禁止浏览器默认 focus ring |
| **图标语义** | 纯装饰图标 `aria-hidden="true"`，功能图标有 `aria-label` |

## 三、验收标准汇总

### 3.1 登录页验收标准

| # | 验收项 | 验收标准 | 优先级 |
|---|--------|---------|--------|
| L-01 | 正常登录 | 输入正确邮箱密码 → 点击登录 → 2s 内跳转仪表盘 → Toast 显示"登录成功，欢迎回来，[姓名]" | P0 |
| L-02 | 邮箱格式校验 | 输入非法格式邮箱 → 失焦后即时显示"请输入有效的邮箱地址" | P0 |
| L-03 | 密码为空校验 | 密码为空时点击登录 → 密码框下方显示"请输入密码" | P0 |
| L-04 | 登录失败提示 | 输入错误密码 → Toast 显示"邮箱或密码错误，还剩 X 次尝试机会" | P0 |
| L-05 | 账号锁定 | 连续 5 次失败 → 按钮置灰 → 显示 15 分钟倒计时 → 倒计时结束后按钮恢复可用 | P0 |
| L-06 | 密码显示/隐藏 | 点击眼睛图标 → 密码在明文/密文间切换 → 图标相应变化 | P0 |
| L-07 | 记住我功能 | 勾选"记住我" → 登录成功 → 关闭浏览器 → 7 天内重新打开直接进入仪表盘 | P0 |
| L-08 | 会话超时 | 不勾选"记住我" → 登录后静置 2 小时 → 显示会话过期遮罩 → 点击重新登录跳转登录页 | P0 |
| L-09 | 超时前提醒 | 会话剩余 5 分钟时 → 右上角弹出提醒 → 点击"延续会话"后计时重置 | P1 |
| L-10 | 首次登录改密 | 使用临时密码登录 → 强制弹出修改密码弹窗 → 修改成功后才能进入后台 | P0 |
| L-11 | 主动退出 | 点击退出登录 → 确认弹窗 → 确认后跳转登录页 → Toast"已安全退出" | P0 |
| L-12 | 键盘提交 | 在密码输入框按 Enter → 触发登录（等同点击登录按钮） | P1 |
| L-13 | Loading 态防重复 | 点击登录后按钮变为 Loading + Disabled → 不可重复点击 | P0 |

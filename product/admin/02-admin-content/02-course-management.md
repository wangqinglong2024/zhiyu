## 一、课程总览页（Level 列表）

### 1.1 页面入口

- 侧边栏路径：**内容管理 > 课程管理**
- 页面标题：「课程管理」
- 页面说明文字（标题下方灰色小字）：「管理 12 个级别的完整课程内容，包括教学素材和练习题目」

### 1.2 Level 卡片网格

页面以**卡片网格**形式展示 12 个 Level，每行 4 张卡片，共 3 行。

**单张 Level 卡片信息**：

| 区域 | 内容 | 说明 |
|------|------|------|
| 顶部标签 | Level 编号 + 阶段标签 | 如「L1」+ 「启蒙段」标签（Amber 色）；阶段标签颜色：启蒙段 Amber / 基础段 Sky / 核心段 Rose / 高阶段 绿色 |
| 标题 | Level 名称 | 如「识字启蒙」 |
| 对标体系 | K-12 年级 + HSK 等级 | 如「一年级 / HSK 1 / A1」 |
| 统计数据 | Unit 数量 / Lesson 总数 / 题目总数 | 如「8 个 Unit · 40 个 Lesson · 186 道题」 |
| 定价标签 | 免费/付费 | L1-L3 显示绿色「免费」标签；L4-L12 显示「$6」标签 |
| 状态标签 | 课程状态 | 开发中（灰色）/ 已上线（绿色）/ 已下线（红色） |
| 操作区 | 「进入管理」按钮 | Sky 色文字按钮，点击进入该 Level 的 Unit 列表页 |

### 1.3 Level 卡片操作

- 点击卡片任意区域或「进入管理」按钮 → 进入该 Level 的 Unit 列表页
- 卡片 Hover 效果：阴影提升（Level 1 → Level 2），鼠标变为 pointer

### 1.4 Level 状态切换

- 每张卡片右上角显示状态切换开关（仅超级管理员可见）
- 切换操作需二次确认：
  - 上线：「确定将 Level X 上线吗？上线后用户端即可看到该级别课程。」
  - 下线：「确定将 Level X 下线吗？下线后用户端将无法访问该级别课程。已购买用户不受影响。」

## 二、Unit 列表页（某 Level 下）

### 2.1 页面入口

- 点击 Level 卡片进入
- 面包屑：**课程管理 > Level X「{Level 名称}」**

### 2.2 页面头部信息

| 信息项 | 内容 |
|--------|------|
| Level 编号 + 名称 | 如「Level 1 — 识字启蒙」 |
| 对标体系 | 如「一年级 / HSK 1 / A1」 |
| 统计信息 | 「共 X 个 Unit，Y 个 Lesson，Z 道题」 |
| 右上角按钮 | 「+ 新建 Unit」（Rose 实色按钮） |

### 2.3 Unit 列表

列表以**可拖拽排序的列表**形式展示。每个 Unit 为一行，行左侧有拖拽手柄图标（六点图标）。

**每行 Unit 信息**：

| 列名 | 宽度 | 内容 | 说明 |
|------|------|------|------|
| 拖拽手柄 | 32px | ⠿ 六点图标 | 鼠标悬停变为抓手光标，按住拖拽可排序 |
| Unit 编号 | 80px | 如「Unit 1」 | 编号随拖拽排序自动更新 |
| 主题名称 | 自适应 | 中文主题名称 | 如「你好中国」 |
| Lesson 数量 | 120px | 如「5 个 Lesson」 | — |
| 题目总数 | 120px | 如「23 道题」 | 统计该 Unit 下所有 Lesson 的题目数量 |
| 操作 | 200px | 操作按钮组 | — |

**操作列按钮**：

| 按钮标签 | 样式 | 点击行为 |
|---------|------|---------|
| 进入 | Sky 色文字按钮 | 进入该 Unit 的 Lesson 列表页 |
| 编辑 | Sky 色文字按钮 | 弹出编辑 Unit 弹窗（修改主题名称） |
| 删除 | 红色文字按钮 | 仅当 Unit 下无 Lesson 时可用；有 Lesson 时置灰 + hover 提示「请先删除该 Unit 下的所有 Lesson」；弹出确认弹窗「确定删除 Unit X 吗？」 |

### 2.4 拖拽排序规则

- 拖拽时被拖行呈半透明状态，目标位置显示蓝色插入线
- 松手后 Unit 编号自动重新排列（Unit 1, Unit 2, Unit 3...）
- 排序变更自动保存（无需手动点击保存）
- 保存成功后 Toast：「排序已更新」

### 2.5 新建 Unit 弹窗

点击「+ 新建 Unit」按钮后弹出 Modal 弹窗。

| 字段 | 类型 | 必填 | 校验规则 | 说明 |
|------|------|------|---------|------|
| Unit 主题名称（中文） | 文本输入框 | 是 | 限 30 字以内 | placeholder：「请输入 Unit 主题名称」 |
| Unit 主题名称（英文） | 文本输入框 | 是 | 限 60 字符以内 | placeholder：「Enter unit topic name」 |
| Unit 主题名称（越南语） | 文本输入框 | 是 | 限 60 字符以内 | placeholder：「Nhập tên chủ đề unit」 |

**弹窗按钮**：
- 「取消」灰色边框按钮 → 关闭弹窗
- 「确认创建」Rose 实色按钮 → 校验→创建成功→ Toast「Unit 已创建」→列表末尾新增一行→弹窗关闭

## 四、Lesson 编辑页

### 4.1 页面入口

- 面包屑：**课程管理 > Level X「{名称}」> Unit Y「{主题}」> Lesson Z「{标题}」**
- 页面布局：单栏布局，最大宽度 960px 居中，底部固定操作按钮栏

### 4.2 基本信息区

| 字段 | 类型 | 必填 | 校验规则 | 说明 |
|------|------|------|---------|------|
| Lesson 标题（中文） | 文本输入框 | 是 | 限 30 字以内 | 右侧字数统计「X/30」 |
| Lesson 标题（英文） | 文本输入框 | 是 | 限 80 字符以内 | — |
| Lesson 标题（越南语） | 文本输入框 | 是 | 限 80 字符以内 | — |
| Lesson 类型 | 下拉单选（只读） | 是 | — | 创建时已选定，编辑页仅展示不可修改（灰色标签显示） |
| 知识模块 | 下拉单选 | 是 | — | 可修改，选项同新建弹窗 |

### 4.3 讲解型内容编辑

当 Lesson 类型为「讲解型」时，基本信息区下方显示以下内容编辑区域。

#### 4.3.1 教学正文

多语言 Tab 切换（中文版 / 英文版 / 越南语版），交互规则同文章编辑页。

**中文版 Tab**：
- 富文本编辑器（支持格式同文章编辑器：标题、加粗、斜体、列表、引用、图片、音频）
- 拼音标注功能（同文章编辑器的拼音标注功能，详见 01-article-management.md §2.6）

**英文版 / 越南语版 Tab**：
- 富文本编辑器（不含拼音标注按钮）

#### 4.3.2 示例句列表

可动态添加多条示例句。每条示例句为一组**结构化输入**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 中文例句 | 文本输入框 | 是 | 如「你好，我叫小明。」 |
| 拼音 | 文本输入框 | 是 | 如「Nǐ hǎo, wǒ jiào Xiǎo Míng.」；支持自动生成拼音，可手动修正 |
| 英文翻译 | 文本输入框 | 是 | 如「Hello, my name is Xiao Ming.」 |
| 越南语翻译 | 文本输入框 | 是 | 如「Xin chào, tôi tên là Tiểu Minh.」 |

**操作**：
- 「+ 添加示例句」按钮，点击后在列表末尾新增一组空白输入
- 每组示例句右上角有「删除」按钮（× 图标），删除时不需二次确认
- 示例句支持拖拽排序（拖拽手柄在每组左侧）

#### 4.3.3 配套音频

| 字段 | 类型 | 必填 | 校验规则 | 说明 |
|------|------|------|---------|------|
| 课时朗读音频 | 文件上传 | 否 | 格式：MP3/WAV；大小：≤ 50MB | 标准发音朗读音频，管理员上传（非 TTS 生成） |

上传区域展示：
- 未上传时：虚线框 + 上传图标 + 文案「点击或拖拽上传音频文件（MP3/WAV，≤ 50MB）」
- 已上传：显示文件名 + 音频时长 + 播放按钮（可试听）+ 「重新上传」按钮 + 「删除」按钮

#### 4.3.4 配图

| 字段 | 类型 | 必填 | 校验规则 | 说明 |
|------|------|------|---------|------|
| 教学配图 | 图片上传 | 否 | 格式：JPG/PNG/WebP；大小：≤ 5MB；最多上传 5 张 | 教学辅助配图 |

上传区域展示：
- 多图上传，已上传图片以缩略图网格展示，可拖拽排序
- 每张图片右上角有 × 删除按钮

#### 4.3.5 重点词汇列表

可动态添加多条重点词汇。每条为一组**结构化输入**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 汉字 | 文本输入框 | 是 | 如「你好」 |
| 拼音 | 文本输入框 | 是 | 如「nǐ hǎo」；支持自动生成，可手动修正 |
| 英文释义 | 文本输入框 | 是 | 如「hello」 |
| 越南语释义 | 文本输入框 | 是 | 如「xin chào」 |
| 例句（中文） | 文本输入框 | 否 | 如「你好，很高兴认识你。」 |

**操作**：
- 「+ 添加词汇」按钮
- 每条词汇右上角「删除」按钮
- 支持拖拽排序

#### 4.3.6 语法点列表

可动态添加多条语法点。每条为一组**结构化输入**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 语法规则名称 | 文本输入框 | 是 | 如「"我是……"句型」 |
| 语法说明（中文） | 多行文本框 | 是 | 语法规则的中文描述 |
| 语法说明（英文） | 多行文本框 | 是 | 英文翻译 |
| 语法说明（越南语） | 多行文本框 | 是 | 越南语翻译 |
| 例句 | 示例句组（同 4.3.2） | 是 | 至少一条例句 |

**操作**：
- 「+ 添加语法点」按钮
- 每条语法点右上角「删除」按钮
- 支持拖拽排序

### 4.4 练习型 / 复习型 — 题目管理

当 Lesson 类型为「练习型」或「复习型」时，基本信息区下方显示**题目管理区域**。

#### 4.4.1 题目列表

以**可拖拽排序的列表**形式展示该 Lesson 下的所有题目。

**每行题目信息**：

| 列名 | 宽度 | 内容 | 说明 |
|------|------|------|------|
| 拖拽手柄 | 32px | ⠿ 六点图标 | 拖拽排序 |
| 题号 | 64px | 如「#1」 | 随拖拽自动更新 |

**操作列按钮**：

### 4.5 底部操作按钮栏

| 保存并返回 | Rose 实色按钮 | 保存当前所有编辑内容→ Toast「已保存」→返回 Lesson 列表页 |

## 五、题目编辑

### 5.1 题目编辑弹窗

### 5.2 通用字段（所有题型共有）

弹窗顶部为通用字段区域：

| 字段 | 类型 | 必填 | 校验规则 | 说明 |
|------|------|------|---------|------|
| 题型选择 | 下拉单选 | 是 | 必须选择 | 选项见 §5.3；新建时默认未选择；编辑时显示已选题型（可切换，切换时弹出确认「切换题型将清空已填内容，确定吗？」） |

| 难度标记 | 星级选择（1-5 星） | 是 | 必须选择 | 点击对应星号设置难度 |

### 5.4 单选题

通用字段之后，显示以下专有字段：

**选项列表**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 选项内容（中文） | 文本输入框 | 是 | 如「A. 你好」 |
| 选项内容（英文） | 文本输入框 | 是 | 英文翻译 |
| 选项内容（越南语） | 文本输入框 | 是 | 越南语翻译 |
| 是否正确答案 | 单选按钮（radio） | 是 | 在所有选项中选一个为正确答案 |

**交互**：
- 默认显示 4 个选项（A/B/C/D）
- 「+ 添加选项」按钮，最多 6 个选项
- 每个选项右上角有 × 删除按钮，最少保留 2 个选项
- 必须且只能选择一个正确答案，未选则校验报错：「请选择正确答案」

**答案解析**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 解析内容（中文） | 多行文本框 | 是 | 解释为什么正确答案是对的 |

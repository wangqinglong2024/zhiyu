## 一、金句列表页

### 1.1 页面入口

- 侧边栏路径：**内容管理 > 每日金句**
- 页面标题：「每日金句管理」
- 右上角主操作按钮：「+ 新建金句」（Rose 实色按钮，药丸形）

### 1.2 视图切换

页面标题右侧提供**视图切换**按钮组（两个图标按钮）：

| 视图 | 图标 | 说明 | 默认 |
|------|------|------|------|
| 列表视图 | ☰（列表图标） | 以表格形式展示所有金句 | 是 |
| 日历视图 | 📅（日历图标） | 以月历形式展示金句排期 | 否 |

### 1.3 筛选栏（列表视图 + 日历视图共用）

| 筛选项 | 类型 | 选项 | 默认值 |
|--------|------|------|--------|
| 状态 | 下拉单选 | 全部状态 / 草稿 / 已排期 / 已发布 / 已过期 | 全部状态 |
| 日期范围 | 日期范围选择器 | 起始日期 — 结束日期 | 空（不限） |
| 关键字搜索 | 文本输入框 | placeholder：「搜索金句内容或出处」 | 空 |

**交互规则**：
- 筛选项变更后即时触发刷新，300ms 防抖
- 「重置」文字按钮位于筛选栏末尾

### 1.4 列表视图 — 表格

| 列名 | 宽度 | 内容 | 排序 |
|------|------|------|------|
| 序号 | 64px | 自增 ID | 支持升序/降序 |
| 金句内容 | 自适应（最小 250px） | 中文金句内容摘要（单行截断，最多显示 30 字） | — |
| 来源/出处 | 150px | 作者或典籍名称 | — |
| 排期日期 | 120px | 格式：YYYY-MM-DD；未排期显示「—」 | 支持升序/降序 |
| 状态 | 100px | 状态标签 | — |
| 创建时间 | 140px | 格式：YYYY-MM-DD | 支持升序/降序 |
| 操作 | 200px | 操作按钮组 | — |

### 1.5 状态标签颜色

| 状态 | 标签颜色 | 触发条件 |
|------|---------|---------|
| 草稿 | 灰色底（`#a3a3a3` 文字，`#f5f5f5` 背景） | 新创建，未设置排期日期，或排期日期被清除 |
| 已排期 | Amber 底（`#d97706` 文字，`#fef3c7` 背景） | 已设置排期日期且排期日期为未来日期 |
| 已发布 | 绿色底（`#22c55e` 文字，`#f0fdf4` 背景） | 排期日已到，系统自动发布 |
| 已过期 | 红色底（`#ef4444` 文字，`#fef2f2` 背景） | 发布日期已过（排期日期的次日 00:00 UTC+7 后自动标记） |

### 1.6 操作列按钮

| 按钮标签 | 样式 | 触发条件 | 点击行为 |
|---------|------|---------|---------|
| 编辑 | Sky 色文字按钮 | 「草稿」「已排期」状态可用；「已发布」「已过期」状态仅可查看不可编辑 | 跳转至编辑页 |
| 取消排期 | Amber 色文字按钮 | 仅「已排期」状态可见 | 弹出确认弹窗「确定取消该金句的排期吗？取消后状态将变为草稿。」→确认→状态变为「草稿」→排期日期清除→ Toast「排期已取消」 |
| 删除 | 红色文字按钮 | 仅「草稿」状态可见 | 弹出确认弹窗「确定删除这条金句吗？」→确认→ Toast「金句已删除」 |

**规则**：
- 「已排期」状态的金句不可直接删除，必须先取消排期
- 「已发布」和「已过期」状态的金句不可删除和编辑（仅可查看）
- 操作按钮之间用分隔线 `|` 分隔

### 1.7 分页组件

- 位置：列表底部右侧
- 每页条数选择：20 / 50 / 100 条/页，默认 20 条
- 显示总条数：「共 XX 条金句」
- 页码导航 + 跳页输入框

## 二、日历视图

### 2.1 日历布局

- 以**月历网格**形式展示，每个格子代表一天
- 日历顶部：「◀ 上月 | 2026 年 4 月 | 下月 ▶」导航
- 今天的日期格子有蓝色边框高亮

### 2.2 日期格子内容

| 格子状态 | 展示内容 | 样式 |
|---------|---------|------|
| 已有排期金句 | 日期数字 + 金句内容摘要（最多显示 15 字，截断） + 状态小标签（已排期/已发布/已过期） | 白色背景 |
| 无排期金句（未来日期） | 日期数字 + 红色底纹 + 文案「未排期」 | **红色浅底纹**（`#fef2f2`），醒目提醒 |
| 无排期金句（过去日期） | 日期数字 + 灰色底纹 | 灰色浅底纹 |
| 今天无排期 | 日期数字 + 红色底纹 + 蓝色边框 + 文案「今日未排期 ⚠️」 | 红色底纹 + 蓝色边框 |

### 2.3 日历交互

- **点击已排期格子**：弹出浮层卡片，显示该日金句的完整信息（金句内容、出处、状态），卡片底部有「编辑」和「取消排期」按钮
- **点击未排期格子（未来日期）**：直接跳转至新建金句页面，排期日期自动填充为点击的日期
- **点击过去日期格子**：无操作（过去的日期不可新建排期）

### 2.4 日历统计信息

日历下方显示当月统计：

| 统计项 | 说明 |
|--------|------|
| 本月已排期天数 | 如「已排期 22 天」 |
| 本月未排期天数 | 如「未排期 8 天」（红色文字） |
| 本月已发布天数 | 如「已发布 15 天」 |

## 三、新建/编辑金句

### 3.1 页面入口

- 新建：金句列表页右上角「+ 新建金句」按钮，或日历视图点击未排期的未来日期
- 编辑：金句列表页操作列「编辑」按钮
- 面包屑：内容管理 > 每日金句 > 新建金句 / 编辑金句「{金句内容前 10 字}…」

### 3.2 页面布局

单栏布局，最大宽度 720px 居中，底部固定操作按钮栏。

### 3.3 编辑字段

| 字段 | 类型 | 必填 | 校验规则 | 说明 |
|------|------|------|---------|------|
| 金句内容（中文） | 多行文本框 | 是 | 限 100 字以内 | 右上角显示实时字数统计「X/100」；placeholder：「请输入中文金句」 |
| 拼音标注 | 多行文本框 | 是 | — | 支持自动生成（点击「自动生成」按钮，根据中文内容自动填充拼音），可手动修正；支持声调符号和数字声调输入 |
| 英文翻译 | 多行文本框 | 是 | 限 200 字符以内 | placeholder：「请输入英文翻译」 |
| 越南语翻译 | 多行文本框 | 是 | 限 200 字符以内 | placeholder：「请输入越南语翻译」 |
| 来源/出处 | 文本输入框 | 是 | 限 50 字以内 | placeholder：「请输入作者/典籍来源，如"李白"或"《论语》"」 |
| 背景图 | 图片上传 | 否 | 格式：JPG/PNG/WebP；大小：≤ 2MB；建议尺寸：1080×1920px（手机壁纸比例） | 用于用户分享时的金句图片生成；未上传时使用系统默认背景 |
| 排期日期 | 日期选择器 | 否 | 不可选择过去的日期（今天及以后可选）；不可选择已被其他金句排期的日期 | 日历形式选择器；已被排期的日期在日历中置灰不可选，hover 提示「该日期已有排期金句」 |

### 3.4 排期冲突处理

- **冲突检测**：选择排期日期时，系统实时检查该日期是否已被其他金句占用
- **已占用日期**：日期选择器中已占用的日期显示为灰色不可选，hover 提示「该日期已有排期金句："{金句摘要前 10 字}…"」
- **同一天只能排一条金句**：若尝试通过其他方式为已排期日期再排一条金句，保存时校验报错：「{日期} 已有排期金句，请选择其他日期」
- **修改排期日期**：编辑已排期金句时，可修改排期日期到其他未被占用的日期

### 3.5 拼音自动生成

拼音标注字段右上方有一个「自动生成」按钮（Sky 色文字按钮）。

**交互流程**：
1. 运营人员在「金句内容（中文）」中输入中文内容
2. 点击「自动生成」按钮
3. 系统调用拼音转换服务，将中文内容转换为拼音，自动填充到「拼音标注」字段
4. 若拼音字段已有内容，弹出确认弹窗：「自动生成将覆盖已有的拼音标注，确定吗？」→确认→覆盖
5. 生成后 Toast：「拼音已自动生成，请检查多音字是否正确」
6. 运营人员可手动修正多音字

### 3.6 底部操作按钮栏

固定在页面底部，白色背景 + 上方阴影分隔线。

| 按钮标签 | 样式 | 位置 | 点击行为 |
|---------|------|------|---------|
| 取消 | 灰色边框按钮 | 左侧 | 若有未保存修改→弹出确认弹窗「当前页面有未保存的修改，确定离开吗？」→「继续编辑」/「放弃修改」；无修改→返回列表页 |
| 保存草稿 | Sky 色边框按钮 | 右侧 | 校验：金句内容（中文）为必填→保存为「草稿」状态（不设排期日期或清除排期日期）→ Toast「草稿已保存」→停留当前页 |
| 确认排期 | Rose 实色按钮 | 最右侧 | 校验全部必填字段（金句内容+拼音+英文+越南语+出处+排期日期）→任一校验失败→滚动到第一个错误字段并标红→ Toast「请完善必填信息」；排期日期未填→ Toast「请选择排期日期」；全部通过→弹出确认弹窗「确定将此金句排期到 {日期} 吗？到期后将自动发布。」→确认→保存并状态变为「已排期」→ Toast「金句已排期到 {日期}」→跳转回列表页 |

## 四、自动发布规则

### 4.1 自动发布机制

| 规则 | 说明 |
|------|------|
| 发布时间 | 排期日 **00:00 UTC+7**（越南时间零点）自动将状态从「已排期」变为「已发布」 |
| 发布方式 | 系统定时任务自动执行，无需人工操作 |
| 发布后展示 | 已发布的金句在应用端首页每日金句区域展示 |
| 过期时间 | 排期日的次日 **00:00 UTC+7** 自动将状态从「已发布」变为「已过期」 |

### 4.2 空白日处理规则

| 场景 | 处理方式 | 说明 |
|------|---------|------|
| 某天无排期金句 | **复用上一天的已发布金句** | 系统自动查找最近一天的已发布金句继续展示；状态保持「已发布」不变为「已过期」，直到有新金句发布 |
| 连续多天无排期 | 持续复用最后一条已发布金句 | 应用端用户看到的金句不会变空，但管理后台日历视图会标红未排期天数 |
| 系统首日无金句（无任何已发布历史） | 展示系统内置默认金句 | 默认金句内容：「学而时习之，不亦说乎。— 孔子《论语》」；该默认金句不在管理后台列表中展示，是系统兜底逻辑 |

### 4.3 空白日提醒

- **管理后台仪表盘提醒**：当未来 7 天内存在无排期日期时，仪表盘快捷操作区显示 Amber 色提醒卡片：「未来 7 天有 X 天未排期金句」，点击跳转至金句日历视图
- **日历视图标红**：未排期的未来日期格子以红色底纹高亮（详见 §2.2）

### 4.4 金句状态完整流转

```
新建金句 ──→ 草稿
                │
                ├── [保存草稿] ──→ 草稿（保持）
                │
                ├── [确认排期] ──→ 已排期
                │                    │
                │                    ├── [取消排期] ──→ 草稿
                │                    │
                │                    ├── [修改排期日期] ──→ 已排期（更新日期）
                │                    │
                │                    └── [排期日 00:00 UTC+7 到达] ──→ 已发布
                │                                                        │
                │                                                        └── [次日 00:00 UTC+7 且有新金句发布] ──→ 已过期
                │
                └── [删除] ──→ 永久删除（仅草稿状态可删）
```

**规则补充**：
- 「已发布」状态的金句只有在有新的金句发布时才变为「已过期」（即如果后续无新金句，该金句持续展示）
- 「已排期」状态的金句可编辑内容和修改排期日期，但不可删除
- 「已发布」和「已过期」的金句不可编辑和删除（历史记录保留）

## 五、状态矩阵

### 5.1 金句列表页（列表视图）

| 状态 | 表现 | 操作引导 |
|------|------|---------|
| **空状态 (Empty)** | 列表区域显示插画（灯泡图标）+ 文案「暂无金句，创建第一条每日金句吧」+ 「+ 新建金句」CTA 按钮 | 引导创建 |
| **加载中 (Loading)** | 表格区域显示 Skeleton 骨架屏 | 等待 |
| **首次加载 (First Load)** | 同加载中 | 等待 |
| **成功 (Success)** | 列表正常展示，操作成功后显示 Toast（2s 自动消失） | — |
| **错误 (Error)** | 列表区域显示错误插画 + 「数据加载失败」+ 「重新加载」按钮 | 重试 |
| **部分加载 (Partial)** | 分页切换时表格区域 Skeleton 加载 | 翻页 |
| **离线 (Offline)** | 顶部 Amber 色 Banner 提示网络断开；操作按钮置灰禁用 | 恢复网络 |

### 5.2 金句日历视图

| 状态 | 表现 | 操作引导 |
|------|------|---------|
| **空状态 (Empty)** | 日历网格正常显示但所有日期格子显示「未排期」红色底纹 + 底部统计「本月已排期 0 天」 | 引导创建 |
| **加载中 (Loading)** | 日历网格显示 Skeleton（每个格子内容区为占位灰块） | 等待 |
| **首次加载 (First Load)** | 同加载中 | 等待 |
| **成功 (Success)** | 日历正常展示已排期/已发布/未排期状态 | — |
| **错误 (Error)** | 日历区域显示错误文案 + 「重新加载」按钮 | 重试 |
| **部分加载 (Partial)** | 切换月份时日历内容区 Skeleton 加载，导航栏保持可操作 | 切换月份 |
| **离线 (Offline)** | 顶部 Amber 色 Banner；已加载的月份数据保持展示；点击格子操作不可用 | 恢复网络 |

### 5.3 金句编辑页

| 状态 | 表现 | 操作引导 |
|------|------|---------|
| **空状态 (Empty)** | 新建模式：所有字段为空；从日历点击进入时排期日期自动填充 | 开始填写 |
| **加载中 (Loading)** | 编辑模式：页面显示 Skeleton 骨架屏 | 等待 |
| **首次加载 (First Load)** | 同加载中 | 等待 |
| **成功 (Success)** | 保存草稿→ Toast「草稿已保存」停留当前页；确认排期→ Toast「金句已排期到 {日期}」跳转列表页 | — |
| **错误 (Error)** | 校验失败→错误字段红色边框 + 行内错误提示→滚动到第一个错误字段；保存失败→ Toast（红色）「保存失败，请重试」；排期冲突→排期日期字段标红 + 行内提示「该日期已有排期金句」 | 修正并重试 |
| **部分加载 (Partial)** | 背景图上传中→上传区域显示进度条 + 百分比 | 等待上传 |
| **离线 (Offline)** | 顶部 Amber 色 Banner；编辑器仍可编辑但「保存草稿」和「确认排期」按钮禁用 | 恢复网络 |

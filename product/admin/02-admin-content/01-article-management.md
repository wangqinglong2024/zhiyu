## 一、文章列表页

### 1.1 页面入口

- 侧边栏路径：**内容管理 > 文章管理**
- 页面标题：「文章管理」
- 右上角主操作按钮：「+ 新建文章」（Rose 实色按钮，药丸形）

### 1.2 筛选栏

筛选栏位于列表上方，横向排列，支持组合筛选。

| 筛选项 | 类型 | 选项 | 默认值 |
|--------|------|------|--------|
| 文章分类 | 下拉单选 | 全部分类 / 中国历史 / 中国美食 / 名胜风光 / 传统节日 / 艺术非遗 / 音乐戏曲 / 文学经典 / 成语典故 / 哲学思想 / 当代中国 / 趣味汉字 / 中国神话传说 | 全部分类 |
| 文章状态 | 下拉单选 | 全部状态 / 草稿 / 待审核 / 已上架 / 已下架 | 全部状态 |
| 内容来源 | 下拉单选 | 全部来源 / Dify 自动生成 / 人工创建 | 全部来源 |
| 关键字搜索 | 文本输入框 | placeholder：「搜索文章标题或正文关键字」 | 空 |
| 创建时间 | 日期范围选择器 | 起始日期 — 结束日期 | 空（不限） |

**交互规则**：
- 筛选项变更后即时触发列表刷新，无需点击"搜索"按钮
- 关键字搜索输入后 300ms 防抖触发
- 筛选条件可组合叠加
- 「重置」文字按钮位于筛选栏末尾，点击后清空所有筛选条件

### 1.3 列表表格

| 列名 | 宽度 | 内容说明 | 排序 |
|------|------|---------|------|
| 多选框 | 48px | 复选框，用于批量操作 | — |
| 序号 | 64px | 自增 ID | 支持升序/降序 |
| 标题 | 自适应（最小 200px） | 中文主标题，超长截断（单行省略号） | — |
| 分类 | 120px | 12 类目之一，标签样式展示 | — |
| 来源 | 100px | 「Dify」标签（Sky 色）或「人工」标签（Amber 色） | — |
| 状态 | 100px | 状态标签（颜色规则见下方） | — |
| 创建时间 | 160px | 格式：YYYY-MM-DD HH:mm | 支持升序/降序 |
| 更新时间 | 160px | 格式：YYYY-MM-DD HH:mm | 支持升序/降序 |
| 浏览量 | 80px | 用户端累计浏览量数字 | 支持升序/降序 |
| 操作 | 200px | 操作按钮组 | — |

### 1.4 状态标签颜色

| 状态 | 标签颜色 | 说明 |
|------|---------|------|
| 草稿 | 灰色底（`#a3a3a3` 文字，`#f5f5f5` 背景） | 新创建或 Dify 自动写入，尚未审核 |
| 待审核 | Amber 底（`#d97706` 文字，`#fef3c7` 背景） | 已提交审核，等待运营确认 |
| 已上架 | 绿色底（`#22c55e` 文字，`#f0fdf4` 背景） | 已发布，用户端可见 |
| 已下架 | 红色底（`#ef4444` 文字，`#fef2f2` 背景） | 已从用户端撤下 |

### 1.5 操作列按钮

| 按钮标签 | 样式 | 触发条件 | 点击行为 |
|---------|------|---------|---------|
| 编辑 | Sky 色文字按钮 | 所有状态可用 | 跳转至编辑页，加载当前文章数据 |
| 预览 | Sky 色文字按钮 | 所有状态可用 | 打开预览弹窗（手机端模拟） |
| 上架 | 绿色文字按钮 | 仅「草稿」「已下架」状态可见 | 弹出二次确认弹窗→确认后状态变为「已上架」→ Toast「文章已上架」 |
| 下架 | 红色文字按钮 | 仅「已上架」状态可见 | 弹出二次确认弹窗→确认后状态变为「已下架」→ Toast「文章已下架」 |
| 删除 | 红色文字按钮 | 仅「草稿」「已下架」状态可见 | 弹出二次确认弹窗（文案：「删除后不可恢复，确定删除这篇文章吗？」）→确认后删除→ Toast「文章已删除」 |

**规则**：
- 「已上架」状态的文章不可直接删除，必须先下架
- 操作按钮之间用分隔线 `|` 分隔

### 1.6 分页组件

- 位置：列表底部右侧
- 每页条数选择：20 / 50 / 100 条/页，默认 20 条
- 显示总条数：「共 XX 篇文章」
- 页码导航：上一页 / 页码 / 下一页
- 跳页输入框：「前往第 __ 页」

### 1.7 表头多选与批量操作

- 表头复选框：全选当前页 / 取消全选
- 勾选任意行后，列表上方出现批量操作栏

**批量操作栏**：

| 信息/按钮 | 说明 |
|----------|------|
| 「已选 X 篇」 | 显示当前勾选的文章数量 |
| 「批量上架」按钮 | 仅对草稿/已下架文章生效；弹出确认弹窗：「确定上架选中的 X 篇文章吗？」→确认→ Toast「X 篇文章已上架」 |
| 「批量下架」按钮 | 仅对已上架文章生效；弹出确认弹窗：「确定下架选中的 X 篇文章吗？」→确认→ Toast「X 篇文章已下架」 |
| 「批量删除」按钮 | 仅对草稿/已下架文章生效；弹出确认弹窗（红色警告图标）：「删除后不可恢复，确定删除选中的 X 篇文章吗？」→确认→ Toast「X 篇文章已删除」 |
| 「取消选择」按钮 | 清空所有勾选 |

**批量操作规则**：
- 若勾选的文章中包含不符合条件的文章（如批量上架时勾选了已上架的文章），弹窗提示：「选中文章中有 Y 篇不符合操作条件，将自动跳过，实际操作 Z 篇。确定继续吗？」
- 批量操作完成后自动取消所有勾选并刷新列表

## 二、新建/编辑文章页

### 2.1 页面入口

- 新建：文章列表页右上角「+ 新建文章」按钮
- 编辑：文章列表页操作列「编辑」按钮
- 面包屑：内容管理 > 文章管理 > 新建文章 / 编辑文章「{文章标题}」

### 2.2 页面布局

页面分为**左侧主编辑区**（占 70% 宽度）和**右侧设置区**（占 30% 宽度，固定定位随页面滚动），底部为操作按钮栏（固定在页面底部）。

### 2.3 左侧主编辑区 — 基本信息

| 字段 | 类型 | 必填 | 校验规则 | 说明 |
|------|------|------|---------|------|
| 文章分类 | 下拉单选 | 是 | 必须选择一个分类 | 12 个分类选项（同列表页筛选项，不含"全部分类"） |
| 封面图 | 图片上传 | 是 | 格式：JPG/PNG/WebP；大小：≤ 2MB；尺寸：建议 16:9 比例，最小 800×450px | 上传后显示缩略图预览，支持裁剪（裁剪框比例锁定 16:9）；点击已上传图片可重新上传 |
| 标签/关键词 | 标签输入 | 否 | 每个标签最多 10 个字；最多添加 10 个标签 | 输入后按 Enter 添加标签；标签以药丸形标签展示；每个标签右侧有 × 删除按钮 |

### 2.4 左侧主编辑区 — 多语言内容编辑

内容编辑区顶部为**语言 Tab 切换栏**，三个 Tab：

| Tab 标签 | 状态标记 | 说明 |
|---------|---------|------|
| 中文版 | 必填标记（红色星号） | 中文内容为必填，是内容的主版本 |
| 英文版 | 「待翻译」/「已填写」标记 | 英文翻译版，必填 |
| 越南语版 | 「待翻译」/「已填写」标记 | 越南语翻译版，必填 |

**每个语言 Tab 下的编辑字段**：

| 字段 | 类型 | 必填 | 校验规则 | 说明 |
|------|------|------|---------|------|
| 文章标题 | 文本输入框 | 是（中文版必填，英文/越南语版必填） | 中文标题限 50 字以内；英文/越南语标题限 120 字符以内 | 输入框右侧显示实时字数统计：「X/50」 |
| 文章正文 | 富文本编辑器 | 是（中文版必填，英文/越南语版必填） | 正文字数限 300-5000 字 | 所见即所得编辑器，详见下方编辑器规格 |

**语言 Tab 交互规则**：
- 切换 Tab 时自动保存当前 Tab 的编辑内容到内存（非持久化）
- Tab 标签旁显示填写进度：「待翻译」（灰色，未填写任何内容时）/「已填写」（绿色，标题和正文均有内容时）
- 中文版 Tab 旁固定显示红色星号，不显示进度标记

### 2.5 富文本编辑器规格

**工具栏按钮（从左到右排列）**：

| 按钮 | 图标 | 功能说明 |
|------|------|---------|
| 标题 | H▾ | 下拉菜单：H2 / H3 / H4（不提供 H1，H1 为文章标题） |
| 加粗 | **B** | 加粗选中文字 |
| 斜体 | *I* | 斜体选中文字 |
| 有序列表 | 1. | 有序列表 |
| 无序列表 | • | 无序列表 |
| 引用 | " | 引用块 |
| 分隔线 | — | 水平分隔线 |
| 拼音标注 | 拼 | **仅中文版 Tab 显示此按钮**，详见下方拼音标注功能 |
| 插入图片 | 🖼 | 上传图片（格式：JPG/PNG/WebP/GIF；大小：≤ 5MB）或输入图片 URL |
| 插入视频 | ▶ | 输入视频嵌入链接（YouTube/Bilibili） |
| 插入音频 | ♪ | 上传音频文件（格式：MP3/WAV；大小：≤ 20MB），或输入音频 URL |

### 2.6 拼音标注功能（核心特色）

拼音标注功能仅在**中文版 Tab** 下可用，用于在中文汉字上方添加 HTML `<ruby>` 标注，以便在应用端展示拼音。

**操作方式**：

1. **选中汉字**：运营人员在富文本编辑器中用鼠标选中一段中文文字（单字或多字均可）
2. **点击「拼」按钮**：工具栏中的拼音标注按钮激活
3. **弹出拼音输入浮层**：在选中文字正上方弹出一个输入浮层
   - 浮层标题：「拼音标注」
   - 显示已选中的汉字：如「中国历史」
   - 拼音输入框：每个汉字对应一个独立输入框，横向排列，输入框宽度跟随汉字宽度
   - 自动填充：系统自动根据选中汉字生成拼音建议（调用拼音转换服务），预填到输入框中
   - 运营人员可手动修改每个汉字的拼音（处理多音字）
   - 声调标记：拼音输入支持声调符号输入（如 zhōng guó lì shǐ），也支持数字声调输入（如 zhong1 guo2 li4 shi3），系统自动转换为声调符号
4. **确认标注**：点击浮层中的「确认」按钮
   - 编辑器中选中的文字变为带拼音标注的样式：汉字上方显示小号拼音
   - 生成的 HTML 结构：`<ruby>中<rp>(</rp><rt>zhōng</rt><rp>)</rp></ruby>`
5. **取消标注**：点击浮层中的「取消」按钮，关闭浮层，不做任何修改

**已标注内容的编辑**：
- 在编辑器中，已标注拼音的文字以**浅蓝色底纹**高亮显示，上方显示小号拼音
- 双击已标注的文字，重新弹出拼音输入浮层，可修改拼音
- 选中已标注的文字后按 Delete 键，弹出选择弹窗：「删除拼音标注」/「删除文字和标注」

**全文自动标注**：
- 工具栏「拼」按钮右侧有下拉箭头，点击展开菜单：
  - 「选中文字标注」：对选中文字标注拼音（默认行为）
  - 「全文自动标注」：对正文中所有中文汉字自动生成拼音标注，运营人员在标注完成后逐一检查多音字
- 全文自动标注完成后弹出 Toast：「已自动标注 X 个汉字的拼音，请检查多音字是否正确」

### 2.7 右侧设置区

| 字段 | 类型 | 必填 | 校验规则 | 说明 |
|------|------|------|---------|------|
| 关联课程 | 下拉单选 | 否 | — | 选项：无关联 / L1 启蒙之旅 / L2 基础读写 / ... / L12 母语精通；关联后文章底部展示课程推荐卡片 |
| 排序权重 | 数字输入框 | 否 | 正整数，范围 0-9999 | 默认值 0；值越大在分类列表中越靠前；相同权重按创建时间倒序 |
| SEO 描述 | 多行文本框 | 否 | 限 200 字符以内 | placeholder：「请输入文章的 SEO 描述，用于搜索引擎展示」 |

### 2.8 底部操作按钮栏

固定在页面底部，白色背景 + 上方阴影分隔线。

| 按钮标签 | 样式 | 位置 | 点击行为 |
|---------|------|------|---------|
| 取消 | 灰色边框按钮 | 左侧 | 若有未保存修改→弹出确认弹窗：「当前页面有未保存的修改，确定离开吗？」，选项「继续编辑」/「放弃修改」；无修改则直接返回列表页 |
| 保存草稿 | Sky 色边框按钮 | 右侧 | 校验必填字段（仅中文标题和分类为必填）→保存为「草稿」状态→ Toast「草稿已保存」→停留在当前页 |
| 上架发布 | Rose 实色按钮 | 最右侧 | 校验全部必填字段（三语言标题+正文+分类+封面图）→任一校验失败→滚动到第一个错误字段并标红→ Toast「请完善必填信息」；全部通过→弹出二次确认弹窗：「确定上架发布这篇文章吗？发布后用户端即可查看。」→确认→保存并状态变为「已上架」→ Toast「文章已上架发布」→跳转回列表页 |

### 2.9 内容状态流转

```
Dify 自动写入 ──→ 草稿
人工新建 ──────→ 草稿
                    │
                    ├── [保存草稿] ──→ 草稿（保持）
                    │
                    ├── [上架发布] ──→ 已上架
                    │                    │
                    │                    └── [下架] ──→ 已下架
                    │                                    │
                    │                                    ├── [重新上架] ──→ 已上架
                    │                                    │
                    │                                    └── [删除] ──→ 已删除（永久）
                    │
                    └── [删除] ──→ 已删除（永久）
```

**规则**：
- 「已上架」状态不可直接删除，必须先下架
- 编辑已上架的文章并保存后，状态保持「已上架」，修改立即生效
- Dify 自动写入的文章初始状态为「草稿」，来源标记为「Dify」

## 三、文章详情预览

### 3.1 预览入口

- 文章列表页操作列「预览」按钮
- 文章编辑页（任何时候均可预览当前编辑内容）

### 3.2 预览弹窗

- 弹窗类型：全屏 Modal
- 弹窗标题栏：「文章预览」+ 关闭按钮（右上角 ×）

### 3.3 预览区域布局

| 区域 | 说明 |
|------|------|
| 左侧控制面板（宽 240px） | 语言切换、外观模式切换 |
| 右侧手机预览框（居中） | 模拟手机屏幕的预览窗口，宽 375px，高 812px（iPhone 比例），圆角边框 + 阴影 |

### 3.4 左侧控制面板

| 控制项 | 类型 | 选项 | 默认值 |
|--------|------|------|--------|
| 预览语言 | 单选按钮组 | 中文版 / 英文版 / 越南语版 | 中文版 |
| 显示模式 | 单选按钮组 | 拼音+中文 / 纯中文 | 拼音+中文（仅中文版生效） |
| 外观模式 | 切换开关 | 浅色模式 / 深色模式 | 浅色模式 |

### 3.5 手机预览框内容

按照应用端实际排版呈现：

1. **封面图**：顶部全宽显示
2. **文章标题**：对应语言的标题
3. **分类标签 + 浏览量 + 发布时间**
4. **正文内容**：渲染富文本内容，拼音标注以 `<ruby>` 样式展示
5. **核心词汇表**（如有）
6. **关联课程推荐**（如有关联）：底部卡片，显示 Level 名称和「去学习」按钮

**交互**：
- 预览框内可正常滚动
- 切换语言/模式后预览实时更新
- 若英文版/越南语版未填写内容，预览区域显示空状态：「该语言版本尚未填写」

## 四、批量操作

（已在第一节 1.7 详细定义，此处为规则汇总）

### 4.1 批量操作规则汇总

| 操作 | 适用源状态 | 目标状态 | 确认方式 |
|------|----------|---------|---------|
| 批量上架 | 草稿、已下架 | 已上架 | 二次确认弹窗 |

## 五、状态矩阵

### 5.1 文章列表页

| 状态 | 表现 | 操作引导 |
|------|------|---------|
| **空状态 (Empty)** | 列表区域显示插画（文档图标）+ 文案「暂无文章，快去创建第一篇吧」+ 「+ 新建文章」CTA 按钮 | 引导创建 |
| **加载中 (Loading)** | 表格区域显示 Skeleton 骨架屏（行数与每页条数一致），保持表头和筛选栏可见 | 等待 |
| **首次加载 (First Load)** | 同加载中，全表骨架屏 | 等待 |
| **成功 (Success)** | 列表正常展示数据，操作成功后显示 Toast（绿色，2s 自动消失） | — |
| **错误 (Error)** | 列表区域显示错误插画 + 文案「数据加载失败，请检查网络后重试」+ 「重新加载」按钮 | 重试 |
| **部分加载 (Partial)** | 分页器显示总页数，切换页码时仅表格区域显示 Skeleton 加载 | 翻页 |
| **离线 (Offline)** | 页面顶部显示 Amber 色 Banner：「网络连接已断开，当前显示的可能不是最新数据」；已加载的数据保持展示但操作按钮全部置灰禁用 | 恢复网络 |

### 5.2 文章编辑页

| 状态 | 表现 | 操作引导 |
|------|------|---------|
| **空状态 (Empty)** | 新建模式：所有字段为空，分类默认未选择；富文本编辑器显示 placeholder：「请输入文章正文…」 | 开始填写 |
| **加载中 (Loading)** | 编辑模式：页面显示 Skeleton 骨架屏（左侧编辑区 + 右侧设置区均显示骨架） | 等待 |
| **首次加载 (First Load)** | 同加载中 | 等待 |
| **成功 (Success)** | 保存草稿→ Toast「草稿已保存」停留当前页；上架发布→ Toast「文章已上架发布」跳转列表页 | — |
| **错误 (Error)** | 校验失败→错误字段红色边框 + 行内错误提示文案→自动滚动到第一个错误字段；保存失败→ Toast（红色）「保存失败，请重试」 | 修正并重试 |
| **部分加载 (Partial)** | 图片/音频上传中→对应上传区域显示进度条 + 百分比；上传完成前「保存草稿」和「上架发布」按钮禁用 | 等待上传完成 |
| **离线 (Offline)** | 页面顶部显示 Amber 色 Banner：「网络连接已断开，请恢复网络后再保存」；「保存草稿」和「上架发布」按钮置灰禁用；编辑器仍可正常编辑 | 恢复网络后保存 |

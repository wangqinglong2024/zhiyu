# 管理端模块：用户管理

> 依赖全局文件：`product/GLOBAL.md`
> 涉及 API：`GET /admin/users`, `GET /admin/users/{id}`, `POST .../freeze`, `POST .../unfreeze`

---

## 一、路由

`/admin/users`

---

## 二、用户列表

### 表格字段

| 字段 | 说明 |
|------|------|
| 注册时间 | MM-DD HH:mm |
| 手机号 | 脱敏展示 |
| 累计消费 | 该用户所有付费订单金额总和 |
| 邀请人数 | 通过该用户邀请码注册并付费的人数 |
| 累计佣金 | 该用户历史获得佣金总额 |
| 状态 | 正常 / 冻结 |
| 操作 | 查看详情 / 冻结 / 解封 |

### 筛选项
- 注册日期范围
- 账号状态
- 搜索手机号

**分页：** 每页 20 条。

---

## 三、用户详情（弹窗）

- 基本信息：手机号、注册时间、微信 openid（是否绑定）
- 消费记录：该用户所有订单列表
- 佣金明细：每笔佣金的来源订单和状态
- 提现记录

---

## 四、账号冻结

### 冻结效果
- 被冻结用户无法登录
- 待结算佣金暂停结算
- 提现申请自动挂起

### 使用场景
发现刷单、薅羊毛行为（如短时间内大量退款 + 佣金套现）时使用。

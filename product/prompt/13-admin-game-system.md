# 提示词 13：管理后台 — 游戏管理与系统设置 PRD

---

## 你的角色

你是「知语 Zhiyu」的高级产品经理，正在为开发团队编写**开发级产品需求文档（PRD）**。你写的文档必须精确到每个按钮的标签文字、每个操作的触发条件和结果、每个页面的 7 种状态表现，让前端和后端开发人员拿到后**零疑问直接开发**。

---

## 本次任务

生成**管理后台游戏管理模块和系统设置模块**的详细 PRD。这是管理后台 PRD 的最后一部分。

覆盖范围：
1. **皮肤管理**（游戏皮肤 CRUD、上架/下架、定价）
2. **赛季管理**（赛季创建/配置/开启/结束、赛季奖励配置）
3. **排行榜管理**（查看/重置、异常数据处理）
4. **推送管理**（推送消息创建/发送/历史记录）
5. **多语言管理**（UI 文案的中英越三语管理）
6. **管理员管理**（管理员 CRUD、角色分配、密码重置）
7. **系统日志**（操作日志查看、审计追踪）

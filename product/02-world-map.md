## 一、世界地图概述

### 1.1 用户故事

> 作为学习者，我想在一张直观的世界地图上看到我的学习进程和接下来的目标，以便有明确的方向感和探索欲。

### 1.2 地图结构

世界地图名为「汉字大陆」（Lục địa Hán tự），包含 4 大区域，严格按照汉语学习顺序从左到右/从下到上展开：

| 区域编号 | 区域名（中文） | 区域名（越南语） | 主题 | 关卡数 | 解锁条件 |
|---------|-------------|---------------|------|--------|---------|
| 1 | 拼音群岛 | Quần đảo Pinyin | 语音基础（声母/韵母/声调） | 20 关 | 注册即解锁 |
| 2 | 汉字谷地 | Thung lũng Hán tự | 笔画/部首/独体字 | 15 关 | 拼音群岛通关 |
| 3 | 词汇平原 | Đồng bằng Từ vựng | HSK1 核心 150 词 | 15 关 | 汉字谷地通关 |
| 4 | 语法要塞 | Pháo đài Ngữ pháp | 基础句式/助词 | 10 关 | 词汇平原通关 |

**区域解锁规则**：
- 完成上一区域的 Boss 关卡（每个区域最后 1 关是 Boss 关），解锁下一区域
- 未解锁区域在地图上显示为灰色半透明 + 锁图标
- 点击未解锁区域弹出提示："Hoàn thành [上一区域名] để mở khóa!"（完成 XX 解锁！）

### 1.3 免费/会员内容划分

| 区域 | 免费关卡 | 会员关卡 |
|------|---------|---------|
| 拼音群岛（20关） | 全部 20 关免费 | — |
| 汉字谷地（15关） | 前 8 关免费 | 后 7 关需会员 |
| 词汇平原（15关） | 前 7 关免费 | 后 8 关需会员 |
| 语法要塞（10关） | 前 5 关免费 | 后 5 关需会员 |

- 免费关卡和会员关卡在地图上无视觉区别（用户不知道哪里是分界线）
- 当用户尝试进入会员关卡时，弹出会员引导弹窗（见 [会员与支付](07-membership-payment.md)）

## 二、地图视觉设计

### 2.1 整体视觉风格

- **俯视角手绘风插画地图**，类似《Kingdom Rush》/《旅行青蛙》的世界地图风格
- 每个区域有独特的地形和色调：
  - 拼音群岛：海洋蓝 + 绿色岛屿，热带风情
  - 汉字谷地：棕色峡谷 + 古朴建筑，水墨元素
  - 词汇平原：金黄稻田 + 集市小镇，越南风情
  - 语法要塞：灰色要塞 + 军事风格，城堡城墙
- 区域之间用道路/桥梁/河流连接
- 全地图为一张完整的长图，用户通过拖拽/滑动浏览

### 2.2 关卡节点设计

每个关卡在地图上表现为一个**圆形节点**，沿道路排列：

| 节点状态 | 视觉表现 |
|---------|---------|
| **已完成（3 星）** | 金色圆形 + ★★★ 标记 + 发光特效 |
| **已完成（2 星）** | 银色圆形 + ★★☆ 标记 |
| **已完成（1 星）** | 铜色圆形 + ★☆☆ 标记 |
| **当前可玩** | 白色圆形 + 呼吸脉冲动画（引导用户点击） |
| **未解锁** | 灰色圆形 + 小锁图标 |
| **Boss 关** | 比普通节点大 1.5 倍 + 皇冠图标 |

### 2.3 关卡节点信息

点击已解锁的关卡节点，弹出关卡信息卡片（从底部上滑的 Bottom Sheet）：

```
┌─────────────────────────────┐
│  Bài 1-3: Vần mẫu đơn      │
│  （第 1-3 关：单韵母）        │
│                             │
│  🎯 Mục tiêu（目标）:        │
│  学会 6 个单韵母 a o e i u ü │
│                             │
│  ⭐ Thành tích（成绩）:       │
│  ★★☆ 最佳得分 85 分          │
│  （已重复 1 次完美目标 = 3 星）│
│                             │
│  ┌───────────────────────┐  │
│  │   Bắt đầu（开始）     │  │ ← 主按钮
│  └───────────────────────┘  │
│   Chơi lại（重新玩）         │ ← 文字链接（已完成关卡可见）
└─────────────────────────────┘
```

### 2.4 小游戏入口

地图上在每个区域内有一个**小游戏岛**（特殊图标节点），点击后进入小游戏选择：

- 拼音群岛区域：声调狙击手入口
- 汉字谷地区域：部首大爆炸入口
- 词汇平原区域：拼音漂移入口

小游戏节点无需通关解锁，跟随区域解锁即可进入。

## 三、地图交互规范

### 3.1 H5 端（手机）

- **纵向滚动的全屏地图**：地图方向为自下而上（拼音群岛在底部，语法要塞在顶部）
- 用户通过上下滑动浏览整张地图
- 双指缩放支持（缩放范围 0.8x ~ 1.5x）
- 当前进度节点自动居中显示（打开地图时定位到当前可玩节点）
- 底部 Tab Bar 在地图页常驻
- 区域标题（如"拼音群岛"）作为分隔带固定在地图中

### 3.2 PC 端

- 地图占据中间主内容区域，两侧留空或显示辅助信息
- 鼠标拖拽移动地图 + 滚轮缩放
- 鼠标 hover 关卡节点时显示简要信息 tooltip
- 左侧可选侧边栏：显示 4 个区域的快速跳转列表
- 详见 [响应式布局](10-responsive-layout.md)

### 3.3 地图加载

- 地图背景静态图通过 CDN 加载（首次加载后缓存到 PWA Service Worker）
- 关卡节点数据从 API 获取（用户进度 + 关卡状态）
- 首次加载显示地图骨架屏（区域形状轮廓 + 节点圆点占位）
- 节点数据加载完成后动画渐入

## 四、过场动画触发点

在世界地图上，以下时机自动触发过场动画：

| 触发时机 | 动画编号 | 说明 |
|---------|---------|------|
| 首次进入拼音群岛 | #1 | "欢迎来到汉字大陆"世界观介绍 |
| 拼音群岛第 18 关完成后 | #2 | "声调的魔法"Boss 前剧情 |
| 首次进入汉字谷地 | #3 | "文字的秘密"汉字起源 |
| 汉字谷地第 13 关完成后 | #4 | "古老的封印"Boss 前剧情 |
| 首次进入词汇平原 | #5 | "阿明来到美食街"日常场景 |
| 词汇平原第 13 关完成后 | #6 | "集市的考验"Boss 前剧情 |
| 首次进入语法要塞 | #7 | "组合的力量"句型概念 |
| 语法要塞第 8 关完成后 | #8 | "最终对话"综合运用 |

- 过场动画自动播放，播放完成后返回地图
- 用户可随时点击「Bỏ qua」（跳过）跳过动画
- 每段动画仅首次触发时自动播放，后续可在关卡信息卡片中手动回看

## 五、地图上的信息展示

### 5.1 顶部状态栏

地图页面顶部固定显示：

```
┌─────────────────────────────┐
│ 🔥 15 │ ⭐ 42 │ 🪙 1,200 │ 💎 50 │
│ streak │ 星星  │  金币    │  钻石  │
└─────────────────────────────┘
```

| 数值 | 含义 | 来源 |
|------|------|------|
| 🔥 Streak | 连续打卡天数 | 每日完成 ≥1 关 +1，中断归零 |
| ⭐ 星星 | 累计获得星星总数 | 每关可获 1-3 星 |
| 🪙 金币 | 游戏内货币 | 完成关卡/成就/每日奖励获得 |
| 💎 钻石 | 高级货币 | 会员每月赠送 300 / 未来购买获得 |

### 5.2 每日任务提示

地图上方（状态栏下方）有一个可折叠的每日任务条：

```
┌─────────────────────────────────┐
│ 📋 Nhiệm vụ hôm nay（今日任务）  │
│ ☑ Hoàn thành 1 bài học（完成 1 关）│ ✓ 已完成
│ ☐ Ôn tập 10 từ（复习 10 个词）   │ → 点击跳转复习
│ ☐ Chơi 1 mini game（玩 1 局小游戏）│ → 点击跳转小游戏
│ 奖励：50 🪙 + 10 XP              │
└─────────────────────────────────┘
```

- 每日任务固定 3 个（完成 1 关 + 复习 10 词 + 玩 1 局小游戏）
- 全部完成领取金币奖励
- 第二天 0:00（用户本地时间）重置

## 六、验收标准

| 项目 | 标准 |
|------|------|
| 地图加载速度 | 首次 ≤ 3 秒（CDN 缓存后 ≤ 1 秒） |
| 关卡状态同步 | 完成关卡后返回地图，节点状态即时更新 |
| 进度定位 | 打开地图自动滚动到当前可玩节点 |
| 区域解锁动画 | 完成 Boss 关后播放区域解锁动画（雾气消散效果） |
| 地图缩放 | H5 双指缩放、PC 滚轮缩放均流畅无卡顿 |
| 过场动画触发 | 首次到达触发点自动播放，仅一次 |
| 免费/会员卡点 | 用户尝试进入会员关卡时弹出付费引导 |

## 七、用户故事与页面状态

### 7.1 核心用户故事

| # | 功能 | 用户故事 | P |
|---|------|---------|---|
| 1 | 地图浏览 | 作为学习者，我想看到清晰的学习路径和进度，以便知道我的位置和下一步方向 | P0 |
| 2 | 关卡进入 | 作为学习者，我想点击地图节点直接进入关卡，以便快速开始学习 | P0 |
| 3 | 区域解锁 | 作为学习者，我想看到 Boss 战胜利后新区域解锁的动画，以便产生探索的成就感 | P0 |
| 4 | 会员卡点 | 作为免费用户，当我点击付费关卡时能看到清晰的会员引导，以便决定是否升级 | P0 |

### 7.2 地图页面 7 状态处理

| 状态 | 处理说明 |
|------|----------|
| Empty | 新用户首次进入：显示完整地图，仅第一关解锁可点击，其余变灰 + 引导气泡“Bắt đầu từ đây!” |
| Loading | Skeleton 屏：地图区域用火山灰/海洋占位色块 + 节点圆圈脉冲动画 |
| First Load | 加载完成后自动滚动到当前可玩节点 + “点击开始”引导动画 |
| Success | 关卡完成后返回地图，节点状态即时更新 + 下一节点解锁动画 |
| Error | 地图加载失败：显示重试按钮 + “Không thể tải bản đồ”，后台自动重试 3 次 |
| Partial | 地图加载成功但进度同步失败：显示缓存进度 + “Đang đồng bộ”标记 |
| Offline | PWA 离线模式：地图可浏览但无法进入关卡，显示“Bạn đang ngoại tuyến”横幅 |

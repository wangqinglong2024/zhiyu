## 一、词库系统

### 1.1 用户故事

> 作为学习者，我想查看我学过的所有词汇，按不同方式浏览和搜索，以便复习和巩固。

### 1.2 词汇来源

词汇通过以下途径自动加入用户词库：
- 完成关卡时，该关新教的词汇自动添加
- 过场动画中点击高亮词后自动添加
- Boss 战中涉及的词汇自动添加

用户**不能**手动添加或删除词汇（MVP 版本简化）。

> **免费用户限制**：免费用户每日最多复习 20 个词汇。触达限制时提示"Nâng cấp Premium để ôn tập không giới hạn"（升级 Premium 无限复习）。会员用户无此限制。

## 二、SRS 间隔重复复习系统

### 2.1 用户故事

> 作为学习者，我想系统地复习已学词汇，根据我的记忆程度智能安排复习时间，以便长期记住。

### 2.2 SRS 算法设计

采用简化版 SM-2 算法（SuperMemo-2）：

**掌握度等级（5 级）**：

| 等级 | 掌握度 | 含义 | 进度条颜色 |
|------|--------|------|-----------|
| 1 | 0-20% | 完全不记得 | 红色 |
| 2 | 20-40% | 模糊印象 | 橙色 |
| 3 | 40-60% | 有时能想起 | 黄色 |
| 4 | 60-80% | 基本掌握 | 浅绿 |
| 5 | 80-100% | 完全掌握 | 绿色 |

**复习间隔（首次学习后）**：

| 复习次数 | 上次答对 | 上次答错 |
|---------|---------|---------|
| 第 1 次 | 4 小时后 | 1 小时后 |
| 第 2 次 | 1 天后 | 4 小时后 |
| 第 3 次 | 3 天后 | 1 天后 |
| 第 4 次 | 7 天后 | 3 天后 |
| 第 5 次 | 14 天后 | 7 天后 |
| 第 6 次 | 30 天后 | 14 天后 |
| 第 7 次+ | 上次间隔 × 2 | 上次间隔 |

**复习答题影响**：
- 答对：掌握度 +20%，间隔按上表递增
- 答错：掌握度 -10%（不低于 0%），间隔重置到上一级
- 掌握度达到 100%：进入"已掌握"状态，60 天后再次复习

### 2.3 复习入口与提醒

**入口**：
- 底部 Tab Bar「词库」Tab 上有红色角标（显示待复习词数）
- 词库页面底部固定的「立即复习」按钮
- 世界地图页面：如果有 ≥ 10 词待复习，顶部弹出提示条

**推送提醒**：
- 每日用户设定的提醒时间，推送复习提醒
- 提醒内容："Bạn có X từ cần ôn tập！"（你有 X 个词需要复习！）

### 2.4 复习界面

每次复习一组（10-20 个词，取决于待复习数量）：

**复习模式：闪卡翻转**

```
正面（先显示）：
┌─────────────────────────────┐
│  Ôn tập（复习）  5/15        │ ← 当前进度
│  ━━━━━━━░░░░░░░ 33%         │ ← 进度条
│                             │
│                             │
│          好                 │ ← 只显示汉字
│                             │
│                             │
│                             │
│  ┌──────────┐ ┌──────────┐ │
│  │Không nhớ │ │  Nhớ rồi │ │ ← "不记得" / "记得了"
│  │（不记得） │ │（记得了） │ │
│  └──────────┘ └──────────┘ │
└─────────────────────────────┘

点击"不记得"或"记得了"后 → 翻转到背面：
┌─────────────────────────────┐
│  Ôn tập（复习）  5/15        │
│  ━━━━━━━░░░░░░░ 33%         │
│                             │
│          好                 │
│          hǎo                │ ← 显示拼音
│          🔊 [播放发音]        │
│                             │
│  tốt, hay, giỏi            │ ← 越南语释义
│  Ví dụ: 你好！Xin chào!     │ ← 例句
│                             │
│  [    Tiếp theo（下一个）  ]  │
└─────────────────────────────┘
```

**答对/答错的判定**：
- 用户自评："记得了"= 答对，"不记得"= 答错
- MVP 不做客观题型复习，依赖用户诚实自评
- 后续版本可增加选择题/听写题等客观题型

### 2.5 复习结算

```
┌─────────────────────────────┐
│  Hoàn thành ôn tập!          │

│  +10 XP                     │
│                             │
│  [   Ôn lại từ sai（再练错词）]│ ← 针对答错的词再复习一轮
│  [   Quay lại（返回）       ] │
└─────────────────────────────┘
```

## 三、掌握度与进度数据

### 3.1 用户学习数据

系统需要追踪每个词汇的以下数据：
- 首次学习时间
- 总复习次数
- 最近一次复习时间
- 下次计划复习时间
- 连续答对次数
- 当前掌握度百分比

### 3.2 词库统计（个人主页可见）

- 已学词汇总数
- 完全掌握词汇数（≥ 80%）
- 今日复习词汇数
- 平均掌握度

## 四、验收标准

| 项目 | 标准 |
|------|------|
| 词汇添加 | 关卡完成后新词 ≤ 1 秒同步到词库 |
| 搜索 | 支持汉字/拼音/越南语模糊搜索，≤ 200ms 返回结果 |
| SRS 调度 | 算法正确计算下次复习时间 |
| 复习提醒 | 待复习词数 ≥ 1 时角标显示 |
| 闪卡翻转 | 翻转动画流畅（300ms 3D 翻转） |
| 数据持久 | 掌握度数据同步到服务端，换设备不丢失 |
| 离线复习 | 会员用户已缓存的词汇可离线复习 |

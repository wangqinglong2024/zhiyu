## 一、过场动画概述

### 1.1 实现形式

**视觉小说风格（Visual Novel）**：
- 背景图 + 角色立绘 + 对话框 + 文字逐字显示 + 配音
- 类似《逆转裁判》《Doki Doki Literature Club》的对话演出风格
- 不需要帧动画/骨骼动画，成本可控

### 1.2 角色设定

| 角色 | 身份 | 形象 | 出场动画 |
|------|------|------|---------|
| **阿明（Minh）** | 越南青年主角 | 20 岁左右，穿休闲服，表情自信阳光 | 全程出场 |
| **小龙（Xiǎo Lóng）** | 可爱龙精灵引导角色 | Q版蓝色中国龙，大眼睛，漂浮在空中，友善可爱 | 教学引导时出场 |
| **声调守卫** | Boss 1 | 穿铠甲的武士，手持写着声调符号的武器 | 拼音群岛 Boss 前 |
| **美丽老师（Měi Lì）** | 汉字谷地 NPC | 古风女教师形象，手持毛笔，温柔有智慧 | 汉字谷地章节 |
| **汉字封印师** | Boss 2 | 古老神秘的封印师，披赫蒙袍，手持发光的文字卷轴 | 汉字谷地 Boss 前 |
| **集市掌柜** | Boss 3 | 越南老板形象，围裙+算盘 | 词汇平原章节 |
| **语法将军** | Boss 4 | 军装将军形象，架势威武 | 语法要塞章节 |

每个角色需要 2-3 个表情变体（普通/高兴/严肃）。

## 二、动画交互规范

### 2.1 画面布局（横屏 16:9）

> 过场动画属于游戏模式，H5 和 PC 端统一采用横屏 16:9 全屏展示。进入动画时 H5 端自动切换到横屏，退出后恢复竖屏。详见 [响应式布局](10-responsive-layout.md)。

```
┌────────────────────────────────────────────────┐
│                                        Bỏ qua │ ← 右上角跳过按钮
│                                                │
│  ┌──────┐                                      │
│  │角色   │     场景背景图                        │ ← 横屏 16:9 背景（AI 生成）
│  │立绘   │     （横屏 1920×1080）                │
│  └──────┘                                      │
│                                                │
│ ┌──────────────────────────────────────────────┐│
│ │ 小龙：                                        ││ ← 对话框（底部 1/3）
│ │ "Chào mừng đến với Lục địa Hán tự!"          ││    说话角色名 + 对话内容
│ │                                               ││
│ │ 中文：欢迎来到汉字大陆！                        ││ ← 三行字幕
│ │ Pīnyīn: Huānyíng lái dào Hànzì Dàlù!        ││
│ │ Tiếng Việt: Chào mừng đến Lục địa Hán tự!    ││
│ │                                    ▶ 继续    ││ ← 点击/自动推进
│ └──────────────────────────────────────────────┘│
└────────────────────────────────────────────────┘
```

### 2.2 交互细节

**文字显示**：
- 逐字显示效果（打字机效果），速度：每字 50ms
- 点击屏幕任意位置 → 立即显示全部文字
- 全部文字显示后，自动等待 2 秒或用户点击推进到下一段

**三行字幕规则**：
- 第一行：中文（汉字）— 加粗，字号最大
- 第二行：拼音 — 浅灰色，字号中等
- 第三行：越南语翻译 — 正常粗细，字号中等
- 如果 UI 语言设为英语，第三行改为英语翻译
- 如果 UI 语言设为印尼语，第三行改为印尼语翻译
- 如果 UI 语言设为中文，第三行隐藏（因为第一行已是中文）

**配音**：
- 中文对话行由 Azure Neural TTS 生成中文语音
- 配音与文字同步播放
- 用户可随时点击暂停/继续
- 右上角音量按钮可静音

**角色立绘动效**：
- 角色出场：从画面外侧滑入（300ms ease-out）
- 说话时：立绘微弱呼吸动效（上下浮动 2px，周期 3s）
- 表情切换：交叉淡入淡出（200ms）
- 退场：向画面外侧滑出

**场景切换**：
- 背景图切换使用交叉淡入淡出（500ms）

**关键词高亮**：
- 对话中的新词汇加粗 + 金色高亮
- 点击高亮词弹出词义卡片（汉字 + 拼音 + 释义 + 发音按钮）
- 词义卡片关闭后继续对话
- 高亮词自动加入用户词库

**跳过**：
- 右上角始终显示「Bỏ qua」（跳过）按钮
- 点击跳过二次确认："Bỏ qua sẽ không tự động phát lại, nhưng bạn có thể xem lại trên bản đồ. Bạn chắc chứ?"（跳过后不会自动播放，但可以在地图上手动回看。确定吗？）
- 确认跳过 → 直接进入关卡

## 三、8 段过场动画内容

### 动画 #1：欢迎来到汉字大陆

**位置**：首次进入拼音群岛
**时长**：约 40 秒（6-8 段对话）
**背景**：海洋+小岛远景（拼音群岛概览）
**角色**：阿明 + 小龙

**对话**：
```
[场景：阿明站在海边，远处是拼音群岛]

小龙（出场动画，从天而降）：
  "Chào Minh! Chào mừng đến với Lục địa Hán tự!"
  中文：你好阿明！欢迎来到汉字大陆！
  Nǐ hǎo Ā Míng! Huānyíng lái dào Hànzì Dàlù!

阿明（好奇表情）：
  "Đây là đâu vậy? Tôi muốn học tiếng Trung..."
  中文：这是哪里？我想学中文...
  Zhè shì nǎlǐ? Wǒ xiǎng xué Zhōngwén...

小龙：
  "Đây là nơi bạn sẽ học tiếng Trung qua phiêu lưu!"
  中文：这里是你通过冒险学中文的地方！

小龙：
  "Hành trình bắt đầu từ Quần đảo Pinyin — nơi bạn sẽ học cách phát âm."
  中文：旅途从**拼音**群岛开始——你将在这里学习发音。

阿明（自信表情）：
  "Được! Bắt đầu thôi!"
  中文：好！开始吧！
  Hǎo! Kāishǐ ba!

[阿明向拼音群岛方向跑去，渐隐]
```

### 动画 #2：声调的魔法

**位置**：拼音群岛第 18 关完成后（Boss 前）
**时长**：约 30 秒
**背景**：声调山脚，云雾缭绕
**角色**：阿明 + 小龙 + 声调守卫（远景剪影）

**对话**：
```
小龙：
  "Cẩn thận! Phía trước là Núi Thanh điệu..."
  中文：小心！前面是**声调**山...

小龙：
  "Thanh điệu rất quan trọng! Cùng một chữ, thanh điệu khác = nghĩa khác!"
  中文：声调非常重要！同一个字，声调不同=意思不同！

小龙（展示例子，画面叠加文字特效）：
  "mā 妈 = mẹ（妈妈）"
  "má 麻 = tê（麻）"
  "mǎ 马 = ngựa（马）"
  "mà 骂 = mắng（骂）"

阿明（惊讶表情）：
  "4 thanh điệu thay đổi nghĩa hoàn toàn!"
  中文：四个声调完全改变意思！

小龙：
  "Đúng vậy! Hãy vượt qua Người gác Thanh điệu để chứng minh bạn đã nắm vững!"
  中文：没错！通过声调守卫的考验来证明你已经掌握了！

[声调守卫剪影在山顶出现]
```

### 动画 #3-#8（简述）

| # | 标题 | 核心内容 | 关键对话要点 |
|---|------|---------|------------|
| 3 | 文字的秘密 | 汉字起源介绍：象形文字→现代汉字 | 展示"日月山水"的象形演变动画。小龙：汉字是画出来的文字！美丽老师出场讲解 |
| 4 | 古老的封印 | 汉字封印师挑战 | 阿明博物馆场景，汉字封印师现身，需要写出正确汉字才能解除封印 |

## 四、过场动画回看

### 4.1 回看入口

- 在关卡信息卡片（地图上点击已完成关卡节点）中，有「Xem lại phim」（回看动画）按钮
- 仅对已触发过的动画可见

### 4.2 回看界面

- 与首次播放完全相同
- 右上角有「跳过」按钮可随时退出
- 不再触发词汇添加（已添加的不重复）

## 五、验收标准

| 项目 | 标准 |
|------|------|
| 加载速度 | 背景图+立绘预加载，启动播放 ≤ 1 秒 |
| 文字同步 | 逐字显示与配音同步 |
| 点击响应 | 点击加速/推进 ≤ 100ms |
| 跳过 | 跳过后直接进入关卡/返回地图 |
| 词汇标注 | 高亮词点击弹出词义卡片 |
| 多语言 | 根据 UI 语言（越南语/中文/英语/印尼语）自动切换第三行字幕 |
| 配音 | Azure TTS 中文发音自然、停顿合理 |

## 七、用户故事与边界条件

### 7.1 核心用户故事

| # | 功能 | 用户故事 | P |
|---|------|---------|---|
| 1 | 过场动画 | 作为学习者，我想通过故事动画了解学习背景，以便产生情感连接而不是干巴巴地做题 | P0 |
| 2 | 跳过功能 | 作为老用户，我想跳过已看过的动画，以便立刻开始学习而不浪费时间 | P0 |
| 3 | 词汇标注 | 作为学习者，我想点击动画中的中文新词查看释义，以便在故事中自然学习新词 | P0 |

### 7.2 边界条件

| 场景 | 处理方式 |
|------|----------|
| TTS 音频加载失败 | 显示字幕但无声音，右下角显示“🔇 音频加载失败” + 点击重试 |
| 背景图加载失败 | 显示纯色渐变占位背景（Sky → Rose 渐变），不阻塞剩余内容 |
| 快速点击跳过 | 非首次观看的动画可直接跳过；首次观看至少播放 3 秒后才显示跳过按钮 |
| 用户在动画中切后台 | 暂停播放，回到前台从上次位置继续 |

## 一、产品全景

### 1.1 产品定义

**知语 Zhiyu** 是面向东南亚市场的中文学习平台，包含两个产品端：

| 产品端 | 形态 | 用户 | 核心职责 |
|--------|------|------|---------|
| **应用端（Apps）** | PWA Web（首发）→ Android/iOS（后续） | 最终学习用户 | 文化浏览、课程学习、游戏练习、个人中心 |
| **管理后台（Admin）** | Web 后台 | 运营人员 | 内容管理、用户管理、订单管理、数据看板 |

### 1.2 核心运营流程

```
运营人员每日流程：
  ① 爬取开源数据 → Dify 工作流总结生成多语言素材 → 写入 Supabase 数据库
  ② 管理后台查看新生成的素材 → 编辑调整 → 确认无误后手动发布
  ③ 发布后用户端可见（类似商品上架）
  ④ 随时可在后台上架/下架任何内容
```

### 1.3 数据流全景

```
Dify 工作流（内容生产工具）
       ↓ 生成后写入
Supabase 数据库（唯一数据主存储）
       ↓                    ↓
  管理后台（运营端）      应用端（用户端）
  - 内容审核/编辑        - 内容展示（已发布）
  - 上架/下架            - 课程学习
  - 用户管理             - 游戏练习
  - 订单管理             - 个人中心
  - 数据统计
```

## 四、多语言体系（贯穿全局）

### 4.1 UI 语言

- 前期支持：**汉语、英语、越南语** 三种切换
- 后期扩展：印尼语、泰语、马来语
- 管理后台：仅中文 + 英文

### 4.2 内容语言

用户可配置：

| 配置项 | 选项 | 默认值 |
|--------|------|--------|
| 学习语言模式 | 拼音+中文 / 纯中文 | 拼音+中文 |
| 解释语言开关 | 开启/关闭 | 开启 |
| 解释语言 | 跟随系统 UI 语言 | 越南语 |

### 4.3 内容展示规则

```
Zuótiān qīngchén, wǒ zuò zài yángtái shàng kàn rìchū.    ← 拼音（选择"拼音+中文"时显示）
昨天清晨，我坐在阳台上看日出。                                ← 中文（始终显示）
Sáng qua, tôi ngồi trên ban công ngắm bình minh.           ← 解释语言（开启时显示）
```

**特殊规则**：
- 若用户系统语言为中文 → 禁止开启解释语言
- 所有内容在 Supabase 中存储多语言字段（pinyin, zh, en, vi），前端按用户配置渲染

## 五、核心业务规则

### 5.1 课程购买

| 课程等级 | 定价（USD）| 有效期 |
|---------|----------|--------|
| **L1-L3** | **免费** | 永久 |
| **L4-L12**（每级） | **$6 / 级** | **3 年** |
| **L4-L12 全部**（9 级） | **$54** | **3 年** |

- 支付通道：Paddle MoR（Web 端）；iOS 走 Apple IAP（$7.99/级）
- 到期前 30/7/1 天推送提醒，续购享 20% 折扣

### 5.2 知语币体系

| 规则 | 说明 |
|------|------|
| 比价 | 1 币 = $0.01（固定） |
| 用途 | 兑换课程 / 皮肤 / 特效 |
| 限制 | 不可提现、不可转让、不可交易、不可充值 |
| 上限 | 单用户 100,000 币 |

**来源**：

| 来源 | 数量 | 条件 |
|------|------|------|
| 推荐付费 | 付费金额的 20%（推荐人和被推荐人各得） | 30 天无退款 |
| 每日签到 | 1-10 币（1=40%, 2=20%, 3-4各8%, 5-6各6%, 7-8各4%, 9-10各2%）| 付费用户双倍 |
| 游戏连胜 | 1 币 | PK 5 连胜 |
| 新手引导 | 100 币 | 完成入学测试 |
| 节日活动 | 不定 | 运营配置 |

**负数余额规则**：退款扣除知语币可至负数，提示用户："因您推荐的用户发起了课程退款，对应的推荐奖励已扣回，您的知语币余额为 XX 币。后续获得新奖励时将自动抵充。"

### 5.3 三级考核体系

| 层级 | 名称 | 触发时机 | 通过标准 |
|------|------|---------|---------|
| 第一级 | 课时小测验 | 每课时学完后 | 无门槛，答错加入 SRS |
| 第二级 | 单元阶段测评 | 每单元学完后 | 70 分 |
| 第三级 | 级别综合考核 | 每 Level 全部单元学完后 | 85 分，单项不低于 60 分 |

- 考核融合课程内容与对应 HSK 考点（不是独立的 HSK 专项）
- 综合考核不通过可无限重考，间隔 24 小时
- 通过获得电子证书，标注 HSK 等级和 CEFR 等级

### 5.4 游戏段位系统

- 12 款游戏共享统一段位：青铜→白银→黄金→铂金→钻石→星耀→王者
- 赢 +1 星，输 -1 星，最低青铜 III 1 星
- MVP 仅陌生人匹配 PK（不含好友系统）
- 赛季 3 个月，段位软重置
- 游戏皮肤/特效/赛季通行证 = 游戏侧变现（纯装饰，不影响平衡）

## 一、落地页（Landing Page）

**Hero 区域**：
- PlayLingo Logo + Slogan "Chơi để Học"（越南语："玩来学"）

- 主 CTA 按钮：「Bắt đầu miễn phí」（免费开始）— Rose 色实色药丸按钮
- 副 CTA：「Đã có tài khoản? Đăng nhập」（已有账号？登录）— Ghost 按钮
- 背景：游戏世界地图的模糊远景图（AI 生成）

**卖点展示区**（3 列卡片，H5 单列堆叠）：
| 卖点 | 图标 | 描述（越南语） |
|------|------|-------------|
| 🎮 Học như chơi game | 游戏手柄图标 | 60 关卡 + 3 mini game，不是背单词 |
| 💰 Rẻ hơn 4 lần Duolingo | 价格标签图标 | Chỉ $2.99/tháng, tiết kiệm hơn 77% |
| 🤝 Giới thiệu bạn, kiếm tiền | 握手图标 | 佣金 5%-20%，推荐越多赚越多 |

**游戏画面预览区**：
- 3-4 张游戏截图横向滚动（H5 左右滑动，PC 自动轮播）
- 截图内容：世界地图 / 声调狙击手 / 部首大爆炸 / 关卡界面

**价格对比区**：
- 表格对比 PlayLingo vs Duolingo vs HelloChinese
- 突出我们的价格优势和游戏特色

**底部 CTA**：
- 重复「免费开始」按钮
- Footer：语言切换（🇻🇳 Tiếng Việt / 🇬🇧 English）、服务条款、隐私政策链接

### 1.3 验收标准

- 首屏加载 ≤ 2 秒（CDN 加速）
- 主 CTA 在 H5 端位于首屏可见区域内
- 越南语为默认语言，可切换为英语
- 如果 URL 携带 `?ref=xxx` 参数，静默存储到 localStorage，注册时自动绑定

### 1.4 情绪目标

兴奋、好奇、低门槛

## 二、注册流程

### 2.1 用户故事

> 作为新用户，我想通过最少步骤完成注册，以便尽快开始体验游戏。

### 2.2 注册方式（3 种）

**方式一：Google 一键登录**
- 点击「Đăng nhập bằng Google」按钮
- 调起 Google OAuth 弹窗
- 授权后自动创建账号并登录
- 跳转到「选择学习目标」页

**方式二：Facebook 一键登录**
- 点击「Đăng nhập bằng Facebook」按钮
- 调起 Facebook OAuth 弹窗
- 授权后自动创建账号并登录
- 跳转到「选择学习目标」页

**方式三：邮箱注册**
- 输入邮箱地址 + 设置密码（≥ 8 位，含字母、数字和特殊字符）
- 点击「Đăng ký」（注册）
- 发送验证邮件（6 位数字验证码，有效期 10 分钟）
- 输入验证码 → 注册完成
- 跳转到「选择学习目标」页

### 2.4 推广链接绑定逻辑

- 用户通过 `playlingo.app/?ref=XXXXX` 访问时，将 ref 值存入 `localStorage`
- 注册时自动读取 ref 值提交到后端，绑定推荐关系
- 如果 ref 无效（用户不存在），静默忽略，不影响注册流程
- 绑定成功后清除 localStorage 中的 ref 值
- 推荐关系一旦绑定不可更改

### 2.5 验收标准

- Google/Facebook 一键注册：从点击到完成 ≤ 5 秒（不含第三方弹窗时间）
- 邮箱注册：验证码邮件 ≤ 30 秒送达
- 密码强度实时提示（弱/中/强，blur 后校验）
- 邮箱已注册时提示「此邮箱已注册，请直接登录」并提供登录链接
- 注册成功立即自动登录，无需再次输入账号密码

## 三、登录流程

### 3.1 登录页 UI

与注册页结构一致，但标题改为「Đăng nhập」（登录），邮箱区域增加「Quên mật khẩu?」（忘记密码？）链接。

### 3.2 忘记密码流程

1. 点击「忘记密码」→ 进入重置页
2. 输入注册邮箱 → 点击「发送重置链接」
3. 邮箱收到重置链接（有效期 1 小时）
4. 点击链接 → 输入新密码（≥ 8 位）→ 确认
5. 重置成功 → 自动登录并跳转到世界地图

### 3.3 登录状态

- 登录态通过 JWT Token 维持
- Token 有效期 7 天，到期自动刷新（refresh token 30 天）
- Refresh token 每次刷新时重置 30 天有效期（滑动窗口）
- 设备同时在线不限（Web 多端可同时登录）
- 连续 30 天无任何操作（无 API 请求）自动退出需重新登录

- 引导只出现一次，不重复触发

- 学习目标选择 → 角色创建 → 进入地图，整个流程 ≤ 60 秒
- 引导气泡不遮挡核心操作区域
- 引导结束后用户自然落在可操作的世界地图上

## 六、设置页

### 6.1 设置项列表

| 设置项 | 操作 | 说明 |
|--------|------|------|
| **头像/昵称** | 点击编辑 | 可修改头像选择和昵称 |
| **学习目标** | 点击修改 | 重新选择学习目标（工作/旅游/考试/兴趣） |
| **界面语言** | 下拉选择 | 🇻🇳 Tiếng Việt / 🇬🇧 English |
| **每日提醒** | 开关 + 时间选择 | 默认开启，默认 20:00，可选 06:00-23:00 |
| **音效** | 开关 | 默认开启 |
| **推送通知** | 开关 | 默认开启（仅 PWA 环境且用户授权时可用） |

| **账号安全** | 修改密码入口 | 仅邮箱注册用户可见 |

- 语言切换后页面立即刷新为对应语言

- 退出登录后清除 Token，跳转到落地页

## 八、用户故事与边界条件

### 8.1 核心用户故事

| # | 功能 | 用户故事 | P |
|---|------|---------|---|
| 1 | Google 登录 | 作为新用户，我想通过 Google 一键登录，以便无需填写表单就能开始学习 | P0 |
| 2 | 昵称设置 | 作为新用户，我想在首次登录时设置昵称，以便在排行榜和社交分享中展示我的身份 | P0 |
| 3 | 学习提醒 | 作为老用户，我想设置每日提醒时间，以便不会忘记学习而中断 Streak | P0 |
| 4 | 删除账号 | 作为用户，我想能彻底删除我的账号和数据，以便维护我的数据隐私权 | P0 |
| 5 | 语言切换 | 作为越南用户，我想在越南语和英语 UI 之间切换，以便用我熟悉的语言操作 | P0 |

### 8.2 边界条件与异常处理

| 场景 | 处理方式 |
|------|----------|
| Google 登录授权失败 | 显示“Đăng nhập thất bại, vui lòng thử lại” + 重试按钮 |
| 昵称重复 | 实时检查，输入时示“Đã được sử dụng” + 禁用提交按钮 |
| 昵称含违禁词 | 过滤禁止提交 + 提示“Đặt tên không hợp lệ” |
| Token 过期 | 自动 refresh；若 refresh 失败则跳转登录页，已有本地进度自动同步 |
| 同一 Google 账号多端登录 | 允许多端同时登录，最后一次操作为准（无强制踢下线） |
| 删除账号后重新用同一 Google 登录 | 视为全新用户，历史数据不可恢复 |
| 删除账号对推荐关系的影响 | 推荐关系解除；已结算佣金不追回；未结算佣金作废；被推荐人的后续续费不再产生佣金 |
| 删除账号数据保留 | 用户数据在删除后 30 天内彻底清除（法律要求的支付记录保留 5 年） |
| 网络断开时进入 App | 显示离线提示页，PWA 缓存的静态资源可浏览，无法操作 |

### 8.3 输入验证规则

| 字段 | 规则 |
|------|------|
| 昵称 | 2-20 字符，支持 Unicode（越南语/中文/英文/数字），禁止纯空格、特殊符号、违禁词 |
| 密码 | ≥ 8 字符，必须含字母、数字和至少 1 个特殊字符（!@#$%^&*等），禁止纯空格 |
| 头像 | 仅预设头像选择（MVP 不支持上传），12 个选项 |
| 提醒时间 | 00:00-23:59，整点/半点可选，默认 20:00 |
| DELETE 确认 | 必须精确输入 "DELETE"（区分大小写） |

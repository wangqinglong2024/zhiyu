## 一、小游戏系统概述

### 1.1 设计目标

小游戏是 PlayLingo 的核心差异化武器——它们不是"换皮练习题"，而是真正让人上瘾的游戏，学习发生在游戏过程中。

### 1.2 入口与访问

- **入口 1**：底部 Tab Bar 「Mini Game」Tab → 小游戏列表页
- **入口 2**：世界地图上各区域内的小游戏岛节点
- **解锁规则**：跟随地图区域解锁
  - 声调狙击手：注册即可玩（拼音群岛区域）
  - 部首大爆炸：汉字谷地解锁后可玩
  - 拼音漂移：词汇平原解锁后可玩

- 小游戏得分可获得金币（每局 5-15🪙）和 XP（每局 10-30）

## 二、声调狙击手（Tone Sniper）

### 2.1 核心创意

Beat Saber × 中文声调：把四声变成可视化的「声波弹」，沿正确声调走势方向滑动斩击。

### 2.2 游戏画面布局（横屏 16:9）

> 小游戏属于游戏模式，H5 和 PC 统一横屏 16:9 全屏显示。进入小游戏时 H5 端自动切换横屏，退出后恢复竖屏。详见 [响应式布局](10-responsive-layout.md)。

```
┌────────────────────────────────────────────────┐
│  Score: 420       Combo: x8  🔥      ❤️ ❤️ ❤️  │ ← 得分 + 连击数 + 生命值
│                                                │
│                   ┌──────┐                     │
│  🀄「妈」mā →    │ 准星  │    ← 🀄「马」mǎ    │ ← 声波弹从四面飞向准星
│                   │ 区域  │                     │
│      ↗            └──────┘         ↙           │
│                                                │
│  ── 声调滑动提示 ──────────────────────          │
│  一声: →  二声: ↗  三声: ↘↗  四声: ↘            │
└────────────────────────────────────────────────┘
```

### 2.3 核心玩法详细设计

**声波弹**：
- 从屏幕四面八方向准星区域飞来
- 每个声波弹上方显示汉字 + 拼音（含声调符号）
- 同时自动播放该字的发音音频
- 声波弹到达准星区域的滞留时间 = 玩家的操作窗口（约 1.5 秒→难度递增到 0.5 秒）

**滑动操作**：
| 声调 | 手势方向 | 视觉辅助线 |
|------|---------|-----------|
| 一声（阴平） | 水平横划 →（从左到右） | ── 平直线 |
| 二声（阳平） | 右上斜划 ↗ | ╱ 向上斜线 |
| 三声（上声） | V 形划 ↘↗ | ╲╱ 先降后升 |
| 四声（去声） | 右下斜划 ↘ | ╲ 向下斜线 |

- 系统检测滑动方向，与目标声调匹配（容差 ±30°）
- 三声需要检测 V 形路径（先下后上），可简化为：只检测最终是向上的趋势

**正确斩击反馈**：
- 声波弹爆裂为彩色粒子特效（颜色随声调不同：一声蓝、二声绿、三声黄、四声红）
- Combo 数 +1
- 得分 +10（基础分 × combo 倍率）
- 短促的切割音效

**连击特效**：
| Combo 数 | 特效 |
|---------|------|
| 5 combo | 屏幕边缘闪金光 |
| 10 combo | 背景变亮 + 准星区域扩大 20% |
| 20 combo | 全屏金色粒子爆炸 + 「Hoàn hảo!」（完美！） |

**失误反馈**：
- 滑动方向错误：声波弹撞碎准星 + combo 归零 + 屏幕红色闪烁 + 扣 1 心
- 声波弹飞出屏幕未操作：视为 Miss + combo 归零 + 扣 1 心

**难度递进**：
| 时间 | 速度 | 频率 | 特殊 |
|------|------|------|------|
| 0-15s | 慢 | 1.5s/个 | 仅一二声 |
| 15-30s | 中 | 1.0s/个 | 四声全开 |
| 30-60s | 快 | 0.7s/个 | 开始出现连续同声调 |
| 60s+ | 极快 | 0.4s/个 | 混乱模式 |

### 2.4 游戏模式

**生存模式**（MVP 默认）：
- 3 条命
- 错 3 次 = Game Over
- 目标：存活尽可能久 + 最高分

**限时模式**：
- 60 秒倒计时
- 无生命限制
- 目标：60 秒内最高分
- 排行榜竞赛

**每日挑战**：
- 固定词库 + 固定速度曲线
- 全服统一排名
- 每日 0:00 重置
- 奖励：前 100 名额外金币

### 2.5 结算画面

```
┌─────────────────────────────┐
│       Game Over!             │
│                             │
│    Điểm（得分）: 1,240       │
│    Max Combo: 15             │
│    Chính xác（准确率）: 82%   │
│    Thời gian（时间）: 45s     │
│                             │
│    🏆 Xếp hạng #42          │ ← 排名
│    🏅 Kỷ lục mới!（新纪录！）│ ← 如果破纪录
│                             │
│    +15 🪙  +20 XP           │
│                             │
│  [Chơi lại] [Chia sẻ] [Thoát]│
│  （再玩）  （分享）  （退出）   │
└─────────────────────────────┘
```

### 2.6 学习效果

- 每局 60s 可练习 40-80 次声调判断
- 肌肉记忆：手指滑动方向与声调走势绑定
- 高频重复：比传统选择题效率高 5 倍以上

## 三、部首大爆炸（Radical Blitz）

### 3.1 核心创意

水果忍者 × 汉字拆解：目标汉字的幽灵轮廓悬浮在上方，偏旁部首像陨石一样从天而降，斩中正确偏旁让它飞入正确位置完成组装。

### 3.2 游戏画面布局（横屏 16:9）

```
┌────────────────────────────────────────────────┐
│  Score: 680  连锁: x3                 ❤❤❤ ⏱60s│
│                                                │
│     ┌───────────────────┐                      │
│     │     好（hǎo）      │   释义:              │
│     │   ┌──┐  ┌──┐     │   tốt, hay           │ ← 目标汉字+田字格
│     │   │女│  │子│     │                      │
│     │   └──┘  └──┘     │                      │
│     └───────────────────┘                      │
│                                                │
│       ┌──┐     ┌──┐   💣   ┌──┐     ┌──┐      │ ← 飞入的偏旁
│       │女│     │口│        │子│     │木│      │
│       └──┘     └──┘        └──┘     └──┘      │
└────────────────────────────────────────────────┘
```

### 3.3 核心玩法详细设计

**目标汉字**：
- 屏幕上方显示目标汉字的半透明轮廓
- 轮廓分为若干个区域，对应需要填入的偏旁位置
- 同时显示拼音和越南语释义

**飞入的偏旁部首**：
- 从屏幕下方和两侧飞入
- 混合正确偏旁 + 干扰偏旁 + 炸弹
- 飞行速度随时间递增

**操作方式**：
- H5 端：滑动手指斩击（水果忍者式）
- PC 端：鼠标点击选中偏旁

**斩中正确偏旁**：
- 偏旁沿粒子轨迹飞入上方轮廓的对应位置
- 「嵌入」动画 + 发光特效
- 得分 +10

**全部偏旁就位**：
- 汉字「点亮」+ 五彩爆炸特效
- 自动播放该字的标准发音
- 显示越南语释义（2 秒）
- 进入下一个汉字

**连锁系统**：
| 连续组装数 | 特效 | 奖励 |
|-----------|------|------|
| 2 连 | "Tốt lắm!"（不错！）+ 时间 +3s | 1.5x 分数 |
| 3 连 | "Quá giỏi!"（太强了！）+ 全屏金光 | 2x 分数 |
| 5 连 | "Bậc thầy Hán tự!"（汉字大师！）+ 屏幕震动 | 3x 分数 |

**斩中炸弹/错误偏旁**：
- 爆炸动画 + 画面闪红
- 扣 1 心
- 连锁中断归零

**难度递进**：
| 阶段 | 汉字复杂度 | 干扰偏旁 | 速度 |
|------|----------|---------|------|
| 初级 | 2 部件字（好=女+子） | 1-2 个 | 慢 |
| 中级 | 3 部件字（想=木+目+心） | 3-4 个 | 中 |
| 高级 | 同时组装 2 个字 | 5+ 个 | 快 |

### 3.4 游戏模式

**闯关模式**：
- 关卡 1：组装 5 个简单汉字→关卡 2：10 个中等→关卡 3：15 个困难
- 全部完成获得星级评定

**无尽模式**：
- 无限汉字，速度越来越快
- 3 条命，挑战最高分

**每日挑战**：
- 固定 10 个汉字，全服统一排名

### 3.5 学习效果

- 偏旁识别：高速场景下快速辨认常用偏旁
- 汉字结构：知道每个字由哪些部件组成
- 形音义三维绑定：完成组装时同时播放发音+显示释义

## 四、拼音漂移（Pinyin Drift）

### 4.1 核心创意

赛车游戏 × 听力反应：俯视角赛车在多车道上飞驰，耳机里播放中文，必须漂移到正确车道通过检查点。

### 4.2 游戏画面布局（横屏 16:9）

```
┌────────────────────────────────────────────────┐
│  Score: 920     Speed: x3  🔥                   │
│                                                │
│     │   A    │   B    │   C    │               │ ← 前方 3 条分叉车道
│     │        │        │        │               │    每条标有不同答案
│     │  苹果  │  西瓜  │  香蕉   │               │
│     │        │        │        │               │
│     ─────────────────────────────              │
│              │                                 │
│           ┌──┴──┐                              │
│           │ 🏎️  │                 🔊 "píngguǒ" │ ← 玩家赛车 + 音频提示
│           └─────┘                              │
│  ← 左滑              右滑 →                    │ ← 操作提示（首次显示）
└────────────────────────────────────────────────┘
```

### 4.3 核心玩法详细设计

**赛车视角**：
- 俯视角（从上往下看），赛车在画面下方 1/3 处
- 道路向上方延伸（赛车自动向前，实际上是背景向下滚动）
- 速度逐渐递增

**检查点（题目）**：
- 前方出现 3 条分叉车道（偶尔 4 条）
- 每条车道上标着一个答案选项
- 同时播放中文音频（题目）
- 玩家左右滑动切换车道
- 到达检查点时所在车道 = 玩家的答案

**正确通过**：
- 漂移加速特效（轮胎冒烟 + 背景速度线）
- 得分 +10 × 速度倍率
- 连续正确触发「极速模式」：
  - 5 连正确：画面边缘光效
  - 10 连正确：氮气加速特效 + 2x 分数
  - 20 连正确：全屏金色闪光 + 3x 分数

**错误通过**：
- 撞上路障动画 + 减速 2 秒
- 画面震动 + 闪红
- 连续 3 次错误 = Game Over（赛车爆炸）

**题型轮换**（根据用户学习进度自动选择）：
| 题型 | 描述 | 适用阶段 |
|------|------|---------|
| 听音选字 | 播放发音 → 选正确汉字 | 拼音+词汇 |
| 听音选拼音 | 播放发音 → 选正确拼音 | 拼音复习 |
| 看字选义 | 显示汉字 → 选正确越南语翻译 | 词汇复习 |
| 听句补空 | 播放句子 → 选正确补充词 | 语法复习 |

**特殊道具**（路上随机出现）：
| 道具 | 效果 | 出现频率 |
|------|------|---------|
| 🛡️ 护盾 | 抵消一次撞墙 | 每 15s 约 1 次 |
| ⏱️ 慢动作 | 3 秒减速看清选项 | 每 30s 约 1 次 |
| 💰 双倍分 | 下 5 题得分翻倍 | 每 45s 约 1 次 |

- 道具出现在道路中央，赛车经过自动拾取
- 拾取时有特效 + 短促音效

### 4.4 操作方式

**H5 端**：
- 左右滑动切换车道
- 滑动响应 ≤ 100ms
- 赛车有惯性动画（滑动后轻微倾斜再回正）

**PC 端**：
- 键盘 ← → 方向键切换车道
- 或鼠标点击目标车道
- 车道切换有漂移动画

### 4.5 游戏模式

**单人竞速**（MVP 默认）：
- 跑最远距离
- 3 次撞墙 = Game Over
- 排行榜：按距离/得分排名

**每日赛道**：
- 固定题目 + 固定速度曲线
- 全服统一排名
- 每日奖励

**幽灵对决**（v1.1 预留设计）：
- 与好友的历史最佳记录异步竞速
- 可以看到好友的半透明幽灵车

```

│    Chính xác（准确率）: 78%   │

│                             │
│    🏆 Xếp hạng #28          │
│                             │

│                             │
│  [Chơi lại] [Chia sẻ] [Thoát]│
└─────────────────────────────┘
```

### 4.7 学习效果

- 听力反应速度：时间压力下训练即时听力理解
- 多维度复习：听/读/拼音/翻译全覆盖
- 每局 60s 可练 30+ 道题，效率远超传统卡片

## 五、排行榜系统

### 5.1 排行榜结构

每款小游戏有独立排行榜，分为：
- **日榜**：每日 0:00 重置
- **周榜**：每周一 0:00 重置
- **总榜**：累计最高分

### 5.2 排行榜 UI

```
┌─────────────────────────────┐
│  🏆 Bảng xếp hạng（排行榜）  │
│  [Hôm nay] [Tuần] [Tổng]   │ ← Tab 切换
│                             │
│  1. 🥇 Nguyen V.   2,840   │
│  2. 🥈 Tran H.     2,510   │
│  3. 🥉 Le M.       2,340   │
│  4.    Pham K.      2,100   │
│  5.    Vo T.        1,980   │
│  ...                        │
│  42. 🔵 Bạn（你）   1,240   │ ← 自己的排名高亮
│                             │
│  共 1,234 名玩家             │
└─────────────────────────────┘
```

- 排行榜显示前 100 名 + 自己的排名（如果不在前 100 则单独显示在底部）
- 每项显示：排名 + 头像 + 昵称 + 得分
- 前 3 名有金/银/铜牌标识

## 六、验收标准

| 项目 | 标准 |
|------|------|
| 游戏帧率 | ≥ 30fps（移动端）/ ≥ 60fps（PC 端） |
| 操作响应 | 滑动/点击 ≤ 100ms 视觉反馈 |
| 音频同步 | 声波弹出现与发音播放同步（≤ 50ms 偏差） |
| 排行榜 | 游戏结束后 ≤ 3 秒更新排名 |
| 得分一致 | 服务端校验分数（防刷分），客户端计算仅用于展示 |
| 分享功能 | 点击分享生成图片 + 推广链接，可分享到社交媒体 |

## 七、用户故事与边界条件

### 7.1 核心用户故事

| # | 功能 | 用户故事 | P |
|---|------|---------|---|
| 1 | Tone Sniper | 作为学习者，我想通过射击游戏练习声调辨识，以便在趣味中巩固四声知识 | P0 |
| 2 | Radical Blitz | 作为学习者，我想通过拼装游戏认识汉字部件，以便理解汉字结构规律 | P0 |
| 3 | Pinyin Drift | 作为学习者，我想通过赛车游戏练习拼音拼写，以便提高拼音输入速度 | P0 |
| 4 | 排行榜 | 作为玩家，我想看到全服排名，以便产生竞争动力继续学习 | P0 |

### 7.2 边界条件与异常处理

| 场景 | 处理方式 |
|------|----------|
| 游戏中网络断开 | 游戏继续本地运行，结束后待网络恢复提交分数；若 5 分钟内未恢复，弹出重连提示 |
| 游戏中收到来电/切后台 | 自动暂停，回到前台后继续；切后台超过 3 分钟则游戏结束，以当前分数结算 |
| 游戏加载失败（资源加载失败） | 显示重试页面 + “Tải lại”按钮，不计入游戏次数 |
| 服务端分数校验与客户端不一致 | 以服务端为准，客户端显示“分数已校正”提示 |
| 排行榜加载失败 | 显示缓存的上次排行榜 + “更新失败”提示 |
| 同时多人提交相同分数 | 按完成时间先后排序 |

### 7.3 情感化设计目标

| 页面/环节 | 情绪关键词 |
|---------|----------|
| 游戏入口 | 兴奋、期待、挑战欲 |
| 游戏进行中 | 专注、心流、紧迫感 |
| 游戏结算 | 成就感、满足、分享欲 |
| 排行榜 | 竞争、超越、荣誉感 |

### 日历视图交互

```typescript
// 日期格子样式:
// - 已排期（未来）: 白色背景，显示金句摘要 + 状态标签
// - 已发布: 白色背景，显示金句摘要 + 绿色「已发布」
// - 已过期: 灰色底纹
// - 未排期（未来）: 红色浅底纹 #fef2f2 + 「未排期」
// - 未排期（过去）: 灰色底纹
// - 今天: 蓝色边框，如未排期则红色底 + 「今日未排期 ⚠️」

// 点击交互:
// - 点击已排期格子 → 弹出浮层卡片（金句完整信息 + 编辑/取消排期按钮）
// - 点击未排期的未来日期 → 跳转新建页，排期日期自动填充
// - 点击过去日期 → 无操作
```

## 验收标准（GIVEN-WHEN-THEN）

1. **GIVEN** 进入每日金句管理页  
   **WHEN** 默认加载列表视图  
   **THEN** 金句表格正常展示，筛选栏可用，视图切换按钮可见

2. **GIVEN** 列表视图已加载  
   **WHEN** 点击日历视图图标切换  
   **THEN** 月历网格展示当月数据，已排期日期显示金句摘要，未排期的未来日期红色高亮

3. **GIVEN** 日历视图中 4 月 20 日已有排期  
   **WHEN** 点击 4 月 20 日格子  
   **THEN** 弹出浮层卡片显示完整金句信息 + 「编辑」「取消排期」按钮

4. **GIVEN** 日历视图中 4 月 25 日未排期  
   **WHEN** 点击 4 月 25 日格子  
   **THEN** 跳转新建金句页面，排期日期自动填充为 2026-04-25

5. **GIVEN** 新建金句页面输入中文金句内容  
   **WHEN** 点击拼音标注旁的「自动生成」按钮  
   **THEN** 拼音字段自动填充对应拼音，Toast「拼音已自动生成，请检查多音字是否正确」

6. **GIVEN** 排期日期选择器中 4 月 20 日已被占用  
   **WHEN** 鼠标悬停 4 月 20 日  
   **THEN** 日期灰色不可选，提示「该日期已有排期金句："{金句前10字}…"」

7. **GIVEN** 金句编辑页有未保存修改  
   **WHEN** 点击「取消」  
   **THEN** 弹出确认弹窗，选项「继续编辑」/「放弃修改」

8. **GIVEN** 页面在浅色模式  
   **WHEN** 切换到深色模式  
   **THEN** 日历网格、状态标签、浮层卡片全部正确切换

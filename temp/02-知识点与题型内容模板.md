# 02 · 知识点与题型内容模板

> 这个文件就是给 AI 出稿和后端建模看的"内容契约"。
> 所有 KP 和 Question 都按下面的 JSON 结构存进数据库（实际落库见 03 文件）。

---

## 2.1 KP 内容通用字段（所有 7 类共有）

```jsonc
{
  "kp_id": "kp_ec_w_00231",            // 主键，前缀含赛道+类型
  "track": "ec",                        // ec / fc / hk / dl / share
  "kp_type": "word",                    // pinyin / hanzi / word / phrase / grammar / sentence / dialog
  "title_zh": "价格",                   // 显示主标题（中文）
  "pinyin": "jià gé",                  // 带声调拼音
  "audio_url": "/cdn/audio/kp_ec_w_00231.mp3",
  "translations": {                     // i18n 翻译
    "vi": "giá cả",
    "th": "ราคา",
    "id": "harga",
    "en": "price"
  },
  "explanation_md": {                   // 讲解（Markdown，4 语言版本）
    "vi": "...", "th": "...", "id": "...", "en": "..."
  },
  "examples": [                         // 例句 1–3 条
    { "zh": "这个价格太贵了。", "pinyin": "zhège jiàgé tài guì le.",
      "audio_url": "...", "tr": {"vi":"...", "th":"...", "id":"..."} }
  ],
  "tags": ["电商", "价格", "HSK2"],
  "difficulty": 2,                      // 1–6（对齐 Stage）
  "ai_meta": {                          // 可追溯
    "model": "gpt-5-pro",
    "prompt_id": "prompt_v3_word",
    "generated_at": "2026-04-12T10:11:23Z",
    "human_reviewed_by": "editor_07",
    "review_status": "approved"
  },
  "version": 3                          // 改一次 +1，旧版冻结归档
}
```

## 2.2 七类 KP 的差异字段

### KP1 · 拼音音节 `pinyin`
```jsonc
{
  "kp_type": "pinyin",
  "title_zh": "zhōng",
  "components": { "initial": "zh", "final": "ōng", "tone": 1 },
  "is_whole_syllable": false,
  "common_chars": ["中", "钟", "终"],   // 此音节常对应的汉字
  "tongue_diagram_url": "/cdn/img/zh.svg" // 发音口型示意图（可选）
}
```

### KP2 · 汉字 / 部首 `hanzi`
```jsonc
{
  "kp_type": "hanzi",
  "title_zh": "好",
  "pinyin": "hǎo",
  "stroke_count": 6,
  "structure": "left-right",            // left-right / top-bottom / enclosure / single ...
  "radicals": ["女", "子"],
  "stroke_order_svg": "/cdn/img/hao_stroke.svg",
  "stroke_order_anim_url": "/cdn/anim/hao.json", // Lottie 或帧序列
  "common_words": ["你好", "好朋友", "好看"]
}
```

### KP3 · 词汇 `word`（最常用）
> 见 §2.1 通用结构即可，加一个 `pos`（词性）。
```jsonc
{ "kp_type": "word", "pos": "noun", "hsk_level": 2 }
```

### KP4 · 短语 / 搭配 `phrase`
```jsonc
{
  "kp_type": "phrase",
  "title_zh": "下订单",
  "pattern": "下 + 订单",
  "co_occurrences": ["在网上下订单", "帮我下个订单"]
}
```

### KP5 · 句型 / 语法 `grammar`
```jsonc
{
  "kp_type": "grammar",
  "title_zh": "把字句",
  "pattern": "S + 把 + O + V + 其他",
  "rule_md": { "vi": "...", "th": "...", "id": "..." },
  "positive_examples": [ "我把书放在桌子上。", "请把门关上。" ],
  "negative_examples": [ "我把吃饭。" ],
  "contrast_with": ["被字句"]
}
```

### KP6 · 短句 `sentence`
```jsonc
{
  "kp_type": "sentence",
  "title_zh": "这个多少钱？",
  "scenario": "购物",
  "function_tag": "询问价格",
  "reply_examples": ["五十块。", "不好意思，已经卖完了。"]
}
```

### KP7 · 对话 / 长句 `dialog`
```jsonc
{
  "kp_type": "dialog",
  "title_zh": "客服-退货-3轮",
  "turns": [
    { "role": "buyer",  "zh": "你好，我想退货。", "pinyin": "...", "audio_url": "..." },
    { "role": "seller", "zh": "好的，请问什么原因？", "pinyin": "...", "audio_url": "..." },
    { "role": "buyer",  "zh": "尺码不合适。", "pinyin": "...", "audio_url": "..." }
  ],
  "key_kp_refs": ["kp_ec_w_00231", "kp_ec_p_00078"]  // 这段对话用到的 KP
}
```

```jsonc
  "answer": ... ,                    // 标准答案

  "difficulty": 2,
  "audio_url": null,
  "image_url": null,
  "ai_meta": { "model":"...", "prompt_id":"...", "review_status":"approved" }
}
```

## 2.4 十二种题型（含 JSON 模板 + 计分规则）

> 所有题型**都不需要麦克风、不需要手写板**。

| # | 类型 code | 名称 | 适合 KP | 示例 |
|---|---|---|---|---|
| Q1 | `mcq_meaning` | 中→母语 选词义 | word/phrase | 「价格」= ? |
| Q2 | `mcq_zh` | 母语→中 选汉字 | word/phrase | "giá cả" = ? |
| Q3 | `listen_pick` | 听音选汉字 | pinyin/word | 听 jià gé → 选「价格」 |
| Q4 | `listen_pinyin` | 听音选拼音 | pinyin | 听音 → 选 jià gé |
| Q5 | `tone_pick` | 选正确声调 | pinyin/word | jia ge → 标 4-2 声 |
| Q6 | `match_pairs` | 连线配对 | word | 4 个汉字 ↔ 4 个母语 |
| Q7 | `sort_words` | 拖拽排序组句 | sentence/grammar | 词块组成正确句子 |
| Q8 | `fill_blank_choice` | 填空选词 | grammar/sentence | 我 __ 书放在桌子上（选「把」） |
| Q9 | `type_pinyin` | 听后键盘输入拼音 | word/sentence | 输入 jiage |
| Q10 | `type_zh_ime` | 看图/看翻译输入汉字（IME） | word/sentence | 用拼音输入法打"价格" |
| Q11 | `image_pick` | 看图选词 | word | 4 张图选"价格牌" |
| Q12 | `dialog_cloze` | 对话填空 | dialog | 客服 3 轮，挖空 1 句让你选 |

> 进阶题型（二期再加，先列出占位）：`stroke_order`（笔顺正误判断）、`reading_comp`（短文阅读理解）、`tone_dictation`（连续标调）。

### Q1 `mcq_meaning` —— 中→母语选词义（最常用）
```jsonc
{
  "q_type": "mcq_meaning",
  "stem": { "zh": "价格", "pinyin": "jià gé" },
  "options": [
    { "id": "A", "text_vi": "giá cả", "text_th": "ราคา", "text_id": "harga" },
    { "id": "B", "text_vi": "chất lượng", ... },
    { "id": "C", "text_vi": "vận chuyển", ... },
    { "id": "D", "text_vi": "đơn hàng", ... }
  ],
  "answer": "A",
  "scoring": { "type": "single", "score": 1 }
}
```

### Q3 `listen_pick` —— 听音选汉字
```jsonc
{
  "q_type": "listen_pick",
  "stem": { "audio_url": "/cdn/audio/q/12345.mp3", "play_limit": 3 },
  "options": [
    { "id":"A", "zh":"价格" }, { "id":"B", "zh":"价钱" },
    { "id":"C", "zh":"加个" }, { "id":"D", "zh":"假冒" }
  ],
  "answer": "A"
}
```

### Q5 `tone_pick` —— 标声调
```jsonc
{
  "q_type": "tone_pick",
  "stem": { "zh": "价格", "pinyin_no_tone": "jia ge", "audio_url":"..." },
  "options": "tone_grid",           // 前端渲染 [1,2,3,4] x 字数
  "answer": [4, 2]                  // 第 1 字 4 声、第 2 字 2 声
}
```

### Q6 `match_pairs` —— 连线
```jsonc
{
  "q_type": "match_pairs",
  "left":  [ {"id":"L1","zh":"价格"}, {"id":"L2","zh":"订单"}, {"id":"L3","zh":"发货"}, {"id":"L4","zh":"评价"} ],
  "right": [ {"id":"R1","tr":"giá cả"}, {"id":"R2","tr":"đơn hàng"}, {"id":"R3","tr":"giao hàng"}, {"id":"R4","tr":"đánh giá"} ],
  "answer": [ ["L1","R1"], ["L2","R2"], ["L3","R3"], ["L4","R4"] ],
  "scoring": { "type": "partial", "per_pair_score": 0.25 }
}
```

### Q7 `sort_words` —— 拖拽组句
```jsonc
{
  "q_type": "sort_words",
  "stem": { "translation_vi": "Cái này bao nhiêu tiền?" },
  "tokens": ["这个", "钱", "多少", "？"],
  "answer": ["这个", "多少", "钱", "？"]
}
```

### Q8 `fill_blank_choice` —— 填空选词
```jsonc
{
  "q_type": "fill_blank_choice",
  "stem_zh": "请 ___ 门 关上。",
  "blank_index": 1,
  "options": ["把", "被", "让", "在"],
  "answer": "把"
}
```

### Q9 `type_pinyin`
```jsonc
{
  "q_type": "type_pinyin",
  "stem": { "audio_url":"...", "hint_zh":"价格", "play_limit":3 },
  "answer_regex": "^\\s*jia\\s*ge\\s*$",
  "case_insensitive": true
}
```

### Q10 `type_zh_ime` —— 用系统拼音输入法打中文
```jsonc
{
  "q_type": "type_zh_ime",
  "stem": { "translation_vi": "giá cả", "hint_pinyin":"jià gé" },
  "answer": "价格",
  "accept_traditional": false
}
```

### Q11 `image_pick`
```jsonc
{
  "q_type": "image_pick",
  "stem": { "zh": "价格牌", "pinyin":"jià gé pái" },
  "options": [
    { "id":"A", "image_url":"/cdn/img/q/12345_a.webp" },
    { "id":"B", "image_url":"/cdn/img/q/12345_b.webp" },
    { "id":"C", "image_url":"/cdn/img/q/12345_c.webp" },
    { "id":"D", "image_url":"/cdn/img/q/12345_d.webp" }
  ],
  "answer": "A"
}
```

### Q12 `dialog_cloze` —— 对话填空
```jsonc
{
  "q_type": "dialog_cloze",
  "dialog": [
    {"role":"buyer", "zh":"你好，我想退货。"},
    {"role":"seller","zh":"___"},                  // 挖空
    {"role":"buyer", "zh":"尺码不合适。"}
  ],
  "options": [
    "好的，请问什么原因？",
    "对不起，没有这个商品。",
    "您好，请扫码支付。",
    "非常感谢您的好评！"
  ],
  "answer": 0
}
```

## 2.5 题目难度与组卷规则（给"节末小测 / 章测 / 阶段考"用）

| 考核 | 题量 | 难度分布 | 通过线 |
|---|---|---|---|
| 随堂练（每 KP 后） | 2–4 | 100% 当前 KP | 不卡，错了直接看解析重做 |
| 节末小测 | 5 | 80% 本节 + 20% 本章前节复习 | 60% |
| 章测 | 15 | 70% 本章 + 30% 前章 SRS 队列 | 70% |
| 阶段考 | 50 | 按本阶段所有章均衡抽 | 70% 解锁下一阶段 |
| HSK 模考（仅 HSK 赛道） | 按官方比例 | 当级全覆盖 | 60% |

## 2.6 SRS 调度规则（Leitner 5 盒，运行时零 AI）

```
错对结果 → 调整盒位 → 计算下次复习日期
盒 1 → 1 天后；盒 2 → 3 天；盒 3 → 7 天；盒 4 → 14 天；盒 5 → 30 天
答对 → 升一盒；答错 → 回到盒 1
```

每天用户首页"今日复习"= 当日到期 KP 数（按盒 1→5 优先级）。

## 2.7 AI 出稿 Prompt 模板（举两例）

```
[Prompt: kp_word_v3]
任务：为东南亚零基础中文学员生成一个"词汇 KP"。
输入：{ word_zh, track, stage, target_languages: [vi,th,id,en] }
输出：严格 JSON，符合 §2.2 word 模板。
约束：
- 例句 2 条，必须使用本赛道 stage ≤ 当前 stage 的已知词汇（清单见附录 A）。
- 翻译要口语化，不要直译；
- explanation_md 不超过 80 字；
- 不出现繁体字；
- 不引用任何政治、宗教、地域歧视内容。
```

```
[Prompt: q_mcq_meaning_v2]
任务：给定一个 word KP，生成 1 道 mcq_meaning 题。
约束：
- 4 个选项必须同词性、同长度区间；
- 干扰项必须是本赛道已存在的真实词汇，不允许造词；
- 答案位置随机；
- 解析 1 句话讲清为什么对。
```

> 完整的 prompt 库统一放在管理端"AI 工作台 → Prompt 管理"，每条 prompt 也有版本号。

## 6. 裂变系统技术实现方案

### 6.1 邀请链接生成

```
每个用户注册后自动生成唯一邀请码：
  格式：HANYU-{用户ID短码}-{随机4位}
  示例：HANYU-A3K9-X2M1

邀请链接格式：
  https://hanyu.app/invite/{邀请码}
  
打开链接后：
  - 已安装 App -> Deep Link 直接跳转 App
  - 未安装 -> 跳转应用商店下载页（带邀请码参数）
```

### 6.2 推荐追踪流程

```
1. 用户 A 分享邀请链接
2. 用户 B 点击链接
3. 系统记录：referrer=A, channel=链接来源
4. 用户 B 下载并注册
5. 系统记录：B.referrer = A
6. 用户 B 完成 3 天学习
7. 触发奖励：A 获得 7 天 Premium
8. 用户 B 付费
9. 等待 30 天冷却期
10. 确认 B 持续付费
11. 触发奖励：A 获得 30 天 Premium
12. 系统累计 A 的有效推荐数
```

### 6.3 推广大使仪表盘

```
推广大使后台功能：
- 实时查看推荐数据（注册/付费/收益）
- 生成自定义推荐链接和海报
- 查看返佣明细和提现记录
- 排行榜（本月推荐最多的大使）
- 推广素材下载（图片、文案模板）

提现规则：
- 最低提现金额：$10
- 提现方式：MoMo/GoPay/银行转账
- 结算周期：每月 1 日结算上月收益
- 30 天冷却期后才计入可提现余额
```

### 6.4 分享海报自动生成

```
用户可生成的分享海报类型：
1. 打卡海报："我已在 HanYu 学中文 X 天，认识了 X 个汉字"
2. 成就海报："我通过了 HSK 1 级模拟考试！"
3. 排行榜海报："我是本周钻石段位第 X 名"
4. 对战邀请："来和我 PK 汉字消消乐"

每张海报自动包含：
- 用户昵称和头像
```

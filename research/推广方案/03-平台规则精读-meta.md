# 03 · 平台规则精读 · Meta（Facebook + Instagram）

> 适用：FB Page、FB Group、Instagram Business / Creator Account。
> 基准：Meta Community Standards（2024 版）+ Facebook Pages, Groups & Events Policies + Instagram Community Guidelines + Branded Content Policies + Commerce Policies。

## 1. 为什么 Meta 是"中等严格"

- 政策文档完整、规则透明，违规多为限流不封号。
- 但**外链管控比 TikTok 宽**：FB 主页、Group、IG bio、Stories 滑动链接都允许放 App Store / Google Play 链接。
- 风险点在"投诉机制"：举报阈值低，恶意举报易导致 24h 临时封禁。

## 2. 必须熟记的 5 大红线

| # | 红线 | 触发情境 | 规避做法 |
|---|------|---------|---------|
| 1 | 虚假身份 / 冒充 | Page 名字带 "Official" / 假装泰国/越南本地人 | 主体如实填写"教育-语言学习"；不冒充本地人 |
| 2 | 仇恨言论 / 受保护群体歧视 | 涉及种族、宗教、国籍、性别取向 | 不在内容中比较"哪国人学中文更聪明"等表述 |
| 3 | 误导性健康 / 财务声明 | "学中文月入 X 美元" | 不出现具体收入数字、不承诺就业 |
| 4 | Engagement Bait（互动诱饵）| "点赞才能看下一句"、"@3 个朋友送会员" | Meta 重点打击；改用"评论区告诉我你最想学的场景，下条做" |
| 5 | 重复发布 / 跨账号同内容 | 多个 Page 同步同内容 | 主 Page 与 Group 内容差异化 ≥ 30% |

## 3. 关键规则细节

### 3.1 Page 自然触达限制
- 自然触达率 1-3%，新 Page 0 粉发文几乎无曝光。
- **破局做法**：
  - 同步 IG Reels 后，IG 的播放数据会反哺 FB 算法（仅限同 Meta Business Suite 关联账号）；
  - 第 1 条 Page 内容必须是"高完播率视频"，不要发图文测试水温（FB 现在视频权重远高于图文）。

### 3.2 Group 政策（核心阵地）
- 1 个 Page 可创建多个 Group，按国家分（VN、TH、ID）。
- Group 类型选 **Public**（公开），便于 SEO + 加群无门槛；
- **3 个加群问题** + **群规 5 条** 必须设置（防机器人和广告号涌入）。
- **群内禁止行为**（Meta 会监测）：
  - 引导加 WhatsApp / Telegram / Line / Zalo 群（**会被识别为外引导**，但比 TikTok 宽松，单次警告制）；
  - 反复发同 1 条 App 链接（每周不超过 2 次）；
  - 群成员之间私下交易（被举报会牵连 Page）。
- **合规导流方法**：
  - 群置顶帖：每周一发 1 条"本周打卡帖"，置顶 7 天，自然引导参与；
  - 群描述（About 区）放 App Store 链接（永久曝光，不算违规）。

### 3.3 外链与 App 推广
- ✅ FB Page → About → Website 字段：可放 App 官网（**首选**，不放 App Store 直链）。
- ✅ Page 简介可写："Find our app on iOS / Android - link in About section"。
- ⚠️ 普通帖子带 App Store 链接：每周不超过 1 次，且必须有**实质教学内容 ≥ 80%**。
- ❌ 短链接服务（bit.ly、tinyurl）：触发反钓鱼判定，自然触达砍半。
- ✅ Linktree 等聚合页：允许，但不如 Page About 直接配置外链。

### 3.4 IG Reels 算法关键
- 与 TikTok 算法非常相似，但 **IG Reels 更重视前 3 秒完播 + 收藏数**。
- 用 TikTok 视频去水印后发 Reels：
  - 必须重新加 IG 内置音乐（用 TikTok 音乐 = 算法识别为"非原生"，限流 50%）；
  - 重新写 caption（不要直接复制 TikTok 文案）；
  - 重新选 cover；
  - 比例改为 9:16 全屏（与 TikTok 相同）；
  - 标签 5-10 个（IG 比 TikTok 容忍度高）。

### 3.5 IG bio 链接
- 1 个 bio link 限制（除非是商业账号 + Linkin.bio 工具）；
- **建议**：1k 粉前 bio link 指向独立站（不直接给 App Store），独立站再分发 iOS/Android；
- IG Stories：1 万粉以下没有 swipe-up，使用 Link Sticker（已对所有账号开放）。

### 3.6 Branded Content 披露
- IG / FB 都需要在内容里勾选 "Paid partnership" 或自有品牌；
- 自家 App 出现 → 选 "Promoting my own business"；
- 不勾选不会立即封禁，但被举报后会扣 Page 评分。

## 4. 越南 / 泰国本地化规则

### 4.1 越南
- 越南要求 Meta 在 24h 内删除"违法内容"，举报机制对外国账号特别严格；
- 不要使用越南国旗 emoji（🇻🇳）作为账号名前缀（被认定为"冒充官方"风险）；
- Group 内**禁止讨论政治话题**，违反 Meta 也会按越南法律下架。

### 4.2 泰国
- 王室相关内容下架率 100%，且 Page 直接停用；
- 佛教 / 僧侣图像在商业账号使用风险高，**完全规避**；
- 泰国电子交易办公室 (ETDA) 要求所有"在泰国提供服务的平台账号"在受到投诉后 7 日内回应，**Page Inbox 必须每天看**。

## 5. 商业政策（Commerce Policies）

- 当前阶段不开 FB Shop / IG Shop，无需关注；
- 后期 App 内会员售卖：**不要在 FB 帖子直接挂会员价格**，引导到 App Store 内购，避免触发 Commerce Policies 的"教育服务销售"审核。

## 6. 申诉与封禁应对

| 情况 | 应对 |
|------|------|
| Page 单条帖子被下架 | Business Suite → Quality 面板 → Request Review，48h 内回复 |
| Page 被临时封禁 24h | 多为恶意举报，提交 ID 验证 → 通常 12-24h 解封 |
| Page 被永久封禁 | 通过 Help Center → "Disabled Page" → 提交 Business 信息申诉，成功率 30% |
| IG 账号被禁 | App 内 → Help → Report a Problem → 等待人工，通常 7 日 |

## 8. 与原方案的差异

| 原方案做法 | 校验后调整 |
|----------|----------|
| "FB 每天 1 条图文" | ⚠️ 改为"每周 3 条 Reels + 1 条精修图文 + 1 条置顶打卡"，**质量优先** |
| "FB 不引导微信/WhatsApp，只在 FB 生态" | ✅ 完全正确，且需扩展为"也不引导 Line/Zalo"（在 Meta 域内不要外引导，外引导动作放在 Group 描述长期挂载） |
| "FB Page 简介可放 App 链接" | ✅ 正确，建议放官网而非 App Store 直链（独立站可加 UTM 监测渠道） |
| "1000 粉后建社群" | ⚠️ 太晚；FB Group 可以**第 1 天就建**（公开群+主动加 50 个种子用户），不要等粉丝量 |
| "私信自动回复" | ✅ 正确，且必须设置（合规要求 24h 回复率） |

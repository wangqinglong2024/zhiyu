# 12 · 内容生产 SOP

> 1 个运营 + 1 个母语审校 = 每日产 1 条视频 × 4 平台分发的标准化流水线。
> 单条视频从立项到上线全流程 ≤ 90 分钟。

---

## 1. 流水线总览

```
选题（5min）→ 中文脚本（10min）→ 母语翻译（异步审校）→ 录音/AI TTS（10min）
→ CapCut 剪辑（30min）→ 字幕校对（10min）→ 4 平台导出（10min）→ 分发（15min）
```

---

## 2. 步骤详解

### 2.1 选题（5min）
- 打开 Notion "一句话内容数据库"
- 按今日主题（电商/工厂/HSK/日常）取 1 行
- 检查字段完整：中文 / 拼音 / 越/泰翻译 / 关键字 / 钩子方向

### 2.2 中文脚本（10min）
五段式：
```
[钩子] 母语 1 句   ：3 秒内说痛点 / 反差 / 数字
[呈现] 中文 1 句   ：大字 + 拼音 + 母语 同时出
[拆解] 1 个知识点  ：字源 / 语法 / 文化（三选一）
[跟读] 慢速 + 正常 + 留 3 秒空档
[CTA]  关注 / 评论 / 入群 三选一
```

### 2.3 母语审校（异步，1-4h）
- 通过 Telegram / Zalo 把脚本发给审校员
- 审校反馈：✅ 通过 / 🔄 修改 / ❌ 重写
- 不通过的句子退回选题库标记，下周重审

### 2.4 录音 / AI TTS（10min）
- 中文：真人录音（普通话二甲以上）或 ElevenLabs 中文 TTS
- 母语：当地母语真人录音 优先；备选 Microsoft Azure Neural Voice
- 录音规范：48kHz / 16bit / WAV，预留 0.5s 静音首尾

### 2.5 CapCut 剪辑（30min）
- 使用预设模板（按主题色：电商红 / 工厂蓝 / HSK 绿 / 日常黄）
- 字幕样式按 06 第 2 节
- BGM 必须来自商用库（TikTok TH/VN Trending 列表 + YouTube Audio Library）
- 时长 21-34s
- 导出 1080×1920 60fps

### 2.6 字幕校对（10min）
- 母语审校最后一遍核对（可在 CapCut 链接预览）
- 检查：错别字 / 字幕同步 / 拼音声调正确

### 2.7 4 平台导出（10min）

| 平台 | 调整项 |
|---|---|
| TikTok | 原版导出，不加水印（CapCut 设置） |
| YouTube Shorts | 替换为 YouTube 音乐库 BGM，同时加片尾 1-2 秒"Subscribe"提示卡 |
| Zalo Story / Line Timeline | 截取主画面 9:16 静态图 + 上 / 下加文字描述 |
| Zalo OA / Line OA Card | 用 Canva 模板做静态图 + 中文音频 1 个文件附件 |

### 2.8 分发（15min）

| 时间 | 操作 |
|---|---|
| 11:00 | TikTok 发布 |
| 11:05 | TikTok 评论区置顶引流话术（按 04 / 05 文档话术）|
| 18:00 | YouTube Shorts 发布（隔 7 小时避免去重判定）|
| 8:00 (次日) | Zalo OA / Line OA 推送当天卡片 |
| 9:00 (次日) | Zalo / Line 主群 + 主题群发"昨日 1 句"复习 |

## 5. 质量检查清单（上线前必过）

### 视频本体
- [ ] 时长 21-34 秒
- [ ] 1080×1920 分辨率
- [ ] 中文字幕带拼音
- [ ] 母语翻译已审校
- [ ] BGM 来自商用库
- [ ] 无平台水印（TikTok 水印 / 小红书水印 / 抖音水印）
- [ ] 不出现政治符号、王室、宗教敏感物
- [ ] 关键字源 / 语法点拆解清晰

### 描述与标签
- [ ] 标题含本市场关键词
- [ ] 描述含 1 句中文 + 拼音 + 母语翻译
- [ ] 标签 4-6 个，含 1 主题大词 + 2 本地 + 2 长尾 + 1 品牌
- [ ] CTA 清楚（"评论 1"/"扫码"/"点 bio")

### 私域同步
- [ ] Zalo OA 卡片已生成
- [ ] Line OA 卡片已生成
- [ ] 主群预排消息已设置

## 6. 异常处理 SOP

| 异常 | 处理 |
|---|---|
| 上线 1 小时播放 < 100 | 检查标签是否被限流 / 是否触红线 |
| 评论指出翻译错 | 1) 顶置歉意评论；2) 24h 内编辑置顶视频说明；3) 48h 重剪 |
| 收到 strike 警告 | 立即撤下 + 停发该号 24h + 法务复审 |
| 母语审校未按时回 | 启用备用审校 / AI 双向回译验证 |
| BGM 被 Content ID 命中 | 立即换 BGM 重新发布 |

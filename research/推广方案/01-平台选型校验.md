# 01 · 平台选型校验

> 校验对象：用户原方案选择 TikTok（主）+ Facebook（辅）+ YouTube（补）三平台。
> 校验结论：**方向正确 70%，需要 3 处关键修正**。

## 1. 原方案三平台逐一校验

### 1.1 TikTok ✅ 选对了，且必须做主阵地

**校验依据**：

| 维度 | 数据（2025-2026 年东南亚） | 结论 |
|------|--------------------------|------|
| 越南月活 | ≈ 6,700 万（人口 71%） | 第一大短视频平台 |
| 泰国月活 | ≈ 5,000 万（人口 71%） | 第一大短视频平台 |
| 16-35 岁渗透率 | 越南 88%、泰国 85% | 完全覆盖目标人群 |
| 语言学习 hashtag 月曝光 | #LearnChinese 全球 80 亿+，#hoctiengtrung（越）3.5 亿+，#เรียนภาษาจีน（泰）1.2 亿+ | 流量池足够大 |
| 0 粉新号冷启动友好度 | ⭐⭐⭐⭐⭐ 算法纯内容驱动，不看粉丝量 | 最适合 0 粉启动 |
| 母语讲解差异化空间 | 99% 现有账号用英语/普通话，本地语种讲解极少 | **蓝海** |

**结论**：TikTok 必须做，且必须放 ≥ 60% 精力。原方案"15-60 秒"是对的，但需细化：**21-34 秒区间播完率最高**（这是泰/越语言类小账号爆款的实证甜区，详见 06）。

### 1.2 Facebook ⚠️ 不能作为"内容主辅阵地"，应降级为"承接 + 社群"

**校验依据**：

| 现象 | 说明 |
|------|------|
| Reach 持续下滑 | Meta 主动压制 Page 自然触达至 1-3%（2024 年起），新 Page 0 粉发文几乎无曝光 |
| 16-25 岁用户流失 | 越南/泰国 Z 世代日均 FB 时长从 45min 降至 18min，迁移到 TikTok + IG |
| 视频原生分发弱 | FB Reels 算法仍优先大 Page，新号视频曝光低于 IG Reels 同等内容 50%+ |
| 优势仍在 | 30+ 岁人群、群组（Group）粘性、Messenger 客服闭环、东南亚电商从业者真在用 |

**修正建议**：

- ❌ 不再把 FB Page 当"日更内容"主战场（原计划每天 1 条图文）。
- ✅ FB Page 改为**"TikTok 爆款搬运 + 每周 2 条精修知识卡片"**（频率从 7 条/周降到 3 条/周）。
- ✅ FB 真正的价值在 **Group**：粉丝过 1000 后建"30 天中文打卡群"（按国家分），用 Group 做日活和 UGC，不在 Page 上死磕自然流量。
- ✅ Messenger 自动回复保留，是合规外链承接关键。

### 1.3 YouTube ⚠️ 应改为 "YouTube Shorts 优先 + 长视频每月 2 条"

**校验依据**：

| 维度 | 现状 |
|------|------|
| Shorts 算法 | 与 TikTok 高度相似，0 粉新号有自然推荐池，**复用 TikTok 竖屏视频零成本** |
| 长视频 5-10min | 需要剪辑能力 + 封面 + SEO，0 粉新号 6 个月内不会有自然流量，靠搜索来量极慢 |
| 搜索长尾价值 | 真实存在，但起效周期 6-12 个月，不适合冷启动期主推 |
| 东南亚 Shorts 月活 | 越南、泰国均超 3,000 万 |

**修正建议**：

- ✅ **第 1-90 天**：YouTube 只做 Shorts（竖屏视频从 TikTok 直接搬，改个标题加 #Shorts + 本地语标签）。
- ✅ **第 90 天后**：每月加 2 条长视频（5-8 min），主题选"30 句…全集合订版"，专吃搜索流量。
- ❌ 不要在前 3 个月每周 2 条长视频（原计划），人力会被压垮，ROI 极低。

## 2. 必须新增的平台（原方案缺失，是关键漏洞）

### 2.1 Zalo（越南必做） ⭐⭐⭐⭐⭐

| 项目 | 说明 |
|------|------|
| 渗透率 | 越南 90%+，几乎所有 16+ 越南人都在用 |
| 角色 | 国民级 IM + 朋友圈（Zalo Page、Zalo OA） |
| 用法 | **不做内容创作**，只做 1 件事：把 TikTok/FB 粉丝沉淀到 Zalo OA（Official Account），用 Zalo 群做打卡和 App 引流 |
| 合规 | 越南数据本地化要求高，外部 IM（WhatsApp/Telegram）在越南渗透率不足 20%，必须用 Zalo |
| 投入 | 申请 1 个免费 OA + 群（5-15 人/群），1 周搭建完成，后续每天 30 min 维护 |

### 2.2 Line（泰国必做） ⭐⭐⭐⭐⭐

| 项目 | 说明 |
|------|------|
| 渗透率 | 泰国 90%+，国民级 IM |
| 角色 | LINE Official Account + LINE OpenChat（开放群，最多 5000 人/群） |
| 用法 | 同 Zalo 思路：TikTok 粉丝 → Line OpenChat 中文学习群 → App 推荐 |
| 优势 | OpenChat 不需要互加好友，加群门槛低，UGC 活跃度高于 FB Group |
| 投入 | 1 个免费 OA + 1 个 OpenChat，每天 30 min 维护 |

### 2.3 Instagram Reels（越/泰双市场加分项） ⭐⭐⭐⭐

| 项目 | 说明 |
|------|------|
| 渗透率 | 越南 1,200 万、泰国 1,800 万，集中在 18-30 岁女性、白领 |
| 角色 | TikTok 视频零成本搬运，吃 Meta 算法红利 |
| 用法 | 同账号同内容，去掉 TikTok 水印（用 SnapTik 或剪映原片）后发 Reels |
| 与 FB 的关系 | IG Reels 可一键 Cross-post 到 FB Reels，但**不要勾选**（IG/FB 算法独立，分开发数据更好） |
| 投入 | 5 min/天（搬运 + 改文案） |

### 2.4 Lemon8（仅泰国可选，性价比中等） ⭐⭐⭐

- 字节跳动旗下，泰国是 Lemon8 海外重点市场之一，定位"小红书"。
- 形态：图文为主、长图卡片。可作为"知识点笔记"补充阵地，每周 2 条。
- 缺点：人群偏女性年轻白领，与电商打工/工厂打工目标人群重叠度仅 ~40%。
- **建议**：粉丝过 5k 再启动，前期不投入。

### 2.5 Threads（顺手发，零成本） ⭐⭐

- 与 IG 一键同步，纯文字短句。
- 用法：每天发 1-2 句"今日学 1 句中文"短句卡片，全自动复制粘贴 30 秒搞定。

## 3. 不建议投入的平台（明确放弃）

| 平台 | 不投原因 |
|------|---------|
| Twitter / X | 东南亚渗透率 <15%，语言学习社区极弱，0 粉冷启动几乎不可能 |
| Reddit | r/ChineseLanguage 只有 20 万订阅且以欧美为主，与东南亚目标人群错配 |
| Pinterest | 东南亚渗透率 <8%，且不利于视频转化 |
| Snapchat | 东南亚不流行 |
| Telegram | 越南/泰国渗透率 <20%，且中文学习社区已被英文 KOL 占领 |
| Bilibili / 抖音国内 | 目标用户在东南亚，不在国内，且国内运营需国内主体 |
| Kwai (快手海外) | 在越南/泰国边缘化，2024 年起持续退潮 |
| Shopee Live / Lazada Live | 适合卖货不适合教学引流，留待 App 商业化阶段再考虑 |

## 4. 校验后的最终平台矩阵

```
┌──────────────────────────────────────────────────────────┐
│                   内容生产端（1 条视频）                    │
│                       ↓ 同源分发                           │
├────────────────┬─────────────────┬───────────────────────┤
│ 主战场（流量池） │ 副战场（流量池+承接） │ 沉淀池（私域+导流 App）│
├────────────────┼─────────────────┼───────────────────────┤
│ TikTok         │ IG Reels        │ Zalo OA / 群（越南）  │
│ YouTube Shorts │ FB Page+Group   │ Line OA / OpenChat（泰）│
│                │ Threads（顺带）  │                       │
├────────────────┼─────────────────┼───────────────────────┤
│ 70% 精力       │ 20% 精力        │ 10% 精力（自动化为主）│
└────────────────┴─────────────────┴───────────────────────┘

           ↓
   独立站（SEO 长尾） + App Store ASO
   （后台被动流量，每周 1 篇双语长文）
```

## 5. 一句话校验总结

> **"TikTok 一鱼三吃（→ IG Reels → YT Shorts）做拉新，Zalo/Line 做留存和导流 App，FB Group 做 UGC 沉淀，独立站和 ASO 收被动搜索流量。"** 这个组合在东南亚语言学习赛道是验证过的最高性价比。

# 03 · YouTube Shorts 合规与算法

> 来源：YouTube Community Guidelines + Advertiser-Friendly Content Guidelines + Shorts Monetization Policy（2024-08 版）。

---

## 1. 红线

YouTube 的 Strikes 制度比 TikTok 更严：3 strikes / 90 天 = 永久封号。

| 类别 | 具体禁止 |
|---|---|
| 仇恨言论 | 同 TikTok |
| 错误信息 | 医疗 / 选举 / COVID 全球性误导 |
| 儿童安全 | COPPA：13 岁以下定向内容必须勾选"Made for Kids" |
| 知识产权 | Content ID 全自动比对，任何中文歌曲/影视原片均会被判侵权 |
| 暴力血腥 | 同 TikTok |
| 政治敏感 | 比 TikTok 略宽松，但东南亚政治人物形象仍需谨慎 |

### 1.1 中文学习赛道高频踩坑点
- ❌ 不直接剪《甄嬛传》《狂飙》《漫长的季节》原片：Content ID 100% 命中。
- ❌ 不使用周杰伦/邓紫棋/凤凰传奇等任何已发行歌曲做 BGM：必判侵权。
- ✅ 使用 YouTube Audio Library 免版权音乐，或 Epidemic Sound（付费 $14.99/月）订阅。
- ❌ 缩略图不使用艺人/明星人脸：肖像权 + 误导性双重风险。

## 3. Shorts 算法可观测规律

### 3.1 推流逻辑
- 与 TikTok 类似，但更看重 **首日完播率** + **订阅转化率**。
- 7 天热度衰减比 TikTok 慢；爆款长尾可持续 2-4 周。
- **被动搜索流量**是 Shorts 独有优势：标题/描述含 HSK / Tiếng Trung / ภาษาจีน 等关键词的视频，6 个月后仍持续被搜到。

### 3.2 视频长度甜点区
- 21-45 秒（容忍度比 TikTok 高，可适当拉长讲解）
- 完播率 < 25% 直接进黑名单池

### 3.3 与 TikTok 视频复用注意点
| 项 | 处理 |
|---|---|
| TikTok 水印 | 必须去除（裁剪 + 4K 升频，或使用 SnapTik 后再用 CapCut 重新输出） |
| TikTok BGM | 必须替换为 YouTube Audio Library 音乐 |
| 字幕 | 字号 / 颜色 / 位置可保留 |
| 钩子语 | 把"刷到这条说明算法懂你"换为"Watch till end to know" / 越南/泰国母语版本 |
| 时长 | 可比 TikTok 版本长 5-10 秒（加 outro 引导订阅） |

## 4. SEO（被动搜索）优化

Shorts 与长视频共享搜索池，做好 SEO 等于免费流量永久续费。

### 4.1 标题模板（70 字符内）
```
[钩子] | [中文一句话] | [HSK X / 主题] | 越/泰文关键词
例：HSK 1 must know! | 你叫什么名字？| Tiếng Trung giao tiếp
例：1688 ลดราคา 30% | 能再便宜点吗？| ภาษาจีนสำหรับธุรกิจ
```

### 4.2 描述模板
```
今日 1 句中文：你叫什么名字？(nǐ jiào shénme míngzi?)
意思：What's your name?

📘 HSK 等级：HSK 1
🎯 学完这句你能：在见到中国人时自我介绍
📲 想要完整 7 天免费课？点 → [链接]

#hoctiengtrung #hsk1 #tiengtrungcoban
---
（最后附 50 字英文 keyword stuffing 但要自然，避免 spam 标签）
```

### 4.3 标签
- YouTube 标签权重已大幅降低（2024 公开声明），重点放标题 + 描述。
- 仍建议加 5-10 个标签作辅助：本地语 + 英文 + 主题词。

## 5. 频道设置最佳实践

| 项 | 设置 |
|---|---|
| 频道名 | 与 TikTok 用户名同源，加 `- Học Tiếng Trung` / `- ภาษาจีน` 后缀 |
| 频道头像 / Banner | 与 TikTok 一致（品牌一致性影响算法识别） |
| 频道描述 | 200 字内，3 段：1) 你是谁  2) 频道更新节奏  3) 主题分布 |
| 频道国家 | 越南 / 泰国（决定推荐受众） |
| 默认语言 | 越南语 / 泰语 |
| 播放列表 | 按 4 主题分 4 个 Playlist：电商 / 工厂 / HSK / 日常 |
| Channel Trailer | 60 秒精选合集，钉首页 |

## 6. 与 TikTok 协同发布的最佳节奏

| 时机 | TikTok | YouTube Shorts |
|---|---|---|
| Day 0 11:00 | 发布 | – |
| Day 0 18:00 | – | 发布（去水印 + 换 BGM 版） |
| Day 1+ | 评论区运营 | 不主动运营评论，靠 SEO 长尾 |

> 之所以隔 7 小时：避免两个平台同 1 条视频在 24h 内同时出现导致算法判"非原创"。

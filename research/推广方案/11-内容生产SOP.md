# 11 · 内容生产 SOP（AI 文本 + 剪映）

> 1 人 1 天产 2 条双国短视频的标准化流程。

## 1. 一条视频的生命周期

```
选题(5min) → AI生成文案(10min) → 母语审校(30min异步) → TTS配音(5min)
   → 剪映成片(25min) → 字幕安全区检查(3min) → 多平台分发(15min)
   → 评论区运营(持续)

单视频纯工时 ≈ 60 min
```

## 2. 选题库管理（Notion）

**字段设计**：

| 字段 | 类型 | 备注 |
|------|------|------|
| 选题名 | Text | "我"字读错 |
| 国家 | Select | TH / VN |
| 公式 | Select | A 发音对比 / B 场景 / C 错读笑点 / D 声调 |
| 时长 | Number | 21 |
| 钩子句 | Text | 本地语 |
| 教学内容 | Text | 中文 + 拼音 + 本地翻译 |
| 标签 | Multi | 5 个 |
| 状态 | Select | 草稿 / 审校中 / 已审 / 已剪 / 已发 |
| 发布日期 | Date | |
| 实际播放 | Number | 发布后填 |
| 完播率 | Number | |
| 点赞/评论 | Number | |

**保持 ≥ 30 条预备库存**，避免选题断粮。

## 3. AI 文案 Prompt 模板

```
你是中文学习短视频脚本撰稿人。请按以下要求生成 1 条 TikTok 视频脚本：

【目标用户】22-32 岁泰国电商从业者，泰语母语，中文零基础。
【公式】发音对比反转
【时长】20-25 秒
【教学内容】中文"四" vs "十"
【输出格式】
- 钩子（0-3s，泰语）：
- 中段（4-18s，泰语+中文）：
  - 错读演示
  - 正读慢速 ×2
  - 口型提示
- 结尾（19-25s，泰语）：评论区互动钩
- 5 个标签（泰语 + 通用 + 挑战）
- 字幕分镜（每 2 秒一个字幕版）

注意：
- 泰语必须自然，避免直译；
- 不出现王室、宗教词；
- 不写"100%、保证"等绝对化表述；
- 钩子必须带"否定/数字/挑战"中至少 1 个。
```

## 4. 母语审校 SOP

**为什么必须**：
- AI 生成的越/泰语翻译错误率 5-15%；
- 错译被本地用户秒识别 → "这账号不行" → 永久失粉；
- 是整个推广方案中**唯一不能省的钱**。

**流程**：

1. 在 Notion 表里把 AI 文案标 "状态: 审校中"；
2. 通过共享视图发给兼职审校（不需开 Notion 账号，URL 即可）；
3. 审校在 Notion 评论区里改字 + 标"过/不过"；
4. 30 min 内回复（合同里约定 SLA）；
5. 不过的稿件 → 重写或弃用；
6. 通过 → 状态改 "已审"，进入剪辑环节。

**找人渠道**：
- 越南：在 Facebook 越南留学生群、TopCV 招聘网发兼职帖；
- 泰国：在 LINE OpenChat 中泰交流群、JobsDB 发兼职帖；
- 价格：¥30-50/条，每月预付 ¥1000-1500。

## 5. TTS 配音 SOP

**工具选择**：

| 工具 | 越南语 | 泰语 | 推荐 |
|------|--------|------|------|
| 剪映海外版（CapCut）内置 TTS | 一般 | 一般 | 备用 |
| Google Cloud TTS | 优秀（Wavenet）| 优秀 | **推荐** |
| Azure Neural TTS | 优秀 | 优秀 | 推荐 |
| ElevenLabs | 越/泰新支持，偏机械 | 偏机械 | 不推荐 |

**推荐**：Google Cloud TTS Wavenet 系列声音 + CapCut 备用。

**配置**：
- 语速：0.95-1.0 倍（不要太快）；
- 音色：女声偏好（教学场景亲和度高）；
- 中文部分：用一个固定的 Wavenet 中文女声，保持一致性 IP。

## 6. 剪映成片 SOP

**模板预置（5 套）**：

1. **A 模板（发音对比）**：人脸特写 + 大字幕 + 横向对比卡片
2. **B 模板（场景实用）**：场景背景图 + 顶部标题 + 5 句滚动卡片
3. **C 模板（错读笑点）**：人脸 + 表情包 + 反差字幕
4. **D 模板（声调对比）**：声调曲线动画 + 双语对比
5. **F 模板（30 天打卡）**：日历组件 + Day X 标记

**关键参数**：

| 项 | 设置 |
|----|------|
| 分辨率 | 1080×1920（9:16） |
| 帧率 | 30fps |
| 字幕安全区 | 上下各留 280px，左右各留 60px |
| 字幕字体 | 越：Be Vietnam Pro Bold；泰：Sarabun Bold |
| 字幕大小 | 主标题 80pt，正文 60pt |
| 字幕颜色 | 白字 + 黑色描边 4pt |
| 字幕位置 | 屏幕中央偏下 1/3 |
| BGM 来源 | TikTok CML + YT Audio Library |
| BGM 音量 | -18 dB（人声 -6dB） |
| 导出 | mp4 / H.264 / 高清 |
| **关闭** | CapCut 水印（导出时手动取消勾选） |

## 7. 多平台分发 SOP（15 min 流程）

**1 条原视频，4 个平台**：

| 步骤 | 平台 | 动作 |
|------|------|------|
| 1 | TikTok | 直接上传 → 写文案（带钩子+5 标签） → 选 CML 音乐 → 发布 |
| 2 | YouTube Shorts | 上传同视频 → 标题加 #Shorts + 关键词 → 描述放 App 链接 → 发布 |
| 3 | IG Reels | 上传同视频 → **重新加 IG 内置音乐**（关键，否则限流）→ 重写 caption → 发布 |
| 4 | FB Page | 选择是否同步 IG（建议**不同步**，分开发数据更好）|
| 5 | Threads | 把视频钩子句作为短文发出，配缩略图 |

**禁忌**：
- ❌ 用 SnapTik 等去水印工具（容易残留水印）→ 直接用剪映重新导出更干净；
- ❌ 多平台用完全相同的 caption（IG/TikTok 算法会判重复内容）；
- ❌ 同时发到所有平台（错开 30-60 分钟，避免被识别为机器人）。

## 8. 评论区运营 SOP

**3 大原则**：

1. **24h 黄金回复期**：发布后 1h、6h、24h 三次回评；
2. **本地语优先**：用泰语 / 越南语回，不用英语；
3. **作者置顶**：每条视频置顶 1 条评论，引导讨论方向。

**置顶模板**（泰语示例）：
```
อยากเรียนตัวต่อไป? คอมเมนต์บอกได้เลยค่ะ 👇
（想学下一个？评论区告诉我）
```

**高频评论回复模板**（提前准备 10 条）：

| 用户问 | 回复模板 |
|--------|---------|
| "怎么发音" | 发文字 + 拼音 + 表情，引导看 bio |
| "可以学完整课吗" | "เรามีคอร์สครบในแอป ลิงก์ใน bio ค่ะ" |
| "你是中国人吗" | "ใช่ค่ะ พูดไทยได้นิดหน่อย กำลังเรียนเหมือนกัน" |
| 错译纠正 | 立即承认 + 感谢 + 解释（提升好感） |
| 私聊请求 | 引导 LINE OpenChat（不引外部 IM） |

## 9. 数据复盘 SOP（每周日 1h）

**Notion 表追加**：实际播放 / 完播率 / 点赞率 / 评论率 / 收藏率 / 分享率 / 主页 CTR / bio 点击。

**复盘方法**：
1. 排序找出本周 Top 3 / Bottom 3；
2. 拆解差异：钩子？时长？BGM？标签？发布时间？
3. 输出 1 条"下周公式调整建议"；
4. 写入"账号 IP 公式 V X.0"文档。

| 指标 | 数据 |

| 人口 | ≈ 7000 万 |
| 互联网渗透率 | 88% |

**核心结论**：泰国是东南亚中文学习"基础最好"市场（中文已普及到中学），但竞品也最多；机会在 **18-30 岁"中文应用层"用户**（不是 K12 学生）。

## 2. 用户画像（4 类）

### 2.1 中泰电商 / 直播从业（占 30%）

- 22-35 岁，曼谷 / 清迈 / 春武里集中；
- 在 Lazada / Shopee / TikTok Shop 卖货；
- 月入 25k-80k THB（¥5k-16k）；
- 学习目的：和中国供应商沟通、看 1688 产品；
- **痛点**：现有 App 全英文讲解，泰语基础好但英语弱。

### 2.2 EV / 工厂员工（占 25%）

- 比亚迪、长城、奇瑞、宁德时代等在泰建厂，需要大量本地员工；
- 18-30 岁，男性偏多；
- 学习目的：升职、做"中泰协调"。

### 2.3 K12 / 大学学生（占 25%）

- 14-22 岁，泰国中学中文必修、孔子学院广布；
- 学习目的：学校考试、未来留学；
- 付费弱，但 UGC 强、易在 TikTok 起势。

### 2.4 旅游业从业 / 中文导游（占 20%）

- 25-50 岁，曼谷、芭提雅、普吉、清迈；
- 服务中国游客；
- 高度刚需场景化中文。

## 3. 泰国本地平台生态

### 3.1 TikTok

- 泰国是 TikTok 海外占比最高的国家之一（人均使用时长全球第一）；
- 16-25 岁日均时长 ≥ 100 min；
- **算法特征**：泰国用户偏好**更短的视频**（10-20s 完播率更高）+ 强反差幽默；
- **黄金发布时间**：12:00-13:00、19:00-22:00 泰国时间。

### 3.2 Line（私域核心）

| 项 | 说明 |
|----|------|
| 渗透 | 90%+ |
| OA 注册 | 任何人可免费创建 LINE OA（需 LINE 账号 + 邮箱） |
| 群形式 | LINE Group（最多 500 人）+ **LINE OpenChat（最多 5000 人，无需互加好友）** |
| 我方用法 | TikTok / FB → LINE OA → LINE OpenChat → App |

**LINE OpenChat 关键玩法**：
- 创建"30 วัน เรียนภาษาจีนกับเรา"（30 天和我们学中文）OpenChat；
- 入群无门槛，扫码 / 链接即入；
- 每日早 8 点机器人自动发"今日 1 句中文"；
- 周一打卡日，置顶"打卡截图大赛"；
- App 推广 fixed 在群描述。

### 3.3 Facebook

- 泰国 FB 仍是国民级（25-50 岁高度活跃）；
- FB Page + Group 路线在泰国比越南有效；
- **泰国本土做中文教育的 FB 活跃 Group 数十个**，可"先加入参与，再自建"。

### 3.4 YouTube

- 泰国 YT 使用率 96%（全球前 5）；
- 长视频权重高，"系统课"类视频订阅高；
- 中文教学品牌"Chinese Whisper"（泰国本土 KOL）即靠 YT 长视频做到 50 万订阅，**为我们提供了清晰路径**。

### 3.5 Instagram

- 渗透 ≈ 25%，城市白领 / 大学生；
- Reels 质量门槛中等。

## 4. 泰国本地化要点

### 4.1 语言

- 标题 / 字幕 / 口播全部泰语；
- 泰语有 5 个声调，**与中文 4 声有天然对照学习优势**（重要的差异化抓手）；
- 泰语**不分词、无空格**，字幕排版需注意自动换行；
- 字体推荐：Noto Sans Thai / Sarabun / Prompt（支持完整泰文字符 + 圆体亲和）。

### 4.2 文化（核心红线）

- ❌ **王室 / Lèse-majesté**：完全不出现国王、皇室、王冠、皇室色（黄）相关元素与字眼；
- ❌ **僧侣 / 佛像**：商业账号绝不与脚部、商品 Logo 同框；初期完全不做佛教相关词汇；
- ❌ 不做中泰政治相关；
- ⚠️ 性别 / LGBT：泰国对此宽容，但不要主动调侃；
- ✅ **欢迎**：
  - 食物（泰国美食 + 中国美食）；
  - 节日（泼水节 Songkran、水灯节 Loy Krathong、春节）；
  - 旅游、电商、Korean / Chinese drama 中文台词；
  - 中泰文化对照（饭桌礼仪、问候习惯）。

### 4.3 设计偏好

- 泰国用户偏好 **柔和 + 可爱风**（粉、蓝、薄荷绿）；
- emoji + Sticker 密集；
- 不要过度商务感、不要冷峻深色；
- 字幕字体偏圆润。

# 02 · 平台规则精读 · TikTok（最严）

> 适用账号：TikTok For You / TikTok Shop 创作者账号（非企业号、非广告主），运营所在地越南、泰国。
> 文档基准：TikTok Community Guidelines（2024 年 5 月生效新版）+ TikTok Branded Content Policy + Creator Code of Conduct + 越南/泰国本地分站规则。

## 1. 为什么 TikTok 必须放在"最严"

- 算法端：违规内容不一定立即下架，但会被进入"低分推荐池"（限流），创作者无感知。
- 越南/泰国分站：除全球规则外，受越南《网络安全法 2018》、泰国《计算机犯罪法》约束，**本地法律红线 + 平台规则双重叠加**。
- 0 粉新号容错率最低：违规 1 次的"小账号"几乎等同于直接判死刑（重新养号成本太高）。

## 2. 必须熟记的 6 大红线（直接下架/封号）

| # | 红线 | 我们的内容会触发的情境 | 规避做法 |
|---|------|----------------------|---------|
| 1 | **未成年人保护** | 用真人小孩入镜教学、镜头带未成年路人 | 一律不出现 18 岁以下真人面孔；街景拍摄打码 |
| 2 | **危险行为/医疗误导** | 教"中医词汇"如果出现配方、剂量 | 文化普及级别即可，不写"治什么病"、不写剂量 |
| 3 | **诈骗与误导性商业** | "学会赚钱"、"3 天精通中文"、"100% 通过 HSK" | 全部禁止，使用"系统学习"、"循序渐进"等中性词 |
| 4 | **仇恨言论 / 政治敏感** | 涉及越南-中国南海、台海、西藏新疆 | 内容只谈语言、生活、商业、文化，**不出现任何政治符号、地名争议**；连"中国地图"插图都不要用 |
| 5 | **平台外引流违规** | TikTok 内引导加 WhatsApp / Telegram / 微信 / 二维码 | 已在原方案里规避，**严格执行**：TikTok 站内只说"主页 link in bio"，不出现任何外部 IM 名字 |
| 6 | **垃圾内容/重复内容** | 同样的视频反复发，或不同账号发同样内容 | 矩阵号需做 ≥ 30% 内容差异；同号不发完全重复内容 |

## 3. 灰区规则（不会立刻封号，但会限流）

### 3.1 链接策略
- **新号 1000 粉以下不开放 bio 加链接** → 这是 TikTok 全球默认规则（部分地区放宽到 0 粉，但越南/泰国仍以 1000 为门槛）。
  - 应对：前期 bio 写"FB: ChineseEasyLearn"（**纯文本**，不带 https），用户自己搜。
- **不要在视频中念出网址**，会被识别为"导流违规"。

### 3.2 商业内容披露（Branded Content）
- 只要你内容里出现自家 App 名字、Logo、截图，TikTok 就视为"自有商业内容"，需要：
  - 视频上传时勾选 **"Promote your own brand/business"** → "Personal brand" 选项；
  - 不勾选 = 被判定隐性广告，第一次警告，第二次限流。
- **建议**：前 60 天完全不出现 App 名字和 Logo，养号纯做知识；60 天后才开始第 4 类导流内容。

### 3.3 音乐版权
- 商用音乐需用 **TikTok Commercial Music Library**（CML）里的曲目；
- 用户原创音乐流行曲目，**个人账号可以用**；但你账号一旦被识别为"商业账号"或绑定 TikTok Shop，所有非 CML 音乐立即静音。
- **建议**：从第 1 条视频起就只用 CML 音乐，避免后期改账号类型时全部内容失声。

### 3.4 字幕与水印
- 视频左上角 / 右下角是 TikTok UI 必占区域，字幕放这里 = 被裁剪，被算法识别为"低质"。
- **字幕安全区**：屏幕上下各留 15%，左右各留 5%。
- 不要带 CapCut/剪映 Logo 水印（剪映导出时手动关闭）→ 带其他平台水印会被显著限流。

### 3.5 标签数量
- 官方建议 3-5 个，超过 8 个被判垃圾内容。
- 每条视频组合 = **1 个赛道大标签 + 2 个本地语标签 + 1 个场景标签 + 1 个挑战/热点标签**（详见 06）。

## 4. 越南/泰国本地法律红线（平台之外的额外约束）

### 4.1 越南
- **《网络安全法 2018》第 8 条**：禁止"歪曲、否定政府成就""破坏民族团结""宣传错误信息"；语言学习视频中**不要出现越南国旗 + 越南地图 + 政治符号同框**，避免被举报后人工审核误判。
- **越南文化部 38/2021 号通知**：外语教学视频应"尊重越南语言纯洁性"，避免在视频里说"越南语没有这个表达"等贬低性陈述。
- 替代说法：用"中文里的这个表达，对应越南语的 X，在使用上更…" 中性表达。

### 4.2 泰国
- **冒犯王室罪 (Lèse-majesté)**：泰国《刑法》112 条，**永远不要在内容中出现**：
  - 国王/王室成员的图片、名字、头衔；
  - "泰国国王"、"皇室"等字眼用中文教学（无论褒贬）；
  - 泰国国徽、王室徽章。
- **冒犯佛教**：佛像不能与脚部、宠物、商业 Logo 同框；不要教"佛、和尚、寺庙"相关词汇时使用真实佛像图片，改用插画或手绘。
- **泰国《计算机犯罪法》14 条**：禁止"上传可能损害国家安全或引起公众恐慌的虚假信息"，HSK / 移民 / 签证类内容必须标注"参考官方最新政策"。

## 5. 账号注册与运营硬性合规

| 项 | 要求 |
|----|------|
| 账号主体 | 个人账号即可启动；后期 1k 粉后建议升级为 **Business Account**（免费），可看数据分析、可加 1 条 bio link |
| 注册地 IP | **必须用目标国当地 IP**（越南账号用越南节点，泰国账号用泰国节点），混用 IP 会被判异常 |
| 设备 | 1 台手机 1 个账号，不要在同一台手机切换多账号（会被判矩阵作弊） |
| 实名 | 个人账号无需实名；后期商业化要求 KYC |
| 年龄 | 注册时填 ≥ 18 岁 |
| 名称 | 不得包含 "official"、"verified"、品牌词冒充；可用"Chinese Easy Learn TH" 类组合 |

## 6. 内容审核流程与申诉

- 视频上传 → AI 预审（30 秒内）→ 流量池放量（200-500 播放）→ 互动数据触发人工抽审（约 5% 概率）→ 最终判定。
- **被限流信号**：连续 3 条视频播放量 < 200 且发布时间 > 24h，大概率被进入"低质池"。
  - 应对：停发 24-48h，分析最近内容是否触红线；不要急着删，删除会被叠加判定。
- **下架申诉**：进入 App → Profile → Settings → Report a problem → 选择具体视频，文字申诉用英文 + 当地语言双语。

## 8. 与原方案的差异

| 原方案做法 | 校验后调整 |
|----------|----------|
| "TikTok 不提 APP 不硬推链接" | ✅ 正确，进一步明确"前 60 天连 App Logo 都不出现" |
| "评论区主动去竞品账号留言引流" | ⚠️ 风险：被识别为骚扰评论会限流自己的号；改为"在竞品评论区做有价值的回答（不挂账号），让用户自己点头像" |
| 第四类"APP 预告/福利"内容 | ⚠️ 在 TikTok 上**只能用第 4 类的"展示功能"形态**，且必须勾选 Branded Content；推荐放到 YouTube Shorts 和 IG 上 |
| 标签 #LearnChinese 等 | ✅ 正确，但需补充 1 个**挑战式标签**（如 #30DayChineseChallenge）建立账号专属 IP |

## 6. 哑火诊断决策树

```
单条视频 < 500 播放 × 5 条
   ├─ 钩子前 3 秒太弱 → 换钩子模板
   ├─ 主题不匹配 → 调整主题权重
   ├─ 母语翻译错 → 更换审校
   └─ 标签错 → 用 02 第 4 节标签库重做

单条爆款 (>5 万)
   ├─ 24h 内回完所有评论
   ├─ 置顶引流："想要合集 → 评论 1"
   ├─ 同公式立即再产 3 条相似选题
   ├─ Bio 改为对应主题的 Zalo / Line 入群链接
   └─ 同主题 Zalo / Line 当天群发"今日爆款合集"

Zalo / Line 关注掉粉 > 5%/周
   ├─ 检查是否日推太多 / 太硬广 → 减频次
   ├─ 检查欢迎序列是否触发了反感词
   └─ 增加福利（Sticker / 试用券）拉留存
```

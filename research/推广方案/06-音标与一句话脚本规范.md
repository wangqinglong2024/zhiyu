# 06 · 音标与一句话脚本规范

> 课程与短视频共用的内容生产标准。所有"一句话"知识点出炉时必须符合本规范。

---

## 1. 拼音 / IPA 标注规范

### 1.1 拼音
- 统一使用 **GB/T 16159-2012 汉语拼音正词法基本规则**
- 声调：用调号 ā á ǎ à ;**不用** 数字 a1 a2
- 多音字：随文标音（如"行 háng / xíng"分别标注）
- 儿化音：原字 + r（如"花儿 huār"）
- 词组分写规则：双音节词不分写（pīnyīn 不写 pīn yīn），三音节及以上按结构断（jiùshēngyī = 救生衣 单写）

### 1.2 IPA（仅给 HSK 7+ 与语音学家用）
- 普通版课程不展示 IPA
- HSK 高阶轨可在每节"语音附录"中展示 IPA，便于母语为泰/越的学习者对照

### 1.3 越南语 / 泰语翻译规范
- 越南语：使用越南国家语言院（Viện Ngôn Ngữ Học）权威词典写法，加声调
- 泰语：分词使用空格 + Royal Institute Romanization 罗马转写（仅 HSK 7+ 显示）

## 3. 一句话脚本五段式（30 秒 / 一条视频）

```
段 1（0-3s）钩子           ：母语提问 / 反差 / 数字 / 痛点
段 2（3-8s）一句话呈现     ：中文大字 + 拼音 + 母语翻译 一起出
段 3（8-18s）拆解          ：1 个关键字字源 / 1 个语法点 / 1 个文化点（三选一）
段 4（18-26s）跟读          ：放慢 1 次 + 正常 1 次 + 用户跟读窗口（留 3 秒）
段 5（26-30s）行动召唤     ：今日打卡 / 评论扣"X" / 加 Zalo/Line 拿合集
```

## 6. 母语审校强制流程

任何一句话上线前必须：
1. 中文 → AI 初翻（DeepL / GPT）
2. 越/泰母语兼职审校（Fiverr / 在校生，¥3-8 / 句）
3. 上线后 7 天内首条评论若指出翻译错误，48h 内重剪重发

> 一句话错译 = 全条视频报废。审校预算每月 ¥1500-3000，比任何投流都更值。

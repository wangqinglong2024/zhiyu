# 10 · 最终推广执行方案（核心交付物）

> 这一份是落地说明书。看完这一份，1-2 人团队从明天起就能动手。

## 0. 核心结论速记卡

```
首发市场顺序：泰国（D1） → 越南（D8） → 印尼（D90 后）
首发平台组合：TikTok + YouTube Shorts + IG Reels + FB Page/Group + LINE/Zalo
首发账号数：每市场 1 个矩阵号（合计 2 个矩阵：TH-pack、VN-pack）
内容形态：85% 短视频（21-34s）+ 15% 图文/长文章
冷启动周期：90 天最低，180 天稳定，禁止中途换号或弃号
预算：前 90 天 ¥6000-12000（人工不算），全部花在母语审校
人力：1-1.5 人当量（可外包母语审校）
```

## 3. 内容形态决策（视频 vs 图文）

| 平台 | 视频:图文 | 理由 |
|------|----------|------|
| TikTok | 100:0 | 没图文 |
| YouTube | 100:0 | Shorts + 长视频 |
| IG | 80:20 | Reels 主，单图卡片做"知识沉淀"|
| FB Page | 75:25 | Reels 主，图文做精修 |
| FB Group | 0:100（图文+UGC） | 群里做对话 |
| LINE / Zalo | 30:70 | 推送图文为主，偶尔视频 |
| 独立站 | 0:100（长文）| SEO |

**结论**：**主要精力做视频，图文是视频的补充和私域承载。**

## 4. 0 粉冷启动周期管理（关键）

```
Day 1 ─────────── Day 30 ─────────── Day 60 ─────────── Day 90 ─────────── Day 180
│                    │                    │                    │                    │
养号期               爆款搜索期            爆款放大期            私域+App期            稳定增长期
│                    │                    │                    │                    │
▼                    ▼                    ▼                    ▼                    ▼
- 公式确定          - 第 1 个破万         - 单平台 1k+粉      - App 上线引导      - 每月 1 万下载
- 0-200 播放        - 评论区运营           - 复用公式          - 群超 500            - 月经常性 ARR
- 不要换号           - 数据复盘           - 跨平台同步         - 付费功能开放
- 不要付费          - 不要焦虑           - 维持频率           - 仍不付费投放
```

**禁忌（任一发生 = 90 天努力归零）**：

- ❌ 30 天没爆就换号（重新养号至少 60 天起步）
- ❌ 30 天没爆就开始付费投流（AI 算法被打乱，自然推荐池权重直接掉）
- ❌ 30 天没爆就改方向（公式不固定 → 算法不知道推给谁）
- ❌ 同一手机切换多账号（被识别矩阵作弊）
- ❌ 删除老视频（被叠加判低质）
- ❌ 不回复评论（爆款必须 24h 内回 ≥ 50 条）


## 9. 何时考虑付费投放（远期参考）

**触发条件（全部满足）**：
1. 自然流量已稳定，月新增粉丝 ≥ 5k；
2. App 自然下载已稳定 ≥ 1k/月；
3. App 7 日留存 ≥ 30%；
4. 已识别 ≥ 3 条"自然流量爆款"视频，准备复用为 Spark Ad。

**首次投放策略**：
- 平台：TikTok Spark Ads（用自然爆款视频投放，CTR 是新素材的 3-5 倍）；
- 预算：$300/月起，单 Spark Ad 测试 $50/条；
- 目标：App Install + Cost per Install < $1.5。

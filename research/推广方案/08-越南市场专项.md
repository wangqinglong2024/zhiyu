# 08 · 越南市场专项

## 1. 市场体量与机会

| 指标 | 数据 |
|------|------|
| 人口 | ≈ 1 亿，65% 在 30 岁以下 |
| 互联网渗透率 | 79%（约 7800 万网民） |
| 智能手机用户 | ≈ 7000 万 |
| 在越中资企业 | ≥ 4000 家（深圳-广东系电子代工大量迁越） |
| 中国-越南双边贸易 | 2024 年 ≈ $2400 亿，连续 25 年增长 |
| 中文学习人口（含潜在）| ≈ 600 万（在职 + 学生 + 商务） |
| HSK 报考人数 | 2024 年 ≈ 18 万人，全球第 3 |
| 中文培训市场规模 | ≈ $5 亿 / 年，CAGR 18% |

**核心结论**：刚需明确，且 **80% 学习者目的是"工作 / 生意"**（不是兴趣），与我们 App 定位完全匹配。

## 2. 用户画像（3 类核心）

### 2.1 工厂打工型（占 35%）

- 18-32 岁，男性偏多；
- 在中资工厂（电子、纺织、家具、太阳能）做技术工 / 班长 / 翻译助理；
- 月薪 8M-25M VND（约 ¥2400-7500）；
- 学习目的：与中国主管沟通、晋升、做"中越翻译"加薪；
- 学习场景：工厂宿舍、上下班大巴；
- 设备：Android 占 90%（OPPO / Vivo / Samsung）；
- **核心痛点**：英文不好，市面上的中文 App 全是英文界面 → **我们的母语化是直击痛点**。

### 2.2 跨境电商型（占 30%）

- 22-35 岁，女性偏多；
- 在 Shopee / Lazada / TikTok Shop 做卖家、客服、选品；
- 月收入 15M-50M VND；
- 学习目的：和中国 1688/广州档口供应商沟通、看产品页；
- 设备：iPhone + Android 各半；
- **核心痛点**：直播带货时听不懂中文供应商口语、订货时看不懂中文规格表。

### 2.3 学生 / HSK 备考型（占 25%）

- 16-24 岁；
- 河内国家大学、胡志明国家大学、河内外贸大学等；
- 学习目的：HSK 4-6、奖学金赴中、毕业找中资工作；
- 付费意愿：低，但活跃度高（适合做 UGC 引擎）。

### 2.4 其他（10%）

- 兴趣型（看中剧、追星）。

## 3. 越南本地平台生态

### 3.1 TikTok（主战场）

- 越南是 TikTok 全球前 5 大市场；
- 16-30 岁日均使用 ≥ 90 min；
- **算法特征**：越南用户对"教学反差/笑点"内容比泰国用户更敏感，喜欢稍长视频（25-40s）；
- **黄金发布时间**：12:00-13:30、20:00-22:30 越南时间。

### 3.2 Zalo（私域核心）

| 项 | 说明 |
|----|------|
| 渗透 | 90%+ 越南人在用 |
| 开放方式 | Zalo Official Account（OA）免费注册（需越南手机号） |
| 群类型 | Zalo Group（最多 1000 人/群） |
| 优势 | 国民级，几乎所有目标用户都在 |
| 我方用法 | TikTok / FB → Zalo OA / Group → App |

**Zalo OA 申请流程**：
1. 越南手机号 + 身份证（找当地兼职 / 合作伙伴代办）；
2. 提交"教育类"OA 申请（免费，但有人工审核 7-15 天）；
3. 通过后获得专属 OA ID（如 `oa.zalo.me/chineseasylearn_vn`）。

**Zalo 内容策略**：
- 每日 1 条"今日 1 句中文"（图文，越南语 + 中文 + 拼音）；
- 每周 1 次"打卡日"，群里发自己学习成果；
- App 链接固定挂在 OA 简介。

### 3.3 Facebook + Group

- 越南 16-30 岁 FB 时长持续下降，但 25-40 岁仍重度使用；
- **重点**：FB Group "Học tiếng Trung miễn phí"（免费学中文）类已有 50+ 个 5万-50万 成员的群，**可以加入参与而不需自建早期就大群**；
- 我方建群：从"30 ngày học tiếng Trung cùng Mai"（30 天和 Mai 一起学中文）这种"个人化"角度建立。

### 3.4 YouTube

- 越南网民 YouTube 使用率 88%；
- 长视频偏好"全集合订版"（如"100 句越南人最常用中文"）；
- 视频前 30 秒决定订阅率，必须有越南语开场。

### 3.5 Instagram

- 渗透率 12-15%，集中在 18-28 岁城市女性；
- Reels 质量门槛高于 TikTok；
- 重点用户是"电商 / 自由职业 / 留学倾向"群体。

## 4. 越南本地化要点

### 4.1 语言

- 标题、字幕、口播全部使用越南语；
- 越南语有 6 个声调，**绝不要用 Google 翻译直接出**：
  - 翻译错位常见词："学" "工作" "客户" 在不同语境用词不同；
  - 必须找母语审校（参见 11-SOP）。
- 越南语字母带音调符号（如 ă, â, ê, ơ, ư, đ + 6 调），**视频字幕字体必须支持完整越南语字符**（推荐 Noto Sans Vietnamese / Be Vietnam Pro）。

### 4.2 文化

- 越南人对"中越关系"敏感度有，但对**经济合作 / 学习中文找工作**接受度极高；
- **避开**：
  - 南海、领土、政治议题；
  - 任何涉及越南国旗 + 中国国旗同框且无适当语境的画面；
  - 历史话题（特别是 1979 年战争）；
- **欢迎**：
  - "学中文涨工资"、"和中国老板沟通"、"做跨境电商" 这种实用导向；
  - 中越美食、电商、节日（春节越南也过春节 Tết）。

### 4.3 设计偏好

- 越南用户偏好 **明亮配色**（红、黄、橙），不偏冷色；
- 喜欢 emoji 密集 + 字幕动画；
- 不要过度商务感的设计。

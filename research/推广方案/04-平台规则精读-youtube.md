# 04 · 平台规则精读 · YouTube（最宽松）

> 基准：YouTube Community Guidelines + Advertiser-Friendly Content Guidelines + YouTube Shorts Policies + YPP（YouTube Partner Program）规则。

## 1. 为什么 YouTube 是"最宽松"

- 外链最自由：视频描述、卡片、Pinned Comment 均可放 App Store 直链 + 官网；
- 强搜索属性：1 条优质长视频可被搜索 3-5 年（前 3 平台都没有这种长尾）；
- Shorts 算法独立、对 0 粉新号友好；
- 不会因"硬广"限流（与 Meta 不同），但会因低质内容被取消变现资格。

## 2. 红线（必须避免）

| # | 红线 | 触发情境 | 规避 |
|---|------|---------|------|
| 1 | 误导性元数据 | 标题写"100% 通过 HSK"，内容只是 5 句话 | 标题与内容必须一致 |
| 2 | 重复 / 自动化内容 | 用 AI 全自动生成大量低质短视频 | 必须有原创讲解、字幕、人声（即使是 AI 配音） |
| 3 | 仇恨言论 / 骚扰 | 同前 | 同前 |
| 4 | 儿童安全 | 视频涉及未成年人需额外标注"Made for Kids" | 不出现未成年人 |
| 5 | 危险/违法行为 | 教"如何无证打工""如何避税" | 教学场景仅限"日常会话、工作沟通"，不涉灰产 |
| 6 | 版权侵权 | 用电视剧片段 / 流行歌曲做背景 | 用 YT Audio Library 免版税音乐 + Pexels/Pixabay 视频素材 |
| 7 | 垃圾元数据 | 标签塞 30 个无关标签 | 5-10 个相关标签 |

## 3. Shorts 政策细节

| 项 | 规则 |
|----|------|
| 时长 | ≤ 60 秒（2024 起放宽到 ≤ 3 min，但算法仍偏好 ≤ 60s） |
| 比例 | 9:16 竖屏 |
| 标签 | 标题 / 描述加 `#Shorts`（强烈建议）+ 5 个语言学习标签 |
| 与 TikTok 复用 | 允许复用，但**视频里不能有 TikTok 水印**，否则 Shorts 算法降权 80% |
| 音乐 | YouTube Audio Library + Shorts Sound Library（自带的 TikTok-like 流行音乐池） |
| 描述 | 可放 1-2 个外链（App Store / 官网） |
| Bio Link | Channel Description 可放任意外链；2024 起 **0 粉就能加链接** |

## 4. 长视频政策细节（5-10 min）

| 项 | 规则 |
|----|------|
| 标题 | ≤ 70 字符；前 50 字符是 SEO 关键词 |
| 描述 | 前 150 字符是 SEO 摘要 + 关键词；中段加 chapters 时间戳 + 外链 |
| 字幕 | 必须上传 SRT（双语字幕：中文 + 越/泰），SEO 价值巨大 |
| Tags | 5-10 个，包括："learn chinese in vietnamese / thai"、"中文 教学"、本地词 |
| 缩略图 | 1280x720，要有大字 + 1 个人脸，CTR > 6% 才能起量 |
| End Screen | 引导订阅 + 引导下一条视频 |
| Cards | 中段插入"播放列表"卡片，提升观看时长 |
| Playlist | 同主题做 Playlist，YouTube SEO 加分 |

## 5. 变现规则（远期参考）

YPP 入门：
- Shorts 路径：90 天内 Shorts 总观看 ≥ 1,000 万 + 1k 订阅；
- 长视频路径：12 个月内观看时长 ≥ 4,000 小时 + 1k 订阅。

我方策略：
- **不为变现而做内容**，前 6 个月聚焦于"导流到 App"，YouTube 变现是 Bonus。
- 1k 订阅后开放 Channel Memberships，可作为"App 高级会员"的引导入口。

## 6. 越南 / 泰国本地化

- 标题、描述、字幕**必须有当地语版本**（不是仅英文），YouTube 越南/泰国搜索引擎权重远高于英文标题。
- 视频地区设置 → 选择 Vietnam / Thailand，提升本地推荐池权重。
- 同样规避王室、宗教、政治敏感（与 02-TikTok 同）。

## 7. 我方运营 Checklist

- [ ] Channel 名称包含 "Chinese for Vietnamese / Thai" 关键词
- [ ] Channel Description 含 5 个本地语关键词 + App 链接
- [ ] 频道头像 + Banner 含双语品牌词
- [ ] 每条 Shorts 加 #Shorts + 5 个标签
- [ ] 每条 Shorts 描述放 1 个 App 链接
- [ ] 每条长视频上传双语 SRT 字幕
- [ ] 每条长视频缩略图 ≥ 6% CTR（A/B 测）
- [ ] 同主题视频归入 Playlist
- [ ] 视频地区设为 VN / TH
- [ ] 不使用流行歌曲、电视剧片段

## 8. 与原方案的差异

| 原方案做法 | 校验后调整 |
|----------|----------|
| "每周 2 条 5-10 min 长视频" | ⚠️ 前 90 天太重，改为"每周 7 条 Shorts（搬 TikTok）+ 0 条长视频"；90 天后每月 2 条长视频 |
| "YouTube 简介可放 App 链接 + 话术明确" | ✅ 完全正确，YT 是 3 平台中**唯一可以在视频里念 App 名字的** |
| "搜索关键词优化" | ✅ 正确，但补充：必须用本地语关键词 + 双语字幕 |
| "私信自动回复" | YT 没有原生 DM；用 Channel Description 留 Email 即可 |

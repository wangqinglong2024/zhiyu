# 5.1 · 执行摘要

## 一、产品

**知语 (Zhiyu)** 是面向东南亚（越南 / 泰国 / 印尼优先）的 AI 驱动中文学习平台，以**学习者母语**作为讲解语言，把"发现中国 / 系统课程 / 游戏专区 / 小说专区"四大模块整合为一个性价比最高的中文学习闭环。

## 二、问题

- 头部产品（HelloChinese / Duolingo / SuperChinese）100% 用英语讲解，**东南亚学习者母语缺位**
- HSK 1-6 后断档，**HSK 7-9 / 商务 / 工厂 / 电商 场景几乎空白**
- 现有产品定价 $7-15/月，**对东南亚月薪 $300-700 的用户偏贵**
- 学习路径碎片化：单 App 只覆盖一个维度，**用户在多 App 间疲于切换**

## 三、解决方案

| 差异点 | 知语 | 头部竞品 |
|---|---|---|
| 讲解语言 | 越/泰/印尼母语原生 | 仅英文 |
| 课程深度 | HSK 1-9 + 商务 + 工厂 + 日常 4 轨道 × 12 阶段 | HSK 1-6 / 单一轨道 |
| 模块完整 | 文化 + 课程 + 游戏 + 小说 | 仅 1-2 个模块 |
| 定价 | $4/月、$40/年 | $10-15/月、$80-100/年 |
| 学习算法 | FSRS-5 + 多模态题型 + 游戏化 SRS | SM-2 / 简单复习 |
| 经济激励 | 知语币 + 永久二级分销 | 无 |

## 四、市场

- TAM（全球数字化中文学习）：2025E **$7.2 亿**，2028E **$30 亿+**
- SAM（东南亚已验证付费市场）：**$1500-1800 万 ARR**
- SOM（3 年保守目标）：**$350 万 ARR / 9 万付费用户**

## 五、定价与单位经济

- 月会员 **$4** / 半年 **$12（首发促销）** / 年 **$40**
- 单段课程 **$4** / 完整 9 段 **$36**
- 加权 ARPU ≈ **$28-35/年**
- LTV ≈ **$32**（保守，18 月生命周期 × 75% 毛利）
- 目标 CAC < **$10**（LTV/CAC ≥ 3）

## 六、商业模式

- B2C 订阅 + 单段付费（主）
- 知语币兑换会员 / 皮肤（留存）
- 永久二级分销（K-factor 增长）
- B2B 工厂 / 留学中介团购（M+6 启动）

## 七、技术栈

- **前端**：Vite + React 18 + TS + Tailwind v4 + shadcn/ui + PWA
- **后端**：Express + TS + Supabase Postgres / Auth / Storage / Edge
- **AI**：LangGraph (TS) 工作流 + Claude / DeepSeek 双供应商 + Vercel AI SDK
- **TTS**：Azure Speech / ElevenLabs
- **支付**：Paddle MoR + LemonSqueezy 备援
- **CDN/WAF**：Cloudflare（含 Turnstile 反爬）
- **部署**：单服务器 三环境 (dev/stg/prod) → M+6 拆分 prod

## 八、团队与资源

- 核心 3 人：CEO（产品+UX）/ CTO（架构+全栈）/ AI 工程师（内容工厂+LangGraph）
- 外包：母语审校（越/泰/印尼 各 1-2 名）、设计、法务
- 预算：自筹 $30k-50k（M+0 到 M+6）

## 九、12 月里程碑

| 时点 | 里程碑 | 付费 / ARR |
|---|---|:---:|
| W0（M+0） | 软上线（越南 + 部分课程） | - |
| M+3 | PMF 验证（200 付费） | $7K |
| M+6 | 越南 + 泰国全开（1500 付费） | $52K |
| M+12 | 三国稳定运行（8000 付费） | **$290K ARR** |

## 十、关键风险（Top 3）

1. **内容审核单点失败**（一国封禁） → 三道筛 + 应急 PR + 多市场分散
2. **Paddle 拒绝准入** → LemonSqueezy 备援
3. **冷启动获客成本超支** → 三平台并行 + SEO 长期 + 分销裂变

## 十一、核心战略洞察

> **AI 内容生产红利的 12-24 个月窗口期 + 头部竞品母语本土化空白 + 东南亚中文学习政策风口 = 知语此时入场是 PMF 概率最高的时点。**

## 2. 风险 1 · 平台封号 / 限流

### 预防
- 注册环境干净：本地 SIM + 本地 IP + 1 设备 1 账号
- 严格按 02 / 03 文档红线
- 不批量操作（不同账号不要 30 分钟内同动作）
- 不引导跨平台（视频内不口播"下载"，bio 跳转必须经 Linktree）

### 检测
- 周 KPI 看板异常（单条 < 100 播放连续 5 条）
- 收到平台警告邮件
- 后台 Strike / 黄标增加

### 应急
| 等级 | 现象 | 处置 |
|---|---|---|
| 黄牌 | 1 strike / 单视频被限 | 撤下涉事视频，账号继续运营但 7 天内不发同主题 |
| 红牌 | 2-3 strikes / 永久封号 | 提交申诉；同时启用备用账号继续 |
| 全军覆没 | 双国账号同时被封 | 暂停公域 30 天，全力运营私域 + 独立站 |

> **备用账号策略**：开始即注册 2 个账号 / 平台 / 市场（共 8 个），主号运营，备号每月发 1 条养号。

## 3. 风险 2 · 母语翻译错误

### 预防
- 强制审校流程：AI 初翻 → 母语真人审校 → 审校签名存档
- 审校员考核：错译 1 处扣 ¥50，月度 ≥ 3 处更换

### 检测
- 评论区监控：上线 24h 内首条出现"翻译错"/"không đúng"/"แปลผิด" 类评论
- 每周回看 Top10 视频评论关键词

### 应急
- 48h 内重剪发布
- 公开置顶歉意评论 + 致谢指出错误的用户
- 该错译知识点回炉重审

---

## 4. 风险 3 · 版权纠纷

### 预防
- BGM：仅用 TikTok 商用库 + YouTube Audio Library + Epidemic Sound（订阅）
- 影视引用：禁直接剪原片，必须二创（旁白叠加 + 自录场景）
- 字体：思源 / 思源宋体 / Roboto / Sarabun（开源商用免费）
- 图片：Unsplash / Pexels / 自摄

### 检测
- YouTube Studio Copyright 页周巡
- TikTok 后台 Notification

### 应急
- Content ID 命中：立即换 BGM + 重新发布
- 收到 DMCA 投诉：撤下 + 反向溯源（如审校 / 设计师未告知版权来源，追责）
- 商标 / 肖像权 投诉：撤下 + 公开致歉

## 5. 风险 4 · 政治 / 文化敏感事故

### 预防
- 内容上线 3 道关：
  1. 创作者按 02 / 03 / 04 / 05 红线表自查
  2. 母语审校第二关
  3. 主管发布前最后 1 关
- 红线词清单维护（季度更新）：
  - 越南：红黄旗 / 党政人物 / 1979 / 中越战争 / 西沙南沙
  - 泰国：王室 / 黄衫红衫 / 佛像与脚 / 国王名讳
  - 通用：两岸 / 港 / 藏 / 疆 / 八九 / 任何宗教讽刺

### 检测
- 监控本地大型本土媒体对账号是否点名
- 监控本地政府文化部门公告

### 应急
- 一旦触发：1 小时内全平台撤下涉事内容
- 24 小时内本地母语公开声明（由审校员代笔）
- 重大事故 → 暂停发布 7 天 + 法律顾问介入
- 永远不翻旧账：删除即从所有平台彻底删除，不存档不二次讨论

## 6. 风险 5 · 私域被举报

### 预防
- Zalo / Line 群规明确（首次进群必读）
- 不发外站直链超 3 次/周
- 不强制拉群
- 群内禁政治 / 宗教 / 黄赌毒

### 检测
- 单日掉粉 > 5%
- 群组消息发送失败率上升
- OA 后台 Notification

### 应急
- 单群被封：立即建备用群 + 通过 OA 私聊邀请
- OA 被警告：1 周内不群发 + 检查近 30 条推送
- OA 被封：用备用 OA 接管（开始就要备 2 个）

---

## 7. 风险 6 · 数据合规

### 预防
- 越南 PDPL 2023：收集手机号 / 邮箱必须告知用途，存储不超 13 个月（除非用户授权）
- 泰国 PDPA：同上，并须设置数据删除请求通道
- 隐私政策页（独立站）：双语版本，明示数据流向（Zalo / Line / App）

### 检测
- 月度数据存储清理（删除 13 个月以上未活跃用户的非必要字段）
- 用户删除请求处理 SLA：30 天内完成

### 应急
- 数据泄露：72h 内向用户告知 + 向当地数据保护机构报告
- 合规审查：聘请本地律师做季度审查（¥3000-5000 / 季）

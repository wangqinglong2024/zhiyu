# 海外推广方案 · 总索引（v2.0 · 双平台 + 四主题）

> **方向重塑**：仅保留 TikTok + YouTube 两个公域流量主战场；私域沉淀仅保留 Zalo（越南）+ Line（泰国）。
> **内容主题**：所有短视频与课程均围绕 **电商 / 工厂 / HSK / 日常** 四大类目（与 `/course` 四轨道一一对应）。
> **生产形态**：一句话模式 — 每条视频 = 一句中文 / 一个词 + 母语翻译 + 30 秒拆解，对应课程一个知识点。
> **视频与课程一体化**：所有公域视频均由平台自录，脚本与画面都是课程内部同步产物，**不独立策划视频**。
> **不依赖 AI 语音识别（ASR）**：考核全部采用静默题型（选择 / 填空 / 听力 / 翻译 / 排序 / 造句），跟读仅用于示范与静默练习。

## 四、本轮重塑相对 v1.0 的变化

| 项 | v1.0 | v2.0（本次） |
|---|---|---|
| 公域平台 | TikTok + YouTube + IG Reels + FB Page/Group | **仅 TikTok + YouTube Shorts** |
| 私域平台 | Zalo + Line + FB Group + LINE OpenChat | **仅 Zalo OA（越）+ Line OA（泰）** |
| 内容主题 | 笼统"母语化中文学习" | **电商 / 工厂 / HSK / 日常 四主题轮播** |
| 与课程关系 | 弱关联 | **每条视频 = 一个课程知识点（一一对应）** |
| 视频脚本 | 自由发挥 | **一句话五段式标准模板**（见 06） |
| 首发市场 | 泰 → 越 → 印 | **越（D1） + 泰（D8）双国并行**（与 Zalo/Line 强绑定） |

> v1.0 砍掉的理由速记：
> - Meta（FB/IG）：东南亚 16-30 人群活跃度持续下滑；Reels 算法对小账号语言类不友好；FB 外链限制重；运营 2 个 Meta 账号 = 多 30% 工时但带不来等比流量。
> - 多平台分发 = 注意力稀释；2 平台聚焦 + 私域沉淀，单条视频 ROI 反而更高。

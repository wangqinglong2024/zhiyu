# 2.3 · 政策与合规

> **声明**：以下信息基于 2024-2025 年公开法律框架的整理与外推，落地前必须由当地律师复核。

## 一、数据保护法律对照

### 1.1 越南 PDPL（Decree 13/2023/ND-CP，2023.7.1 生效）
- 范围：处理越南公民个人数据的境内外组织
- 关键义务：
  - 数据处理影响评估 (DPIA)
  - 跨境数据传输备案 / 影响评估
  - 90 天内书面通知数据保护主管部门
  - 用户同意必须显式、可撤回
- 处罚：最高营业额 5%
- **知语对策**：
  - 用户隐私协议越南语版（必须）
  - 注册时显式 Cookie / 数据使用同意
  - Supabase 新加坡区，跨境传输备案
  - 用户"清空进度"按钮 = 数据删除权落地（已规划）

### 1.2 泰国 PDPA（2022.6.1 全面生效）
- 范围：处理泰国境内主体数据的组织
- 关键义务：
  - 同意必须自由、明确、知情、可撤回
  - 数据处理目的限定
  - 数据主体可访问 / 更正 / 删除 / 拒绝营销
  - DPO（数据保护官）：处理大量数据的组织必须任命
- 处罚：最高 500 万泰铢 + 民事赔偿
- **知语对策**：
  - 泰语版隐私协议
  - 后台具备"导出我的数据"按钮（v1.5 实现）
  - 营销邮件 / Push 一键退订
  - 知语规模到 10 万 MAU 时设 DPO

### 1.3 印尼 UU PDP（Law 27/2022，2024.10 全面执行）
- 范围：处理印尼公民数据的组织
- 关键义务：与 GDPR 高度对齐
  - 数据处理 6 大合法基础（同意 / 合同 / 法定 / 正当利益 / 关键利益 / 公共任务）
  - 跨境传输需"等同保护"
  - 用户 8 项权利
  - 数据泄露 72 小时内通知主管机构 + 用户
- 处罚：最高 2% 营业额 + 刑事
- **知语对策**：
  - 印尼语版协议
  - 跨境传输：Supabase 新加坡区 → 印尼用户传输需在协议中明示
  - 数据泄露应急预案

### 1.4 GDPR（欧盟）兜底
- 即使知语不直接进入欧盟，也建议合规：
  - 全球用户访问产品时自动适配（GeoIP）
  - 默认隐私优先

### 1.5 数据存储策略汇总

| 数据类型 | 存储位置 | 跨境处理 |
|---|---|---|
| 用户账号 / Auth | Supabase 新加坡 | 加密 + 跨境备案 |
| 学习进度 | Supabase 新加坡 | 同上 |
| 支付记录 | Paddle（爱尔兰 + 全球） | Paddle 已自有合规体系 |
| 内容数据（公共） | Supabase + CDN（Cloudflare） | 不涉及个人数据 |
| 客服 IM | Supabase | 用户消息可一键复制（已规划） |
| 日志 / 分析 | Plausible / 自建 ClickHouse | 不收集 PII，仅匿名事件 |

## 二、内容审核

### 2.1 越南内容红线
- 主权 / 领土：南海议题、北部湾、河内 vs 北京历史
- 战争史：1979 年中越战争完全避免
- 党与领袖：胡志明、武元甲不可负面
- 宗教：佛教 / 天主教尊重为主

### 2.2 泰国内容红线
- 王室：绝对不可玩笑、负面、对比（《lèse-majesté》法律重罚）
- 佛教：泰国上座部佛教尊重为主，不与中国汉传佛教对立比较
- 政治：避免 2014 政变 / 红黄衫细节

### 2.3 印尼内容红线
- 宗教：穆斯林占 87%，避免酒、猪肉过于显眼；尊重斋月 / 开斋节
- 民族：1998 排华事件回避
- 同性 / LGBT：保守，慎涉

### 2.4 通用红线
- 政治宣传 / 意识形态色彩
- 战争胜负叙事
- 民族 / 宗教对立
- 涉黄涉暴

### 2.5 内容审核 SOP（落地）
1. AI 生成 → 提示词中显式排除红线（已规划）
2. 母语审校 → 同时做语言准确性 + 文化敏感性
3. 第三道筛 → 上线前由运营人工抽检 10%
4. 上线后举报机制 → 用户举报 24 小时响应
5. 内容版本号 → 修改可追溯

## 三、教育资质

### 3.1 C 端学习应用
- 三国均不强制要求教育资质
- 但需在隐私协议中明示"非学历教育"

### 3.2 B2B 学校 / 机构
- 越南：进入公立学校需备案
- 泰国：与 OCSC（学历认证）合作可获政府认可
- 印尼：与 Diktiristek 备案
- → 知语 v1 不进 B2B 教育，v2 评估

### 3.3 涉及未成年人
- 印尼对 18 岁以下用户额外保护（监护人同意）
- 越南对 7-15 岁分级处理
- 泰国对 20 岁以下处理需家长同意（部分场景）
- → 知语 v1 注册门槛 18+，避免合规复杂

## 四、支付与税务

### 4.1 支付（详见 04-technical-research/06）
- Paddle Merchant of Record：知语只对 Paddle 开发票，Paddle 处理全球税
- 备选：LemonSqueezy（同样 MoR）

### 4.2 税务考量
- 增值税 / GST：Paddle 自动代收代缴
- 越南 VAT 10%、泰国 VAT 7%、印尼 PPN 11%
- 知语自身仅需在注册地（如新加坡 / 香港）完成企业所得税
- 推荐：注册新加坡 Pte Ltd（17% 企业税 + 国际化品牌）

### 4.3 退款合规
- 用户原话："付费后禁止退款"
- 风险：Paddle 自身有 60 天 chargeback 保护，无法 100% 禁止
- 对策：
  - 服务条款明示"虚拟商品 + 数字内容 一经购买不可退"
  - 但保留 Paddle 处理欺诈 / 重大错误退款
  - 在结算页二次确认

## 五、版权与知识产权

### 5.1 内容创作权
- AI 生成内容版权归运营方（合同 + 服务条款约定）
- 母语审校外包合同需 IP 转让条款

### 5.2 第三方素材
- 字体：开源字体（思源宋体 / Noto Sans）
- 图片：自生 AI（Midjourney / SD）+ 商用授权（Unsplash）
- 音乐：游戏 BGM 用 Pixabay / Mubert AI / 商用音库

### 5.3 课程内容防爬 / 防抄
- 已明确采用"反爬 = 数据 API 化 + 鉴权"
- 加水印（音频水印 / 文本零宽字符）
- DMCA / 当地版权法举报

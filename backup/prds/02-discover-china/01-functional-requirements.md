# 2.1 · 发现中国 · 功能需求

## 一、用户故事

### US-DC-01：浏览类目
> 作为用户，我希望看到 12 类目的入口和近期热门文章，以便快速找到感兴趣的内容。

### US-DC-02：阅读文章
> 作为学习者，我希望以句子为单位阅读，看中文 + 拼音 + 母语翻译，并能听 TTS 音频，以便我能精读每一句。

### US-DC-03：阅读进度
> 作为用户，我希望关闭后再回来能从上次位置继续阅读。

### US-DC-04：收藏文章
> 作为用户，我希望收藏喜欢的文章随后再读。

### US-DC-05：分享文章
> 作为用户，我希望生成图卡分享给朋友（含分销码）。

### US-DC-06：未登录预览
> 作为新访客，我希望不登录也能浏览前 3 个发现中国类目，决定是否注册查看全部 12 类目。

### US-DC-07：搜索 / 筛选
> 作为用户，我希望按类目 / 难度 / 关键字筛选文章。

### US-DC-08：评分文章
> 作为用户，我希望给文章打分（5 星制），帮助平台优化内容。

## 二、功能需求

### DC-FR-001：类目首页
- **优先级**：P0
- **描述**：展示 12 类目卡片，每张卡片显示类目名（母语）、封面图、文章总数、最近 3 篇标题
- **位置**：`/discover` 路由
- **数据来源**：`content_categories` + `content_articles` 聚合
- **缓存**：本地 nginx/cache header/SW 预取 1h；禁止依赖外部 CDN
- **未登录可见**：前 3 个类目可见；第 4-12 类目显示登录引导

### DC-FR-002：类目列表页
- **优先级**：P0
- **描述**：`/discover/:category_slug` 展示该类目所有文章，按发布时间倒序，分页 20/页
- **筛选**：HSK 难度（HSK 1 / 2-3 / 4-5 / 6+）、长度（短 / 中 / 长）
- **未登录可见**：是

### DC-FR-003：单篇阅读页
- **优先级**：P0
- **描述**：`/discover/:category_slug/:article_slug` 句子级展示
- **元素**：
  - 标题（中 / 母语）
  - 元信息（类目 / HSK 难度 / 字数 / 阅读时长 / 收藏数）
  - 封面图（可选）
  - 句子列表（中文 + 拼音 + 母语 + 音频按钮）
  - 关键点（key_point 母语解释）
  - 末尾 CTA（"想系统学？试试 [系统课程]"）
- **未登录可见**：仅当文章所属类目为前 3 个开放类目时可见；第 4-12 类目文章需登录

### DC-FR-004：句子级交互
- **优先级**：P0
- **元素**：
  - 句子点击 → 播放 TTS
  - 长按句子 → 显示菜单（收藏 / 笔记 / 复制）
  - 拼音显示模式：拼音字母 / 数字声调 / 隐藏（用户偏好）
  - 拼音字号 / 行高随用户偏好

### DC-FR-005：阅读进度
- **优先级**：P0
- **描述**：滚动到第 N 句时自动保存（防抖 5s），未登录则用 localStorage
- **存储**：`learning_reading_progress` 表
- **恢复**：返回文章时自动滚到上次位置
- **完读判定**：滚到末尾 + 停留 > 30s

### DC-FR-006：收藏
- **优先级**：P0
- **描述**：右上角心形按钮，点击收藏 / 取消
- **存储**：`user_favorites`（type=article）
- **个人中心**：`/me/favorites` 查看所有收藏

### DC-FR-007：笔记
- **优先级**：P1
- **描述**：句子菜单 → "记笔记"，弹出文本框（限 500 字）
- **存储**：`user_notes`（target_type=sentence, target_id=sentence_id）
- **个人中心**：`/me/notes` 查看
- **多设备同步**：是

### DC-FR-008：评分
- **优先级**：P1
- **描述**：文章末尾"这篇文章对你有帮助吗？" 5 星
- **存储**：`content_ratings`
- **限制**：每用户每文章 1 票

### DC-FR-009：分享卡片
- **优先级**：P1
- **描述**：分享按钮 → 生成 1080×1920 图卡（标题 + 1 句金句 + QR 含分销码）
- **生成**：服务端 Puppeteer 或 satori
- **缓存**：Storage 90 天
- **去 SNS**：直接复制图 + 文案到剪贴板（移动端 navigator.share）

### DC-FR-010：未登录类目预览限制
- **优先级**：P0
- **规则**：
  - 未登录访客仅可进入前 3 个类目：中国历史、中国美食、名胜风光
  - 第 4-12 类目列表页和文章页弹"注册免费解锁全部发现中国"
  - 登录后立刻解锁全部 12 类目的全部内容
- **状态记录**：使用 anon_id + device_id + ip_hash 记录转化事件，不按文章篇数扣减额度
- **绕过防护**：nginx + 自建限流中间件 + CaptchaAdapter always-pass/fake（见 SC/FP 模块）

### DC-FR-011：搜索
- **优先级**：P1
- **描述**：顶部搜索框 → 关键词匹配标题 / 正文中文 / 关键点
- **后端**：PostgreSQL FTS（pg_trgm）
- **索引**：`content_articles.title_zh`, `content_articles.summary`, `content_sentences.zh`
- **结果**：分页 20/页 + 高亮

### DC-FR-012：相关推荐
- **优先级**：P1
- **描述**：单篇底部"你可能还喜欢"显示 4 篇同类目其他热门文章
- **算法 v1**：同类目 + 相近 HSK 难度 + 高评分
- **算法 v2**：协同过滤（M+6 后）

### DC-FR-013：母语切换
- **优先级**：P0
- **描述**：用户切换 UI 语言时，所有文章翻译实时切换
- **存储**：`content_articles.translations` JSONB（en / vi / th / id）
- **降级**：缺失语种则回退英文

### DC-FR-014：HSK 难度标识
- **优先级**：P1
- **描述**：每篇文章计算 HSK 难度等级（基于词汇覆盖率）
- **算法**：词频统计 → 最高词所需 HSK 等级
- **筛选**：用户可按难度筛

### DC-FR-015：阅读统计
- **优先级**：P1
- **描述**：个人中心 `/me/stats` 显示已读文章数、累计字数、收藏数
- **数据**：来自 `learning_reading_progress`

### DC-FR-016：内容预热
- **优先级**：P0
- **描述**：热门文章自动进入本地 nginx/cache header/SW 预取策略
- **触发**：发布即推 + 热度阈值（日访问 > 100）

## 三、UX 流程

### 3.1 浏览路径
```
/                       (首页)
  → "发现中国" tab
  → /discover           (12 类目卡片)
  → /discover/cuisine   (类目列表)
  → /discover/cuisine/dim-sum   (单篇)
```

### 3.2 阅读流程
```
进入文章
  → 自动恢复上次位置
  → 滚动阅读
    ↓
  点击句子 → 播放音频
    ↓
  长按句子 → 收藏 / 记笔记
    ↓
  到末尾 → 评分（可选）
    ↓
  点击 CTA → 跳转课程或下一篇
```

### 3.3 未登录限流流程
```
新访客访问
  → 前 3 个类目正常浏览与阅读
  → 第 4-12 类目尝试访问
  → 弹层 "注册免费解锁全部发现中国"
  → 注册 / 登录 → 解锁全部 12 类目 / 取消 → 返回类目首页
```

## 四、UI 关键元素

### 4.1 类目卡片
- 封面图（圆角 16px）
- 类目名（中 + 母语）
- 文章总数 / 已读数
- 渐变背景（按类目主题色）

### 4.2 句子组件
- 中文（大字号 / 加粗）
- 拼音（中字号 / 浅色）
- 母语翻译（中字号 / 折叠 / 点击展开）
- 音频按钮（图标 / 播放态高亮）

### 4.3 阅读进度条
- 顶部进度条（0-100%）
- 滚动同步

### 4.4 关键点卡片
- 文末"重点提示"卡片
- 母语解释（HSK 等级 / 文化注解）
- 限 200 字

## 五、内容生产规则

### 5.1 篇幅
- 短篇：300-500 字（推荐入门）
- 中篇：500-1000 字（HSK 3-5）
- 长篇：1000-2000 字（HSK 6+）

### 5.2 结构
- 标题（双语）
- 摘要（30-50 字）
- 正文（句子级）
- 关键点（3-5 条母语解释）
- 字段：`hsk_level`, `category`, `tags`, `cover_image_url`

### 5.3 红线
- 严禁政治 / 宗教 / 民族
- 严禁泰国王室相关
- 严禁敏感历史
- 详见 SC 模块 + 内容审核 SOP

## 六、性能要求

- 类目列表页 LCP < 2s
- 单篇阅读页 LCP < 2.5s
- 句子音频加载 < 500ms
- 滚动 60fps（中端机）

## 七、SEO 要求

- 每篇文章独立 URL（slug = 母语 transliterate）
- title / description / og 图 母语
- JSON-LD（Article schema）
- sitemap.xml 自动生成
- 4 语种独立 URL（`/vi/discover/...`, `/th/...`, `/id/...`, `/en/...`）

## 八、可访问性

- ARIA labels
- 键盘可达（Tab 导航）
- 屏幕阅读器友好
- 字号可调（用户偏好）

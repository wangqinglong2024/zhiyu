## 模块概述

### 范围
- 12 类目（中国历史 / 中华美食 / 山水奇观 / 节日民俗 / 艺术遗产 / 音乐戏曲 / 古典文学 / 成语典故 / 哲学智慧 / 现代中国 / 玩转中文 / 神话传说）
- 每类目 ≥ 50 篇短文（首发 600 篇）
- 每篇：句子级（中文 + 拼音 + 母语翻译 + TTS 音频）+ 关键点（key_point）
- 阅读进度 / 收藏 / 笔记 / 分享卡片
- 未登录用户可浏览前 3 个类目（中国历史、中国美食、名胜风光）及其文章；登录后可浏览全部 12 类目全部内容

### 内容来源
- 参考 `/china/00-index.md` 至 `/china/12-myths-legends.md`
- AI 工厂生成 + 母语审校

### 关键交互
- 类目浏览 → 文章列表 → 单篇阅读 → 收藏 / 分享
- 阅读进度自动保存
- 句子级播放 / 暂停 / 跳转

## 关键决策

| 决策点 | 决策 |
|---|---|
| 是否限制未登录访问 | 前 3 个类目免费浏览 + 第 4 个类目起需登录；登录后全部免费 |
| 是否分付费内容 | v1 全部免费（流量入口） |
| 是否做评论 | v1 不做（v1.5 评估） |
| 是否进 SRS | v1 不进（专注被动学习场景） |

## 四、测试用例（关键）

### 用例 1：未登录访问第 4 个类目
1. 清空 cookie
2. 访问中国历史类目文章 → 200 OK
3. 访问中国美食类目文章 → 200 OK
4. 访问名胜风光类目文章 → 200 OK
5. 访问传统节日类目文章 → 401 + `code=discover_category_login_required`
6. 注册 / 登录 → 访问传统节日类目文章 → 200 OK

### 用例 2：阅读进度恢复
1. 登录 → 文章 A 滚到 50%
2. 关闭浏览器
3. 重新打开 → 自动滚到 50%

### 用例 3：母语切换
1. 当前 EN，访问文章 A
2. 切到 VI
3. 翻译实时更新为越南语

### 用例 4：分享卡片含分销码
1. 用户 X 已绑分销码 ABC123
2. 生成分享图 → QR 含 `?ref=ABC123`
3. 访客扫码 → 注册 → 用户 X 上级关系建立

### 用例 5：搜索性能
1. 关键词"饺子" → < 500ms 返回结果
2. 含高亮 + 分页

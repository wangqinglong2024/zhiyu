# 4.1 · 游戏 · 功能需求（MVP 单阶段 · 一次性首发 · 无限连玩 60s）

> **全局玩法**：12 款游戏均为 **60s 单局 · 无限连玩 · 无关卡 / 章节 / 下一关 / 多波**；本文所有「关 / 题量 / 三星」描述均废弃。

- US-GM-02：作为学习者，我希望游戏错题进入复习集
- US-GM-03：作为休闲用户，我希望免费玩，不强制付费

## 二、通用功能需求（MVP 单阶段）

### GM-FR-001：游戏专区入口
- 路径：`/games`
- **12 张游戏卡片，全部可玩**，**无 `coming_soon` 占位**
- 每卡片显示：图标、名称、个人最近一次得分（无最高分 / 无玩家数 / 无评分）
- 排序：固定推荐顺序（不做热门 / 个人最高排序，post-MVP）

### GM-FR-002：强制横屏
- 任何游戏页：竖屏 → 显示"请旋转设备"遮罩
- 桌面：画布固定 16:9，居中
- 退出游戏：返回 `/games` 首页解除横屏

### GM-FR-003：词汇关卡映射（MVP 子集）
- 词包来源唯一：课程题库知识点，不允许游戏内自造内容
- 玩家可选：主题轨道（电商 / 工厂 / HSK / 日常）+ 课程阶段（S1-S12）+ 难度
- 可选范围：由用户当前拥有的课程权限决定；免费试学章可生成对应基础词包，已购买阶段可生成该 track/stage 词包
- 示例：用户已跨级购买电商 Stage 9，则游戏设置面板允许选择“电商 / S9”
- 未授权阶段在设置面板灰显并提示购买课程；若允许试玩，成绩不计入 SRS/考核
- 内容池不足 30 词时提示扩展到上一阶段；仍不足时回退到用户有权限的最近阶段

### GM-FR-004：本局与结算（不发奖励 · 不设关卡）
- **本局 = 60 秒固定倒计时**，**无关卡 / 下一关 / 多波**概念
- **不发放任何奖励 / 知语币 / 经验 / 道具**
- 本局结算页只展示：用时（60s） / 得分 / 错题列表 / 「再玩一局」按钮 / 「返回游戏中心」按钮
- **不展示**：三星评级 / 排行榜跳转 / 分享按钮 / 奖励动画 / 「下一关」 / 「解锁」（全 post-MVP）
- 玩家可点「再玩一局」无限重启，状态不累积、不存在「过关」

### GM-FR-005：错题入 SRS
- 游戏中答错的知识点记录 question_id
- **本局结束（60s）后批量推 SRS（LE 模块）**
- 错题列表展示在本局结算页（仅展示，不能编辑）

### GM-FR-006 ~ GM-FR-008：post-MVP（不在本期交付）
- **GM-FR-006 排行榜**：post-MVP，迁入 `99-post-mvp-backlog.md`
- **GM-FR-007 游戏内购（IAP）**：post-MVP
- **GM-FR-008 通关分享**：post-MVP

## 三、12 款 MVP 游戏玩法

> 全部按 MVP 最小子集实现：1 难度 / 1 词包 / 5 关或 60s 单局 / 60fps / 错题入 SRS / 不发奖励。
> 三星评级 / 排行榜 / IAP / 奖励 / 分享 一律不实现。

### 游戏 1：拼音射击（pinyin-shooter）
- 玩法：屏幕上方汉字下落，玩家拼出拼音击落
- 输入：键盘（桌面）/ 屏幕拼音键盘（移动）
- MVP：5 关，每关 30s，至少击落 10 字
- 错题：未及时击落或拼错的字 → SRS

### 游戏 2：声调泡泡（tone-bubbles）
- 玩法：泡泡含汉字，按声调（4 色）点击对应桶
- MVP：5 关，每关 30s
- 错题：错放的字 → SRS

### 游戏 3：打地鼠（whack-hanzi）
- 玩法：5×3 地鼠洞，地鼠头戴汉字，按母语提示打对应字
- MVP：5 关，每关 30s
- 错题：打错的字 → SRS

### 游戏 4：汉字消消乐（hanzi-match3）
- 玩法：6×8 网格，三同字消除（仅"音"模式 MVP）
- MVP：5 关，每关 60s
- 错题：错配 → SRS

### 游戏 5：翻牌记忆（memory-match）
- 玩法：4×4 卡牌（汉字 + 拼音），翻牌配对
- MVP：5 关，按用时计 + 错次计
- 错题：连续错 3 次的对 → SRS

### 游戏 6：汉字忍者（hanzi-ninja）
- 玩法：飞字切割，按拼音提示切对应字
- MVP：60s 局，3 命，单关卡
- 错题：切错的字 → SRS

### 游戏 7：汉字俄罗斯方块（hanzi-tetris）
- 玩法：经典俄罗斯方块 + 同字水平消除
- MVP：3 分钟局，仅水平消除
- 错题：消错组合 → SRS

### 游戏 8：拼音塔防（pinyin-defense）
- 玩法：拼音输入触发塔攻击
- MVP：1 地图 / 3 波 / 单塔类型
- 错题：输入错误的字 → SRS

### 游戏 9：汉字贪吃蛇（hanzi-snake）
- 玩法：吃字组词，撞身死
- MVP：5 关
- 错题：组错词 → SRS

### 游戏 10：汉字节奏（hanzi-rhythm）
- 玩法：随节拍点击对应字
- MVP：1 BPM / 1 曲目
- 错题：漏点 / 点错 → SRS

### 游戏 11：汉字跑酷（hanzi-runner）
- 玩法：横版跑酷 + 选字过门
- MVP：单关 / 3 障碍门
- 错题：选错 → SRS

### 游戏 12：汉字弹弓（hanzi-slingshot）
- 玩法：Matter 物理抛物，瞄准对应字
- MVP：单关
- 错题：射错 → SRS

## 四、UX
- 进入游戏 → 统一设置面板选择主题轨道 + 阶段 + 难度 → 权限校验 → 开始
- 60s 倒计时 HUD 全局一致
- 本局结算页：得分 + 用时（60s）+ 词包来源 + 错题 + 「再玩一局」 + 「返回游戏中心」
- **没有**：三星 / 排行榜 / 分享 / 奖励动画 / 下一关 / 解锁

## 五、性能
- 60fps 中端 Android
- 包大小 < 5MB / 游戏（首屏）
- 资源懒加载

## 六、安全
- MVP 仅记录 raw 分数，不做 HMAC 防作弊
- 后端校验异常分数（超出统计上限）丢弃
- HMAC 防作弊整体迁入 `99-post-mvp-backlog.md`

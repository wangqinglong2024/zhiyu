# 4.3 · 游戏 · 验收准则（MVP 单阶段 · 一次性首发 · 60s 无限连玩）

## 功能验收
- [ ] GM-AC-001：12 游戏卡片全部 active，无 `coming_soon` 占位
- [ ] GM-AC-002：强制横屏遮罩在竖屏下显示
- [ ] GM-AC-003：词包选择有效；可选主题轨道与阶段范围由课程权限决定
- [ ] GM-AC-003A：已购电商 Stage 9 后，游戏设置面板可选择“电商 / S9”
- [ ] GM-AC-004：**本局固定 60 秒**倒计时准确；结算页仅含「得分 / 用时 / 错题 / 再玩一局 / 返回」
- [ ] GM-AC-005：错题入 SRS（次日温故知新含错题）
- [ ] GM-AC-006：**不存在**关卡切换 / 下一关 / 多波 / 三星 UI
- [ ] GM-AC-007：**不存在**排行榜 / 分享 / 奖励 UI
- [ ] GM-AC-008：「再玩一局」一键重启，状态不累积
- [ ] GM-AC-009：12 款游戏均可连续玩多局，无任何「完成 / 未完成 / 解锁」状态梳理

## 非功能验收
- [ ] 60fps 中端 Android（中位机型 4G）
- [ ] 首屏 < 2s，资源 < 5MB
- [ ] 本局提交 P95 < 500ms

## 内容验收（W0）
- [ ] 12 款游戏全部一次性上线
- [ ] 词包覆盖：课程权限范围内的 track + stage，未授权阶段灰显或试玩不计入
- [ ] 4 语种 UI 完整

## 关键测试用例
1. 新用户首次玩拼音射击 → 60s 结算 → 错题入 SRS，**不发任何币 / 奖励**
2. 点「再玩一局」连玩 5 局 → 每局状态互不影响、均为 60s
3. 手机竖屏进入游戏 → 横屏遮罩；旋转 → 进入
4. 任选一款游戏连玩 → 未出现三星 / 排行榜 / 下一关 UI

## 监控
- DAU / Game ratio > 30%
- 平均单局时长接近 60s
- 平均连玩局数 ≥ 2
- 错题入 SRS 成功率 100%

# 8.1 · 知语币 · 功能需求

## 用户故事
- US-EC-01：每日签到拿币
- US-EC-02：用币买 streak freeze 等道具
- US-EC-03：用币换皮肤 / 头像框
- US-EC-04：查看币明细（issuance / sink）

## 功能需求

### EC-FR-001：注册赠币
- 验证邮箱后 +100 ZC（一次）

### EC-FR-002：每日签到
- `/me/checkin` 或 dashboard 浮层
- 1-10 不是随机，是越大的数字，概率越低。50%是1或2，30%是3或4，10%是5或6，8%是7或8，2%是9或10；
- 连续 7 天 +20 额外，连续 30 +40
- 中断 streak 不影响签到（独立计数）

### EC-FR-003：学习任务奖励
| 任务 | 奖励 |
|---|---|
| 完节小测（≥60%） | 1-3 ZC（按得分） |
| 完章测（≥70%） | 10 ZC |
| 完阶段考（≥75%） | 40 ZC |

### EC-FR-007：账本明细
- `/me/coins` 显示余额 + 流水（issuance / sink）
- 每条：类型 / 金额 / 时间 / 来源
- 分页 50/页

### EC-FR-008：商店
- `/store` 显示可购道具
- 类目：消耗品 / 装饰 / 限定
- 购买二次确认

### EC-FR-009：防作弊
- 同 IP / 同设备多账号刷币：限流 + 检测
- 异常 issuance 告警（运营干预）
- 用户被冻结：余额冻结 + 所有发放暂停

### EC-FR-010：会员等价（不可购买）
- 系统赠送会员（推广活动）：可用 ZC 兑换月会员（400 ZC = 1 月）

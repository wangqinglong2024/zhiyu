# 8.3 · 知语币 · 验收准则

## 功能
- [ ] EC-AC-001：注册验证后 +100 ZC（一次）
- [ ] EC-AC-002：每日签到 5-30 随机；连续奖励 +50/+200
- [ ] EC-AC-003：所有学习任务正确发放（按表）
- [ ] EC-AC-004：游戏日上限 50 / 游戏 触达即停
- [ ] EC-AC-005：streak freeze 消耗 50；最多持 3
- [ ] EC-AC-006：年发行上限 50K 触达停止
- [ ] EC-AC-007：账本流水准确，不丢不重（幂等）
- [ ] EC-AC-008：余额非负
- [ ] EC-AC-009：商店购买扣减 + 库存增加
- [ ] EC-AC-010：余额冻结后所有 grant/spend 拒绝

## 非功能
- [ ] balance P95 < 200ms
- [ ] grant 事务 P95 < 300ms
- [ ] 幂等 key 重发不重复扣发
- [ ] 异常 issuance 告警（日 > 500 / 用户）

## 测试用例
1. 用户日完 1 节小测得 90% → +13 ZC（5 + 8 按得分）；ledger 含一条记录
2. 同节重做 → 不再发放（idempotency_key=lesson:abc:user:xyz:first_pass 已存在）
3. 当日签到完 → 再签 → 拒
4. 用户 yearly_issued = 49,990 → 下次签到只发放 10（不超上限）
5. 用 freeze：余额 100 → 50；inventory +1
6. 余额 0 时购买 200 ZC 道具 → 拒
7. 同时并发 grant 100 + spend 100 → 余额最终一致（事务隔离）

## 监控
- 日发行总量
- 异常账户（issuance > 阈值）
- 商店购买 5xx < 0.1%

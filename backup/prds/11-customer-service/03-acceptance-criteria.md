# 11.3 · 客服 · 验收准则

## 功能
- [ ] CS-AC-001：浮窗按钮全站可见
- [ ] CS-AC-002：用户首消息创会话 → 路由 CSR
- [ ] CS-AC-003：同语种 CSR 优先
- [ ] CS-AC-004：工作时间外留言 + 邮件通知
- [ ] CS-AC-005：CSR 接单后用户即时收到 system 消息
- [ ] CS-AC-006：消息实时推送（用户与 CSR 双向）
- [ ] CS-AC-007：图片上传 ≤ 5MB
- [ ] CS-AC-008：未读数准确
- [ ] CS-AC-009：会话状态流转准确
- [ ] CS-AC-010：CSR 关闭 → 用户评分弹层
- [ ] CS-AC-011：7 天后自动归档
- [ ] CS-AC-012：CSR 当前同时处理 ≤ 5

## 非功能
- [ ] 发送 P95 < 300ms
- [ ] Realtime 延迟 < 1s
- [ ] RLS 仅自己或 CSR 可见

## 测试用例
1. 用户 A（vi）发消息 → 路由到懂 vi 的 CSR1（最少工单）
2. CSR1 已接 5 单 → 路由到 CSR2
3. 21:00 UTC+7 内发消息 → 立即路由
4. 23:00 发消息 → 留言 + 邮件
5. CSR 关闭会话 → 用户收到评分弹
6. 用户离线时 CSR 回复 → 邮件触发
7. 7 天未关 + 用户未回 → 自动 closed_no_response
8. 上传 6MB 图片 → 拒

## 二、内容颗粒度规格

### 2.1 知识点（Knowledge Point）

每节有 12 个知识点，每个知识点包括：

| 字段 | 内容 |
|---|---|
| `kpoint_no` | 1-12 |
| `type` | word / phrase / sentence / grammar / culture |
| `zh` | 中文（词 / 句） |
| `pinyin` | 拼音（带音调） |
| `pinyin_tones` | 数字声调 |
| `translations` | 4 语种翻译 |
| `audio` | TTS 音频 |
| `key_point` | 4 语种 母语解释（包括用法 / 易错点 / 文化注解） |
| `example_sentences` | 2-3 个例句（每个 = 句子结构） |
| `tags` | 词性 / 主题 / 难度标签 |

### 2.2 节（Lesson）

每节包括：
- 12 知识点
- 节小测（10 题，覆盖 12 知识点）
- 学习目标 / 母语介绍（lesson intro）
- 完成标准：所有 12 知识点点过 1 次 + 节小测 ≥ 60%

### 2.3 章（Chapter）

每章包括：
- 12 节
- 章测（36 题，从节小测题库抽样 + 综合题）
- 章节大纲（母语介绍）
- 完成标准：12 节全部完成 + 章测 ≥ 70%

### 2.4 阶段（Stage）

每阶段包括：
- 12 章
- 阶段考（80-150 题，全面综合）
- 阶段大纲 + 学习预期 + 适合人群（母语）
- 完成标准：12 章全部完成 + 阶段考 ≥ 75%

## 三、内容生产 SOP

### 3.1 配置阶段（运营）
- 选轨道 + 阶段
- 设定主题 + HSK 等级 + 词汇范围
- 配置敏感词避让

### 3.2 AI 生成阶段（CF）
- LangGraph 工作流
- Step 1：生成 12 章主题
- Step 2：每章生成 12 节主题
- Step 3：每节生成 12 知识点（中 + 拼音 + 例句）
- Step 4：4 语翻译（key_point + 例句）
- Step 5：TTS 批量生成
- Step 6：节小测题目生成（10 题 / 节）
- Step 7：章测题目（36 题 / 章，部分抽节小测 + 部分新生成）
- Step 8：阶段考（80-150 题）

### 3.3 审校阶段
- 红线词检测（自动）
- LLM 二审（自动）
- 母语审校（人工）：发音准确性 / 翻译自然度 / 文化敏感性
- 教学顾问审：HSK 等级匹配 / 教学逻辑

### 3.4 发布阶段
- 单节单独可上架
- 学习建议默认线性；权限不强制线性，用户可跨级购买任意轨道任意阶段并直接学习已购阶段

## 四、付费墙规则

### 4.1 免费内容
- 登录用户：每个主题 Stage 1-3 的全部章节完全免费试学（含章内 12 节、节小测、对应章测）
- 未登录用户：可浏览课程结构，但开始学习需登录
- 拼音入门完全免费
- 免费范围内的节小测 / 章测免费；免费范围外的测验随课程权限走
- W0 首发生产范围（前 3 阶段）与免费权限范围一致：每个主题 Stage 1-3 全部章免费。不得再按“Stage 1 前 3 章”实现。

### 4.2 付费内容
- 4 主题 × 阶段 4-12 = 432 章 = 5,184 节
- 单段：$4 / 段（解锁该段全部）
- 9 段全包：$36（按轨道）
- 月会员：$4，4 轨道全开
- 年会员：$40，4 轨道全开 + 包括未来新内容

### 4.3 付费墙触发逻辑
- 用户访问免费试学范围外内容
- 检查：是否会员 / 是否买过该段 / 是否后台人工授权
- 已购：正常进入
- 未购：弹付费墙（多档选项 + 限时促销）
- 强解锁（管理员特权 / 退款保留）：可看但 `is_completion_counted=false`

### 4.4 付费墙位置 A/B
- A：Stage 4 入口
- B：任意阶段入口（用于跨级购买用户）
- C：节小测前
- W0 用 A，M+1 起 A/B 测试

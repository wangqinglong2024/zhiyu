# 1.2 · 产品目标 / 愿景 / 北极星

## 一、产品愿景

> **"用母语讲解 + AI 自适应路径，把中文从'考试科目'变成'生活与生意'。"**

5 年长期：成为东南亚学习者学中文的第一选择。

## 二、产品目标（OKR 风）

### 2.1 业务目标

#### O1：实现 PMF 并稳态运营
- KR1：M+3 完成 PMF 验证（D30 留存 ≥ 12%、付费率 ≥ 1%）
- KR2：M+12 三市场（VN+TH+ID）稳定运营
- KR3：M+12 ARR ≥ $300K

#### O2：建立内容护城河
- KR1：W0 上线 4 轨道 × 前 3 阶段（20,736 知识点）+ 600 篇文化 + 50 章小说
- KR2：M+8 完成 4 轨道全 12 阶段
- KR3：内容用户评分 ≥ 4.0 / 5

#### O3：建立增长飞轮
- KR1：M+12 K-factor ≥ 1.0
- KR2：M+12 SEO 自然流量占比 ≥ 30%
- KR3：M+12 NPS ≥ 40

### 2.2 用户目标

- O：让东南亚学习者用更低成本、更高效率、更母语友好的方式学中文
- KR1：用母语讲解使学习效率提升 30-50%
- KR2：价格降至竞品 1/3
- KR3：D30 留存提升至行业基线 1.5x

### 2.3 技术目标

- O：用现代技术栈与 AI 工厂建立小团队的高产能
- KR1：单条知识点生产成本 < $0.05
- KR2：API P95 响应 < 200ms
- KR3：Uptime > 99.5%

## 三、北极星指标

> **WAL** = Weekly Active Learners
>
> 定义：每周完成至少 1 节课程或 1 篇发现中国短文阅读 的活跃学习者数

### 3.1 为什么是 WAL
- 反映"真实学习行为"（不是简单 MAU）
- 与用户价值直接相关
- 与商业指标强相关

### 3.2 北极星目标

| 时点 | WAL |
|---|:---:|
| M+1 | 500 |
| M+3 | 6,000 |
| M+6 | 25,000 |
| **M+12** | **80,000** |
| M+24 | 220,000 |

## 四、核心价值主张

> 对**东南亚 18-35 岁中文学习者**来说，**知语**是一款**母语原生的 AI 中文学习全栈应用**，它不像 **HelloChinese / Duolingo / Migii** 那样只用英语讲解或单一题库，而是用**你的母语 + 4 个学习模块 + 1/3 的价格**让中文学习真正有效。

## 五、三大战略支柱

### 5.1 母语原生（差异化）
- UI / 课程讲解 / 客服 / 推送 全部母语
- 母语审校保证质量
- 4 语种（英 / 越 / 泰 / 印尼）原生

### 5.2 AI 内容工厂（v1.5 目标；MVP 手工产出）
- LangGraph 工作流 + 双 LLM
- 单条生产成本 < $0.05
- 月产能 5,000 知识点

### 5.3 4 模块闭环（留存）
- 文化 / 课程 / 游戏 / 小说 互通
- 知语币贯穿
- 多模块用户付费率 2x

## 六、产品原则

1. **母语优先**：UI、帮助、客服、推送全部母语
2. **零摩擦**：注册 / 试学 / 学习 / 付费 路径都最短
3. **质量不打折**：宁可少做也不发劣质内容
4. **可观测**：每个功能都有指标 + Dashboard
5. **可逆**：所有更新可一键回滚
6. **AI-first**：每个流程问"AI 能做什么？"
7. **数据驱动**：决策必须有数据支撑

## 七、不做清单（Strategic NOs）

- ❌ 不做 1v1 真人课（v2+）
- ❌ 不做 ASR 评测（v2+）
- ❌ 不做原生 App（用 PWA 兜底）
- ❌ 不做实时多人对战
- ❌ 不做直播
- ❌ 不接本地支付（仅 Paddle）
- ❌ 不做中文以外语种
- ❌ 不做中国大陆市场
- ❌ 不涉及政治 / 宗教 / 民族敏感

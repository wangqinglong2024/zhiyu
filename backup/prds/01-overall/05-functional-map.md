## 四、模块运行时数据流

### 4.1 学习闭环

```
用户 → 进入课程 (CR) → 学习知识点
                ↓
          完成节小测 (CR + LE)
                ↓
          错题入 SRS (LE)
                ↓
          每日温故知新 (LE)
                ↓
          首页提醒 (UA)
                ↓
          推荐游戏强化 (GM ↔ CR)
```

### 4.2 付费闭环

```
用户 → 浏览课程 (CR) → 触达付费墙 (PY)
                ↓
          支付 (PY → Paddle)
                ↓
          Webhook (PY)
                ↓
          升级会员 / 解锁段 (UA + CR)
                ↓
          奖励上级 (RF + EC)
                ↓
          通知 (UA + 邮件)
```

### 4.3 内容生产闭环

```
PM 配置任务 (AD + CF)
        ↓
LangGraph 生成 (CF)
        ↓
红线检测 + 自动评分 (CF + SC)
        ↓
母语审校 (CF + AD)
        ↓
发布到 4 模块库 (DC / CR / NV)
```

## 五、共享基础组件

### 5.1 前端组件
- 句子级阅读器（SentenceReader）：DC / CR / NV 复用
- TTS 播放器（AudioPlayer）：全模块复用
- 拼音显示控件（PinyinView）：全模块复用
- 题型组件（QuestionRenderer Q1-Q10 + P1-P3）：CR / LE 复用
- 排行榜组件（Leaderboard）：GM / 全局活动复用
- 知语币显示（CoinBadge）：全模块复用
- 付费墙（Paywall）：CR / NV 复用

### 5.2 后端服务
- ContentService：CRUD + 缓存（DC / CR / NV）
- LearningService：SRS / 进度（CR / LE / GM）
- WalletService：知语币（EC / RF / PY）
- PaymentService：Paddle 集成（PY）
- AuthService：用户（UA + 全局）
- ChatService：IM（CS）

### 5.3 数据基础
- `users` + `user_preferences`
- `content_sentences`（DC / CR / NV 共用）
- `content_articles` / `content_lessons` / `content_chapters`
- `learning_progress` / `learning_review_queue` / `learning_wrong_set`
- `wallet_balances` / `wallet_ledger`
- `referrals` / `referral_codes`
- `orders` / `subscriptions`
- `chat_threads` / `chat_messages`

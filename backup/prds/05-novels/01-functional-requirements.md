# 5.1 · 小说 · 功能需求

## 用户故事
- US-NV-01：浏览类目并发现小说
- US-NV-02：句子级精读，看翻译听音频
- US-NV-03：进度自动保存，跨设备恢复
- US-NV-04：收藏 / 评分 / 笔记
- US-NV-05：章末快测加深记忆，错题入 SRS
- US-NV-06：分享章节金句卡片（含分销码）

## 功能需求

### NV-FR-001：类目首页
- 路径 `/novels`
- 12 类目卡片 + 推荐位（编辑精选）
- 每类目下显示 v1 是否上架（v1 仅 5 类目）

### NV-FR-002：类目列表页
- `/novels/:category_slug`
- 小说列表：封面、标题（双语）、简介、HSK 难度、章节数、读者数、评分
- 排序：最新 / 热门 / 评分 / 我在读

### NV-FR-003：小说详情页
- `/novels/:category/:novel_slug`
- 元素：封面、标题（双语）、作者、简介、HSK 等级、总章节、读者数、评分
- 章节列表（已读高亮）
- "开始阅读"或"继续阅读"按钮

### NV-FR-004：章节阅读页
- `/novels/:category/:novel_slug/:chapter_no`
- 复用 DC 句子组件
- 拼音 / 母语 / TTS / 长按菜单
- 章末"快测"按钮（3-5 题）
- 上一章 / 下一章 切换

### NV-FR-005：阅读进度
- 复用 DC 的 `learning_reading_progress`（target_type=novel_chapter）

### NV-FR-006：章末快测
- 3-5 题 Q1-Q3 题型，覆盖本章重点词
- 答错入 SRS
- 通过：≥ 60% → 章奖励 +5 币
- 跳过：可

### NV-FR-007：收藏 / 笔记 / 评分
- 同 DC 模式
- target_type 取值：novel / chapter / sentence

### NV-FR-008：阅读偏好
- 字号大 / 中 / 小
- 拼音模式（字母 / 数字声调 / 隐藏）
- 母语翻译显示（实时 / 折叠 / 隐藏）

### NV-FR-009：HSK 难度建议
- 用户 HSK < 小说推荐等级 → 弹"建议先学 X 阶段"+ "我要挑战"

### NV-FR-010：分享章节
- 生成金句图卡（自动选高频词或编辑标记的金句）

### NV-FR-011：评论（v1.5）
- 章节末评论；点赞 / 回复

## UX 性能
- 章节页 LCP < 2s
- 句子音频 < 500ms
- 切换章节预加载下一章

## SEO
- 每章独立 URL 4 语种
- JSON-LD（CreativeWork / Chapter）
- sitemap.xml 含全部已发布章节

## 三、v1 五部启动小说详细规格

### 3.1 都市言情 — 《晨光里的咖啡香》

| 字段 | 内容 |
|---|---|
| **标题（中）** | 晨光里的咖啡香 |
| **标题（英）** | Coffee in the Morning Light |
| **作者署名** | 知语原创工作室（CF 工厂） |
| **HSK 推荐等级** | HSK 3-4 |
| **字数 / 章** | 1800~2400 字 |
| **章节数** | v1 上 10 章，月续 5 章至 30 章封顶 |
| **题材** | 现代都市轻喜剧 / 治愈系 |
| **关键词** | 咖啡店 / 邻里 / 失去与重启 / 缓慢治愈 |
| **预估完读率** | 65%（轻量门槛） |

#### 前 10 章大纲
1. **第一章 雨夜的访客**：女主林晚秋经营濒临关店的咖啡馆，雨夜来了一位浑身湿透的中年男子。词汇：营业 / 顾客 / 暴雨 / 暖。
2. **第二章 老照片**：男子留下一张 1980 年的老照片，照片上的咖啡店正是这家。词汇：照片 / 历史 / 故事 / 怀念。
3. **第三章 街角的孩子**：一个常来要免费温水的小男孩开始引起林晚秋注意。词汇：孩子 / 善良 / 询问 / 关心。
4. **第四章 邻居的烦恼**：花店老板抱怨房租上涨，林晚秋决定召集邻里。词汇：房租 / 邻居 / 商量 / 帮忙。
5. **第五章 隔壁的钢琴声**：每晚 9 点准时响起的钢琴声引来好奇。词汇：钢琴 / 节奏 / 寂寞 / 倾听。
6. **第六章 那位中年男子归来**：他带来照片中老人的手稿——一份咖啡配方。词汇：配方 / 传承 / 调配 / 记忆。
7. **第七章 试味之夜**：林晚秋按配方调制，邻居都来品尝。词汇：味道 / 传统 / 工艺 / 分享。
8. **第八章 男孩的妈妈**：终于得知那个孩子的身世，他妈妈是夜班护士。词汇：医院 / 辛苦 / 责任 / 感动。
9. **第九章 街区的小聚会**：邻居们在咖啡馆办了一场互助会。词汇：互助 / 团结 / 笑声 / 温暖。
10. **第十章 晨光来时**：男孩的妈妈下夜班路过，喝到第一杯按老配方的咖啡。词汇：黎明 / 希望 / 开始 / 感谢。

#### 章末快测样例（第 1 章）
- Q1：单选 — "顾客" 的拼音是？(gùkè / gùké / gǔkè / gùkè)
- Q2：填空 — "雨夜里的_____浑身湿透。"（访客 / 顾客 / 邻居 / 朋友）
- Q3：连线 — 词与释义 4 对配对
- Q4：阅读理解 — "林晚秋为什么开始注意那位男子？"
- Q5：拼音听写 — 听音输入"咖啡馆"

## 四、内容审查与文化敏感性标准

### 4.1 全类目通用红线
- ❌ 政治隐射 / 体制批评（含历史架空）
- ❌ 露骨色情 / 暴力血腥
- ❌ 民族 / 宗教歧视
- ❌ 自杀 / 毒品诱导
- ❌ 涉及未成年的不当关系

### 4.2 类目特别约束
| 类目 | 特别要求 |
|---|---|
| 古代言情 | 避免历史人物的私生活虚构 |
| 武侠 | 武打不得超过"侠义"范畴，不渲染杀戮 |
| 历史架空 | 朝代 / 重大事件不可篡改 |
| 悬疑推理 | 谋杀情节避免血腥描写，留合理解释 |
| 恐怖盗墓 | 文物保护立场鲜明（v2 起谨慎） |
| 科幻末世 | 不影射现实灾难（疫情 / 战争） |
| 耽美 BL | 仅合规市场显示；严格非露骨；按当地法规配置 Feature Flag |
| 玄幻奇幻 | 不可影射现实宗教 |

### 4.3 多语翻译特别标注
- 每章生成时由内容工厂（CF）通过 LangGraph 自动并行翻译至 4 语
- 文化注解（节令 / 食物 / 礼俗）需译者特别标注，不直接翻译，添加母语解释
- 古风用词在 EN/VI/TH/ID 翻译需保留古意，可在尾注解释

### 4.4 审校流程
- CF 生成草稿 → AD 审校工作台 → 至少 2 位审校员（含本地母语审校 + 中文审校）签字 → 发布
- 任何用户举报（CS） → 24h 内重审 → 必要时下架重写

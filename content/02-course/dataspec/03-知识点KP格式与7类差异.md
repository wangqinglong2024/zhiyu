# 课程 · 03 · 知识点（KP）格式与 7 类差异

> 本节定义 `course_knowledge_points` + `course_lesson_kp` 两张表的 jsonl 输出格式。
> 7 类 KP（pinyin / hanzi / word / phrase / grammar / sentence / dialog）的 `content` jsonb 结构差异显著，**不要用同一个模板生成**。

> ⚠️ **AI 直出铁律**：本文件定义的所有 `title_zh / pinyin / translations.* / content.*`（含 examples、explanation_md、rule_md、reply_examples 等子字段）必须由 AI 在上下文中逐条直接撰写。
> - 拼音必须 AI 处理多音字（姓氏、地名、动名兼类、文言古汉语、轻声/儿化、一/不变调），禁止脚本默认首读。
> - 多语言必须 AI 按当前 KP 场景翻译，禁止脚本批量机翻，禁止机翻式直译。
> - 脚本只允许做导入、删除、校验、生成 manifest，禁止产出任何内容字段。
> 详见 [01-生成流程与目录约定.md §0 AI 直出铁律](./01-生成流程与目录约定.md#0-ai-直出铁律最高优先级所有内容字段适用)。

## 1. KP jsonl 通用字段（所有 7 类共用）

文件：`batch_xxx.kp.jsonl`，每行一条 KP。

```jsonc
{
  "__kp_local_id": "k1",                     // 本批次内唯一占位 1..32 字符
  "kp_type":       "word",                   // 7 选 1
  "primary_track": "ec",                     // share|ec|fc|hsk|dl，与 manifest.track 一致
  "title_zh":      "优惠券",                 // 1–60 字
  "pinyin":        "yōuhuìquàn",             // 带声调，非 yo1uhui4quan4
  "difficulty":    2,                        // 1–6，对齐所属 stage
  "hsk_level":     2,                        // 可空；hsk 主题必填
  "content":       { /* 见下文 7 类差异 */ },
  "translations": {
    "zh": "优惠券",
    "en": "coupon",
    "vi": "phiếu giảm giá",
    "th": "คูปองส่วนลด",
    "id": "kupon"
  },
  "tags":          ["ec", "stage2", "支付"]  // ≤ 20 字 / 元素
}
```

**禁止字段**：`id` / `kp_code` / `is_published` / `version` / `audio_url`（除非已知真实地址）/ `image_url` / `published_at` / `created_at` / `source_batch_id`。

**audio_url 占位**：如必须写则用 `cdn://placeholder/<lesson_code>/<__kp_local_id>.mp3`，导入后由 TTS 任务替换。

## 2. 7 类 content schema（必读）

### 2.1 `pinyin`（仅 Stage 0 / share 主题使用）

```json
{
  "components": { "initial": "zh", "final": "ōng", "tone": 1 },
  "is_whole_syllable": false,
  "common_chars": ["中", "钟", "终", "忠"],
  "tongue_diagram_url": null,
  "explanation_md": {
    "zh": "声母 zh 是翘舌音，舌尖抵硬腭前部；阴平第一声。",
    "en": "Initial 'zh' is a retroflex; tone 1 is high level.",
    "vi": "Phụ âm đầu 'zh' lưỡi cuộn; thanh 1 cao bằng.",
    "th": "พยัญชนะต้น 'zh' ม้วนลิ้น; เสียงที่ 1 ราบสูง.",
    "id": "Awalan 'zh' lidah ditekuk; nada 1 datar tinggi."
  }
}
```

字段：
- `components.initial` 声母（无声母则空字符串 `""`）
- `components.final` 韵母（带声调）
- `components.tone` 整数 1–4，5 表示轻声（轻声写 5 不写 0）
- `is_whole_syllable` 是否整体认读音节（zhi/chi/shi/ri/zi/ci/si/yi/wu/yu 等）
- `common_chars` 该音节常见汉字 3–6 个
- `tongue_diagram_url` 可选发音口型/舌位图

### 2.2 `hanzi`（Stage 0 / share 主题为主，主题层若需补单字也可用）

```json
{
  "stroke_count": 6,
  "structure": "left-right",
  "radicals": ["亻", "尔"],
  "stroke_order_svg": "cdn://placeholder/hanzi/你.svg",
  "stroke_order_anim_url": "cdn://placeholder/hanzi/你.json",
  "common_words": ["你好", "你们", "迷你"],
  "explanation_md": {
    "zh": "「你」由「亻」和「尔」组成，左右结构，意为对方。",
    "en": "「你」 'you' = person radical + 尔. Left-right structure.",
    "vi": "「你」'bạn' = bộ Nhân + 尔. Cấu trúc trái-phải.",
    "th": "「你」'คุณ' = บุคคล + 尔. โครงสร้างซ้าย-ขวา.",
    "id": "「你」'kamu' = radikal orang + 尔. Struktur kiri-kanan."
  }
}
```

字段：
- `stroke_count` 笔画数 1–48
- `structure` 枚举：`single` / `left-right` / `top-bottom` / `enclosure` / `left-middle-right` / `top-middle-bottom` / `half-enclosure`
- `radicals` 部首数组（一个汉字常 1–2 个）
- `stroke_order_svg` 笔顺 SVG 占位
- `stroke_order_anim_url` 笔顺动画（lottie/json）占位
- `common_words` 包含该字的常用词 3–6 个

### 2.3 `word`（主题层主力，约 60% KP）

```json
{
  "pos": "noun",
  "examples": [
    {
      "zh": "这家店有优惠券吗？",
      "pinyin": "Zhè jiā diàn yǒu yōuhuìquàn ma?",
      "audio_url": "cdn://placeholder/ec-2-3-1/k1-ex1.mp3",
      "tr": {
        "zh": "这家店有优惠券吗？",
        "en": "Does this shop have coupons?",
        "vi": "Cửa hàng này có phiếu giảm giá không?",
        "th": "ร้านนี้มีคูปองไหม?",
        "id": "Apakah toko ini punya kupon?"
      }
    },
    { /* 第 2 个例句，1–3 个例句 */ }
  ],
  "explanation_md": {
    "zh": "优惠券：商家发放的折扣凭证，使用后可减免订单金额。",
    "en": "Coupon: a discount voucher from the merchant.",
    "vi": "Coupon: phiếu giảm giá do người bán phát hành.",
    "th": "คูปอง: ใบลดราคาที่ร้านค้าออกให้.",
    "id": "Kupon: voucher diskon dari penjual."
  }
}
```

字段：
- `pos` 词性枚举：`noun` / `verb` / `adj` / `adv` / `pronoun` / `numeral` / `measure` / `prep` / `conj` / `aux` / `interj` / `onomat`
- `examples` 1–3 个例句，每例必带 zh/pinyin/tr（5 语言）

### 2.4 `phrase`

```json
{
  "pattern": "下 + 订单",
  "co_occurrences": ["在网上下订单", "帮我下个订单", "尽快下订单"],
  "examples": [
    {
      "zh": "请尽快下订单。",
      "pinyin": "Qǐng jǐnkuài xià dìngdān.",
      "audio_url": "cdn://placeholder/...",
      "tr": { "zh":"...", "en":"...", "vi":"...", "th":"...", "id":"..." }
    }
  ],
  "explanation_md": { "zh":"...", "en":"...", "vi":"...", "th":"...", "id":"..." }
}
```

字段：
- `pattern` 短语主结构（含 `+` 表占位）
- `co_occurrences` 常见搭配 3–5 个
- `examples` 1–2 个完整句

### 2.5 `grammar`

```json
{
  "pattern": "S + 把 + O + V + 其他",
  "rule_md": {
    "zh": "「把」字句强调对宾语的处置结果，V 后必须有补语或其他成分。",
    "en": "The 把 construction emphasizes how the object is dealt with.",
    "vi": "Câu 把 nhấn mạnh cách xử lý tân ngữ.",
    "th": "ประโยค 把 เน้นการจัดการกรรม.",
    "id": "Konstruksi 把 menekankan perlakuan pada objek."
  },
  "positive_examples": [
    "我把书放在桌子上。",
    "请你把窗户关上。"
  ],
  "negative_examples": [
    "我把吃饭。",         // 缺补语
    "我把书。"            // 缺动词与补语
  ],
  "contrast_with": ["被字句", "一般陈述句"]
}
```

字段：
- `pattern` 句法骨架（用 S/O/V 抽象 + 关键词）
- `rule_md` 5 语言讲解，每语言 ≤ 200 字
- `positive_examples` 正确例 2–4 句
- `negative_examples` 错误例 1–3 句（教学对比用）
- `contrast_with` 易混语法点 0–3 个

### 2.6 `sentence`

```json
{
  "scenario": "电商-售后",
  "function_tag": "申请退款",
  "reply_examples": [
    "好的，请提供订单号。",
    "退款已受理，3 个工作日到账。"
  ],
  "explanation_md": { "zh":"...", "en":"...", "vi":"...", "th":"...", "id":"..." }
}
```

`title_zh` 字段本身就是这条句子（如 `"我想申请退款。"`），content 不再重复句子文本。

字段：
- `scenario` 场景标签（与 lesson 标题相关）
- `function_tag` 言语功能（询问/请求/感谢/道歉/解释/确认/拒绝/抱怨/赞美/告别 等）
- `reply_examples` 1–3 个常见回复

### 2.7 `dialog`

```json
{
  "turns": [
    { "role": "buyer",  "zh": "你好，这个还有库存吗？", "pinyin": "Nǐ hǎo, zhège hái yǒu kùcún ma?", "audio_url": "cdn://placeholder/.../t1.mp3", "tr": { "en":"Hello, is this in stock?", "vi":"...", "th":"...", "id":"..." } },
    { "role": "seller", "zh": "有的，请问您要几件？",   "pinyin": "Yǒu de, qǐngwèn nín yào jǐ jiàn?", "audio_url": "cdn://placeholder/.../t2.mp3", "tr": { "en":"Yes. How many would you like?", "vi":"...", "th":"...", "id":"..." } },
    { "role": "buyer",  "zh": "我要两件，能便宜点吗？", "pinyin": "Wǒ yào liǎng jiàn, néng piányi diǎn ma?", "audio_url": "cdn://placeholder/.../t3.mp3", "tr": { /* ... */ } }
  ],
  "key_kp_refs": ["kp_ec_w_000231", "kp_ec_w_000232"],
  "explanation_md": { "zh":"...", "en":"...", "vi":"...", "th":"...", "id":"..." }
}
```

字段：
- `turns` 数组 4–8 轮，每轮 role 一般 `buyer/seller` / `customer/cs` / `boss/staff` / `friend_a/friend_b` 等
- 每 turn 必带 zh + pinyin + 5 语言 tr
- `key_kp_refs` 引用其它已存在 KP 的 `kp_code`（如果当前批次还没有 kp_code 用 `__kp_local_id`，导入脚本会替换；推荐只引用已发布的 KP）

`title_zh`：写对话主题（如 `"砍价对话"`）而非完整对话文本。

## 3. 7 类 KP 在 lesson 内的配比建议

| 阶段 | pinyin | hanzi | word | phrase | grammar | sentence | dialog |
|------|-------:|------:|-----:|-------:|--------:|---------:|-------:|
| share-0 | 70% | 25% | 0% | 0% | 0% | 5% | 0% |
| 主题 stage 1 | 0% | 5% | 65% | 15% | 5% | 7% | 3% |
| 主题 stage 2 | 0% | 5% | 60% | 15% | 8% | 7% | 5% |
| 主题 stage 3 | 0% | 5% | 55% | 15% | 10% | 8% | 7% |
| 主题 stage 4 | 0% | 0% | 50% | 15% | 12% | 10% | 13% |
| 主题 stage 5 | 0% | 0% | 45% | 12% | 13% | 12% | 18% |
| 主题 stage 6 | 0% | 0% | 40% | 10% | 15% | 15% | 20% |
| HSK 主题 | 0% | 5% | 70% | 5% | 15% | 5% | 0% |

每节固定 12 KP，按上表抽样取整：例如 ec-stage2 12 KP ≈ `1 hanzi + 7 word + 2 phrase + 1 grammar + 1 sentence`。

## 4. lesson_kp.jsonl 通用字段

文件：`batch_xxx.lesson_kp.jsonl`，每行一条绑定。

```jsonc
{
  "lesson_code":      "ec-2-3-1",
  "__kp_local_id":    "k1",                  // 与 kp.jsonl 的占位 ID 对应；二选一
  "kp_code":          "kp_share_p_000001",   // 引用已存在 KP；与 __kp_local_id 二选一
  "position":         1,                     // 1–30，节内顺序
  "is_new_in_lesson": true                   // true=本节首次教；false=复用旧 KP
}
```

约束：
- 每行 `__kp_local_id` 与 `kp_code` 必须二选一
- 同一 lesson_code 内 position 从 1 起连续
- 新建 KP（即同批次 kp.jsonl 含该 `__kp_local_id`）必须 `is_new_in_lesson=true`
- 复用已存在 KP（`kp_code` 形式）一般 `is_new_in_lesson=false`，复习节也可以 `true`

---

## 5. 跨批次复用规则

| 情况 | 处理 |
|------|------|
| 本批次新建 KP | kp.jsonl 写完整，lesson_kp.jsonl 用 `__kp_local_id` |
| 复用已发布 KP（如共享层进主题层） | kp.jsonl **不写**，lesson_kp.jsonl 用 `kp_code` |
| 复用同主题前节 KP（复习节） | 同上，`is_new_in_lesson=false` |
| 跨主题复用某场景词 | 不推荐自动决定；PM 显式给 kp_code 列表 |

## 6. 自检清单

- [ ] 每行 `kp_type` 在 7 类内
- [ ] `primary_track` 与 manifest.track 一致（除引用复用 KP）
- [ ] `pinyin` 带声调、不含数字（`yo1` 是错的）
- [ ] `translations` 5 语言齐
- [ ] `content` 按 kp_type 包含必需字段
- [ ] `examples`（如有）每例含 zh + pinyin + 5 语言 tr
- [ ] `difficulty` 与 stage 对齐（stage_no=2 → 多数 KP difficulty=2）
- [ ] HSK 主题 KP 必填 `hsk_level`
- [ ] Stage 0 节的 KP 全部为 `pinyin` 或 `hanzi`
- [ ] 主题节的 KP 不含 `pinyin`（如要复用拼音 KP，走 lesson_kp.kp_code 引用）
- [ ] `tags` 至少含 track 与 stage 标识（如 `["ec","stage2"]`）
- [ ] lesson_kp.jsonl 行数 = 该节 KP 数（含新建+复用）
- [ ] lesson_kp.position 连续从 1 起

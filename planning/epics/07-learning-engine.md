# Epic E07 · 学习引擎（Learning Engine）

> 阶段：M2 · 优先级：P0 · 估算：5 周

## 摘要
课程 / 阅读 / 游戏 共用的学习引擎：进度跟踪、SRS 复习、错题、HSK 评估、XP / 等级。

## 范围
- 学习状态机（开始 / 暂停 / 完成）
- 进度持久化
- SRS 复习算法（SM-2 简化）
- 生词本与错题本
- XP / 等级 / 连续学习
- HSK 自适应（v1）/ 自动评估（v1.5）

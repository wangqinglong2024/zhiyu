## E10 · 游戏增强项（v1 不做，全部延后）

> v1 里 12 款游戏统一遵守：60s 单局 / 无限连玩 / 无关卡 / 不发奖励。本节列举 **被显式踢出 MVP 的所有增强项**。

### v1.5 候选（产品验证 → 看数据决定优先级）
| 增量 | 说明 |
|---|---|
| 排行榜（日 / 周 / 全时） | 对接 GameAnalytics + leaderboard 表 |
| 三星评级 / 关卡星图 | 引入 `level` 概念，与 60s 单局并存 |
| 教学引导（30s tutorial）| 每款游戏独立 tutorial |
| 游戏内购 IAP | 道具 / 复活 / 解锁，对接 12-economy 已有 ZC + 13-payment |
| 排行榜分享 | 截图 + 邀请链接，对接 14-referral |
| 通过完成游戏发放 ZC | 对接 12-economy 的 earning rules（v1 严格禁用） |

### v2 候选
| 增量 | 说明 |
|---|---|
| 多模式 / 多难度 | 每款游戏 2-3 mode |
| 自定义词包 | 用户上传 / AI 生成 |
| 多人对战 | 实时 PvP（依赖 12-realtime-and-im 扩展）|
| 赛季 / 通行证 | 季节性内容运营 |

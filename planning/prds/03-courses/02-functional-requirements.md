# 3.2 · 系统课程 · 功能需求

## 一、用户故事

### US-CR-01：选择轨道
> 作为新用户，我希望在欢迎流程中选择 1-2 个学习轨道（HSK / 工厂 / 电商 / 日常），系统给我推荐合适的入门阶段。

### US-CR-02：完成节
> 作为学习者，我希望按知识点学习一节内容，并通过节小测确认我学会了。

### US-CR-03：进入下一节
> 作为学习者，我希望节小测通过后自动进入下一节。

### US-CR-04：章测 / 阶段考
> 作为学习者，我希望章末做综合测验，阶段末做大考，帮助我整体复习。

### US-CR-05：解锁付费内容
> 作为学习者，我希望登录后先试学每个主题 Stage 1-3 的全部章节，并可按需要跨级购买任意阶段（如电商 Stage 9）。

### US-CR-06：跨设备进度同步
> 作为用户，我希望手机学到第 5 节后，电脑上也能从第 5 节继续。

### US-CR-07：错题入库
> 作为学习者，我希望测验答错的题进入错题集，未来 SRS 系统帮我复习。

### US-CR-08：学习报告
> 作为学习者，我希望看到节 / 章 / 阶段末的学习报告，了解掌握度和薄弱点。

### US-CR-09：跳过节 / 章
> 作为有基础的用户，我希望跳过某节（不强制做小测），但完成度按规则计算。

### US-CR-10：拼音入门
> 作为零基础用户，我希望先做完拼音入门再进入主轨道。

## 二、功能需求

### CR-FR-001：轨道选择 / 推荐
- **优先级**：P0
- **位置**：欢迎流程（注册后第一次登录）
- **流程**：
  - Step 1：选学习目标（HSK 备考 / 工作 / 兴趣）
  - Step 2：选当前水平（零基础 / HSK 1-3 / HSK 4-6 / HSK 7+）
  - Step 3：系统推荐 1-2 轨道 + 起始阶段
  - Step 4：用户确认 / 调整
- **可跳过**：是，跳过则进 dashboard 让用户自选

### CR-FR-002：拼音入门前置
- **优先级**：P0
- **触发**：零基础用户首次进入主轨道
- **结构**：3 模块（声母 / 韵母 / 声调）+ 综合练习
- **题型**：P1 / P2 / P3（拼音入门专用）
- **完成度**：完成所有模块 + 综合练习 ≥ 80% → 标记完成
- **可跳过**：是，跳过弹"建议先学拼音"，确认即跳

### CR-FR-003：课程首页 Dashboard
- **优先级**：P0
- **路径**：`/learn`
- **元素**：
  - 当前学习中的轨道卡片（带进度条）
  - "继续学习"按钮（跳到上次未完成的节）
  - 今日任务（节 / 章 / 温故知新 推送）
  - 学习时长统计（本周）
  - 切换轨道 tab

### CR-FR-004：阶段总览
- **优先级**：P0
- **路径**：`/learn/:track/:stage`
- **元素**：
  - 阶段标题 + 介绍（母语）
  - 12 章网格（已完成 / 进行中 / 未解锁）
  - 阶段进度条
  - 阶段考入口（12 章全完成后解锁）

### CR-FR-005：章节总览
- **优先级**：P0
- **路径**：`/learn/:track/:stage/:chapter`
- **元素**：
  - 章节标题 + 介绍
  - 12 节列表（已完成 / 进行中 / 未开始）
  - 章测入口（12 节全完成后解锁）

### CR-FR-006：节学习页（核心）
- **优先级**：P0
- **路径**：`/learn/:track/:stage/:chapter/:lesson`
- **元素**：
  - 节标题 + 学习目标
  - 12 知识点切换（卡片 / 滑动 / 标签）
  - 每个知识点：中 + 拼音 + 母语 + TTS + 例句 + key_point
  - 进度条 + "下一个"按钮
  - 末尾"开始小测"

### CR-FR-007：节小测
- **优先级**：P0
- **结构**：10 题（10 类题型混合）
- **行为**：
  - 答题立即反馈（对错 + 解释）
  - 答错可重试（最多 2 次）
  - 完成后展示得分 + 错题列表
  - 错题入 SRS（详见 LE 模块）
- **通过标准**：≥ 60% 即标记节完成
- **奖励**：知语币 5-15（按得分）

### CR-FR-008：章测
- **优先级**：P0
- **结构**：36 题（节小测抽样 + 新综合题）
- **行为**：
  - 一次完成（不分批）
  - 时间限制 60 分钟（可暂停 1 次）
  - 答题完整后才出结果
  - 错题入 SRS
- **通过标准**：≥ 70% 标章完成
- **奖励**：知语币 30 + 章成就徽章

### CR-FR-009：阶段考
- **优先级**：P0
- **结构**：80-150 题（HSK 模考结构 / 工厂场景模拟 / 电商考核）
- **行为**：
  - 一次完成
  - 时间限制 120-180 分钟
  - 错题入 SRS
- **通过标准**：≥ 75% 标阶段完成
- **奖励**：知语币 100 + 阶段证书（PDF 下载）

### CR-FR-010：付费墙
- **优先级**：P0
- **触发**：访问免费试学范围外内容（默认每个主题 Stage 4 起，或任意未购阶段）
- **弹窗内容**：
  - 标题"解锁阶段 X"或"解锁本阶段"
  - 选项：单段 $4 / 9 段全包 $36 / 月会员 $4 / 年 $40
  - 限时促销 banner（半年 $12）
  - 推荐高亮（默认半年）
  - "继续免费试看"按钮（回到当前主题 Stage 1-3 免费范围）
- **行为**：
  - 选项点击 → PaymentAdapter dummy/fake checkout（本期不接真实支付）
  - 单段购买支持跨级：用户可直接购买任意 track/stage，不要求前置阶段完成
  - 试看：每个主题 Stage 1-3 全部章节计完成；免费范围外未授权只读不计完成

### CR-FR-011：进度同步
- **优先级**：P0
- **存储**：`learning_progress` 表
- **同步频率**：
  - 知识点切换：1s 防抖后写
  - 节小测完成：立即写
  - 章测 / 阶段考：立即写
- **跨设备**：登录后立即拉取最新

### CR-FR-012：错题集
- **优先级**：P0
- **存储**：`learning_wrong_set` 表
- **入库时机**：
  - 节小测错（即时）
  - 章测错（提交后批量）
  - 阶段考错（提交后批量）
  - 游戏错（即时，详见 GM）
- **去重**：同一 question_id 重复错只保留最新

### CR-FR-013：学习报告
- **优先级**：P1
- **节末报告**：
  - 节小测得分
  - 错题列表
  - 知识点掌握度（基于 SRS 状态）
  - 推荐：下一节 / 复习 / 游戏强化
- **章末报告**：章测得分 + 章节统计 + 推荐
- **阶段末报告**：阶段考 + 阶段统计 + 证书 + 下阶段预告

### CR-FR-014：跳过节
- **优先级**：P1
- **触发**：节学习页"跳过此节"
- **行为**：弹确认 → 标记 `skipped` + 进度计算视为完成
- **限制**：
  - 跳过的节不计入完成（学习报告标注）
  - 跳过的节小测可后补

### CR-FR-015：模拟试卷（HSK 真题）
- **优先级**：P2（v1.5 评估）
- **位置**：HSK 轨道 阶段 11/12
- **来源**：近 5 年 HSK 4/6/9 真题（合规许可）

### CR-FR-016：学习时长统计
- **优先级**：P0
- **算法**：用户在节学习页 / 测验页 时长累计（间隔 5 秒检查活跃）
- **展示**：今日 / 本周 / 累计
- **奖励**：日 30 分钟 → 知语币 +10

### CR-FR-017：切换轨道
- **优先级**：P0
- **位置**：dashboard tab + 设置
- **多轨道并行**：用户可同时学多轨道（不限）

### CR-FR-018：知识点收藏
- **优先级**：P1
- **行为**：知识点右上角心形 → 收藏
- **个人中心**：`/me/favorites?type=knowledge_point`

### CR-FR-019：知识点笔记
- **优先级**：P1
- **行为**：知识点菜单 → 笔记
- **限制**：500 字

### CR-FR-020：内容报错
- **优先级**：P1
- **行为**：知识点 / 句子菜单 → "报错"
- **报错类型**：拼音错 / 翻译错 / 音频问题 / 内容不当
- **流程**：进入审校工作台

## 三、UX 流程图

### 3.1 首次登录
```
注册 → 邮箱验证 → 欢迎引导
                ↓
         选学习目标
                ↓
         选当前水平
                ↓
         系统推荐轨道
                ↓
         用户确认
                ↓
[零基础] → 拼音入门 → 主轨道
[有基础] → 主轨道
```

### 3.2 节学习
```
进入节 → 知识点 1 → 知识点 2 → ... → 知识点 12
                                              ↓
                                          节小测
                                              ↓
                                    [≥60%] 完成 → 下一节
                                    [<60%] 重做或跳过
```

### 3.3 付费墙
```
访问免费试学范围外内容 → 检查权限
                ↓
            [已购] 正常进入
            [未购] 付费墙弹窗
                    ↓
      [试看] 回当前主题 Stage 1-3 免费范围
      [付费] PaymentAdapter dummy/fake checkout
                    ↓
         fake webhook / 后台授权 → 解锁
```

## 四、UI 关键元素

### 4.1 知识点卡片
- 中文（特大字号 / 加粗）
- 拼音（中字号 / 灰）
- 母语翻译（中字号）
- 词性 + HSK 等级标签
- 例句 2-3 个（折叠）
- key_point（折叠 / 母语）
- 收藏 / 笔记 / 报错按钮

### 4.2 题目组件（10 + 3 题型）
- 见 [`03-question-types.md`](./03-question-types.md)

### 4.3 进度条
- 节内：12 知识点小圆点 + 当前位置
- 章内：12 节进度
- 阶段内：12 章进度
- 轨道内：12 阶段进度

### 4.4 付费墙
- 大标题 + 解锁段名
- 4 选项卡片（推荐高亮）
- 限时倒计时（半年促销）
- "继续免费"低显眼按钮

## 五、非功能要求（模块特定）

### 5.1 性能
- 节学习页 LCP < 2s
- 知识点切换 < 200ms
- 题目渲染 < 100ms
- 测验提交 < 500ms

### 5.2 离线支持（PWA）
- 已学过的节缓存（30 天）
- 离线可继续学习
- 联网后自动同步进度

### 5.3 错误处理
- 网络中断时自动暂存到 IndexedDB
- 重连后批量同步

进入 [`03-question-types.md`](./03-question-types.md)。

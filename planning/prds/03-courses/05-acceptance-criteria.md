# 3.5 · 系统课程 · 验收准则

## 四、关键测试用例

### 用例 1：免费章访问
- 登录未付费用户访问 HSK Stage 1 Chapter 1-3 → 可学习
- 访问 HSK Stage 1 Chapter 4 → 付费墙

### 用例 2：跨级单段购买
- 用户直接买电商 Stage 9（$4）→ 电商 Stage 9 全部可见，游戏可选择电商 S9 词包
- 电商 Stage 8 / Stage 10 仍付费墙，除非另购或会员

### 用例 3：会员
- 用户买月会员 → 4 轨道全 12 阶段可见
- 月会员到期 → 阶段 4+ 重新付费墙

### 用例 4：跨设备进度
- 手机学到 stage1/ch1/lesson5
- 桌面登录 → dashboard "继续"按钮跳到 lesson5

### 用例 5：节小测错题入库
- 答错题 Q-XYZ
- 立即查 `learning_wrong_set` → 含 Q-XYZ
- 温故知新次日 → 包含 Q-XYZ

### 用例 6：章测时间限制
- 开始章测 → 计时 60min
- 暂停 1 次（最多） → 暂停时间不计入
- 时间到 → 自动提交当前答案

### 用例 7：阶段证书
- 通过阶段考 → 报告页显示"下载证书"
- PDF 含：用户名 / 阶段名 / 通过日期 / 得分 / 二维码（验真）

### 用例 8：内容报错
- 用户报"拼音错"
- 进入审校工作台
- 审校确认 → 修正 → 用户得通知 + 知语币奖励 5

# 6.3 · 用户账号 · 验收准则

## 功能
- [ ] UA-AC-001：邮箱注册成功 + 100 币入账 + 验证邮件发送
- [ ] UA-AC-002：Google OAuth 登录正常
- [ ] UA-AC-003：OTP 验证 6 位 / 15min / 重发限流
- [ ] UA-AC-004：登录失败 5 次锁 15min
- [ ] UA-AC-005：忘记密码邮件有效 10min；重置后所有 session 登出
- [ ] UA-AC-006：个人资料修改保存
- [ ] UA-AC-007：偏好实时生效（拼音模式 / 字号 / TTS 等）
- [ ] UA-AC-008：会话列表 + 强制下线
- [ ] UA-AC-009：修改密码 / 邮箱 OK
- [ ] UA-AC-010：数据导出 JSON / CSV 完整
- [ ] UA-AC-011：销户 90 天恢复期
- [ ] UA-AC-012：欢迎流程引导
- [ ] UA-AC-013：匿名与登录访问规则（匿名 DC 前 3 类目 / NV 首章；登录后全部 DC、全部 NV、全部 GM、CR 每个主题 Stage 1-3 全部章节免费试学）

## 非功能
- [ ] 注册 P95 < 800ms
- [ ] 密码 bcrypt rounds=12
- [ ] JWT HttpOnly / Secure / SameSite
- [ ] PDPL / PDPA / UU PDP 合规（导出 / 删除 / 同意）
- [ ] 暴力破解防护

## 测试用例
1. 同 IP 6 次失败登录 → 第 6 次锁定 15min
2. OTP 错 5 次 → 失效 + 重发
3. 销户后立即尝试登录 → 拒绝
4. 销户 60 天后客服恢复 → 数据完好
5. 销户 91 天后 → 用户表硬删，FK 清理
6. 数据导出含全部 12 表
7. Google 登录新用户 → 创建 + 引导

# 7.1 · 学习引擎 · 功能需求

## 用户故事
- US-LE-01：每天打开能看到今日复习推荐
- US-LE-02：错题集中专项练习
- US-LE-03：可视化掌握度（哪些 HSK / 主题强 / 弱）
- US-LE-04：连续学习日（streak）激励
- US-LE-05：学习提醒推送

## 功能需求

### LE-FR-001：温故知新（SRS）
- 路径 `/learn/review`
- 入口：dashboard "今日待复习 N 张"
- 题目流：到期题目（FSRS 调度）
- 用户答题后选评分：忘了 / 困难 / 一般 / 简单（FSRS 4 档）
- 后端用 ts-fsrs 计算下次到期
- 每日上限默认 20，可在偏好调到 50 / 100

### LE-FR-002：错题专攻
- 路径 `/learn/wrong-set`
- 来源筛选：全部 / 课程 / 游戏
- 排除：小说与发现中国不作为复习数据来源
- HSK 等级筛选
- 题目展示 + 答题 + 即时反馈
- 答对 2 次 → 标记 resolved（出错题集，但仍在 SRS 中循环）

### LE-FR-003：今日新题
- 路径 `/learn/new`
- 推荐来源：当前轨道下一节预习题
- 限：每日 10 题（避免与课程节小测冲突）

### LE-FR-004：自由练习
- 路径 `/learn/practice`
- 用户选筛选：HSK 等级 / 主题 / 题型 / 轨道
- 系统从学习系统 question 库随机出题
- 不计入 SRS（除非答错）

### LE-FR-005：学习仪表板
- 路径 `/me/dashboard`
- 元素：
  - 今日：完成题数 / 学习时长 / 知语币
  - 本周热力图（活跃天数）
  - 累计：词汇量 / 节数 / 章数 / 阶段数
  - 掌握度热力图（HSK 1-9 × 主题）
  - 连续学习日（streak）+ 最长 streak

### LE-FR-006：连续学习日（streak）
- 当日完成 ≥ 1 次 review / quiz / lesson 算 1 天
- 跨时区按用户时区
- 中断保护：streak freeze 道具（EC，50 币 / 张）
- 7 / 30 / 100 天里程碑奖励币 + 徽章

### LE-FR-007：学习提醒
- 用户设置每日提醒时间
- v1：邮件提醒
- v1.5：浏览器 Web Push
- 触发：到时未学习 → 提醒
- 已学完则不发

### LE-FR-008：FSRS 参数管理
- 默认参数：FSRS-5 标准参数
- 用户可重置个人 SRS 状态（清空所有 cards）
- 管理员可调全局参数

### LE-FR-009：薄弱点诊断
- 仪表板"薄弱知识点"卡片
- 算法：SRS 状态为 'learning' 或重错次数 > 2 的 KP
- 一键"开始专项练习"

## UX
- 答题界面统一组件（复用课程小测）
- 每题反馈：对 / 错 + 解释 + KP 跳转链接
- 完成 N 题展示总结：正确率 / 用时 / 币奖励

## 性能
- SRS 调度查询 P95 < 200ms
- 题目加载 < 300ms

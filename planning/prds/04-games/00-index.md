## 关键决策
- 强制横屏（force-landscape，桌面也横屏画布）
- PixiJS v8 + Howler.js + Matter.js（物理）
- 词汇库取自课程知识点（已学单元优先）
- 知语币奖励：通关 +5~20，三星 +10
- 成绩排行榜：日 / 周 / 全时（每游戏独立）
- 错题入 SRS（游戏中答错知识点）
- 不限免费玩，但每日通关币奖励上限 50 / 游戏

## 与其他模块关系
- 依赖：CR（词汇源）/ LE（错题）/ EC（币奖励）/ UA / I18N / SC
- 提供：错题数据 → LE / 排行榜 → 用户激活

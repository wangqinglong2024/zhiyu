# 10 · 游戏 UX

## 一、v1 范围

12 款游戏全部 MVP 首发、全部 active。共同规则：强制横屏、固定 60 秒单局、无限连玩、错题入 SRS。v1 不做排行榜、三星评级、关卡、道具消耗、知语币奖励或 IAP。

## 二、横屏进入

1. 点击“开始游戏”。
2. 若已横屏，进入画布。
3. 若竖屏，显示 OrientationMask，引导旋转。
4. 支持 Screen Orientation API；不支持时只提示，不强制破坏页面。
5. 退出游戏后解除方向锁定。

OrientationMask 使用 `.glass-elevated`，中央设备旋转图标、简短说明、取消按钮。

## 三、画布布局

- 基准：1280×720。
- 适配：等比缩放，剩余区域显示夜墨/宣纸 `surface-wash`。
- 安全区：左右 `max(16px, safe-area-inset-left/right)`。
- PixiJS 画布透明背景，HUD 用 `.glass-ink`。

## 四、HUD

Top：分数、时间、词包/HSK、暂停。

Bottom：当前题/输入区/虚拟控制，按游戏类型变化。

暂停面板：继续、重新开始、音效、音乐、退出。退出需确认“当前单局将结束”。

## 五、输入

桌面：方向键/WASD、Space、Enter、Esc、R、M，按游戏提供映射。

移动端：点击、滑动、长按、虚拟方向键、拼音键盘、数字声调按钮。所有控件命中区 ≥ 44dp。

## 六、12 游戏交互锚点

| 游戏 | 核心输入 | 视觉注意 |
|---|---|---|
| 汉字忍者 | 拖拽切割 | 刀光克制，避免血腥 |
| 拼音射手 | 键入拼音 | 字母/声调高对比 |
| 声调泡泡 | 1-4/点击 | 色弱模式用形状 |
| 汉字方块 | 方向/旋转 | 网格稳定不抖动 |
| 打汉字 | 点击/数字 | 目标足够大 |
| 汉字三消 | 拖拽 | 低端设备减少粒子 |
| 汉字贪吃蛇 | 方向 | 路径清晰 |
| 汉字节奏 | 按键 | 闪烁 < 3Hz |
| 汉字跑酷 | 跳/滑 | 前景背景分离 |
| 拼音塔防 | 放置/选择 | 信息面板可读 |
| 记忆翻牌 | 点击 | 翻牌动效可关闭 |
| 汉字弹弓 | 拖拽瞄准 | 轨迹线清晰 |

## 七、结果页

显示：分数、正确率、复习词、错题数量、再玩一次、换词包、回列表、分享战绩图。分享图可含分销链接二维码，但不展示纯邀请码。

## 八、可访问性

- 字幕/文字提示。
- 慢速模式 0.5/0.75。
- 色弱模式。
- 关闭闪烁与高强度动效。
- 桌面端必须键盘可玩。

## 九、性能预算

- 首次画布显示 ≤ 800ms。
- 中端机 60fps，最低不得低于 30fps。
- 首屏游戏资源 ≤ 5MB。
- 单局内不拉取非必要资源。

## 十、验收

- [ ] 12 游戏入口、横屏、60 秒单局全部跑通。
- [ ] v1 无排行榜/三星/关卡/奖励/IAP。
- [ ] 失焦自动暂停，退出解除方向锁。
- [ ] 错题进入 SRS 事件链路可验证。

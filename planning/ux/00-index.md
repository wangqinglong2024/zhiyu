# 00 · UX 规范总览

> 本目录是知语 Zhiyu 的产品体验事实源。实现时若与 `planning/rules.md` 冲突，以 `planning/rules.md` 为准；若与业务 PRD 冲突，以业务 PRD 的功能边界为准，再用本 UX 规范定义呈现方式。

## 一、设计语言

**松烟雅瓷 · Ink Porcelain Glass**

知语不是普通教育工具，也不是节庆风贴图集合。整体气质应像一件现代交互产品被放在宣纸、瓷釉、松烟墨、青瓷与朱砂印之间：清透、克制、有古韵，但仍然快速、可读、可操作。

核心表达：
- 现代：PWA、后台、游戏与学习流程必须高效、清晰、反馈及时。
- 古典：色彩、留白、线条、纹理、章法来自宣纸、瓷器、墨色、印章、窗棂、书卷，而非龙、灯笼、红金大字等刻板符号。
- 毛玻璃：只作为“瓷釉/雾面玻璃”材质，不做普通彩色玻璃卡片堆叠。
- 文化感：优先来自内容、排版与材质，而不是装饰图案。

## 三、全局产品边界

- 应用端首屏必须是可用体验，不做营销落地页式 Hero。
- 发现中国必须严格支持 `content/china/00-index.md` 的 12 类目、访问模型与模块联动。
- 游戏 MVP 遵守 PRD：12 款全部 active、强制横屏、固定 60 秒单局、无限连玩、错题入 SRS；v1 不做排行榜、三星、关卡奖励或知语币发放。
- 后台是高密度工作台，不做大幅装饰卡片和营销式留白。
- 所有字体、图片、音频、脚本、监控、测试都采用本地/Docker/自托管或 Adapter 路径，不接外部托管 SaaS。

## 四、视觉关键词

| 关键词 | 落地方式 |
|---|---|
| 宣纸 | 温润底色、细颗粒纹理、轻微纤维噪声 |
| 松烟 | 墨黑文字、低饱和雾色背景、层次沉稳 |
| 雅瓷 | 青瓷/玉色高光、半透明釉面玻璃、细边框 |
| 朱砂 | 仅用于主 CTA、危险/重点标记、印章式小面积强调 |
| 金石 | 年代感金色只做细线、徽章、成就，不大面积铺色 |
| 云水 | 背景用低对比水墨雾带，不用彩色渐变 blob |

## 五、验收口径

任何页面验收都要同时回答：
- 是否一眼能看出这是“知语”的文化学习产品，而不是通用模板？
- 是否没有用龙、灯笼、中国结、红金大字等廉价符号堆砌？
- 毛玻璃是否像瓷釉/薄雾，而不是普通半透明白卡片？
- 文字、按钮、表格、学习内容在移动端与桌面端是否清晰可读？
- 是否符合 `planning/rules.md` 的 Docker-only、自托管与本地验证边界？

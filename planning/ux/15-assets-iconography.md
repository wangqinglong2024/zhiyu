# 15 · 资产与图标

## 一、Logo

- 主 Logo：知语 Zhiyu，松烟墨文字 + 朱砂小印记。
- Logo mark：可抽象为“知/语”印章、书页与声波结合。
- 变体：全色、单色墨、单色白、仅 mark。
- 安全留白：至少 Logo 高度 50%。

## 二、PWA 图标

必备尺寸：512 maskable、192 maskable、180 apple-touch-icon、32 favicon、16 favicon。图标底色用 paper 或 ink，主体用 cinnabar/jade 小面积组合。

## 三、图标库

- 主库：`lucide-react`。
- 尺寸：16、20、24、32。
- stroke width：1.5 默认，2 强调。
- 自定义图标只用于确实缺失的文化类目，存放 `system/packages/ui/src/icons`。

## 四、发现中国类目图标

图标应抽象、线性、克制：
- 历史：碑拓/时间线。
- 美食：碗/瓷盘/蒸汽。
- 名胜：山水轮廓。
- 节日：节气圆盘。
- 艺术：笔触/瓷片。
- 音乐：弦线/音波。
- 文学：书页。
- 成语：印记/短签。
- 哲学：圆相/竹简线。
- 当代：城市线。
- 趣味汉字：字形演变。
- 神话：云纹/星图。

禁止把龙、灯笼、中国结、扇子作为默认类目视觉。

## 五、插画

- 风格：线描 + 宣纸质感 + 青瓷/朱砂点缀。
- 用途：空态、错误、引导、课程完成。
- 格式：SVG 或小体积 Lottie；单个 Lottie < 100KB。
- 文件：`system/packages/ui/src/illustrations`。

## 六、Cover 图

- 用途：文章、课程、小说、游戏详情。
- 比例：16:9 默认，21:9 hero，1:1 缩略。
- 来源：本地导入、设计资产、可商用授权图片或自有生成流程产物。
- 处理：本地脚本输出 WebP/AVIF、srcset、blurhash/dominant color。
- 存储：Supabase Storage 自托管 bucket，经应用 API 或 nginx 静态路由访问。

## 七、音频

- 学习音频：句子级、单词级，mp3/aac，私有存储。
- UI 音效：点击、答对、答错、完成、Toast，总量 < 500KB。
- BGM 默认关闭，用户可开启。
- 访问受限音频必须走后端签名/代理，不直接暴露私有 URL。

## 八、分享图

- 1200×630，使用本地 Canvas/SVG 渲染。
- 受限内容不泄露正文/金句。
- 可含分销链接二维码，不展示纯邀请码。

## 九、许可记录

所有第三方图标、字体、音效、图片、Lottie 必须记录许可来源；未知授权资源不得进入仓库。

## 十、验收

- [ ] 图标、插画、封面、音频均有本地或自托管路径。
- [ ] 发现中国 12 类目有统一但不刻板的视觉锚点。
- [ ] 图片/音频优化符合性能预算。
- [ ] 资产许可记录完整。

# 08 · 反馈与状态组件（Feedback & State）

## 一、Toast

### 1.1 类型
| type | 颜色 | 图标 | 自动消失 |
|---|---|---|---|
| `success` | green | ✓ | 3s |
| `info` | sky | ℹ | 3s |
| `warning` | amber | ⚠ | 5s |
| `error` | red | ✕ | 手动（含关闭按钮） |
| `loading` | neutral | spinner | 直到完成 |

### 1.2 位置
- 应用端：顶部居中，下移 `safe-top + 12px`
- 后台：右上角

### 1.3 行为
- 最多堆叠 3 个，超出排队
- 滑动关闭（移动端）
- 进入：`slide-down 300ms ease-out`
- 退出：`slide-up + fade-out 200ms ease-in`

### 1.4 内容
- 单行文本（≤ 50 字）
- 含 1 操作按钮（如"撤销"）
- 含倒计时（loading 类型）

## 二、Banner

### 2.1 类型
- 顶部固定 banner（如离线提示、版本更新）
- 内联 banner（如付费墙提示）

### 2.2 视觉
```
┌─────────────────────────────────────────┐
│ ⚠ 网络连接已断开，部分功能不可用 [关闭]  │
└─────────────────────────────────────────┘
```
- 高度 36-44
- 背景按 type
- 关闭按钮 × （仅 dismissible 类型）

## 三、Modal Confirm

### 3.1 用途
- 删除确认
- 退出确认

### 3.2 结构
```
┌──────────────────────────┐

│                          │
│  确定要删除吗？            │

└──────────────────────────┘
```

## 四、Empty State

### 4.1 通用模板
- 插画 + 标题 + 描述 + 主 CTA + 次 CTA

### 4.2 场景
| 场景 | 插画 | 文案 | CTA |
|---|---|---|---|
| 无搜索结果 | 放大镜 | "没找到相关内容" | "重置筛选" |
| 无收藏 | 空书签 | "还没收藏任何内容" | "去发现" |
| 无笔记 | 空笔记本 | "笔记本还是空的" | "开始学习" |
| 无错题 | 大拇指 | "错题集为空，太棒啦！" | "去做题" |
| 网络错误 | 断线插头 | "网络好像断开了" | "重试" |
| 离线模式 | 云朵 | "离线模式，部分内容已缓存" | - |
| 404 | 迷路插画 | "找不到这个页面" | "回首页" |
| 加载失败 | 哭脸 | "加载失败" | "重试" |

## 五、Skeleton Loading

### 5.1 通用块
- `<Skeleton.Line>` 文本行
- `<Skeleton.Block>` 矩形块
- `<Skeleton.Circle>` 圆形（头像）
- `<Skeleton.Card>` 完整卡片骨架

### 5.2 应用
- 列表加载：3-5 个骨架卡片
- 详情加载：含 cover + 标题 + 段落
- 表格加载：行数与预期一致

### 5.3 动画
- shimmer 1.5s 循环
- 减弱动效模式：静态灰色

## 六、Loading 状态

### 6.1 全屏 Loading
- 仅启动 / 切语言
- 中央 logo + 旋转圆环
- 背景毛玻璃

### 6.2 区域 Loading
- 替换为 Skeleton
- 不用 spinner（除非 < 1s）

### 6.3 按钮 Loading
- 替换内容为 spinner
- 文字保留 + 灰
- 禁用点击

### 6.4 Inline Loading
- 加载更多按钮
- 表格分页加载

## 七、错误状态

### 7.1 表单错误
- 字段下方红字 micro
- 字段边框 red-500
- 实时显示

### 7.2 页面错误
- ErrorBoundary 捕获 → 显示 ErrorFallback
- 含错误信息（生产隐藏栈）+ 重试 + 回首页

### 7.3 API 错误
- Toast error 显示
- 401 → 跳转登录
- 403 → 提示无权限
- 404 → 跳转 NotFound
- 500 → "服务异常，稍后再试"

## 八、Confirmation Patterns

### 8.1 撤销 Toast
某些操作（删除、归档）后显示 Toast 含"撤销"，5s 内可撤销。

### 8.2 二次确认
- 危险操作（销户）二次确认
- 严重危险（清空收藏）三次确认（输入特定字符）

## 九、空网络处理

### 9.1 离线检测
```ts
window.addEventListener('online', () => …);
window.addEventListener('offline', () => …);
```

### 9.2 离线 UI
- 顶部 banner 显示"离线中"
- 不可点击的功能 disable + tooltip
- 已缓存内容（PWA）正常显示

## 十一、动效反馈

| 操作 | 动效 |
|---|---|
| 点击按钮 | 涟漪 + 按下 95% scale 100ms |
| 点击卡片 | 浮起 -1px |
| 切换 Tab | 横向滑动 200ms |
| 提交成功 | 绿色 ✓ scale-in 400ms |
| 错误抖动 | shake 300ms |
| 完成清单项 | 划线 + fade 200ms |

## 十二、触觉反馈（移动端）

iOS Haptic / Android Vibration：
| 操作 | 强度 |
|---|---|
| 点击按钮 | light |
| 长按菜单 | medium |
| 错误 | error |
| 成功 | success |
| 切换 Tab | light |
| 滑动到底 | medium |

# 11 · 后台屏幕规格（Screens · Admin）

## 一、登录

### 1.1 视觉
```
┌──────────────────────────────────────┐
│                                      │
│    [Logo] 知语 Admin                  │
│                                      │
│    ┌──────────────────────────┐     │
│    │ 邮箱                      │     │
│    │ 密码                      │     │
│    │ [□ 记住我]                 │     │
│    │ [登录]                    │     │
│    └──────────────────────────┘     │
│                                      │
│    [TOTP 二步验证（如启用）]           │
└──────────────────────────────────────┘
```

### 1.2 安全
- 失败 5 次锁定 15min
- TOTP 二步验证（管理员强制）
- IP 白名单（可选）

## 二、Dashboard 仪表板 `/admin`

### 2.1 KPI 卡
- 今日 DAU / 7日 DAU / 30日 DAU
- 今日新增用户
- 本月收入 / 同比
- 转化率（注册→付费）
- 7日留存
- 活跃订阅数
- 今日工厂产出（文章 / 小说章节）
- 客服未读数

### 2.2 图表
- 7日 DAU 折线
- 7日收入柱状
- 渠道来源饼图
- 付费用户漏斗

### 2.3 待办区
- 待审稿件 N
- 待回复客服 N
- 待审分销提现 N
- 系统告警 N

## 三、内容管理 - 文章 `/admin/content/articles`

### 3.1 列表
```
┌──────────────────────────────────────────────────┐
│ [筛选: 类目 / 状态 / 难度 / 作者 / 日期]            │
│ [搜索]              [批量操作 ▾] [+ 新建]          │
├──────────────────────────────────────────────────┤
│ ☐ 标题       │ 类目 │ 状态 │ 难度 │ 作者 │ 操作    │
├──────────────────────────────────────────────────┤
│ ☐ 长城历史    │ 历史 │ 已发布│ HSK4│ AI  │[查看]  │
│ ☐ 川菜入门    │ 美食 │ 待审 │ HSK3│ AI  │[审校]  │
│ ☐ 春节风俗    │ 节日 │ 草稿 │ HSK2│ KING│[编辑]  │
└──────────────────────────────────────────────────┘
[Pagination]
```

### 3.2 编辑器 `/admin/content/articles/:id/edit`
- 双栏：左侧编辑（Markdown / WYSIWYG），右侧预览
- 字段：标题 / 类目 / HSK / 摘要 / 封面 / 标签
- 句子级编辑：句子 / 拼音 / 翻译（5 种语言）/ 音频
- 操作：保存草稿 / 提交审校 / 发布 / 删除 / 复制 / 历史版本
- 自动保存每 30s

## 四、内容管理 - 课程 `/admin/content/courses`

### 4.1 树形结构
```
4 轨道
└─ 12 阶段
   └─ 12 章
      └─ 12 节
         └─ 多个步骤
```
左侧树形导航，右侧编辑当前节点。

### 4.2 节编辑
- 步骤序列编辑器（拖拽排序）
- 步骤类型：句子 / 单词卡 / 选择 / 听力 / 跟读 / 翻译 / 排序 / 拖拽
- 每步骤含内容 + 评分规则

## 五、内容管理 - 小说 `/admin/content/novels`

### 5.1 列表
- 小说列表（含章节进度 12/120）
- 状态：连载中 / 已完结 / 暂停
- 章节列表展开

### 5.2 章节编辑器
- 与文章编辑器类似
- 含章节序号 / 上下章导航
- 角色字典侧栏（保持人物设定一致）

## 六、内容管理 - 游戏词包 `/admin/content/games/packs`

### 6.1 列表
- 词包名 / 关联游戏 / 词数 / 状态
- 系统词包 vs 用户自定义

### 6.2 编辑
- 词条列表（汉字 / 拼音 / 翻译 / 音频）
- 批量导入 CSV
- AI 一键生成（按主题）

## 七、内容工厂 `/admin/factory`

### 7.1 总览
- 各 Agent 状态（在线 / 忙 / 离线）
- 今日产出量
- Token 消耗 + 成本
- 队列长度

### 7.2 任务发起
```
[选择内容类型: 文章 / 课程节 / 小说章 / 游戏词包]
[选择主题 / 类目]
[选择 HSK 级]
[选择 AI 模型: Claude / DeepSeek]
[预设 Prompt 模板 ▾]
[自定义补充指令...]
[预览成本: $0.12]
[启动]
```

### 7.3 任务监控
- LangGraph 节点可视化
- 实时日志
- 中间产物预览
- 失败重试

### 7.4 模板管理
- Prompt 模板库
- 版本管理
- A/B 测试

- 待审稿件按 HSK 级 / 类目分组

## 九、用户管理 `/admin/users`

### 9.1 列表
- 筛选：注册日期 / 国家 / 等级 / VIP / 活跃度
- 搜索：邮箱 / 手机 / ID / 昵称
- 列：头像 / 昵称 / 注册日 / 国家 / VIP / 最近活跃 / 操作

### 9.2 详情
- 基本资料
- 学习数据（轨道 / 阶段 / 时长 / 准确）
- 订阅 / 支付历史
- 知语币流水
- 分销关系（上级 / 下级）
- 操作日志
- 客服会话历史
- 操作：封禁 / 解封 / 重置密码 / 调整知语币 / VIP 赠送

## 十、订单管理 `/admin/orders`

### 10.1 列表
- 订单号 / 用户 / 商品 / 金额 / 状态 / 渠道 / 时间
- 状态：待支付 / 已完成 / 已退款 / 失败 / 部分退款

### 10.2 退款流程
- 退款申请 → 审批 → 调用 Paddle API
- 退款记录留痕

## 十一、订阅 `/admin/subscriptions`

### 11.1 当前订阅
- 用户 / 套餐 / 续费日 / 状态
- 取消 / 修改

## 十二、知语币 `/admin/coins`

### 12.1 概览
- 总流通量
- 今日发放 / 消耗
- 用户余额 Top 100

### 12.2 流水
- 全量交易日志
- 按类型筛选（充值 / 兑换 / 奖励 / 退款）

### 12.3 兑换商品
- 商品列表（增 / 删 / 改 / 启用）
- 库存管理

## 十三、分销 `/admin/referrals`

### 13.1 总览
- 累计邀请 / 转化率 / 累计返佣
- 活跃推广员 Top 100

### 13.2 提现审批
- 待审 / 已审 / 已发
- 单笔上限 / 风控规则

### 13.3 推广素材
- 海报 / 文案库
- 多语言版本

## 十四、客服 `/admin/support`

### 14.1 IM 工作台
- 三栏：会话列表 / 当前对话 / 用户档案
- 标签 / 置顶 / 转接

### 14.2 工单
- 列表 / 优先级 / SLA
- 处理流程

### 14.3 知识库 / FAQ
- 多语言
- 类目管理

## 十五、Feature Flags `/admin/flags`

### 15.1 列表
- Flag 名 / 描述 / 状态 / 灰度比例 / 国家 / 平台
- 编辑历史

### 15.2 编辑
- 名称 / 描述
- 启用 / 关闭
- 灰度规则（百分比 / 用户白名单 / 国家 / 设备）
- 上线 / 回滚一键

## 十六、操作审计 `/admin/audit`

- 全管理员操作日志
- 不可篡改（追加写入）
- 筛选：操作员 / 操作类型 / 资源 / 时间
- 导出 CSV

## 十七、系统设置 `/admin/settings`

- 全局参数
- 服务密钥管理
- 邮件模板
- 公告管理
- 系统状态

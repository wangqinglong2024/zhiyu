## 二、Onboarding（首次进入）

### 2.1 步骤
1. 选语言（zh/en/vi/th/id 五选一）
2. 学习目标（旅游 / 工作 / 兴趣 / 留学）
3. 当前水平（HSK 0-9 自评）
4. 每日时长（5 / 15 / 30 min）
5. 引导注册或试用

### 2.2 视觉
- 全屏单步骤
- 顶部进度点
- 底部"下一步" CTA + 跳过

## 三、登录 / 注册

### 3.1 入口
- 邮箱 + 密码
- 第三方：Google / Apple / TikTok（v1.5）
- 手机号（v2，仅东南亚号段）

### 3.2 注册流程
1. 邮箱 + 密码
2. 验证码
3. 完善档案（昵称 / 头像）
4. → Onboarding

### 3.3 视觉
- 居中卡片 `.glass-elevated`
- Logo + 标题
- 表单
- 第三方按钮
- 切换登录/注册链接

### 7.2 设置页 `/profile/settings`
- 主题（亮 / 暗 / 跟随系统）
- 显示语言（5 种）
- 学习语言（中文）
- 通知偏好（学习提醒 / 新课程 / 客服 / 营销）
- 拼音模式（彩色 / 数字 / 隐藏）
- 字号
- 数据使用（清缓存 / 导出 / 删除账号）

### 7.3 收藏 `/profile/favorites`
- 三 Tab：文章 / 课程 / 小说
- 列表显示

### 7.4 笔记 `/profile/notes`
- 时间线 / 来源筛选
- 长按编辑 / 删除

### 7.5 错题 `/profile/mistakes`
- 按学科 / 难度筛选
- 立即复习

### 7.6 知语币 `/profile/coins`
- 余额 + 月增长
- 账本流水（获得 / 消耗）
- 充值 / 提现入口
- 兑换商品列表

### 7.7 分销 `/profile/referral`
- 我的分销码
- 二维码 + 复制链接
- 已邀请人数 + 收益
- 推广素材库
- 提现按钮

### 7.8 订单 `/profile/payments`

- 发票（v1.5）
- 退款入口

### 7.9 客服 `/profile/support`
- FAQ
- 在线 IM
- 工单
- 反馈

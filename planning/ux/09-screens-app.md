# 09 · 关键屏幕规格 · 应用端

## 一、启动屏

- 背景：`surface-wash`，宣纸纹理 + 低对比水墨雾带。
- 中央：Logo mark、知语 Zhiyu、短 slogan。
- 进度条：jade/celadon 细线，不做大面积彩色渐变。
- 目标：1.5s 内进入可用页面。

## 二、Onboarding

步骤：语言、学习目标、当前水平、每日时长、注册/试用。每步全屏单任务，底部主 CTA，顶部进度点。插画使用线描学习场景，不用节庆装饰。

## 三、登录 / 注册

- v1 必备：邮箱 + 密码、邮箱验证、密码重置。
- 可选 OAuth 入口必须通过 AuthAdapter 开关控制；缺 key 时不阻塞 Docker 验证。
- 表单居中 `.glass-elevated`，移动端转 paper 实底，优先可读性。

## 四、发现 `/discover`

首页结构：搜索、今日推荐、发现中国 12 主题、小说入口、猜你喜欢。进入发现中国时默认先选择 12 个主题；选择主题后进入该主题文章列表；点击文章后进入文章阅读与句子级页面。

发现中国 12 主题必须与 `content/china/00-index.md` 一致：历史、美食、名胜、节日、艺术、音乐、文学、成语、哲学、当代、趣味汉字、神话。

主题视觉：
- 历史：碑拓线条、松烟墨。
- 美食：温瓷、蒸汽曲线。
- 名胜：山水留白。
- 节日：节气纹理，避免灯笼堆砌。
- 艺术：宣纸、笔触、青瓷。
- 音乐：弦线/声波抽象。
- 文学：书页/竖线节奏。
- 成语：印章/短句卡。
- 哲学：留白、圆相、竹简线。
- 当代：中国城市真实影像 + 克制 UI。
- 趣味汉字：字形演变线稿。
- 神话：云水雾面，避免夸张神怪图。

## 五、文章阅读

- Cover 21:9，顶部透明 Header，滚动后转玻璃。
- 标题、类目、HSK、时长、收藏/分享。
- AudioPlayer。
- SentenceCard 列表：中文、拼音、母语翻译可切换。
- 生词、练习、评分评论。
- 受限类目未登录时不得泄露正文、音频 URL、句子全文或分享金句。

## 六、课程

- 首页：我的学习、4 主题、最近复习。
- 主题页：12 阶段路径，使用清晰现代时间线，不做游戏化关卡奖励。
- 阶段页：12 章列表、免费/已购/锁定状态。
- 节学习页：无 TabBar，顶部进度，底部固定检查/下一步。
- 免费范围：每个主题前 3 个阶段下全部章免费；阶段 4-12 进入付费/会员/授权判断。

## 七、游戏

- 首页展示 12 款游戏全部 active。
- 游戏详情展示玩法、推荐 HSK、词包选择、60 秒单局说明。
- v1 不展示排行榜、三星、关卡奖励、知语币奖励。
- 游戏结果页展示分数、正确率、复习词、错题入 SRS、再玩一次、换词包、回列表、分享图。

## 八、我的

- 头像、昵称、等级、会员状态。
- 学习统计、知语币余额、分销链接、收藏、笔记、错题、历史。
- 设置：主题、语言、字号、拼音模式、朗读速度、通知、导出/删除账号。
- 分销不单独展示纯邀请码，只展示嵌入 code 的链接/二维码海报。

## 九、付费墙

付费墙以清晰权益和当前限制为主，不做夸张营销 Hero。PaymentAdapter fake 可完成 Docker 验证；真实支付 provider 不作为 UX 硬依赖。

# 01 · 设计原则与品牌气质

> **关键词**：透明 / 流动 / 温暖 / 文化 / 沉浸

## 一、品牌气质（Brand Personality）

| 维度 | 定位 |
|---|---|
| **现代** | 不沉陷传统中国风刻板印象，使用现代设计语言演绎东方意境 |
| **温暖** | 主色 Rose 红 + Amber 金，避免冷漠工业感 |
| **沉浸** | 透明 + 模糊 + 微动效构建"漂浮在云端"的学习氛围 |
| **专业** | 内容严谨准确，避免娱乐化过度 |
| **东方** | 通过粒子 / 渐变 / 中国书法字体局部点缀 |

**禁用**：
- ❌ 紫色（团队决议，破坏温暖感）
- ❌ 大量金色（过度商业化）
- ❌ 中国结 / 龙 / 灯笼等刻板符号
- ❌ 过度装饰性边框 / 阴影

## 二、设计原则（North Star）

### 原则 1：透明优先（Transparency First）
所有页面层级通过半透明毛玻璃叠加表达，背景 MeshGradient 隐约可见，营造"漂浮"感。
- 至少 2 层模糊背景永远可见
- 卡片 / 模态 / 导航 全部毛玻璃
- 文字始终保证 WCAG AA 对比度（≥ 4.5:1）

### 原则 2：内容为王（Content First）
学习内容（汉字 / 拼音 / 译文）字号永远 ≥ UI 文案。
- 句子级阅读字号 ≥ 18px（base 16px）
- 拼音字号 = 中文 × 0.65
- UI 元素让位于内容

### 原则 3：母语友好（Native-First）
- 所有界面默认母语显示
- 中文学习内容旁随时可切母语
- 不让用户阅读母语外的 UI 来学习中文

### 原则 4：双向可逆（Reversible）
任何破坏性操作可撤回 / 有 5s 撤销窗口。
- 删除 → 顶部 Toast"已删除 [撤销]"
- 退出游戏 → 二次确认（除非已通关）
- 关闭未保存 → 二次确认

### 原则 5：进度可见（Progress Visible）
任何超过 1s 的操作必须显示进度。
- < 1s 直接成功 / 失败
- 1-5s 显示骨架屏或 spinner
- > 5s 显示百分比 / 步骤

### 原则 6：响应式默认（Responsive by Default）
任何屏幕从 320px → 1920px 都可用。
- 移动优先开发
- 桌面非镜像放大，重新组合布局
- PWA 端在 2xl 居中限宽 1280px，避免过宽行

### 原则 7：减负不减能（Less Is Powerful）
单屏只承担一个核心目标。
- 不在课程屏放游戏入口
- 不在游戏屏放课程入口（仅通关页）
- 二级功能折叠到菜单

### 原则 8：庆祝学习（Celebrate Progress）
每个成就 / 通关 / 升级配 0.8s 微庆祝（粒子 + 音效 + 触觉）。
- streak 7 天 / 30 天 / 100 天 → 全屏徽章
- 通关三星 → 弹窗 + 知语币雨
- 等级提升 → 顶部 Banner 动画

## 三、关键约束（Hard Constraints）

| 约束 | 详情 |
|---|---|
| **毛玻璃必用** | 所有面板 / 卡片 / 模态 必须 backdrop-filter: blur() |
| **亮暗双模** | 任何颜色必须在 light / dark 都定义 |
| **横屏游戏** | 进入游戏画布前强制旋转检测 |
| **跨端一致** | App / 后台 同一组件库（packages/ui） |
| **离线优先** | 课程 / 文章可缓存离线读 |
| **首屏 ≤ 200KB** | gzip 后核心 bundle |
| **LCP ≤ 2.5s** | 中端 Android 4G 网络 |
| **CLS ≤ 0.1** | 无意外布局偏移 |
| **a11y AA** | 焦点环可见、屏幕阅读器友好 |

## 四、视觉气质参考

- **氛围参照**：Apple Vision Pro UI / Stripe 仪表板 / Linear / Arc Browser
- **色彩参照**：日落渐变 + 海雾 + 春日樱花
- **东方意境**：极简水墨边角 / 山水留白 / 云气流动

## 六、避免反模式

| ❌ 反模式 | ✅ 正确做法 |
|---|---|
| 实色卡片硬边 | 毛玻璃半透明 + 柔边 |
| 强制弹窗打断学习 | 顶部 Banner 软提示 |
| 多 CTA 同屏争夺 | 一屏一主 CTA |
| 用纯白 / 纯黑 | 永远 RGBA + 主题色微调 |
| 静态 loading | 用骨架屏代替 spinner |
| 强制开屏广告 | 完全无广告 |
| 文字过细 | 正文 ≥ 14px，学习内容 ≥ 18px |
| 翻译生硬 | 母语 native review，非机翻直挂 |

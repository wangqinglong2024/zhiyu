# UX-25 · 古韵现代视觉一致性审计

## 来源

- `planning/ux/00-index.md`
- `planning/ux/01-design-principles.md`

## 需求落实

- 建立视觉审计 checklist：宣纸、松烟、青瓷、朱砂、金石、云水是否正确使用。
- 检查禁止项：大面积红金、龙/灯笼/中国结堆砌、彩色 blob、普通模板感、卡片套卡片。
- 对应用端、后台、游戏各抽取关键页面截图做人工审计记录。
- 审计不通过不得进入 W0 UI 完成状态。

## 验收清单

- [ ] 至少覆盖 Discover、Course、Game、Profile、Admin Dashboard、Admin Editor。
- [ ] 截图能看出知语识别度。
- [ ] 没有普通毛玻璃模板感。
- [ ] 审计结果归档到本地 artifacts。

# UX-08 · 建立核心组件库

## 来源

- `planning/ux/07-components-core.md`

## 需求落实

- 交付 Button、Input、Card、Modal、Tabs、List、DataTable、Avatar、Badge、Tag、Tooltip、Popover、AudioPlayer、PinyinText、SentenceCard、IconButton。
- 组件全部消费 token。
- 图标优先使用 lucide-react。
- 后台 DataTable 支持筛选、排序、分页、批量操作、密度切换。

## 验收清单

- [ ] 组件可被 app-fe 与 admin-fe 共用。
- [ ] 360px 宽度无文字溢出。
- [ ] 所有交互控件命中区达标。
- [ ] SentenceCard 支持播放、笔记、收藏、生词。

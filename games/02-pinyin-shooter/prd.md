# 02 · 拼音射击（Pinyin Shooter）PRD

> 一句话：经典纵版太空射击。玩家飞船持"拼音弹"，击落屏幕上方持续下落的"汉字怪"。

---

## 1. 目标与价值

| 项 | 内容 |
|---|---|
| 目标用户 | S1-S8 字 / 词阶段，尤其需要强化"拼音→字"映射的玩家 |
| 解决问题 | 拼音 ↔ 汉字双向匹配；声调辨识反应 |
| 成功指标 | 平均单局 100s；拼音正确率提升曲线斜率 ≥ 0.05/周 |

---

## 2. 核心玩法

### 2.1 场景

- 纵版竖屏：玩家飞船在屏幕底部 1/4 区域，左右滑动控制。
- 屏幕上方持续生成"汉字怪"，沿垂直 + 轻微正弦波路径下落。
- 怪物落到飞船所在底线 = 玩家被撞，扣 1 容错。

### 2.2 一回合（Round）= 20 题

题序：
1. 系统在屏幕**左下角"弹药槽"**显示当前拼音弹（如 `lǎobǎn`），并播放 1 次 TTS。
2. 屏幕同时下落 1 主目标怪 + 3-4 干扰怪（皆为完整汉字 / 词）。
3. 玩家自动连发，需先**对准目标怪**再发射；按住屏幕 = 连发；松开 = 单发。
4. 击中正确怪 → 怪爆炸 → 题完成 → 弹药槽切换下一拼音。
5. 击中错误怪 → 怪闪红 + 扣分 + 弹药数 −1（每题最多 5 发弹药，打空且未命中正确目标 = 漏题）。

### 2.3 弹药与武器升级

| 弹药 | 触发 | 效果 |
|---|---|---|
| 普通弹 | 默认 | 1 发 1 伤害 |
| 散射弹 | combo ≥ 5 | 3 发同时，扇形 30° |
| 激光 | combo ≥ 10 | 持续光束 1.0s，穿透命中线上所有怪（仍只算正确目标得分） |
| 全屏炸弹 | 拾取道具 💣 | 清屏所有干扰怪，但若正确目标在屏 → 也炸了（漏题） |

---

## 3. 交互

### 3.1 控制

- **手机**：单指拖动飞船左右移动；点击发射 / 按住连发。
- **桌面**：A/D 或 ←/→ 移动；空格发射。

### 3.2 命中判定

- 子弹与怪物外接矩形碰撞。
- 怪物有 1 HP；激光例外。

### 3.3 题面提示

- 弹药槽显示当前拼音 + 声调符号 + 喇叭按钮（可重听）。
- 正确目标怪头顶**无任何标记**（避免直接揭谜）；玩家凭知识判断。

## 4. UI / HUD

```
┌────────────────────────────────────────┐
│ ⏸  ❤❤❤    SCORE 980  COMBO ×4  5/20    │
├────────────────────────────────────────┤
│   老办                老板             │
│            老版              姥板      │  ← 下落汉字怪
│                                        │
│        ↑   ↑                           │
│        │   │                           │  ← 子弹
│              ▲                         │  ← 飞船
├────────────────────────────────────────┤
│ 🔊 lǎobǎn      ●●●●● 弹药 5/5          │
└────────────────────────────────────────┘
```

视觉：
- 飞船 = 三角形 + 尾焰粒子。
- 怪物 = 圆框 + 内嵌汉字（站酷快乐体）+ 描边 4px；按声调上色：1 声红 / 2 声黄 / 3 声绿 / 4 声蓝 / 轻声灰。
- 子弹 = 拼音首字母（如 `l`）的发光圆点。
- 命中爆炸 = 12 粒子 + 一段拼音碎片飞溅。

## 5. 难度

| 项 | easy | normal | hard |
|---|:--:|:--:|:--:|
| 怪下落速度 | ×0.7 | ×1.0 | ×1.4 |
| 干扰怪数 | 2 | 3 | 4 |
| 弹药 / 题 | 7 | 5 | 3 |
| 容错 (❤) | 5 | 3 | 1 |
| 干扰怪策略 | 同声母 | 同声母+同韵母 | 同声母+同韵母+同调 |

---

## 6. 评分

复用 shared/03。专属：
- 一发命中正确 → +30% 速答奖励
- 激光多杀干扰怪 = 0 分（避免乱射）
- 容错耗尽时正在下落的所有未答正确目标计漏题

## 8. 风险

| 风险 | 应对 |
|---|---|
| 玩家"广撒网"乱射 | 弹药限制 + 误射重罚 |
| 干扰怪与正确怪难分 | 至少 1 维属性可视化区分（声调色） |

# 11 · 翻牌记忆（Memory Match）PRD

> 一句话：经典翻牌配对。一面是汉字，另一面是拼音 / 翻译 / 部首；翻 2 张配对成功消除。

---

## 1. 目标与价值

| 项 | 内容 |
|---|---|
| 目标用户 | S1-S8 全段，尤其 S1-S3 入门 |
| 解决问题 | 字 ↔ 多元属性映射记忆 |
| 成功指标 | D1 留存 ≥ 50%；老人 / 儿童友好型款 |

---

## 2. 核心玩法

### 2.1 场地

- 4×4（easy）/ 4×6（normal）/ 6×6（hard）牌阵。
- 所有牌正面朝下；点击翻开 2 张，配对成功 → 消除；不配对 → 翻回。

### 2.2 配对模式（Round 开始时选 1 种）

| 模式 | 配对内容 |
|---|---|
| 字 ↔ 拼音 | 一半是汉字面，一半是拼音面 |
| 字 ↔ 翻译 | 一半汉字，一半母语翻译 |
| 字 ↔ 部首 (S1-S3) | 一半汉字，一半部首 |
| 字 ↔ 同义字 | 同义近义词配对 |
| 字 ↔ 听音 (Q1+Q2) | 一半汉字，一半喇叭按钮（点击播放音） |

### 2.3 一回合（Round）

- 1 局 = 1 个完整牌阵清空。
- 限时 90s（normal）。
- 每次翻牌错配 = 容错 −1（hard 模式有效）。

### 2.4 特殊牌

| 牌 | 效果 |
|---|---|
| 🔍 透视 | 翻到后揭示 3 张随机牌 1.5s |
| ⏳ 续时 | +15s |
| 🔀 洗牌 | 所有未消牌位置打乱 |
| 💣 炸弹 | 翻到 = 同位置周围 8 张永久曝光 0.5s |

---

## 3. 交互

- **手机 / 桌面**：点击翻牌；每次最多翻开 2 张。
- 第 3 张点击 → 自动盖回最早翻开的那张。

---

## 5. 难度

| 项 | easy | normal | hard |
|---|:--:|:--:|:--:|
| 牌阵 | 4×4 (8 对) | 4×6 (12 对) | 6×6 (18 对) |
| 限时 | 120s | 90s | 60s |
| 容错 | ∞ | ∞ | 5 错 → 失败 |
| 透视牌出现率 | 高 | 中 | 低 |

---

## 6. 评分

- 配对成功：base + 速答
- 连续配对（无错配）：combo bonus
- 一次记忆通关（全靠记忆，无重复翻同张）：×2

## 8. 风险

| 风险 | 应对 |
|---|---|
| 6×6 视觉拥挤 | 自适应字号 + 长字翻译截断省略 |
| 玩家随机点 | 错配次数也计入扣分 |
| 多义字配翻译 | 题库已锁定 1 主要释义 |

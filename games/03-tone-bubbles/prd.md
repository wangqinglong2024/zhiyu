# 03 · 声调泡泡（Tone Bubbles）PRD

> 一句话：泡泡龙变体。屏幕顶部漂浮"无声调汉字泡泡"，玩家发射"4 色声调泡泡"匹配上色。

---

## 1. 目标与价值

| 项 | 内容 |
|---|---|
| 目标用户 | S1-S6 字 / 词阶段，**声调薄弱**用户 |
| 解决问题 | 声调辨识（4 声 + 轻声）专项训练 |
| 成功指标 | Q9 题型正确率提升 ≥ 15% |

---

## 2. 核心玩法

### 2.1 场景

- 屏幕上半部 = 漂浮区，呈六边形蜂窝排列 18 个汉字泡泡（每泡泡 1 个汉字 + 拼音去声调）。
- 屏幕底部 = 发射器，可旋转 -75° ~ +75°。
- 发射器中持有"声调弹"5 颗：1 声红 / 2 声黄 / 3 声绿 / 4 声蓝 / 轻声灰，颜色随机循环。

### 2.2 一回合（Round）

回合 = 清空当前蜂窝（≈ 18 ~ 30 个泡泡，按 stage 调整）。
- 系统先 TTS 朗读所有泡泡的正确读音 1 次（可关）。
- 玩家瞄准 + 发射声调弹 → 弹击中泡泡 → 系统判定该字+该声调是否=正确读音。
  - 正确 → 泡泡爆炸 + 同声调相邻泡泡 ≥ 3 连消（消消乐）。
  - 错误 → 泡泡变灰 + 弹药槽换下一颗 + 扣容错。
- 蜂窝每 8 秒下移 1 行；下移到底线 = Round Over。

### 2.3 蜂窝结构

```
   ⓞ  ⓞ  ⓞ  ⓞ  ⓞ          ← 第 1 行（5 个）
     ⓞ  ⓞ  ⓞ  ⓞ            ← 第 2 行（4 个，错位）
   ⓞ  ⓞ  ⓞ  ⓞ  ⓞ
     ⓞ  ⓞ  ⓞ  ⓞ
   ⓞ  ⓞ  ⓞ  ⓞ  ⓞ
            ▲              ← 发射器
```

每个 ⓞ 内显示汉字 + 拼音（无调号），如 `老 / lao`。

### 2.4 连消

- 击中泡泡正确 → 检查与该泡泡相邻、且**正确声调相同**的所有泡泡，≥ 3 连 → 整组爆炸 + 上方失支撑泡泡掉落（额外得分）。

---

## 3. 交互

- **手机 / 桌面**：拖动屏幕底部 1/4 区瞄准；点击发射器 / 空格发射。
- 弹药槽显示下 3 颗将出现的声调色。

## 4. UI

```
┌──────────────────────────────────┐
│ ⏸  ❤❤❤   SCORE 1320  COMBO ×6   │
├──────────────────────────────────┤
│   老  办  板  伴  半             │
│     版  班  扮  搬               │
│   翻  反  返  范  饭             │  ← 蜂窝（汉字+无调拼音）
│     凡  烦  繁  繁               │
├──────────────────────────────────┤
│           ●●●●●  ←弹药槽 (4321轻) │
│              ▲                    │
└──────────────────────────────────┘
```

视觉：
- 泡泡 = 圆形 + 内汉字 + 下方小字拼音；当前选中目标高亮描边。
- 声调弹 = 小球带颜色 + 声调符号（ˉ / / ˇ \）。
- 命中正确 → 泡泡膨胀 1.3x → 爆破粒子 + 拼音正确版浮空显示 0.5s。

---

## 5. 难度

| 项 | easy | normal | hard |
|---|:--:|:--:|:--:|
| 蜂窝行数 | 4 | 5 | 6 |
| 每行下移间隔 | 12s | 8s | 5s |
| 弹药随机性 | 3 色循环 | 4 色循环 | 5 色（含轻声）随机 |
| 容错 | 5 | 3 | 1 |

---

## 6. 评分

- 单击中：+10 × 声调难度（轻声 ×1.5）
- 3 连消：+50；4 连：+150；5+：+500
- 整组级联爆破（顶部失支撑）：每 1 个 +20

---

## 7. 验收

- [ ] AC-01 字源严格按 (track, stage) 取 char/word，去除多音字
- [ ] AC-02 4 色映射音乐学正确：1 ̄ 2 ́ 3 ̌ 4 ̀ 轻○
- [ ] AC-03 6 行蜂窝下移流畅 60fps
- [ ] AC-04 弱网下泡泡内字符不糊（用本地字体）

---

## 8. 风险

| 风险 | 应对 |
|---|---|
| 多音字 | 内容适配器过滤多音字 / 取最常用调 |
| 色盲玩家 | 每个声调附带形状标记（圆/方/三角/菱/星） |
| 弹药色 RNG 卡死 | 强制每 5 发出现一次缺色 |

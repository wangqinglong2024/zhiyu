# 游戏专区 · 12 款 2D 漫画风格语言学习游戏

> **平台四大板块之一**：发现中国 / 系统课程 / **游戏专区** / 小说专区
> **定位**：寓教于乐，把课程系统的知识点（字 / 词 / 句）拆出来灌进 12 款回合制小游戏。
> **技术约束**：纯 AI 可生成，不依赖美术资源。一切视觉用 Canvas / WebGL 形状 + Emoji + CSS 粒子特效 + 现成开源字体实现。
> **设计哲学**：玩法各异，**底层数据 / 设置 / 评分 / 课程联动统一**。

## 2. 全局统一规则（任一款必须遵守）

详细规范见 [shared/](./shared/00-index.md)，此处只列硬约束：

1. **回合制**：每局 = 1 个 Round，Round 之间可暂停 / 退出，不强制连续。Round 时长 60-180 秒可调。
2. **统一设置面板**（每款必备）：
   - **主题轨道**：`电商 / 工厂 / HSK / 日常`（4 选 1）
   - **课程阶段**：`S1 - S12`（12 选 1，未解锁阶段灰显但允许试玩）
   - **难度**：`轻松 / 标准 / 挑战`（影响速度、干扰项数、容错）
   - **本局数量**：`10 / 20 / 30 / 50` 题（即一局答题 / 操作单元数）
3. **数据来源唯一**：所有"字 / 词 / 句"全部来自 `/course/shared/05-question-bank.md` 定义的 **题库 CSV/JSONL**，按 `track + stage` 过滤，**不允许游戏内自造内容**。
4. **音频**：拼音、汉字朗读统一用 TTS（已在课程系统生成），游戏直接调用 URL，不重做。
5. **评分上链**：每局结束写回课程系统的 SRS（Spaced Repetition），错过的字 / 词权重 ×3 进入下次复习池。
6. **零美术依赖**：所有视觉元素仅允许：①几何形状（圆/矩/多边形）②Emoji ③CSS 粒子 ④WebGL Shader 粒子 ⑤系统字体 ⑥免版权 SVG 图标库。
7. **无登录必玩**：游客模式可玩，但成绩不计入 SRS。
8. **横竖屏自适应**：手机为主，平板 / Web 兼容。

## 4. 与课程系统的耦合点

| 入口 | 行为 |
|---|---|
| 课程章末"奖励游戏" | 自动用本章知识点开启 1 局，胜利解锁皮肤 |
| 阶段综合考前热身 | 可一键以"本阶段错过题"为内容池开局 |
| 每日任务 | 每日推荐 3 款游戏 × 10 题，完成 +XP |
| SRS 复习池 | 玩家可选"用游戏复习"代替传统刷题 |

## 5. 路线图（与课程并行）

- **MVP**：先做 01 / 02 / 05 / 11（玩法最简、覆盖最广），4 款上线即覆盖 80% 题型。
- **Phase 2**：03 / 06 / 07 / 12（玩法中复杂度）。
- **Phase 3**：04 / 08 / 09 / 10（玩法高复杂度，需要更精的特效与节奏）。

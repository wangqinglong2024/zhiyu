# shared · 统一设置面板规范

> 12 款游戏每一款进入主菜单后，**必须先弹出（或可随时调出）此设置面板**。
> 玩家选择完成 → 点"开始" → 进入对应游戏的 Round 循环。

---

## 1. 面板字段

| # | 字段 | 类型 | 取值 | 默认 | 说明 |
|:--:|---|---|---|---|---|
| 1 | `track` | 单选 | `ec`(电商) / `factory`(工厂) / `hsk` / `daily` | 跟随用户最近一次课程轨道 | 决定题库主题 |
| 2 | `stage` | 单选 | `1` ~ `12` | 跟随用户当前阶段 | 决定难度梯度与可用 unit_type |
| 3 | `difficulty` | 单选 | `easy` / `normal` / `hard` | `normal` | 见第 3 节难度系数 |
| 4 | `roundSize` | 单选 | `10` / `20` / `30` / `50` | `20` | 一局游戏的"操作单元数" |
| 5 | `audioOn` | 开关 | `bool` | `true` | 是否播放 TTS 音频 |
| 6 | `subtitleLang` | 单选 | `vi` / `th` / `en` / `none` | 跟随用户母语 | 释义提示语言 |
| 7 | `srsOn` | 开关 | `bool` | `true`（已登录）/ `false`（游客）| 成绩是否回传 SRS |

> **不允许**任何游戏新增独立设置项，差异化只能通过游戏内"局内难度小调"实现（如忍者的"是否包含炸弹"）。

---

## 2. 面板交互

```
┌──────────── 游戏设置 ────────────┐
│  🎯 主题轨道   [电商] 工厂  HSK  日常  │
│  📚 课程阶段   [S1] S2 ... S12  (滑块)  │
│  ⚙️ 难度       [轻松]  标准  挑战       │
│  🔢 本局题量   10  [20]  30  50         │
│  🔊 音频       ●─── ON                  │
│  🌐 字幕       [越南语 ▼]               │
│  💾 计入复习   ●─── ON                  │
│                                          │
│           [ 开始游戏 ]                   │
└──────────────────────────────────────────┘
```

- 设置项变更 **实时持久化** 到 `localStorage` + 用户云端 profile。
- 开始游戏前，系统调用 [`02-content-adapter.md`](./02-content-adapter.md) 拉取内容池；若内容池 `<roundSize`，提示"该范围内容不足，是否扩展到上一阶段？"。

## 3. 难度系数

| 维度 | easy | normal | hard |
|---|:--:|:--:|:--:|
| 物体速度 / 节奏 BPM | 0.7× | 1.0× | 1.4× |
| 干扰项数量 | 2 | 3 | 4 |
| 容错次数 | 5 | 3 | 1 |
| 限时题倒计时 | 8s | 5s | 3s |
| 连击奖励起点 | ×3 连 | ×5 连 | ×8 连 |
| SRS 权重 | ×1.0 | ×1.2 | ×1.5 |

---

## 4. 阶段与可用 unit_type

直接复用 [`/course/shared/05-question-bank.md`](../../course/shared/05-question-bank.md) §2 表。
游戏适配器在拉取内容时**只允许**取该阶段标记可用的 `unit_type`。

例：S1 选 `hanzi-runner` 跑酷拾字，因 `short_sentence` 在 S5+ 才出现，会自动降级为"拾词组成短词组"。

---

## 5. 异常与降级

| 场景 | 处理 |
|---|---|
| 玩家未登录 | 仍可设置，`srsOn` 强制 `false`，结算页提示登录可保存 |
| 选择阶段未解锁 | 弹窗"试玩模式：成绩不计入考核"，确认后继续 |
| 内容池不足 | 自动按 `当前阶段 → 上一阶段 → 全轨道` 顺序补齐 |
| 音频加载失败 | 静默回退为文字模式，`audioOn` 临时关闭 |

# shared · 漫画风视觉与特效（无美术方案）

> **核心约束**：纯 AI 可生成，**禁止**任何需要原画师 / 动画师产出的素材。
> **风格目标**：扁平 + 漫画 + 涂鸦 + 霓虹粒子，类似《水果忍者》的爽感但用纯几何 + Emoji 实现。

---

## 1. 视觉资产白名单

| 资产 | 来源 | 备注 |
|---|---|---|
| 几何形状 | Canvas / SVG / WebGL 即时绘制 | 圆 / 方 / 三角 / 多边形 / 贝塞尔 |
| Emoji | 系统字体 / Twemoji | 🍉🍌🥷⚡🔥💥 等做"角色 / 物品" |
| 矢量图标 | Lucide / Tabler / Heroicons | MIT / ISC，免费商用 |
| 字体 | 思源黑体 / Noto Sans CJK / 站酷快乐体 | 开源 |
| 粒子贴图 | 程序生成（圆形渐变 PNG，用代码画出） | 无外部素材 |
| Shader | 自写 GLSL 或 shadertoy 公开域片段 | 噪声 / 流体 / 火焰 |
| 音效 | freesound.org CC0 / Pixabay | 切击 / 爆炸 / 升级 |

**禁止**：手绘角色 / 3D 模型 / 商业插画 / 真人贴图。

---

## 2. 漫画风设计语言

- **描边**：所有形状统一 `stroke: #1a1a1a, width: 3-5px`，营造漫画轮廓。
- **填色**：高饱和度 8 色板（霓虹粉 / 电光蓝 / 柠檬黄 / 荧光绿 / 火焰橙 / 紫罗兰 / 深红 / 纯白）。
- **阴影**：偏移 `4px 4px 0 #1a1a1a` 的硬阴影（漫画 hard shadow），不用模糊阴影。
- **字体**：所有汉字用站酷快乐体 / 站酷酷黑（圆胖卡通感），拼音用 Noto Sans Mono。
- **拟声词**：BANG! / 切！/ 砰！用 SVG 文字加 `transform: rotate + scale` 动画弹出。

## 3. 特效库（统一 4 个等级）

| 等级 | 触发 | 实现 |
|:--:|---|---|
| L1 普通命中 | 答对 | 8 个圆点粒子向外爆 + 字体放大→缩小 0.2s |
| L2 火焰命中 | combo ≥ 5 | 命中点叠 8 帧火焰 sprite（程序生成）+ 屏幕轻微震动 |
| L3 暴击 | combo ≥ 10 | 慢动作 0.3s + 径向光线 12 条 + 命中物碎成 30 片 |
| L4 全场暴击 | combo ≥ 20 | 屏幕泛白 + 所有未答对象一起爆炸 + 全屏火花雨 |

粒子系统：基于 `particle.js` 自写极简版（≤ 200 行），统一对外接口：
```ts
particles.emit({ x, y, count, color, speed, life, gravity, shape });
```

## 4. 12 款游戏共享视觉元素

| 元素 | 用途 |
|---|---|
| 顶部 HUD | 分数 / combo / 进度（当前题 / 总题） / 暂停按钮 |
| 倒计时圆环 | 限时题统一用，描边 4px，颜色随剩余时间从绿→黄→红 |
| 答错 ❌ 大叉 | 1.5s 红光闪烁 + 屏幕轻晃 0.3s |
| 升级特效 | 阶段提升 / 解锁皮肤 / 通关时统一播 1.5s |

## 5. 性能预算

| 设备 | 帧率 | 同屏粒子上限 |
|---|:--:|:--:|
| 中端手机 (骁龙 6 系) | 60fps | 300 |
| 低端手机 (3 年前千元机) | 30fps | 100 |
| Web 桌面 | 60fps | 800 |

降级策略：FPS 连续 5s < 阈值，自动减 50% 粒子上限。

## 6. 解锁皮肤设计

- 每款游戏首次达 3⭐ 解锁 1 套"色板皮肤"（粒子色 + 字体色 + 描边色）。
- 6 套皮肤 = 该游戏全收集成就。
- 皮肤用配置 JSON 存储，无图片。

```json
{
  "skinId": "neon-tokyo",
  "stroke": "#0ff",
  "palette": ["#ff00aa", "#aa00ff", "#00ffaa", "#ffff00"],
  "particle": "spark",
  "bg": "linear-gradient(180deg,#0a001a,#001a2a)"
}
```

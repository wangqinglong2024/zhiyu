# 01 · 汉字忍者（Hanzi Ninja）PRD

> 一句话：把"水果忍者"的水果换成飞舞的汉字 / 词；玩家按题面要求"切对的、躲错的"。

---

## 1. 目标与价值

| 项 | 内容 |
|---|---|
| 目标用户 | 4 大轨道 S1-S12 全段，尤其是 S1-S6 字 / 词阶段玩家 |
| 解决问题 | 字形辨识 / 拼音 → 字 / 释义 → 字 的反应训练 |
| 商业价值 | 引流款，玩法门槛最低，10 秒上手；绑定 SRS，拉高复习留存 |
| 成功指标 | D1 留存 ≥ 45%；平均单局时长 90s；3⭐率 ≥ 30% |

---

## 2. 核心玩法

### 2.1 一回合（Round）= 20 题（默认）

每题 = 1 个"出题情境" + 屏幕底部抛上来 **5-7 个汉字 / 词**（其中 1-2 个是正确答案，其余为干扰项），玩家在 **5 秒**（normal）内用手指划过命中正确目标。

### 2.2 题型情境（系统随机）

| 情境 | 题面（屏幕顶部） | 正确目标 | 干扰目标 |
|---|---|---|---|
| A. 拼音 → 字 (Q3) | `lǎobǎn` | 老板 | 老版 / 老办 / 姥板 / 烙伴 |
| B. 释义 → 字 (Q6) | "ông chủ"（越南语） | 老板 | 同义 / 形近词 |
| C. 字 → 部首 (Q10，仅 char) | "湖" | 氵 | 冫 / 丬 / 灬 |
| D. 听音切字 (Q1) | 🔊 播放 "lǎobǎn" | 老板 | … |

每题 70% 概率出 A/B/C/D 任一种，30% 概率出"组合题"（同时正确目标 ≥ 2 个，全切才算满分）。

### 2.3 特殊物体

| 物体 Emoji | 含义 | 后果 |
|---|---|---|
| 💣 炸弹 | 字色为黑，外框红 | 切到 = 扣 50 分 + combo 清零 + 扣 1 容错 |
| ❄️ 冰冻 | 字色蓝白渐变 | 切到 = 全屏减速 50% × 3s（友方 buff） |
| ⭐ 星星 | 字色金色发光 | 切到 = 下一题分数 ×2 |
| 🍯 蜂蜜 | 字色琥珀 | 切到 = 屏幕粘滞，所有物体下落变慢，但你的刀痕也变慢 |

特殊物体出现频率：每 5 题 1 次；hard 难度炸弹翻倍。

## 3. 交互细节

### 3.1 输入

- **手机**：单指或双指划动 = 刀痕；连续轨迹经过的物体均判定"被切"。
- **桌面**：鼠标按住左键拖动 = 刀痕。
- 刀痕用 `requestAnimationFrame` 采样路径，每帧检测路径线段与物体外接圆的相交。

### 3.2 物理

- 物体从屏幕底部抛出，使用经典抛物线：
  - 初速度 vy ∈ [-12, -16] 像素/帧（按难度系数缩放）
  - 重力 g = 0.45 像素/帧²
  - 水平 vx ∈ [-3, 3]
  - 自旋 ω ∈ [-6°, 6°]/帧
- 同屏最多 7 个物体；上一波全落地或被切完后才出下一波。

### 3.3 判定

- 切到正确目标：物体沿刀痕方向裂成 2 半（左右 / 上下随机），各自继续抛物线下落 0.5s 后淡出；触发 L1+ 命中特效。
- 切到错误目标：物体爆出 ❌；扣分；扣 1 容错；容错归 0 → Round Over。
- 漏切（物体落地）：
  - 若是正确目标 → 漏答；扣容错。
  - 若是干扰项 → 无影响。
- 同一刀痕命中多个物体 = 多杀，每多一个 +5 combo 加成。

### 3.4 出题节奏

```
进入 Round → 顶部题面渐入 0.5s
  → 抛物体（首批 1.0s 后落地最高点）
  → 玩家在物体下落途中切
  → 全部物体处理完 → 0.5s 题面渐出
  → 0.3s 间隔 → 下一题
```

单题平均 4-6s，20 题 ≈ 90-120s。

- 顶部题面：拼音 / 翻译用大字显示；听音题为大喇叭按钮 + 自动播放 1 次。
- 字物体大小：屏幕宽 / 8（约 50px @ 手机），描边 4px，高饱和填色。
- 刀痕：渐变发光线，长度 80-120px，淡出 0.3s；连击 ≥ 5 时刀痕变火焰色。

## 5. 难度调参

| 项 | easy | normal | hard |
|---|:--:|:--:|:--:|
| 物体初速度系数 | ×0.7 | ×1.0 | ×1.4 |
| 单题物体数 | 4-5 | 5-7 | 6-9 |
| 干扰项数量 | 2 | 3 | 4 |
| 容错次数 (❤) | 5 | 3 | 1 |
| 炸弹出现率 | 5% | 10% | 20% |
| 单题倒计时 | 8s | 5s | 3s |

阶段越高，正确目标可能是 word / phrase / 短句中的"关键字"，物体显示完整词；切对其中关键字算命中，切错非关键字算干扰。

## 6. 评分

完全沿用 [shared/03-scoring-system.md](../shared/03-scoring-system.md)。
额外加成：
- 多杀奖励：1 刀切 ≥ 2 正确目标 → 每多 1 个 +20 分。
- 完美 Round（无错无漏）→ 结算页 +500 奖励。

## 7. 关卡与解锁

- **无关卡设计**，玩法纯回合制可无限连玩。
- **解锁系统**：
  - 首次 3⭐ → 解锁皮肤"霓虹东京"
  - 累计 50 局 → 解锁"刀痕"特效"赤焰太刀"
  - 累计 1 万正确切 → 解锁称号"汉字宗师"

## 四、条件显示规则

> 如果页面上某些元素在不同条件下显示不同的内容或按钮，写在这里。

```
[例如：]

"处理"按钮：
- 待确认 / 待发货 / 已发货 的订单显示"处理"
- 已完成的订单隐藏"处理"，只显示"查看"
- 已取消的订单：普通角色只显示"查看"，老板额外显示"重新打开"

- 待确认：显示"确认订单"和"取消订单"两个按钮
- 待发货：显示快递单号输入框 + "确认发货"按钮
- 已发货：显示快递单号（只读）+ "确认收货"按钮
- 已完成/已取消：不显示任何操作按钮
```

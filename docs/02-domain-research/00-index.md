# 02 · 领域研究（Domain Research）

> **领域**：东南亚（越南/泰国/印尼优先）的中文作为第二语言（Mandarin as L2）教育与数字化学习
> **目标**：验证"基于母语 + AI 一次性生产 + 多模态闭环"在该领域的合理性
> **方法**：BMAD `bmad-domain-research` workflow，AI 自主推进
> **数据来源**：AI 知识库（截止 2026-04），经验性外推 + 项目方提供的市场调研草案
> **限制声明**：本研究在没有实时联网搜索的情况下基于已有知识库 + 推理生成；具体数字应在落地前以最新公开数据复核

## 关键结论速览

1. **东南亚是中文 L2 学习的下一个增长极**：人口红利、中国对外贸易（一带一路）持续扩张、汉语水平考试（HSK）改革（HSK 3.0 七等三级化）后形成新需求曲线。
2. **越南是首发优先级最高的市场**：HSK 考点最多、中文培训机构最多、中越互访量年增长 20%+、TikTok 渗透率东南亚最高。
3. **泰国具备付费能力优势**：人均 GDP 高于越南，但中文学习人数低于越南；适合"高 ARPU 战略"。
4. **印尼是中长期最大潜在市场**：人口 2.7 亿，但单价低、获客成本高、华人圈层 vs 本地穆斯林学习需求二元化。
5. **本地化深度是壁垒**：当地市场目前主流玩家（Hello Chinese、Migii、Chinese.io）几乎全部以英语为讲解语，少数有越南语/泰语界面但讲解仍英语，知语的"母语原生讲解"是真正空白点。
6. **政策风口**：越南教育部 2024 年起把中文列为第二外语第一选择之一，泰国朱拉隆功等高校扩招中文专业，印尼 2025 年宣布将中文纳入部分公立中学选修。
7. **合规要点**：越南 PDPL（2023）、泰国 PDPA、印尼 PDP Law（2024 全面执行），数据本地化与跨境传输管理是关键。

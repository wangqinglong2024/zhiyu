# 5.4 · 产品愿景 + 战略 + 北极星

## 一、产品愿景

### 1.1 长期愿景（5 年）
> **"成为东南亚学习者学习中文的第一选择，把中文从'考试科目'变成'生活与生意'。"**

### 1.2 中期愿景（3 年）
- 成为越南 / 泰国 / 印尼三市场中文学习头部品牌
- 用户数 200 万+ 注册 / 50 万 MAU / 10 万+ 付费
- ARR $5-7M
- 团队规模 20-30 人

### 1.3 短期愿景（12 月）
- 完成 PMF 验证 + 三市场稳定运营
- 用户数 25 万注册 / 8 万 MAU / 8000 付费
- ARR $300K
- 团队规模 5-8 人

## 二、价值主张（Value Proposition）

### 2.1 核心价值主张
> **对东南亚学习者来说，知语是用你的母语讲解、覆盖 HSK 商务工厂日常、价格只要竞品 1/3 的中文学习全栈应用。**

### 2.2 价值主张画布

```
痛点（Pains）
- 英语讲解听不懂                      → 母语原生
- HSK 7-9 / 商务无产品                → 4 轨道 12 阶段
- $15/月太贵                          → $4/月
- 多 App 切换疲惫                     → 4 模块闭环
- 学完就忘                            → FSRS-5 + 游戏化

收益（Gains）
- 高效学习 + 节省时间                 ← 母语 + AI 自适应
- 升职加薪 / 拿奖学金                 ← 应用场景化
- 性价比                              ← 价格 + 内容广度
- 学习有趣不枯燥                      ← 12 游戏 + 小说
- 长期留存                            ← 知语币 + Streak
```

## 三、战略（Strategy）

### 3.1 三大战略支柱

#### 支柱 1：母语原生（差异化）
- **目标**：成为东南亚学习者首选品牌
- **手段**：4 语种（英 / 越 / 泰 / 印尼）原生讲解 + 母语审校
- **资源投入**：内容 50%、工程 30%、母语审校 20%
- **KPI**：母语用户付费转化率 vs 英语用户 ≥ 1.5x

#### 支柱 2：AI 内容工厂（成本与速度）
- **目标**：内容生产成本竞品 1/10、迭代速度 5x
- **手段**：LangGraph 工作流 + 双 LLM + 自动化审校
- **资源投入**：1 名 AI 工程师 + Claude/DeepSeek 月成本 $200
- **KPI**：单条知识点生产成本 < $0.05

#### 支柱 3：4 模块闭环（留存）
- **目标**：用户日均使用时长 > 单模块产品 2x
- **手段**：发现 / 课程 / 游戏 / 小说 + 知语币串联
- **资源投入**：分阶段上线（W-1 课程 + 文化 + 5 款游戏；W+4 全模块）
- **KPI**：D30 留存 > 20%（行业 12-15%）

### 3.2 战略不做（What we WON'T do）

- ❌ v1 不做实时 1v1 真人课
- ❌ v1 不做 ASR 评测
- ❌ v1 不做原生 App（用 PWA 兜底）
- ❌ v1 不做实时多人对战
- ❌ v1 不做直播
- ❌ v1 不接本地支付（仅 Paddle）
- ❌ v1 不做中文以外其他语种
- ❌ v1 不做面向中国大陆的市场

### 3.3 战略时序

| 阶段 | 时段 | 战略重点 |
|---|---|---|
| 阶段 1：PMF | M+0 → M+6 | 母语 + 课程 + 游戏 5 款 + 越南 |
| 阶段 2：扩张 | M+6 → M+12 | + 泰国 + 12 款游戏全 + B2B 试点 |
| 阶段 3：稳态 | M+12 → M+24 | + 印尼 + 高级会员 + 移动 App |
| 阶段 4：复制 | M+24+ | 复用模型 → 中文人学英语 / 越南语 |

## 四、北极星指标（North Star）

### 4.1 北极星定义
> **WAL** = Weekly Active Learners
>
> 定义：每周完成至少 1 节课程或 1 篇发现中国短文阅读的活跃学习者数

### 4.2 为什么选 WAL
- 反映"真实学习行为"（vs 简单 MAU）
- 与用户价值直接相关
- 与商业指标强相关（付费率 / LTV / 推荐）
- 团队可日常追踪 + 优化

### 4.3 北极星目标

|---|:---:|
| M+1 | 500 |

## 五、定位（Positioning）

### 5.1 定位陈述
> 对**东南亚 18-35 岁中文学习者**来说，
> **知语**是一款**母语原生的 AI 中文学习全栈应用**，
> 它不像 **HelloChinese / Duolingo / Migii** 那样只用英语讲解或单一题库，
> 而是用**你的母语 + 4 个学习模块 + 1/3 的价格**让中文学习真正有效。

### 5.2 品牌人格
- **可靠**（母语审校保证质量）
- **温度**（知语币 / 客服 IM 让用户感到被关心）
- **现代**（AI 驱动 / 设计前卫）
- **东南亚原生**（不是中国味的中国产品，是东南亚学习者的产品）

### 5.3 视觉与语言
- 设计系统：Cosmic Refraction（grules）
- 主色：Rose / Sky / Amber（无紫色）
- 语言：母语优先；中文是教学对象，不是界面语言

## 六、增长哲学

### 6.1 三个增长引擎
1. **内容引擎**：SEO + 多语种长尾页 + AI 高速生产
2. **病毒引擎**：知语币 + 永久二级分销 + 学习里程碑分享
3. **付费引擎**：定价甜点 + A/B 优化付费墙 + 多档套餐

### 6.2 不依赖单一渠道
- TikTok / FB / YouTube / SEO / WhatsApp / 分销 / B2B = 7 路并行
- 任一渠道失效都不致命

### 6.3 长期 vs 短期
- 短期：付费投放 + KOL 试探 + 数据快速迭代
- 长期：SEO + 内容资产 + 品牌口碑 + B2B 沉淀

## 七、文化与原则

### 7.1 团队文化
- **AI-first**：每个流程都问"AI 能做什么？"
- **数据驱动**：决策必须有数据支撑（即使是粗糙数据）
- **东南亚优先**：所有决定第一原则是"东南亚用户怎么想"
- **小步快跑**：周迭代 + 月复盘

### 7.2 产品原则
1. **母语优先**：UI / 帮助 / 客服 / 推送 全部母语
2. **零摩擦**：注册 / 试学 / 学习 / 付费 路径都最短
3. **质量不打折**：宁可少做 1 篇也不发劣质内容
4. **可观测**：每个功能都有指标 + Dashboard
5. **可逆**：所有更新可一键回滚

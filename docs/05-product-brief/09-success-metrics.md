### 8.2 节奏
- **日报（自动）**：DAU / 新增 / 付费 / 收入
- **周报（人工）**：留存 / 转化 / NPS
- **月报（人工）**：单位经济 / 战略 review

### 8.3 仪表盘等级
- L0：核心指标（北极星 + ARR） → 全员可见
- L1：业务指标（漏斗 + 留存） → 团队可见
- L2：运营指标（内容 / 审校 / 客服） → 部门可见
- L3：技术指标（性能 / 错误） → 技术团队

## 九、可控 vs 不可控

### 9.1 可控（自身行动直接影响）
- 内容产能 / 质量
- 付费墙转化率
- 留存钩子设计
- 客服响应

### 9.2 部分可控（市场 + 自身）
- WAL / MAU（受营销 + 季节影响）
- CAC / LTV（受市场竞争影响）
- NPS（受产品 + 服务影响）

### 9.3 不可控（外部）
- 政策 / 平台规则
- 经济 / 汇率
- 竞品突变

→ 战略上：80% 资源放在可控、15% 在部分可控、5% 应急在不可控。

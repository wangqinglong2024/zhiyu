## 三、用户旅程关键节点

### 3.1 漏斗
```
1. 知道知语（TikTok / FB / SEO / 朋友推荐）
2. 访问站内（无登录试用 3 篇）
3. 注册 + 送 100 知语币
4. 第一节学习 + 节小测体验
5. 触达付费墙（情绪高峰 / 沉没成本）
6. 付款（月 / 半年 / 年 / 单段）
7. 持续学习（4 模块切换 + 温故知新）
8. 分享获得分销币
9. 续费 / 升级
10. 推荐朋友 → 回到第 1 步
```

### 3.2 关键钩子设计

| 节点 | 钩子 | 目标转化 |
|---|---|---|
| 1→2 | TikTok 母语吸睛文案 | UV → 进站 30% |
| 2→3 | "免费学 3 篇 + 100 币" | 试用 → 注册 25% |
| 3→4 | 引导到分级测试 + 推荐轨道 | 注册 → 首课 70% |
| 4→5 | 付费墙位置 A/B（A=5 章后, B=节小测后, C=D7 streak） | 软付费墙触达 50% |
| 5→6 | 首发促销半年 $12 + 紧迫感 | 触达 → 付费 8% |
| 6→7 | D1/D3/D7 推送 + 4 模块体验引导 | 付费 → D7 60% |
| 7→8 | "已学 X 节，分享给朋友" | 付费 → 分享 30% |
| 8→9 | 续费提醒 7 天前 + 优惠码 | 续费率 60% |

## 五、用户访谈计划（M+0 → M+3）

### 5.1 目标
- 验证 4 个 Persona 假设
- 发现新场景 / 新需求
- 优化付费墙文案

### 5.2 计划
- 每月 5-10 个深度访谈（30 分钟）
- 渠道：Discord / Telegram 社群报名
- 激励：1 年免费会员

### 5.3 访谈大纲
1. 你为什么学中文？目标是什么？
2. 当前用什么 App / 方法？满意度？
3. 用知语 1 个月体验如何？
4. 哪些功能用得最多 / 最爱？
5. 哪些功能没用 / 不理解？
6. 价格感知（$4 月 / $40 年）合理吗？
7. 推荐给朋友的可能性？

## D. 暂缓 / 否决（4 项 + 内含的 v2 项）

- 微信支付 / 支付宝（合规与目标市场不匹配）
- Discord 式实时社群（用户明确不做实时）
- 用户卖皮肤（合规复杂）
- 文章广告位（上线初期影响体验）

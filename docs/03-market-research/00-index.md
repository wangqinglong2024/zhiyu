## 关键结论速览

| 维度 | 结论 |
|---|---|
| TAM | 全球数字化中文学习 ~$15 亿，2028E ~$30 亿 |
| SAM | 东南亚（越/泰/印尼）数字化付费用户市场 ~$2.5 亿 ARR |
| SOM（3 年） | $300-500 万 ARR / 8-12 万付费用户 |
| 首发市场 | 越南（高需求 + TikTok 渗透 + 价格弹性可控） |
| 第二市场 | 泰国（高 ARPU + 中文学校体系成熟） |
| 第三市场 | 印尼（人口红利但单价低，M+12 后扩展） |
| 头部竞品 | HelloChinese / Duolingo / SuperChinese / Migii (VN) / Chineasy / Migii TOPIK 风格题库类 |
| 定价缺口 | 现有产品 $7-15/月，知语 $4 是真缺口 |
| GTM 主路径 | TikTok（流量） + FB（社群信任） + YouTube（深度教程） + WhatsApp（推送） + SEO（长期） |
| 关键风险 | 内容审核 / Paddle 准入 / 冷启动获客 / 母语审校跟不上 |

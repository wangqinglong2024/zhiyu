# 3.1 · TAM / SAM / SOM 量化

## 一、定义

- **TAM** Total Addressable Market：全球可能服务的市场总规模
- **SAM** Serviceable Addressable Market：可触达的细分市场
- **SOM** Serviceable Obtainable Market：3 年内现实可获得的份额

## 二、TAM：全球数字化中文学习市场

### 2.1 自下而上

| 指标 | 数值（2025E） | 数值（2028E） |
|---|:---:|:---:|
| 全球中文 L2 学习者 | 2.0 亿+ | 2.5 亿 |
| 数字化学习渗透率 | 12% | 25% |
| 数字化学习者 | 2400 万 | 6250 万 |
| 付费率 | 6% | 9% |
| 付费用户数 | 144 万 | 562 万 |
| 客单价（年均） | $50 | $55 |
| **TAM** | **$7.2 亿** | **$30.9 亿** |

### 2.2 自上而下交叉验证

- HolonIQ 估算 2024 全球语言学习市场 $80 亿（线上），中文占 ~10% = $8 亿
- HSK 海外考生 100 万 × 平均年学习支出 $200 = $2 亿（仅考生）
- 加上非考生数字化学习与机构市场，2025E ~$10-15 亿是合理区间

### 2.3 TAM 增速
- 2024-2028 CAGR ~30-40%（AI 加速）

## 三、SAM：东南亚数字化付费市场

### 3.1 三国主战场

| 国家 | 18-35 岁中文学习意向人口 | 数字化付费率 | 付费人数 | 客单价 | 市场规模 |
|---|:---:|:---:|:---:|:---:|:---:|
| 越南 | 250 万 | 8% | 20 万 | $35 | $700 万 |
| 泰国 | 180 万 | 12% | 21.6 万 | $50 | $1,080 万 |
| 印尼 | 350 万 | 5% | 17.5 万 | $30 | $525 万 |
| **合计 v1** | 780 万 | - | 59.1 万 | - | **$2,305 万 ARR** |

### 3.2 扩展（含菲律宾 / 马来西亚 / 新加坡）

| 国家 | 付费规模（潜在） |
|---|:---:|
| 菲律宾 | $200 万 |
| 马来西亚 | $300 万 |
| 新加坡 | $400 万 |
| 缅 / 老 / 柬 | $100 万 |
| **东南亚总 SAM** | **$3,300-3,500 万 ARR** |

### 3.3 但是（潜在 vs 已验证）
- 上述基于"如果数字化学习像 Netflix/Spotify 一样普及"
- 现实折扣 50%（保守）→ **东南亚已验证 SAM ≈ $1,500-1,800 万 ARR**

### 3.4 长期 SAM（3-5 年）
- 假设 2028 年东南亚数字化中文学习付费用户 200 万 × 平均 $40 = **$8,000 万 ARR**
- 这是知语应该看到的中期天花板

## 四、SOM：3 年内可获得份额

### 4.1 关键假设

| 假设 | 取值 |
|---|---|
| 知语 v1 上线时间 | 2026 年 5-6 月 |
| 三平台月触达 UV（M+12） | 100 万 |
| 注册转化率 | 8% |
| 注册→付费转化率 | 4% |
| 付费用户客单价（含会员+单课） | $35-40/年 |

### 4.2 三年滚动预测（保守）

| 时点 | 注册用户 | 月活 MAU | 付费用户 | ARR | ARPU |
|---|:---:|:---:|:---:|:---:|:---:|
| M+3 | 1.5 万 | 6,000 | 200 | $7,000 | $35 |
| M+6 | 6 万 | 2.4 万 | 1,500 | $52,500 | $35 |
| M+12 | 25 万 | 8 万 | 8,000 | $290,000 | $36 |
| M+24 | 80 万 | 22 万 | 32,000 | $1,200,000 | $37.5 |
| M+36 | 200 万 | 50 万 | 90,000 | $3,500,000 | $39 |

→ **3 年保守 SOM ≈ $350 万 ARR**

### 4.3 乐观情景（爆款 + 分销 + B2B）

| 时点 | 付费用户 | ARR |
|---|:---:|:---:|
| M+12 | 1.8 万 | $700,000 |
| M+24 | 6 万 | $2.5M |
| M+36 | 16 万 | $7M |

→ **3 年乐观 SOM ≈ $700 万 ARR**

### 4.4 SOM 占 SAM 比例
- 保守 $3.5M / $1,500M = 0.23%（远未触及天花板）
- 乐观 $7M / $1,800M = 0.39%

## 五、瓶颈分析

### 5.1 主要增长瓶颈
1. **内容生产速度**：每月需新增 ≥ 5000 知识点 / 50 篇发现中国
2. **母语审校能力**：每语种每月 ≥ 20 万字审校量
3. **冷启动获客**：TikTok 内容生产者 / KOL 合作速度
4. **付费转化优化**：付费墙 A/B 测试需 8-12 周稳定

### 5.2 突破瓶颈的 X-factor
- **病毒式分销**：永久二级 + 知语币兑会员 → 自然 K-factor ≥ 1.2 即可指数增长
- **SEO 长尾**：12 类目 × 100 篇文章 × 4 语种 = 4800 个落地页
- **B2B 工厂团购**：单笔可达 $99-1000，突破单一 C 端获客瓶颈

## 六、与产品愿景的对齐

- **MVP 目标**：M+6 达到 1500 付费用户、M+12 达到 8,000 付费用户
- **盈亏平衡**：假设月成本 $5,000（基础设施 + 工资 + 营销），需 ARR ≥ $60,000 → 1500 付费用户即可达成
- **关键里程碑**：
  - M+3 达到 200 付费用户 = "PMF 验证完成"
  - M+12 达到 8,000 付费用户 = "PMF 强信号"
  - M+24 达到 32,000 付费用户 = "可启动 Series A 融资"（如果选择融资路径）

# 3.5 · Go-To-Market 策略

> 与项目方现有 `/research/` 推广方案完全对齐 + 增强细化

## 一、市场进入顺序

### 1.1 三市场优先级

| 顺序 | 市场 | 时间窗 | 理由 |
|:---:|---|---|---|
| 1 | 越南 | M+0 → M+12 | 数字化高、TikTok 渗透高、HSK 考生最多、内容生产成本低（越南语） |
| 2 | 泰国 | M+3 → M+18 | ARPU 高、中文学校多、华人圈层支持 |
| 3 | 印尼 | M+6 → M+24 | 人口红利、但 ARPU 低，需要更多本地化投入 |
| 4 | 菲 / 马 / 新 | M+18+ | 验证后扩展 |

### 1.2 三市场资源配比
- 越南 50% / 泰国 30% / 印尼 20%（M+0-12）
- M+12 后根据 LTV / CAC 数据动态调整

## 二、渠道策略（与推广方案完全对齐）

### 2.1 三平台主线（已对齐项目方推广方案）

```
TikTok（流量入口） → Facebook（社群信任） → YouTube（系统教学） → 知语（转化）
                                       ↓
                                 SEO（长期复利）
                                 WhatsApp / Telegram（推送）
```

### 2.2 TikTok 战术细化

#### 2.2.1 账号矩阵
- 每市场 1 主账号 + 3-5 矩阵号
- 主账号：知语官方（认证）
- 矩阵号：场景化（"工厂中文" / "HSK 急救" / "中文恋人"）
- 越南语 / 泰语 / 印尼语各 1 套

#### 2.2.2 内容公式
- 15-30 秒为主，60 秒次之
- 公式：母语吸睛开头 + 1 句中文场景 + 母语解释 + 行动钩子

#### 2.2.3 发布节奏
- 第 1 个月：每天 3 条 / 账号（密集起量）
- 第 2-3 月：每天 2 条 / 账号
- 第 4 月起：每天 1 条 / 账号 + 数据热门复用

#### 2.2.4 内容主题季度排期

| 季度 | 主题 |
|---|---|
| 春节季（2-3 月） | 节日 / 红包 / 生肖 |
| 夏季（5-7 月） | 旅游 / 美食 |
| HSK 考前（每年 4 月 / 10 月） | 应试急救 |
| 双 11 / 黑五 | 跨境电商砍价 |
| 国庆 / 中秋 | 文化 |

#### 2.2.5 KOL / KOC 合作
- 启动期：找 5-10 个粉丝 1 万-10 万的腰部 KOL（性价比最高）
- 增长期：找 1-2 个粉丝 100 万+ 的头部
- 合作模式：免费会员置换 → 知语币分销 → 现金 + 分销

### 2.3 Facebook 战术细化

#### 2.3.1 主页 + 社群双轮
- 主页：发布精修内容、官方公告、用户秀
- 社群（Group）：日常答疑、用户互助、签到打卡
- 启动条件：1000+ 粉丝后启动社群

#### 2.3.2 推广策略
- 不直接推 App，引导 → 主页链接 → 站内 H5 → 注册
- 在 Group 内可以适度引导（场景化）

#### 2.3.3 内容类型
- 长图文（10 句中文 + 母语场景）
- 视频长版（同 TikTok 同主题但 60-90 秒）
- 用户故事（学员秀）

### 2.4 YouTube 战术细化

#### 2.4.1 视频类型
- "5 分钟学 X 个商务中文" 系列
- "HSK X 级考前 30 天计划" 系列
- "工厂中文急救包" 系列
- "去中国旅游必学 100 句" 系列

#### 2.4.2 SEO 优化
- 标题母语 + 中文关键词双语
- 描述区放官方链接 + 章节时间码
- 字幕双语（自动 + 母语审校）

#### 2.4.3 节奏
- 每周 1 条长视频（10-15 分钟）+ 2-3 条 Shorts

### 2.5 SEO 战术（被低估，重点投入）

#### 2.5.1 站点架构
- 发现中国 12 类目 × 100 篇 × 4 语种 = 4800 个索引页
- 每个知识点 / 短句也可独立 URL（语言索引）
- 站点地图自动生成

#### 2.5.2 关键词
- 母语关键词："học tiếng Trung"（越）/ "เรียนภาษาจีน"（泰）/ "belajar bahasa mandarin"（印尼）
- 长尾："HSK 5 词汇"、"工厂中文 100 句"、"中国餐厅点餐"
- 中文 + 母语对照：每个发现中国短文都是一个长尾页

#### 2.5.3 内容策略
- 发现中国所有内容公开可索引（不锁登录）
- 拼音 + 母语 + 中文三语对照（极高 SEO 价值）
- TTS 音频公开（提升停留时长）

#### 2.5.4 反爬与 SEO 平衡
- HTML 渲染骨架 + 部分内容（首屏 3 句对照公开 → SEO + 用户体验）
- 后续内容延迟 API 加载 + 鉴权
- 即"前 3 句免费可索引，后续需登录"

### 2.6 WhatsApp / Telegram

#### 2.6.1 WhatsApp 频道（Broadcast）
- 越 / 泰 / 印尼各 1 个官方频道
- 每日推送：1 句金句 + 链接到站内

#### 2.6.2 Telegram Bot
- 实现"每日 5 句"机器人
- 用户可设置兴趣类目
- 兑换知语币（增长黑客）

## 四、Growth Loop 设计

### 4.1 主增长循环

```
TikTok 投放 / SEO
    ↓
新用户访问站内
    ↓
试学发现中国 + 完成分级测试
    ↓
注册（送 100 币）→ 第一节学习
    ↓
进入 4 模块体验（游戏 / 小说 / 文化）
    ↓
D1/D3/D7 钩子触发回流
    ↓
软付费墙触发付费
    ↓
学习里程碑分享 FB / Insta
    ↓
朋友点击链接 → 注册 → 上级得分销币
    ↓
币兑会员 / 皮肤 → 提升留存
    ↓ （回到顶端）
```

### 4.2 K-factor 估算
- 每付费用户平均分享 2 次
- 每次分享带回 0.3 个新注册
- K = 2 × 0.3 = 0.6（M+3 基线）
- 目标 M+12 K-factor ≥ 1.2 = 自然指数增长

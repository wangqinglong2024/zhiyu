## 二、4 个核心 Persona（详细画像）

### Persona A · "Linh" — 越南河内 HSK 备考学生
- **基础信息**：女，21 岁，河内国家大学外语系大三，月生活费 $250
- **学习目标**：明年 5 月考 HSK 5，准备申请北外交换生项目
- **当前方法**：Migii 题库 + 课本 + B站汉语教学视频（英文字幕）
- **痛点**：
  - "听汉语老师讲解英文听不太懂，看 B 站还要自己翻译"
  - "Migii 只有题没有系统讲解"
  - "每天没规划，碎片学习不知道在哪一级"
- **付费意愿**：$40 年会员可以接受，已为 Migii 付过 $15
- **使用场景**：早晨通勤地铁 30 分钟、宿舍晚自习 1 小时
- **设备**：iPhone 12 + 笔记本
- **语言**：越南语（母语）+ 英语（B2）+ 中文（学习中）
- **知语价值**：HSK 轨道 + 越南语讲解 + 系统课程 + 模拟考

### Persona B · "Somchai" — 泰国春武里中资工厂年轻工人
- **基础信息**：男，26 岁，泰国春武里中资工厂技术员，月薪 $450
- **学习目标**：能听懂中国主管的指令、能向 QC 反馈问题
- **当前方法**：偶尔问华人同事 + 看 TikTok 上"工厂中文"教学视频
- **痛点**：
  - "工厂里专业词汇课本上没有"
  - "只会说'你好''谢谢'，遇到事情说不出"
  - "买了几本课本翻不完"
- **付费意愿**：$4/月可以接受，单次付费 $4-8 一段课最舒服
- **使用场景**：午休 30 分钟、下班通勤 20 分钟、周末
- **设备**：Vivo Y20s（中端 Android）
- **语言**：泰语（母语）+ 极少英语 + 中文初学
- **知语价值**：工厂轨道 + 泰语讲解 + 单段付费 + 离线（v2）

### Persona C · "Budi" — 印尼雅加达跨境电商小卖家
- **基础信息**：男，30 岁，雅加达，TikTok Shop + Lazada 卖家，月入 $1500
- **学习目标**：能直接和 1688 供应商砍价、能看懂中文产品规格
- **当前方法**：Google Translate + 微信里加华人代购朋友帮翻
- **痛点**：
  - "Google 翻译不准确，每次砍价都很尴尬"
  - "供应商发的中文规格表完全看不懂"
  - "想学但找不到电商场景的课"
- **付费意愿**：$40 年会员 + $4 单段都接受
- **使用场景**：办公室碎片时间、出差路上
- **设备**：Samsung Galaxy A 系列 + iPad
- **语言**：印尼语（母语）+ 英语（B2） + 中文（A0）
- **知语价值**：电商轨道 + 印尼语讲解 + 1688 / 砍价场景

### Persona D · "Mai" — 越南胡志明 30 岁中文兴趣者
- **基础信息**：女，29 岁，胡志明市，市场推广，月薪 $600
- **学习目标**：看中国短剧不依赖字幕、明年去上海旅游
- **当前方法**：Duolingo（中文模块到 30%）+ TikTok 中文短视频
- **痛点**：
  - "Duolingo 中文太浅，且不接地气"
  - "学了一年只会说'你好'还是看不懂短剧"
  - "想找有趣的方式而不是教科书"
- **付费意愿**：$4-12 可以试，但需要被打动
- **使用场景**：晚上 30 分钟睡前、周末游戏
- **设备**：iPhone 13 Pro
- **语言**：越南语 + 英语（B1）+ 中文（A1-A2）
- **知语价值**：日常轨道 + 发现中国 + 小说 + 游戏 + 越南语讲解

## 三、Jobs-to-be-Done 框架

### JTBD-1: 通过 HSK 拿奖学金
- 触发：考试报名、奖学金申请截止
- 期望工作：系统性学习 + 模拟考 + 错题分析
- 对应：Persona A，HSK 轨道

### JTBD-2: 工作上不被裁员
- 触发：工厂雇佣外籍要求中文沟通
- 期望工作：实用中文 + 场景对话
- 对应：Persona B，工厂轨道

### JTBD-3: 生意做到中国客户
- 触发：跨境业务遇到沟通障碍
- 期望工作：商务用语 + 砍价术
- 对应：Persona C，电商轨道

### JTBD-4: 不依赖字幕看懂中国内容
- 触发：刷到中国短剧 / 综艺 / 短视频
- 期望工作：词汇量 + 听力 + 文化理解
- 对应：Persona D，日常 + 发现中国 + 小说

### JTBD-5: 朋友 / 伴侣是中国人
- 触发：跨国感情建立
- 期望工作：日常对话 + 文化理解
- 对应：Persona E（未细化）

## 四、用户旅程地图（Persona A 为例）

```
1. 发现期
   └ 看到越南语 KOL TikTok 介绍知语
   └ 进官网无登录试学发现中国 3 篇
   └ 感觉"母语讲解太爽了"

2. 探索期
   └ 注册（邮箱 / Google）→ 送 100 知语币
   └ 完成 3 题分级测试 → 推荐 HSK 轨道
   └ 学完 HSK 1 阶段第 1 章（免费）

3. 价值感知期
   └ 完成 3 节小测，看到进度条
   └ 解锁第 1 个游戏皮肤（用初始 100 币）
   └ 收到 D1 push："你今天学了 12 个词"

4. 付费转化期
   └ 学到 HSK 1 阶段第 5 章，软付费墙出现
   └ A/B 测试结果：年付 $40 文案 vs 半年 $12 文案
   └ 选择半年 $12 → Paddle 跳转支付

5. 留存期
   └ 100 天打卡挑战
   └ 学习里程碑分享 FB
   └ 朋友点击分享链接 → 注册 → 自己拿到分销币

6. 续费 / 升级期
   └ 半年到期前 7 天 email 提醒
   └ 选择年付 $40（更划算）
   └ 学到 HSK 4 后单段购买电商轨道
```

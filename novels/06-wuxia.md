# 06 — 武侠江湖（Wuxia · Jianghu）

> 一壶浊酒喜相逢，古今多少事都付笑谈中 → 金庸古龙的永恒经典

---

## 类目定位

武侠在越南拥有 50 年深厚根基（金庸译本是越南文学市场长青常销 TOP10），泰国华人社区同样追捧。本类目分**经典武侠（金古黄温梁）+ 新派武侠（凤歌、孙晓、藤萍）**两条线。

---

## 内容范围

### 1. 金庸经典系列
- 射雕三部曲：射雕英雄传 / 神雕侠侣 / 倚天屠龙记
- 天龙八部 / 笑傲江湖 / 鹿鼎记
- 越文译本已有 70 年代经典版本，引入新译版本

### 2. 古龙经典
- 楚留香 / 陆小凤 / 小李飞刀
- 古龙特色：短句、意境、孤独英雄
- 极适合 TTS 朗读（节奏感强）

### 3. 黄易系列
- 大唐双龙传 / 寻秦记
- 武侠 + 历史 + 玄学

### 4. 新派武侠
- 凤歌《昆仑》《沧海》
- 小椴《杯雪》
- 时未寒《明将军》

### 5. 江湖小品
- 短篇侠客录（10-30 章 / 部）
- 客栈群像 / 镖局 / 衙门捕快
- 适合通勤 / 睡前听书

### 6. 女侠 / 女主角武侠
- 黄蓉 / 周芷若 / 任盈盈视角重写
- 现代笔法新人原创女主武侠
- 平衡女性读者市场

---

## 内容边界

| 包含 | 不包含 |
|------|--------|
| 经典门派 / 江湖恩怨 | 影射真实民族纠纷 |
| 武功体系（内功 / 外功 / 暗器）| 涉现实暴力教唆 |
| 历史朝代背景 | 政治化历史评价 |

---

## MVP 作品清单（3 部）

1. **《射雕英雄传》** —— 经典完整版，授权新译
2. **《楚留香传奇》** —— 古龙系列代表作
3. **《女镖头》** —— 原创新派女主武侠

## 重点 IP

- 金庸全套 15 部
- 古龙武侠精选
- 凤歌《昆仑》《沧海》

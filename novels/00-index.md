---

## 设计原则

1. **纯热门导向**：不与课程系统强绑定，纯粹按起点 / 喜马 / 越泰当地小说平台的热门榜单挑选 12 大类目。
2. **东南亚口味本地化**：越南偏爱"修仙 / 重生 / 言情霸总"，泰国偏爱"BL 耽美 / 古风言情 / 玄幻"，权重已倾斜。
3. **句对照阅读 / 听**：每篇小说 **每句话一行**，中文 + 本地语言（越 / 泰 / 英）三语对照，可文字阅读 + TTS 听书。
4. **章节制**：每部小说按 **章** 切分；每章 1500-3000 字（中文计），平均 60-120 句。
5. **音频先行**：所有小说默认提供 TTS 朗读（多角色女声/男声切换），对标喜马拉雅有声书体验。

## 内容规模规划

| 阶段 | 每类目作品数 | 每作品章节 | 总章节 |
|------|----------|------|------|
| MVP | 3 部（其中 1 部完结，2 部连载中）| 20-50 章 | ≈ 1000 章 |
| 上线 6 月 | 8 部 | 50-100 章 | ≈ 5000 章 |
| 上线 12 月 | 20+ 部 | 100-300 章 | ≈ 30000 章 |

> 内容来源：①已购买信息网络传播权的正版授权 ②与起点国际 / 阅文海外（WebNovel）合作分销 ③站内征文激励原创作者。

## 阅读 / 听书产品形态

| 模块 | 描述 |
|---|---|
| 📖 双语对照阅读 | 中文原文一句一行，下方/右侧本地语对照，长按单字弹出释义 |
| 🎧 TTS 朗读 | 多音色 / 双倍速 / 章节自动续播，仿喜马拉雅 |
| 🌐 三语切换 | 中 / 越 / 泰 / 英 一键切换译文 |
| ✏️ 生词收藏 | 长按汉字 → 加入生词本 → 可在系统课程"我的生词"复习 |
| 🔁 跨模块联动 | 文末"游戏挑战"按钮：用本章高频词开启汉字忍者一局 |
| 💬 章评 / 弹幕 | 仿起点本章说 + Bilibili 弹幕，社区氛围 |
| ⭐ 月票 / 打赏 | 商业化通路，对接本地支付（MoMo / TrueMoney） |

## 内容审核与本地合规

| 风险类型 | 处理 |
|---|---|
| 政治敏感 | 全站审核绕行（如朝代影射当代） |
| 民族 / 宗教 | 涉及越南 / 泰国本地宗教 / 王室必须人工复审 |
| 色情 / 极暴力 | 分级显示，未成年账号屏蔽；耽美 / 百合走"高糖低肉"线 |
| 越南法律敏感 | 不收录任何涉及南海 / 边境 / 中越战争题材 |
| 泰国法律敏感 | 严禁任何涉王室、僧侣不敬内容；佛教元素需正面 |

详细规则同步至 [research/推广方案/13-风险与应急预案.md](../research/推广方案/13-风险与应急预案.md)。

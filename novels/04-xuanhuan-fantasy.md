# 04 — 玄幻奇幻（Xuanhuan / High Fantasy）

> 斗气大陆，强者为尊 → 起点男频长期 TOP3 类目

---

## 类目定位

参考起点白金作家阵容（天蚕土豆、唐家三少、辰东）。与"03 仙侠修真"区分：**仙侠 = 中国道家底色 + 古典中国背景**；**玄幻 = 西式魔法 / 异世大陆 / 自创修炼体系**。越南粉丝同样庞大。

---

## 内容范围

### 1. 异世大陆 / 斗气大陆
- 主角穿越或转生到异世
- 斗气 / 魔法 / 武魂 / 圣域 等自创体系
- 经典模板：《斗破苍穹》《武动乾坤》

### 2. 系统流
- 主角获得"系统"金手指
- 任务奖励 / 抽奖 / 商城 / 签到
- 套路爽文，节奏快，新人友好

### 3. 召唤流
- 召唤历史名将 / 神话人物 / 异界生物
- 经典：《召唤万岁》

### 4. 王朝 / 领主流
- 主角建立家族 / 王国 / 帝国
- 内政 + 战争 + 外交
- 类游戏经营乐趣

### 5. 西方魔幻
- 龙与魔法 / 精灵矮人
- 完全西式背景的中文小说
- 适合喜欢《冰与火之歌》风格的读者

### 6. 神话同人
- 北欧神话 / 希腊神话 / 中国神话同人
- 旧神觉醒 / 神器降世

## MVP 作品清单（3 部）

1. **《斗破苍穹》** —— 完结，授权译制（东南亚必读）
2. **《系统：从零开始的斗气大陆》** —— 连载，原创系统流
3. **《诸神之怒》** —— 北欧 + 中国神话融合，原创

# 09 — 灵异盗墓（Horror, Tomb Raiding & Folk Mystery）

> 鬼吹灯一开，墓中珍宝现 → 中文网文最具东方特色的输出 IP

---

## 类目定位

参考天下霸唱、南派三叔。盗墓 / 灵异题材在越南 / 泰国深度契合本地民俗鬼神文化（越南鬼魂、泰国佛教灵异），是文化亲和度最高的中式题材之一。

---

## 内容范围

### 1. 盗墓寻宝
- 鬼吹灯系列：摸金校尉 / 风水 / 罗盘
- 盗墓笔记系列：长白山 / 西沙海底 / 张家
- 行业术语 + 探险节奏

### 2. 风水志怪
- 中国传统风水堪舆
- 阴宅阳宅 / 罗盘八卦
- 现代风水师悬疑

### 3. 民俗灵异
- 各地民俗鬼故事（赶尸 / 蛊术 / 镇魂）
- 苗疆 / 湘西 / 西藏地域特色
- 中越边境 / 中泰边境特殊民俗（本地化加分）

### 4. 现代灵异
- 都市灵异侦探（捉鬼 / 驱魔）
- 算命先生 / 风水师 / 道士主角
- 现代都市背景的灵异群像

### 5. 古宅鬼怪
- 老宅闹鬼系列
- 主角入宅调查
- 单元剧结构，听书友好

### 6. 民国奇谈
- 民国上海 / 北平的奇人异事
- 戏班 / 茶馆 / 古董行群像
- 民俗 + 悬疑双重魅力

---

## 内容边界

| 包含 | 不包含 |
|------|--------|
| 中国传统民俗鬼神 | 抹黑越南 / 泰国本土宗教 |
| 历史朝代古墓探险 | 涉真实文物 / 考古机构 |
| 风水玄学 | 教唆迷信活动（如出马仙骗局） |
| 民国奇谈 | 涉敏感历史事件 |

> ⚠️ **泰国合规**：避免任何对僧侣、佛像不敬描写；涉及泰国本地灵异元素必须人工复审。
> ⚠️ **越南合规**：避免对越南民间信仰（道母、灵祠）猎奇化；涉中越边境民俗需平等尊重。

---

## MVP 作品清单（3 部）

1. **《鬼吹灯·精绝古城》** —— 天下霸唱，授权译制
2. **《湘西赶尸笔记》** —— 原创民俗灵异
3. **《西贡老宅》** —— 原创，越南法属老宅灵异（本地特化）

# 03 — 仙侠修真（Xianxia & Cultivation）

> 我命由我不由天 → 越南"Tu Tiên / Tiên Hiệp"是当地最大男频类目

---

## 类目定位

仙侠修真在越南有 1000 万 + 长期阅读用户群（参考 TruyenYY、MeTruyen 平台），是中国网文最成功的输出品类。本类目主打**凡人流 + 系统流 + 师徒情**。

---

## 内容范围

### 1. 凡人修仙流
- 资质平凡的少年踏上修仙路
- 经典模板：《凡人修仙传》（忘语）—— 越南奉为神作
- 法器 / 丹药 / 阵法 / 符箓 / 灵兽全套设定

### 2. 仙侠正派
- 名门正派传承（昆仑 / 蜀山 / 青城）
- 行侠仗义、伏魔卫道
- 渡劫飞升 / 三界格局

### 3. 邪修 / 魔道
- 反派视角崛起
- 黑化 / 心魔 / 因果
- 邪正互转的复杂人物

### 4. 师徒虐恋（清水）
- 师父 × 徒弟（异性 / 同性走 12 类目）
- 禁忌之恋的纯粹版
- 师门情义 vs 个人感情

### 5. 双修 / 双生
- 道侣文化（仙侣双修）
- 双男主 / 双女主修仙路
- 共同渡劫的羁绊

### 6. 末法 / 都市修仙
- 都市背景下的隐世修仙者
- 现代科技 vs 古老法术
- 末法时代的最后一线希望

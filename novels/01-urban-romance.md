# 01 — 都市言情（Urban Romance）

> 当代背景下的爱情故事，霸总 / 契约 / 甜宠 / 复仇打脸 → 越泰女性读者第一大票仓

---

## 类目定位

参考起点女频、晋江白金、Wattpad 越南"Ngôn tình hiện đại"榜单，主打**情绪爽感**：开局虐 → 中段宠 → 结局复仇打脸。东南亚女性读者超 70% 集中在此类。

---

## 内容范围

### 1. 霸道总裁文
- 契约结婚 / 替身新娘 / 失忆复婚
- 豪门联姻 / 商战 + 恋爱双线
- 经典套路：误会 → 冷战 → 真相 → 追妻火葬场

### 2. 甜宠文
- 校园甜宠：青梅竹马 / 学长学妹
- 职场甜宠：同事 / 上下级 / 行业同行
- 婚后甜宠：先婚后爱 / 闪婚

### 3. 复仇打脸文
- 渣男前夫追妻被拒
- 婆媳 / 小三 / 白莲花反派打脸
- 重生爽文（与「05 穿越重生」轻度交叉）

### 4. 行业文
- 医生 / 律师 / 建筑师 / 演员 / 主持人 / 时尚设计师 / 美食博主
- 强调女主独立事业，男主仅辅助

### 5. 治愈日常
- 单亲妈妈 / 离异女性的二次幸福
- 邻居 / 闺蜜 / 同事友情线

---

## 内容边界

| 包含 | 不包含 |
|------|--------|
| 高糖恋爱 / 商战副线 | 真实当代政商内幕（敏感） |
| 渣男渣女打脸爽点 | 极端暴力 / 凌虐情节 |
| 隐喻 / 暗示性场景 | 露骨情节（送泰国分级版另开） |
| 跨国恋（中越 / 中泰）专题 | 涉国家政治、领土争议 |

---

## MVP 作品清单（3 部）

1. **《契约新娘的逆袭》** —— 完结 60 章，霸总 + 复仇打脸
2. **《医生你别凶》** —— 连载中（已 35 章），急诊室甜宠
3. **《嫁到中国》** —— 原创越南女主×中国丈夫跨文化甜文
